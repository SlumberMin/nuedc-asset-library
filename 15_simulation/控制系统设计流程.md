## 2. 需求分析与性能指标

### 2.1 典型性能指标

#### 时域指标

| 指标 | 符号 | 典型要求 | 说明 |
|------|------|---------|------|
| 上升时间 | $t_r$ | < 0.5s | 首次到达终值的时间 |
| 调节时间 | $t_s$ | < 2s | 进入±2%或±5%误差带 |
| 超调量 | $\sigma\%$ | < 10%~20% | 峰值超过终值的百分比 |
| 稳态误差 | $e_{ss}$ | < 1% | 稳态时与期望值之差 |
| 跟踪精度 | - | < 0.5° | 对时变信号的跟踪 |

#### 频域指标

| 指标 | 符号 | 典型要求 | 说明 |
|------|------|---------|------|
| 增益裕度 | $G_m$ | > 6dB | 稳定性裕量 |
| 相位裕度 | $\phi_m$ | > 45° | 稳定性裕量 |
| 带宽 | $\omega_b$ | 按需确定 | 系统响应速度 |
| 灵敏度峰值 | $M_s$ | < 2 (6dB) | 鲁棒性指标 |

#### 电赛常见约束

- **控制精度**: 角度 ±0.5°, 位置 ±1mm, 温度 ±0.5°C
- **响应速度**: 阶跃响应 < 1~3s
- **抗扰动**: 负载突变后恢复 < 2s
- **执行器限制**: PWM占空比 0~100%, 电压 ±24V

## 3. 系统建模

### 3.1 机理建模法

基于物理定律建立微分方程。

#### 电机系统
```
电气方程: L·di/dt + R·i = V - Ke·ω
机械方程: J·dω/dt + B·ω = Kt·i - T_load
传递函数: Ω(s)/V(s) = Kt / [(Ls+R)(Js+B) + Kt·Ke]
```

#### 温度系统
```
热平衡方程: C·dT/dt = P·u - k·(T - T_amb)
传递函数: T(s)/U(s) = P/(Cs+k) · 1/s
```

#### RLC电路
```
V(s)/I(s) = (Ls² + Rs + 1/C) / s
```

#### 倒立摆
```
非线性方程: (M+m)·x'' + m·l·θ''·cos(θ) - m·l·θ'²·sin(θ) = F
            l·θ'' + g·sin(θ) = -x''·cos(θ)
线性化(小角度): 近似为二阶不稳定系统
```

### 3.2 实验辨识法

#### 阶跃响应法

```
给系统一个阶跃输入 u₀，记录输出 y(t)

一阶系统: G(s) = K/(τs+1)
  - K = Δy/Δu (稳态增益)
  - τ = 输出到达63.2%终值的时间

二阶系统: G(s) = K·ωn²/(s² + 2ζωn·s + ωn²)
  - 从超调量和振荡频率推算 ζ 和 ωn
```

#### 频率响应法

```
输入一系列正弦信号，测量幅值比和相位差
绘制Bode图，拟合传递函数
```

### 3.3 模型简化原则

1. **保留主导极点**: 远离虚轴的极点可忽略
2. **小惯量近似**: τ << T 的小时间常数可忽略
3. **延迟近似**: e^(-Ls) ≈ 1/(Ls+1) (当 L << T 时)
4. **非线性线性化**: 在工作点附近Taylor展开

## 4. 控制器选型

### 4.1 选型决策树

```
被控对象是否线性？
├── 是 → 模型是否精确已知？
│   ├── 是 → 性能要求高？
│   │   ├── 是 → LQR / H∞最优控制
│   │   └── 否 → PID / 状态反馈
│   └── 否 → 参数变化快？
│       ├── 是 → 自适应控制 (MRAC / 梯度下降)
│       └── 否 → 鲁棒控制 (H∞ / μ综合)
└── 否 → 非线性程度？
    ├── 弱 → 在工作点线性化 + PID
    ├── 中 → 反馈线性化 / 滑模控制
    └── 强 → 反步法 / 非线性MPC
```

### 4.2 各方法对比

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **PID** | 简单、成熟、无需精确模型 | 非线性/时变系统性能差 | 大多数工业过程 |
| **状态反馈** | 可配置所有极点 | 需要全状态测量或观测器 | 状态可测系统 |
| **LQR** | 最优性、保证稳定性 | 需要精确模型 | 模型已知的线性系统 |
| **MPC** | 处理约束、多变量 | 计算量大 | 有约束的优化问题 |
| **自适应** | 处理参数不确定性 | 收敛性分析复杂 | 参数缓慢变化 |
| **鲁棒(H∞)** | 对不确定性有保证 | 保守、设计复杂 | 不确定性可描述 |
| **滑模** | 对匹配不确定性鲁棒 | 抖振问题 | 非线性鲁棒控制 |
| **反馈线性化** | 精确线性化 | 需要精确模型 | 可线性化非线性系统 |
| **反步法** | 系统化设计、Lyapunov稳定 | 仅限严格反馈形式 | 级联非线性系统 |

### 4.3 电赛推荐方案

| 题目类型 | 推荐方案 | 备选方案 |
|---------|---------|---------|
| 温度控制 | PID + 前馈 | 自适应PID |
| 电机调速 | 增量式PID | 状态反馈 + 观测器 |
| 平衡控制 | PID + 卡尔曼滤波 | LQR + LQE |
| 信号跟踪 | 重复控制 + PID | 自适应控制 |
| 电源稳压 | 电压/电流双环PID | 滑模控制 |
| 运动控制 | 位置-速度-电流三环PID | MPC |

## 5. 仿真验证

### 5.1 仿真流程

```
1. 建立被控对象模型
2. 设计控制器
3. 开环仿真 → 验证模型
4. 闭环仿真 → 验证控制性能
5. 参数灵敏度分析 → 验证鲁棒性
6. Monte Carlo仿真 → 验证不确定性下的性能
```

### 5.2 Python仿真模板

```python
import numpy as np
import matplotlib.pyplot as plt

# 系统参数
dt = 0.01       # 采样周期
T = 10.0        # 仿真时长
N = int(T/dt)

# 被控对象（以二阶系统为例）
A = np.array([[0, 1], [-2, -3]])
B = np.array([[0], [1]])
C = np.array([[1, 0]])
x = np.array([[0], [0]])  # 初始状态

# 控制器参数
Kp, Ki, Kd = 5.0, 2.0, 0.5
integral = 0; prev_e = 0

# 记录数据
t_log = []; y_log = []; u_log = []

for i in range(N):
    t = i * dt
    ref = 1.0  # 阶跃参考
    y = (C @ x)[0, 0]
    e = ref - y
    
    # PID控制
    integral += e * dt
    de = (e - prev_e) / dt
    u = Kp*e + Ki*integral + Kd*de
    prev_e = e
    
    # 状态更新（Euler法）
    x = x + (A @ x + B * u) * dt
    
    t_log.append(t); y_log.append(y); u_log.append(u)

# 绘图
fig, (ax1, ax2) = plt.subplots(2, 1)
ax1.plot(t_log, y_log, label='输出')
ax1.axhline(y=1.0, linestyle='--', color='gray', label='参考')
ax1.legend(); ax1.grid(True)
ax2.plot(t_log, u_log, label='控制量')
ax2.legend(); ax2.grid(True)
plt.savefig('simulation_result.png', dpi=150)
```

### 5.3 验证清单

- [ ] 阶跃响应：上升时间、超调量、调节时间
- [ ] 斜坡跟踪：稳态跟踪误差
- [ ] 正弦跟踪：不同频率下的跟踪精度
- [ ] 阶跃扰动抑制：负载突变后的恢复
- [ ] 参数摄动：±20%参数变化下的稳定性
- [ ] 噪声影响：测量噪声对控制量的影响
- [ ] 执行器饱和：控制量限幅后的行为
- [ ] 采样率影响：不同采样周期的性能

```c
// 主循环框架
int main(void) {
    System_Init();
    Timer_Init(CONTROL_PERIOD);  // 定时器中断

        // 主循环：通信、显示、非实时任务

// 控制中断（定时器触发）
void TIM_IRQHandler(void) {
    // 1. 采样传感器

    // 2. 计算参考值

    // 3. 控制算法
    float u = Controller_Update(ref, y);
    
    // 4. 输出执行
    PWM_Set(u);
    
    // 5. 记录数据
    Log_Data(y, u);
}
```

## 7. 调试与整定

### 7.1 PID整定方法

#### Ziegler-Nichols法（经典）

```
1. 令 Ki=0, Kd=0
2. 逐渐增大 Kp 直到系统等幅振荡
3. 记录临界增益 Ku 和振荡周期 Tu
4. 按下表计算:

┌──────────┬──────────┬──────────┬──────────┐
│ 控制类型  │   Kp     │   Ki     │   Kd     │
├──────────┼──────────┼──────────┼──────────┤
│ P        │ 0.5·Ku   │    -     │    -     │
│ PI       │ 0.45·Ku  │ 0.54·Ku/Tu│   -     │
│ PID      │ 0.6·Ku   │ 1.2·Ku/Tu│ 0.075·Ku·Tu│
└──────────┴──────────┴──────────┴──────────┘
```

#### 工程整定口诀

```
先比例，后积分，最后加微分
比例带大慢慢调，振荡发赶紧减
积分消除稳态差，太大会有积分饱
微分预测变化势，噪声大了要慎用
```

### 7.2 常见问题诊断

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 持续振荡 | Kp过大 | 减小Kp |
| 缓慢爬行 | Ki过大 | 减小Ki或加积分限幅 |
| 噪声大 | Kd过大或采样噪声 | 减小Kd，加滤波 |
| 稳态误差 | 缺少积分 | 增加Ki |
| 超调大 | Kp或Ki过大 | 加微分，减小Kp |
| 响应慢 | Kp过小 | 增大Kp |
| 饱和振荡 | 积分饱和 | 抗积分饱和(Anti-windup) |

### 7.3 抗积分饱和

```python
# 条件积分法
if abs(u) < u_max:
    integral += e * dt  # 只在未饱和时积分

# 回算法
if abs(u) > u_max:
    integral -= (u - np.clip(u, -u_max, u_max)) / Ki
```

### 8.2 加分策略

1. **基础分必拿**: 先实现基本要求，再追求发挥部分
2. **仿真先行**: 用Python快速验证算法，节省硬件调试时间
3. **模块化设计**: 传感器、控制器、执行器独立模块，方便替换
4. **参数自整定**: 加入在线参数调整功能，应对不同工况
5. **鲁棒性设计**: 控制器在宽参数范围内都能工作

### 8.3 常用电赛控制技巧

#### 前馈+反馈复合控制
```python
# 前馈: 基于模型的开环补偿
u_ff = ref * model_gain

# 反馈: PID消除残余误差
u_fb = Kp*e + Ki*integral + Kd*de

# 复合
u = u_ff + u_fb
```

#### 增量式PID（防突变）
```python
delta_u = Kp*(e - prev_e) + Ki*e*dt + Kd*(e - 2*prev_e + prev2_e)
u += delta_u
```

#### 变参数PID（分段控制）
```python
if abs(e) > threshold:
    Kp, Ki, Kd = Kp_fast, 0, Kd_fast  # 大误差：快速响应
else:
    Kp, Ki, Kd = Kp_slow, Ki_slow, Kd_slow  # 小误差：精确控制
```
