## 概述

本目录包含电赛常用控制算法的仿真程序，用于：
- **算法验证**：在实际硬件之前验证控制算法可行性
- **参数调优**：通过仿真快速找到合适的 PID 参数
- **教学演示**：直观理解控制原理和参数影响

| 文件 | 功能 | 核心算法 |
|------|------|----------|
| `pendulum_swing_up.py` | 倒立摆摆起 | 能量控制 + LQR |
| `ball_tracking_simulation.py` | 小球视觉追踪 | PID + 云台控制 |
| `line_following_pid_simulation.py` | 循迹小车 | PID 转向控制 |
| `servo_tuning_simulation.py` | 舵机参数调优 | PID 响应分析 |

## 倒立摆摆起仿真

### 文件：`pendulum_swing_up.py`

### 原理
倒立摆从下垂位置摆起到倒立位置，分两个阶段：
1. **能量控制阶段**：通过往复摆动注入能量，逐步增大摆幅
2. **LQR 稳定阶段**：当摆接近倒立位置时，切换到 LQR 线性最优控制

### 物理模型
- 小车质量 M = 0.5 kg
- 摆杆质量 m = 0.1 kg
- 摆杆半长 l = 0.5 m
- 摩擦系数 b = 0.1

### 控制策略
```
能量控制: u = k * (E - E_desired) * θ̇ * cos(θ)
切换条件: |θ - π| < 14.3° 且 |θ̇| < 3 rad/s
LQR: u = -K * x (状态反馈)
```

### 输出
- 摆杆角度曲线（标注切换时刻）
- 控制力输入曲线
- 摆杆能量变化曲线
- 相图（角度 vs 角速度）

### 参数修改
```python
# 修改物理参数
M = 0.5     # 小车质量
m = 0.1     # 摆杆质量
l = 0.5     # 摆杆半长

# 修改控制参数
k_energy = 2.0  # 能量控制增益

# 修改 LQR 权重矩阵
Q = np.diag([100, 1, 200, 1])  # 状态权重
R = np.array([[0.01]])          # 控制权重
```

## 循迹PID仿真

### 文件：`line_following_pid_simulation.py`

### 原理
模拟 5 路传感器循迹小车在不同赛道上的 PID 转向控制。

### 赛道类型
| 赛道 | 特点 | 难度 |
|------|------|------|
| 直线 | 基准测试 | ★ |
| 弯道 | 平缓弯道 | ★★ |
| S弯 | 连续反向弯 | ★★★ |
| 急转弯 | 大角度转弯 | ★★★★ |
| 复杂赛道 | 综合测试 | ★★★★★ |

### PID 参数对比
程序自动对比三组参数：
- **保守**：Kp=1.0, Kd=0.5（稳定但慢）
- **标准**：Kp=2.0, Kd=1.0（平衡）
- **激进**：Kp=4.0, Kd=2.0（快但可能振荡）

### 输出
- 5种赛道的追踪轨迹图
- 不同PID参数在S弯上的对比
- 横向误差随时间变化对比

### 参数修改
```python
# 车辆参数
car.speed = 200          # 行驶速度 mm/s
car.max_steer = 0.4      # 最大转向角 rad
sensor_spacing = 15      # 传感器间距 mm

# PID 参数
pid_configs = {
    'my_config': {'kp': 3.0, 'ki': 0.1, 'kd': 1.5},
}
```

## 舵机调参仿真

### 文件：`servo_tuning_simulation.py`

### 原理
建立舵机二阶系统模型（含死区、饱和、响应延迟），分析 PID 参数对响应的影响。

### 舵机模型特性
```python
自然频率 ωn = 30 rad/s
阻尼比 ζ = 0.6
死区 = 0.5°
最大角速度 = 600°/s
响应延迟 = 3ms
```

### 分析内容

#### 1. Kp 影响
| Kp | 上升时间 | 超调 | 适用场景 |
|----|----------|------|----------|
| 0.5 | 慢 | 无 | 不推荐（太慢） |
| 1.5 | 中等 | 小 | 低精度要求 |
| 3.0 | 快 | 中等 | 一般应用 |
| 6.0 | 很快 | 大 | 不推荐（振荡） |

#### 2. Kd 影响
- Kd 增大 → 超调减小，但噪声放大
- Kd 过大 → 高频振荡

#### 3. Ki 影响
- Ki 消除稳态误差
- Ki 过大 → 积分饱和，超调增大

### 输出
- Kp/Kd/Ki 参数影响对比图（各4组参数）
- 推荐参数对比图

### 参数修改
```python
# 修改舵机模型
servo.omega_n = 30.0    # 自然频率
servo.zeta = 0.6        # 阻尼比
servo.deadzone = 0.5    # 死区

# 添加自定义 PID 配置
my_config = {'kp': 2.5, 'ki': 1.0, 'kd': 0.12, 'name': '自定义'}
```

## 常见问题

### Q: 运行报错 `ModuleNotFoundError: No module named 'matplotlib'`
```bash
pip install matplotlib numpy
```

### Q: 图表中文显示为方块
```python
# 在代码开头添加
import matplotlib
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
matplotlib.rcParams['axes.unicode_minus'] = False
```

### Q: 倒立摆摆不起来
- 检查能量控制增益 `k_energy` 是否足够
- 增大控制力限幅 `np.clip(u, -20, 20)` 中的 20

### Q: 循迹小车出界
- 减小 `car.speed`
- 增大 Kp 值
- 检查传感器间距是否合理

### Q: 如何将仿真参数应用到实际硬件？
1. 仿真中找到合适的 PID 参数范围
2. 实际硬件上从保守参数开始
3. 逐步增大 Kp 直到出现轻微振荡
4. 加入 Kd 抑制振荡
5. 如有稳态误差，少量加入 Ki
