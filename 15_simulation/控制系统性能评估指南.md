## 1. 性能指标体系

### 1.1 基本性能指标分类

| 类别 | 指标 | 符号 | 单位 | 说明 |
|------|------|------|------|------|
| 快速性 | 上升时间 | $t_r$ | s | 首次达到终值的时间 |
| 快速性 | 峰值时间 | $t_p$ | s | 达到最大值的时间 |
| 平稳性 | 超调量 | $\sigma\%$ | % | $(y_{max}-y_{ss})/y_{ss} \times 100\%$ |
| 平稳性 | 衰减比 | $\psi$ | - | 相邻超调之比 |
| 准确性 | 稳态误差 | $e_{ss}$ | - | $r(t) - y(\infty)$ |
| 准确性 | 跟踪精度 | - | % | 稳态相对误差 |
| 抗扰性 | 最大偏差 | $\Delta y_{max}$ | - | 扰动引起的最大偏差 |
| 抗扰性 | 恢复时间 | $t_{rec}$ | s | 偏差回到容许范围的时间 |

### 1.2 性能指标详解

#### 1.2.1 阶跃响应指标

```
           y(t)
           ↑
    y_max ─┤·····*  ← 超调
           │    / \
    y_ss ──┤───/───\─────────  ← 稳态值
           │  /     ‾‾‾‾‾‾‾
           │ /
     0% ───┤/
           └──┬──┬──┬──────── t
              t_r t_p  t_s
```

**关键指标计算公式：**

- **上升时间**: $t_r = t_{90\%} - t_{10\%}$（10%-90%定义）
- **调节时间**: 最后进入±Δ%带的时间（通常Δ=2%或5%）
- **超调量**: $\sigma\% = \frac{y_{max} - y_{ss}}{y_{ss}} \times 100\%$

#### 1.2.2 误差积分指标

| 指标 | 公式 | 特点 | 适用场景 |
|------|------|------|----------|
| IAE | $\int_0^\infty |e(t)|dt$ | 简单直观 | 一般评估 |
| ISE | $\int_0^\infty e^2(t)dt$ | 惩罚大误差 | 安全关键 |
| ITAE | $\int_0^\infty t|e(t)|dt$ | 抑制长时间误差 | 调节时间优化 |
| ITSE | $\int_0^\infty te^2(t)dt$ | 综合考虑 | 性能优化 |
| ISTE | $\int_0^\infty t^2e^2(t)dt$ | 强调后期误差 | 精密控制 |

## 2. 时域评估方法

### 2.1 阶跃响应测试

```python
import numpy as np
from scipy.signal import lti, step

def step_response_analysis(sys, setpoint=1.0):
    """阶跃响应性能分析"""
    t, y = step(sys)
    
    y_ss = y[-1]  # 稳态值
    y_max = np.max(y)
    
    # 超调量
    overshoot = max(0, (y_max - y_ss) / y_ss * 100)
    
    # 上升时间 (10%-90%)
    try:
        t_10 = t[next(i for i, v in enumerate(y) if v >= 0.1 * y_ss)]
        t_90 = t[next(i for i, v in enumerate(y) if v >= 0.9 * y_ss)]
        rise_time = t_90 - t_10
    except StopIteration:
        rise_time = float('inf')
    
    # 调节时间 (2%)
    tolerance = 0.02 * y_ss
    settling_time = t[-1]
    for i in range(len(t)-1, -1, -1):
        if abs(y[i] - y_ss) > tolerance:
            settling_time = t[min(i+1, len(t)-1)]
            break
    
    return {
        'overshoot': overshoot,
        'rise_time': rise_time,
        'settling_time': settling_time,
        'steady_state_error': abs(setpoint - y_ss),
        'IAE': np.trapz(np.abs(setpoint - y), t),
        'ITAE': np.trapz(t * np.abs(setpoint - y), t)
    }
```

```python
def ramp_response_analysis(sys, slope=1.0, T=10.0, dt=0.01):
    """斜坡跟踪性能分析"""
    t = np.arange(0, T, dt)
    _, y, _ = lsim(sys, slope * t, t)
    
    # 跟踪误差
    error = slope * t - y
    
    # 稳态跟踪误差斜率
    ss_error_rate = np.mean(error[-int(0.1*len(error)):]) / (slope * T)
    
    return {
        'max_tracking_error': np.max(np.abs(error)),
        'steady_state_error_rate': ss_error_rate,
        'RMSE': np.sqrt(np.mean(error**2))
    }
```

```python
def square_wave_tracking(sys, amplitude=1.0, period=2.0, cycles=5):
    """方波跟踪性能"""
    dt = 0.01
    t = np.arange(0, cycles * period, dt)
    ref = amplitude * np.sign(np.sin(2 * np.pi * t / period))
    
    _, y, _ = lsim(sys, ref, t)
    
    # 每个周期的性能
    per_cycle = []
    for i in range(cycles):
        idx_start = int(i * period / dt)
        idx_end = int((i + 0.5) * period / dt)  # 半周期
        cycle_error = ref[idx_start:idx_end] - y[idx_start:idx_end]
        per_cycle.append({
            'overshoot': ...,
            'settling_error': np.mean(np.abs(cycle_error[-int(0.1*len(cycle_error)):]))
        })
    
    return per_cycle
```

## 4. 鲁棒性评估

### 4.1 参数灵敏度分析

```python
def parameter_sensitivity(sys_func, nominal_params, perturbation=0.1):
    """
    参数灵敏度分析
    对每个参数±10%变化, 评估性能变化
    """
    results = {}
    base_perf = evaluate_performance(sys_func(nominal_params))
    
    for param_name, nominal_value in nominal_params.items():
        perturbed = nominal_params.copy()
        
        # 正向扰动
        perturbed[param_name] = nominal_value * (1 + perturbation)
        perf_plus = evaluate_performance(sys_func(perturbed))
        
        # 负向扰动
        perturbed[param_name] = nominal_value * (1 - perturbation)
        perf_minus = evaluate_performance(sys_func(perturbed))
        
        # 灵敏度 = Δ性能 / Δ参数
        sensitivity = {}
        for key in base_perf:
            S = (perf_plus[key] - perf_minus[key]) / (2 * perturbation * nominal_value)
            sensitivity[key] = S
        
        results[param_name] = sensitivity
    
    return results
```

```python
def monte_carlo_robustness(sys_func, param_distributions, N=1000):
    """Monte Carlo鲁棒性评估"""
    results = []
    
    for _ in range(N):
        # 随机生成参数
        params = {k: np.random.normal(v['mean'], v['std'])
                  for k, v in param_distributions.items()}
        
        try:
            perf = evaluate_performance(sys_func(params))
            results.append(perf)
        except:
            results.append({'stability': False})
    
    # 统计分析
    stable_count = sum(1 for r in results if r.get('stability', True))
    
    return {
        'stability_robustness': stable_count / N,
        'overshoot_mean': np.mean([r.get('overshoot', 100) for r in results]),
        'overshoot_std': np.std([r.get('overshoot', 100) for r in results]),
        'settling_time_mean': np.mean([r.get('settling_time', 100) for r in results]),
    }
```

## 5. 抗扰性能评估

### 5.1 阶跃扰动响应

```python
def disturbance_rejection_test(sys_cl, disturbance_magnitude=0.2, 
                                 disturb_time=5.0, T=20.0):
    """阶跃扰动抗扰测试"""
    dt = 0.01
    t = np.arange(0, T, dt)
    
    # 设定值为0, 在t=disturb_time加入阶跃扰动
    setpoint = np.zeros_like(t)
    disturbance = np.zeros_like(t)
    disturbance[t >= disturb_time] = disturbance_magnitude
    
    # 仿真
    y, _, _ = lsim(sys_cl, setpoint + disturbance, t)
    
    # 性能指标
    y_after = y[t >= disturb_time]
    t_after = t[t >= disturb_time]
    
    max_deviation = np.max(np.abs(y_after))
    max_deviation_time = t_after[np.argmax(np.abs(y_after))]
    
    # 恢复时间
    tolerance = 0.02
    recovery_time = T
    for i in range(len(y_after)-1, -1, -1):
        if abs(y_after[i]) > tolerance:
            recovery_time = t_after[min(i+1, len(y_after)-1)] - disturb_time
            break
    
    return {
        'max_deviation': max_deviation,
        'max_deviation_time': max_deviation_time,
        'recovery_time': recovery_time,
        'IAE_disturbance': np.trapz(np.abs(y_after), t_after)
    }
```

```python
def noise_rejection_test(sys_cl, noise_power=0.01, T=20.0):
    """噪声抑制性能"""
    dt = 0.01
    t = np.arange(0, T, dt)
    
    # 白噪声
    np.random.seed(42)
    noise = np.sqrt(noise_power) * np.random.randn(len(t))
    
    setpoint = np.ones_like(t)
    _, y, _ = lsim(sys_cl, setpoint + noise, t)
    
    # 输出噪声功率
    output_noise = y - setpoint
    output_noise_power = np.var(output_noise)
    
    # 噪声衰减比
    noise_attenuation = 10 * np.log10(noise_power / output_noise_power)
    
    return {
        'output_noise_power': output_noise_power,
        'noise_attenuation_dB': noise_attenuation,
        'output_noise_rms': np.sqrt(np.var(output_noise))
    }
```

## 6. 能耗与效率评估

### 6.1 控制能量评估

```python
def control_energy_analysis(u, t):
    """控制能量分析"""
    dt = t[1] - t[0]
    
    return {
        'total_energy': np.trapz(u**2, t),  # ∫u²dt
        'peak_control': np.max(np.abs(u)),
        'rms_control': np.sqrt(np.mean(u**2)),
        'control_variation': np.sum(np.abs(np.diff(u))),  # 总变差
        'control_smoothness': np.std(np.diff(u))  # 平滑度
    }
```

### 6.2 能效比指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 控制代价 | $J_u = \int_0^T u^2(t)dt$ | 控制能量 |
| 跟踪代价 | $J_e = \int_0^T e^2(t)dt$ | 跟踪误差 |
| 综合代价 | $J = J_e + \lambda J_u$ | 权衡跟踪与能耗 |
| 能效比 | $J_e / J_u$ | 单位控制能量产生的跟踪精度 |

### 6.3 执行器约束评估

- **饱和比例**: 控制信号超出限幅的时间占比
- **变化率限制**: $|du/dt|_{max}$ 是否满足执行器要求
- **功耗估算**: $P = \int_0^T |u(t) \cdot i(t)| dt$

```python
def quantization_analysis(Kp, Ki, Kd, adc_bits=12, dac_bits=10):
    """量化效应分析"""
    adc_resolution = 1.0 / (2**adc_bits - 1)
    dac_resolution = 1.0 / (2**dac_bits - 1)
    
    # PID参数量化
    Kp_quantized = round(Kp / dac_resolution) * dac_resolution
    
    # 死区评估
    dead_band = adc_resolution / Kp  # 输入侧死区
    
    # 极限环振荡评估
    limit_cycle_amplitude = dac_resolution / (1 - Ki * 0.01)  # 简化估算

        'limit_cycle_amplitude': limit_cycle_amplitude,

    }
```

## 8. 综合评分方法

### 8.1 加权评分法

```python
def comprehensive_score(metrics, weights=None):
    """
    控制系统综合评分
    metrics: 性能指标字典
    weights: 权重字典
    """
    if weights is None:
        weights = {
            'overshoot': 0.15,      # 超调量越小越好
            'settling_time': 0.20,  # 调节时间越短越好
            'steady_state_error': 0.15,  # 稳态误差越小越好
            'IAE': 0.15,            # 积分误差越小越好
            'robustness': 0.15,     # 鲁棒性越高越好
            'control_energy': 0.10, # 控制能量越小越好
            'noise_rejection': 0.10 # 噪声抑制越好越好
        }
    
    # 归一化各项指标到[0,1]区间
    normalized = {}
    limits = {
        'overshoot': (0, 50),       # 0%-50%
        'settling_time': (0, 10),   # 0-10s
        'steady_state_error': (0, 0.1),
        'IAE': (0, 5),
        'robustness': (0, 1),
        'control_energy': (0, 100),
        'noise_rejection': (0, 40)  # dB
    }
    
    total_score = 0
    for key, weight in weights.items():
        if key in metrics:
            low, high = limits.get(key, (0, 1))
            val = np.clip((metrics[key] - low) / (high - low), 0, 1)
            # 某些指标越小越好
            if key in ['overshoot', 'settling_time', 'steady_state_error', 
                       'IAE', 'control_energy']:
                val = 1 - val
            total_score += weight * val
    
    return total_score * 100  # 0-100分
```

```python
def performance_radar(metrics_dict):
    """绘制性能雷达图"""
    categories = ['快速性', '平稳性', '准确性', '鲁棒性', '抗扰性', '能效性']
    N = len(categories)
    
    angles = np.linspace(0, 2*np.pi, N, endpoint=False).tolist()
    angles += angles[:1]
    
    values = [
        1 / (1 + metrics_dict['settling_time']),  # 快速性
        1 / (1 + metrics_dict['overshoot'] / 100),  # 平稳性
        1 / (1 + metrics_dict['steady_state_error'] * 100),  # 准确性
        metrics_dict.get('robustness', 0.5),  # 鲁棒性
        1 / (1 + metrics_dict.get('disturbance_deviation', 0.5)),  # 抗扰性
        1 / (1 + metrics_dict.get('control_energy', 10) / 10)  # 能效性
    ]
    values += values[:1]
    
    fig, ax = plt.subplots(figsize=(8, 8), subplot_kw=dict(polar=True))
    ax.fill(angles, values, alpha=0.25)
    ax.plot(angles, values, 'o-', linewidth=2)
    ax.set_thetagrids(np.degrees(angles[:-1]), categories)
    ax.set_ylim(0, 1)
    ax.set_title('控制系统性能雷达图', fontsize=14)
    
    return fig
```

| 电机调速 | 稳态误差<1%, 响应<0.5s | 准确性+快速性 |
| 温度控制 | 超调<2%, 精度±0.1°C | 平稳性+准确性 |
| 平衡车 | 姿态角<±3°, 响应快 | 抗扰性+快速性 |
| 电源变换 | 纹波<1%, 效率>90% | 准确性+效率 |
| 信号跟踪 | 跟踪误差<5%, 带宽足够 | 带宽+准确性 |

### 9.3 快速调参策略

**第一步: 粗调 (确定大致范围)**
- 先用Ziegler-Nichols获得初始参数
- 观察阶跃响应趋势

**第二步: 细调 (平衡性能)**
- 超调大 → 减小Kp, 增大Kd
- 响应慢 → 增大Kp, 适当增大Ki
- 振荡 → 增大Kd, 减小Kp
- 稳态误差 → 适当增大Ki

**第三步: 微调 (优化积分指标)**
- 使用ITAE准则指导微调
- 考虑噪声对Kd的影响 (加入低通滤波)

### 9.4 性能评估检查清单

- [ ] 上升时间是否满足快速性要求
- [ ] 超调量是否在允许范围内
- [ ] 调节时间是否满足要求
- [ ] 稳态误差是否满足精度要求
- [ ] 增益裕度 > 6 dB
- [ ] 相位裕度 > 30°
- [ ] 参数±20%变化时系统仍稳定
- [ ] 阶跃扰动下偏差和恢复时间满足要求
- [ ] 控制信号无剧烈抖振
- [ ] 采样频率满足要求 (>10×闭环带宽)
- [ ] 无极限环振荡
- [ ] 执行器未长期饱和
