# 控制系统设计检查清单

## 一、需求分析阶段

### 1.1 控制目标明确
- [ ] 明确被控量（电压、电流、速度、位置、温度等）
- [ ] 确定设定值/参考信号类型（阶跃、斜坡、正弦、任意曲线）
- [ ] 确定允许的稳态误差范围
- [ ] 确定允许的超调量上限
- [ ] 确定调节时间/响应速度要求
- [ ] 确定抗扰动性能要求

### 1.2 系统边界条件
- [ ] 确定执行器类型（MOSFET、继电器、舵机、电机等）
- [ ] 确定执行器的驱动能力上限（最大电压/电流/功率）
- [ ] 确定传感器类型及量程
- [ ] 确定采样频率及A/D分辨率
- [ ] 确定控制周期（与采样周期的关系）

---

## 二、被控对象建模

### 2.1 机理建模
- [ ] 列写系统物理方程（基尔霍夫、牛顿、热力学等）
- [ ] 进行拉普拉斯变换得到传递函数 G(s)
- [ ] 确定系统阶次（一阶/二阶/高阶）
- [ ] 确定关键参数（增益K、时间常数τ、阻尼比ζ、自然频率ωn）
- [ ] 识别是否有纯滞后环节 e^(-Ls)

### 2.2 实验辨识
- [ ] 阶跃响应法：记录开环阶跃响应曲线
- [ ] 从响应曲线提取 K、τ、L（S型曲线法/两点法）
- [ ] 频率响应法（可选）：Bode图辨识
- [ ] 验证模型准确性：仿真与实测对比，误差<10%

### 2.3 非线性特性识别
- [ ] 饱和非线性（执行器限幅）
- [ ] 死区非线性（传感器/执行器不灵敏区）
- [ ] 滞环非线性（继电器、齿轮间隙等）
- [ ] 量化误差（A/D分辨率限制）

---

## 三、控制器设计

### 3.1 PID参数整定
- [ ] 选择PID类型（位置式/增量式）
- [ ] 使用Ziegler-Nichols法获取初始参数
- [ ] 仿真微调参数（先P→再PI→最后PID）
- [ ] 验证稳定性裕度（相位裕度>30°，增益裕度>6dB）
- [ ] 记录最终 Kp、Ki、Kd 值

### 3.2 进阶控制策略选择
- [ ] **积分抗饱和**：是否需要？选择条件积分/反计算/限幅方案
- [ ] **微分滤波**：噪声大时是否需要不完全微分？选择滤波系数α
- [ ] **微分先行**：设定值频繁变化时是否采用？
- [ ] **串级控制**：内环是否有独立被控量？内外环带宽比≥3:1
- [ ] **前馈补偿**：是否有可测量的扰动？前馈增益计算
- [ ] **Smith预估**：纯滞后大(>3τ)时是否采用？
- [ ] **模糊PID**：模型不精确时是否采用？隶属函数和规则表设计
- [ ] **自适应PID**：工况变化大时是否采用？自适应律选择

### 3.3 控制量限幅
- [ ] 设定控制量上限 umax 和下限 umin
- [ ] 确认限幅值与执行器能力匹配
- [ ] 积分项独立限幅（防止积分饱和）

---

## 四、传感器与信号处理

### 4.1 传感器选型
- [ ] 量程覆盖被控量全范围
- [ ] 精度满足控制精度要求（一般为控制精度的3~5倍）
- [ ] 响应速度满足采样频率要求
- [ ] 输出信号类型匹配MCU A/D输入（0~3.3V / 4~20mA等）

### 4.2 信号调理
- [ ] 硬件滤波：RC低通滤波器截止频率设计
- [ ] 软件滤波：选择合适滤波算法
  - [ ] 移动平均滤波（简单，适合慢变信号）
  - [ ] 一阶低通滤波（实时性好）
  - [ ] 中值滤波（抗脉冲干扰）
  - [ ] 卡尔曼滤波（最优估计，需模型）
- [ ] 数值微分的噪声放大问题处理
- [ ] 标定：零点校准、增益校准

---

## 五、数字实现注意事项

### 5.1 离散化
- [ ] 选择离散化方法（欧拉前差分/后差分/双线性变换）
- [ ] 确认采样周期 Ts << 系统时间常数 τ（一般 Ts ≤ τ/10）
- [ ] 微分项使用后向差分：d(k) = (e(k) - e(k-1)) / Ts
- [ ] 积分项使用梯形法或前向欧拉法

### 5.2 数值问题
- [ ] 整数运算还是浮点运算？量化误差评估
- [ ] 积分项累积溢出保护（限幅或使用更大变量类型）
- [ ] 微分项对测量噪声的敏感度评估
- [ ] 控制周期抖动对性能的影响评估

### 5.3 代码实现检查
- [ ] PID计算中除零保护
- [ ] 变量初始化正确（历史值清零）
- [ ] 控制量输出限幅
- [ ] 异常处理：传感器故障时的安全输出
- [ ] 看门狗定时器使能

---

## 六、仿真验证

### 6.1 开环仿真
- [ ] 验证被控对象模型的阶跃响应
- [ ] 验证模型频率特性（Bode图）
- [ ] 与实测数据对比验证

### 6.2 闭环仿真
- [ ] 阶跃响应测试：记录 tr、ts、Mp、ess
- [ ] 方波跟踪测试：验证跟踪能力
- [ ] 正弦跟踪测试：验证不同频率下的跟踪精度
- [ ] 抗扰动测试：加入阶跃扰动，验证恢复时间
- [ ] 噪声抑制测试：加入测量噪声，验证输出平滑性
- [ ] 参数灵敏度测试：Kp、Ki、Kd各±20%变化的影响
- [ ] 鲁棒性测试：对象参数变化±30%时的稳定性

### 6.3 极端工况仿真
- [ ] 初始误差很大时的响应（不饱和/不发散）
- [ ] 设定值突变时的响应
- [ ] 传感器断线/故障时的行为
- [ ] 执行器卡死/饱和时的行为

---

## 七、硬件调试

### 7.1 电路检查
- [ ] 电源纹波<50mV（示波器确认）
- [ ] 信号线与功率线隔离/屏蔽
- [ ] ADC参考电压稳定
- [ ] PWM驱动信号完整性（上升沿/下降沿/死区时间）
- [ ] 电机/MOSFET驱动电路的栅极电阻和续流二极管

### 7.2 软硬件联调
- [ ] 传感器标定：实测值与ADC读数的对应关系
- [ ] PWM占空比与实际输出的线性度
- [ ] 控制周期实测抖动<5%
- [ ] 通信延迟（如有无线控制）评估

---

## 八、性能测试与验收

### 8.1 静态性能
- [ ] 稳态误差：|ess| < 允许值
- [ ] 稳态输出纹波：峰峰值 < 允许值
- [ ] 温漂测试：温度变化时的输出稳定性

### 8.2 动态性能
- [ ] 阶跃响应指标：tr、ts、Mp、ess均满足要求
- [ ] 跟踪性能：对变化设定值的跟踪误差
- [ ] 扰动抑制：加入负载扰动后的恢复时间

### 8.3 鲁棒性验证
- [ ] 供电电压变化±10%时的稳定性
- [ ] 负载变化50%时的性能
- [ ] 长时间运行稳定性测试（>30分钟）

---

## 九、常见问题排查指南

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| 持续振荡 | Kp过大 / 积分时间太短 | 减小Kp或Ki |
| 响应太慢 | Kp太小 / Ki太大 | 增大Kp，减小Ki |
| 超调过大 | Kp过大或Ki过大 | 减小Kp，适当增大Kd |
| 稳态误差不消除 | Ki太小 | 增大Ki |
| 输出抖动/噪声大 | Kd过大 / 传感器噪声 | 减小Kd，增加滤波 |
| 积分饱和 | 输出长时间限幅 | 加入抗积分饱和措施 |
| 启动冲击大 | 积分项初始累积 | 上电积分清零/软启动 |
| 控制量频繁切换极性 | 死区设置不合理 | 调整死区阈值 |
