# 仿真平台使用手册

## 一、平台概述

本仿真平台是nuedc-asset-library的控制理论仿真模块，提供从经典控制到现代控制的完整仿真工具集。所有仿真程序基于 Python + SciPy + Matplotlib 实现，开箱即用。

---

## 二、环境要求

| 项目 | 要求 |
|------|------|
| Python | 3.8+ |
| 依赖包 | numpy, scipy, matplotlib |
| 安装命令 | `pip install numpy scipy matplotlib` |
| 推荐IDE | VS Code / PyCharm / Jupyter |

## 三、仿真程序清单

### 3.1 PID 控制仿真 (`pid_simulation.py`)
- **功能**：PID 参数整定、阶跃响应分析
- **方法**：Ziegler-Nichols、Cohen-Coon、手动调参
- **运行**：`python pid_simulation.py`

### 3.2 频域分析仿真 (`frequency_response.py`)
- **功能**：Bode 图绘制、Nyquist 图、稳定裕度
- **方法**：频率响应法、相位裕度/增益裕度计算
- **运行**：`python frequency_response.py`

### 3.3 根轨迹仿真 (`root_locus.py`)
- **功能**：根轨迹绘制、增益选择、稳定性分析
- **运行**：`python root_locus.py`

### 3.4 解耦控制仿真 (`decoupling_simulation.py`)
- **功能**：2×2 MIMO 系统解耦
- **内容**：
  - RGA（相对增益矩阵）计算 → 判断配对
  - 静态解耦矩阵设计
  - 解耦前后阶跃响应对比
- **运行**：`python decoupling_simulation.py`
- **关键输出**：
  - RGA 热力图（配对推荐）
  - 交叉耦合消除效果

### 3.5 状态反馈仿真 (`state_feedback_simulation.py`)
- **功能**：极点配置、状态反馈控制
- **内容**：
  - 能控性矩阵计算与判断
  - Ackermann 公式极点配置
  - 不同极点配置响应对比
- **运行**：`python state_feedback_simulation.py`
- **关键输出**：
  - s 平面极点位置图
  - 开环 vs 闭环阶跃响应

### 3.6 观测器仿真 (`observer_simulation.py`)
- **功能**：Luenberger 全阶观测器 + 状态反馈
- **内容**：
  - 能观性判断
  - 观测器增益设计（对偶极点配置）
  - 真实状态 vs 估计状态对比
  - 分离原理验证
- **运行**：`python observer_simulation.py`
- **关键输出**：
  - 状态估计误差收敛曲线
  - 控制器 + 观测器极点分布

### 3.7 鲁棒控制仿真 (`robust_control_simulation.py`)
- **功能**：H∞ 简化版鲁棒控制
- **内容**：
  - 加权函数设计（性能权重 Wp、鲁棒权重 Wt）
  - LQR + ARE 求解
  - 参数摄动下的鲁棒性分析
  - 灵敏度/补灵敏度函数分析
- **运行**：`python robust_control_simulation.py`
- **关键输出**：
  - 加权函数 Bode 图
  - 参数摄动下开环/闭环响应对比
  - S/T 函数与权重上界

## 四、使用流程

```
1. 安装依赖 → pip install numpy scipy matplotlib
2. 进入目录 → cd 15_simulation/
3. 运行仿真 → python xxx_simulation.py
4. 查看结果 → 自动生成 PNG 图片 + 终端打印数据
5. 修改参数 → 打开 .py 文件，修改顶部参数区
```

## 五、参数修改指南

### 5.1 修改系统模型
```python
# 例：修改传递函数 G(s) = num / den
num = [5]           # 分子系数
den = [1, 3, 2]     # 分母系数: s² + 3s + 2
```

### 5.2 修改状态空间
```python
A = np.array([[0, 1], [-2, -3]])   # 系统矩阵
B = np.array([[0], [1]])           # 输入矩阵
```

```python
desired_poles = [-3, -5]           # 实数极点
desired_poles = [-4+2j, -4-2j]    # 共轭复数极点
```

```python
t = np.linspace(0, 10, 500)       # 0~10秒, 500个点
```

## 七、常见问题

### Q1: 中文显示乱码
```python
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
plt.rcParams['axes.unicode_minus'] = False
```

### Q2: 矩阵求解报错
检查矩阵是否奇异（`np.linalg.det(M)`），确保系统能控/能观。

### Q3: 阶跃响应发散
检查开环极点是否在右半平面，检查反馈增益符号。

### Q4: 如何保存数据

```python
np.savetxt('data.csv', np.column_stack([t, y]), delimiter=',',
           header='time,output', comments='')
```
