# 视觉算法测试数据集说明

## 1. 概述

本文档说明电赛常用视觉算法所需的测试数据集准备方法，涵盖颜色识别、形状检测、目标追踪等场景。

---

## 2. 数据集分类

### 2.1 颜色识别测试集

| 类别 | 数量要求 | 采集条件 | 格式 |
|------|---------|---------|------|
| 红色目标 | ≥50张 | 不同光照/角度/距离 | JPG/PNG |
| 绿色目标 | ≥50张 | 同上 | JPG/PNG |
| 蓝色目标 | ≥50张 | 同上 | JPG/PNG |
| 黑色线条 | ≥30张 | 白底/灰底 | JPG/PNG |

**采集建议：**
- 每类至少覆盖3种光照条件（强光/正常/弱光）
- 目标大小覆盖传感器实际识别范围
- 包含干扰物（相似颜色物体）

### 2.2 形状/标志识别测试集

| 类别 | 说明 |
|------|------|
| 圆形 | 不同大小、部分遮挡 |
| 矩形 | 带倾斜角度 |
| 数字/字母 | 0-9, A-Z 印刷体 |
| 特殊标志 | 赛题指定的目标图案 |

### 2.3 循线/车道测试集

- 白底黑线：直线、弯道、十字、断线
- 黑底线：同上
- 分辨率建议：320×240 或 640×480
- 包含阴影干扰

### 2.4 动态场景视频

| 场景 | 帧率 | 时长 | 格式 |
|------|------|------|------|
| 目标移动 | ≥15fps | ≥10s | MP4/AVI |
| 相机移动 | ≥30fps | ≥15s | MP4/AVI |
| 多目标 | ≥15fps | ≥10s | MP4/AVI |

## 3. 数据采集方法

### 3.1 使用 OpenCV 摄像头采集

```python
import cv2
import os

cap = cv2.VideoCapture(0)
save_dir = "dataset/color_red"
os.makedirs(save_dir, exist_ok=True)
count = 0

while True:
    ret, frame = cap.read()
    cv2.imshow("Capture", frame)
    key = cv2.waitKey(1)
    if key == ord('s'):
        cv2.imwrite(f"{save_dir}/{count:04d}.jpg", frame)
        count += 1
        print(f"Saved {count}")
    elif key == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```

### 3.2 图像增强/扩增

```python
import cv2
import numpy as np

def augment_image(img):
    """数据增强：亮度、对比度、旋转"""
    results = []
    # 亮度调整
    for alpha in [0.7, 1.0, 1.3]:
        results.append(cv2.convertScaleAbs(img, alpha=alpha, beta=0))
    # 旋转
    h, w = img.shape[:2]
    for angle in [-10, 0, 10]:
        M = cv2.getRotationMatrix2D((w/2, h/2), angle, 1)
        results.append(cv2.warpAffine(img, M, (w, h)))
    # 高斯噪声
    noise = np.random.normal(0, 25, img.shape).astype(np.uint8)
    results.append(cv2.add(img, noise))
    return results
```

### 3.3 从视频提取帧

```python
import cv2

def extract_frames(video_path, interval=10):
    """每隔interval帧保存一帧"""
    cap = cv2.VideoCapture(video_path)
    count = 0
    saved = 0
    while cap.isOpened():
        ret, frame = cap.read()
        if not ret:
            break
        if count % interval == 0:
            cv2.imwrite(f"frames/frame_{saved:04d}.jpg", frame)
            saved += 1
        count += 1
    cap.release()
```

## 7. 注意事项

1. **光照一致性**：比赛环境光照可能与训练差异大，需多条件采集
2. **分辨率匹配**：确保数据集分辨率与实际使用的摄像头一致
3. **实时性验证**：在目标平台上测试算法耗时，确保满足帧率要求
4. **鲁棒性测试**：加入干扰场景（反光、遮挡、模糊）测试
