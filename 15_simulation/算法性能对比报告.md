# 算法性能对比报告

> 15_simulation 模块 —— 全部仿真算法的横向对比分析

## 一、算法概览

| 算法 | 核心思想 | 适用场景 | 复杂度 |
|------|----------|----------|--------|
| **PID / PID自整定** | 误差的比例-积分-微分 | 线性定常系统 | ★☆☆ |
| **自适应PID** | 在线调整PID参数 | 参数缓变系统 | ★★☆ |
| **模糊PID** | 模糊规则整定PID | 模型不确定系统 | ★★☆ |
| **神经网络PID** | NN学习PID参数 | 复杂非线性系统 | ★★★ |
| **ADRC** | 扩张状态观测器+非线性反馈 | 大范围不确定系统 | ★★★ |
| **MPC** | 滚动优化+预测模型 | 约束优化控制 | ★★★ |
| **滑模控制 (SMC)** | 变结构切换控制 | 不确定/扰动系统 | ★★☆ |
| **反步控制 (Backstepping)** | 逐步Lyapunov递推 | 严格反馈非线性系统 | ★★★ |
| **无差拍 (Deadbeat)** | 有限步收敛 | 数字采样系统 | ★★☆ |
| **重复控制** | 内模原理补偿周期扰动 | 周期信号跟踪 | ★★☆ |
| **Smith预估器** | 纯滞后补偿 | 大滞后系统 | ★★☆ |
| **状态反馈** | 极点配置/LQR | 可建模系统 | ★★☆ |
| **观测器 (Luenberger/Kalman)** | 状态重构 | 不可直接测量状态 | ★★☆ |
| **鲁棒控制 (H∞)** | 最坏情况优化 | 强不确定性系统 | ★★★ |
| **PD+重力补偿** | 前馈重力+PD反馈 | 机械臂控制 | ★★☆ |

## 四、场景选型指南

### 4.1 线性系统、模型已知
- **首选**: PID、状态反馈、MPC
- **约束优化**: MPC
- **快速收敛**: Deadbeat

### 4.2 线性系统、模型不确定
- **首选**: ADRC、自适应PID、鲁棒控制
- **参数缓变**: 自适应PID
- **结构不确定**: 鲁棒控制(H∞)

### 4.3 非线性系统
- **严格反馈型**: 反步控制
- **含不确定/扰动**: 滑模控制
- **通用非线性**: ADRC、模糊PID、神经网络PID

### 4.4 特殊场景
- **大滞后**: Smith预估器
- **周期信号**: 重复控制
- **状态不可测**: 观测器+状态反馈
- **机械臂**: PD+重力补偿 / 计算力矩法
- **倒立摆**: LQR / 滑模 / 反步

### 4.5 电赛实战推荐

| 控制任务 | 推荐算法(简单) | 推荐算法(高级) |
|----------|---------------|---------------|
| 温度控制 | PID + Smith预估 | ADRC / MPC |
| 电机调速 | 增量式PID | ADRC / SMC |
| 平衡小车 | PID双环 | LQR / 滑模 |
| 倒立摆 | LQR + 观测器 | 反步 / SMC |
| 球板系统 | PID串级 | MPC / ADRC |
| 循迹小车 | PID | 模糊PID |
| 摆杆控制 | 能量+PD | 反步 / SMC |
| 机械臂 | PD+重力补偿 | 计算力矩法 |

## 六、总结

1. **工程首选**: PID 及其变体(自适应/模糊/神经网络) — 成熟可靠
2. **抗扰动利器**: ADRC — 不依赖精确模型, 工程适应性强
3. **非线性首选**: 滑模控制 — 鲁棒性极强, 但需解决抖振
4. **理论优美**: 反步控制 — Lyapunov稳定性保证, 适合理论分析
5. **约束优化**: MPC — 处理多约束的最优方案, 计算量大
6. **电赛实战**: PID+前馈/ADRC 是性价比最高的选择
