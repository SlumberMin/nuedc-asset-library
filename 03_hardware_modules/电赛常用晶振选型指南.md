## 1. 晶振基础概念

### 1.1 关键参数

| 参数 | 符号 | 含义 | 典型值 |
|------|------|------|--------|
| 标称频率 | f₀ | 晶振工作频率 | 8MHz, 32.768kHz |
| 频率精度 | Δf/f | 出厂频率偏差 | ±10ppm, ±50ppm |
| 频率稳定度 | - | 温度范围内频率变化 | ±10ppm ~ ±100ppm |
| 负载电容 | CL | 匹配的负载电容 | 12pF, 18pF, 20pF |
| 等效串联电阻 | ESR | 晶振等效电阻 | 30~200Ω |
| 驱动功率 | DL | 最大允许驱动功率 | 0.1~1mW |
| 温度范围 | - | 工作温度范围 | -20~+70°C（商业级） |
| 老化率 | - | 频率随时间漂移 | ±3ppm/年 |

### 1.2 晶振工作原理

```
石英晶体等效电路：
    ┌─── L1 ───┐
    │   (等效   │
    │    电感)   │
────┤   C1     ├────
    │ (动态电容)│
    │   R1     │
    │ (等效电阻)│
    └──────────┘
         │
        C0 (并联电容)

串联谐振频率：fs = 1/(2π√(L1·C1))
并联谐振频率：fp = fs·√(1+C1/C0)
```

### 1.3 晶振分类总览

| 类型 | 频率精度 | 频率范围 | 成本 | 应用 |
|------|---------|---------|------|------|
| 无源晶振 | ±10~50ppm | 32.768kHz~200MHz | 最低 | 一般MCU时钟 |
| 有源晶振 | ±25~50ppm | 1MHz~200MHz | 低 | 高速数字电路 |
| TCXO | ±0.5~2ppm | 10MHz~50MHz | 中 | GPS、通信 |
| VCXO | ±50~100ppm | 10MHz~200MHz | 中 | PLL、频率调制 |
| OCXO | ±0.01~0.1ppm | 5MHz~100MHz | 高 | 基站、仪器 |
| MEMS | ±10~50ppm | 1MHz~200MHz | 低 | 抗振动场合 |

## 2. 无源晶振（Crystal）

### 2.1 特点

- 需要外部振荡电路（MCU内部或外部分立元件）
- 功耗最低
- 成本最低
- 电赛最常用

### 2.2 常用封装

| 封装 | 尺寸 | 适用场景 |
|------|------|---------|
| HC-49S | 11.4×4.7mm | 传统插件/贴片 |
| 3215 | 3.2×1.5mm | 小型贴片 |
| 2016 | 2.0×1.6mm | 超小型 |
| 5032 | 5.0×3.2mm | 标准贴片 |
| 圆柱形 | φ2×6mm | 32.768kHz |

### 2.3 电赛常用频率

| 频率 | 用途 | 推荐型号 |
|------|------|---------|
| 32.768kHz | RTC时钟 | FC-135, MC-306 |
| 4MHz | 低成本MCU | HC-49S 4MHz |
| 8MHz | STM32常用 | HC-49S 8MHz |
| 12MHz | USB时钟 | 5032 12MHz |
| 16MHz | Arduino/STM32 | 5032 16MHz |
| 25MHz | 以太网/USB | 5032 25MHz |
| 32MHz | 高速MCU | 5032 32MHz |

### 2.4 负载电容匹配

```
MCU晶振电路：
                ┌──────────────┐
XTAL Pin 1 ──┤              ├── XTAL Pin 2
      │       │    MCU内部    │       │
     C1       │   振荡电路    │      C2
      │       │              │       │
     GND      └──────────────┘      GND

匹配条件：CL = (C1 × C2)/(C1 + C2) + Cstray

Cstray：PCB走线和引脚寄生电容，通常3~5pF

例：晶振CL=18pF, Cstray=5pF
C1 = C2 = 2×(CL - Cstray) = 2×(18-5) = 26pF
取标准值27pF
```

### 2.5 无源晶振电路设计要点

```c
// PCB布局要求：
// 1. 晶振尽量靠近MCU引脚（走线<5mm）
// 2. 走线用地线包围（guard ring）
// 3. 晶振下方不走其他信号线
// 4. 负载电容靠近晶振引脚
// 5. 避免晶振附近有高速数字信号

// MCU振荡器配置（STM32示例）
RCC_OscInitStruct.OscillatorType = RCC_OSCILLATORTYPE_HSE;
RCC_OscInitStruct.HSEState = RCC_HSE_ON;
RCC_OscInitStruct.HSEPredivValue = RCC_HSE_PREDIV_DIV1;
RCC_OscInitStruct.PLL.PLLState = RCC_PLL_ON;
RCC_OscInitStruct.PLL.PLLSource = RCC_PLLSOURCE_HSE;
RCC_OscInitStruct.PLL.PLLMUL = RCC_PLL_MUL9; // 8MHz×9=72MHz
```

### 2.6 无源晶振常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 不起振 | 负载电容不匹配 | 调整C1/C2 |
| 起振慢 | ESR过大或驱动不足 | 更换晶振或增大驱动 |
| 频偏大 | 负载电容不准 | 精确匹配CL |
| 温漂大 | 晶体切型不对 | 使用AT切型 |

## 3. 有源晶振（Oscillator）

### 3.1 特点

- 内置振荡电路，输出标准时钟信号
- 4引脚：VCC, GND, Output, NC（或Enable）
- 频率精度更高（±25~50ppm）
- 不需要外部负载电容

### 3.2 输出类型

| 输出类型 | 特点 | 适用场景 |
|---------|------|---------|
| CMOS | 轨到轨输出，驱动能力强 | 数字电路通用 |
| LVPECL | 差分输出，高速 | 高速通信 |
| LVDS | 差分输出，低功耗 | 低功耗高速 |
| Clipped Sine | 正弦波输出，低抖动 | RF应用 |

### 3.3 引脚定义

```
标准4引脚封装（5032/3225）：
  ┌─────────┐
1 │ NC/EN   │ 4
  │  VCC    │
2 │         │ 3
  │  GND    │
  │  OUT    │
  └─────────┘

Pin1: NC或Enable（使能）
Pin2: GND
Pin3: Output（时钟输出）
Pin4: VCC
```

### 3.4 有源晶振电路设计

```
电路连接：
VCC ──┬── 100nF ──┬── GND
      │           │
      └── Pin4    │
                  │
Pin2 (GND) ──────┘
Pin3 (OUT) ──→ MCU OSC_IN
Pin1 (NC/EN) ── VCC 或 NC

PCB布局：
- 100nF去耦电容紧贴VCC和GND引脚
- 输出走线尽量短
- 不需要外部负载电容
```

### 3.5 电赛常用有源晶振型号

## 4. TCXO温补晶振

### 4.1 特点

- 内置温度补偿电路
- 频率精度 ±0.5~2ppm（-30~+85°C）
- 适合温度变化大的环境
- 成本中等

### 4.2 温度补偿原理

```
TCXO内部结构：
                  ┌─────────────────┐
                  │   温度传感器     │
                  │   (热敏电阻)     │
                  └────────┬────────┘
                           │
                  ┌────────▼────────┐
                  │   补偿电路       │
                  │  (模拟或数字)    │
                  └────────┬────────┘
                           │
                  ┌────────▼────────┐
                  │   变容二极管     │
                  │   (调整频率)     │
                  └────────┬────────┘
                           │
                  ┌────────▼────────┐
                  │   晶体振荡器     │
                  └─────────────────┘

补偿过程：
1. 温度传感器检测环境温度
2. 补偿电路计算频率偏差
3. 通过变容二极管调整负载电容
4. 频率被拉回标称值
```

### 4.3 TCXO关键参数

| 参数 | 普通TCXO | 高精度TCXO |
|------|---------|-----------|
| 频率精度 | ±2ppm | ±0.5ppm |
| 温度范围 | -30~+85°C | -40~+85°C |
| 老化率 | ±1ppm/年 | ±0.3ppm/年 |
| 功耗 | 2~5mA | 1~3mA |
| 启动时间 | 2~10ms | 5~50ms |

### 4.4 电赛常用TCXO

| 型号 | 频率 | 精度 | 价格 | 适用场景 |
|------|------|------|------|---------|
| TG-5006CG | 26MHz | ±2ppm | ¥10 | GPS模块 |
| SG-8002DC | 10MHz | ±2ppm | ¥8 | 高精度计时 |
| ECX-327 | 32MHz | ±1.5ppm | ¥12 | 无线通信 |
| KT2520K | 24MHz | ±0.5ppm | ¥20 | 高精度时钟 |

### 4.5 TCXO应用电路

```
TCXO典型连接：
VCC ──┬── 100nF + 10μF ──┬── GND
      │                   │
      └── VCC Pin         │
                          │
GND Pin ──────────────────┘
OUT Pin ──→ 串联100Ω ──→ MCU OSC_IN

注意：
- TCXO输出阻抗较低，串联阻尼电阻减少反射
- VCC纹波要小（影响频率精度）
- 启动时间较长，注意系统上电时序
```

## 5. VCXO压控晶振

### 5.1 特点

- 频率可通过外部电压调节
- 调谐范围 ±50~±200ppm
- 用于PLL、频率调制、时钟恢复

### 5.2 调谐特性

```
频率 vs 控制电压：
频率
 ^
 |                    ╱
 |                 ╱
 |              ╱
 |           ╱
 |        ╱
 |     ╱
 |  ╱
 +──────────────────→ 控制电压
   Vmin         Vmax

中心频率：Vcontrol = VCC/2时的频率
调谐范围：Δf/f（通常 ±50~200ppm）
调谐灵敏度：Kv (Hz/V 或 ppm/V)
线性度：频率-电压曲线的非线性度
```

### 5.3 VCXO应用：PLL频率合成

```
PLL频率合成器：
参考输入 ──→ [鉴相器] ──→ [环路滤波器] ──→ [VCXO] ──→ 输出
              ↑                                           │
              └──────── [分频器N] ←──────────────────────┘

f_out = f_ref × N

例：f_ref = 100kHz, N = 720
f_out = 100kHz × 720 = 72MHz
```

### 5.4 电赛常用VCXO

| 型号 | 中心频率 | 调谐范围 | 价格 |
|------|---------|---------|------|
| SG-636 | 14.318MHz | ±100ppm | ¥15 |
| VG-4512CA | 27MHz | ±150ppm | ¥18 |
| ECS-VCXO-250 | 25MHz | ±50ppm | ¥20 |

## 6. OCXO恒温晶振

### 6.1 特点

- 晶体放在恒温槽中，温度恒定
- 频率精度最高：±0.01~0.1ppm
- 功耗大（0.5~2W）
- 体积大，成本高

### 6.2 OCXO工作原理

```
OCXO结构：
┌─────────────────────────────┐
│  ┌─────────────────────┐    │
│  │  ┌───────────────┐  │    │
│  │  │    晶体        │  │    │
│  │  │  (AT切/SC切)   │  │    │
│  │  └───────────────┘  │    │
│  │  加热器 + 温控电路   │    │
│  └─────────────────────┘    │
│  振荡电路 + 输出缓冲        │
└─────────────────────────────┘

恒温温度：通常在晶体的零温度系数点
AT切型：约25°C或85°C
SC切型：约90°C
```

### 6.3 OCXO关键参数

| 参数 | 普通OCXO | 高性能OCXO |
|------|---------|-----------|
| 频率精度 | ±0.1ppm | ±0.01ppm |
| 温度稳定度 | ±0.05ppm | ±0.005ppm |
| 老化率 | ±0.1ppm/年 | ±0.005ppm/年 |
| 相位噪声 | -140dBc/Hz@1kHz | -160dBc/Hz@1kHz |
| 功耗 | 1~2W | 0.5~1W |
| 启动时间 | 3~5分钟 | 5~10分钟 |
| 尺寸 | 36×27mm | 25×25mm |

### 6.4 OCXO应用场景

- 频率计数器/合成器
- 基站时钟
- 精密测量仪器
- 电赛中极少使用（成本高、功耗大）

### 6.5 电赛替代方案

| 应用需求 | 推荐方案 | 成本 |
|---------|---------|------|
| ±10ppm足够 | 普通有源晶振 | ¥3 |
| ±2ppm需要 | TCXO | ¥10 |
| ±0.5ppm需要 | 高精度TCXO | ¥20 |
| ±0.1ppm需要 | OCXO（不推荐电赛） | ¥200+ |

## 7. MEMS振荡器

### 7.1 特点

- 基于微机电系统（MEMS）技术
- 非石英晶体，使用硅谐振器
- 抗振动、抗冲击性能优异
- 启动时间极短（<100μs）
- 可编程频率

### 7.2 MEMS vs 石英晶振

| 特性 | MEMS | 石英晶振 |
|------|------|---------|
| 抗振动 | 50,000g | 50g |
| 抗冲击 | 100,000g | 1,000g |
| 启动时间 | <100μs | 1~10ms |
| 温度稳定性 | ±10~50ppm | ±10~50ppm |
| 频率范围 | 1Hz~700MHz | 32.768kHz~200MHz |
| 老化率 | ±1ppm/年 | ±3ppm/年 |
| 尺寸 | 更小 | 一般 |
| 成本 | 接近 | 低 |

### 7.3 可编程MEMS振荡器

```c
// Si5351A可编程时钟生成器
// I2C接口，输出3路独立可编程时钟
// 频率范围：8kHz ~ 160MHz

// I2C配置示例
void si5351_init(void) {
    // 1. 关闭所有输出
    si5351_write(3, 0xFF);
    
    // 2. 配置PLL
    // PLL_A: 25MHz × 36 = 900MHz
    si5351_write(26, 0x00); // PLL_A 参数
    si5351_write(27, 0);    // P1整数部分
    si5351_write(28, 0);    // 
    si5351_write(29, 0);    // P1低8位
    si5351_write(30, 0);    // P2高
    si5351_write(31, 0);    // P2中
    si5351_write(32, 0);    // P2低+P3
    si5351_write(33, 0);    // P3高
    si5351_write(34, 1);    // P3低
    
    // 3. 配置分频器
    // CLK0: 900MHz / 12.5 = 72MHz
    // ...
    
    // 4. 使能输出
    si5351_write(3, 0x00);
}

void si5351_set_freq(uint8_t clk, uint32_t freq_hz) {
    // 根据目标频率计算PLL和分频器参数
    // 自动选择最优PLL配置
    // ...
}
```

## 9. 电路设计要点

### 9.1 无源晶振PCB设计

```
PCB布局规则：
1. 晶振距MCU < 5mm
2. 走线等长（差分对）
3. 地线包围（Guard Ring）
4. 晶振下方铺地铜
5. 不在晶振区走其他信号线
6. 负载电容靠近晶振

           ┌──── Guard Ring (GND) ────┐
           │                          │
    XTAL_IN ──┤  C1    [晶振]    C2  ├── XTAL_OUT
           │  │                      │  │
           │  GND                   GND │
           └──────────────────────────┘
                    GND铜皮
```

### 9.2 有源晶振PCB设计

```
布局要点：
1. 去耦电容紧贴VCC/GND引脚
2. 输出走线作为50Ω阻抗控制
3. 输出端串联阻尼电阻（33~100Ω）
4. Enable引脚不要悬空

电路：
         100nF
VCC ──┬──||──┬── GND
      │      │
      └──VCC─┘
           │
    [有源晶振]
           │
    OUT ──┬──→ 33Ω ──→ MCU
          │
         NC/EN → VCC (或GPIO控制)
```

### 9.3 晶振电源设计

```
晶振电源要求：
- 纹波 < 50mV（TCXO/OCXO更严格）
- 使用LDO供电（非DC-DC直接供电）
- 电源走线远离数字信号
- 增加LC滤波

电源滤波电路：
VCC ──→ 10μH ──┬── 10μF ──┬── 100nF ──┬── VCC_晶振
                │          │          │
               GND        GND        GND
```

### 9.4 晶振信号完整性

```
高速时钟信号处理：
1. 走线阻抗控制（50Ω单端）
2. 避免直角走线（用45°或圆弧）
3. 远离高速数字信号
4. 必要时终端匹配

终端匹配方案：
源端串联匹配：OUT ──→ 33Ω ──→ 走线 ──→ MCU
末端并联匹配：OUT ──→ 走线 ──→ MCU ──→ 50Ω ──→ GND
```
