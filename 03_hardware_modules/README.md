# 🔧 03_hardware_modules

## 模块概述

本目录存放电赛硬件相关的速查文档和模板，包括物料清单模板、PCB设计指南、焊接调试指南等。这些文档帮助参赛队伍快速完成硬件选型和制作。

## 目录结构

```
03_hardware_modules/
├── 电赛物料清单模板.md
├── PCB设计速查指南.md
└── 焊接调试指南.md
```

## 文件清单

| 文件 | 说明 |
|------|------|
| 电赛物料清单模板.md | 电赛常用元器件清单模板（BOM表） |
| PCB设计速查指南.md | PCB布局布线设计要点速查 |
| 焊接调试指南.md | 焊接技巧与硬件调试方法 |

## 使用方法

1. **硬件选型**：参考 `电赛物料清单模板.md` 准备元器件
2. **PCB设计**：查看 `PCB设计速查指南.md` 进行PCB设计
3. **制作调试**：按照 `焊接调试指南.md` 完成焊接和调试

## FAQ

**Q: 与其他硬件相关目录的关系？**
A: `03_hardware_modules` 包含硬件选型指南和故障排查；`06_circuit_templates` 包含具体电路设计方案。本目录侧重实操速查。

**Q: 没有PCB设计经验怎么办？**
A: 建议先使用洞洞板搭建原型，PCB设计可参考嘉立创EDA等免费工具。
