## 一、三款主流单片机最小系统板对比

| 对比项 | STM32F103C8T6 | MSPM0G3507 | TM4C123GH6PZT7 |
|--------|---------------|------------|-----------------|
| **内核** | ARM Cortex-M3 | ARM Cortex-M0+ | ARM Cortex-M4F |
| **主频** | 72 MHz | 80 MHz | 80 MHz |
| **Flash/RAM** | 64KB / 20KB | 128KB / 32KB | 256KB / 32KB |
| **ADC** | 12bit, 1Msps×2 | 12bit, 4Msps×1 | 12bit, 1Msps×2 |
| **定时器** | 4个通用+1高级 | 多路高级定时器 | 6个+QEI编码器 |
| **FPU** | ❌ 无 | ❌ 无 | ✅ 单精度 |
| **开发环境** | Keil/STM32CubeIDE | CCS/Theia | CCS/Keil |
| **TI杯适配** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐（TI推荐） | ⭐⭐⭐⭐ |
| **价格（最小板）** | ~8元 | ~15元 | ~25元 |
| **资料丰富度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| **电赛推荐场景** | 通用首选、资料多 | TI杯年份首选 | 需FPU/编码器场景 |

### 选型建议
- **TI杯年份**：优先 MSPM0G3507 或 TM4C123（TI官方推荐，部分年份有板卡发放）
- **非TI杯年份**：STM32F103C8T6 资料最丰富，上手最快
- **需要浮点运算**：TM4C123GH6PZT7 带FPU，PID运算快10倍
- **需要高速ADC**：MSPM0G3507 的4Msps ADC 优势明显

## 二、电机模块红黑榜

### 🟢 推荐（红榜）

| 模块 | 型号 | 推荐理由 | 适用场景 |
|------|------|----------|----------|
| 直流减速电机 | JGA25-370 | 扭矩大、带编码器、性价比高 | 小车、云台 |
| 步进电机 | 42BYGH34 | 扭矩0.4N·m、精度高 | 位置控制、转台 |
| 舵机 | MG996R | 金属齿轮、扭矩13kg·cm | 机械臂、转向 |
| 无刷电机 | 2212 1000KV | 功率密度高 | 无人机、高速旋转 |
| 电机驱动 | TB6612FNG | 双路H桥、效率高、发热小 | 双路直流电机 |
| 电机驱动 | DRV8825 | 细分32、电流2.5A | 步进电机 |

### 🔴 避坑（黑榜）

| 模块 | 型号 | 避坑理由 |
|------|------|----------|
| 电机驱动 | L298N | 效率极低（压降2V+）、发热严重、体积大 |
| 直流电机 | TT马达 | 齿轮塑料易打滑、无编码器、精度差 |
| 舵机 | SG90 | 塑料齿轮易损坏、扭矩小 |
| 步进电机 | 28BYJ-48 | 扭矩极小、速度慢、仅适合演示 |

## 三、传感器模块红黑榜

### 🟢 推荐（红榜）

| 类型 | 型号 | 推荐理由 |
|------|------|----------|
| **超声波** | HC-SR04 | 便宜、精度±3mm、成熟稳定 |
| **TOF测距** | VL53L0X | 量程2m、精度±3%、I2C接口 |
| **红外循迹** | TCRT5000 | 5路阵列方案成熟 |
| **IMU** | MPU6050 | 六轴、I2C、资料极多 |
| **IMU（升级）** | ICM42688 | 六轴、SPI、低噪声、2024新款 |
| **编码器** | 光电编码器600线 | AB相正交、精度高 |
| **电磁传感** | 工字电感+运放 | 电磁循迹经典方案 |
| **灰度传感器** | 八路灰度 | 多路巡线、模拟量输出 |
| **OpenMV** | OpenMV4 H7 | 视觉巡线、识别色块 |

### 🔴 避坑（黑榜）

| 模块 | 避坑理由 |
|------|----------|
| 超声波 JSN-SR04T | 一体式防水但盲区大、精度差 |
| GY-521 (MPU6050) | 劣质克隆芯片多，温漂严重 |
| DHT11 | 精度±2°C、响应慢，电赛基本用不上 |
| 红外避障模块 | 检测距离短、受环境光影响大 |

## 四、电源模块红黑榜

### 🟢 推荐（红榜）

| 模块 | 型号/方案 | 推荐理由 |
|------|-----------|----------|
| **锂电池** | 18650 3.7V 2600mAh | 能量密度高、可串并联 |
| **充电管理** | TP4056 | 单节锂电充电、带保护 |
| **充电管理（多节）** | BQ24610 | 支持多节串联、均衡充电 |
| **降压（低压差）** | AMS1117-3.3 | 简单可靠、1A输出 |
| **降压（大电流）** | MP1584EN | 3A输出、效率95%、可调 |
| **降压（5V大电流）** | LM2596 | 3A输出、方案成熟 |
| **升压** | MT3608 | 升压至12V、2A |

### 🔴 避坑（黑榜）

| 模块 | 避坑理由 |
|------|----------|
| 7805稳压 | 压差大效率低、发热严重 |
| USB供电 | 电流不足500mA，带不动电机 |
| 劣质充电宝 | 瞬间大电流会保护断电 |
| 万能充改装 | 不安全、无均衡、易爆炸 |

## 五、通信模块红黑榜

### 🟢 推荐（红榜）

| 模块 | 型号 | 推荐理由 | 适用场景 |
|------|------|----------|----------|
| **无线串口** | HC-05/HC-06 | 稳定、简单、调试方便 | 蓝牙遥控 |
| **WiFi** | ESP8266/ESP32 | 便宜、资料多、可当主控 | 远程监控 |
| **2.4G** | NRF24L01+ | 低延迟、功耗低、距离远 | 双机通信 |

| **串口屏** | 迪文/陶晶驰 | 调试显示、人机交互 | 参数显示 |

### 🔴 避坑（黑榜）

| 模块 | 避坑理由 |
|------|----------|
| ZigBee CC2530 | 配置复杂、协议栈臃肿 |
| LoRa SX1278 | 延迟高、带宽小、不适合实时控制 |
| 红外遥控 | 距离短、方向性强、不可靠 |
