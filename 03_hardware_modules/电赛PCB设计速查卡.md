## 一、PCB设计流程速查

```
原理图设计 → 网表导入 → 板框设计 → 元件布局 → 规则设置

```

## 二、布局规则速查

### 2.1 元件布局原则

```
✅ 先大后小：先放MCU、电源、大连接器
✅ 按信号流向：输入→处理→输出，从左到右
✅ 数模分离：数字区和模拟区分开，地线单点连接
✅ 电源靠近：去耦电容紧贴芯片电源引脚
✅ 热源分散：大功率器件分散放置，远离敏感器件
✅ 高频就近：高频器件引脚越短越好
```

### 2.2 推荐布局分区

```
┌──────────────────────────────────┐
│  电源区    │  MCU区  │  通信区   │
│  LDO/DCDC │  STM32  │  蓝牙WiFi │
│  滤波电容  │  晶振   │  串口     │
├───────────┼─────────┼──────────┤
│  驱动区    │  传感器区│  扩展区   │
│  电机驱动  │  MPU6050│  OLED     │
│  MOS管    │  编码器  │  按键LED  │
└──────────────────────────────────┘
```

## 三、布线规则速查

### 3.1 线宽规则

| 信号类型 | 推荐线宽 | 最小线宽 | 备注 |
|---------|---------|---------|------|
| 普通信号线 | 8~10mil | 6mil | 信号完整性好 |
| 电源线 | 20~40mil | 15mil | 按电流计算 |
| 大电流(>1A) | 40~100mil | - | 可铺铜加厚 |
| USB差分 | 90Ω阻抗 | - | 见阻抗匹配章节 |
| 高速信号 | 50Ω单端 | - | 见阻抗匹配章节 |

### 3.2 电流与线宽对照表（1oz铜厚，温升10°C）

| 电流(A) | 外层线宽(mil) | 内层线宽(mil) |
|---------|-------------|-------------|
| 0.5 | 10 | 15 |
| 1.0 | 20 | 30 |
| 2.0 | 40 | 60 |
| 3.0 | 60 | 100 |
| 5.0 | 100 | 150+铺铜 |

### 3.3 过孔规则

| 参数 | 推荐值 | 备注 |
|------|-------|------|
| 通孔孔径 | 0.3mm | 标准 |
| 通孔外径 | 0.6mm | 标准 |

## 四、阻抗匹配（重要！）

### 4.1 什么情况需要阻抗匹配？

```
✅ USB差分对 (90Ω)
✅ CAN总线 (120Ω终端)
✅ 高速SPI (50Ω)
✅ 射频信号 (50Ω)
✅ 以太网 (100Ω差分)
❌ 普通GPIO/UART/I2C (不需要)
```

### 4.2 常见阻抗结构

#### 微带线（外层走线）

```
┌─────────────────┐  ← 铜走线 (W)
│  FR4 介质 (H)    │
└─────────────────┘  ← 参考地平面

单端阻抗 (50Ω):
  W ≈ 1.5~2 × H  (FR4, εr≈4.4)
  
  近似公式 (IPC-2141):
  Z0 = (87 / √(εr + 1.41)) × ln(5.98H / (0.8W + T))
  
  其中：H=介质厚度, W=线宽, T=铜厚, εr=介电常数
```

#### 差分微带线（USB等）

```
    ┌──W──┐   ┌──W──┐
    │     │ S │     │   ← S = 间距
    └─────┘   └─────┘

差分阻抗 (90Ω USB):
  Zdiff = 2 × Z0 × (1 - 0.48 × e^(-0.96 × S/H))

  典型值 (1.6mm四层板):
  W = 5mil, S = 5mil → Zdiff ≈ 90Ω
  
  验证：用阻抗计算器（如Saturn PCB Toolkit）
```

#### 带状线（内层走线）

```
┌─────────────────┐  ← 上参考地平面
│  FR4 介质 (H1)   │
│  ──── 铜走线 ─── │
│  FR4 介质 (H2)   │
└─────────────────┘  ← 下参考地平面

  Z0 = (60 / √εr) × ln(4H / (0.67π(0.8W + T)))
```

### 4.3 阻抗计算实例

```
设计目标：USB差分 90Ω

PCB参数（嘉立创四层板）：
  层叠：L1-Signal / L2-GND / L3-Power / L4-Signal
  板厚：1.6mm
  L1→L2 介质厚度 H = 0.2mm (7.9mil)
  铜厚 T = 1oz = 35μm = 1.38mil
  FR4 εr = 4.4

用Saturn PCB Toolkit计算：
  W = 5mil, S = 6mil → Zdiff ≈ 89.3Ω ✅
  
  或者：
  W = 6mil, S = 5mil → Zdiff ≈ 87.5Ω ✅
```

### 4.4 阻抗匹配布线规则

```
1. 差分对等长：两根线长度差 < 5mil (USB) / < 2mil (高速)
2. 差分对紧耦合：间距S保持恒定
3. 避免参考平面不连续：不要跨分割区域走线
4. 过孔尽量少：每增加一个过孔约增加0.5~1nH电感
5. 差分对换层时加回流过孔：紧贴信号过孔旁放置GND过孔
6. 避免直角走线：用45°或圆弧
7. 阻抗线不要铺铜：保持阻抗一致性
```

## 五、EMC/EMI设计（电磁兼容）

### 5.1 EMC设计三要素

```
干扰源 → 耦合路径 → 敏感设备
  ↓          ↓          ↓
 减小      切断/衰减    提高抗扰度
```

### 5.2 去耦电容设计（最重要！）

```
每个芯片电源引脚必须放置去耦电容！

推荐方案：100nF + 10μF 并联

放置规则：
  ┌───────────────┐
  │   MCU/IC      │
  │  VCC    GND   │
  │   │      │    │
  │  ┌┴┐    │    │
  │ 100nF   │    │  ← 尽量近！<3mm
  │   │     │    │
  │  ─┴──────┴─  │  ← 短粗地线
  └───────────────┘

不同容值覆盖不同频率：
  10μF  → 低频滤波 (<1MHz)
  100nF → 中频滤波 (1~100MHz)
  10nF  → 高频滤波 (>100MHz)

走线顺序：VCC → 电容 → IC引脚（电容先于IC）
```

### 5.3 地平面设计

```
✅ 地平面尽量完整，不要割裂
✅ 高速信号下方必须有完整参考地
✅ 数字地和模拟地单点连接（或用0Ω电阻/磁珠）
✅ 多层板：至少一层完整地平面
✅ 信号换层时，在过孔旁放GND回流过孔

❌ 不要在地平面上走非本层信号线
❌ 不要让信号线跨过地平面分割处
❌ 不要让电源平面和地平面重叠区域太小
```

### 5.4 关键信号EMC处理

| 信号类型 | EMC措施 |
|---------|--------|
| 时钟/晶振 | 包地处理、远离IO、短走线 |
| USB | 差分对走线、共模滤波器 |
| 电机PWM | 远离敏感信号、加滤波 |
| 电源输入 | TVS保护、π型滤波 |
| 通信线 | 串联阻尼电阻（33~100Ω） |
| 按键/IO | RC滤波、TVS保护 |

### 5.5 晶振布局（常见踩坑点）

```
正确布局：
  ┌──────────┐
  │  MCU     │
  │          │
  │  OSC_IN──┼──[晶振]──OSC_OUT  ← 短！<5mm
  │          │     │
  │          │    GND
  └──────────┘

关键规则：
1. 晶振紧贴MCU引脚，走线<5mm
2. 晶振下方铺地，不要走其他线
3. 包地处理：两侧加GND过孔
4. 匹配电容靠近晶振放置（10~22pF）
5. 晶振外壳接地（如果是金属壳）
```

### 5.6 电机驱动EMC

```
问题：电机是最大的EMI源！

解决措施：
1. 电机驱动区远离MCU/传感器区
2. 电机电源线加磁环/共模电感
3. 驱动芯片VS引脚加0.1μF+10μF去耦
4. 续流二极管要快恢复（如SS34）
5. 电机线用屏蔽线，屏蔽层接GND
6. PCB上电机驱动区单独铺铜
7. 信号线不要从电机驱动区穿过
```

## 七、检查清单

### 7.1 DRC前检查

- [ ] 电源和地没有短路
- [ ] 所有网络已连接（无悬空）
- [ ] 去耦电容已放置
- [ ] 关键信号线宽符合要求
- [ ] 差分对等长已匹配
- [ ] 晶振布局规范
- [ ] 丝印不覆盖焊盘

```
输出文件清单：
  *.GTL  - 顶层铜
  *.GBL  - 底层铜
  *.GTO  - 顶层丝印
  *.GBO  - 底层丝印
  *.GTS  - 顶层阻焊
  *.GBS  - 底层阻焊
  *.GTP  - 顶层锡膏
  *.GBP  - 底层锡膏

  *.TXT  - 钻孔文件
  
  四层板额外：
  *.G2   - 内层2（地平面）
  *.G3   - 内层3（电源平面）
```
