# 结构件设计指南

## 1. 概述

电赛中结构件用于搭建车体、固定电路板、安装传感器。选材需权衡**重量、强度、加工难度、成本**。

### 材料对比总览

| 材料 | 密度 | 强度 | 加工难度 | 成本 | 适用场景 |
|------|------|------|----------|------|----------|
| 亚克力(PMMA) | 1.18 g/cm³ | 中 | ⭐易(激光切割) | 低 | 底板、支架 |
| PLA 3D打印 | 1.24 g/cm³ | 中低 | ⭐⭐(需建模) | 低 | 异形件、夹具 |
| ABS 3D打印 | 1.04 g/cm³ | 中 | ⭐⭐ | 低 | 耐热件 |
| 铝型材(2020) | 2.7 g/cm³ | 高 | ⭐⭐⭐ | 中 | 车体框架 |
| 碳纤维板 | 1.6 g/cm³ | 极高 | ⭐⭐⭐⭐ | 高 | 超轻结构 |
| 木板(椴木) | 0.4 g/cm³ | 低 | ⭐易 | 极低 | 临时/测试 |

## 2. 亚克力板设计

### 2.1 常用规格

| 厚度 | 用途 | 承重参考 |
|------|------|----------|
| 2mm | 面板、遮光罩 | 轻载 |
| 3mm | 底板、小支架 | 中载（最常用） |
| 5mm | 主结构板、大底板 | 重载 |
| 8mm | 大型支撑 | 特殊需求 |

### 2.2 激光切割设计规范

**图纸要求：**
- 文件格式：DXF / SVG / CorelDRAW
- 线条颜色区分：红线=切割，黑线=雕刻
- 线宽设为 0.01mm（Hairline）
- 单位：mm

**孔位设计：**
```
M3螺丝孔：直径 3.2mm（留余量）
M4螺丝孔：直径 4.2mm
M3铜柱嵌入孔：直径 5.0mm（热熔嵌入）
M3螺丝过孔：直径 3.5mm
```

**连接方式：**

```
1. 螺丝+铜柱直连（最常用）
   ┌────────┐
   │ 板A    │
   │  ○孔   │
   ├────────┤
   │ 铜柱M3 │  ← 6mm/10mm/15mm
   ├────────┤
   │ 板B    │
   │  ○孔   │
   └────────┘
   M3螺丝从两端锁紧

2. 直角连接件
   板A ←── L型角码 ──→ 板B（垂直连接）

3. 燕尾榫拼接（无螺丝）
   ┌──┐  ┌──┐
   │  └──┘  │  ← 切割互锁结构
   │  ┌──┐  │
   └──┘  └──┘
```

### 2.3 亚克力注意事项
- ❌ 不要弯折（易裂）
- ❌ 避免冲击（脆性材料）
- ✅ 钻孔时用低速、加水冷却
- ✅ 粘接用氯仿（三氯甲烷）或UV胶
- ✅ 边缘用砂纸打磨防割手

## 3. 3D打印设计

### 3.1 材料选择

| 材料 | 打印温度 | 热变形 | 特点 |
|------|----------|--------|------|
| PLA | 190-220°C | ~55°C | 易打印，脆性，够用 |
| ABS | 230-260°C | ~95°C | 韧性好，需封闭仓 |
| PETG | 220-250°C | ~75°C | 综合性能好 |
| TPU | 220-240°C | - | 弹性件、轮胎、减震 |
| PA(尼龙) | 240-270°C | ~80°C | 高强度，吸湿 |

### 3.2 打印参数推荐（PLA）

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 层高 | 0.2mm | 精度与速度平衡 |
| 壁厚 | 1.2-1.6mm | ≥3圈壁厚 |
| 填充率 | 30-50% | 结构件建议40%以上 |
| 填充图案 | 网格/蜂窝 | 蜂窝强度最高 |
| 打印速度 | 50-80mm/s | 精度优先降速 |
| 支撑 | 树形支撑 | 悬垂>45°需支撑 |
| 热床温度 | 60°C | 防翘边 |

### 3.3 设计规范

**壁厚与强度：**
```
最小壁厚：1.5mm（受力件建议≥2mm）
螺丝孔：M3孔径3.0mm（自攻）/3.2mm（预埋螺母）
预埋螺母孔：直径4.8mm（M3热熔螺母）
加强筋：厚度=壁厚×0.6，高度≤壁厚×3
圆角：所有内角加R≥1mm圆角（减少应力集中）
```

**常用结构件设计：**
```
1. 电机座
   ┌─────────┐
   │ ○   ○ │  ← 电机安装孔
   │         │
   │ ○   ○ │  ← 固定到底板的孔
   └─────────┘

2. 传感器支架
   可调角度设计：底部旋转轴 + 手拧螺丝锁紧

3. PCB固定柱
   M3×6mm铜柱热熔嵌入3D打印件
```

### 3.4 打印注意事项
- **公差预留 0.2mm**（配合件间隙）
- 受力方向避免垂直于打印层（层间结合力弱）
- 大平面底部贴防翘边胶或用 Brim
- 模型文件格式：STL（网格）/ STEP（实体）

## 4. 铝型材车体设计

### 4.1 常用型材规格

| 型材 | 截面 | 槽宽 | 适用 |
|------|------|------|------|
| 2020 | 20×20mm | 6mm | 轻型框架、桌面 |
| 2040 | 20×40mm | 6mm | 中型车体 |
| 3030 | 30×30mm | 8mm | 重型车体 |
| 4040 | 40×40mm | 8mm/10mm | 大型设备 |

**电赛推荐：2020 或 2040 铝型材**（轻、易加工、配件丰富）

### 4.2 连接方式

```
1. T型螺栓+弹性螺母（最常用）
   ┌────┐
   │ ╔══╧══╗ │
   │ ║ 型材 ║ │
   │ ╚══╤══╝ │
   └────┘
   T型螺母滑入槽内 → 螺栓穿过角件 → 拧紧

2. 直角连接件（L型/Z型）
   ┌────┐
   │    │←角件
   ├────┤
   │    │
   └────┘

3. 内六角连接（隐藏式）
   型材端面打孔攻丝，螺栓从侧面锁入

4. 弹性扣件（免工具快装）
   弹性卡扣直接卡入槽内
```

### 4.3 车体框架设计示例

```
顶视图（四轮小车）：
  ┌──────────────────────┐
  │  ○               ○  │  ← 2040横梁
  │  │               │  │
  │  电机            电机 │  ← 电机座固定
  │  │               │  │
  │  ○               ○  │
  ├──┬────────────────┬──┤
  │  │    电池仓      │  │  ← 底部安装板
  │  │                │  │
  ├──┴────────────────┴──┤
  │  ○               ○  │
  │  │               │  │
  │  电机            电机 │
  │  ○               ○  │
  └──────────────────────┘

尺寸建议：
  车体宽度：150-200mm（电赛通道适配）
  车体长度：200-300mm
  离地间隙：30-50mm
```

### 4.4 铝型材配件清单

| 配件 | 数量（4轮车） | 用途 |
|------|--------------|------|
| 2020/2040 型材 | 4-8根 | 框架主体 |
| L型直角件 | 8-12个 | 框架连接 |
| T型螺母(M4/M5) | 20-30个 | 滑入槽内固定 |
| 内六角螺栓M4×8 | 20-30个 | 配合T型螺母 |
| 端盖 | 8个 | 封堵型材端口 |
| 亚克力底板 | 1块(3-5mm) | 电路板安装 |

## 5. 传感器安装设计

### 5.1 巡线传感器安装

```
侧视图：
     车体底板
  ══════════════
        │ 可调支架（3D打印）
        │  ← 长槽孔可上下调节
        │
     [传感器]
        ↕ 距地面 5-15mm
  ─────────────── 地面
```

### 5.2 相机/摄像头安装

```
方案1：固定角度（简单可靠）
  车体 ──→ 3D打印支架(固定角度) ──→ 相机

方案2：舵机云台（可调）
  车体 ──→ 舵机(水平旋转) ──→ 舵机(俯仰) ──→ 相机
```

### 5.3 IMU安装
- 固定在车体**重心位置**
- 远离电机（减少振动干扰）
- 用双面胶+螺丝双重固定

## 6. 快速原型流程

```
1. 用纸板/CAD画草图，确定尺寸
2. 选定材料（亚克力/3D打印/铝型材）
3. 出图 → 激光切割/3D打印/型材切割
4. 装配 → 先不拧紧，确认位置后再紧固
5. 走线固定 → 扎带+线槽
6. 测试 → 振动/跌落/运行测试
```
