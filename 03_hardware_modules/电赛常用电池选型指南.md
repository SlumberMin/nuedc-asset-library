# 电赛常用电池选型指南

## 概述

电池是电赛中几乎所有系统的能量来源。合理选型直接影响系统续航、功率能力、重量和安全性。本文档覆盖锂电、铅酸、镍氢、超级电容和燃料电池五大类。

---

## 1 锂离子/锂聚合物电池

### 1.1 核心特性

| 参数 | 数值范围 |
|------|----------|
| 标称电压 | 3.6V / 3.7V（单节） |
| 充电截止电压 | 4.2V |
| 放电截止电压 | 2.5V~3.0V |
| 能量密度 | 150~265 Wh/kg |
| 循环寿命 | 500~2000次 |
| 自放电率 | 2%~5%/月 |
| 工作温度 | -20°C ~ 60°C |

### 1.2 常用电芯型号对比

| 型号 | 容量 | 最大放电 | 尺寸 | 重量 | 适用场景 |
|------|------|----------|------|------|----------|
| 18650 | 2000~3500mAh | 10~30A | 18×65mm | 45~50g | 通用小车、模块供电 |
| 21700 | 4000~5000mAh | 15~40A | 21×70mm | 60~70g | 大功率系统 |
| 14500 | 800~1000mAh | 1~3A | 14×50mm | 20~25g | 轻量化小系统 |
| 软包锂电 | 500~10000mAh | 5~30C | 定制 | 灵活 | 需定制形状的系统 |
| LiFePO4 | 1000~5000mAh | 10~30A | 多种 | 略重 | 安全性要求高 |

### 1.3 LiFePO4（磷酸铁锂）

| 参数 | 数值 |
|------|------|
| 标称电压 | 3.2V |
| 充电截止 | 3.65V |
| 放电截止 | 2.5V |
| 优点 | 安全性高、寿命长（2000+次）、高温性能好 |
| 缺点 | 能量密度略低（90~160 Wh/kg） |

### 1.4 锂电池保护电路

**必须具备的保护功能：**

| 保护功能 | 阈值（典型值） | 说明 |
|----------|----------------|------|
| 过充保护 | 4.25V~4.35V | 切断充电回路 |
| 过放保护 | 2.4V~2.9V | 切断放电回路 |
| 过流保护 | 根据电芯设定 | 短路/过载保护 |
| 过温保护 | 60°C~80°C | NTC检测温度 |

**常用保护IC：**
- 单节：DW01A + 8205A
- 多节串联：BQ76920（3-5串）、BQ76940（6-15串）

### 1.5 充电管理

**充电曲线（CC-CV）：**
```
电压 ↑
4.2V |─────────────────
     |               /
     |              /  ← 恒压阶段（CV）
     |             /     电流逐渐减小
     |  恒流阶段  /
     | (CC=1C)  /
     |________/
     └─────────────→ 时间
```

**常用充电IC：**
| 型号 | 功能 | 特点 |
|------|------|------|
| TP4056 | 线性充电 | 简单便宜，1A |
| MCP73831 | 线性充电 | 小封装，500mA |
| BQ24610 | 开关充电 | 效率高，支持大电流 |
| CN3065 | 太阳能充电 | MPPT功能 |

### 1.6 电池组设计

**串联：** 电压叠加，容量不变
- 3S：11.1V（典型）
- 4S：14.8V
- 6S：22.2V

**并联：** 容量叠加，电压不变
- 2P容量翻倍

**注意事项：**
- 串联必须加均衡电路
- 并联电池需电压一致
- 使用同批次同容量电芯

## 2 铅酸电池

### 2.1 核心特性

| 参数 | 数值范围 |
|------|----------|
| 标称电压 | 2V（单节） |
| 常见规格 | 6V、12V、24V |
| 能量密度 | 30~50 Wh/kg |
| 循环寿命 | 200~500次 |
| 自放电率 | 3%~20%/月 |
| 工作温度 | -20°C ~ 50°C |

### 2.2 类型对比

| 类型 | 特点 | 电赛适用场景 |
|------|------|-------------|
| 普通铅酸 | 需维护，便宜 | 不推荐 |
| AGM（吸液式） | 免维护，内阻小 | 大功率系统 |
| 胶体电池 | 深放电好，耐高温 | 户外系统 |
| SLA（密封铅酸） | 通用免维护 | 常规使用 |

### 2.3 铅酸电池放电特性

**放电截止电压（12V电池）：**

| 放电率 | 截止电压 |
|--------|----------|
| 10h率 | 10.5V |
| 5h率 | 10.2V |
| 1h率 | 9.6V |

**Peukert方程：**
$$C_p = I^n \cdot t$$

n ≈ 1.1~1.3（铅酸电池），大电流放电时容量下降。

### 2.4 铅酸充电

**三阶段充电：**
1. 恒流充电（0.1C~0.2C）→ 电压升至14.4V（12V电池）
2. 恒压充电（14.4V）→ 电流逐渐减小
3. 浮充（13.5V~13.8V）→ 维持满电

### 2.5 优缺点

| 优点 | 缺点 |
|------|------|
| 便宜 | 重（能量密度低） |
| 技术成熟 | 含铅，不环保 |
| 大电流放电好 | 自放电率高 |
| 安全性高 | 低温性能差 |

## 3 镍氢电池（NiMH）

### 3.1 核心特性

| 参数 | 数值范围 |
|------|----------|
| 标称电压 | 1.2V |
| 充电截止 | 1.4V~1.5V |
| 放电截止 | 0.9V~1.0V |
| 能量密度 | 60~120 Wh/kg |
| 循环寿命 | 500~1000次 |
| 自放电率 | 15%~30%/月（普通），1%~5%/月（低自放） |

### 3.2 常见规格

| 型号 | 容量 | 尺寸 | 典型应用 |
|------|------|------|----------|
| AAA | 800~1100mAh | 10.5×44mm | 小型模块 |
| AA | 1700~2800mAh | 14.5×50mm | 通用 |
| C | 4000~5000mAh | 26×50mm | 中等功率 |
| D | 8000~12000mAh | 34×61mm | 大功率 |

### 3.3 充电方法

| 方法 | 条件 | 终止检测 |
|------|------|----------|
| 标准充电 | 0.1C，14~16h | 时间 |
| 快速充电 | 0.5C~1C | -ΔV=5~10mV/节 |
| 涓流充电 | 0.02~0.05C | 时间 |

**-ΔV检测原理：**
充满时电压略微下降，检测到电压回落即终止充电。

### 3.4 优缺点

| 优点 | 缺点 |
|------|------|
| 环保无毒 | 能量密度不如锂电 |
| 无记忆效应 | 自放电率较高 |
| 安全性好 | 单节电压低 |
| 大电流放电好 | 充电检测复杂 |

### 3.5 低自放电NiMH

**代表产品：** eneloop（松下）、XX（爱乐普）

| 特性 | 数值 |
|------|------|
| 1年后容量保持 | 85%~90% |
| 循环寿命 | 1000~2100次 |
| 容量 | 1900~2500mAh（AA） |

## 4 超级电容

### 4.1 核心特性

| 参数 | 数值范围 |
|------|----------|
| 标称电压 | 2.5V~2.7V（单节） |
| 容量 | 0.1F~数千F |
| 能量密度 | 5~15 Wh/kg |
| 功率密度 | 5000~10000 W/kg |
| 循环寿命 | 50万~100万次 |
| ESR | mΩ级别 |
| 工作温度 | -40°C ~ 70°C |

### 4.2 与电池对比

| 特性 | 超级电容 | 锂电池 | 铅酸电池 |
|------|----------|--------|----------|
| 能量密度 | 低 | 高 | 中 |
| 功率密度 | 极高 | 中 | 中低 |
| 充电速度 | 极快（秒级） | 中（分钟~小时） | 慢（小时） |
| 循环寿命 | 极长（100万次） | 中（500~2000次） | 短（200~500次） |
| 温度范围 | 宽 | 中 | 窄 |

### 4.3 常用规格

| 规格 | 容量 | ESR | 尺寸 | 价格参考 |
|------|------|-----|------|----------|
| 0.47F/5.5V | 0.47F | 50Ω | 纽扣 | ￥1~2 |
| 1F/5.5V | 1F | 30Ω | 纽扣 | ￥2~3 |
| 10F/2.7V | 10F | 0.3Ω | 8×20mm | ￥3~5 |
| 100F/2.7V | 100F | 10mΩ | 22×45mm | ￥10~20 |
| 500F/2.7V | 500F | 3mΩ | 35×62mm | ￥30~50 |
| 3000F/2.7V | 3000F | 0.5mΩ | 60×140mm | ￥100+ |

### 4.4 能量计算

$$E = \frac{1}{2}CV^2$$

**示例：** 100F/2.7V超级电容
$$E = 0.5 \times 100 \times 2.7^2 = 364.5J = 0.1Wh$$

**放电到V_min时的可用能量：**
$$E_{usable} = \frac{1}{2}C(V_{max}^2 - V_{min}^2)$$

### 4.5 均压与保护

**串联均压电路：**
- 被动均压：并联电阻（简单但耗能）
- 主动均压：均压IC（效率高）

**常用均压IC：**
- S-8241系列
- LTC3225（LTC专用超级电容充电IC）

### 4.6 典型应用

| 应用 | 说明 |
|------|------|
| 能量缓冲 | 大电流脉冲供电（如电机启动） |
| 后备电源 | 掉电后维持关键数据 |
| 能量回收 | 制动能量存储 |
| 峰值功率补充 | 电池+超级电容混合供电 |

### 4.7 电池+超级电容混合方案

```
电池 ────┬──── DC-DC ──── 负载
         │
超级电容 ─┘
```

电池提供持续功率，超级电容提供瞬时峰值功率。

## 5 燃料电池

### 5.1 核心特性

| 参数 | 数值范围 |
|------|----------|
| 单体电压 | 0.6V~0.7V（工作） |
| 能量密度 | 300~1000 Wh/kg（含燃料） |
| 工作温度 | 60°C~80°C（PEM型） |
| 寿命 | 5000~20000h |

### 5.2 类型对比

| 类型 | 电解质 | 温度 | 特点 | 电赛适用性 |
|------|--------|------|------|------------|
| PEM（质子交换膜） | 质子膜 | 60-80°C | 启动快，功率密度高 | ★★★ |
| DMFC（直接甲醇） | 质子膜 | 60-90°C | 液态燃料，方便 | ★★ |
| SOFC（固体氧化物） | 陶瓷 | 600-1000°C | 效率高 | ★ |

### 5.3 PEM燃料电池原理

```
H₂ → 阳极催化 → 2H⁺ + 2e⁻
                        ↓ 外电路（电流）
O₂ + 4H⁺ + 4e⁻ → 阴极催化 → 2H₂O
```

**总反应：** 2H₂ + O₂ → 2H₂O

### 5.4 燃料电池系统组成

```
氢气瓶 → 减压阀 → 燃料电池堆 → DC-DC → 负载
                        ↓
                   水/热管理
```

### 5.5 电赛可用方案

**微型燃料电池套件：**
- 功率：5W~50W
- 供氢：氢气瓶/金属氢化物储氢
- 优点：续航时间长，安静
- 缺点：成本高，氢气存储安全

## 6 选型决策矩阵

### 6.1 按应用场景选型

| 场景 | 首选 | 备选 | 理由 |
|------|------|------|------|
| 小车供电(通用) | 18650锂电 | NiMH AA | 能量密度高，易获取 |
| 大功率电机 | 21700锂电/软包 | 铅酸AGM | 大电流放电能力 |
| 轻量化飞行器 | 软包锂电(高C) | - | 轻，高倍率 |
| 长时间数据采集 | LiFePO4 | 铅酸 | 长寿命，安全 |
| 脉冲大电流 | 超级电容+电池 | 高C锂电 | 功率密度高 |
| 后备电源 | 超级电容 | NiMH低自放 | 免维护，长寿命 |
| 环保要求高 | NiMH | LiFePO4 | 无毒 |
| 超长续航 | 燃料电池 | 大容量锂电 | 能量密度最高 |

### 6.2 关键参数对比表

| 参数 | 锂离子 | 铅酸 | NiMH | 超级电容 | 燃料电池 |
|------|--------|------|------|----------|----------|
| 能量密度(Wh/kg) | 150-265 | 30-50 | 60-120 | 5-15 | 300-1000 |
| 功率密度(W/kg) | 300-1500 | 100-300 | 200-500 | 5000-10000 | 100-300 |
| 单体电压(V) | 3.6-3.7 | 2.0 | 1.2 | 2.5-2.7 | 0.6-0.7 |
| 循环寿命(次) | 500-2000 | 200-500 | 500-1000 | 50万-100万 | - |
| 充电时间 | 1-4h | 4-10h | 1-4h | 秒~分钟 | 加注燃料 |
| 安全性 | 中 | 高 | 高 | 高 | 中低 |
| 成本 | 中 | 低 | 中 | 中 | 高 |
| 环保 | 中 | 差 | 好 | 好 | 好 |

### 6.3 容量计算公式

**系统续航时间：**
$$T = \frac{C_{battery} \times V_{nom}}{P_{system}} \times \eta$$

- C_battery：电池容量（Ah）
- V_nom：标称电压（V）
- P_system：系统平均功率（W）
- η：放电效率（锂电约0.9，铅酸约0.8）

**考虑Peukert效应（铅酸/高倍率）：**
$$T = \frac{C_{rated}}{I_{actual}^n} \times \left(\frac{I_{rated}}{I_{actual}}\right)^{n-1}$$

## 7 电赛实战建议

### 7.1 通用推荐方案

```
小车/机器人：   3S/4S 18650电池组（2500mAh以上）
飞行器：       3S~6S 高C软包锂电
低功耗系统：   2S 18650 或 AA NiMH
大功率系统：   4S~6S 21700 或铅酸AGM
```

### 7.2 注意事项

1. **锂电池严禁过放过充**，必须使用保护板
2. **铅酸电池避免深度放电**，放电不超过80%
3. **串联电池组必须均衡充电**
4. **超级电容注意电压限制**，单节不超2.7V
5. **比赛时准备备用电池**
6. **注意电池携带安全规定**（航空运输限制等）

### 7.3 常用电池管理芯片速查

| 芯片 | 功能 | 适用 |
|------|------|------|
| TP4056 | 单节锂电充电 | 最常用 |
| DW01A | 单节锂电保护 | 最常用 |
| BQ76920 | 3-5串BMS | 多串锂电 |
| INA219 | 电流/功率监测 | 电量监测 |
| LTC4150 | 库仑计 | 精确电量计量 |
