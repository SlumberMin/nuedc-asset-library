# 焊接调试指南

## 一、概述

本文档提供电赛硬件制作中的焊接技巧、调试工具使用方法和故障排查流程。

---

## 三、焊接技巧

### 3.1 贴片电阻/电容 (0603/0805)

```
步骤:
1. PCB一端焊盘预上锡 (薄薄一层)
2. 镊子夹持元件放到焊盘上
3. 烙铁加热预上锡端，固定元件
4. 另一端加锡焊接
5. 回头修正第一端焊点

    ┌───────────┐
    │  预上锡端   │ ← 先固定这端
    │  ┌─────┐  │
    │  │ 元件 │  │
    │  └─────┘  │
    │  焊接端   │ ← 再焊这端
    └───────────┘

关键点:
- 锡量适中，焊点光亮呈弧形
- 元件居中，不偏移
- 两端焊锡量一致
```

### 3.2 QFP封装IC (如STM32 LQFP-48)

```
步骤:
1. 焊盘上定位 (第1脚标记对齐)
2. 先焊接对角两个引脚固定
3. 加助焊剂到所有引脚
4. 拖焊: 烙铁头蘸少量锡，沿引脚方向拖动
5. 检查桥连: 用放大镜逐脚检查
6. 桥连处理: 加助焊剂，吸锡带吸除多余锡

拖焊示意:
  烙铁 →→→→→→→→ 方向
  ┌──┬──┬──┬──┐
  │1 │2 │3 │4 │  ← 引脚
  └──┴──┴──┴──┘

关键点:
- 助焊剂是关键! 没有助焊剂容易桥连
- 拖焊速度要均匀
- 发现桥连立即用吸锡带修复
```

### 3.3 QFN封装IC

```
步骤:
1. 焊盘预上锡 (薄薄一层)
2. QFN底部热焊盘区域涂锡膏(或预上锡)
3. 对准定位，热风枪350°C均匀加热
4. 观察锡膏融化、元件自动对位
5. 自然冷却

热风枪使用:
  温度: 350~380°C
  风量: 中等 (太大吹飞元件)
  距离: 3~5cm
  方式: 均匀画圈加热

关键点:
- 底部散热焊盘必须有锡，否则散热差
- 过孔阵列帮助导热到背面
- 冷却前不要移动PCB
```

### 3.4 BGA焊接 (进阶)

```
电赛中较少用到BGA，但在使用某些模块时可能遇到:

步骤:
1. 焊盘涂锡膏 (通过钢网)
2. BGA对准 (利用对位标记)
3. 热风枪或回流焊加热
4. 自然冷却

注意: BGA返修需要专业设备，电赛建议尽量避免使用BGA封装器件
```

### 3.5 插件元件焊接

```
步骤:
1. 元件插入PCB孔位
2. 弯脚固定 (或用胶带临时固定)
3. 翻转PCB
4. 烙铁同时加热焊盘和引脚
5. 送锡，锡自然流动覆盖焊盘
6. 移开锡丝，再移开烙铁

良好焊点标准:
  ✓ 焊锡呈圆锥形，光亮
  ✓ 焊锡润湿焊盘和引脚
  ✓ 引脚轮廓可见

不良焊点:
  ✗ 虚焊: 焊锡呈球形，不润湿
  ✗ 冷焊: 焊锡表面粗糙无光
  ✗ 过多: 焊锡淹没元件
  ✗ 过少: 焊盘未完全覆盖
```

## 四、焊接质量检查

### 4.1 目视检查清单

```
检查项目:
□ 元件方向正确 (IC第1脚、二极管极性、LED极性)
□ 无桥连 (相邻引脚间无焊锡连接)
□ 无虚焊 (焊点光亮，焊锡润湿良好)
□ 元件贴紧PCB (无浮起)
□ 无漏焊 (所有焊盘都有焊锡)
□ 无元件损坏 (电阻无裂纹、IC无烧痕)
□ 丝印清晰可读
```

### 4.2 万用表检查

```
上电前必检:
1. 短路检查: VCC对GND是否短路 (电阻模式)
   正常: 应有几KΩ以上 (非0)
   异常: 0Ω或接近0 → 检查焊接桥连

2. 电源检查: 
   不插IC，通电测量各电源点电压
   3.3V ± 5% → OK
   5V ± 5% → OK

3. 连通性检查:
   关键信号线是否导通
   地线是否完整
```

## 五、调试工具使用

### 5.1 万用表

```
常用测量:
  电压: 并联测量 (红笔接正，黑笔接负)
  电流: 串联测量 (断开电路，串入万用表)
  电阻: 断电测量 (不能在线测!)
  通断: 蜂鸣档 (检查焊接连通性)

技巧:
  - 测量前预估量程，从大到小
  - 测电流后记得换回电压档!
  - 红笔插对孔 (mA孔测小电流，10A孔测大电流)
```

### 5.2 示波器

```
常用功能:
  1. PWM波形: 检查频率和占空比
     设置: 时基1ms/div, 触发边沿上升
     测量: 频率、占空比、幅值

  2. UART波形: 检查通信
     设置: 时基按波特率调整 (115200→约8.7μs/bit)
     触发: 下降沿 (起始位)

  3. 电源纹波:
     设置: AC耦合, 时基1ms/div, 幅值10mV/div
     观察: DC-DC输出纹波应<50mV

  4. SPI/I2C: 解码功能
     设置: 协议解码 (选SPI或I2C)
     观察: 数据内容、时序关系

示波器探头:
  - 使用前校准 (探头补偿调整)
  - 10X档减小负载效应
  - 接地夹尽量短 (长地线引入噪声)
```

### 5.3 逻辑分析仪

```
适用: 数字信号时序分析 (SPI/I2C/UART/PWM)

使用方法:
  1. 连接信号线 + GND
  2. 设置采样率 (≥信号频率10倍)
  3. 设置协议解码
  4. 触发捕获
  5. 分析数据

推荐: Saleae Logic / 国产逻辑分析仪 (50~200元)
软件: Logic 2 / PulseView (免费)
```

### 5.4 程序调试器 (ST-Link / J-Link / DAP-Link)

```
功能:
  - 在线烧录固件
  - 单步调试 (断点、变量查看)
  - 实时变量监控 (SWO/RTT)

连接:
  SWD接口 (4线):
    VCC (3.3V) ──> 目标板VCC (可选)
    GND ────────> 目标板GND
    SWDIO ──────> 目标板SWDIO
    SWCLK ──────> 目标板SWCLK

常见问题:
  - 连接失败: 检查接线、目标板供电
  - 速度慢: 降低SWD频率
  - 芯片锁死: 执行全片擦除
```

## 六、故障排查流程

### 6.1 系统性排查步骤

```
Step 1: 外观检查
  ├── 检查焊接质量 (桥连/虚焊/漏焊)
  ├── 检查元件方向 (IC/二极管/LED)
  └── 检查元件值 (电阻色环/电容标识)

Step 2: 电源检查 (不插敏感IC)
  ├── 通电测量各电源点电压
  ├── 检查纹波 (示波器)
  └── 检查电流消耗 (是否异常大)

Step 3: 最小系统测试
  ├── 只接MCU + 电源 + 晶振
  ├── 烧录LED闪烁程序
  └── 确认MCU工作正常

Step 4: 逐步添加模块
  ├── 每次加一个模块
  ├── 测试该模块基本功能
  └── 确认正常后再加下一个

Step 5: 整体联调
  ├── 所有模块连接
  ├── 运行完整程序
  └── 测试边界条件
```

### 6.2 常见故障与解决

#### 电源类故障

| 症状 | 可能原因 | 排查方法 |
|------|----------|----------|
| 通电无反应 | 电源未接通 | 万用表测VCC |
| 电源指示灯不亮 | LED极性反/电源断路 | 检查LED方向和电源通路 |
| 电压偏低 | 稳压器损坏/过载 | 断开负载单独测稳压器 |
| 发热严重 | 短路/过载 | 红外测温或断电摸温 |
| 纹波大 | 滤波电容缺失/损坏 | 示波器测电源波形 |

#### MCU类故障

| 症状 | 可能原因 | 排查方法 |
|------|----------|----------|
| 无法烧录 | 接线错误/芯片锁死 | 检查SWD接线，尝试全擦 |
| 程序不运行 | 晶振不起振/复位异常 | 示波器测晶振波形 |
| 复位 | 电源跌落/看门狗 | 测电源纹波，检查代码 |
| GPIO无输出 | 配置错误/焊接虚焊 | 万用表测引脚电平 |
| ADC读数不准 | 参考电压不稳/干扰 | 测VREF，加滤波 |

#### 通信类故障

| 症状 | 可能原因 | 排查方法 |
|------|----------|----------|
| UART乱码 | 波特率不匹配 | 确认两端波特率一致 |
| I2C无应答 | 地址错误/上拉缺失 | 逻辑分析仪抓波形 |
| SPI数据错误 | 时钟极性/相位错 | 检查CPOL/CPHA配置 |
| RS485丢包 | 终端电阻/偏置电阻 | 检查总线电阻 |
| 蓝牙连不上 | 电源不稳/AT配置错 | 重新配置蓝牙参数 |

#### 电机驱动类故障

| 症状 | 可能原因 | 排查方法 |
|------|----------|----------|
| 电机不转 | 驱动IC未使能/接线错 | 检查使能引脚和接线 |
| 电机单向转 | H桥一路损坏 | 交换AOUT1/AOUT2 |
| 驱动IC过热 | 过流/短路 | 测量电机电流 |
| 电机抖动 | PWM频率太低 | 提高PWM频率到20kHz+ |
| 启动复位 | 电源跌落 | 加大电源电容/分离供电 |

## 七、焊接调试实战技巧

### 7.1 BGA焊盘修复

```
当PCB焊盘被意外撕掉时:
1. 用刀片刮出走线铜箔
2. 用细导线 (0.1mm漆包线) 连接
3. UV胶固定
4. 注意: 修复后强度下降，避免震动
```

### 7.2 芯片引脚桥连修复

```
1. 加足量助焊剂
2. 吸锡带覆盖桥连区域
3. 烙铁压在吸锡带上加热
4. 锡被吸走后抬起
5. 重复直到桥连消除
6. 放大镜确认
```

### 7.3 0402元件焊接要点

```
0402太小，手焊困难:
1. 用镊子+放大镜/显微镜
2. 烙铁温度320°C，尖头烙铁头
3. 先固定一端，再焊另一端
4. 用牙签辅助定位
5. 实在不行就换0603封装
```

### 7.4 飞线技巧

```
当PCB走线断路或设计错误时:
1. 0.1mm漆包线 (刮掉两端漆皮)
2. 从断点一端焊到另一端
3. 用UV胶或热熔胶固定
4. 飞线越短越好
5. 避免飞线跨越高速信号线
```

## 八、安全注意事项

### 8.1 人身安全
- 烙铁使用时注意防烫
- 焊接时保持通风（焊锡烟雾有害）
- 不要触摸通电的裸露电路
- 热风枪远离易燃物品

### 8.2 元件安全
- CMOS器件注意防静电（佩戴防静电手环）
- 电解电容注意极性（反接会爆炸！）
- 锂电池不要过充过放（有起火风险）
- 大电流测试时先检查接线

### 8.3 工具保养
- 烙铁头用完后上锡保护
- 定期清洁烙铁头（湿海绵或铜丝球）
- 热风枪冷却后再收纳
- 万用表用完关机或断开表笔
