# 常用芯片速查手册

> 电赛常用芯片引脚、参数、替代型号速查表

---

## 1. 运算放大器

### LM358 — 双路通用运放

| 参数 | 值 |
|------|-----|
| 供电电压 | 3V ~ 32V (单电源) / ±1.5V ~ ±16V |
| 输入失调电压 | 2mV (typ) |
| 增益带宽积 | 1MHz |
| 输出电流 | ~20mA |
| 封装 | DIP-8 / SOP-8 |

**引脚图 (DIP-8)：**
```
        ┌────────┐
 OUT1  1│        │8  VCC
 IN1-  2│ LM358  │7  OUT2
 IN1+  3│        │6  IN2-
 GND   4│        │5  IN2+
        └────────┘
```
**替代型号：** LM258(工业级)、LM2904(车规)、NE5532(音频升级)、OPA2134(精密升级)

---

### TL072 — 低噪声JFET运放

| 参数 | 值 |
|------|-----|
| 供电电压 | ±3.5V ~ ±18V |
| 输入偏置电流 | 30pA |
| 转换速率 | 13V/µs |
| 增益带宽积 | 3MHz |
| 封装 | DIP-8 / SOP-8 |

**替代型号：** TL082(通用版)、NE5532(低噪声版)、OPA2134(高级版)

---

### OPA2134 — 高精度运放

| 参数 | 值 |
|------|-----|
| 供电电压 | ±2.5V ~ ±18V |
| 输入失调电压 | 100µV |
| 增益带宽积 | 8MHz |
| 失调漂移 | 2µV/°C |
| 封装 | DIP-8 / SOP-8 |

**替代型号：** AD8608、OPA2277(精密直流)

---

## 2. 比较器

### LM393 — 双路比较器

| 参数 | 值 |
|------|-----|
| 供电电压 | 2V ~ 36V |
| 输入失调电压 | 5mV |
| 响应时间 | 1.3µs |
| 输出 | 集电极开路 |

**引脚图 (DIP-8)：**
```
        ┌────────┐
 OUT1  1│        │8  VCC
 IN1-  2│ LM393  │7  OUT2
 IN1+  3│        │6  IN2-
 GND   4│        │5  IN2+
        └────────┘
```
**替代型号：** LM2903(车规)、TLV1701(低功耗)、MAX9117(高速)

> ⚠️ 输出为开漏，需要上拉电阻


## 5. DC-DC转换器

|------|-----|

```
        ┌─────────┐

        └─────────┘
```

**替代型号：** LM2576(同系列)、MP2315(同步整流)、TPS5430(TI)、XL4015(国产)

## 6. H桥 / 电机驱动

### L298N — 双H桥驱动

| 参数 | 值 |
|------|-----|
| 供电电压 | 4.5V ~ 46V |
| 逻辑电压 | 5V |
| 持续输出电流 | 2A/桥 |
| 峰值电流 | 3A |
| 控制方式 | 2路PWM + 方向 |

**引脚图：**
```
┌──────────────────────┐
│  OUT1  OUT2  OUT3  OUT4│ ← 电机输出
│                        │
│  VS    VSS   GND       │ ← 电源
│  ENA   IN1   IN2       │ ← A桥控制
│  ENB   IN3   IN4       │ ← B桥控制
└──────────────────────┘
```
**替代型号：** TB6612FNG(低功耗)、DRV8833(低压)、BTS7960(大电流)

## 11. MCU常用引脚速查

### STM32F103C8T6 (BluePill)

```
        ┌────────────┐
 VBAT  1│            │48  PC13
 PC14  2│            │47  PC14
 PC15  3│            │46  VSSA
 PD0   4│            │45  VDDA
 PD1   5│ STM32F103 │44  PA0 (ADC/USART2)
 NRST  6│   C8T6     │43  PA1
 VSSA  7│            │42  PA2
 VDDA  8│            │41  PA3
 PA0   9│            │40  VSS
 PA1  10│            │39  VDD
 ...   ││            │...
        └────────────┘
```

| 关键引脚 | 功能 |
|----------|------|
| PA0~PA7 | ADC1_CH0~7 / TIM2_CH1~4 |
| PA9/PA10 | USART1_TX/RX |
| PA2/PA3 | USART2_TX/RX |
| PB6/PB7 | I2C1_SCL/SDA |
| PA5~PA7 | SPI1_SCK/MISO/MOSI |
| PA8~PA10 | TIM1_CH1/USART1 |
| PB0/PB1 | ADC1_CH8/CH9 |

---

### ESP32-WROOM-32

| 关键引脚 | 功能 |
|----------|------|
| GPIO34~39 | 仅输入(ADC2不可用) |
| GPIO25/26 | DAC1/DAC2 |
| GPIO21/22 | I2C SDA/SCL |

## 12. 快速选型决策树

```
需要运放?
├── 通用/直流 → LM358 / LM324
├── 低噪声/音频 → NE5532 / OPA2134
├── 高精度 → OPA2277 / AD8608
└── 高速 → LM6172 / AD8066

需要ADC?
├── I2C接口 → ADS1115(16位) / ADS1015(12位)
├── SPI接口 → ADS1256(24位) / MCP3208(12位)
└── MCU内置 → STM32 ADC (12位)

需要电机驱动?
├── 小电机(<1.2A) → TB6612FNG
├── 中电机(<2A) → L298N / DRV8833
├── 大电机(<43A) → BTS7960
└── BLDC → DRV8313 / L6234

需要DC-DC?
├── 降压(<3A) → LM2596 / XL4015
├── 降压(<5A) → TPS54331 / MP2315
├── 升压 → MC34063 / XL6009
└── 负压 → MC34063 (反相) / LM2662
```
