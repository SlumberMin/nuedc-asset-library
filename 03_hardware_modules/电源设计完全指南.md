# 电源设计完全指南

## 一、锂电池充放电设计

### 1.1 常用锂电池规格

| 类型 | 标称电压 | 充满电压 | 放电截止 | 典型容量 | 适用场景 |
|------|---------|---------|---------|---------|---------|
| 18650单节 | 3.7V | 4.2V | 3.0V | 2000-3500mAh | 便携设备 |
| 2S串联 | 7.4V | 8.4V | 6.0V | 2000-3500mAh | 电机驱动 |
| 3S串联 | 11.1V | 12.6V | 9.0V | 2000-5000mAh | 大功率系统 |
| 聚合物单节 | 3.7V | 4.2V | 3.0V | 500-10000mAh | 薄型设备 |

### 1.2 单节锂电池充电电路（TP4056）

```
         ┌─────────────────────────────────┐
  USB 5V──┤BAT+  TP4056模块  BAT+├───┬─── 锂电池+
         │         GND──────────┤   │
         └──────────────────────┘   ├── 保护板
                                    │
                              锂电池- ──┘
```

**TP4056关键参数：**
- 充电电流由 PROG 引脚电阻决定：R = 1200 / I（kΩ）
- 1kΩ → 1.2A，1.2kΩ → 1A，2kΩ → 580mA，10kΩ → 130mA
- 比赛推荐充电电流：500mA-1A（平衡速度与安全）

**带保护功能的完整充电方案：**
```
USB 5V ──→ TP4056 ──→ DW01A + 8205A保护板 ──→ 锂电池
                │              │
                LED充电指示    过充/过放/过流/短路保护
```

### 1.3 多节串联充电（BMS均衡板）

```
电池组负极 ──┐
             │    ┌──────────────────────┐
2S电池中点 ──┼───┤  2S BMS均衡板         ├─── 充电器+/-
             │    │  (过充4.25V断开)      │
3S电池中点1 ─┼───┤  (过放2.5V断开)       │
             │    │  (均衡电流~50mA)      │
3S电池中点2 ─┼───┤                       │
             │    └──────────────────────┘
电池组正极 ──┘
```

**BMS选型要点：**
- 均衡电流：被动均衡一般50-100mA，竞赛够用
- 过流保护值：根据最大负载电流选择（5A/10A/20A等）
- 内阻：越低越好，大电流场景需关注

### 1.4 锂电池放电注意事项

```
                    放电曲线特征
  电压(V)
  4.20 ├────╲
       │     ╲
  3.80 ├      ╲__________  ← 主要工作区间
       │                  ╲
  3.30 ├                   ╲
       │                    ╲
  3.00 ├                     ╲  ← 截止点
       └─────────────────────────
       0    20%   50%   80%  100%
                容量百分比
```

**放电保护设定：**
- 单节截止电压：3.0V（保护板自动断开）
- 实际使用建议3.2V以上截止，延长电池寿命
- 大电流放电时电压会跌落，需留余量

## 二、多路稳压设计

### 2.1 稳压方案选型总览

| 需求 | 方案 | 典型芯片 | 效率 | 纹波 | 适用 |
|------|------|---------|------|------|------|
| 5V/500mA以内 | LDO | AMS1117-5.0 | 低 | 极小 | MCU/传感器 |
| 3.3V/500mA以内 | LDO | AMS1117-3.3 | 低 | 极小 | MCU/通信模块 |
| 5V/2A+ | 降压DC-DC | MP1584/LM2596 | 高 | 较大 | 舵机/外设 |
| 3.3V/2A+ | 降压DC-DC | MP2315/TPS5430 | 高 | 较大 | WiFi/大电流 |
| 负压/-5V | 反相DC-DC | MC34063/ICL7660 | 中 | 中 | 运放供电 |
| 升压5V | 升压DC-DC | SX1308/MT3608 | 高 | 较大 | 单节锂电→5V |

### 2.2 典型多路电源架构（比赛推荐）

```
                        ┌──── AMS1117-3.3 ──→ 3.3V (MCU/传感器)
                        │         (LDO)
  锂电池7.4V ──┬────────┤
  (2S)         │        ├──── MP1584 ──→ 5V (舵机/运放)
               │        │    (DC-DC, 3A)
               │        │
               │        ├──── LM2596 ──→ 12V (若需升压用MT3608)
               │        │    (DC-DC, 3A)
               │        │
               │        └──── LDO/DC-DC ──→ 其他电压
               │
               └──→ 电机驱动直接供电 (VM引脚)
```

### 2.3 LDO电路设计（AMS1117）

```
          ┌────────────────────────┐
 Vin ──┬──┤IN        OUT├──┬── Vout
       │  │   AMS1117   │  │
      C1  └──────┬──────┘  C2
     10μF       GND       10μF
     (输入滤波)            (输出滤波)
```

**设计要点：**
- 输入输出压差：AMS1117需≥1.1V，压差大则功耗大
- 功耗计算：P = (Vin - Vout) × I，超过0.5W需加散热
- 输入电容：10μF陶瓷 + 100nF去耦（靠近芯片）
- 输出电容：22μF陶瓷（ESR要求见数据手册）
- **发热估算**：7.4V→3.3V@300mA = 1.23W，需散热片

### 2.4 DC-DC降压电路设计（MP1584）

```
Vin ──┬────────────────────────────────────┐
      │                                    │
     CIN              ┌────────┐           │
    22μF    BST──┤    │MP1584  │           │
      │         SW├────┤        │           │
      ├───L(4.7μH)┘    │  VIN  ├── EN     │
      │                 │  GND  ├── FB     │
      │                 └───┬───┘          │
      │                     │              │
      │              ┌──┬──┘               │
      │             R1  │                  │
      │              ├──┴── Vout           │
      │             R2  │                  │
      │              │  COUT               │
      │              │ 22μF                │
      │              │                     │
      GND────────────┴─────────────────────┘
```

**反馈电阻计算（MP1584）：**
```
Vout = 0.8 × (1 + R1/R2)
推荐：R2 = 10kΩ
5V输出：R1 = 52.5kΩ (用47k+5.6k串联)
3.3V输出：R1 = 31.25kΩ (用33kΩ)
```

**PCB布局要点：**
- 输入电容紧贴VIN和GND引脚
- 电感紧贴SW引脚，走线短粗
- FB分压电阻靠近FB引脚，远离SW噪声区
- 大面积铺地，SW节点铜箔面积适当

## 三、电源保护设计

### 3.1 过流保护

**方案一：保险丝（最简单）**
```
电源正极 ──→ 自恢复保险丝(PPTC) ──→ 负载
                额定电流 = 最大负载 × 1.5
```
- PPTC推荐：500mA/1A/2A（根据负载选型）
- 优点：免维护，自动恢复
- 缺点：响应慢（ms级）

**方案二：电子保险丝（精确保护）**
```
         ┌──────────────────────────┐
 Vin ────┤S              D├──── Vout (到负载)
         │   SI2301(PMOS) │
         └───────┬────────┘
                 │G
                 │
          ┌──────┴──────┐
          │ 限流比较器    │ ← 检测采样电阻电压
          │ (LM393)      │
          └──────────────┘
```

**方案三：MOSFET开关保护**
```
Vin ──→ P-MOS ──→ 负载
           │G
           │
    N-MOS驱动 ←── MCU GPIO控制
```

### 3.2 过压保护

```
Vin ──┬──→ 稳压器 → Vout
      │
     TVS二极管 (钳位电压 > 正常电压，< 器件最大耐压)
      │
     GND

推荐TVS选型：
- 5V线路：SMAJ5.0A（钳位9.2V）
- 12V线路：SMAJ12A（钳位19.9V）
- 3.3V线路：SMAJ3.3A（钳位7.3V）
```

### 3.3 反接保护

**方案一：串联二极管（简单但有压降）**
```
Vin(+) ──→ 肖特基二极管 ──→ 电路
            (SS34/1N5819)
            压降0.3-0.5V
```

**方案二：P-MOS反接保护（推荐）**
```
         ┌─────────────────────────┐
Vin(+)───┤S              D├──── 电路正极
         │   AO3401(PMOS) │
         └───────┬────────┘
                 │G
                 │
Vin(-) ──────────┘

工作原理：正接时PMOS导通（Vgs<0），反接时PMOS截止
优点：压降极小（mΩ级），功耗低
```

**方案三：理想二极管（最优）**
```
使用LTC4357/LTC4359等理想二极管控制器
+ 外部N-MOSFET
压降接近0V，适合大电流场合
```

### 3.4 软启动电路

```
Vin ──┬──→ P-MOS ──→ 负载
      │       │G
     R1       │
      ├───────┤
     C1      R2
      │       │
     GND     GND

软启动时间 ≈ R1 × C1
上电时C1充电，PMOS缓慢导通，限制浪涌电流
```

## 四、电源监控设计

### 4.1 电池电压检测

**电阻分压 + ADC检测：**
```
Vbat ──→ R1 ──┬──→ ADC引脚 (MCU)
               │
              R2
               │
              GND

分压计算：Vadc = Vbat × R2 / (R1 + R2)
要求：Vadc ≤ 3.3V（MCU ADC量程）

推荐：R1=10kΩ, R2=10kΩ → 2S电池(8.4V)时Vadc=4.2V ❌
修正：R1=20kΩ, R2=10kΩ → 2S电池(8.4V)时Vadc=2.8V ✓

精度提升：并联100nF滤波电容
```

**软件实现（STM32）：**
```c
// 电池电压检测
#define ADC_REF     3.3f    // ADC参考电压
#define ADC_MAX     4095    // 12位ADC
#define R1          20.0f   // 上臂电阻 kΩ
#define R2          10.0f   // 下臂电阻 kΩ

float get_battery_voltage(void) {
    uint16_t adc_val = HAL_ADC_GetValue(&hadc1);
    float v_adc = (float)adc_val / ADC_MAX * ADC_REF;
    float v_bat = v_adc * (R1 + R2) / R2;
    return v_bat;
}

// 电池电量估算（2S锂电池）
uint8_t get_battery_percent(float v_bat) {
    // 8.4V=100%, 7.4V≈50%, 6.0V=0%
    if (v_bat >= 8.4f) return 100;
    if (v_bat <= 6.0f) return 0;
    return (uint8_t)((v_bat - 6.0f) / 2.4f * 100);
}
```

### 4.2 电流检测

**方案一：采样电阻 + 运放**
```
         Rsense           ┌──────────┐
VCC ──→ ┌┤├──── 负载 ──→  │ INA180   ├──→ ADC
         │    (0.1Ω)      │ 增益=50  │
         └────────────────┤          │
                          └──────────┘

电流计算：I = Vout / (Rsense × Gain)
例：Rsense=0.1Ω, Gain=50, Vout=1V → I=0.2A=200mA
```

**方案二：霍尔电流传感器（ACS712）**
```
         ┌──────────────────────┐
VCC ──→  │  ACS712-5A/20A/30A  │──→ 模拟输出到ADC
         │  灵敏度:185mV/A(5A) │
         └──────────────────────┘
         优点：电气隔离，无插入损耗
```

### 4.3 电源健康监测系统

```c
// 电源监控综合代码
typedef struct {
    float battery_voltage;    // 电池电压
    float system_current;     // 系统电流
    uint8_t battery_percent;  // 电池电量%
    uint8_t charging;         // 充电状态
    uint8_t fault;            // 故障标志
} PowerStatus_t;

// 故障标志位定义
#define FAULT_NONE          0x00
#define FAULT_LOW_BAT       0x01  // 电池低压
#define FAULT_OVER_CURRENT  0x02  // 过流
#define FAULT_OVER_TEMP     0x04  // 过温

void power_monitor_task(void) {
    PowerStatus_t ps;
    
    // 采集电压电流
    ps.battery_voltage = get_battery_voltage();
    ps.battery_percent = get_battery_percent(ps.battery_voltage);
    ps.system_current  = get_system_current();
    
    // 故障检测
    ps.fault = FAULT_NONE;
    if (ps.battery_voltage < 6.2f)  ps.fault |= FAULT_LOW_BAT;
    if (ps.system_current > 5.0f)   ps.fault |= FAULT_OVER_CURRENT;
    
    // 低压报警处理
    if (ps.fault & FAULT_LOW_BAT) {
        // 降低系统功耗：关闭非必要外设
        HAL_GPIO_WritePin(LED_GPIO_Port, LED_Pin, GPIO_PIN_RESET);
        // 保存关键数据
        save_critical_data();
    }
    
    // 通过串口/LCD显示电源状态
    display_power_status(&ps);
}
```

| 检查项 | 要求 | ✓/✗ |
|--------|------|-----|

## 五、竞赛电源设计实战建议

### 5.1 推荐电源架构（通用）

```
                    ┌─ TP4056+BMS ──→ 1S/2S/3S锂电池
                    │
 充电宝/USB 5V ────┤
                    │
                    └─ 也可直接用充电宝USB输出

 2S锂电池7.4V ──┬── DC-DC(MP1584) ──→ 5V (舵机/大电流外设)
                │
                ├── LDO(AMS1117) ──→ 3.3V (MCU/传感器/通信模块)
                │
                └── 直接供电 ──→ 电机驱动模块VM
```

### 5.2 常见踩坑与解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| MCU复位 | 电机启动瞬间电压跌落 | 加大输入电容(470μF+)，电源隔离 |
| ADC不准 | 电源纹波干扰 | LDO供电+100nF去耦+取多次平均 |
| 模块烧毁 | 电源反接 | 加P-MOS反接保护 |
| 电池鼓包 | 过充/过放/过流 | 必须用BMS保护板 |
| DC-DC噪声大 | 布局不当 | 电感紧贴SW，FB远离SW |
| 舵机抖动 | 5V供电不足 | 独立DC-DC给舵机，加大电容 |
