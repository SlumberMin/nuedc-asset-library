# 电赛常用测试测量工具使用指南

## 一、示波器

### 1.1 基本操作速查

| 操作 | 按键/旋钮 | 说明 |
|------|----------|------|
| 时基调整 | TIME/DIV | 水平每格时间，调整波形水平密度 |
| 电压档位 | VOLTS/DIV | 垂直每格电压，调整波形垂直大小 |
| 触发电平 | LEVEL | 设置触发阈值，稳定波形 |
| 触发边沿 | Slope | 上升沿↑/下降沿↓触发 |
| 通道耦合 | Coupling | DC(直通)/AC(隔直)/GND |
| 自动量程 | Auto Scale | 一键自动设置（快速但不一定最优） |
| 光标测量 | Cursor | 手动测量ΔV/Δt |
| 自动测量 | Measure | 自动计算频率/幅值/占空比等 |

### 1.2 关键测量技巧

#### 测量PWM波形

```
1. 探头接信号线，地线就近接地
2. 耦合方式：DC
3. 触发方式：边沿触发，触发电平设在PWM幅值50%处
4. 时基：使一屏显示2~3个完整周期
5. 测量项：
   - 频率：Measure → Frequency
   - 占空比：Measure → Duty Cycle
   - 上升时间：Measure → Rise Time
   - 幅值：Measure → Vpp / Vmax / Vmin
```

#### 测量纹波

```
1. AC耦合（去除直流分量）
2. 带宽限制20MHz（减少噪声）
3. 电压档位调到mV级（20mV/div或更小）
4. 探头用×1档（减少噪声，但带宽降低）
5. 测量 Vpp 即为纹波幅度
```

#### 测量电源噪声/毛刺

```
1. DC耦合
2. 触发方式：单次触发（Single）
3. 触发条件：边沿 + 电平设在异常阈值
4. 时基调快（1μs/div~100ns/div）
5. 捕获毛刺波形，记录宽度和幅度
```

### 1.3 探头使用注意事项

```
✅ 正确：
- 使用探头自带接地弹簧（高频测量时）
- 地线尽量短，避免形成环路
- 使用×10档进行高频测量
- 测量前校准探头补偿电容

❌ 错误：
- 用长鳄鱼夹地线（引入电感，高频振铃）
- 未校准探头补偿（波形失真）
- 信号频率超过探头带宽
- 探头过载（超过最大输入电压）
```

### 1.4 常见波形问题诊断

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 波形振铃/过冲 | 探头补偿不足/地线太长 | 调补偿+缩短地线 |
| 波形模糊/重叠 | 触发不稳 | 调触发电平/选正确边沿 |
| 高频丢失 | 带宽不足 | 切×10档/换高频探头 |
| 直流失调 | AC/DC耦合选错 | 检查耦合设置 |
| 幅值不对 | ×1/×10档设置不匹配 | 示波器设探头倍率 |

## 二、逻辑分析仪

### 2.1 基本原理

逻辑分析仪是**数字信号的示波器**，只识别高/低电平，可同时采集多路数字信号，特别适合**通信协议解码**。

### 2.2 常用逻辑分析仪

| 型号 | 通道 | 采样率 | 价格区间 | 推荐 |
|------|------|--------|----------|------|
| Saleae Logic 8 | 8ch | 100MS/s | 贵 | 专业首选 |
| Kingst LA2016 | 16ch | 200MS/s | 中等 | 性价比之选 |
| DSLogic Plus | 16ch | 400MS/s | 中等 | 高性能 |
| 便宜USB逻辑分析仪 | 8ch | 24MS/s | <50元 | 入门够用 |

### 2.3 协议解码配置

#### UART解码

```
设置：
- 通道：CH0 接 TX/RX 信号线
- 采样率：≥10倍波特率（115200时用2MS/s以上）
- 触发：下降沿（起始位）
- 协议分析：Async Serial
  - 波特率：115200
  - 数据位：8
  - 停止位：1
  - 校验：None
  - 极性：Normal
```

#### SPI解码

```
设置：
- 通道：CLK→CH0, MOSI→CH1, MISO→CH2, CS→CH3
- 采样率：≥10倍SPI时钟
- 触发：CS下降沿
- 协议分析：SPI
  - Clock：Rising Edge
  - Bit Order：MSB First
  - CS Active Low
```

#### I2C解码

```
设置：
- 通道：SCL→CH0, SDA→CH1
- 采样率：≥10倍I2C时钟（400kHz时用5MS/s以上）
- 触发：SDA下降沿（起始条件）
- 协议分析：I2C
  - 地址位：7bit
```

### 2.4 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 解码全是FF | 接线反了 | 检查MOSI/MISO |
| 间歇性错误 | 采样率不够 | 提高采样率 |
| I2C解码失败 | SCL/SDA接反 | 交换通道 |
| 通信量太大 | 没设触发条件 | 设精确触发 |

## 三、频谱仪

### 3.1 基本概念

频谱仪显示信号的**频率-幅度**关系（频域），用于分析信号的频率成分、谐波、噪声等。

```
横轴：频率 (Hz)
纵轴：幅度 (dBm 或 dBV)
```

### 3.2 关键参数设置

| 参数 | 说明 | 设置建议 |
|------|------|----------|
| 频率范围 (Span) | 显示的频率区间 | 覆盖信号频率+谐波 |
| 中心频率 (CF) | 屏幕中心的频率 | 设为信号频率 |
| 参考电平 (Ref Level) | 屏幕顶部的电平 | 设为信号幅值附近 |
| 分辨率带宽 (RBW) | 频率分辨率 | 越小精度越高，扫速越慢 |
| 视频带宽 (VBW) | 平滑噪声 | VBW ≤ RBW |
| 衰减 (Attenuation) | 输入衰减 | 避免过载 |

### 3.3 电赛中的应用场景

#### 测量信号源谐波失真

```
1. 连接信号源输出到频谱仪
2. 设置中心频率为基波频率
3. Span设为基波频率的5倍以上
4. 记录基波幅度和各次谐波幅度
5. THD = √(V2²+V3²+...)/V1 × 100%
```

#### 测量电源EMI

```
1. 使用近场探头靠近PCB
2. 频率范围：150kHz ~ 30MHz（传导）或 30MHz ~ 1GHz（辐射）
3. 记录各频率点的峰值
4. 对比标准限值（CISPR 11/EN 55011）
```

#### 分析射频信号

```
1. 设置中心频率和Span覆盖目标频段
2. RBW根据信号带宽设置（一般为信号BW的1%~10%）
3. 测量信号功率、占用带宽、邻道泄漏等
```

### 3.4 替代方案

电赛中通常没有频谱仪，替代方案：

| 方法 | 工具 | 精度 | 适用场景 |
|------|------|------|----------|
| FFT分析 | 示波器内置FFT | 中 | 低频信号频谱 |
| ADC+FFT | MCU+MATLAB | 中 | 自制简易频谱仪 |
| SA44B | 手持频谱仪 | 中 | 有预算时 |
| ADALM-PLUTO | SDR平台 | 中 | 射频频谱分析 |

## 四、网络分析仪

### 4.1 基本概念

网络分析仪测量**网络（电路/器件）的频率响应**，包括：
- **S参数**：S11(反射/驻波)、S21(传输/增益)
- **阻抗**：随频率变化的阻抗
- **群延迟**：信号通过系统的延迟

### 4.2 矢量网络分析仪（VNA）基本操作

```
1. 校准（必须！）
   - 开路校准（Open）
   - 短路校准（Short）
   - 负载校准（Load/50Ω）
   - 直通校准（Through）——双端口测量时

2. 设置
   - 起始/终止频率
   - 扫描点数（一般201~1001点）
   - 中频带宽（IF BW）

3. 测量
   - S11：反射系数 → VSWR、回波损耗
   - S21：传输系数 → 增益/插损
```

### 4.3 电赛中的应用场景

#### 天线匹配测量

```
测S11（反射系数）：
1. 校准VNA
2. 连接天线到Port 1
3. 扫描目标频段
4. S11 < -10dB 的频段即为可用带宽
5. 标记谐振频率和带宽
```

#### 滤波器特性测量

```
测S21（传输特性）：
1. 校准（含直通校准）
2. 滤波器输入→Port 1，输出→Port 2
3. 观察通带增益、阻带衰减、截止频率
4. 测量-3dB带宽
```

### 4.4 替代方案

| 方案 | 工具 | 精度 | 频率范围 |
|------|------|------|----------|
| NanoVNA | 便携VNA | 低~中 | 50kHz~数百MHz |
| LiteVNA | 改进版NanoVNA | 中 | 50kHz~6GHz |
| ADALM-PLUTO | SDR平台 | 中 | 325MHz~3.8GHz |
| 自制扫频 | MCU+DDS+检波 | 低 | 根据DDS而定 |

## 五、其他常用工具

### 5.1 万用表

```
常用测量：
- 直流电压：并联测量
- 直流电流：串联测量（注意档位！）
- 电阻：断电测量
- 二极管/通断：蜂鸣档
- 电容：部分万用表支持

注意事项：
⚠️ 测电流时绝对不能并联！会短路烧保险
⚠️ 测量前确认量程，先用大量程试测
⚠️ 测量高频信号时万用表只显示平均值，不准确
```

### 5.2 信号发生器

```
常用波形：
- 正弦波：测试放大器频率响应
- 方波：测试数字电路时序
- 三角波：测试线性度
- 脉冲：测试响应速度
- 扫频信号：自动扫频测试
- 噪声：测试噪声性能

常用功能：
- AM/FM调制
- 扫频（Sweep）
- 突发（Burst）
- 任意波形（ARB）
```

### 5.3 电烙铁/焊接工具

```
推荐配置：
- 焊台：Hakko FX-888D / 白光 / 快克
- 烙铁头：刀头（拖焊）+ 尖头（精细焊接）
- 焊锡：0.8mm含铅焊锡（电赛比赛时可用）
- 助焊剂：免清洗助焊剂
- 吸锡带/吸锡器

焊接要点：
- 温度：有铅焊锡 300~330°C，无铅 350~380°C
- 先加热焊盘，再送锡
- 焊接时间 < 3秒（防损坏元件）
- 焊后检查：光滑、有光泽、呈圆锥形
```

### 5.4 热风枪

```
用途：
- QFN/BGA等底部焊盘元件的拆焊
- 热缩管收缩
- 批量回流

温度设置：
- 拆焊：300~350°C，风速中
- 热缩管：150~200°C
- BGA返修：250~280°C（需要均匀加热）
```

## 六、工具使用效率技巧

### 6.1 测量效率提升

```
1. 示波器快捷键记忆：Save/Recall → 保存常用设置
2. 逻辑分析仪预设：保存协议解码配置
3. 自动测量：善用示波器的自动测量功能
4. 屏幕截图：USB导出波形，用于报告
5. 光标测量：手动测不规则波形时用光标
```

### 6.2 调试流程建议

```
1. 先用万用表检查供电和基本连接
2. 用示波器看关键节点波形
3. 通信问题用逻辑分析仪抓协议
4. 频域问题用FFT或频谱仪
5. 阻抗匹配问题用VNA
```
