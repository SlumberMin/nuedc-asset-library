# 电赛常用保护器件速查卡

> TVS / ESD / 保险丝 / PPTC / 热敏电阻 | 选型+电路+参数速查

## 一、TVS 二极管（瞬态电压抑制器）

### 1.1 工作原理
```
正常工作：高阻态（不导通）
过压时：纳秒级钳位，将电压限制在安全值
响应时间：< 1ns（极快）
```

### 1.2 关键参数
```
┌──────────────┬──────────────────────────────────────┐
│ 参数          │ 含义                                  │
├──────────────┼──────────────────────────────────────┤
│ VRWM         │ 最大反向工作电压（必须>电路工作电压）   │
│ VBR          │ 击穿电压（TVS开始导通的电压）          │
│ VC           │ 钳位电压（最大钳位，必须<器件耐压）     │
│ IPP          │ 最大脉冲电流                          │
│ PPP          │ 最大脉冲功率（IPP × VC）              │
└──────────────┴──────────────────────────────────────┘
```

### 1.3 电赛常用型号

```
单向TVS（直流线路保护）：
┌──────────┬────────┬────────┬────────┬──────────────┐
│ 型号      │ VRWM   │ VBR    │ VC@IPP │ 应用          │
├──────────┼────────┼────────┼────────┼──────────────┤
│ SMBJ5.0A │ 5.0V   │ 6.4V   │ 9.2V   │ 5V逻辑保护    │
│ SMBJ12A  │ 12V    │ 13.3V  │ 19.9V  │ 12V电源保护   │
│ SMBJ24A  │ 24V    │ 26.7V  │ 38.9V  │ 24V电源保护   │
│ P6KE6.8A │ 5.8V   │ 6.8V   │ 10.5V  │ 5V信号保护    │
│ SMCJ36A  │ 36V    │ 40V    │ 58.1V  │ 电机驱动保护  │
└──────────┴────────┴────────┴────────┴──────────────┘

双向TVS（交流/差分信号）：
┌──────────┬────────┬────────┬──────────────┐
│ 型号      │ VRWM   │ VBR    │ 应用          │
├──────────┼────────┼────────┼──────────────┤
│ SMBJ5.0CA│ ±5.0V  │ ±6.4V  │ USB数据线     │
│ SMBJ15CA │ ±15V   │ ±16.7V │ RS485总线     │
│ P6KE18CA │ ±15V   │ ±18V   │ CAN总线       │
└──────────┴────────┴────────┴──────────────┘
```

### 1.4 选型规则
```
1. VRWM ≥ 电路最大工作电压（留10-20%余量）
2. VC < 被保护器件的最大耐压
3. IPP > 预期最大浪涌电流
4. 信号线用低电容TVS（<100pF），电源线可用电容大的
```

### 1.5 典型电路
```
电源入口保护：
Vin ──┬── TVS ──┬── GND
      │         │
      └─ 保险丝 ┘
      (先熔断，后TVS钳位)

信号线保护：
Signal ──┬── TVS ──┬── GND
         │         │
     信号接入    保护器件
```

## 二、ESD 保护器件

### 2.1 与TVS的区别
```
┌──────────┬──────────────┬──────────────┐
│ 特性      │ ESD器件       │ TVS           │
├──────────┼──────────────┼──────────────┤
│ 功率      │ 较小（<100W） │ 大（600W+）   │
│ 电容      │ 极低（<1pF）  │ 较高（几十pF）│
│ 响应时间  │ <1ns          │ <1ns          │
│ 应用      │ 高速信号线    │ 电源/低速信号 │
│ 封装      │ SOT-23/SOD   │ SMB/SMC      │
└──────────┴──────────────┴──────────────┘
```

### 2.2 电赛常用型号

```
┌──────────┬──────────┬────────┬──────────────────────┐
│ 型号      │ 通道数   │ 电容    │ 应用                  │
├──────────┼──────────┼────────┼──────────────────────┤
│ ESD5V3U2 │ 2路      │ 0.5pF  │ USB 2.0 D+/D-        │
│ USBLC6-2 │ 2路      │ 3.5pF  │ USB全速保护           │
│ PESD5V0L2│ 2路      │ 0.9pF  │ 高速差分信号          │
│ ESD9X5.0 │ 1路      │ 0.5pF  │ 单线ESD保护           │
│ SRV05-4  │ 4路+1电源│ 0.8pF  │ HDMI/高速数字接口     │
└──────────┴──────────┴────────┴──────────────────────┘
```

### 2.3 USB ESD保护电路
```
USB_VBUS ──── SMBJ5.0A ──── GND
USB_D+  ──── ESD5V3U2 ──── GND
USB_D-  ──── ESD5V3U2 ──── GND

注意事项：
- ESD器件尽量靠近连接器放置
- 接地路径要短而宽
- GND引脚直接连到大面积地平面
```

## 三、保险丝（Fuse）

### 3.1 类型对比

```
┌──────────────┬──────────────┬──────────────┬──────────────┐
│ 类型          │ 响应速度      │ 体积          │ 成本          │
├──────────────┼──────────────┼──────────────┼──────────────┤
│ 玻璃管保险丝  │ 中等          │ 大            │ 低            │
│ 贴片保险丝    │ 中等          │ 小（1206等）  │ 低            │
│ 自恢复PPTC   │ 慢（ms级）    │ 小            │ 中            │
│ 快断型        │ 快（ms级）    │ 中            │ 低            │
│ 慢断型        │ 慢（秒级）    │ 中            │ 低            │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

### 3.2 选型要点
```
1. 额定电流 = 正常工作电流 × 1.5~2.0（降额使用）
2. 额定电压 > 电路最大电压
3. 熔断特性：
   - 快断（F）：精密电路、信号线保护
   - 慢断（T）：电机、电源等有浪涌的场合
4. 分断能力 > 电路最大短路电流
```

### 3.3 电赛常用型号

```
贴片保险丝（常用）：
┌──────────┬──────────┬──────────┬──────────────────┐
│ 型号      │ 额定电流  │ 封装      │ 应用              │
├──────────┼──────────┼──────────┼──────────────────┤
│ 0467001   │ 1A       │ 0603     │ 信号/传感器供电    │
│ 0467002   │ 2A       │ 0603     │ 逻辑电路保护       │
│ 0466003   │ 3A       │ 1206     │ 5V电源总线         │
│ 0466005   │ 5A       │ 1206     │ 电机/大功率供电     │
│ F1206HI5A │ 5A       │ 1206     │ 高分断能力          │
└──────────┴──────────┴──────────┴──────────────────┘

玻璃管保险丝（调试用）：
5×20mm：0.5A / 1A / 2A / 3A / 5A
```

### 3.4 保险丝座
```
调试建议：用保险丝座 + 玻璃管保险丝
方便更换，快速恢复，适合比赛现场调试
```

## 四、PPTC（自恢复保险丝）

### 4.1 工作原理
```
正常工作：低阻态（mΩ级），相当于导线
过流时：PTC效应，温度升高→阻值急剧增大→限制电流
故障排除后：温度降低→自动恢复低阻态

响应时间：ms~s级（比普通保险丝慢）
```

### 4.2 关键参数
```
┌──────────────┬──────────────────────────────────────┐
│ 参数          │ 含义                                  │
├──────────────┼──────────────────────────────────────┤
│ Ihold        │ 保持电流（最大持续工作电流）            │
│ Itrip        │ 触发电流（PPTC动作的最小电流）         │
│ Vmax         │ 最大工作电压                          │
│ Imax         │ 最大故障电流（分断能力）               │
│ Rmin         │ 初始最小电阻                          │
│ Rmax         │ 动作后最大电阻                        │
└──────────────┴──────────────────────────────────────┘
```

### 4.3 电赛常用型号

```
┌──────────┬────────┬────────┬────────┬──────────────────┐
│ 型号      │ Ihold  │ Itrip  │ Vmax   │ 应用              │
├──────────┼────────┼────────┼────────┼──────────────────┤
│ MF-MSMF050│ 0.50A │ 1.00A  │ 60V   │ USB/传感器保护    │
│ MF-MSMF100│ 1.00A │ 2.00A  │ 30V   │ 5V逻辑电源        │
│ MF-MSMF200│ 2.00A │ 4.00A  │ 15V   │ 低功率电机        │
│ MF-NSMF110│ 1.10A │ 2.20A  │ 60V   │ 通用12V线路       │
│ MF-JTD250 │ 2.50A │ 5.00A  │ 16V   │ 电机/舵机驱动     │
│ MF-R300   │ 3.00A │ 6.00A  │ 30V   │ 电源总线保护       │
│ MF-R500   │ 5.00A │ 10.0A  │ 16V   │ 大功率电机保护     │
└──────────┴────────┴────────┴────────┴──────────────────┘
```

### 4.4 选型规则
```
1. Ihold > 电路正常最大工作电流 × 1.2~1.5
2. Itrip < 电路允许的最大故障电流
3. Vmax > 电路最大工作电压
4. 注意：高温环境需降额（温度每升高10°C，Ihold降约5%）
```

### 4.5 典型应用电路
```
USB端口保护：
USB_VBUS ── PPTC(0.5A) ── 5V_BUS
USB_D+ ────────────────── MCU
USB_D- ────────────────── MCU

电机驱动保护：
12V ── PPTC(2.5A) ── H桥 ── 电机
        (串联在电源正极)
```

## 五、NTC 热敏电阻

### 5.1 类型
```
NTC（负温度系数）：温度↑ → 阻值↓（常用）
PTC（正温度系数）：温度↑ → 阻值↑
```

### 5.2 应用场景

```
┌──────────────┬──────────────────────────────────────┐
│ 应用          │ 说明                                  │
├──────────────┼──────────────────────────────────────┤
│ 浪涌抑制      │ 串联在电源入口，冷态高阻限制浪涌电流   │
│ 温度测量      │ 分压/电桥测量阻值→查表/公式得温度      │
│ 软启动        │ 串联NTC，上电时限制充电电流            │
└──────────────┴──────────────────────────────────────┘
```

### 5.3 浪涌抑制 NTC

```
常用型号：
┌──────────┬──────────┬──────────┬──────────────────┐
│ 型号      │ 25°C阻值  │ 最大电流  │ 应用              │
├──────────┼──────────┼──────────┼──────────────────┤
│ MF72-5D9  │ 5Ω       │ 1A       │ 小功率电源         │
│ MF72-10D7 │ 10Ω      │ 0.7A     │ LED驱动           │
│ MF72-2.5D11│ 2.5Ω    │ 2A       │ 中功率开关电源     │
│ MF72-1D11 │ 1Ω       │ 4A       │ 大功率电源         │
│ NTC 5D-11 │ 5Ω       │ 3A       │ 通用电源软启动     │
└──────────┴──────────┴──────────┴──────────────────┘

电路：
AC_L ──── NTC ──── 整流桥
AC_N ──────────── 整流桥

工作原理：
上电瞬间：NTC冷态→高阻→限制浪涌电流
正常工作：NTC热态→低阻→几乎无损耗
```

### 5.4 温度测量 NTC

```
常用型号：10kΩ @ 25°C（最常用）
B值：3435 / 3950

温度计算公式（Steinhart-Hart简化）：
1/T = 1/T0 + (1/B) * ln(R/R0)

其中：
T0 = 298.15K (25°C)
R0 = 10kΩ
B = 3950
R = 测量电阻值

实际使用：查表法更高效

电路（分压法）：
VCC ──── R_fixed ───┬─── NTC ──── GND
                     │
                     └── ADC引脚

V_adc = VCC * NTC / (R_fixed + NTC)

代码：
float ntc_calc_temp(uint16_t adc_val) {
    float R_ntc = R_FIXED * adc_val / (4096.0f - adc_val);
    float temp_k = 1.0f / (1.0f/298.15f + logf(R_ntc/10000.0f)/3950.0f);
    return temp_k - 273.15f;  // 返回摄氏度
}
```

## 六、选型速查总结

### 按保护场景选择

```
┌─────────────────────┬──────────────────────────────┐
│ 保护场景              │ 推荐器件                      │
├─────────────────────┼──────────────────────────────┤
│ 电源入口浪涌          │ TVS + 保险丝/PPTC + NTC       │
│ USB端口ESD           │ ESD器件（低电容） + PPTC       │
│ RS485/CAN总线        │ TVS（双向） + PPTC             │
│ 电机驱动电源          │ TVS + PPTC（自恢复）           │
│ 高速信号线            │ ESD器件（<1pF）                │
│ 电源过流保护          │ PPTC（可恢复）/ 保险丝（一次性）│
│ 电源软启动            │ NTC热敏电阻                    │
│ 温度监测              │ NTC 10kΩ B3950                │
└─────────────────────┴──────────────────────────────┘
```

### 典型多级保护电路

```
电源入口完整保护：
Vin ── NTC(浪涌) ── PPTC(过流) ──┬── TVS(钳位) ──┬── GND
                                   │              │
                                   └── 5V稳压 ────┘

信号端口保护：
Connector ── ESD(静电) ── Series R ── MCU Pin
                         (限流100Ω)

总线保护（RS485/CAN）：
Bus_A ──┬── PPTC ──┬── TVS ──┬── 收发器A
        │          │         │
Bus_B ──┘          └── GND   └── 收发器B
```
