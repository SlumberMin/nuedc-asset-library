## 1. 直流有刷电机

### 1.1 特性速查

| 参数 | 典型范围 |
|------|---------|
| 电压 | 3.3V~48V |
| 转速 | 100~20000 RPM |
| 扭矩 | 0.01~10 N·m(加减速箱) |
| 控制方式 | PWM调压 |
| 反馈 | 编码器/电位器/无 |

### 1.2 驱动方案

**方案A: MOSFET H桥 (小功率 <5A)**

```
         VCC
          |
    +-----+-----+
    |Q1(P)  Q3(P)|
    |     |     |
IN1-+  MOTOR  +-IN2
    |     |     |
    |Q2(N)  Q4(N)|
    +-----+-----+
          |
         GND

推荐MOSFET: 
  高侧: AO3401(P-MOS), SI2301
  低侧: AO3400(N-MOS), IRF3205(大电流)
  驱动IC: IR2104, FD6288, BTN7971
```

**方案B: 专用驱动IC**

| IC | 电压 | 电流 | 接口 | 特点 |
|----|------|------|------|------|
| L298N | 5~46V | 2A | GPIO | 经典、压降大 |
| TB6612FNG | 2.5~13.5V | 1.2A | PWM+DIR | 效率高、推荐 |
| DRV8833 | 2.7~10.8V | 1.5A | PWM | 小型、双路 |
| BTN7971 | 6~40V | 43A | PWM+DIR | 大电流 |
| BTS7960 | 6~24V | 43A | PWM+DIR | 大电流、带保护 |

### 1.3 PWM控制代码

```c
// TB6612FNG驱动
typedef struct {
    TIM_HandleTypeDef *htim;
    uint32_t ch1, ch2;     // PWM通道
    GPIO_TypeDef *dir_port;
    uint16_t in1_pin, in2_pin;
} DC_Motor_t;

void DC_Motor_Init(DC_Motor_t *motor) {
    HAL_TIM_PWM_Start(motor->htim, motor->ch1);
    HAL_TIM_PWM_Start(motor->htim, motor->ch2);
}

void DC_Motor_SetSpeed(DC_Motor_t *motor, int16_t speed) {
    if (speed >= 0) {
        HAL_GPIO_WritePin(motor->dir_port, motor->in1_pin, GPIO_PIN_SET);
        HAL_GPIO_WritePin(motor->dir_port, motor->in2_pin, GPIO_PIN_RESET);
        __HAL_TIM_SET_COMPARE(motor->htim, motor->ch1, speed);
        __HAL_TIM_SET_COMPARE(motor->htim, motor->ch2, 0);
    } else {
        HAL_GPIO_WritePin(motor->dir_port, motor->in1_pin, GPIO_PIN_RESET);
        HAL_GPIO_WritePin(motor->dir_port, motor->in2_pin, GPIO_PIN_SET);
        __HAL_TIM_SET_COMPARE(motor->htim, motor->ch1, 0);
        __HAL_TIM_SET_COMPARE(motor->htim, motor->ch2, -speed);
    }
}
```

## 2. 步进电机

### 2.1 特性速查

| 参数 | 典型值 |
|------|-------|
| 步距角 | 1.8°(200步/圈) 或 0.9°(400步/圈) |
| 电压 | 3.3V~48V |
| 电流 | 0.5A~3A/相 |
| 类型 | 两相混合式(最常见) |
| 细分 | 1, 2, 4, 8, 16, 32, 64, 128, 256 |
| 反馈 | 通常无 (开环) |

### 2.2 驱动方案

| 驱动IC | 电流 | 细分 | 接口 | 推荐度 |
|--------|------|------|------|--------|
| A4988 | 2A | 1/16 | STEP+DIR | ⭐⭐⭐⭐ |
| DRV8825 | 2.5A | 1/32 | STEP+DIR | ⭐⭐⭐⭐ |
| TMC2209 | 2A | 1/256 | STEP+DIR/UART | ⭐⭐⭐⭐⭐ |
| TMC2226 | 2.8A | 1/256 | STEP+DIR/UART | ⭐⭐⭐⭐⭐ |
| TB6600 | 4A | 1/16 | STEP+DIR | ⭐⭐⭐ |
| DM542 | 4.2A | 1/256 | STEP+DIR | ⭐⭐⭐ |

### 2.3 驱动电路

```
STM32 ──→ [TMC2209] ──→ 步进电机
            │
     ┌──────┼──────┐
     STEP  DIR   MS1~MS3 (细分设置)
     (PWM) (GPIO) (GPIO)
     
接线:
  VCC: 逻辑电源 3.3V/5V
  VM: 电机电源 8~29V (TMC2209)
  1A,1B,2A,2B: 两相四线
  EN: 使能(低有效)
  STEP: 脉冲输入
  DIR: 方向
  MS1~MS3: 细分选择
```

### 2.4 速度控制代码

```c
// 定时器产生步进脉冲
typedef struct {
    TIM_HandleTypeDef *htim;
    uint32_t channel;
    GPIO_TypeDef *dir_port;
    uint16_t dir_pin;
    float steps_per_rev;    // 每圈步数 (含细分)
    uint32_t current_freq;  // 当前脉冲频率
} Stepper_t;

void Stepper_SetSpeed(Stepper_t *motor, float rpm) {
    if (rpm >= 0) {
        HAL_GPIO_WritePin(motor->dir_port, motor->dir_pin, GPIO_PIN_SET);
    } else {
        HAL_GPIO_WritePin(motor->dir_port, motor->dir_pin, GPIO_PIN_RESET);
        rpm = -rpm;
    }
    
    // 计算脉冲频率
    uint32_t freq = (uint32_t)(rpm * motor->steps_per_rev / 60.0f);
    motor->current_freq = freq;
    
    if (freq > 0) {
        uint32_t arr = (SystemCoreClock / (freq * 2)) - 1;
        __HAL_TIM_SET_AUTORELOAD(motor->htim, arr);
        __HAL_TIM_SET_COMPARE(motor->htim, motor->channel, arr / 2);
        HAL_TIM_PWM_Start(motor->htim, motor->channel);
    } else {
        HAL_TIM_PWM_Stop(motor->htim, motor->channel);
    }
}

// 位置控制 (脉冲计数)
void Stepper_MoveSteps(Stepper_t *motor, int32_t steps, uint32_t speed_hz) {
    if (steps > 0) {
        HAL_GPIO_WritePin(motor->dir_port, motor->dir_pin, GPIO_PIN_SET);
    } else {
        HAL_GPIO_WritePin(motor->dir_port, motor->dir_pin, GPIO_PIN_RESET);
        steps = -steps;
    }
    
    for (int32_t i = 0; i < steps; i++) {
        HAL_GPIO_WritePin(motor->dir_port, motor->dir_pin, GPIO_PIN_SET);
        HAL_Delay_us(1000000 / speed_hz / 2);
        HAL_GPIO_WritePin(motor->dir_port, motor->dir_pin, GPIO_PIN_RESET);
        HAL_Delay_us(1000000 / speed_hz / 2);
    }
}
```

### 2.5 加减速曲线

```
梯形加减速:
  加速段: 频率线性增加 → f(t) = f_start + a*t
  匀速段: 保持最大频率
  减速段: 频率线性减小 → f(t) = f_max - a*t

S形加减速 (更平滑):
  加速度本身也做平滑过渡
  → 7段: 加加速→匀加速→减加速→匀速→加减速→匀减速→减减速
```

## 3. 无刷直流电机 (BLDC)

### 3.1 特性速查

| 参数 | 典型值 |
|------|-------|
| 电压 | 6V~48V (航模可达50V+) |
| KV值 | 100~2000 RPM/V |
| 相数 | 3相 |
| 换相方式 | 电子换相(ESC/驱动IC) |
| 传感器 | 霍尔传感器 或 无感(反电动势) |

### 3.2 驱动方案

**方案A: ESC电调 (航模电机)**

```
STM32 ──PWM(50Hz)──→ ESC ──3相──→ BLDC
                         │
                    电池供电

PWM信号: 1ms=停转, 1.5ms=中速, 2ms=满速
```

**方案B: 三相全桥驱动**

```
       VCC
    ┌──┼──┐
    Q1 Q3 Q5  (上桥臂)
    │  │  │
    A  B  C  ──→ 电机三相
    │  │  │
    Q2 Q4 Q6  (下桥臂)
    └──┼──┘
       GND

推荐驱动IC:
  - DRV8301/DRV8302: 三相预驱+电流检测
  - IR2130/IR2136: 三相预驱
  - FD6288: 低成本三相预驱
  - EG2131/EG2136: 国产替代

MOSFET: IRF3205(大电流), AOD4184, IRLR7843
```

### 3.3 BLDC换相控制 (六步换向)

```c
// 霍尔传感器六步换向
typedef struct {
    GPIO_TypeDef *hall_port;
    uint16_t ha_pin, hb_pin, hc_pin;
    TIM_HandleTypeDef *pwm_htim;
    uint32_t pwm_ch_a, pwm_ch_b, pwm_ch_c;
} BLDC_t;

// 霍尔状态 → 换相表
static const uint8_t hall_to_phase[8] = {
    0,  // 000: 错误
    5,  // 001: 相C+
    3,  // 010: 相B+
    4,  // 011: 相A-
    1,  // 100: 相A+
    6,  // 101: 相C-
    2,  // 110: 相B-
    0,  // 111: 错误
};

void BLDC_Commutate(BLDC_t *motor, uint8_t hall_state, uint16_t duty) {
    uint8_t phase = hall_to_phase[hall_state];
    
    // 全部关闭
    HAL_TIM_PWM_Stop(motor->pwm_htim, motor->pwm_ch_a);
    HAL_TIM_PWM_Stop(motor->pwm_htim, motor->pwm_ch_b);
    HAL_TIM_PWM_Stop(motor->pwm_htim, motor->pwm_ch_c);
    
    switch (phase) {
    case 1: // A+ B-
        Set_Phase_A_PWM(duty);
        Set_Phase_B_Low();
        Set_Phase_C_Float();
        break;
    case 2: // B+ C-
        Set_Phase_B_PWM(duty);
        Set_Phase_C_Low();
        Set_Phase_A_Float();
        break;
    case 3: // B+ A-
        Set_Phase_B_PWM(duty);
        Set_Phase_A_Low();
        Set_Phase_C_Float();
        break;
    case 4: // C+ A-
        Set_Phase_C_PWM(duty);
        Set_Phase_A_Low();
        Set_Phase_B_Float();
        break;
    case 5: // C+ B-
        Set_Phase_C_PWM(duty);
        Set_Phase_B_Low();
        Set_Phase_A_Float();
        break;
    case 6: // A+ C-
        Set_Phase_A_PWM(duty);
        Set_Phase_C_Low();
        Set_Phase_B_Float();
        break;
    }
}
```

### 3.4 FOC (磁场定向控制) 简介

```
比六步换向更高效、更平滑:
  Clarke变换: (Ia,Ib,Ic) → (Iα,Iβ)
  Park变换:  (Iα,Iβ) → (Id,Iq)
  PI控制: Id→0 (无励磁), Iq→I_ref (力矩)
  反Park: (Vd,Vq) → (Vα,Vβ)
  SVPWM: (Vα,Vβ) → 三相PWM占空比

推荐库:
  - SimpleFOC (Arduino/STM32)
  - ODrive (开源高性能)
```

## 4. 音圈电机 (VCM)

### 4.1 特性速查

| 参数 | 典型值 |
|------|-------|
| 类型 | 直线/旋转 |
| 行程 | ±0.1mm ~ ±5mm(直线) |
| 响应 | 极快 (ms级) |
| 力常数 | 1~50 N/A |
| 控制方式 | 电流控制 |
| 应用 | 精密定位、对焦、振动台 |

### 4.2 驱动电路

```
方案: H桥 + 电流采样 + 电流环

STM32 ──PWM──→ [H桥驱动] ──→ 音圈电机
                    │
               [采样电阻] ──→ ADC ──→ 电流PI

关键:
  - 需要高带宽电流环 (10kHz以上)
  - 采样电阻: 0.1~1Ω, 精度1%
  - 运放: 高速运放 (OPA2350, LM358带宽不够)
```

```c
// 音圈电机电流控制
typedef struct {
    PID_t current_pid;      // 电流环PID
    float R_coil;           // 线圈电阻
    float L_coil;           // 线圈电感
    float force_constant;   // 力常数 N/A
    float i_max;            // 最大电流
} VCM_t;

float VCM_Control(VCM_t *vcm, float force_ref, float i_actual) {
    // 力 → 电流参考
    float i_ref = force_ref / vcm->force_constant;
    i_ref = constrain(i_ref, -vcm->i_max, vcm->i_max);
    
    // 电流PI控制
    float pwm = PID_Calculate(&vcm->current_pid, i_ref, i_actual);
    
    return pwm;
}
```

## 5. 压电驱动器 (PZT)

### 5.1 特性速查

| 参数 | 典型值 |
|------|-------|
| 驱动电压 | 0~150V (或±75V) |
| 位移 | 1~100μm |
| 分辨率 | nm级 |
| 响应 | 极快 (μs级) |
| 频率 | DC~kHz |
| 应用 | 精密定位、微调焦、超声换能 |

### 5.2 驱动方案

**方案A: 高压运放驱动**

```
DAC ──→ [高压运放] ──→ 压电片
         (PA95, PA85)
         增益: ×50~×100

推荐:
  PA95 (Apex): ±150V输出
  PA85: ±175V输出, 高带宽
  国产: TPA1260 (120V)
```

**方案B: Boost升压 + 半桥**

```
12V ──→ [Boost升压至150V] ──→ [半桥] ──→ 压电片
                                 │
                            PWM控制输出电压

适用: 成本敏感场景
```

**方案C: 电荷驱动 (高精度)**

```
DAC ──→ [跨导放大器] ──→ 压电片
         (电压→电流→电荷)

优势: 线性度好, 消除压电迟滞
```

### 5.3 驱动代码

```c
// 压电驱动器 (DAC + 高压运放)
typedef struct {
    DAC_HandleTypeDef *hdac;
    uint32_t channel;
    float v_max;           // 最大驱动电压
    float displacement_max; // 最大位移 (μm)
    float resolution;      // DAC分辨率 (V/bit)
    uint16_t dac_max;
} PZT_t;

void PZT_SetPosition(PZT_t *pzt, float displacement_um) {
    // 位移 → 电压 (线性近似)
    float voltage = (displacement_um / pzt->displacement_max) * pzt->v_max;
    voltage = constrain(voltage, 0, pzt->v_max);
    
    // 电压 → DAC值
    uint16_t dac_val = (uint16_t)(voltage / pzt->resolution);
    dac_val = constrain(dac_val, 0, pzt->dac_max);
    
    HAL_DAC_SetValue(pzt->hdac, pzt->channel, DAC_ALIGN_12B_R, dac_val);
    HAL_DAC_Start(pzt->hdac, pzt->channel);
}
```

## 6. 选型决策表

### 6.1 按需求选电机

| 需求 | 推荐电机 | 原因 |
|------|---------|------|
| 调速/力矩 | 直流有刷 | 简单、成本低 |
| 精确定位(角) | 步进电机 | 开环精度高 |
| 精确定位(角)+高效率 | 直流伺服+编码器 | 闭环、效率高 |
| 高速旋转 | 无刷BLDC | 无电刷磨损 |
| 精密微位移(μm) | 音圈电机 | 响应快、线性好 |
| 纳米级定位 | 压电 | 最高精度 |
| 直线运动 | 直线电机/音圈 | 无中间传动 |

## 7. 常见问题速查

| 问题 | 原因 | 解决 |
|------|------|------|
| 电机不转 | 接线错误/驱动IC损坏 | 检查接线、测量电压 |
| 电机只往一个方向 | H桥半桥故障 | 检查MOS管 |
| 步进电机丢步 | 加速过快/电流不足 | 降低加速度/增大电流 |
| 电机发烫 | 电流过大/PWM频率低 | 限流/提高PWM频率 |
| BLDC震动不转 | 换相时序错误 | 检查霍尔接线/反电动势 |
| 电磁干扰大 | PWM开关噪声 | 加TVS/去耦电容/屏蔽线 |
| 音圈振荡 | 电流环带宽不够 | 增大PID带宽 |
