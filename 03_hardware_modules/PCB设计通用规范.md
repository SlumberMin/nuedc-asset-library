## 一、布局规范

### 1.1 分区原则
```
┌─────────────────────────────────┐
│  电源区  │    数字区    │ 通信区 │
│ DC-DC   │    MCU      │ 蓝牙   │
│ LDO     │    ADC      │ WiFi   │
│ 电池管理│    传感器接口│ 串口   │
├─────────┼─────────────┼────────│
│  功率区  │   模拟区    │ 扩展区 │
│ 电机驱动│   运放      │ 排针   │
│ 继电器  │   基准源    │ 测试点 │
└─────────────────────────────────┘
```

### 1.2 元件放置优先级
1. **MCU** → 板中心，引脚朝向信号来源方向
2. **晶振** → 紧贴MCU对应引脚，≤5mm
3. **去耦电容** → 紧贴IC电源引脚，走线≤2mm
4. **电源模块** → 板边，远离敏感信号
5. **电机驱动** → 板边，远离MCU，便于散热
6. **连接器** → 板边，便于接线

### 1.3 间距规范
| 对象 | 最小间距 |
|------|----------|
| 元件焊盘间 | ≥0.3mm |
| 走线间 | ≥0.15mm（常规）/ ≥0.2mm（推荐） |
| 走线到板边 | ≥0.5mm |
| 过孔到焊盘 | ≥0.25mm |
| 高压走线间距 | ≥1mm/kV（按电压等级） |

## 二、布线规范

### 2.1 线宽选择
| 信号类型 | 最小线宽 | 推荐线宽 | 备注 |
|----------|----------|----------|------|
| 信号线 | 0.15mm | 0.2-0.3mm | 一般信号 |
| 电源线（小电流<500mA） | 0.3mm | 0.5mm | MCU供电 |
| 电源线（大电流>1A） | 0.8mm | 1-2mm | 电机供电 |
| 大电流（>3A） | 2mm | 铺铜/多层 | 电池主干线 |

### 2.2 走线规则
1. **避免直角走线** → 用45°或圆弧
2. **避免锐角** → 最小夹角≥90°
3. **信号线等长** → 差分线/编码器线等长误差≤5mm
4. **晶振走线包地** → 晶振信号线两侧走GND保护线
5. **3W原则** → 线中心距≥3倍线宽（减少串扰）
6. **避免平行走线过长** → 跨层走线交叉

### 2.3 过孔使用
| 参数 | 值 |
|------|-----|
| 过孔直径 | 0.3-0.5mm |
| 过孔焊盘 | 0.6-0.8mm |
| 信号过孔 | 可用0.3mm/0.6mm |
| 电源过孔 | 推荐0.5mm/1mm |
| 过孔间距 | ≥0.3mm |

## 三、接地规范

### 3.1 单点接地 vs 大面积铺铜
- **双层板**：底层大面积铺铜做GND平面，配合多个GND过孔
- **四层板**：第二层完整GND平面（不要分割）
- **模拟地与数字地**：单点连接（星型接地），连接点在ADC附近

### 3.2 接地要点
```
                ┌─────────┐
数字GND ────────┤  单点连接 ├────── 模拟GND
(铺铜)          │ (磁珠/0Ω)│       (铺铜)
                └─────────┘
                      │
                   大面积铺铜
                   (最终汇聚到电源GND)
```

### 3.3 常见接地错误
- ❌ 模拟地和数字地大面积混铺
- ❌ 信号回流路径过长（缺少GND过孔）
- ❌ 大电流GND路径过窄
- ✅ 频繁打GND过孔（每5-10mm一个）

## 四、电源设计规范

### 4.1 去耦电容放置
```
IC VDD引脚 ──┬── 100nF (0402/0603, 紧贴引脚)
             ├── 1μF (0805, 较近)
             └── 10μF (电解, 板级储能)

走线路径: VDD → 电容 → IC → 电容 → GND
         (电容在电源进入IC之前滤波)
```

### 4.2 电源走线
- 电源入口加π型滤波（电感+两个电容）
- 大电流走线与信号走线分开层
- 电源走线避免与敏感信号平行

## 五、抗干扰设计

### 5.1 EMI抑制
| 措施 | 适用场景 |
|------|----------|
| 串联电阻（33-100Ω） | 高速信号输出端 |
| RC滤波 | 编码器/按键等输入信号 |
| 铁氧体磁珠 | 电源隔离（数字/模拟） |
| TVS二极管 | 外部接口ESD保护 |
| 差分走线 | 长距离信号传输 |
| 屏蔽线 | 模拟微弱信号 |

### 5.2 电机干扰抑制
```
电机驱动板与主控板分离（推荐）
或在同一板上物理隔离（≥2cm间距）

电机电源入口:
BAT+ ── 自恢复保险丝 ── 大电容(470μF+) ── 驱动IC

电机两端:
电机A ── TVS(SMBJ16A) ── 电机B
(吸收反向EMF)
```

### 5.3 信号完整性
- 关键信号（ADC、编码器）走线尽量短
- 高速信号（SPI、SWD）加串联端接电阻
- 敏感信号远离干扰源（电机、DC-DC电感）
- 信号层相邻层必须是完整GND平面

## 六、PCB打样规范

### 6.1 Gerber文件输出
| 文件 | 内容 |
|------|------|
| .GTL | 顶层铜 |
| .GBL | 底层铜 |
| .GTS | 顶层阻焊 |
| .GBS | 底层阻焊 |
| .GTO | 顶层丝印 |
| .GBO | 底层丝印 |
| .GKO | 板框 |
| .TXT | 钻孔文件 |

### 6.2 打样参数推荐
| 参数 | 推荐值 |
|------|--------|
| 板厚 | 1.6mm |
| 铜厚 | 1oz（一般）/ 2oz（大电流） |
| 最小线宽/间距 | 6mil/6mil (0.15mm) |
| 最小过孔 | 0.3mm钻孔/0.6mm焊盘 |
| 表面处理 | HASL（有铅）或 ENIG（无铅） |
| 阻焊颜色 | 绿色（最便宜） |

### 6.3 丝印规范
- 每个元件标注位号（R1, C1, U1...）
- 接口标注功能（VCC, GND, TX, RX, SWD）
- 板名+版本号+日期（右下角）
- 极性标注（电解电容、二极管、LED）
- 引脚1标识（IC的圆点/缺口）

## 七、设计检查清单

- [ ] 所有去耦电容已放置且贴近引脚
- [ ] 晶振走线≤5mm，下方无铺铜
- [ ] 电源走线宽度满足电流需求
- [ ] 模拟地与数字地正确隔离
- [ ] 未用引脚已处理（上下拉/输出低）
- [ ] 丝印无遗漏，标注清晰
- [ ] DRC检查无错误
- [ ] 板框尺寸正确，安装孔已标注
- [ ] 连接器方向正确（防反插标注）
- [ ] 测试点已预留（关键信号）

## 八、电赛PCB实战经验补充

### 8.1 抗干扰关键措施

| 干扰源 | 干扰路径 | 解决措施 |
|--------|---------|---------|
| DC-DC开关电源 | 电源纹波 | π型LC滤波（10μH+2×100μF） |
| 电机驱动 | 电源线/空间辐射 | 电机独立供电，加TVS（SMBJ16A） |
| 高速数字信号 | 信号走线串扰 | 3W原则，相邻层GND平面 |
| 晶振 | 辐射干扰 | 走线包地，下方禁止铺铜 |
| 继电器/电磁铁 | 反向EMF | 续流二极管+RC吸收 |

### 8.2 电赛常见PCB问题及修复

| 问题 | 表现 | 快速修复方案 |
|------|------|-------------|
| ADC跳动大 | 采样值噪声大 | VDDA加磁珠+独立LDO |
| 运放自激 | 输出高频振荡 | 电源去耦+输出加小电阻（10Ω） |
| 信号串扰 | 相邻信号互相干扰 | 加地线隔离，减少平行走线 |
| 散热不良 | 驱动芯片过热 | 加散热铜箔+过孔阵列 |
| 铺铜孤岛 | 未连接的铜皮 | 删除孤岛或加过孔连接 |

### 8.3 快速打样检查要点

```
提交打样前确认清单：
  □ 电源走线宽度：信号线≥0.2mm，电源线≥0.5mm，大电流≥1mm
  □ 过孔大小：信号过孔0.3mm/0.6mm，电源过孔0.5mm/1mm
  □ 丝印清晰：极性标注、引脚1标识、板名+版本
  □ 板框尺寸：符合机械安装要求
  □ 元件封装：0402/0603/0805尺寸是否可焊接（手工焊接建议≥0603）
  □ 焊盘设计：贴片元件焊盘长度≥1mm（方便手工焊接）
```

### 8.4 四层板推荐叠层结构

```
四层板叠层（推荐）：
  Layer 1 (TOP)  → 信号层+元件面
  Layer 2 (GND)  → 完整地平面（关键！）
  Layer 3 (POWER) → 电源平面（3.3V/5V分割）
  Layer 4 (BOT)  → 信号层+底面元件

优点：
  - 第二层完整GND平面提供优秀参考平面
  - 电源平面与GND平面耦合，降低EMI
  - 信号层相邻有完整参考平面，信号完整性好
```

### 8.5 手工焊接技巧

```
贴片焊接要点：
  1. 先焊小元件（0402/0603），再焊大元件
  2. IC芯片：先固定一个引脚，再焊对角，最后补齐
  3. QFN/BGA：热风枪+助焊膏，温度300-350°C
  4. 焊接后用万用表检查：短路检测（相邻引脚）、虚焊检测

常见焊接失误：
  ❌ 焊锡过多 → 短路
  ❌ 温度过高 → 芯片损坏
  ❌ 时间过长 → 焊盘脱落
  ❌ 方向焊反 → IC、二极管、电解电容
```
