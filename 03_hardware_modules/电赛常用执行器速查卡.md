## 二、直流电机

### 2.1 有刷直流电机

```
参数速查：
├── 额定电压：3V / 6V / 12V / 24V
├── 空载转速：3000-15000 RPM
├── 堵转扭矩：0.01-1 N·m（小电机）
├── 额定电流：0.1-5A
└── 减速比：1:1 - 1:300+

选型口诀：
  要速度 → 低减速比 + 高转速电机
  要扭矩 → 高减速比 + 大电流电机
  要精度 → 加编码器闭环

驱动方案：
  L298N  → 简单双路，效率低，适合入门
  TB6612 → 效率高，推荐首选
  BTS7960 → 大电流（43A），适合大电机
  DRV8871 → 单路小封装，适合空间紧张
```

### 2.2 常用直流电机型号

| 型号 | 电压 | 减速比 | 转速 | 扭矩 | 适用 |
|------|------|--------|------|------|------|
| JGA25-370 | 6-12V | 1:34-1:214 | 60-360RPM | 0.3-1.5kg·cm | 小车驱动 |
| JGB37-520 | 6-24V | 1:17-1:1800 | 10-600RPM | 1-50kg·cm | 中型载重 |
| RS-555 | 12V | 无减速 | 5000-12000RPM | - | 风扇/高速 |
| N20 | 3-6V | 1:10-1:150 | 30-500RPM | 0.1-1kg·cm | 微型精密 |

### 2.3 直流电机驱动电路

```
H桥驱动原理：
        VCC
    ┌───┤├───┐
    │  Q1  Q3 │
  IN+──┤├──┤├──IN-
    │  Q2  Q4 │
    └───┤├───┘
        GND

Q1+Q4导通 → 正转
Q2+Q3导通 → 反转
PWM控制Q1/Q2 → 调速

代码：
  AIN1=1, AIN2=0 → 正转
  AIN1=0, AIN2=1 → 反转
  AIN1=0, AIN2=0 → 制动
  PWM占空比 → 速度
```

## 三、步进电机

### 3.1 步进电机类型

| 类型 | 步距角 | 扭矩 | 细分 | 特点 |
|------|--------|------|------|------|
| 永磁式(PM) | 7.5°-15° | 小 | 有限 | 便宜，精度低 |
| 反应式(VR) | 1.8°-3.6° | 中 | 有限 | 噪声大 |
| 混合式(HB) | 1.8°(标准) | 大 | 高(≤256) | 最常用 |
| 28BYJ-48 | 5.625°/64 | 小 | - | 超便宜，入门用 |
| 42步进(NEMA17) | 1.8° | 0.2-0.6N·m | 高 | 最常用尺寸 |

### 3.2 步进电机参数

```
关键参数：
├── 步距角：1.8°(200步/圈) 最常见
├── 相数：2相(4线) 最常见，也有3相/5相
├── 额定电流：0.5-3A/相
├── 保持扭矩：0.1-5 N·m
├── 转动惯量：影响加减速性能
└── 细分数：1, 2, 4, 8, 16, 32, 64, 128, 256

28BYJ-48：
  5.625°/64减速比 = 每步0.087890625°
  4096步/圈（半步驱动）
  便宜（¥3-5），适合教学和轻载

42步进(NEMA17)：
  1.8°/步 = 200步/圈
  16细分 = 3200步/圈
  扭矩0.2-0.6N·m
```

### 3.3 步进驱动方案

| 驱动芯片 | 电流 | 细分 | 特点 |
|----------|------|------|------|
| ULN2003 | 500mA | - | 最简单，28BYJ-48专用 |
| A4988 | 2A | 1/16 | 最常用，性价比高 |
| DRV8825 | 2.5A | 1/32 | 比A4988细分更高 |
| TMC2209 | 2A | 1/256 | 静音，UART配置 |
| TMC2226 | 2A | 1/256 | TMC2209升级版 |
| DM542 | 4.5A | 1/256 | 大电流，工业级 |

### 3.4 A4988接线与代码

```
接线：
  VDD → 3.3V/5V (逻辑电源)
  VMOT → 8-35V (电机电源)
  GND → 共地
  STEP → MCU PWM/脉冲引脚
  DIR → MCU GPIO
  ENABLE → MCU GPIO (低电平使能)
  MS1/MS2/MS3 → 细分设置

细分设置（MS1/MS2/MS3）：
  L/L/L = Full step
  H/L/L = 1/2
  L/H/L = 1/4
  H/H/L = 1/8
  H/H/H = 1/16
```

```c
// STM32步进电机控制
void stepper_move(int steps, int speed_hz) {
    HAL_GPIO_WritePin(DIR_PORT, DIR_PIN, steps > 0 ? GPIO_PIN_SET : GPIO_PIN_RESET);
    int abs_steps = abs(steps);

    for(int i = 0; i < abs_steps; i++) {
        HAL_GPIO_WritePin(STEP_PORT, STEP_PIN, GPIO_PIN_SET);
        delay_us(2);
        HAL_GPIO_WritePin(STEP_PORT, STEP_PIN, GPIO_PIN_RESET);
        delay_us(1000000 / speed_hz - 2);
    }
}

// 使用定时器产生脉冲（推荐）
void stepper_move_timer(int steps, int speed_hz) {
    __HAL_TIM_SET_AUTORELOAD(&htim, SystemCoreClock / speed_hz - 1);
    HAL_TIM_PWM_Start(&htim, TIM_CHANNEL_1);
    // 计数到steps后停止
}
```

### 4.1 舵机类型对比

| 型号 | 扭矩 | 速度 | 重量 | 价格 | 适用 |
|------|------|------|------|------|------|
| SG90 | 1.8kg·cm | 0.1s/60° | 9g | ¥5 | 轻载入门 |

| 总线舵机 | 可变 | 快 | 可变 | ¥30-100 | 机械臂组 |

### 4.2 舵机PWM控制

```
标准PWM信号：
  周期：20ms (50Hz)
  脉宽：0.5ms - 2.5ms

  0.5ms  → -90° (或0°)
  1.0ms  → -45° (或45°)
  1.5ms  → 0°   (或90°)
  2.0ms  → 45°  (或135°)
  2.5ms  → 90°  (或180°)

  PWM占空比 = 脉宽/周期 × 100%
  1.5ms/20ms = 7.5% → 中位
```

### 4.3 舵机控制代码

```c
// STM32 舵机控制
// 假设定时器时钟72MHz，预分频72-1，周期20000-1 → 50Hz

void servo_set_angle(TIM_HandleTypeDef *htim, uint32_t channel, float angle) {
    // angle: 0-180度
    // 对应pulse: 500-2500us → 500-2500 (TIM计数值)
    uint32_t pulse = (uint32_t)(500 + angle / 180.0f * 2000);
    __HAL_TIM_SET_COMPARE(htim, channel, pulse);
}

// 平滑移动
void servo_smooth_move(float from_angle, float to_angle, float duration_ms) {
    float step = (to_angle - from_angle) / (duration_ms / 10);
    float current = from_angle;
    for(int i = 0; i < (int)(duration_ms/10); i++) {
        servo_set_angle(&htim, TIM_CHANNEL_1, current);
        current += step;
        HAL_Delay(10);
    }
}
```

总线舵机优势：

## 五、音圈电机

### 5.1 音圈电机特性

```
原理：通电线圈在磁场中受力（类似扬声器）
特点：
  ├── 直接驱动，无传动间隙
  ├── 响应极快（<1ms）
  ├── 行程短（±5mm）
  ├── 力与电流成正比（线性好）
  └── 无齿槽效应，运动平滑

关键参数：
  ├── 行程：±2-10mm
  ├── 峰值力：1-100N
  ├── 持续力：0.5-30N
  ├── 峰值电流：0.5-5A
  ├── 电阻：1-20Ω
  └── 力常数 Kf：N/A（典型0.5-10 N/A）
```

### 5.2 音圈电机选型

| 型号 | 行程 | 峰值力 | 应用 |
|------|------|--------|------|
| LV020 | ±1mm | 2N | 光学对焦 |
| LV070 | ±3mm | 7N | 精密定位 |
| LA10-12 | ±5mm | 10N | 振动台 |
| NCC05-18-010 | ±2.5mm | 10N | 硬盘磁头 |

### 5.3 音圈驱动

```
驱动要求：
  - 电流驱动（不是电压驱动！）
  - 双向电流（H桥）
  - 高带宽电流环（>1kHz）

方案：
  1. 高精度DAC + 运放恒流源
  2. PWM + H桥 + 电流采样闭环
  3. 专用音圈驱动芯片

代码（电流环控制）：
```

```c
// 音圈电机电流闭环
float vcm_current_control(float target_current_A) {
    float R_sense = 0.1f;  // 采样电阻
    float voltage = read_adc() * 3.3f / 4096;
    float actual_current = voltage / R_sense;

    float error = target_current_A - actual_current;
    integral += error * dt;
    float duty = Kp * error + Ki * integral;
    // H桥PWM输出
    set_pwm(duty);
    return actual_current;
}
```

## 六、压电执行器

### 6.1 压电效应

```
正压电效应：力 → 电荷（传感器）
逆压电效应：电场 → 形变（执行器）

特点：
  ├── 纳米级分辨率
  ├── 响应极快（μs级）
  ├── 行程极小（μm级）
  ├── 大力输出（kN级，堆叠型）
  ├── 无电磁干扰
  └── 迟滞（5-15%）
```

### 6.2 压电执行器类型

| 类型 | 行程 | 力 | 驱动电压 | 应用 |
|------|------|-----|---------|------|
| 片式 | 1-10μm | 10-100N | 100-200V | 精密调整 |
| 堆叠式 | 10-100μm | 100-10000N | 100-200V | 大力微位移 |
| 弯曲式 | 100μm-1mm | 0.1-1N | 10-100V | 微夹持器 |
| 压电叠堆 | 50-200μm | 1000N+ | 150V | 精密平台 |

### 6.3 压电驱动电路

```
驱动要求：
  - 高压驱动（通常100-200V）
  - 高阻容性负载
  - 需要充放电电路
  - 迟滞补偿（Preisach模型）

方案：
  1. 高压运放（PA85/PA98）
  2. Boost升压 + H桥
  3. 电荷放大器（线性度更好）
```

```c
// 压电驱动（简化的PWM升压方案）
void piezo_set_voltage(float target_voltage) {
    // 目标电压 0-150V
    // 通过DAC控制升压电路
    float duty = target_voltage / MAX_VOLTAGE;
    uint16_t dac_value = (uint16_t)(duty * 4095);
    HAL_DAC_SetValue(&hdac, DAC_CHANNEL_1, DAC_ALIGN_12B_R, dac_value);

    // 迟滞补偿
    float hysteresis_comp = Preisach_Compensate(target_voltage);
    // 叠加补偿信号
    apply_compensation(hysteresis_comp);
}
```

## 七、选型决策矩阵

```
需要什么特性？
│
├── 需要连续旋转？
│   ├── 是 → 直流电机 / 步进电机 / 无刷电机
│   └── 否 → 舵机 / 压电 / 音圈
│
├── 定位精度要求？
│   ├── < 1μm → 压电执行器
│   ├── 1μm-1mm → 音圈电机 / 步进细分
│   ├── 1mm-1cm → 步进电机 / 舵机
│   └── > 1cm → 步进电机 / 直流+编码器
│
├── 速度要求？
│   ├── 极快响应(ms级) → 音圈 / 压电
│   ├── 中等 → 直流电机 / 舵机
│   └── 低速大扭矩 → 步进+减速
│
├── 力/扭矩要求？
│   ├── > 10N·m → 直流减速电机 / 步进+减速
│   ├── 1-10N·m → 步进电机 / 大舵机
│   └── < 1N·m → 小舵机 / 步进
│
└── 成本约束？
    ├── < ¥20 → SG90舵机 / 28BYJ-48步进
    ├── ¥20-100 → NEMA17步进 / MG996R舵机
    └── > ¥100 → 音圈 / 压电 / 总线舵机
```
