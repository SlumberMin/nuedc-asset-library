## 1. 天线基础参数速查

| 参数 | 符号 | 含义 |
|------|------|------|
| 增益 | G (dBi) | 相对各向同性辐射器的增益 |
| 方向性 | D | 最大辐射方向与平均辐射之比 |
| 效率 | η | 辐射功率/输入功率 |
| 带宽 | BW | VSWR≤2的频率范围 |
| 驻波比 | VSWR | ≤2为良好匹配 |
| 输入阻抗 | Zin | 通常匹配到50Ω |
| 极化 | — | 线极化/圆极化 |

### 关键公式

```
波长 λ = c / f = 300 / f(MHz)  [单位：米]
有效长度 le = λ/π (短偶极子)
增益 G(dBd) = G(dBi) - 2.15
```

## 2. 偶极子天线（Dipole）

### 结构

```
        ← λ/4 →
        ┌──────┐
        │      │
────────┘      └────────
馈点（50Ω）
        ← λ/4 →
总长 = λ/2
```

### 参数

| 项目 | 数值 |
|------|------|
| 总长度 | L = λ/2 = 142.5/f(MHz) [cm] |
| 输入阻抗 | 73 + j42.5 Ω（自由空间） |
| 增益 | 2.15 dBi |
| 方向图 | 全向（垂直于天线轴） |
| VSWR带宽 | 约10%（VSWR≤2） |
| 极化 | 与天线臂平行的线极化 |

### 常用频率对应尺寸

| 频率 | 波长λ | 半波长度λ/2 |
|------|-------|------------|
| 433 MHz | 69.3 cm | 34.6 cm |
| 868 MHz | 34.6 cm | 17.3 cm |
| 915 MHz | 32.8 cm | 16.4 cm |
| 2.4 GHz | 12.5 cm | 6.25 cm |
| 5.8 GHz | 5.17 cm | 2.59 cm |

### 折叠偶极子

```
阻抗提升为 ~300Ω，便于匹配
   ┌──────────────┐
   │              │
───┘              └───
馈点
```

## 3. 单极子天线（Monopole）

### 结构

```
        ← λ/4 →
        ┌──────┐
        │ 辐射体 │
────────┴──────┴────────
       接地平面（GND）
馈点在辐射体底部
```

### 参数

| 项目 | 数值 |
|------|------|
| 长度 | L = λ/4 = 71.25/f(MHz) [cm] |
| 输入阻抗 | 36.5 Ω（理想无限大地平面） |
| 增益 | 5.15 dBi（含地面反射） |
| 方向图 | 全向（水平面） |
| 极化 | 垂直线极化 |

### 实际应用要点

- 地平面质量直接影响性能
- PCB天线常采用此结构（如ESP32的板载天线）
- 需要λ/4的径向地平面效果最佳

### 弹簧天线/螺旋天线（缩短型单极子）

```
缩短方式：螺旋缠绕
长度可缩短至 λ/8 ~ λ/10
但带宽变窄、效率降低
常用于小型模块（如433MHz模块）
```

## 4. PIFA天线（平面倒F天线）

### 结构（俯视图）

```
    ┌────────────────────┐
    │     辐射贴片        │
    │                    │
    ├─────┬──────────────┤
    │ GND │  短路片       │
    │     │  ┌──┐        │
    └─────┘  │  │        │
             └──┘        │
    馈点→●               │
    └────────────────────┘
    接地平面
```

### 参数

| 项目 | 数值 |
|------|------|
| 尺寸 | 约 λ/4 × λ/10 |
| 高度 | 6~10mm |
| 增益 | 1~3 dBi |
| 带宽 | 5~10%（VSWR≤2） |
| 阻抗 | 50Ω（通过短路片位置调节） |
| 极化 | 线极化 |

### 优缺点

| 优点 | 缺点 |
|------|------|
| 体积小，适合手持设备 | 带宽较窄 |
| 低剖面，可内置于外壳内 | 效率受地平面影响大 |
| 抗人体干扰较好 | 设计复杂度高 |
| 可共形于曲面 | 频率调谐敏感 |

### 典型应用

- 手机内置天线（2G/3G/4G）
- 蓝牙/WiFi模块天线
- GPS天线（配合圆极化技术）

## 5. 贴片天线（Microstrip Patch）

### 结构（侧视图）

```
    ┌───────────────────────┐
    │      辐射贴片          │  ← W × L
    ├───────────────────────┤
    │    介质基板 (εr, h)     │
    ├───────────────────────┤
    │      接地平面           │
    └───────────────────────┘
    
馈电方式：微带线馈电 / 同轴探针馈电 / 耦合馈电
```

### 设计公式

```
工作频率：f = c / (2·L_eff · √ε_eff)

贴片宽度：
W = c/(2f) · √(2/(εr+1))

有效长度（考虑边缘效应）：
L_eff = L + 2·ΔL
ΔL = 0.412·h·(ε_eff+0.3)·(W/h+0.264) / ((ε_eff-0.258)·(W/h+0.8))

有效介电常数：
ε_eff = (εr+1)/2 + (εr-1)/2 · (1+12h/W)^(-0.5)
```

### 常见基板材料

| 材料 | 介电常数εr | 损耗角正切 | 适用场景 |
|------|-----------|-----------|---------|
| FR4 | 4.4 | 0.02 | 低成本、≤3GHz |
| Rogers RO4003C | 3.38 | 0.0027 | 高频、低损耗 |
| Rogers RO4350B | 3.66 | 0.0037 | 中频应用 |
| 氧化铝陶瓷 | 9.8 | 0.0001 | 微波、毫米波 |
| 聚四氟乙烯 | 2.1 | 0.0002 | 超低损耗 |

### 参数范围

| 项目 | 典型值 |
|------|-------|
| 增益 | 5~8 dBi |
| 效率 | 70~95% |
| 带宽 | 1~5%（VSWR≤2） |
| 极化 | 线极化/圆极化 |
| 阻抗 | 50Ω（边馈）/ 100~200Ω（侧馈） |

### 圆极化实现

```
单馈点：切角法（切去对角）
   ┌──────╲
   │        ╲
   │          ╲
   ╱            │
  ╱             │
 ╱──────────────┘

双馈点：90°功分器馈电
```

## 6. 螺旋天线（Helical）

### 结构

```
      ┌─┐ ┌─┐ ┌─┐
      │ │ │ │ │ │
      │ └─┘ └─┘ │
      │  ┌──┐   │    螺旋线
      │  │  │   │
      └──┘  └───┘
    ┌──────────────┐
    │   接地平面     │
    │   ●馈点       │
    └──────────────┘
    
参数：圈数N, 直径D, 螺距S, 导线直径d
```

### 参数公式

```
轴向模工作条件：
2πD/λ ≈ 1（周长≈波长）

增益（轴向模）：
G ≈ 15·N·S·C/λ³ （C=πD）
简化：G(dBi) ≈ 10·log₁₀(15·N·C²·S/λ³)

输入阻抗：约 120~140 Ω

带宽：10~20%（较宽）
```

### 工作模式

| 模式 | 条件 | 特点 |
|------|------|------|
| 法向模 | D << λ | 辐射方向垂直于轴，类似单极子 |
| 轴向模 | C ≈ λ | 辐射方向沿轴，圆极化，高增益 |
| 锥形模 | C > λ | 多波束 |

### 电赛应用场景

| 场景 | 推荐模式 |
|------|---------|
| 遥控器天线 | 法向模（小尺寸） |
| 无线视频传输 | 轴向模（高增益定向） |
| 无人机通信 | 轴向模（圆极化抗干扰） |

## 7. 对数周期天线（Log-Periodic）

### 结构

```
短 ←─────────────────── → 长

  │  │  │   │    │     │
  │  │  │   │    │     │
  ├──┤  │   │    │     │
  │  ├──┘   │    │     │
  │  │  ├───┘    │     │
  │  │  │   ├────┘     │
  │  │  │   │    ├─────┘
  │  │  │   │    │     │
  
  馈点在短端
  τ = L_n+1 / L_n（比例因子，通常0.8~0.95）
  σ = d_n / (2·L_n)（间距因子，通常0.06~0.2）
```

### 参数

| 项目 | 数值 |
|------|------|
| 频率范围 | f_low ~ f_high（可达10:1带宽） |
| 增益 | 7~12 dBi |
| 阻抗 | 50~100 Ω |
| 极化 | 线极化 |
| 方向图 | 端射（从短端指向长端方向） |

### 设计公式

```
比例因子：τ = f_n / f_{n+1} = L_{n+1} / L_n

偶极子长度：L_n = c / (2·f_n)

间距：d_n = (1-τ)/(4σ) · L_n

带宽：B = f_N / f_1 = (1.1 + 7.7·(1-τ)²/(4τ)) · 1/σ

偶极子数量：N = 1 + log(B)/log(1/τ)
```

### 优缺点

| 优点 | 缺点 |
|------|------|
| 超宽带（10:1） | 物理尺寸大 |
| 增益适中 | 设计复杂 |
| 频率无关特性 | 高频端效率下降 |
| 可用于频谱分析 | 成本较高 |

### 电赛应用

- 宽带信号接收（频谱监测题）
- EMC测试天线
- 多频段通信系统

## 9. 天线匹配与馈电

### 9.1 匹配网络

```
L型匹配网络（最简单）

方案1（低通型）：
    ┌──L──┬──┬─
    │     C  │
    │     │  ZL
    │     GND│
    └────────┘

方案2（高通型）：
    ┌──C──┬──┬─
    │     L  │
    │     │  ZL
    │     GND│
    └────────┘

Smith圆图法：在阻抗点和50Ω圆之间添加LC元件
```

### 9.2 巴伦（Balun）

```
平衡-不平衡转换

偶极子（平衡） ←→ 同轴线（不平衡）

无巴伦时，同轴外皮会有电流，影响方向图

常用实现：
- 1/4波长扼流巴伦
- 传输线变压器巴伦
- 微带线巴伦
```

### 9.3 PCB天线设计注意

| 要点 | 说明 |
|------|------|
| 地平面 | 足够大，至少λ/4 |
| 馈线 | 50Ω微带线，控制阻抗 |
| 隔离 | 天线区域远离数字电路 |
| 介质 | 高频用低损耗基板 |
| 调试 | 预留匹配网络元件位 |

## 10. 天线测试方法

### 10.1 驻波比测量

```
使用矢量网络分析仪（VNA）
S11参数 → VSWR = (1+|S11|)/(1-|S11|)

VSWR ≤ 1.5：优秀
VSWR ≤ 2.0：良好
VSWR ≤ 3.0：可用
VSWR > 3.0：需要重新匹配
```

### 10.2 简易测试（无VNA）

```
1. 信号源 + 频谱仪：远场测试
2. 已知增益天线对比法
3. 场强仪测量
```
