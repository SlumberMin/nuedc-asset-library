# 电赛备赛清单

> 完整备赛清单，覆盖硬件/软件/工具/文档/团队分工

---

## 一、团队分工

### 1.1 三人团队标准配置

```
角色1：硬件工程师（主攻电路设计与焊接）
  ├── 负责电路原理图设计
  ├── 负责PCB绘制或洞洞板搭建
  ├── 负责元器件采购
  ├── 负责焊接与调试
  └── 负责测试仪器操作

角色2：软件工程师（主攻程序开发）
  ├── 负责MCU程序开发
  ├── 负责驱动程序编写
  ├── 负责控制算法实现
  ├── 负责上位机开发（如有需要）
  └── 负责软件调试与优化

角色3：系统工程师（主攻方案设计与报告）
  ├── 负责方案论证
  ├── 负责理论计算
  ├── 负责系统联调
  ├── 负责测试数据记录
  └── 负责设计报告撰写
```

### 1.2 角色能力要求

```
硬件工程师：
  □ 熟练使用Altium Designer/KiCad
  □ 熟练焊接0402/QFP/BGA封装
  □ 熟练使用示波器/万用表/逻辑分析仪
  □ 了解常用电路拓扑（Buck/Boost/运放电路）
  □ 了解PCB布局布线基本原则

软件工程师：
  □ 精通C语言编程
  □ 熟练使用STM32/ESP32开发
  □ 了解常用通信协议（UART/SPI/I2C/CAN）
  □ 了解PID/滤波等常用算法
  □ 熟练使用Keil/IAR/VS Code开发环境

系统工程师：
  □ 较强的文档写作能力
  □ 了解各类电路基本原理
  □ 了解常用测试方法
  □ 具备良好的沟通协调能力
  □ 熟练使用Word/Visio/PPT
```

### 1.3 备赛阶段分工

```
备赛期（赛前1-3个月）：
  硬件：练习焊接、熟悉常用模块
  软件：练习驱动开发、积累算法库
  系统：研究历年题目、整理资料

比赛期（4天3夜）：
  第1天：三人共同审题、方案论证、分工
  第2天：硬件搭建+软件开发并行
  第3天：系统联调+参数优化
  第4天：测试+报告+封箱
```
