# 电赛电机选型指南

## 一、电机类型对比总览

| 特性 | 直流减速电机 | 步进电机 | 舵机 | 无刷电机(BLDC) |
|------|-------------|---------|------|----------------|
| 速度控制 | PWM调速 | 脉冲频率 | 固定角度 | PWM/FOC |
| 位置控制 | 需编码器 | 开环精确 | 内置闭环 | 需编码器 |
| 扭矩 | 大(减速比) | 中等 | 中小 | 大 |
| 精度 | 低(开环) | 高(步距角) | 高(±1°) | 高(闭环) |
| 响应速度 | 快 | 较慢 | 中等 | 快 |
| 噪声 | 中 | 大(低速振动) | 小 | 小 |
| 价格 | 低 | 中 | 低~中 | 高 |
| 驱动复杂度 | 简单(H桥) | 中等(专用IC) | 简单(PWM) | 复杂(ESC/FOC) |
| 适用场景 | 小车、风扇 | 3D打印、转台 | 转向、云台 | 高性能驱动 |

### 28BYJ-48接线（电赛常用）

```
ULN2003驱动板：
  IN1 → PB0    IN2 → PB1
  IN3 → PB2    IN4 → PB3
  
  5V → 电源正   GND → 电源负
```

### 步进电机关键参数计算

```
每圈步数 = 360° / 步距角 = 360° / 1.8° = 200步
细分后每圈步数 = 200 × 细分数
  1/16细分 → 3200步/圈

转速(rpm) = 脉冲频率(Hz) × 60 / (200 × 细分数)
  例: 32kHz, 1/16 → 32000×60/3200 = 600rpm
```

### 舵机控制信号

```
PWM参数：
  周期: 20ms (50Hz)
  脉宽: 0.5ms ~ 2.5ms
    0.5ms → 0°
    1.5ms → 90° (居中)
    2.5ms → 180°

STM32定时器配置(72MHz主频)：
  预分频: 72-1 → 1MHz计数
  周期: 20000-1 → 50Hz
  比较值: 500~2500 → 0°~180°
```

## 六、电机调试常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 电机不转 | 驱动电压不足 | 检查VM引脚电压，用独立电源供电 |
| 电机抖动 | PWM频率太低 | 提高PWM频率至10kHz以上 |
| 步进电机丢步 | 加速度过大 | 降低加速度，使用S曲线 |
| 舵机抖动/啸叫 | 电源纹波大 | 舵机独立供电，加100μF电容 |
| 电机反转 | 接线反了 | 交换电机两根线或修改驱动逻辑 |
| 编码器计数不准 | 信号抖动 | 硬件加RC滤波，软件四倍频 |
| 电机干扰单片机 | 电源未隔离 | 电机驱动板独立供电，共地处理 |
