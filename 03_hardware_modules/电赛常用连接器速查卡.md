## 2. 排针/排母（最常用）

### 规格

```
直插排针 (Pin Header)
  间距：2.54mm（标准）、2.0mm（小间距）、1.27mm（精细）
  排数：单排 / 双排
  针数：1~40 pin 常见
  额定：3A/250V（2.54mm）

排母 (Socket Header)
  与排针配合，板对板连接

弯针 (Right-Angle Header)
  用于 PCB 边缘水平引出
```

### 电赛使用场景

```
场景               推荐规格            注意事项
─────────────────────────────────────────────────
STM32最小系统板     2.54mm 2×20排针    注意引脚定义
传感器模块连接      2.54mm 单排 4~6pin  VCC/GND/SCL/SDA
电机驱动板连接      2.54mm 2×3排针     注意电流能力
调试接口(SWD)      2.54mm 2×5 Cortex  标准SWD接口
杜邦线连接         排针+杜邦线         接触电阻较大
```

### 选型建议

```
推荐型号：
  单排直插：Ckmtw 2.54mm 1×40 排针
  双排直插：Ckmtw 2.54mm 2×20 排针
  排母：Ckmtw 2.54mm 2×20 排母
  
淘宝关键词：2.54排针 排母 弯针 直针

焊接注意：
  - 先焊接排针到 PCB
  - 再插入排母到排针上
  - 将排母焊接到另一块 PCB
  - 保持垂直，避免虚焊
```

## 3. JST 连接器

### 常见型号

| 型号 | 间距 | 额定电流 | 典型用途 |
|------|------|---------|---------|
| JST-PH | 2.0mm | 2A | 电池、小电机 |
| JST-XH | 2.5mm | 3A | 锂电池包、电机 |
| JST-SH | 1.0mm | 1A | 小信号、传感器 |
| JST-ZH | 1.5mm | 1A | 线对板 |
| JST-GH | 1.25mm | 2A | Pixhawk、无人机 |

### 电赛使用场景

```
JST-XH (最常用)：
  - 锂电池连接（2S/3S/4S）
  - 大电流电机连接
  - 电源分配板
  
JST-PH：
  - 小型锂电池（单节18650）
  - 舵机连接（三线 PH2.0）
  - 传感器模块

JST-GH：
  - 无人机模块连接
  - Pixhawk 飞控接口
  - 高密度信号连接
```

### 压接工具

```
必备工具：
  - JST压线钳（棘轮式，推荐 Engineer PA-09）
  - 剥线钳
  - 热缩管

压接步骤：
  1. 剥线 3~4mm
  2. 线芯插入端子压接区
  3. 用压线钳压接（先压内爪，再压外爪）
  4. 插入塑料外壳，听到"咔嗒"声
```

## 4. FPC/FFC 连接器

### 规格说明

```
FPC (Flexible Printed Circuit)：柔性电路板，有补强板
FFC (Flexible Flat Cable)：柔性扁平电缆，无补强

关键参数：
  间距：0.5mm / 1.0mm（电赛最常用）
  pin数：4~60 pin
  类型：同向 / 反向（翻转/不翻转）
  锁紧方式：翻盖式 / 前插式 / 后插式
```

### 电赛常见用途

```
用途              接口类型         pin数    注意
────────────────────────────────────────────────
OLED显示屏        FPC 0.5mm       10~30   注意引脚定义
LCD液晶屏         FPC 1.0mm       30~50   RGB接口需并行
摄像头模块        FPC 0.5mm       15~30   高速信号注意走线
触摸屏            FPC 0.5mm       4~6     I2C接口
打印机机芯        FPC 1.0mm       12~26   注意平整度
```

### 使用注意事项

```
⚠️ 常见错误：
  1. FPC插入方向搞反 → 烧芯片
  2. 未锁紧翻盖 → 接触不良
  3. FPC弯折半径太小 → 断裂
  4. 间距选错（0.5mm vs 1.0mm）→ 插不进去

✅ 正确操作：
  1. 打开锁紧翻盖（轻轻抬起）
  2. FPC金手指朝下插入（或按手册方向）
  3. 平整推入到底
  4. 按下锁紧翻盖
  5. 轻拉确认锁紧
```

## 5. 航空插头

### 常见规格

| 型号 | pin数 | 额定电流 | 防护等级 | 用途 |
|------|------|---------|---------|------|
| GX12 | 2~7 pin | 5~10A | IP40 | 电源/信号 |
| GX16 | 2~10 pin | 5~15A | IP40 | 电机、电源 |
| GX20 | 2~12 pin | 10~20A | IP50 | 大电流 |
| MIL-DTL-38999 | 多种 | 5~23A | IP67 | 高可靠性 |

### 电赛使用场景

```
GX16 (最常用)：
  - 电源输入输出（2/3 pin）
  - 电机动力线（3/4 pin）
  - 传感器信号线（5/7 pin）

连接方法：
  1. 焊接式：线芯焊接到插针
  2. 螺丝压接式：拧螺丝固定线芯
  3. 压接式：专用压接工具

安装步骤：
  1. 将插头底座安装到面板（开孔16mm）
  2. 线缆穿过底座
  3. 焊接/压接到插针
  4. 装入底座，拧紧螺母
```

### 选型建议

```
电赛推荐：
  电源连接：GX16-2（红黑两芯，5~15A）
  电机连接：GX16-3/4（三相+地）
  综合信号：GX16-7（多路信号）
  
注意事项：
  - 注意公母配对（面板装母座，线装公头）
  - 额定电流要留余量
  - 焊接后检查虚焊
```

## 6. 香蕉头

### 规格

```
标准直径：4mm（最常用）、2mm（精密）
额定电流：4mm → 15~30A（弹簧式）/ 5~10A（普通）
颜色编码：
  红色 = 正极 / VCC
  黑色 = 负极 / GND
  黄色 = 信号
  绿色 = 接地
```

### 电赛使用场景

```
用途               推荐型号
─────────────────────────────────
电源测试接口        4mm 弹簧香蕉头
万用表表笔          4mm 插头
电源模块输出端      香蕉插座（面板安装）
实验室电源连接      香蕉头-鳄鱼夹线

使用注意：
  - 弹簧式香蕉头接触更好
  - 注意颜色区分正负极
  - 面板安装时开孔 8mm
```

## 7. BNC 连接器

### 规格

```
全称：Bayonet Neill-Concelman
阻抗：50Ω（射频）/ 75Ω（视频）
频率范围：DC ~ 4GHz
额定电压：500V
连接方式：卡口旋转锁定
```

### 电赛使用场景

```
场景               说明
─────────────────────────────────
示波器探头         标准BNC接口
信号发生器输出     BNC输出端
射频模块           50Ω同轴线
视频信号           75Ω同轴线
滤波器模块         BNC转接

注意事项：
  - 50Ω和75Ω不能混用（反射问题）
  - 旋转锁定要到位（听到"咔嗒"）
  - 频率>1GHz时注意线缆损耗
```

### BNC 转接

```
常用转接件：
  BNC-SMA转接头
  BNC-鳄鱼夹线
  BNC-Banana转接头
  BNC-T型分路器
  BNC终端负载（50Ω/75Ω）
```

## 8. SMA 连接器

### 规格

```
全称：SubMiniature version A
阻抗：50Ω（标准）
频率范围：DC ~ 18GHz
连接方式：螺纹旋转（1/4英寸扳手）
额定电压：500V
```

### 电赛使用场景

```
场景               说明
─────────────────────────────────
WiFi/蓝牙天线      SMA接口天线
GPS模块            SMA天线接口
射频功放           SMA输入输出
LoRa模块           SMA天线

常见天线：
  SMA公头天线（螺纹外螺纹+中心针）
  SMA母头座（面板安装/PCB焊接）
  注意公母配对！
```

### 注意事项

```
⚠️ SMA vs RP-SMA：
  SMA：中心有针（标准）
  RP-SMA：中心无针（反极性，路由器常用）
  → 两者外形一样但不兼容！

连接扭矩：3~5 in-lbs（用扭矩扳手）
不要用钳子拧！会损坏连接器。
```
