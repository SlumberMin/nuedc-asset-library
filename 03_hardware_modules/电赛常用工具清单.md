# 电赛常用工具清单

> 赛前必备工具检查清单，别到了赛场才发现缺东西

## 一、测量仪器

### 万用表

| 项目 | 说明 |
|------|------|
| **推荐型号** | 优利德 UT61E / 福禄克 15B+ |
| **必备功能** | 直流电压、电流、电阻、通断蜂鸣、二极管 |
| **加分功能** | 电容测量、频率测量、自动量程 |
| **使用要点** | 测电流时不要并联！先估范围再选量程 |

**常见使用场景**：
- 检查电源电压（3.3V/5V/12V）
- 测量电阻值确认元件
- 检查线路通断（蜂鸣档）
- 测量电流消耗
- 检查二极管方向

### 示波器

| 项目 | 说明 |
|------|------|
| **推荐型号** | Rigol DS1054Z（四通道，可破解到100MHz） |
| **最低要求** | 双通道，50MHz 带宽 |
| **采样率** | ≥ 1 GSa/s |
| **必备功能** | 自动测量、FFT、存储/导出波形 |

**电赛高频使用场景**：

| 测量对象 | 设置建议 |
|---------|---------|
| PWM 输出 | DC 耦合，时间轴 50μs/div |
| 传感器模拟信号 | AC 耦合可减少偏移 |
| I2C/SPI 通信 | 触发设为边沿，双通道看时钟+数据 |
| 电机反电动势 | 注意探头衰减比（10X） |
| 电源纹波 | AC 耦合，20MHz 带宽限制 |

**使用技巧**：
- 先用 Auto 按钮快速看信号，再手动调
- 测高频信号用 10X 探头（减少负载效应）
- 善用触发功能稳定波形
- 截图保存用于报告

### 逻辑分析仪

| 项目 | 说明 |
|------|------|
| **推荐型号** | Saleae Logic（兼容版亦可） / Kingst LA2016 |
| **通道数** | ≥ 8 通道 |
| **采样率** | ≥ 24 MSa/s |
| **配套软件** | Saleae Logic / KingstVIS / PulseView |

**典型应用场景**：
- 分析 UART/I2C/SPI 通信协议
- 检查多路 PWM 相位关系
- 分析编码器信号
- 调试时序问题

**使用要点**：
- 接地线要短，最好与被测系统共地
- 采样率设为信号频率的 10 倍以上
- 善用协议解码功能

## 二、焊接工具

### 电烙铁

| 项目 | 推荐 |
|------|------|
| **入门级** | 白光 936 / 937 |
| **进阶级** | 白光 FX-888D / Hakko FX-951 |
| **国产性价比** | T12 焊台（快克/安泰信） |
| **功率** | 50-75W |
| **温度设置** | 直插元件 300-320°C，贴片 280-300°C |

### 焊接耗材

| 物品 | 规格 | 用途 |
|------|------|------|
| 焊锡丝 | 0.8mm 含铅（有铅更易上手） | 日常焊接 |
| 焊锡丝 | 0.5mm | 贴片元件 |
| 助焊剂 | 松香膏 / 水溶性助焊剂 | 提高润湿性 |
| 吸锡带 | 2mm / 3mm | 拆焊、清理焊盘 |
| 吸锡器 | 手动吸锡器 | 拆直插元件 |
| 烙铁头 | 刀头、尖头各备一个 | 不同焊接场景 |

### 焊接辅助工具

| 工具 | 用途 |
|------|------|
| **PCB 固定架** | 固定电路板，解放双手 |
| **镊子**（防静电） | 夹持贴片元件 |
| **放大镜/头戴放大镜** | 检查焊点质量 |
| **热风枪** | 拆贴片元件、热缩管 |
| **剪线钳** | 剪引脚、线材 |
| **剥线钳** | 剥导线绝缘层 |

## 五、赛场必备工具箱
