## 2. RC 滤波器

### 2.1 一阶RC低通

```
         R
Vin ──┤├──┬── Vout
           │
          ═══ C
           │
          GND

f_c = 1 / (2π × R × C)
```

**设计示例：**
| 截止频率 | R | C | 用途 |
|----------|---|---|------|
| 1.6kHz | 10kΩ | 10nF | ADC去噪 |
| 160Hz | 10kΩ | 100nF | 慢信号滤波 |
| 16Hz | 10kΩ | 1μF | 温度/电池 |
| 50Hz陷波 | 特殊设计 | - | 工频干扰 |

```c
// 数字等效（一阶IIR低通滤波器）
typedef struct {
    float alpha;    // 滤波系数
    float y_prev;   // 上次输出
} RC_Filter_t;

void RC_Filter_Init(RC_Filter_t* f, float cutoff_hz, float sample_hz) {
    float rc = 1.0f / (2.0f * 3.14159f * cutoff_hz);
    float dt = 1.0f / sample_hz;
    f->alpha = dt / (rc + dt);
    f->y_prev = 0;
}

float RC_Filter_Update(RC_Filter_t* f, float x) {
    f->y_prev = f->y_prev + f->alpha * (x - f->y_prev);
    return f->y_prev;
}
```

### 2.2 二阶RC低通（Sallen-Key无源版）

```
         R1        R2
Vin ──┤├──┬──┤├──┬── Vout
           │       │
          C1      C2
           │       │
          GND     GND

Q < 0.5（过阻尼），-40dB/dec
```

> **注意：** 纯无源二阶RC的Q值很低（最大0.33），选择性差。需要高Q值请用有源滤波器。

## 3. LC 滤波器

### 3.1 LC低通滤波器

```
         L
Vin ──ZZZZ──┬── Vout
             │
            ═══ C
             │
            GND

f_c = 1 / (2π√(LC))
```

### 3.2 设计参数

| 截止频率 | L | C | 应用 |
|----------|---|---|------|
| 10kHz | 1mH | 250nF | 开关电源输出 |
| 50kHz | 100μH | 100nF | DC-DC后级 |
| 100kHz | 47μH | 54nF | DAC输出平滑 |
| 1MHz | 10μH | 2.5nF | 射频前端 |

### 3.3 π型LC滤波器

```
        C1        L
Vin ─┬─┤├─┬──ZZZZ──┬─┤├─┬─ Vout
     │     │        │     │
    GND   GND      GND   GND
                      C2

优点：比单级LC衰减更快（-40dB/dec）
```

### 3.4 电源LC滤波设计

```c
// 设计步骤
// 1. 确定纹波要求：如输出纹波 < 10mV
// 2. 确定开关频率：如 100kHz
// 3. 选择截止频率：f_c = f_sw / 10 = 10kHz
// 4. 计算LC值：
//    L × C = 1 / (2π × f_c)²
//    假设L=220μH → C = 1/(4π² × (10000)² × 220e-6) ≈ 1.15μF
//    选择标准值C=1μF, 加大到2.2μF留余量
```

## 4. 有源滤波器

### 4.1 Sallen-Key 低通滤波器

```
         R1        R2
Vin ──┤├──┬──┤├──┬──┐
           │       │  │
          C1      ┌┴┐ │
           │      │  │ │
          GND     │  ├─┴── Vout
                  │- │
          ┌───────┤  │
          │       │+ │
          │  C2   └┬┘
          ├──┤├────┤
          │        │
         GND      R3→GND（增益设置）
```

**二阶Butterworth（Q=0.707）设计：**

```
对于fc=1kHz, Q=0.707, Gain=1:
R1 = R2 = 10kΩ
C1 = 22nF（= 2Q × C2）
C2 = 15.8nF → 取15nF

实际fc = 1/(2π√(R1×R2×C1×C2)) ≈ 1.03kHz
```

### 4.2 MFB（多反馈）带通滤波器

```
         R1         C2
Vin ──┤├──┬──┤├──┬──┐
           │       │  │
          C1      ┌┴┐ │
           │      │  │ │
          GND     │  ├─┴── Vout
                  │- │
          ┌───────┤  │
          │  R3   │+ │
          └──┤├───┘┬┘
                  GND

中心频率: f0 = 1/(2π) × √((1/R3) × (1/C1 + 1/C2) / (R1×C1×C2))
带宽: BW = 1/(2π×R3×C2)  （窄带用大R3）
Q = f0 / BW
```

### 4.3 有源滤波器运放选型

| 参数 | 要求 | 推荐型号 |
|------|------|----------|
| 低频(<10kHz) | 低失调，低噪 | TL072, OPA2134 |
| 中频(10k-100kHz) | GBW>1MHz | LM358, OPA2340 |
| 高频(>100kHz) | GBW>10MHz | OPA2350, AD8052 |
| 音频 | 低噪，低失真 | NE5532, OPA2134 |
| 精密 | 超低失调 | OPA2188, ADA4522 |

### 4.4 有源滤波器常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 自激振荡 | 相位裕度不足 | 降低增益/增大补偿电容 |
| 截止频率偏移 | 元件精度 | 用1%电阻+5%电容 |
| 失真大 | 运放摆率不足 | 选高SR运放 |
| 输出饱和 | DC偏置 | 加隔直电容或修正偏置 |

## 5. 数字滤波器

### 5.1 移动平均滤波

```c
// 简单移动平均（最常用）
#define WINDOW_SIZE 16
float buf[WINDOW_SIZE];
int idx = 0;
float sum = 0;

float MovingAvg(float new_val) {
    sum -= buf[idx];
    buf[idx] = new_val;
    sum += new_val;
    idx = (idx + 1) % WINDOW_SIZE;
    return sum / WINDOW_SIZE;
}

// 滑动中值滤波（抗脉冲干扰）
float MedianFilter(float* buf, int size) {
    // 排序取中间值
    float sorted[size];
    memcpy(sorted, buf, size * sizeof(float));
    for (int i = 0; i < size - 1; i++) {
        for (int j = i + 1; j < size; j++) {
            if (sorted[i] > sorted[j]) {
                float tmp = sorted[i];
                sorted[i] = sorted[j];
                sorted[j] = tmp;
            }
        }
    }
    return sorted[size / 2];
}
```

### 5.2 IIR 低通滤波器（Butterworth）

```c
// 二阶Butterworth低通滤波器（双线性变换）
typedef struct {
    float b0, b1, b2;  // 分子系数
    float a1, a2;       // 分母系数（a0归一化为1）
    float x1, x2;       // 输入历史
    float y1, y2;       // 输出历史
} IIR_Filter_t;

// 设计参数（离线计算，嵌入式直接填值）
// fc=100Hz, fs=1000Hz, 二阶Butterworth
// omega = 2*pi*fc/fs = 0.6283
// sin=sin(omega/2)=0.3090, cos=cos(omega/2)=0.9511
// alpha = sin/(2*Q) = sin/(2*0.707) = 0.2185
// b0 = (1-cos)/2 = 0.0245
// b1 = 1-cos = 0.0490
// b2 = (1-cos)/2 = 0.0245
// a0 = 1+alpha = 1.2185
// a1 = -2*cos = -1.9021
// a2 = 1-alpha = 0.7815

void IIR_LPF_Init(IIR_Filter_t* f) {
    f->b0 = 0.0245f / 1.2185f;
    f->b1 = 0.0490f / 1.2185f;
    f->b2 = 0.0245f / 1.2185f;
    f->a1 = -1.9021f / 1.2185f;
    f->a2 = 0.7815f / 1.2185f;
    f->x1 = f->x2 = f->y1 = f->y2 = 0;
}

float IIR_Filter(IIR_Filter_t* f, float x) {
    float y = f->b0 * x + f->b1 * f->x1 + f->b2 * f->x2
            - f->a1 * f->y1 - f->a2 * f->y2;
    f->x2 = f->x1; f->x1 = x;
    f->y2 = f->y1; f->y1 = y;
    return y;
}
```

### 5.3 FIR 滤波器

```c
// FIR低通滤波器（线性相位）
// 系数通过MATLAB/Python的firwin设计
#define FIR_ORDER 31
const float fir_coeff[FIR_ORDER] = {
    // 100Hz低通，fs=1000Hz，Hamming窗
    // 由Python: scipy.signal.firwin(31, 100, fs=1000) 生成
    0.0012, 0.0045, 0.0102, 0.0189, 0.0307, 0.0452, 0.0616, 0.0785,
    0.0943, 0.1075, 0.1168, 0.1213, 0.1213, 0.1168, 0.1075, 0.0943,
    0.0785, 0.0616, 0.0452, 0.0307, 0.0189, 0.0102, 0.0045, 0.0012,
    // ... 补齐31个系数
};

float fir_buf[FIR_ORDER];
int fir_idx = 0;

float FIR_Filter(float x) {
    fir_buf[fir_idx] = x;
    float y = 0;
    for (int i = 0; i < FIR_ORDER; i++) {
        int idx = (fir_idx - i + FIR_ORDER) % FIR_ORDER;
        y += fir_coeff[i] * fir_buf[idx];
    }
    fir_idx = (fir_idx + 1) % FIR_ORDER;
    return y;
}
```

### 5.4 卡尔曼滤波器

```c
// 一维卡尔曼滤波（适合单变量估计）
typedef struct {
    float x;    // 状态估计值
    float P;    // 估计误差协方差
    float Q;    // 过程噪声
    float R;    // 测量噪声
} Kalman1D_t;

void Kalman_Init(Kalman1D_t* k, float Q, float R, float x0) {
    k->x = x0;
    k->P = 1.0f;
    k->Q = Q;  // Q大→跟踪快但噪声大
    k->R = R;  // R大→平滑多但响应慢
}

float Kalman_Update(Kalman1D_t* k, float measurement) {
    // 预测
    k->P += k->Q;
    
    // 更新
    float K = k->P / (k->P + k->R);  // 卡尔曼增益
    k->x += K * (measurement - k->x);
    k->P *= (1 - K);
    
    return k->x;
}

// 使用示例
Kalman1D_t km;
Kalman_Init(&km, 0.01f, 0.1f, 0);  // Q=0.01, R=0.1

while (1) {
    float adc_val = ADC_Read();
    float filtered = Kalman_Update(&km, adc_val);
}
```

### 5.5 数字滤波器对比

| 滤波器 | 阶数 | 延迟 | 相位 | 计算量 | 适用场景 |
|--------|------|------|------|--------|----------|
| 移动平均 | N | N/2 | 线性 | O(N) | 一般去噪 |
| 中值 | N | N/2 | 非线性 | O(N·logN) | 脉冲噪声 |
| IIR | 2-4 | 小 | 非线性 | O(N) | **实时滤波首选** |
| FIR | 10-64 | N/2 | 线性 | O(N) | 需线性相位 |
| 卡尔曼 | 1 | 最小 | N/A | O(1) | 状态估计/传感器融合 |

## 6. EMI 滤波器

### 6.1 差模噪声滤波

```
        L
  ┌───ZZZZ───┬───
  │           │
Vin         C_X     Vout     (X电容：跨线，滤差模)
  │           │
  └───────────┴───

推荐C_X: 0.1μF~1μF (X2安规电容)
```

### 6.2 共模噪声滤波

```
       L_CM            L_CM
  ┌──ZZZZ──┬──────┬──ZZZZ──┐
  │        │      │        │
Vin      C_Y1   C_Y2     Vout  (Y电容：跨线到地，滤共模)
  │        │      │        │
  └────────┴──GND─┴────────┘

推荐C_Y: 2.2nF~4.7nF (Y1安规电容)
```

### 6.3 完整EMI滤波器

```
     C_X
  ┌──┤├────────────────────┐
  │                        │
  │  L_CM    L_CM          │
  ├──ZZZZ──┬──ZZZZ──┬──┤├──┤
  │       C_Y1    C_Y2     │
Vin       │       │       Vout
  │       GND     GND      │
  ├──ZZZZ──┬──ZZZZ──┬──────┤
  │  L_CM    L_CM   │      │
  │       C_Y3    C_Y4     │
  │       │       │        │
  └───────┴──GND──┴────────┘

类型：π型EMI滤波器
```

## 7. 电赛滤波器选型实战

### 7.1 ADC输入滤波

```
传感器输出 ── R(100Ω) ──┬── ADC输入
                         │
                        C(100nF)
                         │
                        GND

截止频率: fc = 1/(2π×100×100e-9) = 15.9kHz
作用: 去除高频噪声+抗混叠
```

### 7.2 电源滤波

```
5V电源 ── EMI滤波器 ── LDO ── 10μF ── 100nF ── 3.3V
                          │
                         100nF（每个VDD引脚）
```

### 7.3 DAC输出平滑

```
DAC ── 有源二阶Butterworth ── 功放
        fc = 信号最高频率 × 1.2
```

### 7.4 传感器信号调理

```
传感器 ── 仪表放大器 ── 带通滤波 ── PGA ── ADC
(INA128)    (有源BPF: 0.1-10Hz)
```
