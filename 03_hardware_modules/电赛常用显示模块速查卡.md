## 二、OLED显示模块

### 2.1 SSD1306 0.96" OLED (128×64)

**最常用模块，强烈推荐电赛首选**

```
┌──────────────┐
│  VCC  GND    │  ← 顶部引脚
│  SCL  SDA    │  ← I2C模式
│              │
│  ┌────────┐  │
│  │ 128×64 │  │  ← 显示区域
│  │ 像素   │  │
│  └────────┘  │
└──────────────┘
```

#### 引脚定义（I2C模式）

| 引脚 | 说明 | 接线 |
|------|------|------|
| VCC | 电源 | 3.3V或5V |
| GND | 地 | GND |
| SCL | I2C时钟 | PB6/PB8 |
| SDA | I2C数据 | PB7/PB9 |

#### STM32 HAL驱动（I2C）

```c
#include "ssd1306.h"

// 初始化
void OLED_Init(void) {
    SSD1306_Init();
    SSD1306_Clear();
    SSD1306_UpdateScreen();
}

// 显示字符串
void OLED_Show(void) {
    SSD1306_Clear();
    SSD1306_SetCursor(0, 0);
    SSD1306_WriteString("Voltage: 3.30V", Font_7x10, White);
    SSD1306_SetCursor(0, 12);
    SSD1306_WriteString("Current: 1.50A", Font_7x10, White);
    SSD1306_SetCursor(0, 24);
    SSD1306_WriteString("Power:   4.95W", Font_7x10, White);
    SSD1306_UpdateScreen();
}

// 显示数值（格式化）
void OLED_ShowValue(int x, int y, const char* label, float value, 
                     const char* unit, FontDef font) {
    char buf[32];
    sprintf(buf, "%s:%.2f%s", label, value, unit);
    SSD1306_SetCursor(x, y);
    SSD1306_WriteString(buf, font, White);
}
```

#### Arduino驱动

```cpp
#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>

#define SCREEN_WIDTH 128
#define SCREEN_HEIGHT 64
Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, -1);

void setup() {
    display.begin(SSD1306_SWITCHCAPVCC, 0x3C);
    display.clearDisplay();
    display.setTextSize(1);
    display.setTextColor(SSD1306_WHITE);
}

void loop() {
    display.clearDisplay();
    display.setCursor(0, 0);
    display.printf("Voltage: %.2fV", voltage);
    display.setCursor(0, 16);
    display.printf("Current: %.2fA", current);
    display.display();
    delay(100);
}
```

#### 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 不显示 | I2C地址错误 | 扫描地址0x3C或0x3D |
| 乱码 | 字库不匹配 | 使用配套字库 |
| 闪屏 | 刷新太快 | 降低刷新率或用双缓冲 |
| 对比度差 | 初始化参数 | 调整亮度寄存器 |

### 2.2 SH1106 1.3" OLED (128×64)

与SSD1306类似，驱动略有不同（132×64 RAM，显示128×64）。

```c
// SH1106与SSD1306主要区别：页地址模式不同
// SH1106每页有132字节RAM，显示时偏移2字节
```

## 三、LCD字符显示模块

### 3.1 LCD1602 (16×2字符)

```
┌────────────────────────┐
│ H e l l o   W o r l d  │  ← 第1行 (16字符)
│ T = 2 5 . 3 ℃         │  ← 第2行 (16字符)
└────────────────────────┘
```

#### 引脚定义（I2C背板）

| 引脚 | 说明 | 接线 |
|------|------|------|
| VCC | 电源 | 5V |
| GND | 地 | GND |
| SDA | I2C数据 | PB7 |
| SCL | I2C时钟 | PB6 |

I2C地址：0x27（PCF8574）或0x3F

#### 驱动代码

```c
// LCD1602 I2C驱动（PCF8574背板）
#define LCD_ADDR 0x27

void LCD_Init(void);
void LCD_Clear(void);
void LCD_SetCursor(uint8_t row, uint8_t col);
void LCD_WriteString(const char* str);
void LCD_WriteChar(char c);

// 使用示例
void LCD_ShowData(void) {
    LCD_Clear();
    LCD_SetCursor(0, 0);
    LCD_WriteString("V=3.30V I=1.5A");
    LCD_SetCursor(1, 0);
    LCD_WriteString("P=4.95W T=25C");
}
```

### 3.2 LCD2004 (20×4字符)

与LCD1602类似，4行×20字符，I2C地址相同。

```c
LCD_SetCursor(0, 0); LCD_WriteString("=== System Status ===");
LCD_SetCursor(1, 0); LCD_WriteString("Voltage:  12.00V");
LCD_SetCursor(2, 0); LCD_WriteString("Current:   2.50A");
LCD_SetCursor(3, 0); LCD_WriteString("Temp:     35.2C");
```

## 四、TFT彩色显示模块

### 4.1 ST7735 1.8" TFT (128×160)

```
┌────────────┐
│ VCC  GND   │
│ SCL  SDA   │  ← SPI接口
│ RES  DC    │  ← 复位/数据命令选择
│ CS   BLK   │  ← 片选/背光
└────────────┘
```

#### STM32 SPI驱动

```c
#include "st7735.h"

void TFT_Init(void) {
    ST7735_Init();
    ST7735_SetRotation(1);  // 横屏
    ST7735_FillScreen(ST7735_BLACK);
}

// 显示波形
void TFT_DrawWaveform(float* data, uint16_t len, uint16_t color) {
    for (uint16_t i = 0; i < len - 1; i++) {
        uint16_t x1 = i * 128 / len;
        uint16_t x2 = (i+1) * 128 / len;
        uint16_t y1 = 80 - (uint16_t)(data[i] * 70);
        uint16_t y2 = 80 - (uint16_t)(data[i+1] * 70);
        ST7735_DrawLine(x1, y1, x2, y2, color);
    }
}

// 显示彩色数据
void TFT_ShowDashboard(float v, float i, float p, float t) {
    ST7735_FillScreen(ST7735_BLACK);
    
    ST7735_WriteString(10, 10,  "V:", Font_7x10, ST7735_GREEN);
    ST7735_WriteString(30, 10,  dtostrf(v,5,2), Font_7x10, ST7735_WHITE);
    ST7735_WriteString(80, 10,  "V", Font_7x10, ST7735_GREEN);
    
    ST7735_WriteString(10, 30,  "I:", Font_7x10, ST7735_YELLOW);
    ST7735_WriteString(30, 30,  dtostrf(i,5,2), Font_7x10, ST7735_WHITE);
    ST7735_WriteString(80, 30,  "A", Font_7x10, ST7735_YELLOW);
    
    ST7735_WriteString(10, 50,  "P:", Font_7x10, ST7735_CYAN);
    ST7735_WriteString(30, 50,  dtostrf(p,5,2), Font_7x10, ST7735_WHITE);
    ST7735_WriteString(80, 50,  "W", Font_7x10, ST7735_CYAN);
}
```

### 4.2 ILI9341 2.4" TFT (240×320)

分辨率更高，可显示更多内容。

#### 并行接口接线（8080时序）

| 引脚 | STM32引脚 | 说明 |
|------|-----------|------|
| D0-D7 | PB0-PB7 | 数据总线 |
| CS | PA4 | 片选 |
| DC | PA3 | 数据/命令 |
| WR | PA2 | 写使能 |
| RD | PA1 | 读使能 |
| RST | PA0 | 复位 |
| BL | PA5 | 背光 |

#### 图形界面示例

```c
// 简单GUI元素
void GUI_DrawButton(int x, int y, int w, int h, const char* text, 
                     uint16_t color, bool pressed) {
    uint16_t bg = pressed ? ST7735_WHITE : color;
    uint16_t fg = pressed ? color : ST7735_WHITE;
    
    TFT_FillRect(x, y, w, h, bg);
    TFT_DrawRect(x, y, w, h, fg);
    int tx = x + (w - strlen(text)*8) / 2;
    int ty = y + (h - 12) / 2;
    TFT_WriteString(tx, ty, text, Font_7x10, fg);
}

// 进度条
void GUI_DrawProgressBar(int x, int y, int w, int h, 
                          float percent, uint16_t color) {
    TFT_DrawRect(x, y, w, h, ST7735_WHITE);
    int fill_w = (int)((w-2) * percent / 100.0);
    TFT_FillRect(x+1, y+1, fill_w, h-2, color);
}
```

## 五、墨水屏（E-Paper）

### 5.1 2.13" 墨水屏 (212×104)

```
特点：
- 断电保持显示（超低功耗）
- 阳光下可读
- 刷新慢（2-5秒全刷）
- 只有黑白两色（或黑白红三色）
```

#### 接线

| 引脚 | 说明 | STM32接线 |
|------|------|-----------|
| VCC | 3.3V | 3.3V |
| GND | 地 | GND |
| DIN | MOSI | PA7 |
| CLK | SCK | PA5 |
| CS | 片选 | PA4 |
| DC | 数据/命令 | PA3 |
| RST | 复位 | PA2 |
| BUSY | 忙信号 | PA1 |

#### 驱动代码

```c
#include "epaper.h"

void EPD_ShowInfo(const char* title, const char* line1, 
                   const char* line2, const char* line3) {
    EPD_Init();
    EPD_Clear();
    
    Paint_NewImage(IMAGE_BW, EPD_WIDTH, EPD_HEIGHT, 0, WHITE);
    Paint_Clear(WHITE);
    
    Paint_DrawString_CN(10, 10, title, &Font24CN, BLACK, WHITE);
    Paint_DrawString_EN(10, 50, line1, &Font16, WHITE, BLACK);
    Paint_DrawString_EN(10, 70, line2, &Font16, WHITE, BLACK);
    Paint_DrawString_EN(10, 90, line3, &Font16, WHITE, BLACK);
    
    EPD_Display();
    EPD_Sleep();  // 进入休眠省电
}
```

## 六、LED矩阵

### 6.1 MAX7219 8×8 LED矩阵

```
接线（SPI）:
VCC → 5V
GND → GND
DIN → PA7 (MOSI)
CS  → PA4
CLK → PA5
```

#### 驱动代码

```c
#include "max7219.h"

// MAX7219初始化
void LED_Matrix_Init(void) {
    MAX7219_Init();
    MAX7219_SetIntensity(0x05);  // 亮度 0-15
}

// 显示图案（8字节数组）
void LED_Matrix_ShowPattern(const uint8_t pattern[8]) {
    for (int i = 0; i < 8; i++) {
        MAX7219_SetRow(i + 1, pattern[i]);
    }
}

// 显示简单箭头
const uint8_t arrow_up[] = {
    0b00011000,
    0b00111100,
    0b01111110,
    0b11111111,
    0b00011000,
    0b00011000,
    0b00011000,
    0b00011000
};

// 使用
LED_Matrix_ShowPattern(arrow_up);
```

### 6.2 WS2812B RGB灯带/灯环

```
特点：
- 单线协议（只需一个GPIO）
- 可编程RGB颜色
- 级联扩展方便

接线:
DIN → PA8 (任意GPIO，需精确时序)
VCC → 5V（注意供电能力）
GND → GND
```

#### 驱动代码（STM32 DMA方式）

```c
#include "ws2812.h"

#define LED_COUNT 8  // 8颗LED

void WS2812_ShowStatus(int level, uint32_t color) {
    // level: 0-8, 显示状态条
    for (int i = 0; i < LED_COUNT; i++) {
        if (i < level) {
            WS2812_SetPixel(i, color);
        } else {
            WS2812_SetPixel(i, 0x000000);  // 关闭
        }
    }
    WS2812_Update();
}

// 颜色定义
#define COLOR_RED    0xFF0000
#define COLOR_GREEN  0x00FF00
#define COLOR_BLUE   0x0000FF
#define COLOR_YELLOW 0xFFFF00
#define COLOR_CYAN   0x00FFFF
#define COLOR_WHITE  0xFFFFFF

// 电量指示灯效
void ShowBatteryLevel(float voltage) {
    int level = (int)((voltage - 3.0) / 1.2 * 8);  // 3.0-4.2V → 0-8
    if (level > 8) level = 8;
    if (level < 0) level = 0;
    
    uint32_t color = (level > 2) ? COLOR_GREEN : COLOR_RED;
    WS2812_ShowStatus(level, color);
}
```

## 七、显示模块选型速查

### 7.1 按场景选型

```
需要显示什么？
│
├─ 仅数字/文字（参数值）
│  └─ OLED 0.96" I2C （最简单，推荐首选）
│     或 LCD1602 I2C （更便宜）
│
├─ 需要波形/图表
│  ├─ 简单波形 → OLED 1.3" SPI
│  └─ 彩色波形 → TFT 1.8" / 2.4"
│
├─ 需要复杂图形界面
│  └─ TFT 2.4"/3.5" （带触摸可选）
│
├─ 需要超低功耗/断电保持
│  └─ 墨水屏 2.13"
│
└─ 简单状态指示
   ├─ LED灯/灯带
   └─ LED矩阵
```

### 7.2 接口速度对比

| 接口 | 速率 | 线数 | 适用模块 |
|------|------|------|---------|
| I2C 400kHz | 400kbps | 2线 | OLED, LCD |
| SPI 10MHz | 10Mbps | 4线 | OLED, TFT, 墨水屏 |
| 并行8位 | ~40Mbps | 10+线 | TFT大屏 |
| 并行16位 | ~80Mbps | 18+线 | TFT高分辨率 |

### 7.3 常见问题速查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 无显示 | 电源/接线错误 | 检查VCC、GND、信号线 |
| I2C找不到设备 | 地址不对 | I2C扫描程序确认地址 |
| 显示模糊 | 对比度/背光 | 调初始化参数 |
| 刷屏慢 | SPI频率低 | 提高SPI时钟 |
| 花屏 | 时序问题 | 检查初始化序列 |
| 颜色错乱 | RGB顺序 | 修改颜色宏定义 |
| 墨水屏残影 | 刷新不完全 | 执行全刷（非局刷） |
