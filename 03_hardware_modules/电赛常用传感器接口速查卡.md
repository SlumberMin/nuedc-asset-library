## 一、温度传感器

### 1.1 DS18B20（数字温度）

```
接口：单总线（1-Wire）
精度：±0.5°C（-10~+85°C）
范围：-55~+125°C
供电：3.0~5.5V
引脚：GND(1) DATA(2) VCC(3)

接线：VCC→3.3V, GND→GND, DATA→PA0（需4.7kΩ上拉）

要点：
  - 单总线时序严格，禁中断
  - 12bit转换需750ms
  - 多个DS18B20可挂同一线（ROM寻址）
```

**故障诊断**：
| 现象 | 原因 | 解决 |
|---|---|---|
| 读取85°C | 未完成转换就读 | 等750ms或检查转换命令 |
| 读取全0/全1 | 接线错误/上拉缺失 | 检查4.7kΩ上拉 |
| 时序失败 | 中断干扰 | 读取时关中断 |
| 多设备冲突 | ROM冲突 | 逐个搜索ROM |

### 1.2 NTC热敏电阻（模拟温度）

```
接口：ADC（分压电路）
精度：±1~2°C
范围：-40~+125°C
供电：参考电压
引脚：两端无极性

接线：VCC→NTC→ADC引脚→10kΩ固定电阻→GND

计算：R_ntc = R_fixed * ADC_val / (4096 - ADC_val)
      T = 1 / (1/T0 + ln(R_ntc/R0)/B) - 273.15
      B值：通常3950，查数据手册
```

### 1.3 热电偶（高温测量）

```
型号：K型（最常用）
接口：需冷端补偿 + 放大（约40μV/°C）
范围：-200~+1372°C
芯片：MAX6675（K型专用，SPI接口）/ MAX31855

接线：MAX6675 → SPI(SCK/CS/SO) → MCU
      热电偶 → T+/T-引脚
```

## 二、湿度传感器

### 2.1 DHT11（温湿度一体）

```
接口：单总线
精度：±5%RH, ±2°C
范围：20~90%RH, 0~50°C
供电：3.3~5.5V
引脚：VCC(1) DATA(2) NC(3) GND(4)

接线：VCC→5V, DATA→PA0（5kΩ上拉）, GND→GND
```

### 2.2 DHT22/AM2302

```
精度：±2%RH, ±0.5°C
范围：0~100%RH, -40~80°C
与DHT11引脚兼容，通信协议相同但数据格式不同
```

### 2.3 SHT30（高精度I2C）

```
接口：I2C（地址0x44）
精度：±2%RH, ±0.3°C
供电：2.4~5.5V
优势：I2C接口简单，精度高

接线：VCC→3.3V, SDA→PB7, SCL→PB6, ADDR→GND(0x44)/VCC(0x45)
```

## 三、压力/气压传感器

### 3.1 BMP280（气压+温度）

```
接口：I2C(0x76/0x77) 或 SPI
精度：±1hPa（气压）, ±1°C（温度）
供电：1.71~3.6V
用途：气压测高、天气监测

接线：VCC→3.3V, SDA→PB7, SCL→PB6, SDO→GND(0x76)
```

### 3.2 MPX5010（差压传感器）

```
接口：模拟输出（0.2~4.7V）
量程：0~10kPa
供电：5V
用途：气体流量、风速测量

接线：VCC→5V, GND→GND, Vout→ADC
计算：P(kPa) = (Vout - 0.2) / 0.45
```

## 四、光电/红外传感器

### 4.1 光敏电阻（LDR）

```
接口：ADC（分压）
响应：可见光
特点：响应慢（~ms），非线性

接线：VCC→LDR→ADC引脚→10kΩ→GND
用途：环境光检测、光电计数
```

### 4.2 红外对管（槽型光耦）

```
型号：ITR9608 / TCPT1300
接口：数字（施密特触发器）或模拟
用途：转速测量、编码器

接线：发射管→串联限流电阻(100~330Ω)→VCC
      接收管→集电极→上拉(10kΩ)→VCC，发射极→GND
      集电极→GPIO（或ADC）
```

### 4.3 红外测距（Sharp GP2Y0A21）

```
接口：模拟输出
量程：10~80cm
供电：5V
引脚：VCC(1) GND(2) Vo(3)

接线：VCC→5V, GND→GND, Vo→ADC
注意：输出非线性，需查表或拟合
距离(cm) ≈ 27.86 / (Vout - 0.1)
```

### 4.4 激光测距（VL53L0X）

```
接口：I2C(0x29)
量程：50~1200mm（TOF原理）
精度：±3%
供电：2.6~3.5V
优势：体积小、精度高、不受目标颜色影响

接线：VCC→3.3V, SDA→PB7, SCL→PB6, XSHUT→GPIO（使能）
```

## 五、惯性传感器（IMU）

### 5.1 MPU6050（加速度+陀螺仪）

```
接口：I2C(0x68/0x69)
6轴：3轴加速度 + 3轴角速度
供电：2.37~3.46V

接线：VCC→3.3V, SDA→PB7, SCL→PB6, AD0→GND(0x68)
INT→PA1（数据就绪中断）

DMP库：可直接输出四元数/欧拉角（推荐使用）
```

### 5.2 MPU9250（9轴）

```
加速度+陀螺仪+磁力计
与MPU6050引脚兼容，I2C地址相同
内置磁力计AK8963（辅助I2C地址0x0C）
```

### 5.3 ICM20948（低功耗9轴）

```
MPU9250的升级版，功耗更低
I2C地址：0x68/0x69
自带磁力计
```

### 5.4 BMI270（6轴，低漂移）

```
接口：I2C/SPI
特点：陀螺仪漂移极小，适合长期姿态测量
供电：1.71~3.6V
```

**IMU故障诊断**：
| 现象 | 原因 | 解决 |
|---|---|---|
| I2C无应答 | 地址错/未上电 | 检查AD0电平和供电 |
| 数据跳变大 | 电源噪声 | VCC加100nF去耦 |
| 陀螺仪漂移 | 温漂 | 上电静止校准零偏 |
| 姿态发散 | 融合算法问题 | 降低陀螺仪权重，增大加速度计权重 |

## 六、超声波传感器

### 6.1 HC-SR04

```
接口：Trig(触发)+Echo(回波)
量程：2~400cm
精度：±3mm
供电：5V

接线：VCC→5V, Trig→PA0, Echo→PA1（需分压5V→3.3V）, GND→GND

时序：
  1. Trig拉高10μs触发
  2. Echo输出高电平，宽度=往返时间
  3. 距离 = Echo高电平时间 * 340 / 2 / 10000 (cm)
```

```c
float ultrasonic_measure(void) {
    HAL_GPIO_WritePin(GPIOA, TRIG_PIN, GPIO_PIN_SET);
    delay_us(10);
    HAL_GPIO_WritePin(GPIOA, TRIG_PIN, GPIO_PIN_RESET);

    // 等待Echo上升沿
    while (HAL_GPIO_ReadPin(GPIOA, ECHO_PIN) == GPIO_PIN_RESET);
    uint32_t start = __HAL_TIM_GET_COUNTER(&htim2);

    // 等待Echo下降沿
    while (HAL_GPIO_ReadPin(GPIOA, ECHO_PIN) == GPIO_PIN_SET);
    uint32_t end = __HAL_TIM_GET_COUNTER(&htim2);

    float time_us = (float)(end - start);  // μs
    return time_us * 0.017f;  // cm（340m/s / 2 / 10000）
}
```

**故障诊断**：
| 现象 | 原因 | 解决 |
|---|---|---|
| 测量值不变 | 未触发/超时 | 检查Trig脉冲和Echo接线 |
| 距离偏大 | 温度影响声速 | 加温度补偿 |
| 读数跳变 | 多传感器串扰 | 交替触发，加间隔 |
| 最小距离受限 | 盲区（~2cm） | 正常现象，加红外近距补充 |

## 七、编码器/转速传感器

### 7.1 增量式旋转编码器

```
接口：A相+B相（正交编码）+ Z相（零位）
分辨率：100~2000 PPR
供电：5V

接线：A相→TIM_CH1, B相→TIM_CH2（配置为编码器模式）
      VCC→5V, GND→GND

STM32配置：TIM_EncoderMode_TI12, 4倍频
速度计算：RPM = 脉冲数 * 60 / (PPR * 4 * 采样时间)
```

### 7.2 霍尔传感器（无刷电机）

```
接口：3路霍尔信号（U/V/W）
供电：5V
接线：Ha→TIM_CH1, Hb→TIM_CH2, Hc→TIM_CH3
      或直接GPIO读取

换相逻辑：根据3路霍尔信号的8种组合（有效6种）切换6步换相
```

## 八、电压/电流传感器

### 8.1 ACS712（电流传感器）

```
接口：模拟输出
量程：±5A/±20A/±30A（三种型号）
灵敏度：185/100/66 mV/A
供电：5V

接线：VCC→5V, GND→GND, IP+→被测线路, IP-→被测线路, VIOUT→ADC
计算：I = (Vadc - 2.5) / 0.185  （5A型号）
注意：输出有2.5V零点偏置
```

### 8.2 INA219（电流+功率I2C）

```
接口：I2C(0x40~0x4F)
量程：±3.2A（取决于采样电阻）
供电：3~5.5V
优势：I2C数字输出，自带校准

接线：VCC→3.3V, SDA/SCL→I2C, IN+→被测高侧, IN-→被测低侧
```

### 8.3 电压分压测量

```c
// 电阻分压：R1/(R1+R2) = ADC量程/最大被测电压
// 例：测量0~30V，ADC量程3.3V
// R1=100kΩ, R2=11kΩ → 分压比 11/111 ≈ 0.099
// V_in = V_adc * (R1+R2) / R2

float measure_voltage(uint16_t adc_val) {
    return (float)adc_val / 4096.0f * 3.3f * (100.0f + 11.0f) / 11.0f;
}
```

## 十二、通用故障诊断树

```
传感器无数据输出
├── 检查供电 → 用万用表测VCC-GND电压
│   ├── 无电压 → 检查接线、电源开关、LDO
│   └── 电压正常 → 检查信号线
│       ├── I2C无应答 → 检查地址、上拉电阻(4.7kΩ)、SDA/SCL是否接反
│       ├── SPI无数据 → 检查CPOL/CPHA、CS片选、MOSI/MISO
│       └── UART无数据 → 检查波特率、TX/RX是否接反
├── 有数据但不准
│   ├── 值固定不变 → 传感器损坏或初始化失败
│   ├── 值跳变大 → 加去耦电容(100nF)、软件滤波
│   └── 有固定偏移 → 需校准（零点/满量程标定）
└── 间歇性工作
    ├── 接线松动 → 重新焊接/压接
    ├── 供电不足 → 加大电容或换LDO
    └── 时序竞争 → 加延时/重试机制
```
