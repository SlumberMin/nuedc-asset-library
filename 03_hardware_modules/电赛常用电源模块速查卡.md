## 2. LDO 线性稳压器

### 2.1 特点

- **优点**：输出纹波小、电路简单、响应快、低噪声
- **缺点**：效率低（η ≈ Vout/Vin）、发热大
- **适用**：小电流、低噪声场合（模拟前端、ADC基准）

### 2.2 常用型号速查

| 型号 | 输出电压 | 最大电流 | 压差 | 静态电流 | 封装 | 适用场景 |
|------|---------|---------|------|---------|------|---------|
| **AMS1117-3.3** | 3.3V | 1A | 1.3V | 5mA | SOT-223 | STM32供电 |
| **AMS1117-5.0** | 5.0V | 1A | 1.3V | 5mA | SOT-223 | 通用5V |
| **HT7333** | 3.3V | 500mA | 0.15V | 4μA | SOT-89 | 电池供电 |
| **ME6211** | 3.3V | 500mA | 0.1V | 40μA | SOT-23 | 小型化 |
| **RT9013** | 可调 | 500mA | 0.25V | 25μA | SOT-23 | 低噪声MCU |
| **LP5907** | 可调 | 250mA | 0.12V | 12μA | SOT-23 | 超低噪声 |
| **TPS7A47** | 可调 | 1A | 0.03V | 4μA | SON | 高精度模拟 |
| **XC6206** | 3.3V | 200mA | 0.1V | 1μA | SOT-23 | 超低功耗 |

### 2.3 LDO 电路设计

```
          C_in            C_out
Vin ─┬──[10μF]──┬──[AMS1117]──┬──[22μF]──┬── Vout
     │          │    IN  OUT   │          │
     │         GND    │  ADJ   │         GND
     │               GND       │
     └─────────────────────────┘
     
注意:
1. 输入电容: 10μF 陶瓷电容，靠近 IN 脚
2. 输出电容: 22μF 陶瓷电容，ESR 要求看 datasheet
3. 散热: 压差×电流 = 功耗，必要时加大铜皮散热
4. 有些 LDO 对 ESR 有要求，不能全用陶瓷电容
```

### 2.4 LDO 功耗计算

```
P_loss = (Vin - Vout) × I_load

示例: 12V → 3.3V, I=500mA
P = (12 - 3.3) × 0.5 = 4.35W → 需要散热片！不推荐 LDO

示例: 5V → 3.3V, I=200mA  
P = (5 - 3.3) × 0.2 = 0.34W → 可接受

经验: 压差 > 3V 或 电流 > 500mA → 用 DC-DC
```

## 3. DC-DC 降压 (Buck)

### 3.1 特点

- **优点**：效率高（85%~95%）、电流大、发热小
- **缺点**：输出纹波较大、电路复杂、EMI
- **适用**：大电流、高压差场合

### 3.2 常用型号速查

| 型号 | 输入范围 | 输出 | 最大电流 | 频率 | 效率 | 封装 | 备注 |
|------|---------|------|---------|------|------|------|------|
| **MP2307** | 4.75~23V | 可调 | 3A | 340kHz | 93% | SOP-8 | 经典款 |
| **MP1584** | 4.5~28V | 可调 | 3A | 1.5MHz | 92% | SOT-23 | 小模块常用 |
| **LM2596** | 4.5~40V | 可调 | 3A | 150kHz | 85% | TO-263 | 经典款 |
| **TPS5430** | 5.5~36V | 可调 | 3A | 500kHz | 90% | SOP-8 | TI 品质 |
| **MP2315** | 4.5~24V | 可调 | 3A | 500kHz | 95% | SOT-23 | 高效率 |
| **TPS54331** | 3.5~28V | 可调 | 3A | 570kHz | 90% | SOT-23 | 同步整流 |
| **SY8089** | 2.5~5.5V | 可调 | 3A | 1.5MHz | 95% | SOT-23 | 低压大电流 |
| **RT8059** | 4.5~18V | 可调 | 2A | 1.5MHz | 93% | SOT-23 | 快速响应 |

### 3.3 Buck 电路设计模板

```
                    L (4.7~22μH)
Vin ─┬──[C_in]──┬──[====]──┬──[C_out]──┬── Vout
     │          │          │          │
     │         SW          │          │
     │     [MP2307]        D (肖特基)  │ (有些芯片内置同步整流)
     │     BST  FB         │         GND
     │      │   │          │
     │     Cbst R1         │
     │      │   ├── R2 ── GND
     │     GND  │
     │         GND

关键参数选择:
1. 电感 L: 根据纹波电流 ΔI = (Vin-Vout)*D/(f*L)
   - 一般 ΔI = 20%~40% × I_load
   - 3A 负载: 4.7~10μH
2. 输入电容: 低 ESR 陶瓷电容，22~47μF
3. 输出电容: 低 ESR 陶瓷电容，22~100μF
4. 反馈电阻: Vout = Vref × (1 + R1/R2), Vref 通常 0.6V 或 1.2V
```

### 3.4 Buck 输出电压计算

```
Vout = Vref × (1 + R_top / R_bot)

常见 Vref:
  MP2307: 0.925V
  MP1584: 0.6V
  LM2596: 1.23V

示例: MP1584 输出 5V
  R_bot = 10kΩ
  R_top = 10k × (5/0.6 - 1) = 73.3kΩ → 取 75kΩ
```

## 4. DC-DC 升压 (Boost)

### 4.1 常用型号速查

| 型号 | 输入范围 | 输出 | 最大电流 | 频率 | 封装 | 备注 |
|------|---------|------|---------|------|------|------|
| **MT3608** | 2~24V | 可调至28V | 2A | 1.2MHz | SOT-23 | 最常用 |
| **SX1308** | 2~24V | 可调至28V | 2A | 1.2MHz | SOT-23 | MT3608替代 |
| **XL6009** | 5~32V | 可调至50V | 4A | 400kHz | TO-263 | 大功率 |
| **TPS61088** | 2.7~12V | 可调至12.6V | 10A | 1.2MHz | QFN | 大电流 |
| **FP6276** | 2~5.5V | 可调至5.3V | 6A | 500kHz | SOP-8 | 锂电升5V |

### 4.2 Boost 电路设计

```
         D (肖特基)
    ┌──[▷]──┬──[C_out]──┬── Vout
    │       │          │
Vin ─┬──[C_in]──┬── L ─┘         │
     │         │                  │
     │        SW                  │
     │    [MT3608]                │
     │     EN  FB                 │
     │      │   ├── R_top         │
     │     Vin  R_bot             │
     │         │                  │
     │        GND                GND

反馈: Vout = 0.6 × (1 + R_top/R_bot)

5V 输出: R_bot=10k, R_top=10k×(5/0.6-1)=73.3k → 75k
12V 输出: R_bot=10k, R_top=10k×(12/0.6-1)=190k → 200k
```

### 4.3 Boost 注意事项

```
⚠️ 注意:
1. Boost 输出不能短路！需要加输出保护
2. 肖特基二极管选型: 正向电流 > I_load, 反向电压 > Vout
3. 电感饱和电流 > 峰值电流 (I_peak = I_in_max + ΔI/2)
4. 输入电压不能高于输出电压 (否则直通)
5. 启动时可能有浪涌电流
```

| **CS5080E** | 2A | 5V | 多节串联 | SOP-8 |

### 6.2 TP4056 典型电路

```
        R_prog
USB 5V──[1.2kΩ]──┐      (1A 充电, R=1.2k; 500mA→2k; 200mA→5.1k)
                  │
              [TP4056]
USB ── VIN    BAT──┬── 电池+
        │    STDBY │
        │    CHRG  │        LED 指示
        │     │    │         充电中: CHRG亮, STDBY灭
       GND    │    │         充满:   CHRG灭, STDBY亮
              │    │
             LED  [10μF]
              │    │
             GND  GND

充电电流设置:
R_prog = 1200 / I_charge(A)
1A: 1.2kΩ
500mA: 2kΩ
200mA: 5.1kΩ
```

### 6.3 TP4056 + DW01 完整方案（推荐）

```
USB 5V → [TP4056 充电] → [DW01+FS8205 保护] → 电池 → 负载

保护功能:
- 过充保护: 4.25~4.35V 断开充电
- 过放保护: 2.4~2.5V 断开放电
- 过流保护: 3A 断开
- 短路保护: 快速断开

模块引脚:
  OUT+ ── 负载正极
  OUT- ── 负载负极
```

## 7. 电池保护电路

### 7.1 DW01A + FS8205 方案

```
           DW01A
电池+ ──┬── OC  OD ──┬── 负载+
        │   CS       │
        │   │    [FS8205] (双NMOS)
        │   │   S1  D1  S2  D2
        │   │    │   │   │   │
        │   └────┘   │   │   │
        │            │   │   │
电池- ──┴────────────┴───┴───┴── 负载-

DW01A 引脚:
  OC: 过充控制 → 控制充电 MOSFET
  OD: 过放控制 → 控制放电 MOSFET
  CS: 电流检测
```

### 7.2 多节串联保护

```
2S (7.4V) 串联方案:
- 保护IC: S-8242, DW01 × 2, FS8205 × 2
- 均衡: 被动均衡 (并联电阻) 或主动均衡

3S (11.1V) 串联方案:
- 保护IC: S-8254, BQ77904
- 充电管理: BQ24610 (同步降压充电)
```

## 11. 电源设计检查清单

### 11.1 设计前

- [ ] 确定输入电压范围（电池？USB？适配器？）
- [ ] 确定各路输出电压和最大电流
- [ ] 确定纹波要求（模拟电路 < 1mV，数字电路 < 50mV）
- [ ] 确定效率要求（电池供电尤其重要）
- [ ] 确定 PCB 空间

### 11.2 设计中

- [ ] 选择合适拓扑（LDO / Buck / Boost / Buck-Boost）
- [ ] 计算功耗和效率
- [ ] 选择电感（饱和电流 > 峰值电流 × 1.3）
- [ ] 选择电容（ESR、纹波电流、温度等级）
- [ ] 布局：输入电容→IC→电感→输出电容 靠近放置
- [ ] 反馈走线远离开关节点

### 11.3 调试中

- [ ] 先低压后高压测试
- [ ] 测量输出纹波（示波器 AC 耦合，20MHz 带宽限制）
- [ ] 测量效率 = Pout / Pin
- [ ] 测试负载调整率
- [ ] 测试线性调整率
- [ ] 测试瞬态响应（负载跳变）
- [ ] 温升测试（满载运行 30 分钟）

## 12. 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 输出电压不对 | 反馈电阻错误 | 检查 R1/R2 值 |
| 纹波过大 | 电容 ESR 高 | 换低 ESR 电容/增加电容 |
| 无法启动 | 输入欠压/过流 | 检查输入电压/限流 |
| 发热严重 | 效率低/散热差 | 检查布局/降低负载 |
| 有啸叫 | 电感饱和/频率低 | 换大电感/提高频率 |
| EMI 问题 | 环路面积大 | 缩短走线/加屏蔽 |
| 充电不充满 | 充电截止电压错误 | 检查充电 IC 配置 |
| 电池用不久 | 静态电流大 | 选低 Iq 的 LDO/加使能 |
