```
原理图设计 → 网表生成 → PCB布局 → PCB布线 → DRC检查 → Gerber输出 → 打样下单
  (KiCad/立创EDA/AD)
```

## 二、PCB设计规范

### 2.1 线宽与电流

| 线宽(mil) | 线宽(mm) | 1oz铜(外层)电流 | 2oz铜(外层)电流 |
|-----------|---------|----------------|----------------|
| 6mil | 0.15mm | 0.3A | 0.5A |
| 10mil | 0.25mm | 0.5A | 0.9A |
| 15mil | 0.38mm | 0.8A | 1.4A |
| 20mil | 0.5mm | 1.1A | 1.9A |
| 30mil | 0.76mm | 1.6A | 2.8A |
| 50mil | 1.27mm | 2.5A | 4.3A |
| 100mil | 2.54mm | 4.5A | 7.8A |

**经验值：1A电流约需20mil线宽（1oz铜）**

### 2.2 线间距

| 电压 | 最小间距(mil) | 推荐间距(mil) |
|------|-------------|--------------|
| 5V | 5mil | 8-10mil |
| 12V | 8mil | 12-15mil |
| 24V | 12mil | 20mil |
| 220V | 40mil | 60mil+ |

### 2.3 过孔设计

| 参数 | 标准 | 推荐 |
|------|------|------|
| 过孔直径 | 0.3mm-0.5mm | 0.3mm(信号) |
| 过孔焊盘 | 0.6mm-0.8mm | 0.6mm |
| 最小间距 | 8mil | 10mil |
| 电流能力 | 0.3mm过孔约0.5A | 大电流用多过孔 |

**大电流过孔方案：**
```
电源线旁并排放置多个过孔：
  ───[via][via][via]───
  每个过孔0.5A，3个并排可达1.5A
```

### 2.4 PCB布局原则

```
1. 信号流向：输入→处理→输出，从左到右/从上到下
2. 电源入口：靠近板边，远离敏感信号
3. 晶振：紧贴MCU，走线短，包地
4. 去耦电容：紧贴IC电源脚
5. 数字地/模拟地：单点连接
6. 发热器件：远离敏感元件，留散热空间
7. 接插件：靠板边，便于插拔
```

### 2.5 PCB布线原则

```
1. 避免直角走线（用45°或圆弧）
2. 差分线等长等间距
3. 电源线加粗（参考线宽表）
4. 关键信号线不跨分割平面
5. 地平面完整性
6. 高频信号线短且直
```

## 三、Gerber文件生成

### 3.1 什么是Gerber文件

Gerber是PCB制造的标准格式，包含各层图形信息：
- **GTL**：顶层铜箔（Top Copper）
- **GBL**：底层铜箔（Bottom Copper）
- **GTS**：顶层阻焊（Top Solder Mask）
- **GBS**：底层阻焊（Bottom Solder Mask）
- **GTO**：顶层丝印（Top Silkscreen）
- **GBO**：底层丝印（Bottom Silkscreen）
- **GTP**：顶层锡膏（Top Paste）
- **GBP**：底层锡膏（Bottom Paste）
- **GKO/GML**：板框（Board Outline）
- **TXT/DRL**：钻孔文件（Drill File）

### 3.2 立创EDA生成Gerber

```
步骤：
1. 打开PCB文件
2. 菜单：文件 → 制造文件 → Gerber
3. 设置参数：
   - 层：勾选需要的层（一般全选铜箔+阻焊+丝印+板框）
   - 格式：RS-274X
   - 精度：4:4（或4:5）
   - 单位：mm
4. 点击"生成Gerber"
5. 下载Gerber压缩包（.zip）
6. 同时下载钻孔文件(NCDrill)
```

### 3.3 Altium Designer生成Gerber

```
步骤：
1. 菜单：File → Fabrication Outputs → Gerber Files
2. General标签：
   - 单位：Inches / Millimeters
   - 格式：2:3（精度够用）或 2:4
3. Layers标签：
   - 勾选需要的层
   - Plot Layers: Used On
   - Mirror Layers: All Off
4. Drill Drawing标签：
   - 勾选钻孔图层
5. Apertures标签：
   - 勾选Embedded apertures (RS-274X)
6. 点击OK生成
7. 再生成钻孔文件：File → Fabrication Outputs → NC Drill Files
8. 将所有文件打包为.zip
```

### 3.4 KiCad生成Gerber

```
步骤：
1. 打开PCB编辑器
2. 菜单：File → Fabrication Outputs → Gerbers (.gbr)
3. 选择输出层：
   - F.Cu, B.Cu (铜箔)
   - F.SilkS, B.SilkS (丝印)
   - F.Mask, B.Mask (阻焊)
   - Edge.Cuts (板框)
4. 输出目录：选择文件夹
5. 点击"Plot"生成
6. 生成钻孔文件：File → Fabrication Outputs → Drill Files (.drl)
   - 格式：Excellon
   - 单位：mm
   - 零点：辅助原点
7. 将所有文件打包为.zip
```

### 3.5 Gerber文件验证

**下单前务必验证Gerber：**
1. **在线查看器**：https://gerber-viewer.lceda.cn （立创）
2. **独立查看器**：Gerbv（开源免费）、ZofZPCB
3. **检查项目：**
   - 各层对齐是否正确
   - 板框是否闭合
   - 焊盘/过孔是否完整
   - 丝印是否清晰无遮挡
   - 钻孔文件是否匹配

### 6.3 SMD贴片焊接技巧

**0805/0603电阻电容：**
```
1. 一个焊盘先上锡
2. 镊子夹住元件放到位
3. 烙铁熔化焊锡固定一端
4. 焊另一端
5. 回头补焊第一端
```

**QFP封装（如STM32）：**
```
1. 焊盘上锡（少量）
2. 对准引脚（1脚标记对齐）
3. 先固定对角两个脚
4. 检查对齐
5. 逐边焊接，刀头烙铁顺着引脚拖焊
6. 检查连锡，必要时用吸锡带清理
7. 助焊剂能让拖焊更顺畅
```

**拖焊技巧（QFP/SOP）：**
```
1. 大量涂助焊剂
2. 烙铁头沾少量焊锡
3. 刀头贴着引脚，匀速拖过
4. 焊锡会自动在引脚间分配（表面张力）
5. 连锡处用吸锡带或助焊剂清理
```

**BGA焊接（进阶，比赛一般不涉及）：**
```
需要：热风枪 + BGA钢网 + 锡球 + 对准工具
温度曲线：预热→回流→冷却
比赛不推荐，改用QFN/TQFP替代
```

### 6.4 常见焊接问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 虚焊(假焊) | 温度不够/时间短 | 提高温度，延长加热时间 |
| 连锡(短路) | 锡太多/间距小 | 吸锡带清理，减少锡量 |
| 焊点发暗 | 温度过高/时间长 | 降低温度，快速操作 |
| 立碑 | 两端受热不均 | 两端同时加热，涂锡均匀 |
| 焊盘脱落 | 过热/用力大 | 降低温度，轻操作 |
| 元件偏移 | 锡膏不均/对准差 | 用镊子调整，重焊 |
| 气孔/空洞 | 助焊剂残留 | 涂助焊剂，重新加热 |

### 6.5 焊接质量检查

```
外观检查：
✓ 焊点光亮饱满（有铅焊锡）
✓ 焊锡呈圆锥形覆盖焊盘和引脚
✓ 无连锡、无虚焊、无气孔
✓ 元件方向正确（极性/1脚标记）
✓ 丝印清晰可辨

电气检查：
✓ 万用表测电源对地电阻（不应短路）
✓ 逐个检查关键连接（对照原理图）
✓ 上电前先测各电源轨电压
✓ 逐步上电：先3.3V→确认→再5V→确认→电机电源
```

## 七、比赛PCB设计实战建议

### 7.1 比赛是否需要PCB？

| 场景 | 建议 |
|------|------|
| 开赛前 | 制作通用PCB底板（MCU+电源+接口预留） |
| 比赛中 | 优先洞洞板/面包板搭电路，复杂电路用预制PCB |
| 强电/大电流 | 必须用PCB，洞洞板走大电流不安全 |
| 高频电路 | 必须用PCB，面包板寄生参数太大 |

### 7.2 比赛通用底板设计建议

```
建议设计一块通用底板，预留以下资源：
- STM32F103/F407最小系统（晶振+复位+电源+Boot）
- 电源部分：电池接口+充电+3.3V/5V稳压
- UART接口 ×2（调试+蓝牙/ESP8266）
- SPI接口 ×1（OLED/Flash）
- I2C接口 ×1（传感器）
- ADC接口 ×4（电压/电流检测）
- PWM接口 ×4（电机/舵机）
- CAN接口 ×1（电机控制板）
- GPIO扩展排针 ×8-16
- 大电流接口（螺丝端子，用于电机电源）
- 板载指示LED ×3
- 按键 ×2-3
```

### 7.3 PCB打样时间规划

```
比赛前准备：
├── T-30天：确定通用底板方案，开始画PCB
├── T-14天：完成PCB设计，下单打样
├── T-7天： 收到PCB，焊接调试
├── T-3天： 确认底板功能正常
└── T-day：  比赛开始，专注算法

注意：嘉立创打样24h出货，但快递需1-3天
     至少预留一周时间！
```
