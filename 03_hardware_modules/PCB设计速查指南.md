# PCB设计速查指南

## 一、概述

本文档为电赛PCB设计提供快速参考，涵盖地线处理、电源滤波、信号完整性三大核心领域。

---

## 二、地线处理

### 2.1 地的分类

```
数字地 (DGND): MCU、数字IC、通信模块
模拟地 (AGND): ADC、传感器、基准源
功率地 (PGND): 电机驱动、DC-DC、大电流回路

原则: 三类地最终单点汇合到电源地
```

### 2.2 单点接地拓扑

```
                电源GND
                  │
         ┌────────┼────────┐
         │        │        │
       DGND     AGND     PGND
    (数字区域) (模拟区域) (功率区域)

汇合点选择: 电源输入电容的GND端

错误做法:
  ✗ 三种地混合铺铜
  ✗ 大电流回路经过模拟地区域
  ✗ 多处汇合形成地环路

正确做法:
  ✓ 各区域独立铺铜
  ✓ 单点连接
  ✓ 大电流走粗线回电源
```

### 2.3 地线走线规则

| 场景 | 推荐做法 |
|------|----------|
| 数字信号回流 | 紧贴信号线下方铺地铜 |
| 模拟信号 | 远离数字地回流路径 |
| 大电流回路 | 最短路径回电源，不经过弱信号区 |
| 多层板 | 优先用地层作参考平面 |
| 双层板 | 底层大面积铺铜作地 |

### 2.4 接地常见错误

```
错误1: 地环路
  ┌──GND──┐
  │       │
  信号源  信号接收
  │       │
  └──GND──┘
  → 两个接地点形成环路，拾取电磁干扰
  → 解决: 单点接地或星形接地

错误2: 长地线
  信号──> IC ──[很长的地线]──> GND
  → 长地线有电感，高频时阻抗大
  → 解决: 地线尽量短而宽

错误3: 地线过细
  2A电流走0.25mm地线 → 过热/压降
  → 解决: 1A ≥ 1mm，大电流铺铜
```

## 三、电源滤波

### 3.1 去耦电容放置

```
MCU VCC引脚:
  ┌──────────┐
  │   MCU    │
  │  VCC     │
  │   │      │
  │  [C1 100nF] ← 紧贴引脚，< 5mm
  │   │      │
  │  GND     │
  └──────────┘
  
  再远一点(5~20mm):
  [C2 10μF] ← bulk电容
   │
  GND

规则:
  100nF: 每个VCC引脚一个，紧贴
  10μF: 每个IC区域1~2个
  100μF: 电源入口处
```

### 3.2 LC/RC电源滤波

```
方案一: LC π型滤波 (低噪声供电)
  VCC ──[C1 10μF]──┬──[L 10μH]──┬──[C2 10μF]── VCC_CLEAN
                     │            │
                    GND          GND
  适用: MCU模拟电源、传感器供电

方案二: 铁氧体磁珠滤波 (高频噪声)
  VCC ──[FB 600Ω@100MHz]──┬──[C 100nF]── VCC_CLEAN
                            │
                           GND
  适用: 数字IC电源、通信模块

方案三: RC滤波 (简单低成本)
  VCC ──[R 10Ω]──┬──[C 10μF]── VCC_CLEAN
                   │
                  GND
  注意: R上有压降，电流不宜大（< 50mA）
  适用: 基准电压、运放供电
```

### 3.3 大电容组合策略

```
频率响应覆盖:
  100μF铝电解:  低频滤波 (10Hz~1kHz)
  10μF陶瓷X5R: 中频滤波 (1kHz~1MHz)
  100nF陶瓷X7R: 高频滤波 (1MHz~100MHz)

并联使用可覆盖宽频带:
  VCC ──┬──[100μF]──GND
        ├──[10μF]──GND
        └──[100nF]──GND
```

## 四、信号完整性

### 4.1 高速信号走线规则

```
关键信号线:
  - SPI CLK: 尽量短，< 50mm
  - USB D+/D-: 差分对，90Ω阻抗
  - 晶振: 走线< 10mm，包地处理
  - CAN/RS485: 差分对等长

走线间距规则:
  ┌──────┐    ┌──────┐
  │ 信号 │    │ 信号 │
  └──────┘    └──────┘
  │← S ≥ 2W →│
  
  S (间距) ≥ 2 × W (线宽) → 串扰 < 5%
  S ≥ 3W → 串扰 < 1%
```

### 4.2 走线直角问题

```
✗ 直角走线:
  ───┐
     │
     └───

✓ 45°走线:
  ───╲
      ╲
       ───

✓ 圆弧走线 (最优):
  ───╮
     ╰───

直角走线的阻抗突变会造成信号反射，
对>100MHz信号影响显著。
```

### 4.3 晶振布局

```
         MCU
    ┌───────────┐
    │   XTAL_IN ├──[C1 20pF]──GND
    │           │
    │   晶振    ├──[C2 20pF]──GND
    │   XTAL_OUT│
    └───────────┘

布局规则:
  1. 晶振紧贴MCU引脚 (< 5mm)
  2. 负载电容紧贴晶振引脚
  3. 晶振区域下方不走其他信号线
  4. 晶振外壳接地（金属壳晶振）
  5. 晶振走线包地保护
```

### 4.4 SPI/I2C走线

```
SPI (高速, ~10MHz+):
  ┌─── MCU ───┐         ┌─── Slave ───┐
  │  SCK ──────────────────── SCK      │
  │  MOSI──────────────────── MOSI     │
  │  MISO<────────────────────MISO     │
  │  CS  ──────────────────── CS       │
  │  GND ═══════════════════ GND      │
  └────────────────────────────────────┘
  
  要求: CLK和数据线等长，间距≥2W
  速率>10MHz时: 线长<50mm

I2C (低速, ~400KHz):
  VCC ──[R 4.7K]──┬── SCL ──> 所有设备SCL
                    │
  VCC ──[R 4.7K]──┼── SDA ──> 所有设备SDA
                    │
                   GND
  
  要求: 总线电容 < 400pF
  设备多时: 降低上拉电阻到2.2K
  走线长度: < 30cm (标准模式)
```

## 五、PCB设计速查表

### 5.1 线宽与电流对照

| 电流 (A) | 最小线宽 (外层) | 推荐线宽 (外层) | 铜箔厚度 |
|----------|-----------------|-----------------|----------|
| 0.5 | 0.3mm | 0.5mm | 1oz |
| 1.0 | 0.5mm | 1.0mm | 1oz |
| 2.0 | 1.0mm | 2.0mm | 1oz |
| 3.0 | 1.5mm | 3.0mm | 1oz |
| 5.0 | 2.5mm | 5.0mm或铺铜 | 2oz |

经验公式: I ≈ (ΔT^0.5403) × (W^0.7349) × (0.06746)
简化: 1oz铜, 温升10°C: 1A ≈ 0.5mm

### 5.2 过孔载流能力

| 过孔内径 | 焊盘直径 | 载流 (1oz) | 备注 |
|----------|----------|------------|------|
| 0.3mm | 0.6mm | ~0.5A | 常规 |
| 0.4mm | 0.8mm | ~0.8A | 常规 |
| 0.5mm | 1.0mm | ~1.0A | 推荐 |
| 0.8mm | 1.2mm | ~1.5A | 大电流 |

大电流过孔: 多个过孔并联

### 5.3 常用封装焊盘尺寸

## 六、PCB设计检查清单

### 6.1 布局检查
- [ ] MCU去耦电容紧贴VCC引脚
- [ ] 晶振紧贴MCU，走线包地
- [ ] 电源IC散热面积足够
- [ ] 接插件靠近板边
- [ ] 发热元件远离热敏器件
- [ ] 传感器远离干扰源

### 6.2 布线检查
- [ ] 电源线宽度满足电流需求
- [ ] 地线完整，无断裂
- [ ] 高速信号走线短而直
- [ ] 差分对等长等间距
- [ ] 模拟信号远离数字信号
- [ ] 无直角走线（高速信号）

### 6.3 电源检查
- [ ] 每个IC都有去耦电容
- [ ] 电源入口有大电容
- [ ] DC-DC回路面积最小
- [ ] 功率地和信号地单点连接

### 6.4 DRC检查
- [ ] 无短路
- [ ] 无开路（未连接网络）
- [ ] 线宽/间距满足制造能力
- [ ] 过孔尺寸满足钻孔能力
- [ ] 丝印不覆盖焊盘

## 七、常见问题解答

### Q1: PCB打样选什么参数？
- 板厚：1.6mm（标准）
- 铜厚：1oz（35μm）
- 层数：双层板（电赛足够）
- 阻焊：绿色（便宜）/ 其他颜色（美观）
- 最小线宽/间距：6mil/6mil（推荐8mil/8mil）
- 最小过孔：0.3mm（推荐0.4mm）

### Q2: 如何降低DC-DC纹波？
- 加LC π型滤波输出
- 输出电容并联ESR低的陶瓷电容
- 降低SW节点面积
- 反馈走线远离开关节点

### Q3: ADC采集精度差？
- 参考电压加100nF+10μF滤波
- ADC输入加RC低通滤波（10K+10nF）
- 数字地和模拟地分开
- 采样时避免其他GPIO翻转

### Q4: PCB发热严重？
- 检查铜箔宽度是否足够
- 加散热过孔到背面铜皮
- 检查是否有意外短路
- 功率器件加散热片或铺铜散热
