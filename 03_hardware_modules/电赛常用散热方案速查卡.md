## 2. 散热片选型速查

### 2.1 计算公式

```
Tj = Ta + P × (Rθjc + Rθcs + Rθsa)

Tj: 结温 (°C)
Ta: 环境温度 (°C)
P:  耗散功率 (W)
Rθjc: 结到壳热阻 (°C/W) - 器件手册
Rθcs: 壳到散热片热阻 (°C/W) - 取决于导热材料
Rθsa: 散热片到环境热阻 (°C/W) - 散热片参数
```

### 2.2 常见器件热阻参考

| 器件 | 封装 | Rθjc (°C/W) | 典型应用 |
|------|------|-------------|----------|
| IRF540N | TO-220 | 1.0 | 电机驱动 |
| IRF3205 | TO-220 | 0.75 | 大电流开关 |
| LM317 | TO-220 | 4 | 线性稳压 |
| LM7805 | TO-220 | 5 | 5V 稳压 |
| TIP122 | TO-220 | 4.2 | 达林顿管 |
| MOSFET 模块 | TO-247 | 0.5 | 大功率 |

### 2.3 散热片选型表

| 散热片型号 | 尺寸 (mm) | Rθsa (°C/W) | 适用场景 |
|-----------|-----------|-------------|----------|
| 小型鳍片 | 25×14×10 | 25-30 | < 1W |
| 中型鳍片 | 40×25×15 | 15-20 | 1-3W |
| 大型鳍片 | 60×40×20 | 8-12 | 3-8W |
| 超大鳍片 | 100×60×25 | 4-6 | 8-20W |
| 型材散热片 | 150×80×40 | 2-4 | 20-50W |

### 2.4 快速选型示例

```
场景：IRF540N 驱动电机，持续电流 10A，Rds(on)=44mΩ
功率：P = I²×R = 10²×0.044 = 4.4W
环境：Ta = 40°C
目标：Tj < 125°C

所需总热阻：Rθ_total = (125-40)/4.4 = 19.3 °C/W
Rθjc = 1.0, Rθcs = 0.5 (硅脂)
Rθsa_max = 19.3 - 1.0 - 0.5 = 17.8 °C/W

→ 选择中型散热片 (15-20 °C/W) 即可
```

## 3. 风扇选型速查

### 3.1 风扇规格参数

| 参数 | 说明 | 电赛常用值 |
|------|------|-----------|
| 尺寸 | 长×宽×高 | 40×40×10, 60×60×25 mm |
| 电压 | 额定电压 | 5V, 12V, 24V |
| 风量 | CFM (立方英尺/分) | 5-30 CFM |
| 转速 | RPM | 3000-8000 |
| 噪声 | dB(A) | 20-40 dB |

### 3.2 风扇效果估算

```
强制风冷对散热片热阻的改善：

Rθsa_风冷 ≈ Rθsa_自然 × (0.3 ~ 0.5)

例：自然散热片 Rθsa = 15 °C/W
    加风扇后 Rθsa ≈ 5-8 °C/W
```

### 3.3 风扇驱动电路

```
5V/12V 风扇 + NPN 三极管驱动：

        VCC (12V)
         │
         ├──── 风扇 (+)
         │
         │  风扇 (-)
         │
    R1   ├──── Collector
   1kΩ ──┤
    │    │     NPN (2N2222)
   GPIO ─┘     Emitter
                │
               GND

PWM 控制转速：1kHz ~ 25kHz
```

### 3.4 温控风扇

```c
/* 温控风扇软件实现 */
void fan_control(int16_t temp_c) {
    uint16_t pwm;
    
    if (temp_c < 35) {
        pwm = 0;           /* 关闭 */
    } else if (temp_c < 50) {
        pwm = 30;          /* 低速 30% */
    } else if (temp_c < 70) {
        pwm = 60;          /* 中速 60% */
    } else {
        pwm = 100;         /* 全速 100% */
    }
    
    set_pwm(PWM_FAN, pwm);
}
```

## 4. 热管散热器

### 4.1 热管原理

```
热管结构：
┌──────────────────────────────────┐
│  蒸发段    │   绝热段   │  冷凝段  │
│  (热源)    │           │ (散热)   │
│            │           │          │
│  液态→气态 │   蒸汽流动  │ 气态→液态 │
└──────────────────────────────────┘
导热系数：5000-20000 W/(m·K)  ← 远超铜的 400
```

### 4.2 热管散热器规格

| 类型 | 热管直径 | 散热能力 | 适用场景 |
|------|---------|---------|----------|
| 热管散热片 | 6mm | 30-80W | CPU/功率模块 |
| 热管阵列 | 6mm×4 | 80-150W | 大功率逆变器 |
| 均热板 | - | 50-200W | LED/功率器件 |

### 4.3 电赛应用建议

- 热管散热器适合持续大功率场景（>30W）
- 需要良好接触面（平整+导热硅脂）
- 冷凝端需要足够散热面积
- **注意方向性**：重力辅助型热管需正确安装

## 5. 导热材料选型

### 5.1 导热材料速查表

| 材料 | 导热系数 W/(m·K) | 热阻 °C·cm²/W | 适用场景 |
|------|-----------------|---------------|----------|
| 导热硅脂 | 1-5 | 0.1-0.5 | TO-220 + 散热片 |
| 导热硅胶垫 | 1-6 | 0.5-2 | 绝缘需求 |
| 导热双面胶 | 0.5-1.5 | 1-3 | 小功率固定 |
| 相变材料 | 3-8 | 0.2-0.8 | 高性能需求 |
| 液态金属 | 20-80 | 0.01-0.05 | 极限散热 |
| 石墨片 | 10-15 (面内) | 0.3-1 | 均热/手机 |

### 5.2 导热硅脂使用指南

```
涂抹方法：
1. 清洁散热面（酒精擦拭）
2. 取米粒大小硅脂于芯片中心
3. 安装散热片，自然压平
4. 拧紧螺丝（对角线顺序）

常见品牌：
- 信越 7921/7868（性价比）
- 利民 TF7/TF8（高性能）
- 暴力熊 液态金属（极限）
```

### 5.3 导热垫选型

```
导热垫选择关键：
- 厚度：0.5mm / 1mm / 1.5mm / 2mm
- 硬度：软质适合不平整表面
- 绝缘性：高压场景需要绝缘型

典型应用：
- MOSFET 背面到机箱 → 1mm 导热垫
- 功率模块到散热基板 → 0.5mm 导热垫
```

## 6. 散热设计实战案例

### 6.1 电机驱动散热

```
场景：H桥驱动直流电机，V=24V, I=5A
MOSFET: IRF540N × 4 (半桥×2)
单管功耗: P = I²×Rds(on) = 25×0.044 = 1.1W

方案选择：
├── 轻载 (< 2A) → 无散热片
├── 中载 (2-5A) → 小型散热片 + 导热硅脂
└── 重载 (> 5A) → 中型散热片 + 风扇
```

### 6.2 线性稳压散热

```
场景：LM317 从 12V 降到 5V, 负载电流 500mA
功耗: P = (12-5)×0.5 = 3.5W

方案：
Rθsa_max = (125-40)/3.5 - 4 - 0.5 = 19.8 °C/W
→ 中型散热片即可

⚠️ 注意：线性稳压效率低，大电流建议用开关稳压
```

### 6.3 逆变器散热

```
场景：全桥逆变器, 48V/10A
MOSFET: IRFP250N × 4
单管功耗: ≈ 3-5W (含开关损耗)

方案：
1. 每管配大型散热片
2. 导热硅脂 + 绝缘垫
3. 强制风冷
4. 温度监测 + 过温保护
```
