uint8_t adaptive_threshold(uint16_t *hist, int total) {
    int sum = 0, count = 0;
    for (int i = 0; i < 256; i++) {
        sum += i * hist[i];
        count += hist[i];
    }
    return sum / count;  // Otsu 简化
}

### 2.2 信号类（2021年）
**题目**: 信号失真度测量

**技术要点**:
- 高精度 ADC 采样 (ADS1256, 24bit)
- FFT 频谱分析
- THD (总谐波失真) 计算
- 自适应增益控制

**核心算法**:
```python
import numpy as np

def calc_thd(signal, fs, f0):
    """计算总谐波失真"""
    N = len(signal)
    spectrum = np.abs(np.fft.rfft(signal)) / N
    fundamental_idx = int(f0 * N / fs)
    V1 = spectrum[fundamental_idx]
    
    thd_sq = 0
    for k in range(2, 10):
        idx = int(k * f0 * N / fs)
        if idx < N // 2:
            thd_sq += spectrum[idx] ** 2
    return np.sqrt(thd_sq) / V1
```

**技术要点**:

**PID 调参经验**:
```c
// 增量式 PID
typedef struct {
    float Kp, Ki, Kd;
    float err[3];  // 当前、上次、上上次
} PID_Inc;

float PID_Calc(PID_Inc *pid, float target, float actual) {
    pid->err[2] = pid->err[1];
    pid->err[1] = pid->err[0];
    pid->err[0] = target - actual;
    
    float du = pid->Kp * (pid->err[0] - pid->err[1])
             + pid->Ki * pid->err[0]
             + pid->Kd * (pid->err[0] - 2*pid->err[1] + pid->err[2]);
    return du;
}
```

## 3. 获奖方案设计方法论

### 3.1 系统设计流程
```
需求分析 → 方案论证 → 硬件选型 → 软件架构 → 模块开发 → 系统联调 → 测试优化
```

### 3.2 评分要点
| 评分项 | 权重 | 得分策略 |

| 基本功能 | 40% | 稳定实现所有基本要求 |
| 发挥部分 | 30% | 尽可能多地完成 |
| 创新性 | 15% | 独特的解决方案 |
| 文档报告 | 15% | 逻辑清晰、图表规范 |

| Day 1 | 0-12h | 方案论证、硬件搭建 |

## 4. 硬件选型经验

### 4.1 主控选型
| 需求 | 推荐 | 理由 |
|------|------|------|
| 通用控制 | STM32F103 | 资料丰富、外设齐全 |
| 高性能 | STM32H743 | 480MHz、双精度FPU |
| DSP处理 | STM32F407 | FPU+DSP指令集 |
| 视觉处理 | RK3588S | 6TOPS NPU、丰富接口 |
| 简单视觉 | OpenMV/H7 | 快速开发、MicroPython |

### 4.2 传感器选型
- **编码器**: 增量式 600线/圈（平衡精度与成本）
- **陀螺仪**: MPU6050/ICM20602（注意温漂补偿）
- **摄像头**: OV2640/OV5640（根据分辨率需求选择）
- **超声波**: HC-SR04（短距）/ TFLuna（激光测距）

### 4.3 电源设计
```
锂电池 (12V) 
  ├── DC-DC (5V) → MCU、传感器
  ├── DC-DC (3.3V) → 逻辑电路
  ├── LDO (2.5V) → 模拟电路
  └── 直供 → 电机驱动
```

## 6. 调试经验总结

### 6.1 常见问题
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 电机抖动 | PID 参数不当 | 减小 Kp，增大 Kd |
| 通信丢包 | 波特率/干扰 | 降低波特率、加 CRC |
| 图像丢帧 | 处理太慢 | 降分辨率/优化算法 |
| 电源不稳 | 纹波过大 | 加 LC 滤波、走线加粗 |
| 传感器漂移 | 温度变化 | 软件补偿、定期校准 |

### 6.2 调试工具
- **逻辑分析仪**: 协议调试必备
- **示波器**: 电源纹波、信号质量
- **串口助手**: 实时数据监控
- **上位机**: 数据可视化
