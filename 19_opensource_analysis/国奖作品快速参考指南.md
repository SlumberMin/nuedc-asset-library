## 一、国奖作品必杀技清单

### ✅ 控制类题目

| 技巧 | 说明 | 代码位置 |
|------|------|---------|
| **动态前瞻PID** | 根据曲率自适应调整前瞻距离 | `PreviewPID` |
| **分段PID** | 大/中/小误差用不同参数 | `SegmentPID` |
| **前馈+PID** | 提升跟踪性能，减少滞后 | `FeedforwardPID` |
| **积分抗饱和** | 防止积分累积导致超调 | `Anti-Windup` |
| **微分先行** | 避免设定值突变的冲击 | `Derivative on Measurement` |
| **卡尔曼滤波** | 状态估计，滤除噪声 | `KalmanFilter` |
| **LQR最优控制** | 状态反馈，理论最优 | `LQR Controller` |
| **ADRC** | 不依赖模型，抗干扰强 | `ADRC` |

### ✅ 视觉类题目

| 技巧 | 说明 | 代码位置 |
|------|------|---------|
| **多ROI加权** | 近处权重高，远处权重低 | `multi_roi_tracking()` |
| **HSV颜色空间** | 比RGB更稳定 | `color_detect_hsv()` |
| **形态学处理** | 开运算去噪，闭运算填孔 | `morphologyEx()` |
| **透视变换** | 棋盘/标定板校正 | `warpPerspective()` |
| **模板匹配** | 简单数字/字母识别 | `find_template()` |
| **轮廓分析** | 形状识别 | `shape_recognize()` |

### ✅ 系统集成

| 技巧 | 说明 | 代码位置 |
|------|------|---------|
| **状态机架构** | 清晰的任务流程管理 | `TaskState` |
| **通信协议** | 帧头+长度+命令+数据+CRC | `ProtocolFrame` |
| **降级运行** | 模块失效时自动切换模式 | `SystemMode` |
| **定时器分工** | 不同任务分配不同周期 | `TIM_IRQHandler` |
| **超时保护** | 防止系统卡死 | `timeout_check` |

## 二、PID调参速查卡

### 手动调参口诀
```
先比例，后积分，再微分
小幅调，观响应
振荡减Kp，误差加Ki，超调加Kd
```

### 典型参数范围

| 控制对象 | Kp | Ki | Kd | 备注 |
|----------|----|----|----|----|
| 直流电机速度环 | 0.5~5 (2.0) | 0.1~2 (0.5) | 0~0.5 (0.05) | 编码器反馈 |
| 直流电机位置环 | 1~10 (5.0) | 0~1 (0.1) | 0.5~5 (1.0) | 位置环外环 |
| 舵机角度控制 | 2~15 (8.0) | 0~0.5 (0.01) | 1~5 (2.0) | 电位器反馈 |
| 循线偏移控制 | 1~5 (2.5) | 0~0.5 (0.1) | 0.5~3 (1.2) | 视觉偏移量 |
| 温度控制 | 2~20 (10.0) | 0.1~2 (0.5) | 1~10 (5.0) | NTC/PT100 |

### 问题速查表

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| 响应太慢 | Kp太小 | 增大Kp |
| 振荡/超调大 | Kp太大或Kd太小 | 减小Kp，增大Kd |
| 稳态误差 | Ki太小 | 增大Ki |
| 积分饱和 | Ki太大 | 减小Ki，加积分限幅 |
| 高频抖动 | Kd太大或噪声 | 减小Kd，加低通滤波 |
| 启动冲击 | 微分冲击 | 用微分先行 |
| 负载变化大 | 固定参数不够 | 用自适应/模糊PID |

## 三、报告写作评分要点

### 评分权重
```
方案论证：15% → 货比三家，选择理由充分
理论分析：20% → 公式推导，仿真验证
电路设计：20% → 原理图+PCB，选型依据
软件设计：20% → 流程图+关键代码
测试结果：15% → 数据表格，误差分析
文档规范：10% → 排版整洁，无错别字
```

### 高分技巧
```
1. 方案论证：2-3个方案对比，表格形式
2. 理论推导：数学公式+物理解释
3. 仿真验证：MATLAB/Simulink截图
4. 测试数据：表格+图表，带单位和误差
5. 创新点：单独列出，量化说明
```

## 七、快速排查清单

### 电机不转
- [ ] 电源电压是否正常
- [ ] PWM信号是否输出
- [ ] 驱动芯片是否发烫
- [ ] 电机线是否接反

### 编码器读数异常
- [ ] 供电电压是否正常
- [ ] A/B相是否接反
- [ ] 上拉电阻是否加上
- [ ] 定时器配置是否正确

### 摄像头无图像
- [ ] 供电是否正常
- [ ] 数据线是否松动
- [ ] 分辨率/帧率是否设置
- [ ] DMA是否使能

### 通信中断
- [ ] 波特率是否匹配
- [ ] TX/RX是否交叉
- [ ] 电平是否匹配（3.3V/5V）
- [ ] 地线是否共地
