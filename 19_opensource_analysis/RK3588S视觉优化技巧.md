# RK3588S 视觉优化技巧

## 1. 硬件架构概述

### 1.1 NPU 算力
- **RK3588S NPU**: 6 TOPS INT8 算力，三核架构
- 支持 INT8/INT16/FP16 多精度推理
- 支持 PyTorch/ONNX/TensorFlow/TFLite 模型转换

### 1.2 GPU 架构
- Mali-G610 MP4 GPU
- 支持 OpenCL 3.0、Vulkan 1.1
- 可用于图像预处理和后处理加速

### 1.3 ISP 处理器
- 硬件 ISP，支持 48MP 图像处理
- 支持 HDR、3A（AE/AF/AWB）算法
- 集成畸变矫正硬件模块

## 2. 图像采集优化

### 2.1 V4L2 采集优化
```c
// 设置为 MMAP 模式，减少内存拷贝
struct v4l2_requestbuffers req = {
    .count = 4,
    .type = V4L2_BUF_TYPE_VIDEO_CAPTURE,
    .memory = V4L2_MEMORY_MMAP
};
ioctl(fd, VIDIOC_REQBUFS, &req);

// 设置帧率
struct v4l2_streamparm parm = {
    .type = V4L2_BUF_TYPE_VIDEO_CAPTURE,
    .parm.capture = {
        .timeperframe = { .numerator = 1, .denominator = 60 }
    }
};
ioctl(fd, VIDIOC_S_PARM, &parm);
```

### 2.2 零拷贝方案
- 使用 DMA-BUF 实现 GPU/NPU 零拷贝
- V4L2 buffer → DRM buffer → RGA/NPU 直接访问
- 避免 CPU 参与数据搬运

## 5. NPU 推理优化

### 5.1 模型转换
```bash
# ONNX 转 RKNN
python - << 'EOF'
from rknn.api import RKNN
rknn = RKNN()
rknn.config(mean_values=[123.675, 116.28, 103.53],
            std_values=[58.395, 57.12, 57.375],
            target_platform='rk3588')
rknn.load_onnx(model='yolov8n.onnx')
rknn.build(do_quantization=True, dataset='calibration_list.txt')
rknn.export_rknn('yolov8n.rknn')
EOF
```

### 5.2 多核并行
```python
# 三核 NPU 并行推理
rknn.init_runtime(core_mask=RKNN.NPU_CORE_0_1_2)

# 不同模型分配不同核心
rknn_a.init_runtime(core_mask=RKNN.NPU_CORE_0)  # 检测模型
rknn_b.init_runtime(core_mask=RKNN.NPU_CORE_1)  # 分类模型
rknn_c.init_runtime(core_mask=RKNN.NPU_CORE_2)  # 跟踪模型
```

### 5.3 量化策略
- **INT8 量化**: 默认选择，性能最高，精度损失 <1%
- **混合量化**: 敏感层 FP16，其余 INT8
- **量化感知训练 (QAT)**: 精度最优方案

### 5.4 推理流水线优化
```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 采集T1  │───>│ 预处理T2│───>│ NPU推理T3│───>│ 后处理T4│
└─────────┘    └─────────┘    └─────────┘    └─────────┘
     │              │              │              │
     └──── Frame N-1 ────────────────────────────┘
     └──── Frame N   ────────────────────────────┘
     └──── Frame N+1 ────────────────────────────┘
  流水线并行，整体延迟 = max(T1,T2,T3,T4)
```

                        __global int* hist,
                        int width, int height) {

### 7.2 内存池复用
```python
class BufferPool:
    def __init__(self, buf_size, pool_size=8):
        self.pool = [bytearray(buf_size) for _ in range(pool_size)]
        self.available = list(range(pool_size))
    
    def acquire(self):
        return self.pool[self.available.pop()]
    
    def release(self, idx):
        self.available.append(idx)
```

## 9. 电赛实战技巧

### 9.1 实时性要求
- 目标帧率 ≥ 30fps → 每帧预算 33ms
- 建议分配：采集 5ms + 预处理 5ms + 推理 15ms + 后处理 5ms + 余量 3ms

### 9.2 功耗控制
- 降低 CPU 频率（视觉计算交给 NPU）
- 关闭未使用的外设
- 使用 DVFS 动态调频

### 9.3 稳定性保障
- 添加看门狗定时器
- 异常帧检测与跳过
- 温度过高时降频保护
