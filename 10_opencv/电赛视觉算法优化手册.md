## 1. 优化总原则

### 1.1 优化优先级（ROI由高到低）

```
1. 减少数据量    → 降采样、ROI裁剪 (收益最大)
2. 减少色彩转换  → YUV直接处理、复用灰度图
3. 减少内存分配  → 预分配缓冲区、原地操作
4. 减少函数调用  → 合并形态学操作、向量化
5. 利用硬件加速  → NEON、OpenCL、RGA
```

### 1.2 黄金法则

```
❌ "先写对再优化" → ✅ "从一开始就用正确的方式"
❌ "全图处理更准确" → ✅ "ROI是精度和速度的最佳平衡"
❌ "GPU一定比CPU快" → ✅ "只有大规模并行计算GPU才有优势"
❌ "所有帧都要处理" → ✅ "跳帧处理+结果缓存"
```

## 2. 图像采集优化

### 2.1 V4L2 摄像头配置

```python
# ❌ 错误: 默认配置
cap = cv2.VideoCapture(0)

# ✅ 正确: 优化配置
cap = cv2.VideoCapture(0, cv2.CAP_V4L2)
cap.set(cv2.CAP_PROP_FOURCC, cv2.VideoWriter_fourcc(*'YUYV'))  # 直接YUYV
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 640)   # 足够分辨率即可
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)
cap.set(cv2.CAP_PROP_FPS, 30)
cap.set(cv2.CAP_PROP_BUFFERSIZE, 1)       # 最小缓冲，减少延迟
```

**解释**:
- `CAP_V4L2` 比默认后端更高效
- `YUYV` 避免 MJPEG 解码开销（CPU省 2-3ms）
- `BUFFERSIZE=1` 避免读到旧帧

### 2.2 跳帧策略

```python
# ❌ 错误: 读取每一帧
while True:
    ret, frame = cap.read()  # 可能读到旧帧
    process(frame)

# ✅ 正确: 连续读取丢弃中间帧
while True:
    cap.grab()       # 快速跳过一帧（不解码）
    ret, frame = cap.read()  # 读取最新帧
    process(frame)
```

## 3. 色彩空间优化

### 3.1 避免不必要的色彩转换

```python
# ❌ 错误: 先转BGR再转HSV
frame_yuyv = capture_yuyv()
frame_bgr = cv2.cvtColor(frame_yuyv, cv2.COLOR_YUV2BGR_YUYV)  # 浪费!
hsv = cv2.cvtColor(frame_bgr, cv2.COLOR_BGR2HSV)  # 再浪费!
mask = cv2.inRange(hsv, lower, upper)

# ✅ 正确: YUV空间直接检测（如果可能）
frame_yuv = yuyv_to_yuv(frame_yuyv)  # 零拷贝提取Y/U/V
mask = yuv_inrange(frame_yuv, lower_yuv, upper_yuv)  # numpy向量化
```

### 3.2 YUV vs HSV 选择

```
输入格式    → 推荐色彩空间 → 原因
──────────────────────────────────────
YUYV/NV12  → YUV          → 零转换开销
BGR        → HSV          → 人机视觉直观
灰度图     → 灰度         → 已是单通道
```

### 3.3 HSV 范围优化

```python
# ❌ 错误: 跨0/180度的红色需要两次检测
lower_red1 = np.array([0, 80, 80])
upper_red1 = np.array([10, 255, 255])
lower_red2 = np.array([160, 80, 80])
upper_red2 = np.array([179, 255, 255])
mask = cv2.inRange(hsv, lower_red1, upper_red1) | cv2.inRange(hsv, lower_red2, upper_red2)

# ✅ 正确: 只检测V通道(亮度)+S通道(饱和度)即可区分红色
# 红色特征: 高S + 高V + H在两端
# 如果只需要"有红色"而非精确H范围，只用S+V更简单快速
mask_s = cv2.inRange(hsv[:,:,1], 100, 255)  # 高饱和度
mask_v = cv2.inRange(hsv[:,:,2], 100, 255)  # 高亮度
mask = cv2.bitwise_and(mask_s, mask_v)
```

## 5. 颜色检测优化

### 5.3 颜色标定工具

```python
def calibrate_color(frame, roi):
    """ROI区域自动标定颜色阈值"""
    x, y, w, h = roi
    roi_img = frame[y:y+h, x:x+w]
    hsv_roi = cv2.cvtColor(roi_img, cv2.COLOR_BGR2HSV)
    
    mean = cv2.mean(hsv_roi)[:3]
    std = np.std(hsv_roi.reshape(-1, 3), axis=0)
    
    # 均值 ± 2.5σ 覆盖约99%的样本
    lower = np.array([max(0, int(mean[i] - 2.5 * std[i])) for i in range(3)])
    upper = np.array([min(255, int(mean[i] + 2.5 * std[i])) for i in range(3)])
    
    return lower, upper
```

## 6. 轮廓与形状优化

### 6.1 轮廓检索模式选择

```python
# ❌ 检索所有轮廓（包括嵌套）
contours, _ = cv2.findContours(mask, cv2.RETR_TREE, cv2.CHAIN_APPROX_SIMPLE)

# ✅ 只检索外部轮廓（电赛通常只需要外轮廓）
contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
```

**RETR 模式对比**:

| 模式 | 说明 | 速度 | 适用场景 |
|------|------|------|----------|
| RETR_EXTERNAL | 只有外轮廓 | ⚡最快 | 一般检测 |
| RETR_LIST | 所有轮廓，不建立层级 | 快 | 不需要层级 |
| RETR_TREE | 所有轮廓+完整层级 | 慢 | 嵌套轮廓分析 |
| RETR_CCOMP | 2级层级 | 中等 | 孔洞检测 |

### 6.2 轮廓近似方法

```python
# CHAIN_APPROX_SIMPLE: 只存端点（矩形只需4点而非所有边界点）
contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# ❌ CHAIN_APPROX_NONE: 存所有点（内存大、处理慢）
contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_NONE)
```

### 6.3 面积粗筛

```python
# ✅ 在精细分类前先面积粗筛
min_area = 500
max_area = 100000
valid_contours = []
for cnt in contours:
    area = cv2.contourArea(cnt)
    if min_area < area < max_area:
        valid_contours.append(cnt)
```

### 6.4 多边形逼近技巧

```python
# epsilon 参数决定了逼近精度
perimeter = cv2.arcLength(cnt, True)

# epsilon = 周长的 2-4% 是经验值
epsilon = 0.03 * perimeter
approx = cv2.approxPolyDP(cnt, epsilon, True)
vertices = len(approx)

# 根据顶点数判断形状
if vertices == 3:
    shape = "triangle"
elif vertices == 4:
    # 进一步判断正方形 vs 矩形
    rect = cv2.minAreaRect(cnt)
    (_, (w, h), _) = rect
    aspect = min(w, h) / max(w, h)
    shape = "square" if aspect > 0.85 else "rectangle"
elif vertices == 5:
    shape = "pentagon"
elif vertices == 6:
    shape = "hexagon"
else:
    # 检查圆度
    circularity = 4 * np.pi * area / (perimeter * perimeter)
    shape = "circle" if circularity > 0.85 else "polygon"
```

## 7. 直线检测优化

### 7.1 HoughLinesP 参数调优

```python
# ❌ 默认参数（太慢，检测太多）
lines = cv2.HoughLinesP(edges, 1, np.pi/180, 50)

# ✅ 优化参数
lines = cv2.HoughLinesP(
    edges,
    rho=1,              # 距离分辨率(像素)
    theta=np.pi/180,    # 角度分辨率(弧度)
    threshold=50,       # 累加器阈值(越高越严格)
    minLineLength=50,   # 最短线段长度(过滤短线)
    maxLineGap=15,      # 最大允许间隙(连接断线)
)
```

**参数调优指南**:

| 参数 | 增大效果 | 减小效果 | 推荐值 |
|------|---------|---------|--------|
| threshold | 减少误检，可能漏检 | 增加检测，增加误检 | 30-80 |
| minLineLength | 过滤短线 | 保留短线 | 30-100 |
| maxLineGap | 连接更多断线 | 断线独立 | 5-20 |

### 7.2 自适应 Canny

### 7.3 角度过滤

```python
# ✅ 只保留特定角度范围的直线
def filter_by_angle(lines, target_angle, tolerance=15):
    """过滤掉偏离目标角度太多的直线"""
    filtered = []
    for line in lines:
        x1, y1, x2, y2 = line[0]
        angle = np.degrees(np.arctan2(y2-y1, x2-x1)) % 180
        if abs(angle - target_angle) < tolerance or abs(angle - target_angle - 180) < tolerance:
            filtered.append(line)
    return filtered

# 只保留水平线 (角度≈0或180)
horizontal = filter_by_angle(lines, 0, 15)
# 只保留垂直线 (角度≈90)
vertical = filter_by_angle(lines, 90, 15)
```

## 8. 模板匹配优化

### 8.1 金字塔加速

```python
def pyramid_match(frame, template, levels=3):
    """金字塔加速模板匹配"""
    # 构建搜索图像金字塔
    search_pyramid = [frame]
    for _ in range(levels - 1):
        search_pyramid.append(cv2.pyrDown(search_pyramid[-1]))
    
    # 构建模板金字塔
    tmpl_pyramid = [template]
    for _ in range(levels - 1):
        tmpl_pyramid.append(cv2.pyrDown(tmpl_pyramid[-1]))
    
    # 从顶层开始粗搜索
    result = cv2.matchTemplate(search_pyramid[-1], tmpl_pyramid[-1], cv2.TM_CCOEFF_NORMED)
    _, max_val, _, max_loc = cv2.minMaxLoc(result)
    
    # 逐层细化
    for level in range(levels - 2, -1, -1):
        # 坐标映射到当前层
        x = max_loc[0] * 2
        y = max_loc[1] * 2
        
        # 在候选区域±偏移范围内精确搜索
        margin = 10
        x1 = max(0, x - margin)
        y1 = max(0, y - margin)
        region = search_pyramid[level][y1:y1+tmpl_pyramid[level].shape[0]+2*margin,
                                        x1:x1+tmpl_pyramid[level].shape[1]+2*margin]
        
        result = cv2.matchTemplate(region, tmpl_pyramid[level], cv2.TM_CCOEFF_NORMED)
        _, max_val, _, max_loc = cv2.minMaxLoc(result)
        max_loc = (max_loc[0] + x1, max_loc[1] + y1)
    
    return max_loc, max_val
```

### 8.2 多尺度匹配

```python
def multi_scale_match(frame, template, scale_range=(0.5, 1.5), steps=10):
    """多尺度模板匹配"""
    best_score = 0
    best_loc = None
    best_scale = 1.0
    
    gray_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    gray_tmpl = cv2.cvtColor(template, cv2.COLOR_BGR2GRAY) if len(template.shape) == 3 else template
    
    for scale in np.linspace(scale_range[0], scale_range[1], steps):
        # 缩放模板
        w = int(gray_tmpl.shape[1] * scale)
        h = int(gray_tmpl.shape[0] * scale)
        if w > gray_frame.shape[1] or h > gray_frame.shape[0]:
            continue
        
        scaled = cv2.resize(gray_tmpl, (w, h))
        result = cv2.matchTemplate(gray_frame, scaled, cv2.TM_CCOEFF_NORMED)
        _, max_val, _, max_loc = cv2.minMaxLoc(result)
        
        if max_val > best_score:
            best_score = max_val
            best_loc = max_loc
            best_scale = scale
    
    return best_loc, best_score, best_scale
```

### 8.3 使用 UMat 加速

## 9. 二维码识别优化

### 9.1 预处理增强

```python
def preprocess_for_qr(gray):
    """二维码预处理增强"""
    # 1. CLAHE 对比度增强
    clahe = cv2.createCLAHE(clipLimit=3.0, tileGridSize=(4, 4))
    enhanced = clahe.apply(gray)
    
    # 2. 锐化
    kernel = np.array([[-1,-1,-1], [-1,9,-1], [-1,-1,-1]])
    sharpened = cv2.filter2D(enhanced, -1, kernel)
    
    return sharpened
```

### 9.2 多策略渐进检测

```python
def detect_qr_multi_strategy(frame):
    """多策略QR检测（从快到慢渐进）"""
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    detector = cv2.QRCodeDetector()
    
    # 策略1: 原图直接检测（最快）
    data, pts, _ = detector.detectAndDecode(gray)
    if data:
        return data, pts
    
    # 策略2: 对比度增强
    clahe = cv2.createCLAHE(clipLimit=2.0)
    enhanced = clahe.apply(gray)
    data, pts, _ = detector.detectAndDecode(enhanced)
    if data:
        return data, pts
    
    # 策略3: 二值化
    _, binary = cv2.threshold(enhanced, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    data, pts, _ = detector.detectAndDecode(binary)
    if data:
        return data, pts
    
    # 策略4: 反色
    inverted = 255 - binary
    data, pts, _ = detector.detectAndDecode(inverted)
    if data:
        return data, pts
    
    return None, None
```

## 11. 多线程与流水线

### 11.1 生产者-消费者模式

```python
import threading
import queue

class CameraThread(threading.Thread):
    """独立线程采集摄像头"""
    def __init__(self, cam_id=0):
        super().__init__(daemon=True)
        self.cap = cv2.VideoCapture(cam_id)
        self.frame_queue = queue.Queue(maxsize=2)
        self.running = True
    
    def run(self):
        while self.running:
            ret, frame = self.cap.read()
            if ret:
                if self.frame_queue.full():
                    self.frame_queue.get()  # 丢弃旧帧
                self.frame_queue.put(frame)
    
    def get_frame(self):
        return self.frame_queue.get(timeout=1.0)

# 使用
cam = CameraThread(0)
cam.start()

while True:
    frame = cam.get_frame()  # 总是获取最新帧
    result = process(frame)
```
