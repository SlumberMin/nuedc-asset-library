## 3. 综合选型建议

### 3.1 电赛场景推荐

| 场景 | 推荐模型 | 输入分辨率 | 预期FPS | 说明 |
|------|----------|-----------|---------|------|
| 巡线检测 | PP-PicoDet-XS | 320×320 | 300+ | 最快速度 |
| 通用目标检测 | YOLOv8n | 640×640 | 60-80 | 精度最佳 |
| 实时检测(平衡) | PP-PicoDet-S | 416×416 | 200+ | 速度精度平衡 |
| 多目标跟踪 | YOLOv5n | 416×416 | 120+ | 与跟踪器配合好 |
| 简单分类 | MobileNetV3-S | 224×224 | 800+ | 分类首选 |
| 远距离检测 | YOLOv8n | 640×640 | 60+ | 小目标能力 |

### 3.2 不同精度需求

```
极低延迟 (<5ms):
  → PP-PicoDet-XS 320×320
  → NanoDet 320×320
  → MobileNetV3-S 224×224 (分类)

中等延迟 (5-15ms):
  → YOLOv5n 416×416
  → PP-PicoDet-S 416×416
  → YOLOv8n 416×416

高精度 (>15ms可接受):
  → YOLOv8n 640×640
  → YOLOv5s 640×640
```

## 4. 模型转换注意事项

### 4.1 量化建议

| 模型 | 推荐量化方式 | 校准图片数 | 注意事项 |
|------|-------------|-----------|---------|
| YOLOv8n | INT8, asymmetric | 200+ | 输出head不量化 |
| YOLOv5n | INT8, asymmetric | 200+ | 正常量化 |
| PP-PicoDet | INT8, asymmetric | 300+ | Paddle导出需特殊处理 |
| MobileNetV3 | INT8, channel | 100+ | 精度损失小 |

### 4.2 输入格式

| 模型 | 归一化方式 | 通道顺序 | 值范围 |
|------|-----------|---------|--------|
| YOLOv8n | /255 | RGB | 0-1 |
| YOLOv5n | /255 | RGB | 0-1 |
| PP-PicoDet | ImageNet | RGB | 标准化 |
| MobileNetV3 | ImageNet | RGB | 标准化 |

## 5. 测试环境说明

- **硬件**: RK3588S 开发板 (8GB LPDDR4x)
- **系统**: Ubuntu 22.04, Linux 5.10
- **RKNN版本**: rknn-toolkit2 2.x / rknn-lite2 2.x
- **NPU模式**: 3核联合推理
- **测试方法**: 100次推理取平均值，去除前10次预热

> ⚠️ 实际性能可能因系统负载、散热条件、电源质量等因素有±10%波动。
