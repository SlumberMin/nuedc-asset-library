# 电机驱动电路设计指南

## 一、概述

本文档涵盖电赛常用电机驱动方案：TB6612（小功率DC）、L298N（经典H桥）、IR2104（半桥MOSFET驱动）、DRV8313（三相BLDC驱动）。

---

## 二、TB6612 双路H桥驱动

### 2.1 适用场景
- 小型直流减速电机（额定电流 ≤1.2A，峰值3.2A）
- 适合四轮小车、舵机云台等

### 2.2 电路原理图（ASCII）

```
                    VM (电机电源 4.5~13.5V)
                         |
                    [C1 100uF]---GND
                         |
MCU PWM ──[1K]──> PWMA   TB6612FNG
MCU GPIO ───────> AIN1 ┌─────────────┐
MCU GPIO ───────> AIN2 │             ├──> AOUT1 ──> 电机A+
              STBY ──>H│             ├──> AOUT2 ──> 电机A-
MCU PWM ──[1K]──> PWMB │             │
MCU GPIO ───────> BIN1 │             ├──> BOUT1 ──> 电机B+
MCU GPIO ───────> BIN2 │             ├──> BOUT2 ──> 电机B-
                         │  GND  VM  VCC │
                         └──┼────┼────┼──┘
                            │    │    │
                           GND   │  [C2 100nF]──GND
                                 |
                              VM电源

VCC = 3.3V 或 5V (逻辑电源)
GND 公共连接
```

### 2.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| TB6612FNG | 双H桥 | TB6612FNG | 封装SSOP24 |
| 去耦电容C1 | 100μF/16V | 铝电解电容 | 靠近VM引脚 |
| 去耦电容C2 | 100nF | 贴片0603 | 靠近VCC引脚 |
| 限流电阻 | 1KΩ | 0603贴片 | PWM串联可选 |

### 2.4 PCB布局建议
- VM去耦电容紧贴芯片VM引脚
- 大电流走线（AOUT/BOUT）宽度 ≥ 1mm
- 逻辑地和功率地单点汇合
- 散热焊盘需要过孔到背面散热铜皮

## 三、L298N 双路全桥驱动

### 3.1 适用场景
- 中功率直流电机（2A/路，峰值3A）
- 经典方案，外围简单，压降较大（~2V）

### 3.2 电路原理图（ASCII）

```
           VMS (电机电源 5~46V)
               |
          [C3 470uF/50V]──GND
               |
           ┌───┴───┐
           │ L298N │
MCU PWM──>│ ENA    OUT1├──> 电机A+
MCU GPIO->│ IN1    OUT2├──> 电机A-
MCU GPIO->│ IN2        │
           │        OUT3├──> 电机B+
MCU GPIO->│ IN3    OUT4├──> 电机B-
MCU GPIO->│ IN4        │
MCU PWM──>│ ENB    VSS ├──> 5V (逻辑)
           │ GND   VS  ├──> VMS
           └───┬───┘
               │
              GND

续流二极管(每路4个):
  OUT1 ──[D1 IN5819]──> VMS
  OUT1 ──[D2 IN5819]──> GND
  OUT2 ──[D3 IN5819]──> VMS
  OUT2 ──[D4 IN5819]──> GND
  (OUT3/OUT4同理)
```

### 3.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| L298N模块 | 双H桥 | L298N封装模块 | 带散热片 |
| 续流二极管 | 1A/40V | SS14 / IN5819 | 每桥臂4个 |
| 滤波电容 | 470μF/50V | 铝电解 | 电机电源端 |
| 去耦电容 | 100nF | 瓷片 | VSS端 |

### 3.4 PCB布局建议
- 续流二极管尽量贴近电机输出引脚
- 大面积铺铜散热
- L298N压降大（~2V），注意电机实际得到的电压

## 四、IR2104 半桥MOSFET驱动

### 4.1 适用场景
- 大电流直流电机（取决于MOSFET选型，轻松10A+）
- 高效率，低压降
- 适合四轮驱动、大功率场合

### 4.2 电路原理图（ASCII）

```
         VBAT (7.4V/11.1V 锂电池)
              |
         [C_bulk 470uF]──GND
              |
    ┌─────────┤
    │     D (漏极)
    │  Q1 [NMOS]  IRF3205/IRLR7843
    │     S (源极)──┬──> 电机+
    │              │
    │         [D1 自举二极管 SS14]
    │              │        │
    │         VB───┘   [C_boot 100nF]
    │              │        │
    │         HO───┘        ├──────┐
    │              │        │      │
MCU_H ──> IN   IR2104  VS──┘      │
MCU_L ──> SD        │             │
         LO──> G [Q2 NMOS]        │
              D──> GND             │
              S──> GND             │
                                   │
    电机- ────────────────────────┘

自举电路:
  VB = VS + VCC (通过D1自举充电)
  C_boot 在HO导通期间为高端供电
```

### 4.3 关键设计要点
1. **自举电容**：C_boot ≥ 100nF，陶瓷电容，靠近VB-VS引脚
2. **自举二极管**：快恢复二极管（SS14/UF4007），反向耐压 > VCC
3. **MOSFET选型**：Rds(on)要低，Qg要小，Vgs(th) < 3V（兼容3.3V MCU）
4. **死区时间**：IR2104内置死区（~540ns），无需外加

### 4.4 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| 高端MOSFET | 55V/110A | IRF3205 | Rds=8mΩ |
| 低端MOSFET | 55V/110A | IRF3205 | 同型号 |
| 半桥驱动IC | 600V | IR2104 | SOIC-8 |
| 自举二极管 | 1A/100V | SS14 | 快恢复 |
| 自举电容 | 100nF/50V | 贴片陶瓷 | X7R |
| 大电容 | 470μF/25V | 铝电解 | 电源端 |
| 栅极电阻 | 10Ω | 0805 | 抑制振铃 |

### 4.5 PCB布局建议
- 高频功率回路（Q1-S-D1-C_boot-VS）面积最小化
- 栅极驱动走线尽量短，串联10Ω电阻防振铃
- 大电流铜箔或铺铜，宽度≥2mm
- 散热过孔阵列在MOSFET焊盘下方

## 五、DRV8313 三相BLDC驱动

### 5.1 适用场景
- 三相无刷直流电机（BLDC/PMSM）
- 持续电流2.5A/相，峰值4A
- 适合无人机电调、BLDC直驱

### 5.2 电路原理图（ASCII）

```
         VM (8~60V)
              |
         [C_bulk 100uF]──GND
              |
     ┌────────┴────────────────┐
     │        DRV8313           │
     │  VM  OUTA ──> 电机U相    │
MCU_A──> nFAULT │  OUTB ──> 电机V相    │
MCU_B──> nRESET │  OUTC ──> 电机W相    │
MCU_C──> AIN1  │                     │
MCU_D──> AIN2  │  ENBL ──> 3.3V      │
MCU_E──> BIN1  │  nSLEEP──> 3.3V     │
MCU_F──> BIN2  │  VCP ──[C_cp 100nF] │
MCU_G──> CIN1  │  CP1 ──[C_pump] CP2 │
MCU_H──> CIN2  │  VINT ──[C_int]     │
     │  GND    AVDD──> 3.3V          │
     └────────────────────────────┘

nFAULT ──[R 10K]──> 3.3V (上拉)
```

### 5.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| DRV8313 | 三相半桥 | DRV8313 | QFN/TSSOP |
| VM滤波电容 | 100μF/63V | 铝电解 | 电源端 |
| VCP电容 | 100nF/16V | 陶瓷X7R | 充电泵 |
| CP泵电容 | 1μF/16V | 陶瓷 | CP1-CP2 |
| VINT电容 | 1μF/16V | 陶瓷 | 内部稳压 |
| AVDD电容 | 1μF | 陶瓷 | 模拟电源 |

### 5.4 PCB布局建议
- VM大电容紧贴电源引脚
- 电机输出走线宽而短
- 充电泵电容（VCP/CP）紧贴芯片
- 散热焊盘打过孔，背面大面积铜皮

## 六、常见问题解答

### Q1: 电机启动时MCU复位？
**原因**：电机启动电流大，电源跌落导致MCU欠压。
**解决**：
- 电机电源和MCU电源分开（独立稳压）
- 加大电源端滤波电容（470μF+100nF组合）
- 软件PWM软启动（逐渐增大占空比）

### Q2: MOSFET发热严重？
**排查**：
- Rds(on)是否过大 → 换低Rds(on)的MOSFET
- 栅极驱动电压不足 → 确保Vgs > 10V（用专用驱动IC）
- PWM频率太高 → 降低到20kHz左右
- 死区不足导致直通 → 增加死区时间

### Q3: 电机转动时有异响？
**原因**：PWM频率落在人耳可听范围（20Hz~20kHz）。
**解决**：PWM频率设到18kHz以上（推荐20~25kHz）。

### Q4: L298N驱动效率低、发热大？
**原因**：L298N内部达林顿管压降约2V。
**解决**：改用MOSFET方案（IR2104+MOSFET），效率提升显著。

### Q5: 自举电路不能正常工作？
**排查**：
- 自举二极管是否用了快恢复型号
- 自举电容是否足够大（≥100nF）
- 低端是否先开通让自举电容充电
- VS引脚焊接是否良好
