# 电源模块PCB设计指南

> 电赛电源模块设计规范：LDO、DC-DC、多路电源

---

## 1. 电源架构设计

### 1.1 典型电赛电源架构

```
电池(7.4V/11.1V/12V)
    │
    ├── DC-DC(降压) ── 5V ── 电机驱动/传感器
    │
    ├── LDO ── 3.3V ── MCU/传感器/通信模块
    │
    └── 可调DC-DC ── 输出(题目要求) ── 负载
```

### 1.2 电源选型决策树

```
输入电压 > 输出电压?
├── 是 → 降压(Buck)
│   ├── <500mA → LDO (简单/低噪声)
│   ├── <3A → LM2596 / XL4015
│   └── >3A → TPS54331 / MP2315
└── 否 → 升压(Boost)
    ├── <1A → MC34063
    └── >1A → XL6009 / TPS61088
```

## 2. LDO设计

### 2.1 AMS1117-3.3 设计

**原理图：**
```
VIN(5V) ──┬──[C1: 10µF]── GND
           │
        ┌──┴──┐
        │AMS  │
        │1117 │
        │-3.3 │
        └──┬──┘
           │
VOUT(3.3V)─┬──[C2: 22µF]── GND
           │
           └── 到负载
```

**PCB布局要求：**

| 参数 | 要求 |
|------|------|
| 输入电容C1 | 10µF MLCC，紧贴VIN引脚 |
| 输出电容C2 | 22µF MLCC(必须ESR>0.1Ω)，紧贴VOUT引脚 |
| GND引脚 | 大面积铺铜散热 |
| 散热 | 底部散热焊盘铺铜，加散热过孔 |
| 走线 | 输入/输出线≥20mil |

**散热过孔设计：**
```
       ┌───────────┐
       │  AMS1117  │
       │           │
       └──┬─┬─┬──┘
          ○ ○ ○  ← 0.3mm散热过孔×4~6
       ─────────────  底层大面积铺铜
```

### 2.2 注意事项

- AMS1117压差1.1V，5V→3.3V可以，3.3V→2.5V可能不够
- 输出电容ESR太低可能振荡，MLCC要注意
- 替代方案：RT9013(压差250mV)、ME6211(压差100mV)

## 3. Buck降压设计

### 3.1 LM2596-5.0 设计

**原理图：**
```
         D1(SS34)
VIN ──┬──┐  ┌──┬── L1(33µH) ──┬── VOUT(5V)
      │  └──┘  │               │
   [C_in]      │            [C_out]
   100µF       │            220µF
      │        │               │
     GND      GND             GND

LM2596引脚:
  VIN ── 输入
  GND ── 地
  OUT ── 开关输出(接L1和D1)
  FB  ── 反馈(输出分压到1.23V)
  ON/OFF ── 使能(接VIN)
```

**关键元件选型：**

| 元件 | 推荐值 | 要求 |
|------|--------|------|
| C_in | 100µF/50V 电解 + 100nF MLCC | 低ESR |
| C_out | 220µF/16V 电解 + 100nF MLCC | 低ESR |
| L1 | 33µH/3A | 饱和电流>输出电流×1.3 |
| D1 | SS34(肖特基) | 反向耐压>输入电压 |

**PCB布局：**

```
顶层布局:
┌─────────────────────────────────────┐
│                                     │
│  [C_in]  ┌────────┐   [L1]  [C_out]│
│          │LM2596  │                │
│          └────────┘       [D1]     │
│                                     │
│  VIN─────────────────────── VOUT   │
│  GND ████████████████████████ GND  │
└─────────────────────────────────────┘

关键: 功率环路(C_in→LM2596→L1→C_out→GND→C_in)面积最小
```

**走线要求：**

| 网络 | 线宽 | 说明 |
|------|------|------|
| VIN | ≥30mil | 输入电源 |
| VOUT | ≥30mil | 输出电源 |
| SW(开关节点) | ≥30mil | 尽量短，不要铺铜 |
| GND | 铺铜 | 大面积铺铜 |
| FB(反馈) | 8mil | 远离开关节点 |

---

### 3.2 MP2315 同步Buck (推荐)

**优势：** 3A输出，效率96%，SOT-23-8封装，集成MOSFET

**关键参数：**
- 输入：4.5~24V
- 输出：0.6~23V(可调)
- 开关频率：500kHz~2MHz
- 外部元件少：只需电感+二极管(可选)+电容

**布局要点：**
- 功率环路面积最小化
- BST电容紧贴BST和SW引脚
- 输入电容紧贴VIN和GND引脚

## 4. Boost升压设计

### 4.1 XL6009 设计

**原理图：**
```
VIN ──┬──[C_in: 22µF]── GND
      │
   ┌──┴──┐
   │XL6009│
   └──┬──┘
      │── SW ──┬── L1(33µH) ── VIN
      │        │
      │     D1(SS54)
      │        │
      │        └──┬── VOUT
      │        [C_out: 220µF]
      │           │
      └────────── GND
```

**PCB要点：**
- 输入电容必须紧贴VIN脚
- 输出电容必须紧贴输出二极管
- SW节点尽量短，避免大面积铺铜(天线效应)
- 反馈分压电阻靠近FB引脚

## 5. 多路电源设计

### 5.1 典型多路电源方案

```
电池7.4V
    │
    ├── DC-DC ── 12V (升压，用于电机)
    │
    ├── DC-DC ── 5V (降压，用于逻辑/传感器)
    │    │
    │    └── LDO ── 3.3V (用于MCU)
    │
    └── 可调DC-DC ── 题目要求输出
```

### 5.2 电源树PCB布局

```
┌─────────────────────────────────────────┐
│                                         │
│  [电池接口]                              │
│      │                                  │
│      ├── [Buck: 5V] ── [LDO: 3.3V]     │
│      │                                  │
│      ├── [Boost: 12V]                   │
│      │                                  │
│      └── [可调DC-DC]                    │
│                                         │
│  [输出接口]                              │
└─────────────────────────────────────────┘
```

**关键布局原则：**
- 各路电源物理分区
- GND星形汇聚，避免大电流流过敏感信号地线
- 可调输出的反馈电阻远离其他电源

## 6. 保护电路设计

### 6.1 输入保护

```
电池 ──[自恢复保险丝]──[TVS]──┬── 到电源电路
                              │
                           [C_bulk]
                              │
                             GND
```

| 元件 | 选型建议 |
|------|----------|
| 自恢复保险丝 | 额定电流=正常工作电流×1.5 |
| TVS | 钳位电压=最大输入电压×1.2 |
| C_bulk | 470µF/50V 电解 |

### 6.2 输出保护

```
VOUT ──[保险丝]──[TVS]── 到负载
         │
      [LED指示]──[限流电阻]── GND
```

### 6.3 防反接设计

**方案1：P-MOS防反接**
```
BAT+ ──┬── P-MOS(S极) ── D极 ── 到电路
       │
       └── R(100k) ── G极 ── BAT-
```

**方案2：二极管防反接(简单但有压降)**
```
BAT+ ── SS34(阳极→阴极) ── 到电路
```

## 7. PCB设计规范

### 7.1 线宽与电流

| 铜厚 | 10mil线宽电流 | 20mil | 30mil | 50mil | 100mil |
|------|--------------|-------|-------|-------|--------|
| 1oz | 0.3A | 0.6A | 0.8A | 1.2A | 2A |
| 2oz | 0.5A | 1A | 1.4A | 2A | 3.5A |

> 经验公式: I(A) ≈ 0.024 × ΔT^0.44 × W^0.725 (W为mil, ΔT为温升)

### 7.2 大电流铺铜技巧

```
推荐: 多边形铺铜
  ┌──────────────┐
  │              │
  │  ████████████│  ← 大面积铜皮
  │  ████████████│     额定电流>走线
  │  ████████████│
  └──────────────┘

加锡处理:
  ┌──────────────┐
  │══════════════│  ← 阻焊开窗+上锡
  │══════════════│     额外增加载流
  └──────────────┘
```

### 7.3 热设计

**LDO散热：**
- 底部散热焊盘
- 散热过孔: 0.3mm × 4~8个
- 底层大面积铺铜
- 估算功耗: P = (Vin - Vout) × Iout

**DC-DC散热：**
- 功率器件下方铺铜
- 电感选型注意直流电阻(DCR)
- 效率估算: P_loss = P_out × (1/η - 1)

## 8. 调试流程

### 8.1 焊接后检查

```
1. 目视检查 → 无虚焊/桥接
2. 万用表测电阻 → VIN-GND、VOUT-GND无短路
3. 空载上电 → 测量各路输出电压
4. 轻载测试 → 接小负载测电压稳定性
5. 满载测试 → 额定负载下测电压/纹波/温度
```

### 8.2 测试项目

| 测试项 | 工具 | 合格标准 |
|--------|------|----------|
| 输出电压 | 万用表 | 偏差<±2% |
| 纹波 | 示波器(AC耦合) | <50mV (数字电路) |
| 负载调整率 | 电子负载 | <3% (10%~100%负载) |
| 线性调整率 | 可调电源 | <2% (输入±10%) |
| 效率 | 万用表+电流表 | >80% (DC-DC) |
| 温升 | 热像仪/手摸 | <40°C (环境25°C) |

## 9. 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 输出纹波大 | 电容ESR高/布局差 | 用低ESR电容，缩短走线 |
| LDO发烫 | 压差×电流太大 | 换DC-DC或减小负载 |
| DC-DC不启动 | FB分压错误/EN未拉高 | 检查反馈电阻和使能引脚 |
| 输出电压不准 | 反馈电阻精度/走线电阻 | 用1%精度电阻，加粗走线 |
| 啸叫 | 电感饱和/频率在音频范围 | 换大电流电感/调频率 |
| MCU复位 | 电源干扰/电压跌落 | 加滤波/加大电容 |

## 10. 推荐设计参考

### 电赛常用电源板尺寸
- 简单单路: 30mm × 20mm
- 标准多路: 50mm × 40mm
- 复杂可调: 80mm × 60mm

### 推荐PCB打样参数
- 板厚: 1.6mm
- 铜厚: 1oz(标准) / 2oz(大电流)
- 最小线宽/线距: 6mil/6mil
- 表面处理: HASL
- 阻焊颜色: 绿色(便于观察焊接)
