# 电赛信号调理电路设计

> 放大 / 滤波 / 检波 / 采样保持 / ADC驱动 — 从信号源到数字化的完整链路

---

## 1. 信号调理链路总览

```
信号源 → 前端保护 → 放大/衰减 → 滤波 → 检波/变换 → 采样保持 → ADC驱动 → MCU
         (ESD/钳位)  (增益调节)  (去噪)  (AC→DC等)  (高速信号)  (阻抗匹配)
```

## 2. 信号放大电路

### 2.1 同相放大器

```
              Rf
         ┌───┤├───┐
         │         │
    Vin──┤+  ┌─┐  ├── Vout
         ├──┤Op├──┤
    GND──┤-  └─┘  │
         │   Rg   │
         └───┤├───┘
              │
             GND

Vout = Vin × (1 + Rf/Rg)
增益: Av = 1 + Rf/Rg

特点: 输入阻抗高，适合高阻传感器
```

**设计参数表：**

| 目标增益 | Rf | Rg | 实际增益 | 带宽(MHz) |
|----------|------|------|----------|-----------|
| ×2 | 10kΩ | 10kΩ | 2.00 | GBW/2 |
| ×5 | 40kΩ | 10kΩ | 5.00 | GBW/5 |
| ×10 | 90kΩ | 10kΩ | 10.0 | GBW/10 |
| ×100 | 99kΩ | 1kΩ | 100 | GBW/100 |
| ×1000 | 999kΩ | 1kΩ | 1000 | GBW/1000 |

### 2.2 反相放大器

```
         Rf
    ┌───┤├───┐
    │         │
    │   ┌─┐  │
Vin─┤├──┤Op├──┤── Vout
    Rg  └─┘  │
    │         │
    └─────┬───┘
          │
         GND（虚地）

Vout = -Vin × (Rf/Rg)
增益: Av = -Rf/Rg

特点: 输入阻抗=Rg，有反相
```

### 2.3 仪表放大器（差分放大）

```
       Rg
  ┌───┤├───┐
  │         │
  │  ┌─┐   │        R2
  ├─┤Op├─┐ │   ┌───┤├───┐
  │  └─┘  │ │   │         │
Vin+  R1  └─┤├──┤+  ┌─┐  ├── Vout
             │   ├──┤Op├──┤
Vin-  R1  ┌─┤├──┤-  └─┘  │
  │  ┌─┐  │ │   │   R2   │
  ├─┤Op├─┘ │   └───┤├───┘
  │  └─┘   │         │
  │         │        GND
  └───┤├───┘
       Rg

增益: G = 1 + 2R1/Rg
共模抑制比: CMRR > 100dB (使用匹配电阻)

经典芯片: AD620, INA128, INA333
```

**AD620 增益设定：**

| 增益 | Rg(Ω) | 精度要求 |
|------|--------|----------|
| 1 | ∞（不接） | - |
| 10 | 4.99k | 0.1% |
| 100 | 499 | 0.1% |
| 1000 | 49.9 | 0.1% |

### 2.4 程控增益放大器（PGA）

```
方案1: 运放 + 模拟开关 + 电阻网络

            R1=10k     R2=100k     R3=1M
    Vin──┬──┤├──┐ ┌──┤├──┐  ┌──┤├──┐
         │       │       │        │
         │  ┌────┘ ┌─────┘  ┌─────┘
         │  │      │        │
         │  SW1    SW2      SW3    ← CD4051/ADG1608
         │  │      │        │
         └──┴──────┴────────┴───┤+
                                 ├──┤Op├── Vout
                           Rf──┤-  └─┘
                           │
                          GND

增益选择:
  SW1闭合: Av = -10k/Rf   (×1)
  SW2闭合: Av = -100k/Rf  (×10)
  SW3闭合: Av = -1M/Rf    (×100)

方案2: 专用PGA芯片
  AD8253: G=1,10,100,1000 (数字选择)
  PGA112: G=1,2,4,8,...128 (SPI控制)
```

## 3. 滤波电路

### 3.1 一阶RC低通滤波

```
         R
Vin ──┤├──┬── Vout
            │
           ══ C
            │
           GND

截止频率: fc = 1/(2π·R·C)

常用参数:
  fc=10Hz:   R=16kΩ, C=1μF
  fc=100Hz:  R=16kΩ, C=100nF
  fc=1kHz:   R=16kΩ, C=10nF
  fc=10kHz:  R=1.6kΩ, C=10nF

特点: 简单，衰减慢(-20dB/dec)
```

### 3.2 二阶Sallen-Key低通滤波器

```
         R1        R2
Vin ──┤├──┬──┤├──┬──┬── Vout
            │       │   │
            │       ══C2│
            │       │   │
           ══C1    GND  │
            │           │
           GND     ┌────┘
                   │
              ┌─┐  │
              │Op├──┘
              └─┘
                   │
                  GND

截止频率: fc = 1/(2π·√(R1·R2·C1·C2))
品质因数: Q = √(R1·R2·C1·C2)/(C2·(R1+R2))

常用设计（fc=1kHz, Butterworth, Q=0.707）:
  R1 = R2 = 10kΩ
  C1 = 22nF, C2 = 10nF (近似)
  或使用 FilterPro 工具精确计算
```

### 3.3 二阶有源带通滤波器

```
             C1          R3
    ┌───┤├──┬──┤├──┐
    │         │       │
    │        ══C2    ┌┤+
    │         │      │├─┤Op├── Vout
Vin─┤├──R1──┘     └┤-  └─┘
    R2                  │
    │                  GND
   GND

中心频率: f0 = 1/(2π·√(R1·R3·C1·C2))
带宽: BW = f0/Q
增益: 取决于电阻比值

电赛典型应用:
  信号发生器题目的频率选择
  音频信号的特定频率提取
```

### 3.4 常用滤波器芯片

| 芯片 | 类型 | 截止频率 | 特点 |
|------|------|----------|------|
| MAX7400 | 8阶低通 | 1~10kHz | 时钟可调，椭圆滤波 |
| MAX7404 | 8阶低通 | 1~10kHz | Bessel响应 |
| LTC1064 | 8阶低通 | 20kHz~200kHz | 高速 |
| UAF42 | 通用有源 | 0.1Hz~100kHz | 高通/低通/带通 |

### 3.5 数字滤波（IIR/FIR）

```c
// 一阶IIR低通滤波器（指数移动平均）
float lowpass_filter(float input, float alpha, float prev_output) {
    // alpha = dt / (RC + dt), 0 < alpha < 1
    // alpha越小，滤波越强，延迟越大
    return alpha * input + (1 - alpha) * prev_output;
}

// 二阶IIR (Biquad) 直接I型
typedef struct {
    float b0, b1, b2;  // 分子系数
    float a1, a2;       // 分母系数 (a0归一化为1)
    float x1, x2;       // 输入历史
    float y1, y2;       // 输出历史
} BiquadFilter;

float biquad_process(BiquadFilter *f, float input) {
    float output = f->b0*input + f->b1*f->x1 + f->b2*f->x2
                   - f->a1*f->y1 - f->a2*f->y2;
    f->x2 = f->x1; f->x1 = input;
    f->y2 = f->y1; f->y1 = output;
    return output;
}

// 滑动平均滤波器（FIR，抗脉冲噪声）
#define WINDOW_SIZE 8
float moving_average(float *buffer, float new_sample) {
    static float sum = 0;
    static int idx = 0;
    static int count = 0;
    
    if (count >= WINDOW_SIZE) sum -= buffer[idx];
    buffer[idx] = new_sample;
    sum += new_sample;
    idx = (idx + 1) % WINDOW_SIZE;
    if (count < WINDOW_SIZE) count++;
    
    return sum / count;
}
```

## 4. 检波电路

### 4.1 峰值检波器（包络检波）

```
         D1 (肖特基)
Vin ──┤>|──┬── Vout_peak
            │
           ══ C (保持电容)
            │
           ─┘
           │
    ┌── R ─┘ (泄放电阻，控制下降速率)
    │
   GND

工作原理:
  正半周: D1导通，C充电至峰值
  负半周: D1截止，C通过R缓慢放电

设计要点:
  D1: BAT54S (肖特基二极管, Vf≈0.3V)
  C: 100nF~10μf (充电快/保持时间长)
  R: 控制包络跟踪速度
  充电时间常数: τ_charge = C × R_on(D1) < 信号周期
  放电时间常数: τ_discharge = R×C >> 信号周期
```

### 4.2 精密半波整流

```
         D1
    ┌──┤>|──┐
    │       │
    │  ┌─┐  │
Vin─┤├──┤Op├──┬──┤├── Vout
    R1  └─┘  │   R2
    │       ══
    │        │
   GND      GND

Vout = |Vin| (正半周放大，负半周为0)

消除了普通二极管0.7V的死区
```

### 4.3 精密全波整流（绝对值电路）

```
        R           R
Vin──┤├──┐   ┌──┤├──┐
          │   │        │
          │   │  ┌─┐   │
          └──┤├──┤Op1├─┘  (半波整流)
              │  └─┘      │
             R            │
              │   R    R  │
              └──┤├──┬─┤├─┘
                      │
                      │  ┌─┐
            ┌─────────├──┤Op2├── Vout = |Vin|
            │   R     │  └─┘
            └──┤├──┘
                │
               GND

Vout = |Vin| (全波整流，无死区)
应用: AC信号的幅值测量、RMS计算前端
```

### 4.4 包络检波实用电路（MC33078 + BAT54S）

```
              BAT54S
         ┌───┤>|──┐
         │         │
    ┌─┐  │  100nF  │
    │U1├──┘  ═══   ├──── Vout (包络)
    │  │           │
    └┬─┘     100kΩ │
     │         ┤├  │
     │           │  │
     └───────────┴──┘

适合: AM信号解调、RF信号包络提取
频率范围: 100Hz ~ 1MHz
```

## 5. 采样保持电路（S/H）

### 5.1 原理

```
                控制信号
                 │
    Vin ──┐      │     ┌── Vout → ADC
          │  ┌───┴──┐  │
          └──┤ SW   ├──┘
             │(模拟 │
             │ 开关) │
             └───┬──┘
                 │
                ══ Ch (保持电容)
                 │
                GND

采样阶段: SW闭合，Ch充电至Vin
保持阶段: SW断开，Ch电压保持不变，ADC采样
```

### 5.2 实用S/H电路（LF398）

```
           ┌────────────────┐
    Vin ──┤1  LF398      8├── +Vs (+15V)
           │               │
    Logic─┤7  HOLD/SAMPLE 3├── Vout → ADC
           │               │
   -Vs ──┤4             5├── Ch = 1000pF
  (-15V)  └────────────────┘
               │    │
              GND  GND

控制逻辑:
  Logic=低: 采样模式，输出跟踪输入
  Logic=高: 保持模式，输出保持最后值

采样时间: <4μs (Ch=1000pF)
保持下降率: <0.5mV/ms
孔径抖动: <150ps

应用: 高速ADC前端，多通道同步采样
```

### 5.3 多通道同步采样

```
                 ┌────────┐
    CH0 ───────┤ S/H #0 ├──┐
                 └────────┘  │
                 ┌────────┐  │    ┌──────┐
    CH1 ───────┤ S/H #1 ├──┼────┤ ADC  ├── MCU
                 └────────┘  │    └──────┘
                 ┌────────┐  │
    CH2 ───────┤ S/H #2 ├──┘
                 └────────┘
                     │
               同时采样信号

目的: 消除多通道采样时间差（相位差）
应用: 三相电压/电流同步测量
芯片: AD7606 (内置同步采样S/H)
```

## 6. ADC驱动电路

### 6.1 单端转差分驱动

```
很多高性能ADC是差分输入的（如ADS1256, AD7606）

         R                    ┌── AIN+
Vin ──┤├──┬───────────────┤
         │   │              │
        GND  │   ┌─┐       │
             └──┤Op├─┘      │
                └─┘         ┌── AIN-
                 │
                Vref/2 (共模电压)

或使用专用差分驱动器:
  ADA4940: 单端→差分，高CMRR
  THS4521: 高速差分驱动器
```

### 6.2 阻抗匹配与缓冲

```
高阻传感器输出 → 电压跟随器 → ADC

    传感器         缓冲器           ADC
    ┌─────┐      ┌─────────┐     ┌─────┐
    │ Rs  │      │  ┌─┐    │     │     │
    │很大 ├──────├──┤Op├────├─────┤ADC  │
    │>10kΩ│      │  └─┘    │     │     │
    └─────┘      │  │→Vout│     │ Rin │
                  └─────────┘     │10kΩ│
                                   └─────┘
电压跟随器: Av=1, Zin→∞, Zout→0
```

### 6.3 抗混叠滤波器（ADC前端必备）

```
              R
Vin ──┤├──┬── Vout → ADC
            │
           ══ C
            │
           GND

截止频率: fc ≤ fs/2 (奈奎斯特频率)

电赛常用ADC的抗混叠设计:
  ADS1115 (860SPS): fc ≤ 430Hz → R=3.3kΩ, C=100nF
  STM32 ADC (1MSPS): fc ≤ 500kHz → R=100Ω, C=3.3nF
  AD7606 (200kSPS): fc ≤ 100kHz → R=1kΩ, C=1.5nF
```

### 6.4 ADC参考电压设计

```
精确参考电压对ADC精度至关重要:

方案1: 专用基准芯片
  ┌──────────┐
  │ REF3033  ├── 3.300V (±0.2%)
  │ REF3030  ├── 3.000V
  │ REF3025  ├── 2.500V
  └──────────┘
  
  输出需并联100nF + 10μF去耦

方案2: 电阻分压 + 缓冲
         R1=10kΩ
  3.3V──┤├──┬──┤├──GND
              │  R2=10kΩ
              │
         ┌─┐  │
         │Op├──┘ → Vref = 1.65V
         └─┘
              │
         100nF去耦
```

### 6.5 ADC接地与去耦

```
                数字地(DGND)          模拟地(AGND)
                    │                     │
               ┌────┘                     └────┐
               │                               │
          ┌────┴────┐                    ┌─────┴────┐
          │   MCU   │                    │   ADC    │
          │  数字   │◄──── SPI/I2C ────►│   模拟   │
          │  电源   │                    │   电源   │
          └────┬────┘                    └─────┬────┘
               │    100nF+10μF去耦              │  100nF+10μF去耦
               │                               │
               └───────────┬───────────────────┘
                           │
                      单点接地(星形接地)
                           │
                          GND

关键原则:
  1. 模拟地和数字地在ADC下方单点连接
  2. ADC电源引脚紧贴100nF去耦电容
  3. 模拟信号走线远离数字信号线
  4. 参考电压单独走线
```

## 7. 完整信号调理设计实例

### 7.1 实例：心电信号采集前端

```
电极 → 保护 → 前置放大 → 带通滤波 → 50Hz陷波 → 后级放大 → ADC

设计指标:
  信号: 0.5~5mV (典型1mV)
  带宽: 0.05Hz ~ 150Hz
  增益: ×1000
  CMRR: >100dB

电路实现:
  ① ESD保护: 双向TVS (SMBJ5.0A)
  ② 前置放大: AD620, G=10 (Rg=5.1kΩ)
  ③ 高通: 0.05Hz (RC, R=3.3MΩ, C=1μF)
  ④ 低通: 150Hz (Sallen-Key, fc=150Hz)
  ⑤ 50Hz陷波: 双T网络 + 运放
  ⑥ 后级放大: ×100 (反相放大器)
  ⑦ 电平偏置: +1.65V (单电源ADC)
  ⑧ ADC: ADS1115 (16bit, 860SPS)

总增益: 10 × 100 = 1000
1mV → 1.0V → ADC读数约20000
```

### 7.2 实例：交流电压/电流测量

```
220V AC → 电压互感器/电流互感器 → 调理电路 → ADC

电压通道:
  PT: 220V:2mA + 采样电阻 → 调理
  或电阻分压: 220V → 1MΩ+1kΩ分压 → 220mV

电流通道:
  CT: SCT-013 (100A:50mA) + 采样电阻(33Ω)
  或 ACS712 霍尔电流传感器 (±20A→±1V)

调理电路:
  ┌──────────────────────────────────────────┐
  │ AC信号 → 隔离(光耦/CT) → 整流 → 滤波 → ADC│
  │             │                              │
  │         或直接采样（过零检测+RMS计算）       │
  └──────────────────────────────────────────┘

RMS计算 (软件):
  Vrms = √(1/N × Σ V[n]²)   (采样一个完整周期)
```

### 7.3 实例：光电传感器信号调理

```
光电二极管 → 跨阻放大 → 滤波 → 比较器/ADC

         Rf (可调: 100kΩ~1MΩ)
    ┌───┤├──┐
    │       │
    │  ┌─┐  │
    └──┤Op├──┤── Vout
       └─┘  │
        │    │
  PD──┘   ══Cf (补偿电容)
             │
            GND

跨阻放大器(TIA):
  Vout = -Ipd × Rf
  Cf = √(Cpd / (2π × GBW × Rf))  (稳定性补偿)

  增益带宽积限制:
  带宽 = GBW / (1 + Rf/Rs) ≈ GBW / Rf_gain

设计要点:
  1. 选择低偏置电流运放 (如OPA380, AD8605)
  2. PCB布局: PD紧贴运放输入端，最小化寄生电容
  3. 金属屏蔽罩防止环境光干扰
```

## 9. PCB布局要点

```
信号调理PCB布局检查清单:

□ 输入信号走线尽量短，远离数字信号
□ 运放电源引脚紧贴100nF去耦电容
□ 高阻抗节点（运放输入端）面积最小化
□ 反馈电阻靠近运放放置
□ 模拟地和数字地分离，在ADC下方单点连接
□ 参考电压走线加保护地环
□ 差分信号走线等长等距
□ 多级放大器级间加RC去耦
□ 敏感信号加地平面屏蔽
□ 电源入口加LC滤波
```
