# 电赛通信电路模板库

## 一、UART串口电路

### 1.1 TTL电平直连（3.3V ↔ 3.3V）

```
MCU_A (TX) ────────────── MCU_B (RX)
MCU_A (RX) ────────────── MCU_B (TX)
MCU_A (GND) ────────────── MCU_B (GND)
```

**注意：** 3.3V器件之间可直接连接，无需电平转换。

### 1.2 3.3V ↔ 5V 电平转换

```
方案一：MOS管双向电平转换

          VCC_L(3.3V)        VCC_H(5V)
              │                  │
              R1(10kΩ)           R2(10kΩ)
              │                  │
3.3V_TX ─────┤──────────────────┤──── 5V_RX
              │                  │
            ┌─┘                  └─┐
            │    2N7002(NMOS)      │
            ├───G                  │
            │    S─────────────────┤
            │    D                 │
            └──────────────────────┘
                                    │
5V_TX ──────────────────────────────┤──── 3.3V_RX

方案二：TXB0108自动方向电平转换芯片

VCCA(3.3V) ← 3.3V供电
VCCB(5V)   ← 5V供电
OE         → 接VCCA(使能)
A1~A8      ← 3.3V侧信号
B1~B8      ↔ 5V侧信号
```

### 1.3 USB转TTL串口电路(CH340G)

```
                  CH340G
                 ┌────────┐
    USB_D+ ──────┤D+    TXD├────→ MCU_RX
    USB_D- ──────┤D-    RXD├────← MCU_TX
                 │        │
      5V ───┬───┤VCC   V3├───┬── 100nF → GND
            │   │        │   │
            │   │UD+   UD-│   │
            │   │        │   │
           100nF│DTR   RTS│   │
            │   │        │   │
            └───┤GND     │   │
                └────────┘   │
                              │
  外部12MHz晶振：              │
  XTAL1 ──┤12MHz├── XTAL2    │
  20pF→GND        20pF→GND   │

DTR/RTS → 可用于STM32自动下载电路(接BOOT0/RESET)
```

### 1.4 RS-485通信电路

```
              MAX485 / SP3485
             ┌──────────────┐
    MCU_TX ──┤DI         A├──── 485_A(+)
    MCU_RX ──┤RO         B├──── 485_B(-)
             │            │
    GPIO  ──┤RE(低有效)    │
    GPIO  ──┤DE(高有效)    │
             │            │
      3.3V──┤VCC       GND├─── GND
             └──────────────┘

终端电阻：在A-B之间并联120Ω电阻(总线两端各一个)
偏置电阻：A接10kΩ上拉到VCC，B接10kΩ下拉到GND

典型接线：
  RE和DE可合并控制，高电平发送，低电平接收
  或RE接GND(常收)，DE由GPIO控制
```

### 1.5 RS-232通信电路

```
              MAX232 / SP3232
             ┌──────────────┐
    MCU_TX ──┤T1IN      T1OUT├──── DB9_RX(pin2)
    MCU_RX ──┤R1OUT     R1IN├──── DB9_TX(pin3)
             │              │
   C1+ ──┤C1+       C1-├── 1μF ─┐
   C2+ ──┤C2+       C2-├── 1μF  │
   V+  ←─┤V+          V-├── 1μF  │
             │              │      │
      3.3V──┤VCC         GND├─────┘
             └──────────────┘
             
外部电容：4× 1μF(或100nF，视型号)
```

## 二、SPI通信电路

### 2.1 SPI基本连接（单从机）

```
  MCU (Master)          SPI Slave(如W25Q128)
  ┌─────────┐          ┌──────────────┐
  │    SCK ──┼──────────┼── CLK        │
  │   MOSI ──┼──────────┼── DI (SI)    │
  │   MISO ──┼──────────┼── DO (SO)    │
  │     CS ──┼──────────┼── CS (SS)    │
  └─────────┘          └──────────────┘

接线注意：
  - SCK频率<器件最大频率
  - CS低电平选中
  - 建议MISO加10kΩ上拉
  - 信号线串联22~33Ω阻尼电阻(长线时)
```

### 2.2 SPI多从机连接

```
  MCU (Master)
  ┌──────────┐
  │     SCK ──┼──┬──────────┬──────────┐
  │    MOSI ──┼──┤──────────┤──────────┤
  │    MISO ──┼──┤──────────┤──────────┤
  │     CS1 ──┼──┤          │          │
  │     CS2 ──┼──┤          │          │
  │     CS3 ──┼──┘          │          │
  └──────────┘              │          │
                      ┌──────┘   ┌──────┘
                 Slave1      Slave2      Slave3
                 (Flash)     (LCD)      (ADC)
```

### 2.3 SPI Flash (W25Q128) 电路

```
        VCC(3.3V)
          │
         100nF
          │
    ┌─────┴─────┐
    │   W25Q128  │
    │            │
VCC─┤1        8├─VCC
CS──┤2        7├─HOLD(上拉到VCC)
DO──┤3        6├─CLK
GND─┤4        5├─DI
    └──────────┘

CS:   10kΩ上拉到VCC
HOLD: 10kΩ上拉到VCC
WP:   10kΩ上拉到VCC(禁用写保护)
```

### 2.4 SPI OLED (SSD1306) 电路

```
        VCC(3.3V)
          │
         100nF + 10μF
          │
    ┌─────┴─────┐
    │  SSD1306   │
    │  SPI OLED  │
    │            │
GND─┤GND     SCK├── MCU_SCK
VCC─┤VCC     SDA├── MCU_MOSI
NC──┤NC       CS├── MCU_CS
    │         DC├── MCU_DC(GPIO)
    │        RES├── MCU_RES(GPIO)
    └──────────┘

DC引脚：高=数据，低=命令
RES引脚：低脉冲复位(>3μs)
```

## 三、I2C通信电路

### 3.1 I2C基本连接

```
        VCC(3.3V)
         │
    ┌────┴────┐
    │10kΩ  10kΩ│  上拉电阻
    └┬────┬───┘
     │    │
  SDA│  SCL│
     │    │
  ┌──┴────┴──┐         ┌──────────┐
  │  MCU     │         │  I2C从机  │
  │  (Master)├─SDA─────┤SDA       │
  │          ├─SCL─────┤SCL       │
  └──────────┘         └──────────┘

上拉电阻选择：
  3.3V, 100kHz → 4.7kΩ
  3.3V, 400kHz → 2.2kΩ
  3.3V, 1MHz   → 1kΩ
  5V, 100kHz   → 4.7kΩ
```

### 3.2 I2C多设备总线

```
         VCC(3.3V)
          │
     4.7kΩ + 4.7kΩ (上拉)
          │
  SDA ────┼────┬──────┬──────┬──────
  SCL ────┼────┤──────┤──────┤
          │    │      │      │
       MPU6050 OLED  EEPROM  RTC
       (0x68) (0x3C) (0x50) (0x68)
```

### 3.3 常见I2C器件地址

| 器件 | 7位地址 | 型号 |
|------|---------|------|
| OLED SSD1306 | 0x3C | 0.96寸OLED |
| MPU6050 | 0x68 (AD0=0) | 六轴IMU |
| BMP280 | 0x76 (SDO=0) | 气压计 |
| BH1750 | 0x23 | 光照传感器 |
| ADS1115 | 0x48 | 16位ADC |
| PCF8574 | 0x27 | IO扩展 |
| AT24C256 | 0x50 | EEPROM |
| DS3231 | 0x68 | RTC时钟 |
| INA226 | 0x40 | 电流/功率监测 |
| AHT20 | 0x38 | 温湿度传感器 |

## 四、CAN总线电路

### 4.1 CAN收发器电路(TJA1050)

```
           VCC(5V)
            │
          100nF
            │
     ┌──────┴──────┐
     │   TJA1050    │
     │              │
VCC──┤1          8├─SPLIT(可接0.1μF到GND)
TXD──┤2(来自MCU_CAN_TX)
     │           │
RXD──┤3(到MCU_CAN_RX)
GND──┤4          │
     │           │
CANL┤5───────┐   │
CANH┤6──┐   │   │
     └───┼───┼───┘
         │   │
     120Ω终端电阻(总线两端)
         │   │
     ┌───┘   └───┐
     │  ESD保护   │
     │ (PESD1CAN) │
     └───────────┘
```

### 4.2 CAN总线拓扑

```
  ┌────────┐    ┌────────┐    ┌────────┐
  │ 节点1  │    │ 节点2  │    │ 节点3  │
  │CAN收发器├────┤CAN收发器├────┤CAN收发器│
  └────────┘    └────────┘    └────────┘
      │                              │
    120Ω ←───── 终端电阻 ─────→ 120Ω
```

**STM32 CAN配置要点：**
- 使用PA11(RX)/PA12(TX)或PB8(RX)/PB9(TX)
- 波特率：500kbps(常用)或1Mbps
- 120Ω终端电阻仅在总线两端加

## 五、蓝牙通信电路

### 5.1 HC-05/HC-06经典蓝牙

```
           VCC(3.3~5V)
             │
        ┌────┴────┐
        │  HC-05   │
        │          │
   VCC──┤1      TXD├── MCU_RX
   GND──┤2      RXD├── MCU_TX
        │          │
   STATE├3    EN/KEY├── GPIO(高=AT模式)
        └──────────┘

注意：
  - HC-05: 支持主从模式
  - HC-06: 仅从机模式
  - RXD需3.3V电平，5V系统加分压(51k+100k)
  - 波特率默认9600或38400
```

### 5.2 BLE蓝牙(HC-08/BT05/ESP32内置)

```
           VCC(3.3V)
             │
           100nF
             │
        ┌────┴────┐
        │  HC-08   │
        │  (BLE4.0)│
   VCC──┤1      TXD├── MCU_RX
   GND──┤2      RXD├── MCU_TX
        │      STATE├── GPIO
        │        EN├── GPIO
        └──────────┘
```

## 六、WiFi通信电路(ESP8266/ESP32)

### 6.1 ESP8266 (ESP-01S) 电路

```
           VCC(3.3V, ≥300mA)
             │
        100μF + 100nF
             │
        ┌────┴────┐
        │ ESP-01S  │
        │          │
   VCC──┤1(VCC)    │
   GND──┤2(GND)    │
   TXD──┤3(TXD)────┼── MCU_RX
   RXD──┤4(RXD)────┼── MCU_TX
   CH_PD├5(EN)─────┼── 10kΩ上拉到VCC
   RST──┤6(RST)────┼── 10kΩ上拉到VCC
   GPIO0├7──────────┼── 10kΩ上拉到VCC(正常运行)
   GPIO2├8──────────┼── 10kΩ上拉到VCC
        └──────────┘

注意：
  - 供电必须稳定3.3V，峰值电流可达300mA
  - 建议用AMS1117-3.3独立供电
  - CH_PD(EN)必须拉高
  - GPIO0拉高=正常运行，拉低=烧录模式
```

### 6.2 ESP-12F模块电路

```
        VCC(3.3V)
          │
     100μF + 100nF
          │
     ┌────┴────────┐
     │   ESP-12F    │
     │              │
 GND─┤1          16├─VCC
 TXD─┤2(UTXD)   15├─GND
 RXD─┤3(URXD)   14├─GPIO16(接RST可唤醒)
     │          13├─GPIO14
     │          12├─GPIO12
     │          11├─GPIO13(TX)
     │          10├─GPIO15(MTD0)
  RST┤4           9├─GPIO13(RX)
     │           8├─GPIO0
     │           7├─GPIO4
     │           6├─GPIO5
     │           5├─GPIO2
     └────────────┘

必要上拉：RST→10kΩ→VCC, GPIO0→10kΩ→VCC
GPIO15→10kΩ→GND(MTD0)
```

### 6.3 ESP32-WROOM-32 最小系统

```
        VCC(3.3V)
          │
     100μF + 100nF × 多个
          │
     ┌────┴────────┐
     │ ESP32-WROOM  │
     │              │
     │  EN ← 10kΩ上拉 + 100nF到GND(RC复位)
     │  GPIO0 ← 10kΩ上拉(正常运行)
     │  GPIO2 ← 10kΩ上拉(启动选择)
     │  GPIO12 ← 10kΩ下拉(Flash电压选择)
     │              │
     │  TXD0(GPIO1) → USB转TTL
     │  RXD0(GPIO3) ← USB转TTL
     │              │
     │  40MHz晶振 + 2×12pF
     └──────────────┘
```

## 七、通信方式选型对比

| 方式 | 速率 | 距离 | 线数 | 多设备 | 适用场景 |
|------|------|------|------|--------|---------|
| UART | 115200~1Mbps | <2m | 2(TX/RX) | 点对点 | 调试、模块通信 |
| SPI | 最高18MHz | <1m | 4(CS/SCK/MOSI/MISO) | 多从机 | Flash、LCD、ADC |
| I2C | 100k~1MHz | <1m | 2(SDA/SCL) | 多主机多从机 | 传感器、OLED |
| CAN | 1Mbps | <1km | 2(CANH/CANL) | 多节点 | 总线通信、抗干扰 |
| RS-485 | 10Mbps | <1200m | 2(A/B) | 多节点(32) | 远距离通信 |
| 蓝牙 | 1~3Mbps | <100m | 无线 | 1对多 | 手机控制、数据回传 |
| WiFi | <150Mbps | <100m | 无线 | 多设备 | 远程监控、图传 |
| NRF24L01 | 2Mbps | <100m | SPI | 6对1 | 无线遥控 |
| Zigbee | 250kbps | <100m | 无线 | 网状网络 | 多节点组网 |
