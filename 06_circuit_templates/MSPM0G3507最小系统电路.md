# MSPM0G3507 最小系统电路设计

> 适用封装：LQFP48 / LQFP64
> 核心参数：80MHz Cortex-M0+, 128KB Flash, 32KB RAM, 4Msps ADC
> TI官方推荐电赛主控

---

## 一、最小系统原理图描述

### 1.1 电源部分

```
VDD (多个引脚) ──┬── 100nF×N (每个VDD引脚一个)
                 └── 10μF×1-2 (储能)
VSS ── GND
VDDA ──┬── 100nF + 4.7μF (模拟电源滤波)
       └── 铁氧体磁珠隔离
VDDS ── 3.3V (系统电源域)
```

**电源要求：**
- 供电电压：2.25V ~ 3.6V（推荐3.3V）
- 去耦电容必须就近放置，距离≤2mm
- 模拟电源独立滤波走线

### 1.2 晶振部分

**高频晶振 HFXT (32MHz 或 48MHz)：**
```
HFXTIN  ──┬── 晶振 ──┬── HFXTOUT
          └── CL1 ── GND  └── CL2 ── GND
```
- 48MHz晶振用于USB场景，32MHz用于一般场景
- 负载电容根据晶振规格书选择（典型12-18pF）

**低频晶振 LFXT (32.768kHz)：**
```
LFXTIN  ──┬── 32.768kHz ──┬── LFXTOUT
          └── 10pF ── GND   └── 10pF ── GND
```

### 1.3 复位电路

```
NRST ──┬── 47kΩ ── VDD (内部弱上拉, 可外部加)
       ├── 100nF ── GND
       └── 按键 ── GND
```

### 1.4 调试接口 (SWD)

```
         ┌───────────┐
VDD  ────┤ 1  VCC    │
SWDIO ───┤ 2  SWDIO  │
GND   ───┤ 3  GND    │
SWCLK ───┤ 4  SWCLK  │
         └───────────┘
```
- 支持 cJTAG（2线调试）和 SWD
- TI XDS110/XDS200 调试器兼容

### 1.5 BOOT / 启动配置
- MSPM0G3507 无需外部BOOT引脚配置
- 通过SWD或BSL（Bootloader串口）下载
- BSL模式：拉低BSL引脚后复位进入

### 1.6 未用引脚
- 未用IO配置为输出低电平（最低功耗）
- 或配置为输入+内部上下拉

## 二、PCB设计要点

### 2.1 布局
- MCU居中，晶振紧贴（≤3mm）
- 去耦电容贴VDD引脚放置
- 模拟区域与数字区域分开
- 天线（如有无线）远离金属区域

### 2.2 布线
| 信号 | 线宽 | 备注 |
|------|------|------|
| 电源 | ≥0.8mm | 主干更宽 |
| 信号 | 0.2-0.3mm | - |
| 晶振 | 0.2mm | 包地、等长 |

### 2.3 特别注意
- MSPM0G3507 的 ADC 性能极好（4Msps），需注意模拟信号走线屏蔽
- 去耦电容选用X7R/X5R材质，避免NPO（容值不够大）
