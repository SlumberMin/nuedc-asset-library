# 常用模块接线速查表

## 一、STM32F103C8T6 最小系统引脚分配

### 1.1 核心引脚固定分配（不可更改）

| 引脚 | 功能 | 接线 |
|------|------|------|
| VDD | 3.3V电源 | 接3.3V（每个VDD加100nF去耦电容） |
| VSS | 接地 | 接GND |
| VDDA | 模拟电源 | 接3.3V（加100nF+1μF去耦） |
| VSSA | 模拟地 | 接GND |
| NRST | 复位 | 10kΩ上拉到VDD + 100nF到GND |
| BOOT0 | 启动选择 | 正常运行接GND，下载模式接VDD |
| OSC_IN | 晶振输入 | 接8MHz晶振 + 20pF到GND |
| OSC_OUT | 晶振输出 | 接8MHz晶振 + 20pF到GND |
| PA13 | SWDIO | SWD调试接口 |
| PA14 | SWCLK | SWD调试接口 |

## 二、传感器模块接线

### 2.1 MPU6050（六轴陀螺仪+加速度计）

**I2C接线：**

| MPU6050引脚 | STM32引脚 | 说明 |
|-------------|-----------|------|
| VCC | 3.3V | 电源（不要接5V） |
| GND | GND | 地 |
| SCL | PB8 (I2C1_SCL) | I2C时钟（需4.7kΩ上拉） |
| SDA | PB9 (I2C1_SDA) | I2C数据（需4.7kΩ上拉） |
| AD0 | GND | I2C地址 = 0x68 |
| AD0 | VCC | I2C地址 = 0x69 |
| INT | PB5 | 中断输出（可选） |

**I2C地址：** AD0=LOW → 0x68 (写0xD0, 读0xD1)

**关键寄存器：**
| 寄存器 | 地址 | 说明 |
|--------|------|------|
| WHO_AM_I | 0x75 | 读出应为0x68 |
| PWR_MGMT_1 | 0x6B | 写0x00唤醒 |
| SMPLRT_DIV | 0x19 | 采样率分频 |
| CONFIG | 0x1A | 低通滤波 |
| GYRO_CONFIG | 0x1B | 陀螺仪量程 |
| ACCEL_CONFIG | 0x1C | 加速度计量程 |

### 2.2 HC-SR04（超声波测距模块）

| HC-SR04引脚 | STM32引脚 | 说明 |
|-------------|-----------|------|
| VCC | 5V | 电源（必须5V） |
| Trig | PB0 (GPIO_OUT) | 触发信号（>10μs高脉冲） |
| Echo | PB1 (GPIO_IN/IC) | 回波信号（高电平时间=距离） |
| GND | GND | 地 |

**测距公式：** 距离(cm) = 高电平时间(μs) × 0.034 / 2

**两种读取方式：**
1. **输入捕获（推荐）：** TIM输入捕获模式测量高电平时间
2. **GPIO中断：** 上升沿开始计时，下降沿停止计时

## 三、执行器模块接线

### 3.1 L298N 双H桥驱动模块

| L298N引脚 | 连接 | 说明 |
|-----------|------|------|
| +12V | 电池正极(7~12V) | 电机电源 |
| GND | 电池负极 + STM32 GND | **共地必须** |
| +5V | 5V输出（可给STM32供电） | 板载稳压5V |
| ENA | PA6 (TIM3_CH1) | 电机A使能（PWM调速） |
| IN1 | PB12 | 电机A方向控制1 |
| IN2 | PB13 | 电机A方向控制2 |
| ENB | PA7 (TIM3_CH2) | 电机B使能（PWM调速） |
| IN3 | PB14 | 电机B方向控制1 |
| IN4 | PB15 | 电机B方向控制2 |
| OUT1/2 | 电机A | 电机A接线 |
| OUT3/4 | 电机B | 电机B接线 |

**L298N使能跳线：**
- 保留跳线 → ENA/ENB常HIGH（全速）
- 去掉跳线 → 外接PWM信号控制速度

### 3.3 SG90/MG996R 舵机

| 舵机线色 | 连接 | 说明 |
|----------|------|------|
| 棕色(GND) | GND | 地 |
| 红色(VCC) | 5V | 电源（SG90: 4.8~6V, MG996R: 6~7.2V） |
| 橙色(信号) | PA0 (TIM2_CH1) | PWM信号 |

**PWM参数：**
| 参数 | SG90 | MG996R |
|------|------|--------|
| 周期 | 20ms (50Hz) | 20ms (50Hz) |
| 0°脉宽 | 0.5ms | 0.5ms |
| 90°脉宽 | 1.5ms | 1.5ms |
| 180°脉宽 | 2.5ms | 2.5ms |
| 分辨率 | 180°/2000 = 0.09° | 同左 |

**PWM计算（72MHz主频，预分频72-1，周期20000-1）：**
```c
// 0° → CCR = 500
// 90° → CCR = 1500
// 180° → CCR = 2500
TIM_SetCompare1(TIM2, 1500); // 90°
```

| LM2596引脚 | 连接 | 说明 |
|------------|------|------|

| GND | GND | 地 |
| OUT | 输出电压 | 通过电位器调节 |

### 4.3 电流采样电路

```
        R_sense (0.1Ω/1W)
电机 ────┤├──── MOS管
              │
              └── INA199/INA219 ── STM32 ADC
                  (电流检测放大器)
```

**常用方案：**
| 方案 | 采样电阻 | 放大器 | 适用电流 |
|------|----------|--------|----------|
| 简单方案 | 0.1Ω | 运放同相放大 | <1A |
| 高侧采样 | 0.05Ω | INA199 | <3A |
| 霍尔方案 | 无 | ACS712 | <5A/20A/30A |
| 精密方案 | 0.01Ω | INA219(I2C) | <3A |

## 六、接线注意事项汇总

### ⚡ 必须遵守

1. **共地原则：** 所有模块GND必须连接在一起
2. **电压匹配：** 3.3V模块不能接5V，5V模块信号输出需分压/电平转换
3. **电源去耦：** 每个芯片/模块的VCC和GND之间加100nF陶瓷电容
4. **电流裕量：** 电源模块输出电流 ≥ 总需求 × 1.5
5. **信号完整性：** 模拟信号线远离数字信号线和电源线

### ⚠️ 常见接线错误

| 错误 | 后果 | 预防 |
|------|------|------|
| 忘记共地 | 通信乱码、ADC不准 | 接线前先确认GND连接 |
| 3.3V接5V | 芯片烧毁 | 双重确认电源电压 |
| TX/RX接反 | 无法通信 | TX接RX，RX接TX |
| 没加去耦电容 | 电源噪声大、不稳定 | 每个模块必加 |
| PWM频率不对 | 电机啸叫/舵机抖动 | 确认定时器配置 |
| 编码器线太长 | 信号干扰 | <30cm，加屏蔽 |
