## 使用方法

1. 根据所选MCU平台选择对应的**最小系统电路**模板
2. 根据外设需求选择**驱动电路**和**传感器电路**模板
3. 参考**设计指南**文件了解设计原理和注意事项
4. 将电路模板复制到原理图工程中，按需修改参数

## 依赖说明

- 电路模板基于常见EDA工具（KiCad/立创EDA/AD）设计
- 部分模板需要配合 `03_hardware_modules/` 中的元件清单选型

## 常见问题

**Q: 电机驱动选L298N还是TB6612？**
A: 小功率（<1A）用TB6612（效率高、发热小）；大功率用L298N或IR2104半桥。详见 `电机驱动电路设计指南.md`。

**Q: 3.3V和5V传感器混用怎么办？**
A: 需要电平转换电路。参考 `信号调理电路模板.md` 中的电平转换方案。

**Q: ADC采集精度不够怎么改善？**
A: 加RC滤波 + 电压跟随器，参考 `信号调理电路模板.md`。
