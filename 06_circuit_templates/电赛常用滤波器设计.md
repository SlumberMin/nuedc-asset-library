## 1. 滤波器设计总览

### 1.1 滤波器类型

| 类型 | 功能 | 应用 |
|------|------|------|
| 低通 (LPF) | 通过低频，抑制高频 | 抗混叠、平滑、去噪 |
| 高通 (HPF) | 通过高频，抑制低频 | 去直流、提取变化 |
| 带通 (BPF) | 通过特定频段 | 选频、解调 |
| 带阻 (BEF/Notch) | 抑制特定频段 | 陷波、去工频干扰 |

### 1.2 滤波器逼近方式

| 类型 | 特点 | 适用 |
|------|------|------|
| Butterworth | 最大平坦幅频 | 通用 |
| Chebyshev I | 通带等纹波，过渡带窄 | 需要快速截止 |
| Chebyshev II | 阻带等纹波 | 通带要求平坦 |
| Elliptic (Cauer) | 通带+阻带等纹波 | 阶数最低 |
| Bessel | 最平坦群延迟 | 信号保真/脉冲 |
| Linkwitz-Riley | -6dB 交叉点 | 音频分频 |

### 1.3 阶数与衰减关系 (Butterworth)

| 阶数 | 每倍频衰减 (dB/oct) | 每十倍频衰减 (dB/dec) | -3dB 处斜率 |
|------|---------------------|----------------------|-------------|
| 1 | -6 | -20 | -20dB/dec |
| 2 | -12 | -40 | -40dB/dec |
| 3 | -18 | -60 | -60dB/dec |
| 4 | -24 | -80 | -80dB/dec |
| 6 | -36 | -120 | -120dB/dec |
| 8 | -48 | -160 | -160dB/dec |

## 2. Sallen-Key 拓扑

### 2.1 特点

- **优点**：同相增益、结构简单、元件少、运放要求低
- **缺点**：Q 值受增益影响、高 Q 值对元件精度敏感
- **适用**：Q < 3 的一般滤波

### 2.2 Sallen-Key 低通滤波器 (2阶)

```
            R1        R2
Vin ──[===]──┬──[===]──┬──┬── Vout
             │         │  │
             C1        │  │
             │         C2 │
            GND        │  │
                      GND │
                          │
                    ┌─────┘
                    │ (+)┌────┐
                    ├───┤OPA ├──┘
                    │(-)│    │
                    │   └──┬─┘
                    │      │
                    └──────┘ (反馈到反相端)

标准形式 (增益 K=1+Rb/Ra):

传递函数:
H(s) = K / (s²·R1·R2·C1·C2 + s·[R1·C1 + R2·C1 + R1·C2·(1-K)] + 1)

参数计算 (给定 fc, Q):
令 m = R1/R2, n = C1/C2

选择策略:
- 设 C2 = C (选标准值)
- C1 = 4Q²·C  (n = 4Q²)
- R1 = 1/(2π·fc·Q·√(n)·C)
- R2 = R1/m
- 增益 K=1 时: m = n = 4Q² 不一定可行
```

### 2.3 Sallen-Key 低通设计工具

```python
import numpy as np

def sallen_key_lpf(fc, Q, K, C2=100e-9):
    """
    Sallen-Key 二阶低通滤波器设计
    fc: 截止频率 (Hz)
    Q: 品质因数
    K: DC 增益 (K=1 为单位增益)
    C2: C2 值 (F)
    返回: R1, R2, C1, C2, Ra, Rb
    """
    w0 = 2 * np.pi * fc
    
    # 选择 C1/C2 比值 (通常 4Q² 倍关系)
    n = max(4 * Q**2, 1)  # C1/C2
    C1 = n * C2
    
    # 计算电阻
    # 使用单位增益简化设计 (K=1)
    if abs(K - 1) < 0.01:
        # K=1 时: R1 = 1/(w0*Q*(2*Q + sqrt(4*Q² - 1))*C2)
        # 简化: 用标准公式
        a = 2 * Q
        b = np.sqrt(4 * Q**2 - 1) if Q > 0.5 else 0
        R1 = 1 / (w0 * Q * (a + b) * C2)
        R2 = 1 / (w0 * Q * (a - b) * C2) if (a - b) > 0 else R1
        Ra = None
        Rb = None
    else:
        # 通用增益设计
        m = 1  # 先假设 R1=R2
        R = 1 / (w0 * np.sqrt(n) * C2)
        R1 = R
        R2 = R
        # Ra, Rb 设置增益: K = 1 + Rb/Ra
        Ra = 10e3
        Rb = (K - 1) * Ra
    
    return {
        'R1': R1, 'R2': R2, 'C1': C1, 'C2': C2,
        'Ra': Ra, 'Rb': Rb, 'gain': K, 'fc': fc, 'Q': Q
    }

def design_sallen_key_butterworth(fc, order, C_base=100e-9):
    """
    设计 Butterworth Sallen-Key 滤波器 (偶数阶)
    通过级联二阶节实现
    """
    sections = []
    n_sections = order // 2
    
    for i in range(n_sections):
        # Butterworth 二阶节的 Q 值
        k = i + 1
        Q = 1 / (2 * np.cos((2*k - 1) * np.pi / (2 * order)))
        
        result = sallen_key_lpf(fc, Q, K=1, C2=C_base)
        sections.append(result)
        print(f"第{k}节: Q={Q:.3f}, R1={result['R1']:.0f}Ω, R2={result['R2']:.0f}Ω, "
              f"C1={result['C1']*1e9:.1f}nF, C2={result['C2']*1e9:.1f}nF")
    
    return sections

# 示例: 设计 4阶 Butterworth 低通, fc=1kHz
print("4阶 Butterworth LPF, fc=1kHz:")
sections = design_sallen_key_butterworth(1000, 4)
```

## 3. 多反馈 (MFB) 拓扑

### 3.1 特点

- **优点**：反相增益、对元件灵敏度低、适合高 Q 值、无增益限制
- **缺点**：反相输出
- **适用**：Q > 3、高精度滤波

### 3.2 MFB 低通滤波器 (2阶)

```
           R1        R2
Vin ──[===]──┬──[===]──┬── Vout (反相)
             │         │
             │   C1    │
             │  ┌─||─┐ │
             │  │    │ │
             │  └────┘ │
             │         │
             │         C2
             │         │
             │   ┌─────┘
             │   │
             │  (-)┌────┐
             └────┤OPA ├┘
                  │(+) │
                  └──┬─┘
                     │
                    GND

传递函数:
H(s) = -1 / (R1·C2) / [s² + s·(1/C1·(1/R1+1/R2+1/R3)) + 1/(R2·R3·C1·C2)]

设计公式 (给定 fc, Q, K):
  C2 选标准值 (如 10nF)
  C1 = C2 × 4Q²  (对 K=1 且 Butterworth)
  
  R2 = 1/(2π·fc·Q·C2) × (K·C1·C2)/(C1+C2)  (简化)
  R1 = R2/K
  R3 = 1/(4π²·fc²·R2·C1·C2)
```

### 3.3 MFB 设计工具

```python
def mfb_lowpass(fc, Q, K, C2=10e-9):
    """
    MFB 二阶低通滤波器设计
    fc: 截止频率
    Q: 品质因数
    K: 中频增益 (绝对值)
    C2: C2 值
    """
    w0 = 2 * np.pi * fc
    
    # 选择 C1 (典型 C1 = 4Q² × C2 用于高 Q)
    C1_ratio = max(4 * Q**2, 2)  # 最小2倍
    C1 = C1_ratio * C2
    
    # 计算电阻
    # 用标准 MFB 公式
    R3 = 1 / (w0**2 * C1 * C2 * (1 + 1/(K)) + w0/Q * C2)  # 简化近似
    
    # 更精确的计算:
    a1 = 1/Q  # 归一化 s 系数
    a0 = 1    # 归一化 s² 系数
    
    R2 = a1 / (2 * w0 * C2)  # 简化
    R1 = R2 / abs(K)
    R3 = 1 / (w0**2 * R2 * C1 * C2)
    
    return {
        'R1': R1, 'R2': R2, 'R3': R3,
        'C1': C1, 'C2': C2,
        'gain': -K, 'fc': fc, 'Q': Q
    }


def mfb_lowpass_precise(fc, Q, K_dc):
    """
    MFB 低通精确设计
    fc: -3dB 频率
    Q: 品质因数  
    K_dc: DC 增益 (正数，实际为反相)
    """
    w0 = 2 * np.pi * fc
    
    # 标准化: H(s) = -K_dc * w0² / (s² + (w0/Q)*s + w0²)
    
    # 设 C2 = C (选择标准值)
    C2 = 10e-9
    
    # 根据增益和 Q 选择 C1
    # C1 的选择影响电阻比
    C1 = 4 * Q**2 * C2  # 经典选择
    
    # 计算电阻
    R2 = Q / (w0 * C2 * (2*Q**2 + K_dc))
    R1 = R2 / K_dc
    R3 = 1 / (2 * Q * w0 * C1) - R2 * C2 / C1
    
    # 确保 R3 为正
    if R3 <= 0:
        # 重新选择 C1
        C1 = 2 * C2
        R2 = 1 / (w0 * C2 * np.sqrt(C1/C2))
        R1 = R2 / K_dc
        R3 = 1 / (w0**2 * C1 * C2 * R2)
    
    return {
        'R1': R1, 'R2': R2, 'R3': R3,
        'C1': C1, 'C2': C2,
        'K_dc': -K_dc, 'fc': fc, 'Q': Q
    }

# 示例
result = mfb_lowpass_precise(1000, 2, 1)
for k, v in result.items():
    if isinstance(v, float) and abs(v) < 1:
        print(f"{k}: {v:.2e}")
    elif isinstance(v, float):
        print(f"{k}: {v:.1f}")
    else:
        print(f"{k}: {v}")
```

### 3.4 MFB 带通滤波器 (2阶)

```
           C1        R2
Vin ──[||]──┬──[===]──┬── Vout
             │         │
             R1       C2
             │         │
             ├──[||]──┤
             │         │
             └──[===]──┘
                 R3
                 │

(简化: R3 接反相输入端到地)

带宽: BW = f0/Q
```

## 6. 陷波滤波器 (Notch Filter)

### 6.1 双 T 陷波器

```
           2R         R
Vin ──┬──[===]──┬──[===]──┬── Vout
      │         │         │
      │        2C         │
```

### 6.2 有源双 T 陷波器（增强 Q 值）

```
          双T网络
Vin ──┬────[双T]────┬── Vout
      │             │
      │    R3       │
      │   [===]     │
      │     │       │
      └─────┤(+)    │
            ├─OPA───┘

Q 值由 R3/R4 比值控制:
```

### 6.3 状态变量陷波器

```python
# 数字陷波滤波器 (二阶 IIR)
def design_notch_filter(f_notch, Q, fs):
    """
    设计二阶陷波滤波器
    f_notch: 陷波频率
    Q: 品质因数 (越大越窄)
    fs: 采样率
    """
    w0 = 2 * np.pi * f_notch / fs
    alpha = np.sin(w0) / (2 * Q)
    
    # 陷波滤波器系数
    b0 = 1
    b1 = -2 * np.cos(w0)
    b2 = 1
    a0 = 1 + alpha
    a1 = -2 * np.cos(w0)
    a2 = 1 - alpha
    
    b = np.array([b0/a0, b1/a0, b2/a0])
    a = np.array([1, a1/a0, a2/a0])
    
    return b, a

# 50Hz 陷波, Q=30, fs=10kHz
import numpy as np
b, a = design_notch_filter(50, 30, 10000)
print(f"b = {b}")
print(f"a = {a}")
```

## 7. 工频干扰消除方案

### 7.1 方案对比

| 方案 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| 模拟双 T 陷波 | 简单、实时 | Q 值固定、有相移 | 信号频段远离 50Hz |

| 硬件屏蔽+差分 | 从根本上解决 | 布线复杂 | 低电平信号 |
| 锁定放大器 | 抗干扰最强 | 需要参考信号 | 微弱信号检测 |

### 7.2 50Hz/60Hz 陷波设计速查

```

  R = 15.92kΩ (取 16kΩ), C = 100nF
```

## 8. 高阶滤波器级联设计

### 8.1 级联原则

```
高阶滤波器 = 多个二阶节级联 (+ 一阶节 if 奇数阶)

级联顺序:
1. Q 值最低的节放前面 (抗过载)
2. 增益最高的节放中间
3. 最后一节做增益调整

增益分配:
- 每级增益乘积 = 总增益
- 每级输出不应饱和
- 建议: 各级增益 ≤ 10 (20dB)
```

### 9.2 MAX7400 典型电路

```
        CLK
Vin ──[MAX7400]── Vout
         │
        GND

fc = f_clk / 100

外接时钟设置截止频率:
  fc=1kHz → f_clk=100kHz
  fc=5kHz → f_clk=500kHz

优点: 截止频率由时钟精确控制
缺点: 有时钟馈通噪声
```

## 10. 数字滤波器 (软件实现)

### 10.1 一阶 IIR 低通

```c
// y[n] = α·x[n] + (1-α)·y[n-1]
// α = 2π·fc·T / (2π·fc·T + 1)

typedef struct {
    float alpha;
    float y;
} lpf_1order_t;

float lpf_update(lpf_1order_t *f, float x) {
    f->y = f->alpha * x + (1.0f - f->alpha) * f->y;
    return f->y;
}

// fc=100Hz, fs=10kHz: alpha = 2*3.14*100*0.0001/(2*3.14*100*0.0001+1) = 0.059
```

### 10.2 二阶 IIR (Biquad)

```c
// Direct Form II Transposed
typedef struct {
    float b0, b1, b2;  // 分子系数
    float a1, a2;       // 分母系数 (a0=1)
    float w1, w2;       // 状态
} biquad_t;

float biquad_update(biquad_t *f, float x) {
    float w0 = x - f->a1 * f->w1 - f->a2 * f->w2;
    float y = f->b0 * w0 + f->b1 * f->w1 + f->b2 * f->w2;
    f->w2 = f->w1;
    f->w1 = w0;
    return y;
}

void biquad_init_lowpass(biquad_t *f, float fc, float Q, float fs) {
    float w0 = 2 * 3.14159f * fc / fs;
    float alpha = sinf(w0) / (2 * Q);
    float cosw0 = cosf(w0);
    
    float a0 = 1 + alpha;
    f->b0 = ((1 - cosw0) / 2) / a0;
    f->b1 = (1 - cosw0) / a0;
    f->b2 = f->b0;
    f->a1 = (-2 * cosw0) / a0;
    f->a2 = (1 - alpha) / a0;
    f->w1 = 0;
    f->w2 = 0;
}

void biquad_init_notch(biquad_t *f, float f_notch, float Q, float fs) {
    float w0 = 2 * 3.14159f * f_notch / fs;
    float alpha = sinf(w0) / (2 * Q);
    float cosw0 = cosf(w0);
    
    float a0 = 1 + alpha;
    f->b0 = 1.0f / a0;
    f->b1 = (-2 * cosw0) / a0;
    f->b2 = 1.0f / a0;
    f->a1 = (-2 * cosw0) / a0;
    f->a2 = (1 - alpha) / a0;
    f->w1 = 0;
    f->w2 = 0;
}
```

### 10.3 滑动平均滤波器

```c
// N 点滑动平均
#define AVG_N 16

typedef struct {
    float buffer[AVG_N];
    int index;
    float sum;
} moving_avg_t;

float moving_avg_update(moving_avg_t *f, float x) {
    f->sum -= f->buffer[f->index];
    f->sum += x;
    f->buffer[f->index] = x;
    f->index = (f->index + 1) % AVG_N;
    return f->sum / AVG_N;
}
```

### 10.4 中值滤波器

```c
// 中值滤波: 去除脉冲噪声
#define MED_N 5

float median_filter(float *buffer, int n) {
    // 简单冒泡排序找中值
    float sorted[MED_N];
    for (int i = 0; i < n; i++) sorted[i] = buffer[i];
    
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (sorted[j] > sorted[j+1]) {
                float tmp = sorted[j];
                sorted[j] = sorted[j+1];
                sorted[j+1] = tmp;
            }
        }
    }
    return sorted[n/2];
}
```

## 11. 滤波器设计速查表

### 11.1 常见滤波需求 → 推荐方案

| 需求 | 推荐方案 | 电路 |
|------|---------|------|
| 去除 ADC 高频噪声 | 一阶 RC 低通 | RC + 运放缓冲 |
| 抗混叠滤波 (12bit) | 4阶 Butterworth | 2×Sallen-Key 级联 |
| 50Hz 工频干扰 | 双 T 陷波 | 双T + 运放 |
| 音频带限 (20Hz~20kHz) | 2阶 HPF + 2阶 LPF | Sallen-Key |
| 提取直流偏置 | 一阶 RC 低通 | fc < 1Hz, R=1M, C=1μF |
| 信号包络检波 | 二极管检波 + LPF | 包络 + 平滑 |
| 脉冲整形 | 带通滤波 | 中心频率=脉冲频率 |
| 频谱分析前端 | 高阶 LPF/HPF | MFB 级联 |

### 11.2 fc/RC 快速计算卡

```
fc = 1 / (2π × R × C)

已知 R, C → fc:
  R=1kΩ,  C=100nF → fc=1.59kHz
  R=10kΩ, C=100nF → fc=159Hz
  R=10kΩ, C=10nF  → fc=1.59kHz
  R=10kΩ, C=1nF   → fc=15.9kHz
  R=100kΩ,C=10nF  → fc=159Hz
  R=100kΩ,C=1nF   → fc=1.59kHz

已知 fc, R → C:
  C = 1 / (2π × R × fc)
  
已知 fc, C → R:
  R = 1 / (2π × C × fc)
```

## 12. PCB 布局要点

### 12.1 模拟滤波器布局

```
1. 信号路径最短: 输入 → R → C → 运放 → 输出，直线排列
2. 地平面完整: 滤波器下方保持完整地平面
3. 输入输出隔离: 输入和输出不要平行走线
4. 电源去耦: 每个运放 VCC/GND 引脚加 100nF + 10μF
5. 远离数字信号: 避免 SPI/UART 等数字信号干扰
6. 屏蔽: 高阻抗节点（运放输入端）走线越短越好
```

### 12.2 注意事项

```
⚠️ 高 Q 值滤波器 (Q>5):
  - 电阻精度 ≤ 1%
  - 电容精度 ≤ 5% (用 C0G/NP0 陶瓷或薄膜电容)
  - Q 值每偏差 1%，中心频率偏差约 0.5%
  
⚠️ 高频滤波器 (>100kHz):
  - 运放 GBW > 100× fc
  - 走线电感影响不可忽略
  - 考虑用 PCB 微带线结构

⚠️ 低噪声设计:
  - 用金属膜电阻 (低噪声)
  - 用薄膜电容或 C0G 陶瓷
  - 前级增益要足够（信噪比在前级决定）
```
