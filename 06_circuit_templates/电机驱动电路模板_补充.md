# 电机驱动电路模板 - 补充

## 一、大电流直流电机驱动

### 1.1 BTS7960全桥驱动模块

**模块特点：**
- 驱动电流：最大43A（持续），峰值可达68A
- PWM频率：最高25kHz
- 内置过流/过温/欠压保护
- 逻辑电平：3.3V/5V兼容
- 供电：6V-28V

**引脚定义：**
| 引脚 | 功能 | 说明 |
|------|------|------|
| VCC | 逻辑供电 | 5V |
| GND | 地 | 共地 |
| IN1 | PWM输入1 | 控制正转/反转 |
| IN2 | PWM输入2 | 控制正转/反转 |
| EN | 使能 | 高电平使能 |
| OUT1 | 输出1 | 接电机一端 |
| OUT2 | 输出2 | 接电机另一端 |
| IS | 电流检测 | 模拟输出 |
| SR | 转换速率 | 可选，接GND降低EMI |

**接线图：**
```
STM32                          BTS7960模块
PA0(TIM2_CH1) ──→ IN1
PA1(TIM2_CH2) ──→ IN2
PA2 ────────────→ EN

BTS7960模块
VM(电机电源7-28V) ←── 电池/电源
OUT1 ────────────→ 电机+
OUT2 ────────────→ 电机-
GND ─────────────→ 电源GND(与MCU共地)
IS ──────────────→ ADC(电流检测，可选)
```

**控制逻辑：**
| IN1 | IN2 | 电机状态 |
|-----|-----|---------|
| PWM | LOW | 正转(PWM调速) |
| LOW | PWM | 反转(PWM调速) |
| LOW | LOW | 制动(短路制动) |
| HIGH | HIGH | 自由滑行 |

**STM32驱动代码：**
```c
// TIM2 CH1/CH2 输出PWM，控制BTS7960
void motor_init(void) {
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_1);
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_2);
    HAL_GPIO_WritePin(EN_GPIO, EN_PIN, GPIO_PIN_SET); // 使能
}

// speed: -1000 ~ +1000 (负值反转)
void motor_set_speed(int16_t speed) {
    uint16_t pwm = abs(speed);
    if (pwm > 1000) pwm = 1000;
    
    if (speed > 0) {
        // 正转：IN1=PWM, IN2=0
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, pwm);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_2, 0);
    } else if (speed < 0) {
        // 反转：IN1=0, IN2=PWM
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, 0);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_2, pwm);
    } else {
        // 制动：IN1=0, IN2=0
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, 0);
        __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_2, 0);
    }
}
```

### 1.2 MOSFET H桥驱动电路（分立元件）

**适用场景：** 需要自定义电流能力、成本敏感

**电路原理图（单桥臂）：**
```
         VCC(电机电源)
          │
    ┌─────┴─────┐
    │           │
  Q1(高侧N-MOS)  Q2(高侧N-MOS)
  IRF3205       IRF3205
    │D          │D
    │S    OUT1──┼──── 电机
    │G          │G
    │    ┌──────┤
    │    │      │
  驱动IC 驱动IC │
  (IR2104)(IR2104)
    │    │      │
    │    │    Q3(低侧N-MOS)
    │    │    IRF3205
    │    │      │D
    │    │      │S    OUT2── 电机
    │    │      │G          │
    │    │    驱动IC         │
    │    │    (IR2104)    Q4(低侧N-MOS)
    │    │      │        IRF3205
    └────┴──────┴────────────┘
                            │
                           GND
```

**简化H桥方案（适合中小电流）：**
```
              VCC(电机电源)
               │
         ┌─────┼─────┐
         │     │     │
       Q1(P)  Q2(N)  Q3(P)  Q4(N)
      AO3401 IRF3205 AO3401 IRF3205
         │S    │D     │S    │D
         ├─────┘      ├─────┘
         │            │
       电机+        电机-
         │            │
         └────────────┘
         
控制：
  正转：Q1+Q4导通 (PWM on Q4)
  反转：Q2+Q3导通 (PWM on Q2)
  制动：Q2+Q4导通 (低侧短路)
```

**高侧驱动难点与解决方案：**

N-MOS做高侧时，需要栅极电压高于源极(VCC+10V)：
```
方案一：IR2104半桥驱动IC（推荐）
  MCU_PWM ──→ IR2104 ──→ 高侧MOS + 低侧MOS
                │
               自举电容(100nF-1μF)
               自举二极管(快恢复)

方案二：电荷泵升压
  用555/MC34063产生VCC+12V供给高侧栅极

方案三：P-MOS做高侧（简单但内阻大）
  直接用GPIO驱动P-MOS栅极（反逻辑）
```

### 1.3 IR2104半桥驱动电路（推荐方案）

```
        VCC(12V电机电源)
         │
         ├── 自举二极管(1N4148) ──┐
         │                        │
         │     ┌──────────────┐   │
PWM_IN ──┤────→│IN   VB├──────┼───┘
         │     │     HO├───┐  │
         │     │IR2104 │  │  C_boot(1μF)
         │     │     VS├──┘  │
         │     │     LO├──┐  │
         │     │IN   GND├│  │
         │     └──────────┘  │
         │                   │
         │      ┌─ Q_high(N-MOS, IRF3205) ─┐
         │      │D                           │
         │      │S ←── VS(半桥中点) ──→ 电机
         │      │G ←── HO                   │
         │      │                            │
         │      └─ Q_low(N-MOS, IRF3205) ──┘
         │          │D
         │          │S
         │          │G ←── LO
         │          │
         GND────────┘
```

**完整H桥使用2片IR2104：**
```
MCU PWM1 ──→ IR2104 #1 ──→ 左桥臂(Q1高+Q3低)
MCU PWM2 ──→ IR2104 #2 ──→ 右桥臂(Q2高+Q4低)

正转：PWM1输出，PWM2=LOW (Q2截止，Q4导通)
反转：PWM2输出，PWM1=LOW (Q1截止，Q3导通)
```

## 二、步进电机驱动

### 2.1 步进电机基础

**常用步进电机类型：**
| 类型 | 步距角 | 特点 | 典型型号 |
|------|--------|------|---------|
| 42步进电机 | 1.8° | 通用，NEMA17 | 42BYGH34 |

| 57步进电机 | 1.8° | 大扭矩 | 57BYGH56 |
| 28BYJ-48 | 5.625°/64 | 内置减速 | ULN2003驱动 |

**两相步进电机接线：**
```
4线步进电机（两相双极性）：
  A+ (红线) ──┐
  A- (蓝线) ──┤── 电机线圈A
              │
  B+ (绿线) ──┐
  B- (黑线) ──┤── 电机线圈B

判断哪两根线是同一相：
用万用表测电阻，有阻值的是一相（几Ω到几十Ω）
```

**步进序列（全步进）：**
| 步序 | A+ | A- | B+ | B- | 角度 |
|------|----|----|----|----|------|
| 1 | H | L | L | H | 0° |
| 2 | H | L | H | L | 90° |
| 3 | L | H | H | L | 180° |
| 4 | L | H | L | H | 270° |

### 2.2 A4988步进驱动模块（最常用）

**模块特点：**
- 最大2A/相（需散热）
- 细分：1/1, 1/2, 1/4, 1/8, 1/16
- 内置过流/过温保护
- 逻辑电平：3.3V/5V兼容
- 供电：8V-35V

**引脚定义与接线：**
```
STM32                    A4988模块
PA0 ────────────────→ STEP (脉冲输入)
PA1 ────────────────→ DIR  (方向控制)
PA2 ────────────────→ EN   (使能，低有效)
                      SLEEP ←── VCC (不休眠)
                      RESET←── VCC (不复位)

A4988模块
VMOT(8-35V) ←──── 电机电源
GND ←─────────── 电机电源GND
1A,1B ──────→ 电机线圈A
2A,2B ──────→ 电机线圈B
VDD(3.3/5V) ←── 逻辑电源
GND ←─────────── 逻辑GND
```

**细分设置：**
| MS1 | MS2 | MS3 | 细分 | 步距角(1.8°电机) |
|-----|-----|-----|------|----------------|
| L | L | L | 1 | 1.8° |
| H | L | L | 1/2 | 0.9° |
| L | H | L | 1/4 | 0.45° |
| H | H | L | 1/8 | 0.225° |
| H | H | H | 1/16 | 0.1125° |

**电流限制调节：**
```
A4988模块上有电位器，调节最大电流：
Vref = I_max × 0.4 (A4988)
例：需要1A电流 → Vref = 0.4V

调节步骤：
1. 万用表测Vref引脚电压
2. 旋转电位器到目标电压
3. Vref不要超过0.8V(对应2A)
```

**STM32驱动代码：**
```c
// 使用TIM输出脉冲 + GPIO方向控制
#define STEP_PIN    GPIO_PIN_0
#define STEP_PORT   GPIOA
#define DIR_PIN     GPIO_PIN_1
#define DIR_PORT    GPIOA
#define EN_PIN      GPIO_PIN_2
#define EN_PORT     GPIOA

#define STEPS_PER_REV   200     // 1.8°电机一圈200步
#define MICRO_STEPS     16      // 1/16细分

// 步进电机状态
typedef struct {
    int32_t position;       // 当前位置(步数)
    int32_t target;         // 目标位置
    uint16_t speed;         // 速度(步/秒)
    uint8_t moving;         // 是否在运动
} Stepper_t;

Stepper_t stepper = {0};

// 使能/禁用
void stepper_enable(void) {
    HAL_GPIO_WritePin(EN_PORT, EN_PIN, GPIO_PIN_RESET);
}

void stepper_disable(void) {
    HAL_GPIO_WritePin(EN_PORT, EN_PIN, GPIO_PIN_SET);
}

// 单步脉冲
void stepper_pulse(void) {
    HAL_GPIO_WritePin(STEP_PORT, STEP_PIN, GPIO_PIN_SET);
    delay_us(2);  // 脉宽>1μs
    HAL_GPIO_WritePin(STEP_PORT, STEP_PIN, GPIO_PIN_RESET);
    delay_us(2);
}

// 设置方向
void stepper_set_dir(uint8_t dir) {
    HAL_GPIO_WritePin(DIR_PORT, DIR_PIN, dir ? GPIO_PIN_SET : GPIO_PIN_RESET);
}

// 移动指定步数
void stepper_move(int32_t steps, uint16_t speed_hz) {
    stepper_set_dir(steps >= 0);
    uint32_t abs_steps = abs(steps);
    uint32_t delay_us = 1000000 / speed_hz;
    
    for (uint32_t i = 0; i < abs_steps; i++) {
        stepper_pulse();
        delay_us(delay_us);
        stepper.position += (steps > 0) ? 1 : -1;
    }
}

// 移动到绝对位置
void stepper_move_to(int32_t target, uint16_t speed_hz) {
    int32_t delta = target - stepper.position;
    stepper_move(delta, speed_hz);
}

// 梯形加减速（平滑运动）
void stepper_move_smooth(int32_t steps, uint16_t max_speed, 
                         uint16_t accel) {
    stepper_set_dir(steps >= 0);
    uint32_t abs_steps = abs(steps);
    uint32_t accel_steps = max_speed * max_speed / (2 * accel);
    if (accel_steps > abs_steps / 2) accel_steps = abs_steps / 2;
    
    uint16_t current_speed = 100;  // 起始速度
    
    for (uint32_t i = 0; i < abs_steps; i++) {
        // 加速阶段
        if (i < accel_steps) {
            current_speed = 100 + (max_speed - 100) * i / accel_steps;
        }
        // 减速阶段
        else if (i > abs_steps - accel_steps) {
            current_speed = 100 + (max_speed - 100) * (abs_steps - i) / accel_steps;
        }
        // 匀速阶段
        else {
            current_speed = max_speed;
        }
        
        stepper_pulse();
        delay_us(1000000 / current_speed);
        stepper.position += (steps > 0) ? 1 : -1;
    }
}
```

### 2.4 28BYJ-48 + ULN2003驱动（最简方案）

```
ULN2003驱动板            28BYJ-48步进电机
  IN1 ←── MCU PA0        红线(公共端) ←── 5V
  IN2 ←── MCU PA1        橙线 ←── OUT1
  IN3 ←── MCU PA2        黄线 ←── OUT2
  IN4 ←── MCU PA3        粉线 ←── OUT3
                          蓝线 ←── OUT4

注意：28BYJ-48是5V供电，MCU 3.3V也能驱动ULN2003
      减速比1:64，实际一圈需要 64×64 = 4096 步(半步)
```

**28BYJ-48半步驱动序列：**
```c
// 半步驱动(8步序列，精度更高)
const uint8_t half_step[8] = {
    0x08,  // 1000
    0x0C,  // 1100
    0x04,  // 0100
    0x06,  // 0110
    0x02,  // 0010
    0x03,  // 0011
    0x01,  // 0001
    0x09   // 1001
};

void stepper_28byj_step(int steps, uint16_t delay_ms) {
    static uint8_t phase = 0;
    for (int i = 0; i < abs(steps); i++) {
        phase = (steps > 0) ? (phase + 1) % 8 : (phase + 7) % 8;
        
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_0, (half_step[phase] & 0x01) ? SET : RESET);
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_1, (half_step[phase] & 0x02) ? SET : RESET);
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_2, (half_step[phase] & 0x04) ? SET : RESET);
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_3, (half_step[phase] & 0x08) ? SET : RESET);
        
        HAL_Delay(delay_ms);
    }
}
```

## 三、舵机驱动

### 3.1 舵机PWM控制

```
信号线(橙/白) ←── MCU PWM输出(50Hz)
电源线(红)   ←── 5V独立供电(不要从MCU取电!)
地线(棕/黑)  ←── GND

PWM参数：
- 周期：20ms (50Hz)
- 脉宽0.5ms → 0°
- 脉宽1.5ms → 90°
- 脉宽2.5ms → 180°
```

**STM32舵机控制：**
```c
// TIM3 CH1, 预分频72-1, 计数20000-1 → 50Hz
void servo_set_angle(uint8_t angle) {
    // 0°=500μs, 180°=2500μs
    // 对应比较值: 500~2500 (1μs=1计数@72MHz/72=1MHz)
    uint16_t compare = 500 + (uint16_t)angle * 2000 / 180;
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, compare);
}
```

### 3.2 舵机供电注意事项

```
❌ 错误接法：
MCU 3.3V ──→ 舵机VCC (电流不够，MCU会复位)

✓ 正确接法：
电池 ──→ 5V DC-DC(MP1584) ──→ 舵机VCC
                               舵机GND ──→ 共地

✓ 多舵机：
电池 ──→ 5V/3A DC-DC ──┬──→ 舵机1 VCC
                        ├──→ 舵机2 VCC
                        ├──→ 舵机3 VCC
                        └──→ 100μF电解电容(滤波)
```

## 四、电机驱动保护电路

### 4.1 续流二极管

```
电机等效为电感+电阻，关断时会产生反向EMF（反电动势）

MOSFET H桥必须加续流二极管：
         VCC
          │
    Q1 ───┤─── Q2
          │
        电机 ←── 需要快速续流二极管
          │
    Q3 ───┤─── Q4
          │
         GND

大部分MOSFET内置体二极管，但速度慢
高速开关时建议并联快恢复二极管(MUR1560/SF16)
```

### 4.2 电机驱动电源隔离

```
问题：电机启停导致电源波动，引起MCU复位

解决方案：
1. 独立供电：MCU用一路LDO，电机用独立电池/DC-DC
2. 加大滤波电容：电机电源入口加470μF+100nF
3. 光耦隔离：PWM信号经光耦传到驱动板
4. 共地处理：MCU地和电机地在电源入口处单点连接
```

### 4.3 电流采样与保护

```
在电机回路串联采样电阻（0.1Ω-0.5Ω），用运放放大后送ADC

         Rsense(0.1Ω)
电机 ──→ ┌┤├──── 驱动桥
         │
         └──→ INA180(增益50) ──→ ADC

过流阈值设定：软件比较，超过则关闭PWM输出
```
