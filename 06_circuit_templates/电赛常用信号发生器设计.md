## 2. DDS 方案（推荐）

### 2.1 DDS原理

```
频率控制字(FTW)     相位累加器      查找表(LUT)      DAC        低通滤波
    ┌──┐        ┌──────────┐    ┌────────┐    ┌─────┐    ┌─────┐
───►│+ │◄──────►│  32位寄存 │───►│ sin表  │───►│DAC  │───►│ LPF │──► 输出
    │  │        │          │    │(256点) │    │     │    │     │
    └──┘        └──────────┘    └────────┘    └─────┘    └─────┘
      ▲
      │
  f_out = f_clk × FTW / 2^32

例: f_clk=25MHz, 要输出1kHz
    FTW = 1000 × 2^32 / 25000000 = 171799
```

### 2.2 AD9833 方案（SPI，最高12.5MHz）

```
STM32 ──SPI──► AD9833 ──► LPF ──► 输出
                │
           25MHz晶振
```

**电路：**
```
         ┌──────────────┐
   SCK ──┤1 SCLK    VDD├── 3.3V
   SDIN──┤2 SDATA  VOUT├──┬── 信号输出
   CS  ──┤3 FSYNC  AGND├──│
   VDD ──┤4 MCLK   DGND├──│
         │              │  │
 25MHz ──┤5 MCLK'       │  │
         └──────────────┘  │
                          ┌┴┐
                 200Ω     │ │ ← 可选衰减电阻
                          └┬┘
                           │
                          ═══ 100nF
                           │
                          GND

后级低通滤波器（可选）：
VOUT ── 200Ω ──┬── 输出
                │
              1nF
                │
               GND
fc = 1/(2π×200×1e-9) ≈ 796kHz
```

**驱动代码：**

```c
// AD9833 SPI驱动
#define AD9833_FCLK  25000000UL  // 25MHz

// 控制寄存器位定义
#define AD9833_CTRL_B28      (1 << 13)
#define AD9833_CTRL_HLB      (1 << 12)
#define AD9833_CTRL_FSELECT  (1 << 11)
#define AD9833_CTRL_PSELECT  (1 << 10)
#define AD9833_CTRL_RESET    (1 << 8)
#define AD9833_CTRL_SLEEP1   (1 << 7)
#define AD9833_CTRL_SLEEP12  (1 << 6)
#define AD9833_CTRL_OPBITEN  (1 << 5)
#define AD9833_CTRL_DIV2     (1 << 3)
#define AD9833_CTRL_MODE     (1 << 1)

// 波形选择
#define AD9833_WAVE_SINE     0x0000
#define AD9833_WAVE_TRIANGLE 0x0002
#define AD9833_WAVE_SQUARE   0x0028
#define AD9833_WAVE_MSBOUT   0x0020

static SPI_HandleTypeDef* ad9833_hspi;

void AD9833_Init(SPI_HandleTypeDef* hspi) {
    ad9833_hspi = hspi;
    AD9833_Write(AD9833_CTRL_RESET);  // 复位
    HAL_Delay(10);
    AD9833_Write(AD9833_CTRL_B28 | AD9833_CTRL_RESET);
}

void AD9833_Write(uint16_t data) {
    HAL_GPIO_WritePin(AD9833_CS_GPIO, AD9833_CS_PIN, GPIO_PIN_RESET);
    HAL_SPI_Transmit(ad9833_hspi, (uint8_t*)&data, 1, 100);
    HAL_GPIO_WritePin(AD9833_CS_GPIO, AD9833_CS_PIN, GPIO_PIN_SET);
}

void AD9833_SetFrequency(float freq_hz, uint16_t phase, uint16_t wave) {
    uint32_t freq_word = (uint32_t)((freq_hz / AD9833_FCLK) * (1UL << 28));
    
    uint16_t freq_lsb = (uint16_t)(freq_word & 0x3FFF) | 0x4000;  // FREQ0寄存器
    uint16_t freq_msb = (uint16_t)((freq_word >> 14) & 0x3FFF) | 0x4000;
    uint16_t phase_word = (phase & 0x0FFF) | 0xC000;  // PHASE0寄存器
    
    // 先复位，再写入频率和相位
    AD9833_Write(AD9833_CTRL_B28 | AD9833_CTRL_RESET);
    AD9833_Write(freq_lsb);
    AD9833_Write(freq_msb);
    AD9833_Write(phase_word);
    
    // 清除复位，设置波形
    AD9833_Write(AD9833_CTRL_B28 | wave);
}

// 使用示例
int main() {
    AD9833_Init(&hspi1);
    
    // 输出1kHz正弦波
    AD9833_SetFrequency(1000.0f, 0, AD9833_WAVE_SINE);
    
    // 输出10kHz三角波
    // AD9833_SetFrequency(10000.0f, 0, AD9833_WAVE_TRIANGLE);
    
    // 输出100Hz方波
    // AD9833_SetFrequency(100.0f, 0, AD9833_WAVE_SQUARE);
    
    while (1);
}
```

### 2.3 AD9850 方案（并行/串行，最高40MHz）

```
STM32 ── 并行8位或串行 ──► AD9850 ──► LPF ──► 功放 ──► 输出
                              │
                         125MHz有源晶振
```

**关键参数：**
- 频率分辨率：0.0291Hz（125MHz时钟）
- 输出频率：DC ~ 40MHz（125MHz × 0.4 = 50MHz理论最大）
- DAC输出：10位，约1Vpp
- 需要外部低通滤波器

```c
// AD9850 串行模式驱动
#define AD9850_FCLK  125000000UL

void AD9850_WriteByte(uint8_t data) {
    for (int i = 0; i < 8; i++) {
        HAL_GPIO_WritePin(W_DATA_GPIO, W_DATA_PIN, 
                          (data & 0x01) ? GPIO_PIN_SET : GPIO_PIN_RESET);
        HAL_GPIO_WritePin(W_CLK_GPIO, W_CLK_PIN, GPIO_PIN_SET);
        HAL_GPIO_WritePin(W_CLK_GPIO, W_CLK_PIN, GPIO_PIN_RESET);
        data >>= 1;
    }
}

void AD9850_SetFrequency(float freq_hz) {
    uint32_t freq_word = (uint32_t)(freq_hz * 4294967296.0f / AD9850_FCLK);
    
    // FQ_UD拉低
    HAL_GPIO_WritePin(FQ_UD_GPIO, FQ_UD_PIN, GPIO_PIN_RESET);
    
    // 写入40位数据（32位频率 + 8位控制）
    AD9850_WriteByte((uint8_t)(freq_word & 0xFF));
    AD9850_WriteByte((uint8_t)((freq_word >> 8) & 0xFF));
    AD9850_WriteByte((uint8_t)((freq_word >> 16) & 0xFF));
    AD9850_WriteByte((uint8_t)((freq_word >> 24) & 0xFF));
    AD9850_WriteByte(0x00);  // 控制字节（相位0，Power-up）
    
    // FQ_UD上升沿更新
    HAL_GPIO_WritePin(FQ_UD_GPIO, FQ_UD_PIN, GPIO_PIN_SET);
    HAL_GPIO_WritePin(FQ_UD_GPIO, FQ_UD_PIN, GPIO_PIN_RESET);
}
```

## 3. DAC + DMA 任意波形发生器

### 3.1 方案概述

```
STM32内部DAC ── DMA传输 ── 波形缓冲区 ── 定时器触发
                                      │
                            ┌─────────┘
                            ▼
                      低通滤波 ── 功放 ── 输出
```

**优势：** 可产生任意波形，成本低（只需STM32），适合低频信号

### 3.2 硬件配置

```c
// CubeMX配置:
// DAC1 → Channel1 → PA4
// TIM6 → 触发DAC
// DMA1 Stream5 → DAC Channel1
```

### 3.3 波形生成代码

```c
#include "math.h"

#define WAVE_SAMPLES  256
#define WAVE_AMPLITUDE 2047  // 12位DAC, 中间值
#define WAVE_OFFSET    2048

uint16_t wave_buf[WAVE_SAMPLES];

// 生成正弦波
void GenerateSine(void) {
    for (int i = 0; i < WAVE_SAMPLES; i++) {
        wave_buf[i] = (uint16_t)(WAVE_OFFSET + 
                      WAVE_AMPLITUDE * sin(2.0 * M_PI * i / WAVE_SAMPLES));
    }
}

// 生成三角波
void GenerateTriangle(void) {
    for (int i = 0; i < WAVE_SAMPLES; i++) {
        if (i < WAVE_SAMPLES / 2)
            wave_buf[i] = (uint16_t)(WAVE_OFFSET + 
                          WAVE_AMPLITUDE * 4.0 * i / WAVE_SAMPLES - WAVE_AMPLITUDE);
        else
            wave_buf[i] = (uint16_t)(WAVE_OFFSET + 
                          WAVE_AMPLITUDE * 3.0 - WAVE_AMPLITUDE * 4.0 * i / WAVE_SAMPLES);
    }
}

// 生成锯齿波
void GenerateSawtooth(void) {
    for (int i = 0; i < WAVE_SAMPLES; i++) {
        wave_buf[i] = (uint16_t)(WAVE_OFFSET - WAVE_AMPLITUDE + 
                      2.0 * WAVE_AMPLITUDE * i / WAVE_SAMPLES);
    }
}

// 生成方波
void GenerateSquare(void) {
    for (int i = 0; i < WAVE_SAMPLES; i++) {
        wave_buf[i] = (i < WAVE_SAMPLES / 2) ? 
                      (WAVE_OFFSET + WAVE_AMPLITUDE) : 
                      (WAVE_OFFSET - WAVE_AMPLITUDE);
    }
}

// 自定义波形（任意函数）
void GenerateCustom(float (*func)(float phase)) {
    for (int i = 0; i < WAVE_SAMPLES; i++) {
        float phase = (float)i / WAVE_SAMPLES;
        float val = func(phase);
        // 限幅
        if (val > 1.0f) val = 1.0f;
        if (val < -1.0f) val = -1.0f;
        wave_buf[i] = (uint16_t)(WAVE_OFFSET + WAVE_AMPLITUDE * val);
    }
}

// 调频：改变定时器周期
void SetFrequency(float freq_hz) {
    // freq_out = f_tim / (samples × period)
    // period = f_tim / (freq_out × samples)
    uint32_t timer_clk = HAL_RCC_GetPCLK1Freq() * 2;  // APB1定时器时钟
    uint32_t prescaler = 1;
    uint32_t period = timer_clk / (freq_hz * WAVE_SAMPLES) - 1;
    
    // 自动选择预分频
    while (period > 65535) {
        prescaler *= 2;
        period = timer_clk / (prescaler * freq_hz * WAVE_SAMPLES) - 1;
    }
    
    __HAL_TIM_SET_PRESCALER(&htim6, prescaler - 1);
    __HAL_TIM_SET_AUTORELOAD(&htim6, period);
    HAL_TIM_GenerateEvent(&htim6, TIM_EVENTSOURCE_UPDATE);
}

// 启动DMA输出
void StartWaveform(void) {
    HAL_DAC_Start_DMA(&hdac, DAC_CHANNEL_1, (uint32_t*)wave_buf, 
                      WAVE_SAMPLES, DAC_ALIGN_12B_R);
    HAL_TIM_Base_Start(&htim6);
}

int main() {
    HAL_Init();
    SystemClock_Config();
    MX_GPIO_Init();
    MX_DAC_Init();
    MX_TIM6_Init();
    MX_DMA_Init();
    
    GenerateSine();
    SetFrequency(1000);  // 1kHz
    StartWaveform();
    
    while (1) {
        // 可动态切换波形和频率
    }
}
```

## 4. PLL 频率合成

### 4.1 原理

```
参考频率(f_ref)     相位比较器     环路滤波器     VCO      分频器(N)
    ┌──┐         ┌─────────┐    ┌────────┐    ┌─────┐   ┌─────┐
───►│  ├────────►│  PD/CP  │───►│  LPF   │───►│ VCO │──►│ /N  │──► f_out
    └──┘         │         │    └────────┘    └─────┘   └─────┘
    ┌──┐         │         │                      │
    │/R│◄────────┤         │                      │
    └──┘         └─────────┘                      │
      ▲                                           │
      │                                           │
   f_ref ─────────────────────────────────────────┘

f_out = f_ref × N / R
```

### 4.2 常用PLL芯片

| 芯片 | 频率范围 | 特点 | 电赛适用 |
|------|----------|------|----------|
| SI5351 | 8kHz~160MHz | I2C控制，3路输出 | **推荐** |
| ADF4351 | 137MHz~4.4GHz | SPI控制，射频 | 高频题 |
| LMX2594 | 10MHz~15GHz | 超高性能 | 非电赛范畴 |
| CD4046 | 1MHz~1MHz | 经典PLL | 低频锁相 |

### 4.3 SI5351 方案（I2C，推荐）

```c
// SI5351 时钟发生器
// 3路独立输出，8kHz~160MHz，I2C控制
// 晶振25MHz或27MHz

#define SI5351_ADDR  0x60  // 7位地址

// 寄存器定义
#define SI5351_REG_CLK_ENABLE    3
#define SI5351_REG_PLL_A         26
#define SI5351_REG_MULTISYNTH_0  42

void SI5351_Init(void) {
    // 禁用所有输出
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_CLK_ENABLE, 0xFF);
    
    // PLL复位
    I2C_WriteReg(SI5351_ADDR, 177, 0xA0);  // PLLA+PLLB复位
    HAL_Delay(10);
}

void SI5351_SetFrequency(uint8_t clk_num, uint32_t freq_hz) {
    // 简化版本：使用整数分频
    // f_out = f_xtal * (a + b/c) / (d * e + f/g)
    // 这里用简化计算，实际需根据AN619文档精确计算
    
    uint32_t xtal_freq = 25000000;  // 25MHz晶振
    
    // 计算PLL参数（简化）
    uint32_t pll_mult = freq_hz / xtal_freq + 1;
    if (pll_mult > 900) pll_mult = 900;
    if (pll_mult < 15) pll_mult = 15;
    
    uint32_t pll_freq = xtal_freq * pll_mult;
    uint32_t div = pll_freq / freq_hz;
    
    // 设置PLL（整数模式）
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_PLL_A, pll_mult - 4);  // P1
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_PLL_A + 1, 0);         // P2[15:8]
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_PLL_A + 2, 0);         // P2[7:0]
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_PLL_A + 3, 0);         // P3[15:8]
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_PLL_A + 4, 1);         // P3[7:0]
    
    // 设置Multisynth分频器
    uint8_t base = SI5351_REG_MULTISYNTH_0 + (clk_num * 8);
    I2C_WriteReg(SI5351_ADDR, base, div - 4);  // P1
    I2C_WriteReg(SI5351_ADDR, base + 1, 0);
    I2C_WriteReg(SI5351_ADDR, base + 2, 0);
    I2C_WriteReg(SI5351_ADDR, base + 3, 0);
    I2C_WriteReg(SI5351_ADDR, base + 4, 1);
    
    // PLL复位
    I2C_WriteReg(SI5351_ADDR, 177, 0xA0);
    
    // 使能输出
    I2C_WriteReg(SI5351_ADDR, SI5351_REG_CLK_ENABLE, ~(1 << clk_num));
}
```

## 5. 函数发生器 IC

### 5.1 ICL8038（经典，已停产但易获取）

```
         ┌──────────────┐
   RA ──┤1           14├── RB
         │              │
   C  ──┤2     ICL    13├── NC
         │    8038      │
  FM_S ──┤3           12├── R_Set
         │              │
  V+  ──┤4           11├── V-
         │              │
  Duty──┤5           10├── C_ext
         │              │
  SqO ──┤6            9├── TrO
         │              │
  SiO ──┤7            8├── FM_I
         └──────────────┘

连接方式：
- 引脚6(方波)通过10kΩ上拉到V+
- 引脚7(正弦波)输出
- 引脚9(三角波)输出
- RA=RB=10kΩ, C=1nF~10μF

f = 1 / (0.33 × C × (RA + RB/2))
f_max ≈ 300kHz
```

### 5.2 XR2206

```
方波/正弦波/三角波输出
频率范围：0.01Hz~1MHz
通过电阻电容设置频率
正弦波失真度<2%

f = 1 / (R × C)
R = R1 或 R2（引脚选择）
```

## 6. 输出调理电路

### 6.1 功率放大

```
信号源 ── 电压跟随器 ── 功率运放 ── 50Ω输出
          (OPA340)     (OPA541/TDA2030)

负载能力：50Ω负载时需要输出电流 I = Vpp/50
```

### 6.2 阻抗匹配（50Ω输出）

```
         R_series
信号 ──┤├──┬── 50Ω输出
           │
         50Ω ← 匹配电阻（当负载开路时）
           │
          GND

当R_series=50Ω时：
- 负载50Ω：输出电压为信号的一半（-6dB）
- 负载开路：输出电压等于信号电压
```

### 6.3 DC偏置叠加

```
         R1(10kΩ)
信号 ──┤├──┬── 输出
           │
         R2(10kΩ)
           │
偏置 ──┤├──┘
(Vref)

Vout = Vsignal × R2/(R1+R2) + Vbias × R1/(R1+R2)
```

## 7. 电赛实战设计模板

### 7.1 基础信号发生器（AD9833方案）

```
STM32F103C8T6
    │
    ├── SPI1 → AD9833
    │           │
    │          25MHz晶振
    │           │
    │          LPF → BNC输出
    │
    ├── 按键 × 4（频率+/频率-/波形/使能）
    ├── 旋转编码器（频率细调）
    └── 0.96" OLED（显示频率/波形/幅度）

预算：STM32(¥8) + AD9833(¥12) + OLED(¥10) + 按键等(¥5) ≈ ¥35
```

### 7.2 高性能信号源（DDS+功放）

```
STM32F407
    │
    ├── SPI → AD9850 (40MHz)
    │          │
    │         LPF (7阶椭圆, fc=20MHz)
    │          │
    │         AD603 (数控衰减)
    │          │
    │         OPA541 (功率放大)
    │          │
    │         50Ω输出
    │
    ├── DAC → AD603控制电压（幅度控制）
    ├── ADC ← 输出检波（幅度反馈）
    ├── TFT LCD（频率/幅度/波形显示）
    └── 旋转编码器 + 按键

预算：约¥80-120
```

### 7.3 频率/幅度/相位可控信号源

```
功能需求：
1. 频率范围：100Hz ~ 10MHz
2. 频率步进：1Hz
3. 输出幅度：0.1Vpp ~ 5Vpp（程控）
4. 相位可调：0° ~ 360°
5. 波形：正弦/方波/三角波
6. 输出阻抗：50Ω

实现方案：
DDS(AD9833/50) + DAC幅度控制 + MCU控制 + LCD显示
```

## 8. 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| DDS输出有杂散 | DAC量化噪声+时钟泄漏 | 加低通滤波器 |
| DAC波形失真 | 采样点不够/滤波器不当 | 增加采样点数/优化LPF |
| 频率不准 | 晶振偏差 | 用温补晶振或校准 |
| 输出幅度不稳定 | 电源噪声/负载变化 | 加反馈稳幅电路 |
| 高频衰减 | DAC建立时间/LPF截止太低 | 用高速DAC/调整LPF |
