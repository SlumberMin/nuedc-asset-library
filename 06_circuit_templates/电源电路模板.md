# 电源电路模板

> 锂电池充放电管理 + 多路稳压输出方案

---

## 一、锂电池充放电方案

### 1.1 单节锂电池充电（TP4056方案）

```
USB 5V ──┬── TP4056 ──── BAT+ (锂电池正极)
         │   Pin: VIN   BAT- (锂电池负极)
         │
         │   PROG引脚 ── Rprog ── GND
         │   Rprog = 1.2kΩ → 充电电流约1A
         │   Rprog = 2kΩ  → 充电电流约580mA
         │
         │   充电指示:
         │   CHRG (Pin 7) ── LED ── 1kΩ ── VCC (充电中亮)
         │   STDBY(Pin 6) ── LED ── 1kΩ ── VCC (充满亮)
```

### 1.2 锂电池保护电路（DW01+8205A）

```
B+ ── DW01 Pin 1 (OC) ── 8205A Gate
BAT+ ── 8205A Source
P+ ── 8205A Drain (输出正极)

BAT- ── DW01 Pin 2 (CS) ── 8205A Source2
P- ── 8205A Drain2 (输出负极)

DW01功能:
- 过充保护: 4.25V断开
- 过放保护: 2.5V断开  
- 过流保护: 3A断开
- 短路保护: 断开
```

### 1.3 集成方案（IP5306）

```
USB 5V ── IP5306 VIN
BAT ── IP5306 BAT (锂电池)
LBO ── IP5306 (升压输出5V)
SYS_OUT ── 5V (系统供电)

特点: 充电+放电+升压+电量指示一体
```

## 二、多路稳压方案

### 2.1 系统电源架构

```
锂电池 3.7-4.2V
    │
    ├── MP1584EN ── 5V / 3A  (DC-DC降压, 给电机/传感器供电)
    │       │
    │       ├── AMS1117-3.3 ── 3.3V / 1A (给MCU供电)
    │       │
    │       └── 5V直接供电 (继电器/舵机/超声波)
    │
    ├── MP1584EN ── 12V / 2A (DC-DC升压, 如需给舵机/电磁铁)
    │
    └── 直接输出 3.7-4.2V (大电流电机直驱)
```

### 2.2 MP1584EN 降压电路 (推荐主力)

```
VIN (4.5-28V) ──┬── 22μF/50V电解 ── GND
                └── MP1584EN
                       │
              BST ── 100nF ── SW
              SW ── 33μH电感 ──┬── VOUT
                              ├── 22μF×2 (输出滤波)
                              └── 反馈分压
                                     │
                         R1 ──┬── FB
                              └── R2 ── GND

输出电压: Vout = 0.8 × (1 + R1/R2)
5V输出: R1=52.3kΩ, R2=10kΩ → 0.8×(1+5.23)=4.98V
12V输出: R1=140kΩ, R2=10kΩ → 0.8×(1+14)=12V
3.3V输出: R1=31.6kΩ, R2=10kΩ → 0.8×(1+3.16)=3.33V
```

### 2.3 AMS1117-3.3 LDO电路

```
VIN (≤15V) ──┬── 10μF电解 ── GND (输入滤波, 必需!)
             └── AMS1117-3.3
                    │
                   VOUT ──┬── 10μF电解 ── GND (输出滤波, 必需!)
                          └── 3.3V输出
注意: AMS1117的输入输出电容是必须的，否则会振荡!
```

### 2.4 MT3608 升压电路

```
VIN (2-24V) ──┬── 10μF ── GND
              └── MT3608
                     │
            SW ── SS34肖特基 ──┬── VOUT
            L ── 22μH电感 ────┘   ├── 22μF×2
                                  └── 反馈分压
                              R1 ──┬── FB
                                   └── R2 ── GND

5V输出: R1=23.7kΩ, R2=7.5kΩ
12V输出: R1=84.5kΩ, R2=7.5kΩ
```

## 三、完整电源板设计

### 3.1 推荐方案（电赛小车）

```
[18650×2 串联 7.4V]
      │
      ├── [MP1584EN] ── 5V/3A (传感器/舵机)
      │       │
      │       └── [AMS1117-3.3] ── 3.3V/800mA (MCU)
      │
      ├── [LM2596] ── 6V/3A (电机可选)
      │
      └── 直接 7.4V (大扭矩电机)

充电: [BQ24610] 或 [TP4056×2]
保护: [DW01+8205A] ×2
```

### 3.2 PCB布局要点

1. **大电流路径走线≥2mm**，最好铺铜
2. **DC-DC电感**靠近IC放置，输出电容靠近电感
3. **模拟电路电源**与数字电源用磁珠隔离
4. **电池接口**用大焊盘，XT60/XT30连接器
5. **保险丝**串联在电池输出端（自恢复5A）

## 四、电源指示与监控

### 4.1 电池电压检测
```
BAT+ ── R1(100kΩ) ──┬── ADC引脚
                     └── R2(100kΩ) ── GND
分压比: 0.5, 4.2V→2.1V (在3.3V ADC范围内)
ADC读数 → 电压 = ADC值 × 3.3 / 4095 × 2
```

### 4.2 LED指示
```
3.3V ── LED(绿色) ── 330Ω ── GND (电源OK)
5V   ── LED(蓝色) ── 470Ω ── GND (5V OK)
```

## 五、常见电源问题排查

| 问题 | 可能原因 | 解决 |
|------|----------|------|
| MCU反复重启 | 电池电压不足/内阻大 | 更换电池/加电容 |
| 电机堵转时MCU复位 | 电源内阻大、电压跌落 | 独立供电/加大电容 |
| ADC读数不准 | 电源纹波大 | 加LC滤波/独立供电 |
| 电机有异响 | PWM频率太低 | 提高PWM≥20kHz |
| 稳压器发烫 | 压差大、电流大 | 用DC-DC替代LDO |
