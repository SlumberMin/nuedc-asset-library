> nuedc-asset-library | 版本 V3 | PCB 接地设计规范与最佳实践

## 1. 接地基础概念

### 1.1 接地类型总览

| 接地类型 | 符号 | 说明 | 颜色标识 |
|----------|------|------|----------|
| 数字地 (DGND) | ⏚D | 数字电路参考地 | 绿色 |
| 模拟地 (AGND) | ⏚A | 模拟电路参考地 | 蓝色 |
| 功率地 (PGND) | ⏚P | 大电流功率电路地 | 棕色 |
| 信号地 (SGND) | ⏚S | 信号传输参考地 | 黄色 |
| 安全地 (PE) | ⏚ | 设备外壳保护地 | 黄绿色 |
| 机壳地 (CHASSIS) | ⏚ | 金属外壳接地 | 灰色 |

### 1.2 接地的目的

```
1. 提供参考电位（0V 基准）
2. 返回电流路径（信号回流）
3. 屏蔽干扰（法拉第笼）
4. 安全保护（漏电保护）
5. EMI 抑制（减小环路面积）
```

## 2. 接地方式详解

### 2.1 单点接地

```
适用场景：低频电路 (< 1MHz)，音频、传感器、低速数字

电路 A ──┐
         ├──── 公共接地点 (星形)
电路 B ──┘

优点：无地环路，低干扰
缺点：高频时阻抗大，走线长

┌──────────────────────────────────────┐
│           电路 A (数字)               │
│     GND ──────────────────────┐      │
│                               │      │
│           电路 B (模拟)        │      │
│     GND ──────────────────┐   │      │
│                           │   │      │
│           电路 C (功率)    │   │      │
│     GND ──────────────┐   │   │      │
│                       │   │   │      │
│                    ┌──┴───┴───┴──┐   │
│                    │  星形接地点  │   │
│                    └──────┬──────┘   │
│                           │          │
│                          GND        │
└──────────────────────────────────────┘
```

### 2.2 多点接地

```
适用场景：高频电路 (> 10MHz)，射频、高速数字

电路 A ──┬── GND平面
         │
电路 B ──┬── GND平面
         │
电路 C ──┴── GND平面

优点：低阻抗，短回流路径
缺点：可能有地环路
```

### 2.3 混合接地

```
适用场景：模数混合系统（电赛最常用）

┌─────────────────────────────────────────┐
│                                         │
│  ┌─────────┐    ┌─────────┐             │
│  │ 数字电路 │    │ 模拟电路 │             │
│  │  (MCU,   │    │ (ADC,   │             │
│  │  逻辑)   │    │  DAC,   │             │
│  └────┬────┘    │  运放)   │             │
│       │         └────┬────┘             │
│       │              │                  │
│       ▼              ▼                  │
│  ┌─────────┐    ┌─────────┐             │
│  │ DGND区域 │    │ AGND区域 │             │
│  └────┬────┘    └────┬────┘             │
│       │              │                  │
│       └──────┬───────┘                  │
│              │                          │
│         单点连接                         │
│         (磁珠/0Ω)                       │
│              │                          │
│              ▼                          │
│         主电源地                        │
│                                         │
└─────────────────────────────────────────┘
```

## 3. 数字地与模拟地分离

### 3.1 分离原则

```
✅ 正确做法：
┌─────────────┐     ┌─────────────┐
│  数字区域    │     │  模拟区域    │
│             │     │             │
│  DGND 铜皮  │     │  AGND 铜皮  │
│             │     │             │
└──────┬──────┘     └──────┬──────┘
       │                   │
       └─────────┬─────────┘
                 │
           单点桥接处
           (靠近 ADC)

❌ 错误做法：
- 数字地和模拟地大面积重叠
- 数字信号回流路径经过模拟区域
- 多处连接导致地环路
```

### 3.2 ADC/DAC 接地设计

```
STM32 ADC 接地最佳实践：

┌────────────────────────────────────────┐
│                                        │
│   ┌────────┐          ┌────────┐       │
│   │ MCU    │          │ 模拟   │       │
│   │        │          │ 信号源 │       │
│   │ VSSA ──┼──┐       │        │       │
│   │ VSS  ──┼──┼──┐    │ GND ───┼──┐   │
│   │        │  │  │    │        │  │   │
│   └────────┘  │  │    └────────┘  │   │
│               │  │                │   │
│    AGND平面   │  │   AGND平面     │   │
│               │  │                │   │
│               │  │                │   │
│               ▼  ▼                ▼   │
│          ┌────────────────────┐       │
│          │    AGND 与 DGND    │       │
│          │    单点桥接 (磁珠)   │       │
│          └─────────┬──────────┘       │
│                    │                  │
│                    ▼                  │
│                   GND                │
└────────────────────────────────────────┘

关键点：
1. VSSA 直接连接 AGND 平面
2. AGND 平面与 DGND 平面在 ADC 附近单点连接
3. 使用磁珠或 0Ω 电阻桥接
4. 模拟信号远离数字信号走线
```

### 3.3 接地连接方式对比

| 连接方式 | 适用场景 | 优缺点 |
|----------|----------|--------|
| 0Ω 电阻 | 简单分离 | 低成本，可调 |
| 磁珠 | 高频噪声抑制 | 阻隔高频，通直流 |
| 小电容 (10-100pF) | 射频旁路 | 高频短路 |
| 直接连接 | 低噪声系统 | 最低阻抗 |

## 4. 功率地设计

### 4.1 功率地布局原则

```
功率电流路径（以 Buck 为例）：

Vin ──┬── L ──┬── Vout
      │       │
     SW       C
      │       │
     Q1       │
      │       │
     GND ─────┘

⚡ 高频大电流回路：Q1 → C → SW → Q1
   这个环路面积必须最小！

布局要点：
1. 功率器件靠近放置
2. 功率地铜皮加宽（≥ 2mm/A）
3. 输入/输出电容靠近开关管
4. 功率地与信号地分离
```

### 4.2 大电流地线设计

```
电流与铜皮宽度关系（1oz 铜，温升 10°C）：

电流      最小宽度    推荐宽度
─────────────────────────────
0.5A      0.3mm       0.5mm
1A        0.5mm       1.0mm
2A        1.0mm       2.0mm
5A        2.5mm       5.0mm
10A       5.0mm       10.0mm

功率地走线建议：
- 宽度 ≥ 电流(A) × 1mm
- 使用铜皮而非走线
- 多层板使用地平面
- 过孔阵列连接各层地平面
```

### 4.3 功率地与信号地连接

```
┌──────────────────────────────────────┐
│                                      │
│   ┌─────────────┐  ┌─────────────┐  │
│   │  功率区域    │  │  信号区域    │  │
│   │ (电机驱动,  │  │ (MCU, ADC,  │  │
│   │  电源)      │  │  传感器)     │  │
│   │             │  │             │  │
│   │  PGND平面   │  │  SGND平面   │  │
│   └──────┬──────┘  └──────┬──────┘  │
│          │                │         │
│          └───────┬────────┘         │
│                  │                  │
│            单点汇合                 │
│           (电源输入端)              │
│                  │                  │
│                  ▼                  │
│                 GND                 │
└──────────────────────────────────────┘
```

## 5. 安全接地

### 5.1 安全地的作用

```
安全接地（PE - Protective Earth）：

1. 漏电保护：故障电流经 PE 流入大地
2. 外壳保护：金属外壳连接 PE
3. 静电泄放：ESD 通过 PE 泄放
4. EMI 屏蔽：机壳接地形成屏蔽体

符号：⏚ 或 ⊥
颜色：黄绿色线
```

### 5.2 安全接地设计要点

```
设备安全接地：

┌─────────────────────────────────────┐
│  金属外壳                           │
│  ┌─────────────────────────────┐    │
│  │                             │    │
│  │      PCB 电路板              │    │
│  │                             │    │
│  │      电路地 ──┬── 机壳地     │    │
│  │              │              │    │
│  │        1MΩ + 电容           │    │
│  │         │    │              │    │
│  └─────────┼────┼──────────────┘    │
│            │    │                   │
│            ▼    ▼                   │
│          金属外壳                    │
│            │                        │
│            ▼                        │
│        安全地 PE ─── 大地            │
│                                     │
└─────────────────────────────────────┘

安全地连接方式：
- 电路地通过 1MΩ 电阻接外壳（泄放静电）
- 并联 100pF-1nF 电容（旁路高频）
- 低阻抗路径接大地
```

### 5.3 实验室接地注意事项

```
⚠️ 电赛场地接地检查：

1. 确认电源插座有地线
2. 示波器探头接地夹 → 系统地
3. 多台仪器共地 → 避免地环路
4. 人体静电 → 摸金属外壳泄放

常见接地问题：
- 示波器地夹接错位置 → 短路
- 两台仪器地电位差 → 信号异常
- 人体静电损坏芯片 → 先触摸接地
```

## 6. PCB 接地布局实战

### 6.1 两层板接地设计

```
两层板接地策略：

顶层：信号走线 + 元器件
底层：尽可能完整的地平面

┌──────────────────────────────────────┐
│  顶层 (Top)                          │
│  ┌──────────────────────────────┐    │
│  │ ┌────┐ ┌────┐ ┌────┐        │    │
│  │ │ U1 │ │ U2 │ │ U3 │        │    │
│  │ └──┬─┘ └──┬─┘ └──┬─┘        │    │
│  │    │      │      │          │    │
│  │  ──┼──────┼──────┼── 信号线  │    │
│  │                              │    │
│  └──────────────────────────────┘    │
│                                      │
│  底层 (Bottom)                       │
│  ┌──────────────────────────────┐    │
│  │                              │    │
│  │      连续 GND 铜皮           │    │
│  │      (不要被信号线割裂)       │    │
│  │                              │    │
│  └──────────────────────────────┘    │
└──────────────────────────────────────┘

关键规则：
1. 底层地平面尽可能完整
2. 信号线下方不要走垂直走线
3. 每个 IC 旁放置去耦电容
4. 过孔旁放置回流地过孔
```

### 6.2 四层板接地设计

```
四层板叠层推荐：

Layer 1: 信号层 (Top)
Layer 2: 地平面 (GND) ← 完整地平面
Layer 3: 电源平面 (VCC)
Layer 4: 信号层 (Bottom)

优势：
- Layer 1 信号 → Layer 2 地平面提供回流路径
- Layer 4 信号 → Layer 3 电源平面提供回流路径
- 地平面完整 → 最低阻抗回流

⚠️ 注意：
- Layer 2 (GND) 不要走信号线
- Layer 3 (VCC) 不要走信号线
- 高速信号尽量走 Layer 1（直接参考 GND）
```

### 6.3 去耦电容接地

```
去耦电容接地方案：

          电源
           │
      ┌────┴────┐
      │    IC    │
      │  VCC  GND│
      └──┬────┬─┘
         │    │
        C1   │   ← 100nF 瓷片电容
         │   │      尽量靠近 IC 电源引脚
         ▼   ▼
        GND 平面

放置顺序（从 IC 引脚出发）：
1. IC VCC 引脚 → 100nF 电容 → GND 引脚
2. 电容接地端直接到地平面（最短路径）
3. 大电容 (10µF) 放在电源入口
4. 小电容 (100nF) 放在每个 IC 旁边
```

## 7. 接地问题排查

### 7.1 常见接地问题

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| ADC 读数跳动 | 数字噪声耦合到模拟地 | 分离数字地/模拟地 |
| 电源纹波大 | 功率回路阻抗高 | 加宽功率地铜皮 |
| 通信丢帧 | 信号回流路径不连续 | 补充地过孔 |
| EMI 超标 | 地环路面积大 | 缩小回路面积 |
| 运放输出噪声 | 地线压降 | 星形接地 |
| 系统复位 | 电机反电动势干扰 | 功率地/信号地分离 |

### 7.2 接地调试技巧

```
调试步骤：

1. 检查地平面连续性
   → 万用表测量各点地间电阻（应 < 0.1Ω）

2. 检查地环路
   → 示波器测量不同接地点间的电压差
   → 正常应 < 50mV

3. 检查高频干扰
   → 示波器 AC 耦合测量地线噪声
   → 频谱分析仪检查 EMI

4. 热像仪检查
   → 地线过热 = 阻抗过高/电流过大
```

## 8. 接地设计速查卡

### 8.1 频率与接地方式

```
频率          接地方式         布局要点
───────────────────────────────────────────
DC ~ 10kHz    单点接地         星形拓扑
10kHz ~ 1MHz  混合接地         区域分离，单点桥接
> 1MHz        多点接地         完整地平面
> 100MHz      多点接地         多层板，紧密耦合
```

### 8.2 接地设计检查清单

#### 设计阶段
- [ ] 确定接地类型（数字/模拟/功率/安全）
- [ ] 选择接地方式（单点/多点/混合）
- [ ] 规划地平面分区
- [ ] 设计桥接点位置

#### 布局阶段
- [ ] 地平面完整，无大面积割裂
- [ ] 去耦电容靠近 IC 放置
- [ ] 功率回路面积最小
- [ ] 模拟信号远离数字信号

#### 验证阶段
- [ ] 各地间电阻 < 0.1Ω
- [ ] 地线压降 < 50mV
- [ ] ADC 噪声 < 1 LSB
- [ ] 无明显 EMI 问题

## 9. 电赛接地设计实战建议

### 9.1 通用建议

```
电赛项目接地设计简化流程：

1. 先规划分区
   ├── 数字区域 (MCU, 逻辑)
   ├── 模拟区域 (ADC, 传感器)
   └── 功率区域 (电机, 电源)

2. 各区域内部多点接地
3. 区域间单点桥接（靠近 ADC/DAC）
4. 功率地单独走宽铜皮
5. 每个 IC 旁放 100nF 去耦电容
```

### 9.2 常见错误避免

```
❌ 错误 1: 信号线跨越地平面分割
   → 信号回流路径断裂 → EMI 大增

❌ 错误 2: 去耦电容离 IC 太远
   → 引线电感抵消电容作用

❌ 错误 3: 示波器地夹夹错位置
   → 短路！正确接法：地夹接系统 GND

❌ 错误 4: 大电流走细线
   → 地线压降导致测量误差
```
