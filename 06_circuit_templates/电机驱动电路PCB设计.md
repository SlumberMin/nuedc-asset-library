# 电机驱动电路PCB设计指南

> 电赛电机驱动板设计规范与注意事项

---

## 1. 设计原则概述

电机驱动电路是电赛中最容易出问题的部分。设计要点：

1. **大电流回路最短** — 功率走线尽量短粗
2. **电源去耦充分** — 大容量电解 + 小容量MLCC
3. **强弱电隔离** — 控制信号远离功率部分
4. **散热设计** — 功率器件必须考虑散热
5. **保护电路** — 续流二极管、TVS、保险丝

---

## 2. 常用驱动方案PCB设计

### 2.1 L298N方案

**原理图要点：**
```
VS ──┬── 电机电源(8~46V)
     │
     ├── 电解电容 100µF/50V (靠近VS引脚)
     └── MLCC 100nF (靠近VS引脚)

VSS ── 逻辑电源5V
      └── MLCC 100nF (靠近VSS引脚)

GND ── 大面积铺铜

OUT1/OUT2 ── 电机A
OUT3/OUT4 ── 电机B
```

**PCB布局：**
```
┌─────────────────────────────┐
│  电源输入区                    │
│  [VS] [C_bulk] [C_bypass]    │  ← 电源输入在板边缘
│                              │
│  ┌──────────┐               │
│  │  L298N   │               │
│  │          │               │
│  └──────────┘               │
│      ↕ 电机输出              │
│  [OUT] [续流二极管]           │
│                              │
│  信号区                      │
│  [MCU接口] [EN/IN信号]       │  ← 信号在板另一侧
└─────────────────────────────┘
```

**走线要求：**
| 网络 | 线宽 | 说明 |
|------|------|------|
| VS (电机电源) | ≥40mil (1mm) | 越宽越好 |
| OUT (电机输出) | ≥40mil | 与VS同宽 |
| GND | 铺铜 | 整面铺铜 |
| VSS (5V) | 15~20mil | 标准信号线 |
| EN/IN | 8~10mil | 信号线 |

---

### 2.2 TB6612FNG方案 (推荐)

**原理图要点：**
```
VM ── 电机电源(2.5~13.5V)
     └── 电解电容 47µF + MLCC 100nF

VCC ── 逻辑电源(2.7~5.5V)
      └── MLCC 100nF

STBY ── 上拉到VCC(使能)

PWMA/PWMB ── PWM输入
AIN1/AIN2 ── A桥方向
BIN1/BIN2 ── B桥方向
```

**PCB布局优势：**
- TB6612FNG内置MOSFET，无需续流二极管
- SOP-24封装，比L298N更紧凑
- 发热小，一般不需散热片

---

### 2.3 BTS7960大电流方案

**适用：** 电流>3A的大电机

**原理图要点：**
```
BTS7960 × 2 构成全桥

VCC ── 4.5~28V (电机电源)
     └── 电解 220µF/35V + MLCC 220nF × 2

IS引脚 ── 电流检测(可选)

INH ── 使能(高有效)
IN  ── PWM输入
OUT ── 电机输出
```

**PCB设计要点：**
- 每个BTS7960散热焊盘必须大面积铺铜
- 建议使用2oz铜厚
- 铺铜面积至少 2cm × 2cm

## 3. 续流保护设计

### 3.1 续流二极管

电机是感性负载，关断时会产生反电动势，必须加续流保护。

**方案选择：**

| 二极管 | 正向压降 | 反向恢复 | 适用场景 |
|--------|----------|----------|----------|
| 1N4007 | 1.1V | 30µs | 低频PWM(<1kHz) |
| UF4007 | 1V | 75ns | 中频PWM |
| SS34(肖特基) | 0.5V | <10ns | 高频PWM(推荐) |
| SS54 | 0.5V | <10ns | 大电流 |

**布局要求：**
```
        ┌─── 电机正极 ───┐
        │                │
    ┌───┴───┐        ┌───┴───┐
    │ H桥高侧│        │D1(续流)│  ← 紧贴电机焊盘
    └───┬───┘        └───┬───┘
        │                │
        └─── 电机负极 ───┘
```

- 续流二极管应紧贴电机接线端子
- 走线尽可能短，减少寄生电感
- 每个半桥都需设置续流回路

## 4. 电源设计

### 4.1 电机电源与逻辑电源分离

```
电池 ──┬── 电机驱动(VM)  ← 大电流
       │
       └── LDO/DC-DC ── 逻辑电源(VCC)  ← 小电流
```

**PCB实现：**
- 电机电源和逻辑电源分别走线
- GND在单点汇聚(星形接地)
- 逻辑电源入口加磁珠隔离高频干扰

### 4.2 去耦电容布局

```
功率器件
    │
    ├──[100nF MLCC]── 紧贴引脚(距离<3mm)
    │
    ├──[10µF~100µF 电解]── 靠近器件(距离<10mm)
    │
    GND (大面积铺铜)
```

**电容推荐值：**

| 位置 | 电解电容 | 陶瓷电容 | 距离 |
|------|----------|----------|------|
| 电机电源入口 | 100~470µF/50V | 100nF | <5mm |
| 驱动芯片VM | 47~100µF | 100nF | <3mm |
| 驱动芯片VCC | — | 100nF + 10µF | <2mm |
| MCU电源 | — | 100nF + 10µF | <2mm |

## 5. PCB布局规范

### 5.1 板层规划(2层板)

```
顶层: 信号线 + 功率线
       ┌─────────────────────────────┐
       │  [电源输入] [驱动芯片]       │
       │  [MCU]     [接插件]          │
       │  [去耦电容]                  │
       └─────────────────────────────┘

底层: GND铺铜 + 散热铺铜
       ┌─────────────────────────────┐
       │  ████████████████████████   │
       │  ███████ GND铺铜 ███████   │
       │  ████████████████████████   │
       └─────────────────────────────┘
```

### 5.2 关键布局规则

| 规则 | 要求 | 原因 |
|------|------|------|
| 功率线宽 | ≥1mm (20A时≥3mm) | 降低电阻和发热 |
| 去耦电容 | 紧贴芯片电源引脚 | 降低阻抗 |
| 功率环路 | 面积最小化 | 降低EMI |
| 大电流铺铜 | 2oz铜厚或加锡 | 提高载流能力 |
| 接插件 | 板边缘 | 方便接线 |
| 散热孔 | 功率器件下方 | 导热到底层铺铜 |
| 信号线 | 远离功率线 | 减少干扰 |

### 5.3 接插件选择

| 类型 | 额定电流 | 适用 | 推荐型号 |
|------|----------|------|----------|
| 排针 | 1~3A | 信号/小电流 | 2.54mm排针 |
| KF2EDGK | 5~10A | 电机/电源 | 5.08mm端子 |
| XT30 | 15A | 电池接口 | Amass XT30 |
| XT60 | 30A | 大电流 | Amass XT60 |
| 螺丝端子 | 10~30A | 大电流可拆卸 | TB系列 |

## 6. EMC设计要点

### 6.1 减少EMI的措施

1. **PWM频率选择**
   - 电机PWM推荐 10~25kHz
   - <20kHz可能有啸叫，>25kHz增加开关损耗

2. **布线技巧**
   - PWM信号线尽量短，加地线包围
   - 电机线使用双绞线
   - 电源入口加TVS管和LC滤波

3. **屏蔽**
   - 电机线使用屏蔽线(可选)
   - 信号线远离电机线

### 6.2 TVS保护

```
电源入口:
  VS ──┬── TVS(SMBJ36A) ── GND
       │
       └── 到驱动电路
```

## 7. 调试检查清单

### 设计阶段
- [ ] 电机驱动方案选择是否合理(电流/电压匹配)
- [ ] 续流保护是否完整(每个半桥)
- [ ] 去耦电容值和位置是否正确
- [ ] 功率走线宽度是否足够
- [ ] 散热设计是否充分
- [ ] 接插件额定电流是否匹配

### 制板阶段
- [ ] 铜厚是否为1oz或2oz
- [ ] 过孔是否做了加锡处理
- [ ] 关键走线是否做了加锡

### 焊接阶段
- [ ] 功率器件散热焊盘是否充分焊接
- [ ] 电解电容极性是否正确
- [ ] 大电流接插件焊接牢固

### 调试阶段
- [ ] 先用万用表检查短路
- [ ] 先不接电机，测试驱动信号
- [ ] 先用低压小电流测试
- [ ] 逐步加负载到额定

## 8. 常见问题与解决

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 电机不转 | 驱动信号错误 / 电源不足 | 检查PWM和方向信号 |
| 电机抖动 | PWM频率太低 | 提高到20kHz以上 |
| 驱动芯片发烫 | 电流过大 / 散热不足 | 加散热片/降负载 |
| MCU复位 | 电机干扰传导到电源 | 加LC滤波/分离电源 |
| 嘎嘎响 | PWM频率在音频范围 | 调整到25kHz以上 |
| 烧保险丝 | 短路 / 堵转电流过大 | 检查接线/加软启动 |

## 9. 推荐设计参考

### 电赛常用电机驱动板尺寸
- 单路驱动(小型车): 40mm × 30mm
- 双路驱动(标准车): 60mm × 40mm
- 四路驱动(复杂题): 80mm × 60mm

### 推荐PCB打样参数
- 板厚: 1.6mm
- 铜厚: 1oz (小电流) / 2oz (大电流)
- 最小线宽/线距: 6mil/6mil
- 最小过孔: 0.3mm
- 表面处理: HASL (有铅) 或 ENIG (无铅)
