# 电赛显示电路模板库

## 一、OLED显示电路

### 1.1 0.96寸 OLED (SSD1306) - I2C接口

```
引脚定义(4针I2C版)：
  VCC → 3.3V
  GND → GND
  SCL → MCU_SCL (如PB6)
  SDA → MCU_SDA (如PB7)

接线图：
  MCU_PB6(SCL) ──┬── OLED_SCL
                  │
                 4.7kΩ → VCC(3.3V)
                  
  MCU_PB7(SDA) ──┬── OLED_SDA
                  │
                 4.7kΩ → VCC(3.3V)
                  
  3.3V ──────────── OLED_VCC
  GND  ──────────── OLED_GND
```

**SSD1306 I2C地址：** 0x3C (SA0=0) 或 0x3D (SA0=1)

### 1.2 0.96寸 OLED (SSD1306) - SPI接口

```
引脚定义(7针SPI版)：
  VCC  → 3.3V
  GND  → GND
  SCL  → MCU_SCK (如PA5)
  SDA  → MCU_MOSI (如PA7)
  RES  → MCU_RESET或GPIO (如PB0)
  DC   → MCU_GPIO (如PB1)  [高=数据, 低=命令]
  CS   → MCU_CS (如PA4)

接线图：
  MCU_PA5(SCK)  ──────── OLED_SCL
  MCU_PA7(MOSI) ──────── OLED_SDA
  MCU_PA4(CS)   ──────── OLED_CS
  MCU_PB1(DC)   ──────── OLED_DC
  MCU_PB0(RES)  ──────── OLED_RES
  3.3V          ──────── OLED_VCC
  GND           ──────── OLED_GND
```

### 1.3 1.3寸 OLED (SH1106) - I2C接口

```
与SSD1306接线完全相同，区别：
  - 分辨率同为128×64
  - SH1106内部RAM为132×64，显示偏移2像素
  - 驱动代码需修改初始化命令
  - I2C地址同为0x3C
```

### 1.4 0.91寸 OLED (SSD1306) - I2C接口

```
分辨率：128×32
接线：同0.96寸I2C版本
初始化需修改COM配置(扫描方向)
```

### 1.5 OLED显示中文注意事项

```
字模取模方式：
  - 取模软件：PCtoLCD2002 / 字模提取工具
  - 取模方式：阴码+逐列式+顺向
  - 16×16中文字模：32字节/字
  - 需要自己编写显示函数，OLED库一般不含中文
```

## 二、TFT显示屏电路

### 2.1 1.8寸 TFT (ST7735) - SPI接口

```
引脚定义(8针)：
  VCC  → 3.3V
  GND  → GND
  SCL  → MCU_SCK
  SDA  → MCU_MOSI
  CS   → MCU_CS
  DC/RS→ MCU_GPIO (高=数据, 低=命令)
  RST  → MCU_GPIO (复位)
  BLK  → MCU_PWM或3.3V (背光)

接线图：
  MCU_SPI_SCK  ──────── TFT_SCL
  MCU_SPI_MOSI ──────── TFT_SDA
  MCU_SPI_CS   ──────── TFT_CS
  MCU_GPIO_1   ──────── TFT_DC
  MCU_GPIO_2   ──────── TFT_RST
  MCU_PWM      ──────── TFT_BLK (背光控制)
  3.3V         ──────── TFT_VCC
  GND          ──────── TFT_GND

特性：
  - 分辨率：128×160 或 128×128
  - 色深：65K色(16bit RGB565)
  - SPI时钟：最高15MHz
  - 驱动IC：ST7735S
```

### 2.2 2.4寸 TFT (ILI9341) - SPI接口

```
接线与ST7735相同，区别：
  - 分辨率：240×320
  - 驱动IC：ILI9341
  - SPI时钟：最高10MHz(写)/20MHz(读)
  - 需要较大RAM用于帧缓冲(全屏150KB)

引脚定义(同上8针)：
  VCC  → 3.3V
  GND  → GND
  SCL  → MCU_SCK
  SDA  → MCU_MOSI
  CS   → MCU_CS
  DC   → MCU_GPIO
  RST  → MCU_GPIO
  BLK  → MCU_PWM/3.3V
  LED  → 3.3V (背光正极)
```

### 2.3 2.4寸 TFT (ILI9341) - 并口16位接口(STM32 FSMC)

```
适用于STM32F103/F407等有FSMC的芯片：
  
  数据线D0~D15 → FSMC_D0~D15
  地址线A0~A16 → FSMC_A0~A16 (DC/RS接到地址线)
  CS → FSMC_NE1/NE2/NE3/NE4
  WR → FSMC_NWE
  RD → FSMC_NOE
  RST → GPIO
  BLK → PWM

FSMC配置要点：
  - 使用Bank1 NOR/SRAM模式
  - DC接到地址线，区分命令(地址=0)和数据(地址=1)
  - 并口速度比SPI快10倍以上
```

### 2.4 2.8寸/3.5寸 TFT (ILI9486/ILI9488) - SPI/并口

```
ILI9486:
  - 分辨率：320×480
  - SPI速度较慢，建议并口
  - 常见于Arduino Mega屏

ILI9488:
  - 分辨率：320×480
  - 色深支持RGB666(262K色)
  - SPI模式较常用

并口接线参考ILI9341 FSMC配置
```

### 2.5 1.54寸 TFT (ST7789) - SPI接口

```
特性：
  - 分辨率：240×240 (方形)
  - 驱动IC：ST7789V
  - SPI最高频率：约60MHz(部分模块)
  - 色深：65K色(RGB565)
  
接线同ST7735，初始化命令不同
偏移设置：行偏移+列偏移各40像素(ST7789V内部240×320)
```

## 三、LCD显示电路

### 3.1 LCD1602 (字符型LCD) - 并口接口

```
引脚定义(16针)：
  VSS(1) → GND
  VDD(2) → 5V
  V0(3)  → 对比度调节(接10kΩ电位器中间脚)
  RS(4)  → MCU_GPIO [0=命令, 1=数据]
  RW(5)  → GND (只写模式)
  E(6)   → MCU_GPIO (使能，下降沿触发)
  D0~D7  → 数据线(4位模式只用D4~D7)
  A(15)  → 背光+(5V)
  K(16)  → 背光-(GND)

4位模式接线(推荐)：
  MCU_PA0 → RS
  MCU_PA1 → E
  MCU_PA4 → D4
  MCU_PA5 → D5
  MCU_PA6 → D6
  MCU_PA7 → D7
  
  V0 → 10kΩ电位器 → GND (调节对比度)
  RW → GND
```

### 3.2 LCD1602 - I2C接口(PCF8574转接板)

```
        PCF8574 I2C转接板
        ┌──────────────┐
  SDA──┤SDA     P0(RS) ├── LCD_RS
  SCL──┤SCL     P1(RW) ├── LCD_RW
       │        P2(EN) ├── LCD_E
       │        P3(BL) ├── 背光
       │     P4~P7     ├── D4~D7
       │              │
  VCC──┤VCC        GND├── GND
        └──────────────┘

I2C地址：0x27(默认) 或 0x3F
优点：只需2根线(SDA+SCL)
```

### 3.3 LCD12864 (图形/汉字LCD) - 并口接口

```
引脚定义(20针)：
  VSS → GND
  VDD → 5V
  V0  → 对比度
  RS(CS) → MCU_GPIO
  RW(SID) → MCU_GPIO (串行数据)
  E(SCLK) → MCU_GPIO (串行时钟)
  DB0~DB7 → 数据线(并口模式)
  PSB → 高=并口, 低=串口
  RST → MCU_GPIO (复位)
  BLA → 背光+(5V)
  BLK → 背光-(GND)

串行模式(3线)：
  PSB → GND (选择串口)
  CS  → MCU_GPIO
  SID → MCU_GPIO (数据)
  SCLK→ MCU_GPIO (时钟)
```

## 六、电赛显示方案推荐

| 竞赛场景 | 推荐显示 | 理由 |
|---------|---------|------|
| 参数显示(电压/频率) | OLED 0.96寸 | 体积小，够用 |
| 图形化界面 | TFT 2.4寸 | 性价比高，分辨率够 |
| 波形显示(示波器类) | TFT 2.8寸/3.5寸 | 大屏显示波形清晰 |
| 调试信息 | LCD1602(I2C) | 只需2线，不占资源 |
| 菜单系统 | OLED + 按键 | 简洁高效 |
| 高级GUI | TFT + LVGL | 专业界面效果 |
| 多参数同时显示 | LCD12864 | 中文支持好 |

## 七、显示初始化代码模板

### STM32 HAL I2C OLED初始化

```c
// OLED I2C地址
#define OLED_ADDR  0x3C
#define OLED_CMD   0x00
#define OLED_DATA  0x40

void OLED_Init(void) {
    uint8_t init_cmds[] = {
        0xAE,  // Display OFF
        0xD5, 0x80,  // Clock div
        0xA8, 0x3F,  // Multiplex(64)
        0xD3, 0x00,  // Display offset
        0x40,  // Start line
        0x8D, 0x14,  // Charge pump ON
        0x20, 0x02,  // Page addressing
        0xA1,  // Segment remap
        0xC8,  // COM scan direction
        0xDA, 0x12,  // COM pins config
        0x81, 0xCF,  // Contrast
        0xD9, 0xF1,  // Pre-charge
        0xDB, 0x40,  // VCOMH
        0xA4,  // Display from RAM
        0xA6,  // Normal display
        0xAF   // Display ON
    };
    
    HAL_Delay(100);
    for (int i = 0; i < sizeof(init_cmds); i++) {
        OLED_WriteCmd(init_cmds[i]);
    }
}

void OLED_WriteCmd(uint8_t cmd) {
    uint8_t data[2] = {OLED_CMD, cmd};
    HAL_I2C_Master_Transmit(&hi2c1, OLED_ADDR<<1, data, 2, 100);
}

void OLED_WriteData(uint8_t dat) {
    uint8_t data[2] = {OLED_DATA, dat};
    HAL_I2C_Master_Transmit(&hi2c1, OLED_ADDR<<1, data, 2, 100);
}
```
