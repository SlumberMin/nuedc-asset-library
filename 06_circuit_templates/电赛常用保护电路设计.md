# 电赛常用保护电路设计

> 电赛电路模板库 | 过流 · 过压 · ESD · 反接 · TVS · 保险丝

---

## 一、保护电路总览

| 保护类型 | 防护对象 | 关键器件 | 响应速度 |
|---------|---------|---------|---------|
| **过流保护** | 负载短路/过载 | 保险丝/MOSFET | μs~ms |
| **过压保护** | 电源尖峰/浪涌 | TVS/稳压管 | ns |
| **ESD保护** | 静电放电 | TVS阵列/ESD二极管 | <1ns |
| **反接保护** | 电源反接 | MOSFET/二极管 | 即时 |
| **欠压锁定** | 电源不稳 | UVLO芯片/比较器 | ms |
| **过温保护** | 器件过热 | 热敏电阻/温控IC | ms |

## 二、过流保护

### 2.1 保险丝（一次性）

```
         ┌───── Fuse ─────┐
VCC ─────┤  额定电流Ifuse   ├─────→ 负载
         └────────────────┘

选型规则：
  Ifuse ≥ 1.5 × I_normal（正常工作电流）
  Ifuse ≤ 0.8 × I_max（导线/PCB最大承载）
```

**常用型号：**

| 类型 | 规格 | 特点 |
|------|------|------|
| 快断型 | 0.5A~10A | 过流立即熔断 |
| 慢断型 | 0.5A~10A | 允许短暂浪涌（电机启动） |
| 自恢复PPTC | 0.1A~3A | 过流断开，冷却后自恢复 |

### 2.2 自恢复保险丝（PPTC）

```
         ┌──── PPTC ────┐
VCC ─────┤  额定电流Ihold  ├─────→ 负载
         └───────────────┘

常用型号：
  MF-MSMF050：Ihold=0.5A, Vmax=6V
  MF-MSMF110：Ihold=1.1A, Vmax=6V
  MF-MSMF200：Ihold=2.0A, Vmax=6V
  MF-MSMF260：Ihold=2.6A, Vmax=6V

优点：自恢复，免更换
缺点：响应慢（ms级），有残余电阻
```

### 2.3 电子过流保护（MOSFET限流）

```
           R_sense
VCC ──┬──/\/\/──┬────→ 负载
      │         │
      │    ┌────┤
      │    │  Q1(P-MOS)
      │    │    │
      └────┤  Gate ←── 运放比较器输出
           │
         GND

原理：
  I_load × R_sense 与 Vref 比较
  过流时运放输出翻转 → 关闭MOSFET
  
R_sense选型：
  R_sense = Vref / I_limit
  功率 = I_limit² × R_sense × 2（留余量）
```

**典型参数：**
```
I_limit = 2A, Vref = 0.2V
R_sense = 0.1Ω, 功率 ≥ 0.8W（选1W）
```

### 2.4 集成过流保护

| 芯片 | 通道 | 限流 | 特点 |
|------|------|------|------|
| **TPS2553** | 1 | 0.1~1.7A | USB限流开关 |
| **TPS25940** | 1 | 可调 | 电子保险丝，带软启动 |
| **LTC4213** | 1 | 可调 | 热插拔保护 |
| **SI5902** | 双 | 固定 | 双通道负载开关 |

## 三、过压保护

### 3.1 TVS二极管

```
         ┌── TVS ──┐
Signal ──┤         ├──→ 后级电路
         └─┬─────┬─┘
          GND   GND

TVS钳位原理：
  正常电压：TVS高阻，不影响电路
  过压时：TVS瞬间导通，钳位到安全电压
  响应时间：<1ns（极快）
```

**常用TVS选型：**

| 型号 | Vrwm | Vc@Ipp | Ipp | 应用 |
|------|------|--------|-----|------|
| **SMBJ5.0A** | 5.0V | 9.2V | 65A | 5V信号线 |
| **SMBJ12A** | 12V | 19.9V | 30A | 12V电源 |
| **SMBJ24A** | 24V | 38.9V | 15A | 24V电源 |
| **SMBJ36A** | 36V | 58.1V | 10A | 36V电源 |
| **PESD5V0S1BA** | 5.0V | 12V | - | USB数据线ESD |
| **SM712** | 3.3/5V | - | - | RS485专用 |

### 3.2 TVS选型规则

```
1. Vrwm ≥ 工作电压（正常时TVS不导通）
2. Vc < 后级器件最大耐压（钳位效果）
3. Ipp > 最大浪涌电流（不被烧毁）
4. 功率 ≥ 预期浪涌能量
```

### 3.3 稳压管保护

```
         ┌── Dz(5.1V) ──┐
Signal ──┤              ├──→ 负载
         └─────┬────────┘
              GND

简单过压保护：
  V > Vz + Vf → Dz导通，钳位到Vz
  适合低能量、非快速过压
  
常用型号：1N4733A (5.1V/1W)
```

### 3.4 齐纳+TVS组合保护

```
                 限流电阻
Input ────┬───/\/\/───┬────→ Output
          │          │
          │    ┌─────┤
          │    │  Dz │ (齐纳：精确钳位)
          │    │     │
          ├─── TVS ──┤ (TVS：吸收能量)
          │          │
         GND        GND

齐纳管：精确钳位到目标电压
TVS：吸收大的瞬态能量
组合：兼顾精度和能量吸收
```

## 四、ESD保护

### 4.1 ESD防护等级

| 标准 | 模型 | 测试电压 |
|------|------|---------|
| HBM (人体模型) | 100pF + 1.5kΩ | ±2kV~±8kV |
| MM (机器模型) | 200pF + 0Ω | ±200V |
| CDM (充电器件模型) | 自身电容 | ±500V~±1kV |

### 4.2 常用ESD保护器件

| 器件 | 通道 | 电容 | 钳位电压 | 应用 |
|------|------|------|---------|------|
| **PESD5V0S1BA** | 1 | 50pF | 12V | 通用I/O |
| **PESD5V0S2BT** | 2 | 35pF | 14V | USB差分对 |
| **TPD2E001** | 2 | 1.5pF | - | USB 2.0 |
| **TPD4E05U06** | 4 | 0.5pF | - | USB 3.0/高速信号 |
| **NUP4114** | 4 | 0.8pF | - | HDMI/高速差分 |
| **ESD5V3U4RSY** | 4 | 0.3pF | - | 超高速信号 |

### 4.3 ESD布局要点

```
✓ ESD器件尽量靠近连接器放置
✓ 接地走线尽量短而宽
✓ 信号线先经过ESD保护再进芯片
✓ ESD接地到机壳地（如果有机壳地）
✗ 不要让ESD电流流经信号地平面
```

## 五、电源反接保护

### 5.1 串联二极管（最简单）

```
VCC+ ──── D(肖特基) ────→ 负载
GND  ──────────────────→ 负载

优点：简单
缺点：正向压降0.3~0.7V，功耗 = I × Vf

适用：小电流（<500mA）场合
推荐型号：SS34 (3A/40V), SS54 (5A/40V)
```

### 5.2 P-MOSFET反接保护（推荐）

```
VCC+ ────┬─────── S ── D ────→ 负载
         │          G
         └──────────┘
         
         ┌── R(10kΩ) ──┐
VCC+ ────┤              ├──→ Gate
         └──────────────┘
                              GND ──→ 负载GND

原理：
  正接：Vgs < Vth → MOSFET导通 → Vf ≈ I×Rds(on) 极小
  反接：Vgs = 0 → MOSFET截止 → 电路断开

推荐型号：
  AO3401 (P-MOS, -30V, -4A, Rds=65mΩ)
  SI2301 (P-MOS, -20V, -2.8A, Rds=120mΩ)
  DMP2045U (P-MOS, -20V, -3.8A, Rds=55mΩ)
```

### 5.3 N-MOSFET反接保护（低压降）

```
GND ────┬─────── S ── D ────→ 负载GND
        │          G
        └──────────┘
        
        ┌── R(10kΩ) ──┐
VCC+ ───┤              ├──→ Gate
        └──────────────┘

原理同P-MOS，但放在GND路径
Rds(on)可更低（N-MOS性能优于P-MOS）
注意：浮地问题，不适合有模拟地的场合
```

### 5.4 理想二极管（集成方案）

| 芯片 | 电压 | 电流 | Vf | 特点 |
|------|------|------|----|------|
| **LTC4357** | 9~80V | - | 极低 | 单通道理想二极管 |
| **LTC4359** | 4~80V | - | 极低 | 正/负压均可 |
| **TPS2419** | 9~20V | - | 极低 | 热插拔+理想二极管 |

## 六、综合保护方案

### 6.1 电源输入全方位保护

```
                PPTC     P-MOS        滤波
VIN+ ──┬──TVS──┤保险丝├──┤反接保护├──┤LC├──→ VCC
       │        │         │           │
       │        │         │     C_out │
      GND      GND       GND   ├─┤   GND
                               
组成：TVS(浪涌) + PPTC(过流) + P-MOS(反接) + LC(滤波)
```

### 6.2 USB端口保护

```
USB_VBUS ──┬── TVS(5V) ──┬── PPTC(500mA) ──→ 5V
           │             │
          GND           GND

USB_D+ ──┬── ESD(TPD2E001) ──→ MCU_D+
         │
        GND

USB_D- ──┬── ESD(TPD2E001) ──→ MCU_D-
         │
        GND
```

### 6.3 电机驱动保护

```
V_Motor ──┬── TVS ──┬── PPTC ──→ 驱动器VCC
          │         │
         GND       GND

H桥输出端：
OutA ──┬── TVS(双向) ──┬── 电机
       │               │
      GND             GND

续流二极管（内置在驱动IC中或外加）：
防止电机反EMF损坏驱动器
```

### 6.4 传感器接口保护

```
Signal ──┬── R(100Ω) ──┬── ESD ──→ MCU
         │             │
         │           GND
         │
         ├── Dz(钳位到VCC+0.3V)
         │
        GND

R：限流，防止ESD残余电压过大
Dz：钳位到安全电压
ESD：TVS阵列吸收静电
```

## 七、PCB布局要点

```
保护器件放置原则：
1. TVS/ESD → 尽量靠近接口连接器
2. 保险丝/PPTC → 电源入口处
3. 过流检测R_sense → 靠近功率MOSFET
4. 接地走线 → 短而宽，避免形成环路
5. 大电流保护器件 → 注意散热

关键布线规则：
- TVS到GND的走线长度 < 5mm
- ESD保护器件到连接器的距离 < 2mm
- 电源保护器件的地要接在一起
- 模拟地和数字地在保护器件处单点连接
```

## 八、电赛常见保护需求速查

| 场景 | 推荐方案 |
|------|---------|
| 5V USB供电 | PPTC + TVS(5V) + ESD |
| 12V适配器输入 | PPTC + P-MOS反接 + TVS(18V) |
| 锂电池供电 | P-MOS反接 + PPTC + UVLO |
| 电机驱动输出 | TVS(双向) + 续流二极管 |
| 按键/旋钮输入 | ESD + RC滤波 |
| RS485通信 | SM712 + 限流电阻 |
| ADC模拟输入 | RC低通 + 钳位二极管 |
| 大功率MOS驱动 | Rg电阻 + TVS(GS间) |
