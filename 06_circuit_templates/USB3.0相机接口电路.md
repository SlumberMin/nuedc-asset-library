# USB 3.0 相机接口电路设计指南

## 1. USB 3.0 基础参数

| 参数 | 规格 |
|------|------|
| 速率 | 5 Gbps (SuperSpeed) |
| 信号对 | TX+/TX-, RX+/RX- (差分, 90Ω阻抗) |
| 电压 | VBUS = 5V, 信号电平 ~400mVp-p |
| 最大电流 | 900mA (USB 3.0), 1.5A/3A (BC 1.2/Type-C) |
| 连接器 | Type-A, Type-C, Micro-B |

---

## 2. 供电设计

### 2.1 基本供电电路

```
VBUS(5V) ──[自恢复保险丝 500mA/1A]──┬──→ 相机 VCC
                                      │
                                    [10μF + 100nF] 去耦
                                      │
                                     GND
```

### 2.2 带负载开关的受控供电

```
VBUS ──→ TPS2553(使能可控负载开关)
          EN ←── GPIO控制（相机电源管理）
          OUT ──→ [10μF + 100nF] ──→ 相机 VCC
          FAULT ──→ GPIO输入（过流报警）
```

**TPS2553 关键参数：**
- 限流可调（外部电阻设置 100mA~1.1A）
- 过流保护自动关断
- 热关断保护

### 2.3 多相机独立供电

```
5V/3A 电源
  ├──[TPS2553]──→ 相机1 (限流1A)
  ├──[TPS2553]──→ 相机2 (限流1A)
  └──[TPS2553]──→ 相机3 (限流1A)
  每路独立控制使能
```

## 3. 信号完整性设计

### 3.1 差分走线要求

| 规范 | 要求 |
|------|------|
| 特征阻抗 | 90Ω ±10% (差分) |
| 走线长度匹配 | TX+/TX- 差值 < 5mil |
| RX+/RX- 差值 | < 5mil |
| 层间过孔 | 尽量避免，必要时 ≤ 2个 |
| 走线间距 | 差分对间距 ≥ 2倍线宽（对间隔离） |
| 参考地平面 | 必须有完整参考地，禁止地平面割裂 |

### 3.2 PCB 布局要点

```
[USB连接器] ──→ [ESD保护] ──→ [共模扼流圈] ──→ [SoC USB PHY]
     │              │              │
   短而直        就近放置       靠近连接器
```

**关键规则：**
1. ESD 器件（TPD4S012 / USBLC6-2）紧贴连接器放置
2. 共模扼流圈（如 ACM2012）放在 ESD 之后、SoC 之前
3. 差分对走线保持平行、等长、等距
4. 走线下方参考地平面完整连续
5. 差分对两侧留 ≥ 3W 间距的禁布区

### 3.3 USB 3.0 连接器引脚定义

```
USB 3.0 Type-A:
  ┌─────────────────────┐
  │ VBUS  D-  D+  GND   │  ← USB 2.0 (前排)
  │ SSTX+ SSTX- GND SSRX+ SSRX- │  ← USB 3.0 (后排)
  └─────────────────────┘

USB 3.0 Type-C (简化):
  A2/A3: SSTX+/SSTX-
  B11/B10: SSRX+/SSRX-
  A6/A7: D+/D- (USB 2.0)
  A1/B1/B12/A12: VBUS
  A4/A9/B4/B9: GND
```

## 4. ESD 与保护电路

### 4.1 推荐 ESD 方案

```
USB连接器 ──→ TPD4S012 (集成ESD + 过压保护)
               │ VCC=5V
               │ OVP=6V
               │ ESD: ±15kV HBM
               ──→ SoC USB端口
```

**备选方案：**
- USBLC6-2SC6（超低电容 ESD，0.5pF）
- PRTR5V0U2X（专用 USB3.0 ESD，0.3pF）

### 4.2 过压/过流保护组合

```
VBUS ──[自恢复保险丝]──[TVS 5V(SMBJ5.0A)]──→ 系统
D+/D- ──[TPD4S012 ESD]──→ SoC
SSTX/SSRX ──[USBLC6-2]──→ SoC
```

## 6. 延长线缆设计

USB 3.0 标准线缆最长 **3米**。电赛中常需延长：

### 6.1 主动式延长方案
```
OPi USB3.0 ──→ [1m无源线] ──→ [USB3.0 Hub/中继器] ──→ [相机]
                              如 TUSB8041 / FE1.1s
```

### 6.2 信号调理（长线缆）
- 差分线上串联 22Ω 匹配电阻（靠近发送端）
- 使用屏蔽双绞线缆
- 避免与电机/电源线并行走线

---

## 7. 电赛实战建议

1. **优先使用短线缆**（<1m），减少信号衰减
2. **相机独立供电**，不要与电机共享 USB 电源
3. **固定相机**，比赛振动会导致图像模糊
4. **测试光照条件**，赛场光线可能与实验室差异大
5. **准备备份相机**，USB接口频繁插拔易损坏
6. **散热考虑**，长时间高分辨率采集会发热降频
