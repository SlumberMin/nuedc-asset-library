# 电源电路设计指南

## 一、概述

电源是系统的命脉。本文档涵盖锂电池充放电管理、多路稳压输出、电源保护电路三大核心模块。

---

## 二、锂电池充放电管理

### 2.1 单节锂电池（3.7V）充电电路 - TP4056

```
         USB 5V
           │
      [C1 100nF]──GND
           │
    ┌──────┴──────┐
    │   TP4056     │
    │  VIN  BAT+──> 锂电池正极
    │  GND  BAT-──> 锂电池负极
    │       STDB ──> [LED绿]──> VCC (充满指示)
    │       CHRG ──> [LED红]──> VCC (充电中指示)
    │       PROG──> [R_prog 1.2K]──> GND
    └─────────────┘

充电电流设定 (R_prog):
  1.2KΩ → 1000mA
  2KΩ  → 580mA
  10KΩ → 130mA

注意: 锂电池必须带保护板（过充/过放/过流保护）
```

### 2.2 多节串联充电管理 - BQ24610 (2~6S)

```
         电源输入 (12~24V)
              │
         [C_in 100uF]──GND
              │
    ┌─────────┴─────────┐
    │     BQ24610         │
    │  VCC  ACDRV ──> PMOS栅极 (输入MOSFET)
    │       BATDRV ──> 充电MOSFET
    │  SRP  SRN ──> 电流检测电阻 (10mΩ)
    │       MPPSET ──> 太阳能MPPT (可选)
    │       VFB ──> 分压电阻设定充电电压
    │       TS ──> NTC热敏电阻 (温度保护)
    │       ISET1 ──> 充电电流设定
    │  GND               │
    └─────────────────────┘

充电电压设定 (VFB):
  V_charge = 4.2V × 串联节数
  R_top / R_bot 分压比: VFB = 2.1V 时满电

充电电流设定:
  I_charge = V_iset / (R_sense × 10)
```

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| 充电IC（单节） | 线性充电 | TP4056 | 带保护板版本 |
| 充电IC（多节） | 同步降压充电 | BQ24610 | 2~6S |
| 充电IC（多节） | 同步充电 | CN3722 | 太阳能充电 |

| 功率MOSFET | P沟道 | AO4407 | 输入开关 |

## 三、多路稳压输出

### 3.1 线性稳压（LDO） - AMS1117

适用：输入输出压差小、电流<1A的场合

```
         VIN (5V)
           │
      [C_in 10uF]──GND
           │
    ┌──────┴──────┐
    │  AMS1117-3.3 │──── VOUT (3.3V)
    │  IN    OUT   │
    │  GND         │
    └──────┬──────┘
           │
      [C_out 22uF]──GND

效率 ≈ Vout/Vin = 3.3/5 = 66%
功耗 = (Vin - Vout) × I_load
例: 1.7V × 0.5A = 0.85W (需注意散热)
```

### 3.2 开关稳压（DC-DC Buck） - MP1584 / LM2596

适用：压差大、电流>1A、效率要求高

```
         VIN (7~28V)
              │
         [C_in 22uF/35V]──GND
              │
    ┌─────────┴─────────┐
    │     MP1584EN        │
    │  VIN  BST ──[C_boot 100nF]
    │       EN  ──> VIN (使能)
    │       FB  ──┬──> R_top ──> VOUT
    │             └──> R_bot ──> GND
    │       SW  ──┬──> L1 (4.7uH/3A)
    │             │         └──> VOUT
    │             └──> D1 (SS34 肖特基)
    │  GND               │     └──> GND
    └─────────────────────┘
                   │
              [C_out 22uF]──GND

输出电压设定:
  Vout = 0.8 × (1 + R_top/R_bot)
  例: R_top=30K, R_bot=10K → Vout = 0.8 × 4 = 3.2V

推荐电阻值:
  5V输出:  R_top=52.5K, R_bot=10K
  12V输出: R_top=140K,  R_bot=10K
  3.3V输出: R_top=31.25K, R_bot=10K
```

### 3.3 典型电赛电源树

```
锂电池 7.4V (2S) / 11.1V (3S)
    │
    ├──[DC-DC Buck MP1584]──> 5V (电机电源 / 逻辑电源)
    │     │
    │     ├──[LDO AMS1117]──> 3.3V (MCU/传感器)
    │     └──[LDO AMS1117]──> 3.3V (通信模块独立供电)
    │
    └──[直接或Boost]──> 电机高压 (特殊需求)

注意: LDO总输出电流不要超过AMS1117的1A限制
若3.3V总电流>800mA，改用DC-DC 3.3V方案
```

## 四、电源保护电路

### 4.1 反接保护 - P-MOSFET

```
         电池+ ──── S (PMOS源极)
                     G ──[R1 100K]──> 电池-
                     │
                  [D1 稳压管12V]
                     │
                    GND
                     D ──── 系统VCC

工作原理:
  正接: Vgs < Vth → PMOS导通 → 系统得电
  反接: Vgs = 0 → PMOS截止 → 系统断电
```

### 4.2 过流保护 - 电子保险丝

```
         V_IN
           │
      [R_sense 50mΩ]
           │
    ┌──────┴──────┐
    │   TPS2592xx  │
    │  IN    OUT ──> V_OUT (受保护输出)
    │       ILIM ──> [R_lim] ──> GND (限流设定)
    │       EN   ──> V_IN (使能)
    │       FLT  ──> [LED] (故障指示)
    │  GND         │
    └──────────────┘

限流设定:
  I_limit = 50mV / R_sense
  例: R_sense=50mΩ → I_limit=1A
```

### 4.3 过压保护 - TVS + 稳压管

```
         V_IN ──┬──[TVS SMBJ16A]──> GND
                │
                └──[稳压管 15V]──> GND (钳位)
                │
           [保险丝 PTC]
                │
            系统电源

TVS选型:
  工作电压 > 正常电源电压
  钳位电压 < 后级器件最大耐压
  功率 > 预期浪涌能量
```

### 4.4 欠压/过压检测 - 分压+比较器

```
         V_BAT
           │
      [R1 100K]──┬──> 比较器+ (LM393)
                 │
           [R2 10K]
                 │
                GND

         V_REF (TL431 2.5V) ──> 比较器-

         比较器输出 ──> MCU GPIO (电压报警)

欠压阈值设定:
  V_trip = V_ref × (R1 + R2) / R2
  例: 2.5V × 110K/10K = 2.75V (单节锂电欠压)
```

## 五、PCB布局建议

### 5.1 通用原则
- 电源走线加粗：1A电流 ≥ 1mm（外层），≥ 0.5mm（内层）
- 输入/输出电容紧贴IC引脚
- DC-DC的SW节点面积最小化（EMI源头）
- 大面积铺铜散热，功率器件下方打散热过孔

### 5.2 DC-DC特殊要求
- 功率电感紧贴SW引脚
- 续流二极管（如有）紧贴SW和GND
- 反馈分压电阻靠近FB引脚，远离SW节点
- 输入电容→IC→电感→输出电容形成紧凑回路

## 六、常见问题解答

### Q1: 电源纹波大，MCU工作不稳定？
- DC-DC输出加LC π型滤波（22μH+22μF×2）
- MCU供电用LDO二次稳压
- 去耦电容100nF紧贴MCU VCC引脚

### Q2: LDO发热严重？
- 检查压差×电流的功耗（>0.5W需散热）
- 大电流场合改用DC-DC方案
- 加铜皮散热面积

### Q3: 电机启动瞬间系统重启？
- 电机电源和逻辑电源分离
- 加大输入电容（470μF以上）
- 软件实现PWM软启动
- 加独立的MCU供电LDO

### Q4: 电池续航短？
- 测量各路实际电流消耗
- 空闲时关闭不必要模块（睡眠模式）
- DC-DC比LDO效率高50%+，优先选择
- 降低MCU运行频率可显著省电

### Q5: 多路电源上电时序问题？
- 先3.3V（MCU），再5V（外设）
- 用EN引脚+RC延时控制上电顺序
- 避免MCU的GPIO在未上电时被外设反灌电流
