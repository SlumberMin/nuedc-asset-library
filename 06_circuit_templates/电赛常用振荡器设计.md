## 2. RC振荡器

### 2.1 文氏桥振荡器 (正弦波)

```
电路:
          R₁    C₁
    ┌───┤R├──┤C├───┐
    │                 │
    │  ┌─────────┐   │
    ├──┤ -   Op  ├───┤
    │  │   Amp   │   │
    │  └────┬────┘   │
    │       │        │
    │  ┌────┘        │
    ├──┤ +           │
    │  │             │
    └──┼──┤R├──┤C├──┘
       │   R₂   C₂
       │
      GND

振荡条件:
  f₀ = 1 / (2π·R·C)
  增益 ≥ 3 (Rf/Rg ≥ 2)
  
稳幅方法:
  - 二极管限幅 (简单)
  - 热敏电阻NTC (线性好)
  - JFET自动增益控制 (失真低)
```

```c
// 文氏桥参数计算器
typedef struct {
    float R;      // 电阻值 (Ω)
    float C;      // 电容值 (F)
} WeinBridge_t;

float WeinBridge_Frequency(WeinBridge_t *wb) {
    return 1.0f / (2.0f * 3.14159f * wb->R * wb->C);
}

// 频率 → RC值推荐 (f: Hz)
// f=1kHz → R=10kΩ, C=15.9nF
// f=10kHz → R=10kΩ, C=1.59nF
// f=100kHz → R=1kΩ, C=1.59nF
```

### 2.2 双T陷波振荡器

```
         R    R
    ┌──┤R├─┤R├──┐
    │              │
    │    2C        │
    ├──┤C├────────├──→ 输出
    │              │
    └──┤R├─┐      │
         R  │ C   │
            └─┤├──┘

f₀ = 1 / (2π·R·C)

特点: 频率稳定性优于文氏桥
```

### 2.3 RC相移振荡器

```
三级RC移相网络 + 反相放大器

f₀ = 1 / (2π·R·C·√6) ≈ 1/(15.4·R·C)
增益 ≥ 29 (三级)

优点: 电路简单
缺点: 波形失真较大
```

## 3. LC振荡器

### 3.1 常见拓扑对比

| 振荡器 | 反馈方式 | 频率 | 特点 |
|--------|---------|------|------|
| Colpitts | 电容分频 | 最常用 | 稳定、易起振 |
| Hartley | 电感分频 | 中频 | 需中心抽头电感 |
| Clapp | 改进Colpitts | 高频 | 频率更稳定 |
| Butler | 晶体+LC | 高频 | 高Q值 |

### 3.2 Colpitts振荡器 (最常用)

```
         VCC
          │
          R_bias
          │
    ┌─────┤
    │     │
    L     C₁ ──→ 输出
    │     │
    ├─────┤
    │     │
    │     C₂
    │     │
   GND   GND

频率: f₀ = 1 / (2π·√(L · C₁·C₂/(C₁+C₂)))

设计示例 (1MHz):
  L = 100μH, C₁ = C₂ = 500pF
  f₀ = 1/(2π·√(100e-6 · 250e-12)) ≈ 1MHz
```

### 3.3 LC振荡器设计要点

```
起振条件:
  A·β ≥ 1 (环路增益 ≥ 1)
  通常设计 A·β ≈ 2~5 (确保起振)

影响频率稳定性的因素:
  1. 电感Q值 → 选用高Q铁氧体磁芯
  2. 电容温度系数 → NPO/C0G陶瓷电容
  3. 有源器件参数 → 选fT >> f₀的晶体管
  4. 电源波动 → 稳压供电
  5. 负载效应 → 缓冲隔离输出

实用建议:
  - 电赛中LC振荡器主要用于射频/通信题
  - 低频场景优先用DDS或RC
```

## 4. 晶体振荡器

### 4.1 晶振参数

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 标称频率 | 4/8/12/16/24/25/32MHz | STM32常用8/25MHz |
| 频率精度 | ±10~±50ppm | 普通晶振 |
| 温度特性 | ±20ppm (-20~70°C) | 普通晶振 |
| 负载电容 | 12~22pF | 需匹配 |
| ESR | 30~80Ω | 等效串联电阻 |

### 4.2 皮尔斯振荡器 (MCU外部晶振)

```
            ┌─────────────┐
    XTAL1 ──┤             ├── XTAL2
      │     │     MCU     │     │
      │     │  (内部反相器) │     │
     C₁     └─────────────┘    C₂
      │                         │
     GND                       GND

C₁, C₂: 负载电容 (通常10~22pF)
  谐振频率取决于晶振本身

设计要点:
  1. C₁=C₂=负载电容值 (查晶振数据手册)
  2. 走线尽可能短 (< 10mm)
  3. 两侧加地屏蔽
  4. R_feedback: MCU内部或外部1~10MΩ
```

### 4.3 有源晶振 (时钟模块)

```
         VCC
          │
    ┌─────┤
    │  有源晶振
    │  (4脚: VCC, GND, OUT, NC)
    │     │
    └──→ OUT ──→ MCU OSC_IN

优点: 输出稳定、无需匹配、抗干扰
缺点: 功耗较高、成本稍高

推荐型号:
  - 32.768kHz: DS3231(带RTC, ±2ppm)
  - 8/25MHz: SG-210 (普通), ECS-2520MV (小封装)
```

### 4.4 STM32晶振配置

```c
// 8MHz外部晶振 → 168MHz系统时钟 (STM32F4)
RCC_OscInitStruct.OscillatorType = RCC_OSCILLATORTYPE_HSE;
RCC_OscInitStruct.HSEState = RCC_HSE_ON;
RCC_OscInitStruct.PLL.PLLState = RCC_PLL_ON;
RCC_OscInitStruct.PLL.PLLSource = RCC_PLLSOURCE_HSE;
RCC_OscInitStruct.PLL.PLLM = 8;    // 8MHz / 8 = 1MHz
RCC_OscInitStruct.PLL.PLLN = 336;  // 1MHz × 336 = 336MHz
RCC_OscInitStruct.PLL.PLLP = 2;    // 336MHz / 2 = 168MHz
RCC_OscInitStruct.PLL.PLLQ = 7;    // 336MHz / 7 = 48MHz (USB)

// 32.768kHz外部晶振 → RTC
RCC_OscInitStruct.LSEState = RCC_LSE_ON;
hrtc.Init.AsynchPrediv = 127;
hrtc.Init.SynchPrediv = 255;
// RTC时钟 = 32768 / (128 × 256) ≈ 1Hz
```

## 5. DDS (直接数字频率合成)

### 5.1 原理

```
相位累加器: phase += freq_word (每个时钟周期)
波形查找表: amplitude = LUT[phase[高位]]
DAC输出: 模拟信号

输出频率: f_out = freq_word × f_clk / 2^N
  N: 相位累加器位数 (通常28~48位)
  f_clk: 参考时钟频率

频率分辨率: Δf = f_clk / 2^N
  例: 125MHz / 2^32 ≈ 0.029Hz
```

### 5.2 STM32软件DDS

```c
// 软件DDS (简单但CPU占用高)
#define DDS_LUT_SIZE    256
#define DDS_PHASE_BITS  32

static const uint16_t sine_lut[DDS_LUT_SIZE] = {
    2048, 2098, 2148, 2198, 2248, 2298, 2348, 2398,
    2447, 2496, 2545, 2594, 2642, 2690, 2737, 2784,
    // ... 完整256点正弦表
};

typedef struct {
    uint32_t phase_acc;     // 相位累加器
    uint32_t freq_word;     // 频率控制字
    uint16_t amplitude;     // 幅度
    DAC_HandleTypeDef *hdac;
    uint32_t dac_channel;
} SoftDDS_t;

void SoftDDS_Init(SoftDDS_t *dds, float freq_hz, float fs_hz) {
    dds->freq_word = (uint32_t)((freq_hz / fs_hz) * (1ULL << 32));
    dds->phase_acc = 0;
    dds->amplitude = 2048;  // 12位DAC
}

// 定时器中断回调中调用 (采样率fs)
void SoftDDS_Update(SoftDDS_t *dds) {
    dds->phase_acc += dds->freq_word;
    uint8_t index = dds->phase_acc >> 24;  // 取高8位
    uint16_t sample = sine_lut[index];
    
    HAL_DAC_SetValue(dds->hdac, dds->dac_channel, DAC_ALIGN_12B_R, sample);
}

// 设置频率
void SoftDDS_SetFreq(SoftDDS_t *dds, float freq_hz, float fs_hz) {
    dds->freq_word = (uint32_t)((freq_hz / fs_hz) * (1ULL << 32));
}
```

### 5.3 AD9833/AD9850 DDS芯片

```
AD9833 (推荐, SPI接口):
  - 频率范围: DC~12.5MHz
  - 分辨率: 28位 (0.1Hz精度)
  - 输出: 正弦/三角/方波
  - 供电: 2.3~5.5V
  - 封装: MSOP-10

接线:
  STM32 ──SPI──→ AD9833 ──→ 模拟输出
                    │
              25MHz有源晶振
```

```c
// AD9833驱动
#define AD9833_FSYNC    GPIO_PIN_4   // CS
#define AD9833_SCLK     GPIO_PIN_5   // SCK
#define AD9833_SDATA    GPIO_PIN_7   // MOSI

void AD9833_WriteWord(uint16_t data) {
    HAL_GPIO_WritePin(GPIOA, AD9833_FSYNC, GPIO_PIN_RESET);
    for (int i = 15; i >= 0; i--) {
        HAL_GPIO_WritePin(GPIOA, AD9833_SCLK, GPIO_PIN_RESET);
        HAL_GPIO_WritePin(GPIOA, AD9833_SDATA, 
                          (data & (1 << i)) ? GPIO_PIN_SET : GPIO_PIN_RESET);
        HAL_GPIO_WritePin(GPIOA, AD9833_SCLK, GPIO_PIN_SET);
    }
    HAL_GPIO_WritePin(GPIOA, AD9833_FSYNC, GPIO_PIN_SET);
}

void AD9833_SetFrequency(float freq_hz, float mclk_hz) {
    uint32_t freq_reg = (uint32_t)((freq_hz / mclk_hz) * (1UL << 28));
    
    uint16_t freq_lsb = 0x4000 | (freq_reg & 0x3FFF);
    uint16_t freq_msb = 0x4000 | ((freq_reg >> 14) & 0x3FFF);
    
    AD9833_WriteWord(0x2100);  // 控制字: B28=1, reset
    AD9833_WriteWord(freq_lsb); // FREQ0 LSB
    AD9833_WriteWord(freq_msb); // FREQ0 MSB
    AD9833_WriteWord(0xC000);   // 相位0
    AD9833_WriteWord(0x2000);   // 正弦输出, 取消reset
}
```

### 5.4 DDS设计要点

```
关键参数:
  采样率 ≥ 2×f_out_max (奈奎斯特, 实际取 2.5~3倍)
  DAC位数: 8位→48dB动态范围, 12位→72dB, 16位→96dB

输出滤波器:
  低通滤波器截止频率: f_cut = f_out_max × 1.2
  推荐: 3~5阶Butterworth或椭圆滤波器

杂散抑制:
  - 提高DAC位数
  - 使用抖动技术
  - 优化滤波器设计
```

## 6. PLL (锁相环频率合成)

### 6.1 PLL基本原理

```
       ┌──────────────────────────────────┐
       │                                  │
  f_ref──→[鉴相器PD]──→[环路滤波器LPF]──→[VCO]──→ f_out
       │      ↑                            │
       │      │                            │
       │  [分频器÷N]←─────────────────────┘
       │      │
       │      └──→ f_out/N = f_ref (锁定时)
       │
       └──→ f_out = N × f_ref

频率分辨率: Δf = f_ref (当N为整数时)
改善: 小数分频 (ΔN PLL) → Δf = f_ref / M
```

### 6.2 PLL芯片选型

| 芯片 | 频率范围 | 分辨率 | 接口 | 特点 |
|------|---------|--------|------|------|
| ADF4351 | 35MHz~4.4GHz | 1kHz | SPI | 电赛射频首选 |
| LMX2572 | 10MHz~13.6GHz | 0.1Hz | SPI | 高性能 |
| Si5351 | 8kHz~160MHz | <1Hz | I2C | 低成本、多输出 |
| HMC833 | 25MHz~6GHz | 极高 | SPI | 低相噪 |
| CDCE913 | 80MHz以下 | - | I2C | 简单时钟 |

### 6.3 ADF4351配置 (电赛常用)

```
ADF4351关键参数:
  - VCO: 2200~4400MHz
  - RF输出: 35~4400MHz (含分频器)
  - 参考时钟: 10~250MHz
  - 分辨率: 取决于参考和分频比
  - SPI接口

输出频率:
  f_out = (INT + FRAC/MOD) × f_ref / R / 分频器
  
  INT: 整数分频比 (23~65535)
  FRAC: 小数分子 (0~MOD-1)
  MOD: 小数分母 (2~4095)
  R: 参考分频器 (1~1023)
  输出分频器: 1/2/4/8/16/32/64
```

```c
// ADF4351 SPI驱动 (简化)
typedef struct {
    SPI_HandleTypeDef *hspi;
    GPIO_TypeDef *cs_port;
    uint16_t cs_pin;
    float ref_freq;     // 参考频率
    float rf_div;       // 输出分频器
} ADF4351_t;

void ADF4351_WriteRegister(ADF4351_t *pll, uint32_t reg) {
    HAL_GPIO_WritePin(pll->cs_port, pll->cs_pin, GPIO_PIN_RESET);
    uint8_t data[4] = {
        (reg >> 24) & 0xFF,
        (reg >> 16) & 0xFF,
        (reg >> 8) & 0xFF,
        reg & 0xFF
    };
    HAL_SPI_Transmit(pll->hspi, data, 4, 100);
    HAL_GPIO_WritePin(pll->cs_port, pll->cs_pin, GPIO_PIN_SET);
}

void ADF4351_SetFrequency(ADF4351_t *pll, float freq_mhz) {
    // 计算VCO频率
    float vco_freq = freq_mhz * pll->rf_div;
    
    // 选择输出分频器
    if (vco_freq < 2200) {
        // 需要增大分频器或调整设计
    }
    
    // 计算整数和小数分频
    float n_div = vco_freq / pll->ref_freq;
    uint32_t integer = (uint32_t)n_div;
    uint32_t mod = 4095;
    uint32_t frac = (uint32_t)((n_div - integer) * mod);
    
    // 寄存器配置 (需根据数据手册详细配置)
    uint32_t reg5 = 0x00580005;  // 寄存器5
    uint32_t reg4 = 0x00808034 | ((uint32_t)(pll->rf_div == 1 ? 0 : 
                  __builtin_ctz((uint32_t)pll->rf_div)) << 20);  // 寄存器4
    uint32_t reg3 = 0x000004B3;  // 寄存器3
    uint32_t reg2 = 0x00004E42;  // 寄存器2 (含电荷泵、R分频器)
    uint32_t reg1 = ((mod & 0xFFF) << 3) | 0x08000001;  // 寄存器1
    uint32_t reg0 = ((integer & 0xFFFF) << 15) | ((frac & 0xFFF) << 3); // 寄存器0
    
    // 按顺序写入 (从高到低)
    ADF4351_WriteRegister(pll, reg5);
    ADF4351_WriteRegister(pll, reg4);
    ADF4351_WriteRegister(pll, reg3);
    ADF4351_WriteRegister(pll, reg2);
    ADF4351_WriteRegister(pll, reg1);
    ADF4351_WriteRegister(pll, reg0);
}
```

## 7. 555定时器

### 7.1 无稳态多谐振荡器 (方波)

```
         VCC
          │
         R₁
          │
    ┌─────┤
    │    DIS(7)
    │     │
    │    R₂
    │     │
    │  THR(6)─┤
    │  TRG(2)─┤──555──→ OUT(3)
    │     │          │
    │     C          │
    │     │         GND
    │    GND

频率: f = 1.44 / ((R₁ + 2R₂) × C)
占空比: D = (R₁ + R₂) / (R₁ + 2R₂)

50%占空比: R₁ << R₂ (如 R₁=1kΩ, R₂=100kΩ)
  f ≈ 1.44 / (2 × R₂ × C)
```

### 7.2 单稳态触发器

```
触发输入(2脚) → 输出固定宽度脉冲
脉宽: t = 1.1 × R × C

应用: 消抖、延时、脉冲展宽
```

### 7.3 555设计速查

| 频率 | R₁ | R₂ | C | 占空比 |
|------|----|----|----|-------|
| 1Hz | 10kΩ | 47kΩ | 10μF | 55% |
| 100Hz | 1kΩ | 47kΩ | 100nF | 52% |
| 1kHz | 1kΩ | 47kΩ | 10nF | 52% |
| 10kHz | 1kΩ | 47kΩ | 1nF | 52% |
| 100kHz | 1kΩ | 4.7kΩ | 1nF | 56% |

## 8. 选型决策指南

### 8.1 按需求选方案

```
需求: 固定频率时钟 → 晶体振荡器
需求: 可变频率低频 (1Hz~100kHz) → 软件DDS / 555
需求: 可变频率中频 (100kHz~12MHz) → AD9833 DDS
需求: 可变频率高频 (>12MHz) → ADF4351 PLL
需求: 多路时钟 → Si5351
需求: 射频载波 → ADF4351 / HMC833
需求: 正弦波 (< 1MHz) → 文氏桥 / DDS
需求: 方波 (< 100kHz) → 555 / MCU定时器
需求: 超高精度 → 恒温晶振OCXO + PLL
```

### 8.2 电赛频率方案推荐

| 频段 | 推荐方案 | 芯片/电路 | 精度 |
|------|---------|----------|------|
| DC~100kHz | MCU+DAC | STM32+DMA | 中 |
| 100kHz~12MHz | DDS芯片 | AD9833 | 高 |
| 12MHz~100MHz | DDS芯片 | AD9850/AD9910 | 高 |
| 100MHz~4GHz | PLL合成 | ADF4351 | 高 |
| 固定时钟 | 晶振 | 晶体+MCU | 极高 |
