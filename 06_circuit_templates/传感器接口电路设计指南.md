# 传感器接口电路设计指南

## 一、概述

本文档覆盖电赛常用传感器的接口电路设计：超声波、红外、灰度、颜色、IMU传感器。

---

## 二、HC-SR04 超声波测距

### 2.1 原理
发射40kHz超声波脉冲，通过回波时间计算距离（距离 = 声速 × 时间 / 2）。

### 2.2 电路原理图（ASCII）

```
              VCC (5V)
               │
          [C1 100uF]──GND
               │
    ┌──────────┴──────────┐
    │      HC-SR04         │
    │  VCC   Trig ──[R1 1K]──> MCU GPIO (5V容忍)
    │         Echo ──[R2 2K]──┬──> MCU GPIO
    │                        │
    │         GND       [R3 2.2K]──> GND
    └──────────────────────┘

Echo分压说明:
  R2=2K 串联, R3=2.2K 到GND
  V_echo = 5V × R3/(R2+R3) ≈ 2.62V (兼容3.3V MCU)

若MCU为5V系统，Echo直接连接即可。
```

### 2.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| 超声波模块 | HC-SR04 | HC-SR04/P | 工作电压5V |
| 滤波电容 | 100μF/10V | 铝电解 | VCC端 |
| 上拉/分压电阻 | 2KΩ | 0603 | Echo串联保护 |
| 下拉电阻 | 2.2KΩ | 0603 | Echo分压到GND |

### 2.4 PCB布局建议
- 超声波模块远离电机等振动源
- Trig/Echo信号线加地线屏蔽
- VCC滤波电容靠近模块

### 2.5 常见问题
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 测量值跳变大 | 多个超声波互相干扰 | 交替触发，间隔>60ms |
| 距离偏大 | 温度影响声速 | 加温度补偿算法 |
| 无法检测 | 被测物太小或倾斜 | 调整角度或换激光测距 |

## 三、红外传感器

### 3.1 红外避障（TCRT5000 / E18-D80NK）

```
         VCC (3.3V/5V)
              │
         [C1 100nF]──GND
              │
    ┌─────────┴─────────┐
    │   TCRT5000模块      │
    │  VCC    AO ──> MCU ADC
    │         DO ──> MCU GPIO (数字输出)
    │  GND               │
    └────────────────────┘

E18-D80NK (NPN开集输出):
         VCC (5~24V) ──── E18 VCC(棕)
         MCU GPIO ──[R 4.7K]──> 3.3V
                   │
         E18 OUT(黑) ──────┘
         GND ───────── E18 蓝线

注意: E18输出为NPN开集，需上拉电阻
```

### 3.2 红外对管（RPR220 / 槽型光耦）

```
         VCC (3.3V)
              │
         [R1 100Ω]──> IR LED阳极
                       IR LED阴极──> GND (由MCU GPIO通过三极管驱动)

         VCC (3.3V)
              │
         [R2 10K]──┬──> MCU GPIO (信号输出)
                   │
              光敏管集电极
              光敏管发射极──> GND

无遮挡时: GPIO = LOW (光敏管导通)
有遮挡时: GPIO = HIGH (光敏管截止)
```

### 3.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| 红外避障模块 | NPN输出 | E18-D80NK | 检测距离3~80cm可调 |
| 红外对管 | 反射式 | TCRT5000 | 检测距离~15mm |
| LED限流电阻 | 100Ω | 0805 | 红外LED驱动 |
| 上拉电阻 | 10KΩ | 0603 | 集电极上拉 |
| 分压电阻 | 4.7KΩ | 0603 | NPN输出电平转换 |

### 3.4 常见问题
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 误触发 | 环境光干扰 | 加遮光罩，软件滤波 |
| 黑色物体检测不到 | 反射率低 | 换用激光对管 |
| 距离不稳定 | 供电不稳 | VCC加滤波电容 |

## 四、灰度传感器（五路/八路）

### 4.1 原理
通过红外反射光强判断地面颜色深浅，用于巡线。

### 4.2 电路原理图（ASCII）

```
         VCC (3.3V/5V)
              │
    ┌─────────┴─────────────┐
    │    灰度传感器模块        │
    │  VCC  S1 ──> MCU ADC/IO│
    │       S2 ──> MCU ADC/IO│
    │       S3 ──> MCU ADC/IO│  (五路)
    │       S4 ──> MCU ADC/IO│
    │       S5 ──> MCU ADC/IO│
    │  GND  OUT ──> (可选模拟总线)
    └─────────────────────────┘

自搭灰度传感器(单路):
    VCC ──[R1 100Ω]──> IR LED
                        IR LED ──> GND
    VCC ──[R2 10K]──┬──> MCU ADC
                    │
               光敏电阻
                    │
                   GND
```

### 4.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| 灰度模块 | 五路模拟输出 | RPR220阵列 | 模块整体购买 |
| 红外LED | 940nm | IR333-A | 限流100Ω |
| 光敏电阻 | 暗阻>1MΩ | GL5528 | 可换光电二极管 |
| 限流电阻 | 100Ω | 0805 | LED限流 |
| 上拉电阻 | 10KΩ | 0603 | ADC参考 |

### 4.4 常见问题
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 巡线漂移 | 传感器间距过大 | 缩小间距至10mm |
| 深浅判别不准 | ADC参考电压不稳 | 加基准或软件校准 |
| 地面材质影响 | 反射率差异大 | 现场标定黑白阈值 |

## 五、颜色传感器（TCS34725）

### 5.1 原理
通过RGB滤光片阵列+光电二极管测量颜色分量，I2C数字输出。

### 5.2 电路原理图（ASCII）

```
         VCC (3.3V)
              │
         [C1 100nF]──GND
              │
    ┌─────────┴─────────────┐
    │     TCS34725 模块       │
    │  VIN  SCL ──[R1 4.7K]──> 3.3V ──> MCU SCL
    │       SDA ──[R2 4.7K]──> 3.3V ──> MCU SDA
    │  GND  INT ──> MCU GPIO (可选)
    │       LED ──> 3.3V (板载LED控制)
    └─────────────────────────┘

I2C地址: 0x29 (ADDR引脚接GND)
```

### 5.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| 颜色传感器 | I2C数字输出 | TCS34725 | 模块推荐 |
| I2C上拉电阻 | 4.7KΩ | 0603 | SCL/SDA各一个 |
| 去耦电容 | 100nF | 陶瓷X7R | VIN端 |
| LED限流电阻 | 按模块 | - | 板载LED |

### 5.4 常见问题
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 颜色识别不准 | 环境光影响 | 遮光+白平衡校准 |
| I2C通信失败 | 地址错误/上拉缺失 | 检查地址，确认上拉 |
| 读数饱和 | 距离太近/增益太高 | 调整增益和积分时间 |

## 六、IMU传感器（MPU6050 / ICM20948 / MPU9250）

### 6.1 原理
六轴（加速度计+陀螺仪）或九轴（+磁力计）惯性测量单元。

### 6.2 电路原理图（ASCII）

```
         VCC (3.3V)
              │
         [C1 100nF]──GND
         [C2 10uF]──GND
              │
    ┌─────────┴─────────────┐
    │      MPU6050 模块       │
    │  VCC  SCL ──[R1 4.7K]──> 3.3V ──> MCU SCL
    │       SDA ──[R2 4.7K]──> 3.3V ──> MCU SDA
    │  GND  INT ──> MCU GPIO (数据就绪中断)
    │       AD0 ──> GND (地址0x68) / VCC (0x69)
    └─────────────────────────┘

I2C地址: AD0=GND → 0x68, AD0=VCC → 0x69

SPI接法(ICM20948):
         VCC (3.3V)
              │
    ┌─────────┴─────────────┐
    │    ICM20948 模块        │
    │  VCC  SCL/SCLK ──> MCU SPI_CLK
    │       SDA/MOSI ──> MCU SPI_MOSI
    │       AD0/MISO ──> MCU SPI_MISO
    │       nCS       ──> MCU GPIO (片选)
    │  GND  INT       ──> MCU GPIO
    └─────────────────────────┘
```

### 6.3 元器件选型表

| 元器件 | 参数 | 推荐型号 | 备注 |
|--------|------|----------|------|
| IMU | 6轴I2C/SPI | MPU6050 | 最常用，精度一般 |
| IMU | 9轴SPI | ICM20948 | 高精度，带磁力计 |
| IMU | 9轴SPI | MPU9250 | 经典方案 |
| I2C上拉电阻 | 4.7KΩ | 0603 | SCL/SDA |
| 去耦电容 | 100nF+10μF | 陶瓷 | VIN端 |

### 6.4 PCB布局建议
- IMU远离电机、电源等振动/干扰源
- 如需精确姿态，模块应远离PCB边缘
- 走线短而直，避免信号完整性问题
- I2C上拉电阻靠近MCU端

### 6.5 常见问题
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 陀螺仪零漂大 | 温度漂移 | 上电静止校准+温度补偿 |
| I2C挂死 | 总线冲突 | 加I2C总线恢复程序 |
| 姿态角累积误差 | 积分漂移 | 互补滤波/卡尔曼滤波融合 |
| 振动导致数据噪声 | 机械振动 | 减震垫+低通滤波 |

## 七、通用注意事项

### 7.1 电平匹配
- 5V传感器 → 3.3V MCU：分压电阻或电平转换芯片
- 3.3V传感器 → 5V MCU：一般可直接连接（5V MCU的高电平阈值通常<3.3V）
- 开集/开漏输出：必须加上拉电阻

### 7.2 抗干扰
- 模拟信号线远离数字信号线
- ADC参考电压加滤波（100nF + 10μF）
- 传感器供电加独立LC滤波

### 7.3 传感器校准
- **零点校准**：静止状态下采集N个样本取平均
- **比例校准**：与标准量具对比修正系数
- **温度补偿**：记录温度-误差关系，软件修正
