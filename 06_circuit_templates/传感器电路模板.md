# 传感器电路模板

> 涵盖红外循迹、超声波测距、IMU姿态、编码器测速

---

## 一、红外循迹传感器

### 1.1 TCRT5000 单路电路

```
VCC (5V) ──┬── TCRT5000 发射端(红外LED) ── 100Ω限流 ── GND
           └── TCRT5000 接收端(光敏三极管)
                        │
                    集电极 ──┬── 10kΩ上拉 ── VCC
                             └── 输出 ── MCU ADC/GPIO
                        │
                    发射极 ── GND
```

### 1.2 五路阵列方案

```
传感器阵列布局（底部视角）:
   S1   S2   S3   S4   S5
   ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐
   └─┘  └─┘  └─┘  └─┘  └─┘
   ◄──────── 15-20mm间距 ────────►

每路输出通过LM339比较器:
  VCC ── 10kΩ电位器 ── 正输入(+)  ── 输出 ── GPIO(带上拉)
  传感器输出 ──── 负输入(-)
  
  比较器输出: 高=白底, 低=黑线
```

### 1.3 PCB要点
- 传感器间距15-20mm（根据赛道宽度调整）
- 红外LED和接收管并排朝下，距离地面5-10mm
- 比较器输出加10kΩ上拉电阻
- 供电加100nF去耦

## 二、超声波测距传感器

### 2.1 HC-SR04 接口电路

```
MCU                      HC-SR04
Trig (GPIO OUT) ──────── Trig
Echo (GPIO IN/Timer) ─── Echo
VCC ──────────────────── VCC (5V)
GND ──────────────────── GND

--- 电平匹配（如MCU为3.3V）---
Echo (5V) ──┬── 1kΩ ── GPIO (3.3V)
            └── 2kΩ ── GND
            (电阻分压: 5V × 2k/(1k+2k) = 3.3V)
```

### 2.2 测距原理
```
1. Trig发送≥10μs高电平脉冲
2. 模块发出8个40kHz超声波脉冲
3. Echo输出高电平时间 = 距离 × 2 / 340
4. 距离(cm) = Echo高电平时间(μs) / 58
```

### 2.3 注意事项
- 最小测距约2cm，最大约400cm
- 测量间隔≥60ms（避免余波干扰）
- 多个超声波模块需交替触发（避免互扰）
- 温度补偿：声速 = 331.3 + 0.606 × T(°C)

## 三、IMU 姿态传感器

### 3.1 MPU6050 电路 (I2C)

```
VCC ── 3.3V (注意: 不是5V!)
GND ── GND
SCL ── 4.7kΩ上拉 ── VCC ── MCU I2C_SCL (PB6/PA8等)
SDA ── 4.7kΩ上拉 ── VCC ── MCU I2C_SDA (PB7/PA9等)
AD0 ── GND (I2C地址 0x68) 或 VCC (地址 0x69)
INT ── GPIO (中断输出, 可选)

--- 去耦 ---
VCC引脚 ── 100nF + 10μF ── GND
```

### 3.2 ICM42688 电路 (SPI, 推荐升级)

```
VDD ── 3.3V + 100nF + 10μF
VDDIO ── 3.3V + 100nF
CS   ── GPIO (片选, 拉低选中)
SCLK ── MCU SPI_SCLK
MOSI ── MCU SPI_MOSI
MISO ── MCU SPI_MISO
INT1 ── GPIO (中断)
INT2 ── GPIO (可选)
```

### 3.3 滤波与安装
- IMU安装在PCB中心，远离电机/电源
- 加橡胶垫减震（电机振动影响加速度计）
- 软件做零漂校准（上电静止采样取平均）

## 四、编码器接口

### 4.1 增量式光电编码器 (A/B相)

```
编码器        MCU
A相 ────────── Timer_CH1 (如PA0/TIM2_CH1)
B相 ────────── Timer_CH2 (如PA1/TIM2_CH2)
VCC ────────── 5V/3.3V (看编码器规格)
GND ────────── GND

--- 信号调理 (推荐) ---
A相 ──┬── 100Ω ── GPIO
      └── 100pF ── GND
      (RC低通滤波, 消除抖动)

--- 长线传输 (>20cm) ---
编码器A ── 差分驱动器(AM26C31) ── 双绞线 ── 差分接收器(AM26C32) ── MCU
```

### 4.2 STM32 编码器模式配置
```c
// 使用TIM2的编码器模式
TIM_Encoder_InitTypeDef encoderConfig;
encoderConfig.EncoderMode = TIM_ENCODERMODE_TI12; // A/B双边沿计数
encoderConfig.IC1Polarity = TIM_ICPOLARITY_RISING;
encoderConfig.IC2Polarity = TIM_ICPOLARITY_RISING;
HAL_TIM_Encoder_Init(&htim2, &encoderConfig);
HAL_TIM_Encoder_Start(&htim2, TIM_CHANNEL_ALL);
```

### 4.3 TM4C123 硬件QEI
```
PD6 ── PhA0 (编码器A相)
PD7 ── PhB0 (编码器B相)  
PD3 ── IDX0 (索引信号, 可选)

// 硬件自动计数, 无需CPU干预
```

### 4.4 速度计算
```
转速(rpm) = 脉冲数 × 60 / (编码器线数 × 倍频数 × 采样时间秒)
例: 600线编码器, 4倍频, 10ms采样
    rpm = count × 60 / (600 × 4 × 0.01) = count × 2.5
```

## 五、电磁传感器（电磁循迹）

### 5.1 工字电感方案

```
工字电感(10mH) ── 谐振电容(匹配频率)
                     │
               ── LM358运放(放大)
                     │
               ── 二极管检波(整流)
                     │
               ── RC滤波 ── MCU ADC

谐振频率: f = 1/(2π√(LC))
20kHz赛道: L=10mH → C=6.3nF (用6.8nF)
```

### 5.2 运放放大电路
```
                    Rf (100kΩ)
            ┌────┤├────┐
信号输入 ──┤─        ┤├── 输出 ── ADC
  Rg(1kΩ) └──[LM358]──┘
           
增益 Av = Rf/Rg = 100
```
