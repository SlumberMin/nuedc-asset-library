# 电赛常用放大电路设计

> 运放放大 | 仪表放大 | 程控增益 | 可变增益 | 隔离放大

---

## 一、运算放大器基础

### 1.1 核心参数速查

| 参数 | 含义 | 电赛关注点 |
|---|---|---|
| GBW(增益带宽积) | 增益×带宽=常数 | 信号频率f时最大增益=GBW/f |
| SR(压摆率) | 输出最大变化速率 | SR > 2πf×Vpeak 才不失真 |
| Vos(输入失调) | 零输入时输出偏移 | 直流放大需关注 |
| Ib(输入偏置电流) | 输入端电流 | 高阻源需关注 |
| CMRR(共模抑制比) | 抑制共模信号能力 | 差分测量关键 |
| PSRR(电源抑制比) | 抑制电源波动能力 | 电源噪声大时关注 |

### 1.2 常用运放选型

| 型号 | GBW | SR | Vos | 特点 | 推荐场景 |
|---|---|---|---|---|---|
| LM358 | 1MHz | 0.3V/μs | 3mV | 便宜，单电源 | 低频直流 |
| OP07 | 0.6MHz | 0.3V/μs | 75μV | 低失调 | 精密直流 |
| OPA2134 | 8MHz | 20V/μs | 0.5mV | 低噪声 | 音频/小信号 |
| OPA2277 | 1MHz | 0.8V/μs | 20μV | 精密双运放 | 仪表放大 |
| TL072 | 3MHz | 13V/μs | 3mV | 通用高速 | 有源滤波 |
| AD8065 | 145MHz | 180V/μs | 0.4mV | 高速 | 高频放大 |
| MCP6002 | 1MHz | 0.6V/μs | 4.5mV | 轨到轨，低功耗 | 电池供电 |

## 二、同相放大电路

### 2.1 基本同相放大

```
        Rf
   ┌───/\/\/───┐
   │           │
   │   ┌───┐  │
Vin├──→│+  ├──┴──→ Vout
      │   │
  ┌──→│-  │
  │   └───┘
  │
 GND (或参考电压)

增益：G = 1 + Rf/Rg
输入阻抗：极高（运放输入阻抗）
输出：Vout = Vin × (1 + Rf/Rg)
```

```c
// 设计公式
// 给定增益G，选择Rg，计算Rf
// Rf = (G - 1) × Rg
// 推荐：Rg = 10kΩ (兼顾噪声和功耗)

// 例：G=10 → Rf=90kΩ(用100kΩ可调电阻)
// 例：G=100 → Rf=990kΩ(分两级: 10×10)
```

### 2.2 电压跟随器（G=1）

```
   ┌───────────┐
   │   ┌───┐   │
Vin├──→│+  ├───┴──→ Vout
      │   │
   ┌──→│-  │
   │   └───┘
   └─────────────┘

用途：阻抗变换、缓冲
增益：1
输入阻抗：极高
输出阻抗：极低
```

## 三、反相放大电路

```
   R1      Rf
Vin──/\/\/──┬──/\/\/──┐
            │         │
        ┌───┤         │
   ┌──→ │-  │         │
   │    │   ├─────────┴──→ Vout
   │  ┌─│+  │
   │  │ └───┘
   │  │
  GND GND

增益：G = -Rf/R1（输出反相）
输入阻抗：R1（较低，注意信号源内阻）
```

```c
// 反相加法器：多路输入求和
// Vout = -(Rf/R1×V1 + Rf/R2×V2 + Rf/R3×V3)
// 例：电平叠加、DAC偏置调整
```

## 四、差分放大电路

### 4.1 基本差分放大

```
        R2          R3
V1──/\/\/──┬──/\/\/──┐
           │         │
       ┌───┤         │
  R1┌─→│-  │         │
    │   │   ├─────────┴──→ Vout
V2──┴──→│+  │
        └───┤
            │
           R4
            │
           GND

当 R1=R2, R3=R4 时：
Vout = (R3/R1) × (V2 - V1)
CMRR取决于电阻匹配精度（推荐0.1%精度电阻）
```

### 4.2 三运放仪表放大器（推荐）

```
          Rg
   ┌───/\/\/───┐
   │           │
   │  ┌───┐    │
V1─┤─→│+  ├──┐ │
   │  │ A1├──┘ │
   │┌─→│-  │   │
   ││  └───┘   │      R3       R4
   ││          │  ┌──/\/\/──┬──/\/\/──┐
   ││          │  │         │         │
   ││          │  │  ┌───┐  │         │
   ││          └──┤─→│+  ├──┘         │
   ││             │  │ A3 ├───────────┴──→ Vout
   ││          ┌──┤─→│-  │
   ││          │  │  └───┤
   ││          │  │      │
   ││  ┌───┐   │  │   R2 │
   │└─→│+  ├──┐│  └──/\/\/┘
   │   │ A2├──┘│
   └──→│-  │   │
       └───┘   │
               │
V2─────────────┘

增益：G = 1 + 2×R1/Rg（R1为A1/A2的反馈电阻）
CMRR > 100dB（远优于单运放差分）

增益设置：
  G = 1:  Rg = ∞ (开路)
  G = 10: Rg = 2×R1/9
  G = 100: Rg = 2×R1/99
  G = 1000: Rg = 2×R1/999
```

### 4.3 仪表放大器IC（推荐方案）

| 型号 | 增益范围 | CMRR | 带宽 | 特点 |
|---|---|---|---|---|
| INA128 | 1~10000 | 120dB | 200kHz | 通用经典 |
| INA828 | 1~10000 | 140dB | 2MHz | 低噪声 |
| AD620 | 1~10000 | 100dB | 1MHz | 低功耗 |
| INA333 | 1~1000 | 100dB | 150kHz | 微型封装 |

```c
// AD620增益设置：G = 1 + 49.4kΩ/Rg
// Rg = 49.4kΩ / (G - 1)
// G=10 → Rg=5.49kΩ
// G=100 → Rg=499Ω
// G=1000 → Rg=49.4Ω
```

## 五、程控增益放大器（PGA）

### 5.1 电阻网络切换型

```
用模拟开关（如CD4051/CD4052）切换反馈电阻

        R_sw1    R_sw2    R_sw3
   ┌───/\/\/──┬──/\/\/──┬──/\/\/──┐
   │  (×1)    │  (×10)  │  (×100) │
   │   S1     │   S2    │   S3    │
   │  ┌───┐   │         │         │
Vin├──→│+  ├───┘         │         │
      │   │             │         │
  ┌──→│-  ├─────────────┴─────────┘
  │   └───┘
 GND

MCU通过GPIO控制S1/S2/S3选择增益
```

### 5.2 专用PGA芯片

| 型号 | 增益范围 | 步进 | 接口 | 推荐 |
|---|---|---|---|---|
| PGA113 | 1~128 | ×1,2,5,10,20,50,100,128 | SPI | 通用 |
| PGA280 | 1~128 | 64级 | SPI | 高精度 |
| AD8253 | 1~1000 | ×1,10,100,1000 | 并口/SPI | 高速 |

```c
// PGA113示例（SPI）
void PGA_set_gain(uint8_t gain_code) {
    // gain_code: 0=×1, 1=×2, 2=×5, 3=×10, 4=×20, 5=×50, 6=×100, 7=×128
    CS_LOW();
    SPI_Write(0x2A);           // 写配置寄存器
    SPI_Write(gain_code << 4); // 增益位在高4位
    CS_HIGH();
}
```

### 5.3 STM32内部PGA

```
STM32G4/L4等系列OPAMP可配置为PGA模式：
  - 增益：×2, ×4, ×8, ×16（硬件固定）
  - 通过HAL库配置
  - 无需外部运放
```

## 六、可变增益放大器（VGA/AGC）

### 6.1 基于压控的VGA

```
方案：用FET或光耦作为可变电阻

FET可变电阻：
  - JFET（如2N5457）的D-S间电阻随Vgs变化
  - Vgs=0时Rds最小（数十Ω），Vgs=Vp时Rds最大（MΩ级）
  - 通过DAC控制Vgs实现增益调节

光耦可变电阻：
  - 如PC817，LED电流控制光电晶体管等效电阻
  - 线性区可作为压控电阻
```

### 6.2 自动增益控制（AGC）

```c
// AGC核心思想：检测输出幅度，自动调节增益
float agc_target = 1.0f;   // 目标幅度
float agc_gain = 10.0f;    // 当前增益
float agc_alpha = 0.01f;   // 调整速率

float agc_process(float input) {
    float output = input * agc_gain;
    float amplitude = fabsf(output);

    // 幅度检测（包络检波）
    static float envelope = 0;
    envelope = 0.99f * envelope + 0.01f * amplitude;

    // 增益调整
    if (envelope > agc_target * 1.1f) {
        agc_gain *= (1.0f - agc_alpha);  // 减小增益
    } else if (envelope < agc_target * 0.9f) {
        agc_gain *= (1.0f + agc_alpha);  // 增大增益
    }

    // 限制增益范围
    if (agc_gain > 1000.0f) agc_gain = 1000.0f;
    if (agc_gain < 1.0f) agc_gain = 1.0f;

    return output;
}
```

### 6.3 专用VGA芯片

| 型号 | 增益范围 | 控制方式 | 带宽 | 推荐 |
|---|---|---|---|---|
| AD8338 | 0~80dB | 模拟/DAC | 14MHz | 通用 |
| LMH6505 | -5~40dB | 模拟 | 150MHz | 高速 |
| AD603 | -11~+31dB | 模拟 | 90MHz | 经典 |

## 七、隔离放大电路

### 7.1 为什么需要隔离

```
场景：
  - 测量高侧电流（共模电压几百伏）
  - 电机驱动（强电与弱电隔离）
  - 医疗设备（安全隔离）
  - 长距离传输（消除地环路干扰）
```

### 7.2 线性光耦隔离

```
HCNR200（高线性模拟光耦）

        R1            Rf
  Vin──/\/\/──┬──[LED]──┬──/\/\/──┐
              │         │         │
              │    PD1  │   PD2   │
              │  ┌───┐  │  ┌───┐  │
              └──│   │──┘  │   │──┘
                 │ A1├─→   │ A2├──→ Vout
             ┌──→│-  │     │   │
             │   └───┘  ┌──│-  │
            GND         │  └───┘
                       GND

增益：Vout/Vin = K × Rf/R1（K≈1，光耦传输比）
隔离电压：5000Vrms
带宽：~1MHz
```

### 7.3 隔离运放IC

| 型号 | 隔离电压 | 带宽 | 精度 | 推荐 |
|---|---|---|---|---|
| AMC1200 | 4250V | 60kSPS | ±0.25% | 电流检测 |
| ISO124 | 1500V | 50kHz | ±0.5% | 通用模拟隔离 |
| ACPL-C87B | 5000V | 100kHz | ±0.3% | 高精度 |

### 7.4 Sigma-Delta隔离ADC方案

```
AMC1306（推荐用于电流采样）
  - 内置隔离的Σ-Δ调制器
  - 输出1bit高速比特流
  - MCU侧数字滤波（SINC滤波器）
  - 适合电机相电流采样
```

## 八、信号调理电路

### 8.1 传感器信号调理典型电路

```
传感器输出 → 仪表放大（差分转单端）
           → 低通滤波（去噪）
           → 电平移位（匹配ADC范围）
           → 限幅保护（防过压）

例：应变片→AD620(×100)→二阶LPF(1kHz)→电平移位→ADC
```

### 8.2 电平移位电路

```
将双极性信号移位到ADC单极性范围

      R1           R3
Vin──/\/\/──┬──/\/\/──┐
            │         │
            │   Vref──┤──→ Vout
            │         │
           R2         │
            │         │
           GND       GND

Vout = Vin × R3/(R1+R3) + Vref × R1/(R1+R3)
选择R1=R3：Vout = (Vin + Vref) / 2
Vref由DAC或电阻分压提供
```

### 8.3 输入保护电路

```
        D1(3.3V钳位)
Vin──/\/\/──┬──|<|──→ 3.3V
   (100Ω)   │
            ├──|>|──→ GND
            │  D2
            │
           ADC引脚

或用专用ESD保护：TVS二极管（如PESD5V0）
```

## 九、PCB布局要点

### 9.1 运放电路布局

```
1. 去耦电容：VCC引脚旁放100nF+10μF
2. 地平面：模拟地和数字地单点连接
3. 反馈路径：短且远离干扰源
4. 输入端：保护环（guard ring）减少漏电流
5. 高阻节点：不要铺铜，减少寄生电容
6. 差分走线：等长、等距、平行走
```

### 9.2 常见布局错误

| 错误 | 后果 | 正确做法 |
|---|---|---|
| 去耦电容离引脚远 | 自激振荡 | 电容紧贴VCC引脚 |
| 输入走线长 | 拾取噪声 | 缩短输入走线，加屏蔽 |
| 地线过细 | 地弹、误差 | 用地平面 |
| 高增益级靠近数字区 | 干扰 | 分区布局 |

## 十、电赛实战设计模板

### 10.1 电压测量电路（0~30V）

```
分压(100k/11k) → 电压跟随器(OPA2277) → 二阶LPF(1kHz) → STM32 ADC
精度：±0.1%（配合校准）
```

### 10.2 电流测量电路（0~5A）

```
采样电阻(0.1Ω) → INA128(×10) → 二阶LPF → ADC
或：ACS712 → 跟随器 → ADC（隔离方案）
```

### 10.3 小信号放大（μV级）

```
信号源 → INA128(×1000) → 有源带通(10Hz~10kHz) → PGA(程控) → ADC
注意：屏蔽、接地、去耦必须到位
```

### 10.4 音频放大（麦克风信号）

```
驻极体麦克风 → 偏置电路(2kΩ→VCC) → AC耦合(1μF) → OPA2134(×100)
→ 有源带通(300Hz~3.4kHz) → ADC
```
