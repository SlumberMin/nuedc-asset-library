### 1.2 LDO选型决策树

```
需要什么？
├── 大电流（>500mA）→ AMS1117 / LM1117 / TLV1117
├── 低功耗（电池供电）→ HT7333 / MCP1700 / XC6206
├── 低噪声（ADC/射频供电）→ LP5907 / RT9013 / TPS7A20
├── 超低压差（电池快没电时）→ HT7333 / MCP1700
└── 最便宜 → AMS1117
```

### 1.3 LDO典型电路

```
Vin ─┬─┤Cin├─┬─[LDO]─┬─ Vout ─┬─┤Cout├─┬─ 负载
     │  10μF  │        │        │  10μF  │
    GND    GND        GND    GND       GND

Cin: 输入滤波电容 10μF（MLCC优先）
Cout: 输出滤波电容 10μF（ESR要求看datasheet）
注意：部分LDO对Cout的ESR有要求！
```

### 1.4 LDO关键参数

| 参数 | 含义 | 影响 |
|------|------|------|
| Dropout Voltage | 最小输入输出压差 | 输入电压选择 |
| Line Regulation | 输入电压变化引起的输出变化 | 输入波动抑制 |
| Load Regulation | 负载变化引起的输出变化 | 负载能力 |
| PSRR | 电源抑制比 | 输入纹波抑制能力 |
| Output Noise | 输出噪声 | ADC/射频电路选择 |
| Thermal Shutdown | 过温保护 | 安全保护 |

### 2.2 Buck电路设计模板

#### LM2596方案（简单可靠）

```
Vin ─┬─[L1]─┬─ Vout
     │       │
    CIN    D1(肖特基)  COUT
   100μF    SS34       220μF/25V
     │       │         │
    GND     GND       GND

输出电压设定：
Vout = 1.23 × (1 + R2/R1)
R1 典型值 1kΩ~10kΩ
R2 = R1 × (Vout/1.23 - 1)

5V输出：R1=1k, R2=3.07k(取3k+68Ω)
3.3V输出：R1=1k, R2=1.68k(取1.5k+180Ω)
12V输出：R1=1k, R2=8.75k(取8.2k+560Ω)
```

#### MP1584方案（高频高效）

```
Vin ─┬─[2.2μH]─┬─ Vout
     │          │
   22μF×2      SS34    22μF×2
   (MLCC)      (肖特基)  (MLCC)
     │          │         │
    GND        GND       GND

优势：频率1.5MHz，电感仅2.2μH，体积小
适合空间受限场景
```

### 2.3 Buck关键设计要点

| 要点 | 说明 |
|------|------|
| 电感选择 | L ≥ (Vin-Vout)×Vout/(Vin×f×ΔI)，ΔI一般取Iout的30%~40% |
| 输入电容 | 需承受纹波电流Irms = Iout×√(D(1-D))，D=Vout/Vin |
| 输出电容 | 决定输出纹波，ΔV = ΔI/(8×f×Cout) |
| 二极管 | 选肖特基（SS34/SS54），注意反向耐压>Vin |
| 布局 | 输入电容、IC、电感形成最小环路，远离敏感信号 |

|------|----------|------|----------|------|------|
| MT3608 | 2~24V | 最高28V | 2A | 1.2MHz | 极便宜，万能升压 |
| XL6009 | 5~32V | 最高50V | 4A | 400kHz | 大电流升压 |
| LM2577-ADJ | 3.5~40V | 最高60V | 3A | 52kHz | 经典方案 |
| TPS61088 | 2.7~12V | 最高12.6V | 10A(峰值) | 200kHz~2MHz | 大电流升压 |
| MC34063 | 3~40V | 可调 | 1.5A | 100kHz | 老牌万能IC |

| LT8330 | 3~40V | 最高60V | 1A | 2MHz | 高压小电流 |

### 3.2 Boost典型电路（MT3608）

```
Vin ─┬─[MT3608]─┬─[4.7μH~22μH]─┬─ D1(肖特基)─┬─ Vout
     │    EN     │                │   SS34      │
     │    SW ────┘                │             │
    CIN         GND             GND           COUT
   22μF                          220μF
     │                            │
    GND                          GND

输出电压设定：
Vout = 0.6 × (1 + R2/R1)

5V输出：R1=13k, R2=95.3k(取100k)
12V输出：R1=10k, R2=190k(取200k)
```

### 3.3 Boost设计要点

```
⚠️ 注意事项：
1. Boost不能短路保护（输出直接通过电感和二极管到Vin）
   → 需要额外加保险丝或电子保护
2. 输出纹波通常比Buck大
3. 占空比不能太接近100%，一般D<90%
4. 最大输出电流受输入电流限制：Iout_max = Iin × (1-D) / η
```

| 型号 | 保护功能 | 过流 | 封装 | 特点 |
|------|----------|------|------|------|
| DW01A | 过充/过放/过流/短路 | 可调 | SOT-23-6 | 最常用单节保护 |
| FS312F | 过充/过放/过流/短路 | 固定 | SOT-23-6 | DW01A替代 |
| S-8241 | 过充/过放/过流 | 可选 | SOT-23-6 | 精度更高 |
| HY2120 | 过充/过放/过流 | 可选 | SOT-23-6 | 国产替代 |
| XB7608 | 过充/过放/过流/短路 | 集成MOS | SOP-8 | 集成度高 |

### 6.2 DW01A + 8205A 保护电路

```
        ┌─────────┐
BAT+ ──→│         │──→ LOAD+
        │ DW01A   │
BAT- ──→│    DOUT │──→ [8205A双NMOS] ──→ LOAD-
        │    COUT │
        └─────────┘

保护功能：
- 过充保护：电池电压 > 4.25~4.35V → 断开充电MOS
- 过放保护：电池电压 < 2.4~2.5V → 断开放电MOS
- 过流保护：电流超过设定值 → 断开MOS
- 短路保护：输出短路 → 快速断开MOS

⚠️ 注意：
1. 8205A是双NMOS封装，需配合DW01A使用
2. MOSFET的Rds(on)决定保护电路的等效内阻
3. 选择低Rds(on)的MOSFET减少压降和发热
```

### 7.1 常用电源监控IC

| 型号 | 功能 | 精度 | 特点 |
|------|------|------|------|
| TLV809 | 复位监控 | ±1.5% | 低成本复位IC |
| MAX809 | 复位监控 | ±2% | 经典方案 |
| TPS3839 | 低功耗复位 | ±2% | 150nA超低功耗 |
| INA219 | 电流/功率监控 | ±1% | I2C接口，最常用 |
| INA226 | 电流/功率监控 | ±0.1% | 高精度，I2C |
| INA169 | 电流检测放大器 | ±2% | 模拟输出 |
| LTC2944 | 电量计 | ±5% | I2C，库仑计 |
| BQ34Z100 | 电量计 | ±3% | 多节电池电量计 |

### 7.2 INA219电流监控电路

```
        ┌───────────────┐
Vcc ────│ VCC     VS+   │──── 负载供电
        │               │
        │ INA219        │
        │               │
        │ A0 A1   SDA   │──── I2C数据
        │     SCL       │──── I2C时钟
        │               │
GND ────│ GND     VS-   │──── 负载地
        └───────────────┘

分流电阻 Rsense 接在 VS+ 和 VS- 之间
Rsense选择：
- 大电流(>1A)：10mΩ~50mΩ（功率足够）
- 小电流(<500mA)：100mΩ~1Ω
- 注意：Rsense上的压降 = I × Rsense

配置要点：
- 总线电压范围：0~32V（INA219）
- 分流电压范围：±320mV
- ADC精度：12bit
- I2C地址：0x40（A0=A1=GND时）
```

```c
/* INA219驱动代码 */
#define INA219_ADDR  0x40
#define INA219_REG_CONFIG     0x00
#define INA219_REG_SHUNT_V    0x01
#define INA219_REG_BUS_V      0x02
#define INA219_REG_POWER      0x03
#define INA219_REG_CURRENT    0x04
#define INA219_REG_CALIB      0x05

void INA219_Init(float r_shunt, float i_max)
{
    /* 计算校准值 */
    /* Cal = 0.04096 / (Current_LSB × Rshunt) */
    float current_lsb = i_max / 32768.0f;
    uint16_t cal = (uint16_t)(0.04096f / (current_lsb * r_shunt));
    I2C_WriteReg16(INA219_ADDR, INA219_REG_CALIB, cal);

    /* 配置：总线电压32V，分流±320mV，连续采样 */
    uint16_t config = 0x399F;  // 典型配置
    I2C_WriteReg16(INA219_ADDR, INA219_REG_CONFIG, config);
}

float INA219_GetCurrent(void)
{
    int16_t raw = (int16_t)I2C_ReadReg16(INA219_ADDR, INA219_REG_CURRENT);
    return raw * 0.0001f;  // Current_LSB = 0.1mA
}

float INA219_GetBusVoltage(void)
{
    uint16_t raw = I2C_ReadReg16(INA219_ADDR, INA219_REG_BUS_V);
    return (raw >> 3) * 0.004f;  // 4mV per LSB
}
```

## 八、电源方案设计速查

### 8.1 常见电源轨

| 电压 | 典型用途 | 推荐方案 |
|------|----------|----------|
| 1.2V | FPGA内核、DDR | LDO (AMS1117可调) |
| 1.8V | FPGA IO、低电压MCU | LDO (AP2112K-18) |
| 2.5V | FPGA辅助、某些传感器 | LDO (XC6206) |
| 3.3V | MCU、数字电路、传感器 | LDO (AMS1117-3.3) 或 Buck |
| 5.0V | USB、运放、传感器 | LDO (AMS1117-5.0) 或 Buck |
| ±5V | 运放双电源 | Buck + Inverter (TPS65133) |
| ±12V | 运放/信号调理 | Boost + Inverter |
| 12V | 电机驱动、继电器 | Buck (LM2596) 或 Boost |
| 24V | 电机驱动、长距离 | Boost (XL6009) |

### 8.2 多路电源方案示例

```
题目要求：STM32系统 + 运放信号调理 + 电机驱动

电源规划：
┌──────────────────────────────────────────────────┐
│ 12V电池/适配器                                     │
│   │                                               │
│   ├──[LM2596 Buck]──→ 5V ──→ AMS1117-3.3 ──→ 3.3V (MCU)
│   │                                               │
│   ├──[LM2596 Buck]──→ ±5V ──→ 运放供电            │
│   │   (或 TPS65133 双路输出)                        │
│   │                                               │
│   ├── 直接12V ──→ 电机驱动 (L298N/TB6612)         │
│   │                                               │
│   └──[LDO]──→ 3.3V ──→ ADC参考电压 (可选独立LDO)   │
└──────────────────────────────────────────────────┘
```

### 8.3 电源设计检查清单

```
□ 输入电压范围确认
□ 输出电压和电流需求计算
□ 效率要求（电池供电必须用DC-DC）
□ 纹波要求（ADC供电需要低纹波 → LDO或低噪声Buck）
□ 启动时序（FPGA等需要特定上电顺序）
□ 保护功能（过流/过压/反接/ESD）
□ 散热计算：P_dissipated = (Vin-Vout)×Iout（LDO）
□ 电感/电容选型和布局
□ 测试验证：空载/满载/瞬态响应
```

## 九、电赛电源设计经验总结

| 场景 | 推荐方案 | 原因 |
|------|----------|------|
| MCU供电 | AMS1117-3.3 | 便宜可靠，电流足够 |
| ADC精密供电 | LP5907-33 | 超低噪声 |
| 电机驱动 | LM2596/MP1584 Buck | 高效率，大电流 |
| 电池充电 | TP4056 | 最简单便宜 |
| 电流检测 | INA219 | 精度好，I2C接口 |
| 单节锂电升降压 | TPS63020 | 高效升降压 |
| 负压生成 | TPS65133 | 正负压输出 |
| 快速搭建 | 用成品DC-DC模块 | 电赛时间紧张时首选 |
