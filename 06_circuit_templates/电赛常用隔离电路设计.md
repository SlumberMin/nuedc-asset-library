# 电赛常用隔离电路设计

## 概述

电气隔离用于切断两个电路之间的直接电气连接，实现信号传输的同时保证安全和抗干扰。在电赛中，隔离电路常用于：高低压隔离、模拟/数字信号隔离、通信接口隔离、电源隔离等场景。

---

## 1 隔离技术对比

| 技术 | 隔离原理 | 带宽 | 隔离电压 | 精度 | 成本 | 典型应用 |
|------|----------|------|----------|------|------|----------|
| 光耦 | 光信号 | ~1MHz | 2.5-5kV | 中 | 低 | 数字信号/开关信号 |
| 磁耦（变压器） | 磁场 | ~100MHz | 1-5kV | 中高 | 中 | 电源/高频信号 |
| 电容耦合 | 电场 | ~150Mbps | 2.5-5kV | 高 | 中 | 高速数字信号 |
| 数字隔离器 | 多种 | ~150Mbps | 2.5-5kV | 高 | 中高 | 高速数字/多通道 |

## 2 光耦隔离

### 2.1 基本原理

```
输入端                    隔离栅                    输出端
        ┌──────────┐     │    ┌──────────┐
Vin ──→│  LED发光  │  光 │    │ 光电晶体管│──→ Vout
        └──────────┘     │    └──────────┘
```

### 2.2 常用光耦型号

| 型号 | CTR | 带宽 | 隔离电压 | 特点 | 价格 |
|------|-----|------|----------|------|------|
| PC817 | 50-600% | ~80kHz | 5kV | 最常用，便宜 | ￥0.2-0.5 |
| TLP521 | 100-600% | ~10kHz | 2.5kV | 经典款 | ￥0.3-0.5 |
| TLP290 | 50-600% | ~1MHz | 3.75kV | 高速 | ￥0.5-1 |
| 6N137 | - | 10Mbps | 5kV | 高速数字 | ￥1-2 |
| HCPL-2630 | - | 10Mbps | 5kV | 双通道高速 | ￥3-5 |

**CTR（电流传输比）：** CTR = I_C / I_F × 100%

### 2.3 开关量光耦电路

#### 基本开关隔离

```
         VCC_IN                VCC_OUT
          │                     │
          R1                    R2
          │                     │
Vin ──┬───┤LED    光耦    C────┤─── Vout
      │        ═══════════         │
     GND_IN   │     │           GND_OUT
              隔离栅（不连接）
```

**设计要点：**
- R1 = (VCC_IN - V_LED) / I_F，I_F一般取5~20mA
- R2 = VCC_OUT / I_C，I_C = I_F × CTR
- R2不宜过大，否则响应慢

#### 反相开关电路

```
VCC_OUT
  │
  R2
  │
  ├──── Vout
  │
  C──── 光耦光电管
  │
 GND

Vin → R1 → LED → GND_IN
```

Vin高 → LED亮 → 光电管导通 → Vout低（反相）

### 2.4 模拟信号光耦隔离

#### 线性光耦方案（PC817线性区）

```
         VCC                        VCC
          │                          │
          R1                         R2
          │    ┌──────┐   ┌──────┐   │
Vin ──┬── R ──┤LED   │   │LED   ├──┤── Vout
      │       │ 光耦1 │   │ 光耦2 │   │
     GND      │      │   │      │   │
              │ CTR匹配│   │ CTR匹配│  │
              └──────┘   └──────┘  GND
```

**反馈式线性隔离原理：**
- 使用两个匹配光耦
- 光耦1的LED由输入驱动
- 光耦2的LED通过运放反馈驱动
- 利用CTR匹配实现线性传输

#### HCNR200 高精度线性光耦

```
         VCC                        VCC
          │                          │
          R1                    Rf   │
          │     ┌────────┐     │    │
Vin ──┬── R ───┤ HCNR200├──── R2 ───┤── Vout
      │        │        │          │
     GND       │ PD1  PD2│          │
               │  LED   │         GND
               └────────┘
                 隔离栅
```

**传递函数：** V_out / V_in = K × (R_f / R1)

K为光电二极管传输系数（典型0.05%），通过运放闭环保证线性度。

### 2.5 光耦驱动LED设计

**限流电阻计算：**
$$R = \frac{V_{in} - V_F - V_{sat}}{I_F}$$

| 参数 | 典型值 | 说明 |
|------|--------|------|
| V_F | 1.2V | LED正向压降（红外） |
| I_F | 5~20mA | 正向电流 |
| V_sat | 0.2~0.3V | 驱动管饱和压降 |

## 3 磁耦隔离（变压器隔离）

### 3.1 基本原理

```
一次侧              磁芯              二次侧
  ┌──┐    ┌────────────────┐    ┌──┐
  │  │    │   ≋≋≋≋≋≋≋≋≋≋   │    │  │
  │N1│◄───┤   铁氧体磁芯   ├───►│N2│
  │  │    │   ≋≋≋≋≋≋≋≋≋≋   │    │  │
  └──┘    └────────────────┘    └──┘
```

**变压比：** V2/V1 = N2/N1

### 3.2 脉冲变压器隔离

#### 信号传输

```
Vin ──┬── C ──┬── 变压器一次侧 ── 变压器二次侧 ──┬── 整形 ── Vout
      │       │                                   │
     GND     R（阻尼）                           GND_ISO
```

**设计参数：**
- 匝比：通常1:1
- 磁芯：高频铁氧体（如EE型、环型）
- 频率范围：1kHz~10MHz

### 3.3 隔离电源变压器

#### 反激式隔离电源

```
        VCC                        V_ISO
         │                          │
         ┤                          │
        ┌┤ ┐                      ┌┤ ┐
Vin ──┤ │变压器│ ── 整流滤波 ──┤ │变压器│ ── Vout
        │N1 N2│                    │N1 N2│
        └┤ ┘                      └┤ ┘
         │                          │
      MOSFET                     GND_ISO
         │
        GND
```

**常用隔离电源模块：**
| 型号 | 输入 | 输出 | 功率 | 隔离电压 |
|------|------|------|------|----------|
| B0505S-1W | 5V | 5V | 1W | 1kV |
| B1205S-2W | 12V | 5V | 2W | 2kV |
| MORNSUN系列 | 多种 | 多种 | 0.1-30W | 1-6kV |

### 3.4 ADuM系列磁耦隔离器

**ADI公司的iCoupler技术：** 芯片级变压器隔离

| 型号 | 通道 | 速率 | 隔离电压 | 特点 |
|------|------|------|----------|------|
| ADuM1201 | 2通道 | 10Mbps | 2.5kV | 通用 |
| ADuM1401 | 4通道 | 10Mbps | 2.5kV | 多通道 |
| ADuM1250 | 2通道（I²C） | - | 2.5kV | I²C专用 |
| ADuM3190 | 隔离误差放大器 | - | 2.5kV | 电源反馈 |

## 4 电容耦合隔离

### 4.1 基本原理

利用芯片内嵌的高压电容实现信号耦合，同时提供电气隔离。

```
信号输入 → 调制 → [隔离电容] → 解调 → 信号输出
```

### 4.2 TI电容耦合数字隔离器

| 型号 | 通道 | 速率 | 隔离电压 | 特点 |
|------|------|------|----------|------|
| ISO7220A | 2通道 | 25Mbps | 4kV | 通用高速 |
| ISO7221A | 2通道(1+1) | 25Mbps | 4kV | 方向可配 |
| ISO7240A | 4通道 | 25Mbps | 4kV | 多通道 |
| ISO7740 | 4通道 | 100Mbps | 5kV | 超高速 |

### 4.3 典型应用电路

```
        VCC1            VCC2
         │    ISO7220A    │
         │   ┌───────┐   │
Vin ─────┤──►│IN1  OUT1├──┤──── Vout
         │   │       │   │
         │   │IN2  OUT2├──┤──── 信号2
         │   └───────┘   │
        GND1           GND2
        (不连接)        (不连接)
```

## 5 数字隔离器

### 5.1 通用数字隔离器

#### 基本参数选型

| 参数 | 考虑因素 |
|------|----------|
| 数据速率 | 信号频率的2-5倍 |
| 通道数 | 需要隔离的信号数量 |
| 通道方向 | 单向/双向/可配置 |
| 隔离电压 | 安全标准要求 |
| 传播延迟 | 高速信号关键参数 |
| 功耗 | 低功耗系统关注 |

#### 选型速查表

| 需求 | 推荐方案 |
|------|----------|
| 低速开关信号(<100kHz) | PC817光耦 |
| 中速数字信号(~1MHz) | 6N137高速光耦 |
| 高速数字(~25Mbps) | ISO7220 |
| 超高速(~150Mbps) | ISO77xx / ADuM4xxx |
| I²C隔离 | ADuM1250 / ISO1540 |
| SPI隔离 | ADuM3150 / ISO7741 |
| CAN总线隔离 | ISO1042 / ADM3053 |
| RS485隔离 | ADM2587E / ISO3082 |

### 5.2 I²C总线隔离

```
        VCC1                VCC2
         │    ADuM1250       │
         │   ┌───────┐      │
SDA1 ────┤──►│SDA1 SDA2├────┤─── SDA2
         │   │       │      │
SCL1 ────┤──►│SCL1 SCL2├────┤─── SCL2
         │   └───────┘      │
        GND1              GND2
```

**注意：** I²C是开漏输出，隔离器需支持双向通信。

### 5.3 SPI总线隔离

```
        VCC1                 VCC2
         │   ADuM3150         │
         │  ┌─────────┐      │
SCLK ────┤─►│SCLK  SCLK├────┤── SCLK
         │  │         │      │
MOSI ────┤─►│MOSI  MOSI├────┤── MOSI
         │  │         │      │
MISO ◄───┤──│MISO  MISO├◄───┤── MISO
         │  │         │      │
CS ──────┤─►│CS    CS ├───►┤── CS
         │  └─────────┘      │
        GND1               GND2
```

### 5.4 CAN总线隔离

**隔离CAN收发器方案：**

```
MCU ── TX ──┐                 ┌── CAN_H
             │  ISO1042       │
         隔离CAN收发器 ────────┤
             │                │
MCU ── RX ──┘                └── CAN_L
```

**ISO1042特点：**
- 集成隔离+收发器
- 隔离电压5kV
- 支持5Mbps CAN FD
- 符合ISO 11898标准

### 5.5 RS485隔离

**方案1：数字隔离器 + RS485收发器**

```
MCU ── TX ── ISO7221 ── TX ── MAX485 ── A/B（总线）
MCU ── RX ── ISO7221 ── RX ── MAX485 ── A/B
              VCC1        VCC2         VCC_ISO
              GND1        GND2         GND_ISO
```

**方案2：集成隔离RS485（ADM2587E）**

```
MCU ── TX ──┐                 ┌── A
             │ ADM2587E       │
         隔离485收发器 ────────┤
             │                │
MCU ── RX ──┘                └── B
```

### 6.3 隔离电源PCB布局

```
┌─────────────────────────────────────┐
│  一次侧区域          二次侧区域      │
│                                     │
│  VIN    ┌──────────┐    VOUT_ISO    │
│   │     │ 隔离模块  │      │        │
│   C1    │          │     C2        │
│   │     │ GND1  GND2│      │       │
│  GND    └──────────┘   GND_ISO     │
│                                     │
│        ←─ 爬电距离 ≥ 6mm ─→        │
└─────────────────────────────────────┘
```

**PCB设计要点：**
1. 一次侧和二次侧地平面分开
2. 隔离区域下方禁布铜
3. 爬电距离满足安全标准
4. 输入输出电容靠近模块引脚

## 7 隔离运放

### 7.1 模拟信号隔离方案

| 方案 | 精度 | 带宽 | 隔离电压 | 适用场景 |
|------|------|------|----------|----------|
| 线性光耦(HCNR200) | 0.5-2% | ~1MHz | 5kV | 通用模拟隔离 |
| 隔离运放(AMC1200) | 0.25% | ~60kHz | 4kV | 电压检测 |
| 隔离Σ-Δ调制器(AMC1306) | 16bit | ~78kSPS | 7kV | 高精度ADC前级 |
| V/F+光耦 | 0.1% | ~100kHz | 5kV | 高精度慢信号 |

### 7.2 AMC1200隔离运放

**参数：**
- 增益：8V/V（固定）
- 输入范围：±250mV
- 精度：±0.25%（增益误差）
- 隔离电压：4kV
- 带宽：60kHz

**典型应用——母线电压检测：**

```
母线电压
    │
    R1（分压）
    │
    ├──── AMC1200输入+ ──── 隔离 ──── AMC1200输出 ──── ADC
    │
    R2
    │
   GND
```

### 7.3 电流检测隔离

**方案：隔离运放 + 采样电阻**

```
大电流母线
    │
    R_shunt（mΩ级）
    │─────────────────────
    │                     │
    ┤ AMC1200 VIN+       │
    │                     │
    ┤ AMC1200 VIN-       负载
    │                     │
    └─────────────────────┘
    
    AMC1200 VOUT ──── ADC
```

**采样电阻选择：**
- R_shunt = V_shunt_max / I_max
- V_shunt_max ≤ 250mV（AMC1200输入范围）
- 功率：P = I²_max × R_shunt

## 8 隔离设计要点

### 8.1 安全标准

| 标准 | 隔离电压要求 | 应用 |
|------|-------------|------|
| 功能隔离 | 基本绝缘 | 信号隔离 |
| 基本绝缘 | 1kV~2.5kV | 消费电子 |
| 加强绝缘 | 2.5kV~5kV | 医疗/工业 |
| 双重绝缘 | 4kV~8kV | 特殊应用 |

### 8.2 PCB爬电距离

| 工作电压 | 最小爬电距离（FR4） |
|----------|---------------------|
| 50V | 0.5mm |
| 100V | 1.0mm |
| 200V | 2.0mm |
| 400V | 3.0mm |
| 600V | 4.0mm |

### 8.3 EMC注意事项

1. **隔离栅两侧地平面不连接**（除非需要Y电容）
2. **输入输出走线远离隔离区域**
3. **去耦电容紧靠芯片电源引脚**
4. **高速隔离器注意阻抗匹配**
5. **必要时在隔离栅加屏蔽层**

### 8.4 常见问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 输出信号毛刺 | 共模干扰 | 加滤波电容/改用磁耦 |
| 传输延迟过大 | 光耦响应慢 | 换用高速光耦/数字隔离器 |
| 模拟精度差 | 光耦CTR漂移 | 用反馈式电路/隔离运放 |
| 隔离电源纹波大 | 负载变化 | 加LC滤波/增大输出电容 |
| 通信误码率高 | 信号完整性差 | 缩短走线/加端接电阻 |

## 9 电赛实战电路

### 9.1 高压侧电流检测隔离

```
       高压母线（如48V）
           │
        R_shunt
           │
    ┌──────┴──────┐
    │  AMC1200    │      低压侧
    │  VIN+  VOUT ├────  ──── ADC
    │  VIN-       │      │
    │  GND1  GND2 │     GND
    └─────────────┘
    高压侧地       低压侧地
```

### 9.2 隔离UART通信

```
MCU_TX ── ISO7221 ──► ISO7221 ──► 远端设备RX
MCU_RX ◄── ISO7221 ◄── ISO7221 ◄── 远端设备TX
         VCC1          VCC2
         GND1          GND_ISO
```

### 9.3 隔离CAN总线

```
MCU ── SPI ── ADuM3150 ── SPI ── MCP2515 ── CAN ── ISO1042 ── CAN总线
     隔离层         隔离层               隔离层
```

或简化方案：
```
MCU ── TX/RX ── ISO1042 ── CAN_H / CAN_L
         隔离层（集成）
```

### 9.4 电源隔离方案选型

| 场景 | 推荐方案 |
|------|----------|
| 数字隔离器供电 | B0505S-1W 模块 |
| 隔离运放供电 | B0505S-1W 模块 |
| 隔离CAN供电 | ISO1042自带隔离侧供电 |
| 隔离485供电 | ADM2587E自带隔离侧供电 |
| 大功率隔离电源 | 反激变换器自搭 |
