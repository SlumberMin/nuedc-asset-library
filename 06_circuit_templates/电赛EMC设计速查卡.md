## 一、EMC基础知识

### 1.1 EMC三要素

```
电磁兼容问题 = 干扰源 + 耦合路径 + 敏感设备

         干扰源
           │
    ┌──────┼──────┐
    │      │      │
  辐射    传导    耦合
    │      │      │
    └──────┼──────┘
           │
       敏感设备

解决思路：
  1. 抑制干扰源（最有效）
  2. 切断耦合路径
  3. 提高设备抗扰度
```

### 1.2 EMC测试项目

| 类别 | 测试项 | 标准 | 限值 |
|------|--------|------|------|
| 发射 | 辐射发射(RE) | CISPR 22 | 30-230MHz: 40dBμV/m |
| 发射 | 传导发射(CE) | CISPR 22 | 150kHz-30MHz: 66dBμV |
| 抗扰 | 静电放电(ESD) | IEC 61000-4-2 | ±8kV接触/±15kV空气 |

| 抗扰 | 传导抗扰(CS) | IEC 61000-4-6 | 3V (150kHz-80MHz) |

## 二、辐射发射（RE）抑制

### 2.1 辐射来源

```
差模辐射：电流环路产生的辐射
  E ∝ f² × I × A / r
  f=频率, I=电流, A=环路面积, r=距离

共模辐射：导体作为天线的辐射
  E ∝ f × I_cm × L / r
  L=导体长度, I_cm=共模电流

高频噪声源：
  ├── 时钟信号及其谐波
  ├── 开关电源（DCDC）
  ├── 数字信号边沿
  └── 电机PWM驱动
```

### 2.2 辐射抑制措施

| 措施 | 效果 | 实施难度 | 适用场景 |
|------|------|---------|---------|
| 缩小环路面积 | ★★★★★ | 低 | PCB布线 |
| 地平面完整 | ★★★★★ | 低 | PCB设计 |
| 串联电阻/铁氧体 | ★★★★ | 低 | 时钟线 |
| 屏蔽 | ★★★★★ | 中 | 整体/局部 |
| 差分信号 | ★★★★ | 中 | 高速信号 |
| 扩频时钟 | ★★★ | 低 | 时钟EMI |

### 2.3 PCB辐射控制

```
关键原则：

1. 最小化高频电流环路面积
   ├─ 信号线紧邻回流地平面
   ├─ 电源和地平面紧密耦合
   └─ 高速信号线参考完整地平面

2. 去耦电容
   ├─ 每个VCC引脚：100nF + 10μF
   ├─ 高速芯片：增加10nF/1nF
   ├─ 电容尽量靠近引脚
   └─ 电容接地过孔直接到地平面

3. 时钟线处理
   ├─ 串联22-330Ω阻尼电阻
   ├─ 走内层（被地平面屏蔽）
   ├─ 避免跨越地平面分割
   └─ 包地处理（两侧地铜皮）

4. 接插件处理
   ├─ 连接器附近增加去耦
   ├─ 屏蔽连接器接地
   └─ 线缆加磁环
```

## 三、传导发射（CE）抑制

### 3.1 传导路径

```
传导发射通过电源线/信号线向外传播：
  ┌──────┐      ┌──────┐
  │ 设备  │─L线──→│ 电网  │
  │      │─N线──→│      │
  │      │─PE线─→│      │
  └──────┘      └──────┘

  共模传导：L/N对PE → EMI滤波器X电容+共模电感
  差模传导：L对N → EMI滤波器Y电容+差模电感
```

### 3.2 EMI滤波器设计

```
标准EMI滤波器拓扑：

L线 ──┤Cx├──┤Lcm├──┤Cx├──→ 设备
         │      │      │
         │    Cm│Cm    │
         │      │      │
N线 ──┤Cx├──┤Lcm├──┤Cx├──→ 设备

Cx = X电容（差模滤波）：0.1-4.7μF
Cm = Y电容（共模滤波）：1-4.7nF（受漏电流限制）
Lcm = 共模电感：1-10mH

选型口诀：
  低频传导 → 大Cx、大Lcm
  高频传导 → 多级滤波、铁氧体磁珠
  漏电流限制 → Y电容 ≤ 4.7nF
```

### 3.3 电源入口滤波模板

```
电路模板（单级EMI滤波）：

VIN+ ──┬──[FB1]──┬──[Lcm]──┬──[FB2]──┬── VCC_IN
       │         │         │         │
     [C_x1]    [C_y1]   [C_y2]    [C_x2]
       │         │         │         │
VIN- ──┴─────────┴─────────┴─────────┴── GND

FB1/FB2 = 铁氧体磁珠 @ 100MHz: 600-1000Ω
C_x1/C_x2 = 0.1-1μF X7R
C_y1/C_y2 = 1-4.7nF Y1/Y2安规电容
Lcm = 共模电感 1-4.7mH
```

## 四、抗扰度设计

### 4.1 ESD（静电放电）防护

```
ESD防护策略：
  1. TVS二极管（最快响应 <1ns）
  2. RC滤波（对外部接口）
  3. 隔离（光耦/磁耦）
  4. 接地设计（单点/多点）

TVS选型：
  ├── 信号线：双向TVS，Vbr = 1.1×Vsignal
  ├── 电源线：单向TVS，Vbr = 1.2×Vsupply
  ├── 响应时间：< 1ns
  └── 钳位电压：< 芯片最大耐压
```

```c
// TVS典型参数速查
// 信号线：SMBJ5.0CA (5V双向)
//   Vbr=6.4V, Vc=9.2V, Ppp=600W
// 电源5V：SMBJ5.0A (5V单向)
//   Vbr=6.4V, Vc=9.2V, Ppp=600W
// 电源12V：SMBJ12A (12V单向)
//   Vbr=13.3V, Vc=19.9V, Ppp=600W
```

### 4.2 EFT（电快速脉冲群）防护

```
EFT特点：高频脉冲群，单脉冲5ns/50ns，5kHz重复
防护方案：
  1. 电源入口：π型LC滤波
  2. 信号线：RC滤波 + TVS
  3. 共模滤波

典型防护电路：
  信号线 ──[R 100Ω]──┬── MCU
                     │
                  [C 100pF]
                     │
                    GND
                     │
                  [TVS双向]
                     │
                    GND
```

### 4.3 浪涌（Surge）防护

```
浪涌特点：高能量，1.2/50μs波形，持续ms级
防护方案：
  1. MOV压敏电阻（吸收能量）
  2. TVS（快速钳位）
  3. GDT气体放电管（大能量泄放）
  4. 保险丝（过流保护）

多级防护：
  电源入口 → GDT → MOV → L(退耦) → TVS → 设备
  (粗保护)                   (细保护)
```

## 五、滤波设计

### 5.1 常用滤波器类型

| 类型 | 频率范围 | 适用 | 缺点 |
|------|---------|------|------|
| RC滤波 | DC-10MHz | 信号线去噪 | 有压降，带宽受限 |
| LC滤波 | 100kHz-100MHz | 电源滤波 | 可能谐振 |
| 铁氧体磁珠 | 10MHz-1GHz | 高频抑制 | 有阻抗频率特性 |
| 共模电感 | 100kHz-100MHz | 差分信号/电源 | 体积大 |
| π型滤波 | DC-1GHz | 电源入口 | 复杂 |

### 5.2 铁氧体磁珠选型

```
关键参数：
├── 阻抗@100MHz：30-1000Ω
├── 直流电阻DCR：< 0.5Ω（避免压降）
├── 额定电流：> 实际电流×1.5
└── 自谐振频率：> 工作频率

选型速查表：
  ┌──────────┬───────────┬────────┬──────────┐
  │ 型号      │ 阻抗@100M │ DCR    │ 额定电流  │
  ├──────────┼───────────┼────────┼──────────┤
  │ BLM18AG  │ 600Ω      │ 0.15Ω  │ 500mA    │
  │ BLM18PG  │ 1000Ω     │ 0.2Ω   │ 1A       │
  │ BLM21PG  │ 600Ω      │ 0.05Ω  │ 3A       │
  │ GZ2012   │ 120Ω      │ 0.1Ω   │ 500mA    │
  └──────────┴───────────┴────────┴──────────┘
```

### 5.3 去耦电容设计

```
去耦电容分频段覆盖：

频率段      │ 电容值    │ 封装    │ 作用
───────────┼──────────┼────────┼────────
DC-10kHz   │ 10-100μF │ 电解    │ 低频储能
10k-1MHz   │ 100nF    │ 0402-0805 │ 中频去耦
1-100MHz   │ 10nF     │ 0402   │ 高频去耦
100MHz-1GHz│ 1nF/100pF│ 0201-0402 │ 超高频

布局要点：
  1. 电容尽量靠近IC电源引脚
  2. 过孔直接连接地平面（不要长走线）
  3. 多个电容并联降低ESL
  4. 0402比0805高频特性更好
```

### 5.4 电源滤波模板

```
3.3V数字电源滤波：
  VIN(5V) ──[LDO]──┬──[FB]──┬── 3.3V_D (数字)
                   │        │
               [10μF]    [100nF+10nF]
                   │        │
                  GND      GND

3.3V模拟电源滤波：
  VIN(5V) ──[LDO]──┬──[FB]──┬──[LC]──┬── 3.3V_A (模拟)
                   │        │        │
               [10μF]    [100nF]  [10μF+100nF]
                   │        │        │
                  GND      GND      AGND

关键：模拟地和数字地单点汇合（星形接地）
```

## 六、屏蔽设计

### 6.1 屏蔽效能

```
屏蔽效能 SE = 20log(E入射/E透射) dB

SE = A(吸收损耗) + R(反射损耗) + M(多次反射修正)

典型值：
  0.1mm铜箔：SE > 60dB @ 1MHz
  0.5mm铝板：SE > 80dB @ 100MHz
  金属外壳：SE > 40-80dB

影响因素：
  ├── 材料导电性（铜>铝>钢）
  ├── 厚度（越厚越好）
  ├── 开孔（孔径越小越好）
  └── 接缝（连续导电接触）
```

### 6.2 屏蔽实践

```
屏蔽壳体设计：

1. 开孔处理
   ├─ 孔径 < λ/20（λ为最高干扰频率波长）
   ├─ 多个小孔优于一个大孔
   ├─ 通风孔用蜂窝状波导
   └─ 显示窗口加导电玻璃/丝网

2. 接缝处理
   ├─ 导电衬垫（铍铜簧片/导电橡胶）
   ├─ 螺钉间距 < λ/20
   └─ 搭接阻抗要低

3. 线缆穿透
   ├─ 屏蔽线缆：屏蔽层360°端接
   ├─ 滤波连接器：穿心电容
   └─ 铁氧体磁环：在线缆上套磁环
```

### 6.3 PCB层叠与屏蔽

```
推荐4层板层叠（EMC最佳）：

Layer 1 (Top)    : 信号 + 元件
Layer 2 (Inner1) : GND平面 ← 关键！完整地平面
Layer 3 (Inner2) : 电源平面
Layer 4 (Bottom) : 信号

优势：
  ├── 内层地平面屏蔽顶层信号辐射
  ├── 电源/地平面紧密耦合提供低阻抗供电
  ├── 信号线参考地平面阻抗可控
  └── 高速信号走内层受双层屏蔽

6层板升级：
  L1: Signal  L2: GND  L3: Signal
  L4: Power   L5: GND  L6: Signal
  → 两个地平面提供更好屏蔽
```

## 七、接地设计

### 7.1 接地策略

```
单点接地（低频 < 1MHz）：
  ┌─────┐  ┌─────┐  ┌─────┐
  │数字  │  │模拟  │  │功率  │
  └──┬──┘  └──┬──┘  └──┬──┘
     └────────┴────────┘
              │
          汇合点 → 大地

多点接地（高频 > 10MHz）：
  ┌─────┐  ┌─────┐  ┌─────┐
  │数字  │  │模拟  │  │功率  │
  └──┬──┘  └──┬──┘  └──┬──┘
     ↓        ↓        ↓
  ═════════════════════════  ← 大面积地平面
              ↓
            大地

混合接地（1MHz-10MHz）：
  各模块多点接地到地平面
  地平面与机壳单点连接
```

### 7.2 地平面分割

```
⚠️ 常见错误：随意分割地平面

正确做法：
  1. 高速信号线不跨越地分割
  2. 如果必须跨越，在跨越处放置桥接电容
  3. 模拟/数字地在ADC下方单点连接
  4. 功率地和信号地在电源入口单点连接

ADC/DAC地线处理：
  ┌───────────────────────────┐
  │        AGND区域            │
  │    ┌───────────────┐      │
  │    │    ADC芯片     │      │
  │    └───────┬───────┘      │
  │            │(单点连接)     │
  ├────────────┼──────────────┤
  │        DGND区域            │
  └───────────────────────────┘
```

## 八、电赛EMC实战清单

### 8.1 设计阶段检查清单

```
□ PCB设计
  □ 4层板（至少内层完整地平面）
  □ 高速信号线参考地平面完整
  □ 去耦电容靠近IC电源引脚
  □ 晶振/时钟线走内层或包地
  □ 信号线避免跨越地分割
  □ 连接器附近放置TVS

□ 电源设计
  □ EMI滤波器（共模电感+XY电容）
  □ 每级电源入口去耦
  □ LDO/DCDC输出充分滤波
  □ 数字/模拟电源分开滤波

□ 外壳/线缆
  □ 屏蔽外壳（如有条件）
  □ 外部线缆套磁环
  □ 连接器选择带屏蔽壳的
  □ 屏蔽线缆360°接地

□ 接地
  □ 统一接地策略（单点/多点）
  □ 大电流回路不经过敏感区域
  □ 数字/模拟地单点连接
```

### 8.2 测试阶段快速排查

```
辐射超标：
  1. 频谱分析确定频率 → 找对应噪声源
  2. 近场探头定位热点
  3. 加磁环/铁氧体 → 改善？
  4. 检查地平面完整性
  5. 加屏蔽 → 最后手段

传导超标：
  1. 确定共模还是差模（LISN分离）
  2. 差模：加大X电容、增加差模电感
  3. 共模：加大Y电容、增加共模电感
  4. 检查PCB布线环路面积

抗扰度失败：
  1. 加TVS（信号/电源入口）
  2. 加滤波（RC/磁珠）
  3. 加屏蔽（局部/整体）
  4. 软件看门狗/重试机制
```

### 8.3 快速整改方案

| 问题 | 紧急整改 | 根本整改 |
|------|---------|---------|
| 时钟辐射超标 | 串联电阻22-100Ω | 重布线到内层+包地 |
| DCDC辐射超标 | 输出加LC滤波 | 优化环路面积+屏蔽 |
| 信号线传导 | 加铁氧体磁珠 | 重新设计滤波器 |
| ESD失败 | 信号入口加TVS | 修改PCB布局+屏蔽 |
| 电源浪涌 | 入口加MOV+TVS | 设计多级防护 |
