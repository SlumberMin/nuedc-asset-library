### 2.3 电赛常备容值清单

```

```

### 2.4 去耦电容规则

```
每个IC电源引脚: 100nF陶瓷（就近放置）
每个电源入口: 10μF~100μF电解 + 100nF陶瓷
MCU: VDD每个引脚100nF + VDDA额外100nF+1μF
高速运放: 10μF钽 + 100nF陶瓷 + 10nF陶瓷
```

## 三、电感

### 3.1 常用电感类型

| 类型 | 感值范围 | DCR | 饱和电流 | 适用 |
|------|---------|-----|---------|------|
| 贴片功率电感 | 1μH~10mH | 0.05~5Ω | 0.5~5A | DC-DC |
| 色环电感 | 1μH~10mH | 0.1~10Ω | 小 | EMI滤波 |
| 共模电感 | 100μH~10mH | — | — | 电源EMI |
| 磁珠 | — | — | — | 高频噪声抑制 |

### 3.2 选型要点

```
DC-DC电感:
  L = Vout × (Vin-Vout) / (Vin × f × ΔI)
  ΔI 通常取负载电流的30%~40%
  饱和电流 > 峰值电流 × 1.2
  DCR越小效率越高

磁珠:
  阻抗@目标频率 > 100Ω
  额定电流 > 实际电流 × 1.5
```

## 四、MOSFET

### 4.1 常用型号速查

| 型号 | 类型 | Vds | Id | Rds(on) | Vgs(th) | 封装 | 适用 |
|------|------|-----|-----|---------|---------|------|------|
| **2N7002** | N-MOS | 60V | 300mA | 5Ω | 1~2.5V | SOT-23 | 信号开关 |
| **AO3400** | N-MOS | 30V | 5.7A | 28mΩ | 0.65~1.4V | SOT-23 | **电赛首选小功率** |
| **IRLML6344** | N-MOS | 30V | 5A | 29mΩ | 1~2.3V | SOT-23 | 逻辑电平驱动 |
| **IRF540N** | N-MOS | 100V | 33A | 44mΩ | 2~4V | TO-220 | 中大功率 |
| **IRF3205** | N-MOS | 55V | 110A | 8mΩ | 2~4V | TO-220 | 大功率电机 |
| **IRF9540** | P-MOS | -100V | -23A | 0.2Ω | -2~-4V | TO-220 | 高边开关 |
| **AO3401** | P-MOS | -30V | -4A | 50mΩ | -0.7~-1.3V | SOT-23 | 逻辑电平P-MOS |

### 4.2 MOSFET驱动要点

```
N-MOS做低边开关:
  Vgs > Vgs(th) + 裕量(>2V)
  逻辑电平MOS: 3.3V/5V直接驱动
  标准MOS: 需要10V栅极驱动

P-MOS做高边开关:
  Vgs < -Vgs(th) - 裕量
  栅极通过电阻下拉到地→导通
  栅极接到VCC→关断

栅极电阻: 10Ω~100Ω（抑制振荡）
栅极下拉: 10K~100K（确保默认关断）
```

### 4.3 关键参数速查

```
导通电阻 Rds(on): 越小越好，决定导通损耗
  P_loss = I² × Rds(on)

最大电流 Id: 实际电流不超过50%额定值

阈值电压 Vgs(th): 确保驱动电压足够
  3.3V MCU → 选Vgs(th) < 1.5V的逻辑电平MOS

最大功耗: P_max = (Tj_max - Ta) / Rθja
  SOT-23: Rθja ≈ 200°C/W → 最大约0.5W
  TO-220: Rθja ≈ 62°C/W → 最大约2W（无散热片）
```

## 五、运算放大器

### 5.1 常用型号速查

| 型号 | 供电 | GBW | SR | 噪声 | 特点 | 适用 |
|------|------|-----|-----|------|------|------|
| **LM358** | 3~32V | 1MHz | 0.3V/μs | 中 | 便宜通用 | **电赛通用首选** |
| **OPA2134** | ±2.5~±18V | 8MHz | 20V/μs | 低 | 低噪声音频 | 音频/精密 |
| **OPA2277** | ±2~±18V | 1MHz | 0.8V/μs | 极低 | 高精度 | 精密测量 |
| **TL072** | ±5~±18V | 3MHz | 13V/μs | 低 | JFET输入 | 高阻源 |
| **MCP6002** | 1.8~6V | 1MHz | 0.6V/μs | 中 | 轨到轨 | **3.3V系统首选** |
| **AD8066** | ±5~±24V | 145MHz | 180V/μs | 低 | 高速 | 高速信号调理 |
| **LMV358** | 2.7~5.5V | 1MHz | 1V/μs | 中 | 低电压轨到轨 | 便携设备 |
| **INA128** | ±2.25~±18V | 200KHz | 0.4V/μs | 极低 | 仪表放大器 | **传感器放大首选** |

### 5.2 选型决策树

```
需要放大信号?
├─ 供电3.3V?
│  ├─ 低速(<100KHz)? → MCP6002 / LMV358
│  └─ 高速? → AD8066
├─ 供电5V/±12V?
│  ├─ 通用? → LM358
│  ├─ 低噪声? → OPA2134 / OPA2277
│  ├─ 高阻源? → TL072 (JFET输入)
│  └─ 高速? → AD8066 / THS4011
└─ 差分/桥式传感器?
   └─ INA128 / INA122
```

### 5.3 运放电路速查

```c
// 反相放大: G = -Rf/Rin, Zin = Rin
// 同相放大: G = 1 + Rf/Rg, Zin = 极高
// 差分放大: Vout = (V2-V1) × Rf/Rin
// 积分器: Vout = -1/(RC) × ∫Vin dt
// 微分器: Vout = -RC × dVin/dt
```

| 电路 | 增益公式 | 输入阻抗 | 注意 |
|------|---------|---------|------|
| 反相放大 | -Rf/Rin | Rin | 输入阻抗有限 |
| 同相放大 | 1+Rf/Rg | 极高 | 共模输入 |
| 差分 | Rf/Rin | Rin | 电阻匹配关键 |
| 电压跟随 | 1 | 极高 | 阻抗变换 |
| I-V转换 | -Rf | 极低 | 光电二极管用 |
| 仪表放大 | 1+2R1/Rg | 极高 | **传感器首选** |

### 5.4 关键参数理解

```
GBW (增益带宽积):
  闭环带宽 = GBW / 闭环增益
  例: GBW=1MHz, G=100 → 带宽=10KHz

SR (压摆率):
  最大输出频率 = SR / (2π × Vpeak)
  例: SR=0.3V/μs, Vpeak=2V → fmax≈24KHz

输入偏置电流 Ib:
  JFET/CMOS: <1nA（高阻源用）
  BJT: 10~500nA

输出摆幅:
  普通: 轨到轨约±1.5V（±15V供电时）
  轨到轨: 几乎到电源轨
```
