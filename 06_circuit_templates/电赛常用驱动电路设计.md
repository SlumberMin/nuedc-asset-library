## 1. LED驱动电路

### 1.1 基础LED驱动（限流电阻）

```
MCU GPIO ──── R ──── LED ──── GND

限流电阻计算：
R = (Vcc - Vf) / If

常见LED参数：
  红色：Vf≈2.0V, If=20mA
  绿色：Vf≈2.2V, If=20mA
  蓝色/白色：Vf≈3.2V, If=20mA
  
例：3.3V驱动红色LED
R = (3.3 - 2.0) / 0.02 = 65Ω → 选68Ω标准值
```

### 1.2 MOSFET驱动大电流LED/LED阵列

```
VCC (12V/24V)
 │
 ├─── LED串 ─── LED串 ─── LED串
 │
 ├─── R ─── LED串
 │
 D (MOSFET Drain)
 │
 ┌─────┐
 │ NMOS │ ← GPIO通过限流电阻(100Ω~1kΩ)驱动Gate
 │(IRLML2502)│
 └─────┘
 S ─── GND

选型注意：
- Vgs(th) < GPIO高电平（逻辑电平MOSFET）
- Rds(on)足够低，减少功耗
- Id > 总电流需求
```

### 1.3 恒流LED驱动

```
方案1：晶体管恒流源
VCC ─── LED ─── Collector(NPN)
                       │
                  ┌────┤
                  │    Emitter
                  R    │
                  │   GND
                  └────┤
                       │
                 Base ─ R ─ MCU

I_LED ≈ (Vb - 0.7) / R

方案2：专用恒流芯片
  PT4115 / AL8805 / TPS92512
  输入PWM调光，输出恒流
```

### 1.4 WS2812B（智能LED）驱动

```
数据时序（VDD=5V）：
  T0H: 350ns ±150ns (高电平)
  T0L: 800ns ±150ns (低电平)  → 逻辑0
  T1H: 700ns ±150ns (高电平)
  T1L: 600ns ±150ns (低电平)  → 逻辑1
  Reset: >50μs

接线：
  MCU DATA ─── 330Ω ─── DIN ─── DOUT ─── DIN（级联）
  VCC(5V) 和 GND 需要并联 100μF 电容

注意：3.3V MCU可能需要电平转换（SN74LVC1T45）
```

## 2. 继电器驱动电路

### 2.1 NPN三极管驱动继电器

```
         VCC(继电器电源，如12V)
          │
       ┌──┴──┐
       │继电器│
       │线圈  │
       └──┬──┘
          │
      C ──┘
      │
  ┌───┤  NPN (2N2222 / S8050)
  │   │
  │   E ─── GND
  │
  └── R(1kΩ) ─── MCU GPIO
  │
  ├── 续流二极管 (1N4148) 反向并联
  │   (阴极接VCC，阳极接集电极)

关键参数：
- 继电器线圈电压：5V/12V/24V
- 线圈电流：通常20~150mA
- 三极管 Ic > 线圈电流
- 续流二极管保护三极管免受反向EMF
```

### 2.2 MOSFET驱动继电器

```
         VCC
          │
       ┌──┴──┐
       │继电器│
       │线圈  │
       └──┬──┘
          │
      D ──┘
      │
  ┌───┤  NMOS (AO3400 / IRLML2502)
  │   │
  │   S ─── GND
  │   │
  G ──┘
  │
  R(10kΩ)──GND (下拉，防误触发)
  │
  └── R(100Ω) ─── MCU GPIO

续流二极管：1N4007 反向并联于继电器线圈
```

### 2.3 光耦隔离驱动（推荐用于强电）

```
MCU GPIO ─── R(330Ω) ─── 光耦LED ─── GND
                (PC817)

光耦集电极 ─── VCC(继电器电源)
                 │
              ┌──┴──┐
              │继电器│
              └──┬──┘
光耦发射极 ─── 三极管Base ─── R(10kΩ)──VCC
                              │
                         NPN C──┘
                              E──GND

优点：MCU与继电器电源完全隔离
```

### 2.4 继电器选型要点

| 参数 | 说明 |
|------|------|
| 线圈电压 | 匹配驱动电源 |
| 触点容量 | 电压/电流超过负载需求 |
| 吸合/释放时间 | 快速切换场景注意 |
| 机械寿命 | 频繁切换选固态继电器 |
| 触点形式 | 常开(NO)/常闭(COM)/转换(CO) |

## 3. 电磁阀驱动电路

### 3.1 电磁阀特性

```
电磁阀：线圈通电产生磁场，驱动阀芯
特点：
- 感性负载（类似继电器，但电流更大）
- 通常DC 12V/24V，电流0.5~2A
- 需要续流保护
- 可能需要快速关断（反向电动势泄放）
```

### 3.2 N-MOSFET驱动方案

```
        VCC(24V)
         │
      ┌──┴──┐
      │电磁阀│
      │线圈  │
      └──┬──┘
         │
     D ──┘
     │
 ┌───┤  NMOS (IRF540N / IRLZ44N)
 │   │
 │   S ─── GND (通过R_sense可选)
 │
 G ─── R(100Ω) ─── 驱动电路
 │
 R(10kΩ)──GND

续流保护：
D ──┬── 电磁阀 ──┬── VCC
    │             │
    └─ 1N5408 ────┘
    (快恢复二极管或TVS)

快速关断方案（加TVS钳位）：
电磁阀两端并联 TVS (如 1.5KE24A)
可加速阀芯回落
```

### 3.3 半桥驱动（需要双向电磁阀）

```
         VCC
          │
       ┌──┴──┐
    Q1(P-MOS)│
       └──┬──┘
          ├─── 电磁阀 ───┐
       ┌──┴──┐           │
    Q2(N-MOS)│           │
       └──┬──┘           │
          │              │
         GND             │
                    ┌──┴──┐
                    │Q4 N │
                    └──┬──┘
                    GND

正向：Q1+Q4导通
反向：Q2+Q3导通
关闭：全部关断（续流通过体二极管）

注意：同一桥臂上下管不能同时导通（直通保护）
```

## 4. 压电蜂鸣器驱动

### 4.1 无源蜂鸣器（需要外部信号）

```
方案1：NPN直接驱动
MCU PWM ─── R(1kΩ) ─── Base
                        │
                   NPN (S8050)
                        │
                   C ───┼── 蜂鸣器 ─── VCC(5V/12V)
                        │
                   E ─── GND

方案2：H桥驱动（可切换极性，提高音量）
         VCC
          │
    Q1(P)─┤──Q2(P)
          │
    蜂鸣器─┤
          │
    Q3(N)─┤──Q4(N)
          │
         GND

驱动频率：通常1~4kHz（蜂鸣器谐振频率）
```

### 4.2 有源蜂鸣器（内置振荡器）

```
MCU GPIO ─── R(1kΩ) ─── Base
                         │
                    NPN (S8050)
                         │
                    C ───┼── 有源蜂鸣器 ─── VCC
                         │
                    E ─── GND

只需给高/低电平即可发声/停止
```

### 4.3 音量控制

```
PWM占空比控制音量：
占空比50% → 最大音量
占空比减小 → 音量减小

电压驱动（通过DAC + 放大器）
```

## 5. 电机驱动电路

### 5.1 直流电机基本驱动

#### N-MOS单向驱动

```
        VCC(Motor)
         │
      ┌──┴──┐
      │ DC   │
      │ Motor│
      └──┬──┘
         │
     D ──┘
     │
 ┌───┤  NMOS (IRF540N / IRLZ44N)
 │   │
 │   S ─── R(0.1Ω) ─── GND  (电流采样可选)
 │
 G ─── 驱动电路

续流二极管：Motor两端反并联快恢复二极管
```

#### H桥双向驱动（分立元件）

```
          VCC
     ┌─────┴─────┐
     │           │
  ┌──┤──┐     ┌──┤──┐
  │Q1(P)│     │Q2(P)│
  └──┤──┘     └──┤──┘
     ├─── Motor ──┤
  ┌──┤──┐     ┌──┤──┐
  │Q3(N)│     │Q4(N)│
  └──┤──┘     └──┤──┘
     │           │
     └─────┬─────┘
          GND

正转：Q1+Q4导通
反转：Q2+Q3导通
制动：Q3+Q4导通（或Q1+Q2）
滑行：全部关断
PWM调速：对导通对进行PWM

死区时间：上下管切换时插入1~2μs死区，防止直通
```

### 5.2 专用电机驱动芯片

#### L298N（双H桥）

```
       ┌─────────────┐
VCC ───┤VS        OUT1├──── Motor A
5V ───┤VSS       OUT2├──── Motor A
       │             │
ENA ───┤EN1       OUT3├──── Motor B
IN1 ───┤IN1       OUT4├──── Motor B
IN2 ───┤IN2       EN2├──── ENB
       │             │
GND ───┤GND         │
       └─────────────┘

IN1=1, IN2=0 → 正转
IN1=0, IN2=1 → 反转
IN1=IN2 → 制动
ENA=PWM → 调速

注意：L298N压降约2V，大电流时发热严重
适用：VCC≤46V, I≤2A
```

#### TB6612FNG（推荐）

```
       ┌─────────────┐
VM ───┤VM       AOUT1├──── Motor A
VCC ───┤VCC      AOUT2├──── Motor A
       │             │
PWMA ─┤PWMA      BOUT1├──── Motor B
AIN1 ─┤AIN1      BOUT2├──── Motor B
AIN2 ─┤AIN2      PWMB├──── PWMB
BIN1 ─┤BIN1         │
BIN2 ─┤BIN2         │
STBY ─┤STBY         │
GND ───┤GND         │
       └─────────────┘

优点：低压降(0.5V)，效率高，体积小
适用：VM≤13.5V, I≤1.2A(持续)/3.2A(峰值)
```

#### DRV8825 / A4988（步进电机）

```
       ┌─────────────┐
VMOT ──┤VMOT     OUT1A├── 步进电机A相
VDD ───┤VDD      OUT1B├── 步进电机A相
       │          OUT2A├── 步进电机B相
STEP ──┤STEP     OUT2B├── 步进电机B相
DIR ───┤DIR         │
EN ────┤EN          │
MS1 ───┤MS1         │
MS2 ───┤MS2         │
MS3 ───┤MS3         │
GND ───┤GND         │
       └─────────────┘

细分设置（MS1/MS2/MS3）：
  0/0/0 → 全步
  1/0/0 → 1/2步
  0/1/0 → 1/4步
  1/1/0 → 1/8步
  1/1/1 → 1/32步 (DRV8825) / 1/16步 (A4988)
```

### 5.3 电调（ESC）驱动无刷电机

```
MCU PWM信号 ──── ESC ──── 三相无刷电机

PWM信号：
  1000μs → 最小油门（停止）
  1500μs → 中位
  2000μs → 最大油门

初始化序列：
  1. 发送1000μs信号持续2秒
  2. 等待ESC"滴滴"确认音
  3. 即可正常控制
```

### 5.4 伺服舵机驱动

```
MCU PWM ──── 舵机信号线(橙/白)
VCC(5V) ──── 舵机电源线(红)
GND ──────── 舵机地线(棕/黑)

PWM参数：
  频率：50Hz（周期20ms）
  0°：500μs 脉宽
  90°：1500μs 脉宽
  180°：2500μs 脉宽

注意：
- 多舵机同时运动时，电源电流可能很大
- 建议舵机独立供电，与MCU共地
- 加100~470μF电解电容稳压
```

## 6. 驱动电路设计检查清单

| 检查项 | 说明 |
|-------|------|
| □ 续流保护 | 所有感性负载（继电器/电机/电磁阀）加续流二极管 |
| □ 驱动能力 | GPIO/MOSFET/三极管电流能力 > 负载电流 |
| □ 电压匹配 | 驱动电压与负载额定电压匹配 |
| □ 散热 | 大电流器件需散热片 |
| □ 隔离 | 强电回路考虑光耦/继电器隔离 |
| □ 死区时间 | H桥必须有死区，防止直通 |
| □ 去耦电容 | 驱动芯片VCC加0.1μF+10μF电容 |
| □ 限流保护 | 过流保护电路或保险丝 |
| □ 反接保护 | 电源输入防反接二极管或MOSFET |
