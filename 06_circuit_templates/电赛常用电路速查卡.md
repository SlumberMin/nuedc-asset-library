# ⚡ 电赛常用电路速查卡（打印用）

---

## 一、电机驱动电路

### 1. L298N双H桥（直流电机，≤2A）
```
        VCC(12V) ──┬── VS
                    │
 IN1 ──┐     ┌─────┤──── OUT1 ──→ 电机+
       │ L298N     │
 IN2 ──┘     └─────┤──── OUT2 ──→ 电机-
                    │
        GND ───────┴── GND    ENA ──→ PWM调速

接线: IN1=HIGH, IN2=LOW → 正转
      IN1=LOW, IN2=HIGH → 反转
      ENA=PWM → 调速（0~255对应0~100%）
注意: VS和VSS分开供电，GND共地
```

### 2. TB6612FNG（直流电机，≤1.2A，效率更高）
```
 VM ──→ 电机电源(4.5~13.5V)
 VCC ──→ 逻辑电源(2.7~5.5V)
 AIN1/AIN2 ──→ 方向控制
 PWMA ──→ PWM调速
 AO1/AO2 ──→ 电机
 STBY ──→ HIGH使能
```

### 3. MOS管H桥（大电流电机）
```
         VCC
          │
    ┌──Q1(P-MOS)──┐
    │              │
PWM─┤    电机M     ├─GND
    │              │
    └──Q2(N-MOS)──┘
          │
         GND

单方向驱动（N-MOS低边驱动）:
  MCU_PWM ──→ [10kΩ] ──→ Gate
                          │
                   Drain ──┤── 电机
                          │
  GND ──────────── Source

双方向需H桥(4个MOS) + 死区控制
```

### 4. 步进电机驱动（A4988/DRV8825）
```
 VDD ──→ 逻辑电源3.3/5V
 VMOT ──→ 电机电源(8~35V)
 STEP ──→ 脉冲信号（每脉冲一步）
 DIR ──→ 方向（HIGH/LOW）
 MS1/MS2/MS3 ──→ 细分设置
 ENABLE ──→ LOW使能

细分设置: MS1=H,MS2=H,MS3=H → 1/16细分
         MS1=L,MS2=L,MS3=L → 全步
```

### 5. 舵机驱动
```
信号线(橙/白) ──→ MCU PWM引脚
电源线(红) ──→ 5V（大舵机用独立电源）
地线(棕) ──→ GND

PWM参数: 周期20ms(50Hz)
  0°  → 脉宽0.5ms → 占空比2.5%
  90° → 脉宽1.5ms → 占空比7.5%
  180°→ 脉宽2.5ms → 占空比12.5%
```

## 二、传感器接口电路

### 1. 电压测量（分压）
```
Vin ──[R1]──┬──[R2]──→ GND
             │
             └──→ ADC引脚

Vout = Vin × R2/(R1+R2)
例: 测0~30V，ADC量程3.3V
  R1=100kΩ, R2=11kΩ → Vout_max = 30×11/111 ≈ 2.97V ✓
建议并联100nF滤波电容
```

### 2. 电流测量（分流器+运放）
```
         Rshunt(0.1Ω)
负载+ ──┤├────→ 负载

INA128/INA219接法:
  IN+ ──→ Rshunt高压侧
  IN- ──→ Rshunt低压侧
  REF ──→ GND或VCC/2
  Vout ──→ ADC

例: Rshunt=0.1Ω, I=1A → Vshunt=0.1V
  增益G=10 → Vout=1V → ADC读数
```

### 3. 红外避障（TCRT5000）
```
VCC(5V) ──┬──→ LED阳极(通过限流电阻)
           │
MCU_ADC ←──┤──→ OUT(集电极开路，需上拉)
           │
GND ───────┴──→ GND

上拉电阻: 10kΩ到VCC
输出: 近→LOW, 远→HIGH（或模拟量）
```

### 4. 超声波测距（HC-SR04）
```
VCC(5V) ──→ VCC
Trig ──→ MCU输出引脚
Echo ──→ MCU输入引脚（或经分压3.3V）
GND ──→ GND

触发: Trig拉高10μs
距离: D = Echo高电平时间 × 340 / 2 (cm)
例: 100μs → D = 0.1ms × 34000 / 2 = 1.7cm
```

### 5. 温度传感器（DS18B20）
```
VCC(3.3/5V) ──[4.7kΩ上拉]──┬──→ DQ
                              │
MCU_IO ←──────────────────────┘
GND ──→ GND

单总线协议，精度±0.5°C
```

### 6. 电位器/角度传感器
```
VCC ──┬──→ 电位器两端
GND ──┘
         滑动端 ──→ ADC
         
并联100nF电容滤波
ADC值 → 角度 = ADC/4096 × 360°
```

## 三、电源电路

### 1. AMS1117-3.3/5.0（LDO线性稳压）
```
Vin(5~12V) ──→ IN    ┌─── 10μF(输入)
                      │
GND ──────────────── GND
                      │
              OUT ────┴─── 10μF(输出) ──→ 3.3V/5V

特点: 简单、纹波小、效率低
最大电流: 800mA
功耗: (Vin-Vout)×Iout，注意散热
```

### 2. LM2596（DC-DC Buck降压）
```
Vin(7~40V) ──→ IN
               │
         ┌─────┤ 330μH电感 ──→ Vout
         │     │                    │
        D(肖特基) C(220μF)          │
         │     │                    │
GND ─────┴─────┴────────────────────┘
         │
        FB ──→ 分压电阻 ──→ 设置输出电压

Vout = 1.23 × (1 + R2/R1)
例: 12V→5V: R1=1kΩ, R2=3kΩ
效率: 80~90%
```

### 3. XL6009（DC-DC Buck-Boost升降压）
```
Vin(5~32V) ──→ IN+
GND ──→ IN-

可输出高于或低于输入的电压
适合电池供电场景
```

### 4. 电池充电管理（TP4056）
```
USB 5V ──→ IN+
GND ──→ IN-
BAT+ ──→ 锂电池正极
BAT- ──→ 锂电池负极

充电电流由RPROG设置:
  1.2kΩ → 1000mA
  2kΩ → 580mA
  10kΩ → 130mA

自带过充/过放/过流保护
```

## 四、信号调理电路

### 1. 有源低通滤波（运放，二阶）
```
输入 ──[R]──┬──[R]──┬──→ [运放+]──→ 输出
            │       │         │
           C       C      [运放-]──→ 反馈
            │       │
           GND     GND

fc = 1/(2π·R·C)（Sallen-Key拓扑）
例: R=10kΩ, C=10nF → fc ≈ 1.6kHz
```

### 2. 比较器（LM393/过零检测）
```
Vin ──→ IN+
Vref ──→ IN-（或分压）
VCC(5V) ──→ VCC
GND ──→ GND
OUT ──→ MCU（需上拉10kΩ到VCC）

Vin > Vref → HIGH
Vin < Vref → LOW
```

### 3. 电压跟随器（阻抗匹配）
```
信号源 ──→ [运放+]──→ 输出
              │
          [运放-]──→ 直接反馈到输出

增益=1，高输入阻抗，低输出阻抗
用于: ADC前端、传感器缓冲
```

## 五、通信接口电路

### 1. UART串口（最常用）
```
MCU_TX ──→ 对端RX
MCU_RX ←── 对端TX
GND ──→ GND（必须共地！）

TTL电平: 0/3.3V 或 0/5V
RS232: 需MAX232转换（±12V）
```

### 2. RS485（长距离通信）
```
MCU_TX ──→ MAX485 DI
MCU_RX ←── MAX485 RO
MCU_IO ──→ MAX485 DE/RE（方向控制）

MAX485:
  A ──→ 总线A（差分+）
  B ──→ 总线B（差分-）
  DE=HIGH → 发送
  RE=LOW → 接收（DE/RE可短接用一个IO控制）

120Ω终端电阻接在A-B之间（总线两端）
```

### 3. I2C（板内传感器通信）
```
VCC(3.3V) ──[4.7kΩ]──┬── SCL
                       ├── SDA
MCU_SCL ───────────────┘
MCU_SDA ───────────────┘

速率: 标准100kHz，快速400kHz
地址: 7位，查阅器件手册
```

## 六、保护电路

### 1. 电源反接保护（P-MOS）
```
       ┌─── S(Drain方向接电源)
       │
Vin ───┤   P-MOS
       │
       └─── D(接负载)

Vin正向: MOS导通，压降小
Vin反向: MOS截止，保护电路
```

### 2. TVS瞬态抑制
```
信号线 ──┬── TVS ──→ GND
         │
         └──→ 后续电路

选型: 工作电压 < TVS击穿电压 < 器件耐压
例: 3.3V系统选SMBJ3.3A
```

### 3. 自恢复保险丝（PPTC）
```
电源正极 ──→ PPTC ──→ 负载

选型: 额定电流 > 正常工作电流
      跳闸电流 < 线路最大承受电流
例: 工作500mA，选0.5A PPTC
```
