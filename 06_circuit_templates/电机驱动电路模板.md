## 一、TB6612FNG 双路H桥驱动（⭐推荐）

### 1.1 电路描述

```
VM (电机电源, ≤13.8V) ── 100μF电解 + 100nF瓷片
VCC (逻辑电源, 5V) ── 100nF
GND ── 公共地

--- A路电机 ---
AIN1 (Pin 5) ── GPIO
AIN2 (Pin 7) ── GPIO
PWMA (Pin 6) ── PWM信号 (10kHz~100kHz)
AO1 (Pin 9)  ── 电机A端1
AO2 (Pin 10) ── 电机A端2

--- B路电机 ---
BIN1 (Pin 3) ── GPIO
BIN2 (Pin 4) ── GPIO
PWMB (Pin 11)── PWM信号
BO1 (Pin 1)  ── 电机B端1
BO2 (Pin 2)  ── 电机B端2

STBY (Pin 8) ── VCC (高电平使能, 低电平待机)
```

### 1.2 真值表

| IN1 | IN2 | PWM | 状态 |
|-----|-----|-----|------|
| H | L | PWM | 正转（速度由PWM占空比决定） |
| L | H | PWM | 反转 |
| L | L | X | 制动（短路制动） |
| H | H | X | 制动 |

### 1.3 PCB要点
- VM电容100μF+100nF紧贴芯片引脚
- 电流回路走线≥1mm
- 散热焊盘连接GND铺铜散热
- 输入信号加33Ω串联电阻防振铃

### 1.4 参数
| 参数 | 值 |
|------|-----|
| 电机电压 | 2.5V ~ 13.8V |
| 逻辑电压 | 2.7V ~ 5.5V |
| 持续电流 | 1.2A/桥 |
| 峰值电流 | 3.2A/桥 |
| 内阻Rds(on) | 0.5Ω |

## 二、L298N 双路H桥驱动（⚠️不推荐新设计）

### 2.1 电路描述

```
VS (电机电源, 5-46V) ── 100μF + 100nF
VSS (逻辑电源, 5V) ── 100nF
GND ── 公共地

ENA (Pin 6) ── PWM/跳线(常高)
IN1 (Pin 5) ── GPIO
IN2 (Pin 7) ── GPIO
OUT1 (Pin 2) ── 电机
OUT2 (Pin 3) ── 电机

ENB ── PWM
IN3, IN4 ── GPIO
OUT3, OUT4 ── 电机

电流检测电阻:
SENSE_A (Pin 1) ── 0.5Ω/1W ── GND
SENSE_B (Pin 15) ── 0.5Ω/1W ── GND
(不需限流则短接到GND)
```

### 2.2 缺点
- **压降大**：内部达林顿管压降≥2V，效率仅60-70%
- **发热严重**：1A电流时功耗≥2W
- **体积大**：Multiwatt15封装
- **速度慢**：PWM频率受限（≤25kHz）

### 2.3 何时使用
- 电机电压>13.8V（超出TB6612范围）
- 已有库存，比赛临时使用

## 三、DRV8825 步进电机驱动（⭐推荐）

### 3.1 电路描述

```
VM (电机电源, 8.2-45V) ── 100μF/50V + 100nF
VDD (逻辑电源, 2.5-5.5V) ── 100nF
GND ── 公共地 (大面积铺铜散热)

STEP (Pin 14) ── GPIO/PWM (步进脉冲)
DIR  (Pin 15) ── GPIO (方向)
nEN  (Pin 10) ── GPIO (低电平使能, 高电平禁止)
nRESET (Pin 11) ── VDD (正常使用) 或 GPIO
nSLEEP (Pin 12) ── VDD (正常使用) 或 GPIO

M0 (Pin 17) ── GPIO (细分设置)
M1 (Pin 18) ── GPIO
M2 (Pin 19) ── GPIO

AOUT1, AOUT2 ── 步进电机A相
BOUT1, BOUT2 ── 步进电机B相

VREF (Pin 12) ── 电流设定 (电位器中间脚)
    电流限制: I = VREF / (5 × Rsense)
    Rsense = 0.1Ω时: I = VREF / 0.5
    例: VREF=0.5V → 限流1A
```

### 3.2 细分设置

| M2 | M1 | M0 | 细分 |
|----|----|----|------|
| L | L | L | 全步 |
| L | L | H | 1/2步 |
| L | H | L | 1/4步 |
| L | H | H | 1/8步 |
| H | L | L | 1/16步 |
| H | L | H | 1/32步 |

### 3.3 PCB要点
- **散热是关键**：底层大面积铺铜+散热过孔（0.3mm×16个以上）
- VM电容100μF紧贴引脚
- 电流采样电阻精度≤1%
- 电机走线≥1.5mm

## 五、通用保护电路

### 5.1 电机续流二极管
```
电机两端并联TVS二极管（如SMBJ16A）或续流二极管
TB6612/DRV8825内部已集成，一般无需外部
L298N建议外部加肖特基二极管（1N5822）
```

### 5.2 电流采样
```
串联采样电阻 + 运放（INA199/INA219）
采样电阻: 0.1Ω-0.5Ω（根据电流选择功率）
输出到MCU ADC引脚
```

### 5.3 电源滤波
```
电机电源入口：100μF电解 + 100nF瓷片
驱动芯片VM引脚：10μF + 100nF
逻辑电源：100nF
```
