# STM32F103C8T6 最小系统电路设计

> 适用封装：LQFP48
> 核心参数：72MHz Cortex-M3, 64KB Flash, 20KB RAM

---

## 一、最小系统原理图描述

### 1.1 电源部分

```
VDD (Pin 24, 36, 48) ──┬── 100nF×3 (去耦, 贴近引脚)
                        └── 10μF×1 (储能)
VSS (Pin 23, 35, 47) ── GND
VDDA (Pin 9) ──┬── 100nF + 10μF (模拟电源, 独立滤波)
               └── 铁氧体磁珠隔离数字电源
VSSA (Pin 8) ── 模拟地 (单点连接数字地)
VBAT (Pin 1) ── 3.3V (无电池则接VDD)
```

**要点：**
- 每个VDD引脚必须就近放置100nF陶瓷电容（0402/0603）
- VDDA需额外加10μF电解电容，远离数字信号
- 电源走线宽度≥0.5mm，主供电走线≥1mm

### 1.2 晶振部分

**高速外部晶振 HSE (8MHz)：**
```
OSC_IN  (Pin 5) ──┬── 8MHz晶振 ──┬── OSC_OUT (Pin 6)
                  └── 20pF ── GND    └── 20pF ── GND
```
- 晶振负载电容 CL = 20pF（匹配晶振规格书）
- 晶振距引脚≤5mm，走线等长、对称
- 晶振下方铺铜禁止，避免寄生电容

**低速外部晶振 LSE (32.768kHz)：**
```
OSC32_IN  (Pin 3) ──┬── 32.768kHz ──┬── OSC32_OUT (Pin 4)
                    └── 10pF ── GND    └── 10pF ── GND
```
- 仅RTC需要时焊接，否则NC

### 1.3 复位电路

```
NRST (Pin 7) ──┬── 10kΩ ── VDD (上拉)
               ├── 100nF ── GND (滤波)
               └── 按键 ── GND (手动复位)
```

### 1.4 BOOT配置

```
BOOT0 (Pin 44) ──┬── 10kΩ ── GND (正常运行, 从Flash启动)
                 └── 跳线/按键 ── VDD (ISP下载模式)

BOOT1 (Pin 20) ── 10kΩ ── GND (通常拉低)
```

| BOOT1 | BOOT0 | 启动模式 |
|-------|-------|----------|
| X | 0 | 从Flash启动（正常） |
| 0 | 1 | 从System Memory启动（ISP） |
| 1 | 1 | 从SRAM启动（调试） |

### 1.5 SWD调试接口

```
         ┌───────────┐
VDD  ────┤ 1  VCC    │
SWDIO ───┤ 2  SWDIO  │  (PA13)
GND   ───┤ 3  GND    │
SWCLK ───┤ 4  SWCLK  │  (PA14)
         └───────────┘
         4pin 2.54mm排针
```
- SWDIO/SWCLK加33Ω串联电阻防反射
- 也可使用标准20pin JTAG接口（引出TDI/TDO/NRST）

### 1.6 未用引脚处理
- 所有未用GPIO悬空或配置为输出低电平
- 未用ADC引脚接地（通过10kΩ电阻）
- 未用模拟引脚接VDDA或GND

## 二、PCB设计要点

### 2.1 板层与尺寸
- **推荐**：双层板（降低成本）或四层板（信号质量更好）
- **最小系统板尺寸**：30mm × 50mm（紧凑型）或 40mm × 60mm（带扩展排针）
- **板厚**：1.6mm（标准）

### 2.2 布局原则
1. MCU居中放置
2. 晶振紧贴MCU对应引脚，≤5mm
3. 去耦电容紧贴VDD引脚，走线尽量短粗
4. 电源部分与信号部分分区
5. 复位按键/BOOT跳线放在板边方便操作

### 2.3 布线规则
| 信号类型 | 最小线宽 | 最大线宽 | 备注 |
|----------|----------|----------|------|
| 电源主干 | 0.8mm | 2mm | 越宽越好 |
| 信号线 | 0.2mm | 0.3mm | 标准即可 |
| 晶振走线 | 0.2mm | - | 等长、包地 |
| SWD调试线 | 0.2mm | - | 短且等长 |

### 2.4 铺铜与接地
- 底层大面积铺铜作为GND平面
- 晶振下方禁止铺铜（去耦区域可铺铜）
- 模拟地与数字地单点连接（星型接地）
- 过孔间距≥0.3mm

### 2.5 丝印标注
- 所有引脚编号丝印
- BOOT模式选择标注
- 电源极性标注（VCC/GND）
- 板名+版本号+日期

## 三、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 无法下载程序 | BOOT0未拉低/接线错误 | 检查BOOT0/SWD接线 |
| 程序跑飞 | 晶振不起振 | 检查晶振/匹配电容 |
| ADC精度差 | VDDA滤波不足 | 加强VDDA滤波 |
| 功耗大 | 未用引脚悬空 | 配置为输出或接上下拉 |
