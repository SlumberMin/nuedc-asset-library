## 1. 电阻电桥

### 1.1 惠斯通电桥

```
        V+
        │
    ┌───┴───┐
    R1      R2
    ├──Vo+──┤
    R3      R4(传感器)
    └───┬───┘
        │
        GND

Vo = V+ × (R4/(R3+R4) - R2/(R1+R2))
平衡条件: R1×R4 = R2×R3
```

### 1.2 半桥差分配置

```
        V+
        │
    ┌───┴───┐
    R+ΔR    R-ΔR   ← 两个应变片（差分）
    ├──Vo+──┤
    R       R
    └───┬───┘
        │
        GND

Vo = V+ × ΔR / (2R)  (小信号近似)
灵敏度是单臂电桥的2倍
```

### 1.3 全桥配置

```
        V+
        │
    ┌───┴───┐
    R+ΔR    R-ΔR
    ├──Vo+──┤
    R-ΔR    R+ΔR
    └───┬───┘
        │
        GND

Vo = V+ × ΔR / R
灵敏度是单臂电桥的4倍
线性度最好
```

### 1.4 电桥激励与ADC接口

```
         5V
          │
     ┌────┴────┐
     │         │
   350Ω     350Ω+ΔR (应变片)
     │         │
     ├─ Vo+ ───┤
     │         │
   350Ω     350Ω-ΔR
     │         │
     └────┬────┘
          │
         GND

满量程输出: ±10mV (1000με应变)
→ 需要仪表放大器放大200~500倍
→ 接24位Σ-Δ ADC (如HX711)
```

### 1.5 电桥调零电路

```
        V+
        │
    ┌───┴───┐
    R1      R2
    ├──Vo+──┤         ┌── 10kΩ多圈电位器 ──┐
    R3      R4        │                     │
    └───┬───┘         │    调零范围: ±5mV   │
        │             └─────────────────────┘
        GND

方法: 电位器中心抽头接Vo+，两端接V+和GND
精度: 选用多圈精密电位器 (0.1%)
```

## 2. 恒流源

### 2.1 基本运放恒流源

```
         V+
          │
          R1 (设定电阻)
          │
    ┌─────┴─────┐
    │   (-)     │
    │   OA ─────┼──→ Iout = Vref / R1
    │   (+)     │
    └─────┬─────┘
          │
        Vref
          │
         GND

Iout = Vref / R1
精度取决于Vref和R1的精度
```

### 2.2 Howland电流泵（浮地恒流源）

```
         V+
          │
          R1
          │
    ┌──┬──┴──┐
    │  │ (-) │
    │  │ OA──┼──┬──→ Iout+
    │  │ (+) │  │
    │  └──┬──┘  R4 (负载)
    │     │     │
    │     R3    │
    │     │     │
    │  ┌──┴──┐  │
    │  │ (-) │  │
    │  │ OA──┼──┴──→ Iout-
    │  │ (+) │
    │  └──┬──┘
    │     R2
    │     │
    │    GND

Iout = Vin / R3 (当R1=R2, R3=R4)
优点: 负载可浮地
```

### 2.3 集成恒流源方案

| 方案 | 电流范围 | 精度 | 说明 |
|------|----------|------|------|
| REF200 | 100μA×2 | 0.5% | 双路恒流 |
| LM334 | 1μA~10mA | 3% | 三端可调 |
| XTR110 | 4-20mA | 0.05% | 电流环变送器 |
| 运放+MOS | μA~A | 取决于电阻 | 灵活 |

### 2.4 PT100恒流源测量电路

```
         3.3V
          │
     ┌────┴────┐
     │   (-)   │
     │   OA────┼──→ 1mA ──→ PT100 ──→ GND
     │   (+)   │         │
     └────┬────┘         │
          │              │
        Vref(1V)         │
          │              │
         GND             ↓
                     Vout = 1mA × Rpt100
                     0℃ → 100Ω → 100mV
                     100℃ → 138.5Ω → 138.5mV

恒流源精度要求: <0.1% (用精密电阻+基准)
```

## 3. 差分放大器

### 3.1 基本差分放大器

```
         Vin+
          │
          R1
          │
    ┌─────┴─────┐
    │   (-)     │
    │   OA ─────┼──→ Vout
    │   (+)     │
    └─────┬─────┘     Vout = (Rf/R) × (Vin+ - Vin-)
          Rf              (当R1=R2=Rf=R3)
          │
    ┌─────┴─────┐
    │           R3
    │           │
    Vin- ─── R2 ─┘

CMRR = 20×log10(Ad/Acm)
实际CMRR受电阻匹配限制，通常40~60dB
```

### 3.2 高CMRR差分放大（三运放）

```
见下方仪表放大器部分
```

### 3.3 电阻匹配与CMRR关系

| 电阻精度 | 典型CMRR | 适用场景 |
|----------|----------|----------|
| 1% | 34dB | 一般用途 |
| 0.1% | 54dB | 中等精度 |
| 0.01% | 74dB | 高精度 |
| 集成IA | >100dB | 专业测量 |

## 4. 仪表放大器（IA）

### 4.1 三运放仪表放大器

```
              V+
              │
    Vin+ ──R1──┤──R3──┤── OA3(-)──┐
              │      │            │
         OA1(+)  Rg(增益设定)   OA3 ──→ Vout
              │      │            │
         OA2(+)  Rg          OA3(+)──┘
              │      │            │
    Vin- ──R1──┤──R3──┤── R3 ────┘
              │                    │
              R2                R2
              │                    │
             GND                  GND

增益: G = 1 + 2×R1/Rg
输出: Vout = G × (Vin+ - Vin-)

CMRR: >80dB (取决于R1匹配)
```

### 4.2 常用仪表放大器IC

| 型号 | CMRR | 增益范围 | 带宽 | 供电 | 价格 | 适用 |
|------|------|----------|------|------|------|------|
| AD620 | 100dB | 1~10000 | 120kHz | ±2.5~±18V | ¥15 | 通用精密 |
| INA128 | 120dB | 1~10000 | 200kHz | ±2.25~±18V | ¥12 | 通用 |
| INA826 | 120dB | 1~10000 | 1MHz | 2.7~36V | ¥8 | 低功耗 |
| INA333 | 100dB | 1~1000 | 150kHz | 1.8~5.5V | ¥6 | 低电压 |
| AD8422 | 110dB | 1~1000 | 1.5MHz | ±2.5~±18V | ¥20 | 高速 |
| INA188 | 110dB | 1~1000 | 500kHz | ±2~±18V | ¥10 | 高压 |

### 4.3 AD620应用电路

```
         +5V
          │
    ┌─────┴─────┐
    │   AD620   │
    │   ┌───┐   │
Vin+→│1─┤   ├─7│→ Vout (Ref=2.5V时双极性输出)
    │   │   │   │
Vin-→│2─┤   ├─6│→ +Vs
    │   │   │   │
REF─→│5─┤   ├─4│→ -Vs
    │   └───┘   │
    │   Rg(8)   │
    └───────────┘

增益设置:
G = 1 + 49.4kΩ/Rg
Rg = 49.4kΩ/(G-1)

| 增益 | Rg | 精密电阻推荐 |
|------|------|------------|
| 10 | 5.49kΩ | 0.1%金属膜 |
| 100 | 499Ω | 0.1%金属膜 |
| 500 | 98.8Ω | 0.1% |
| 1000 | 49.4Ω | 0.1% |
```

### 4.4 低噪声IA设计要点

```
1. 电源去耦: 每个供电脚对地100nF + 10μF
2. 输入保护: TVS二极管 + 限流电阻
3. 屏蔽: 输入线用屏蔽电缆，屏蔽层接REF
4. PCB布局: 输入端远离数字信号线
5. 增益设定: Rg用0.1%精密电阻
6. REF引脚: 接低阻抗基准（如运放缓冲后的VCC/2）
7. 信号带宽: 带通限制在需要的范围内
```

### 5.2 AMC1200隔离测量电路

```
    高压侧 (HV)              低压侧 (LV)
    ┌──────────┐             ┌──────────┐
    │ AMC1200  │             │ AMC1200  │
    │          │   隔离栅     │          │
Vin+→│VINP   VOUTP├───────────┤VINP   VOUTP├→ 到ADC
Vin-→│VINN   VOUTN├───────────┤VINN   VOUTN├→ 到ADC
    │          │             │          │
    │ VDD1  GND1│             │VDD2   GND2│
    └──┬────┬──┘             └──┬────┬──┘
       │    │                   │    │
     +5V   GND1              +5V   GND2(本地)

隔离电压: 4250Vpk
增益: 固定×8
输入范围: ±250mV
输出: 差分, ±2V
```

### 5.3 数字隔离方案

```
方案1: 光耦隔离 (慢速信号)
    MCU_TX ──→ [光耦] ──→ 对端_RX
    速率: <100kbps
    隔离: 2500~5000Vrms

方案2: 磁耦隔离 (高速信号)
    MCU_SPI ──→ [ADUM1402] ──→ 对端_SPI
    速率: 90Mbps
    隔离: 2500Vrms

方案3: 电容隔离 (中速信号)
    MCU_CAN ──→ [ISO1050] ──→ CAN总线
    速率: 1Mbps
    隔离: 2500Vrms
```

### 5.4 电源隔离

```
方案1: 隔离DC-DC模块
    5V ──→ [B0505S-1W] ──→ 隔离5V
    隔离: 1000Vdc
    功率: 1W

方案2: 变压器隔离
    ┌──────┐        ┌──────┐
    │PWM驱动│→ 变压器 →│整流滤波│→ 隔离电源
    └──────┘        └──────┘
    功率: 可达数十W

方案3: 隔离+非隔离组合
    本地5V ──→ [LM2596] ──→ 本地12V
    隔离5V ──→ [B0505S] ──→ 隔离5V (模拟前端)
```

## 6. 典型传感器调理电路

### 6.1 应变片/称重传感器

```
称重传感器(全桥,2mV/V)
    │
    ├─ 5V激励
    │
    信号 ──→ AD620 (G=500) ──→ 低通滤波 ──→ HX711/ADC
         Rg = 99Ω

满量程输出: 10mV (5V激励)
放大后: 5V
分辨率: HX711 24位
```

### 6.2 热电偶（K型）

```
K型热电偶: 41μV/℃
    │
    ├─ 冷端补偿 (DS18B20)
    │
    信号 ──→ INA128 (G=100) ──→ 低通 ──→ ADC
         Rg = 494Ω

0℃ → 0mV → 0V
500℃ → 20.6mV → 2.06V
1000℃ → 41.3mV → 4.13V
```

### 6.3 光电二极管

```
           ┌───┐
    PD ────┤(-)│
    │      │ OA├──→ Vout
    │  Cf──┤(+)│
    │  │   └───┘
    │  │
    │ GND

跨阻放大器(TIA):
Vout = -Ipd × Rf

PD: 硅光电二极管, 暗电流<1nA
Rf = 1MΩ~100MΩ (低光照)
Cf = Rf/带宽补偿 (防止振荡)
```

### 6.4 pH电极

```
pH电极: ~59.2mV/pH (25℃), 阻抗~100MΩ
    │
    ┌──────┐
    │OPA129│ (超低偏置电流 <1pA)
    │  (-) ┤
    │  OA  ├──→ 第一级 (G=1)
    │  (+) ┤
    └──┬───┘
       │
       R(10MΩ,输入保护)
       │
    pH电极

    第一级输出 ──→ AD620 (G=5) ──→ ADC
    
    pH=7 → 0mV → 0V
    pH=0 → +414mV → +2.07V
    pH=14 → -414mV → -2.07V
    (需±5V双电源)
```

### 6.5 加速度计（压电式）

```
压电加速度计: ~100pC/g
    │
    ┌────────┐
    │电荷放大器│
    │   Cf   │
    │  ┌──┐  │
    PD─┤OA├──→ Vout = Q/Cf
    │  └──┘  │     = 100pC/100pF
    └────────┘     = 1V/g

Cf = 100pF → 1V/g
后接高通滤波 (0.1Hz) 去直流
后接低通滤波 (10kHz) 抗混叠
```

## 7. 通用抗混叠滤波器

```
         R (10kΩ)
Vin ──┤├──┬──→ Vout
            │
            C (根据截止频率选择)
            │
           GND

fc = 1/(2πRC)

| 采样率 | 截止频率 | R | C |
|--------|----------|------|------|
| 1kHz | 200Hz | 10kΩ | 80nF |
| 10kHz | 2kHz | 10kΩ | 8nF |
| 100kHz | 20kHz | 1kΩ | 8nF |

更高阶: 用Sallen-Key二阶滤波器
```

## 8. 信号调理电路设计检查清单

| 检查项 | 要求 | 验证方法 |
|--------|------|----------|
| 输入保护 | TVS+限流 | ESD测试 |
| 共模电压范围 | 不超IA输入范围 | 计算+示波器 |
| 电源去耦 | 100nF+10μF | 示波器纹波 |
| 接地回路 | 单点接地 | 测地线电流 |
| 带通限制 | 抗混叠+噪声抑制 | 频响测试 |
| CMRR | 满足共模抑制要求 | 指标计算 |
| 温漂 | <应用要求 | 温度循环测试 |
| PCB布局 | 模拟/数字分区 | 审查 |
| 屏蔽 | 输入线屏蔽 | 噪声测试 |
| 基准精度 | 满足系统精度 | 万用表验证 |
