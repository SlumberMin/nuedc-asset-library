# 信号调理电路模板

## 1. 概述

信号调理是将传感器原始信号转换为MCU/ADC可处理的标准信号。核心环节：**放大 → 滤波 → 电平转换 → ADC驱动**。

### 信号调理链路
```
传感器 → [缓冲/阻抗匹配] → [放大] → [滤波] → [限幅保护] → ADC
```

## 2. 运算放大器电路

### 2.1 常用运放选型

| 型号 | 带宽 | 失调 | 轨到轨 | 供电 | 适用场景 |
|------|------|------|--------|------|----------|
| LM358 | 1MHz | 3mV | 否(输出) | 3-32V | 通用低速 |
| OPA340 | 5.5MHz | 150μV | 是 | 2.5-5.5V | 低电压精密 |
| OPA2188 | 2MHz | 25μV | 是 | 4-36V | 高精度 |
| MCP6002 | 1MHz | 4.5mV | 是 | 1.8-6V | 低功耗/便宜 |
| AD620 | - | 30μV | - | ±2.5-18V | 仪表放大器 |
| TL072 | 3MHz | 5mV | 否 | ±5-18V | 音频/通用 |

### 2.2 同相放大器

```
         Rf
    ┌───[10kΩ]───┐
    │             │
    │   ┌─────┐  │
Vin─┤──→│  +   │  │
    │   │ OPA  ├──┴──→ Vout = Vin × (1 + Rf/Rg)
    │   │  -   │
    │   └──┬──┘
    │      │
    └──[Rg=1kΩ]──→ GND
```

**增益公式：** `Av = 1 + Rf/Rg`
- Rf=10kΩ, Rg=1kΩ → Av = 11倍
- Rf=100kΩ, Rg=10kΩ → Av = 11倍

**设计要点：**
- 输入阻抗高（适合高阻传感器）
- 同相端加偏置可设置直流工作点
- 补偿电阻 `Rp = Rf // Rg` 接到同相端减小偏置电流误差

### 2.3 反相放大器

```
        Rf
   ┌──[10kΩ]──┐
   │           │
   │  ┌─────┐  │
   ├──│  -   │  │
   │  │ OPA  ├──┴──→ Vout = -Vin × (Rf/Rin)
   │  │  +   │
   │  └──┬──┘
   │     │
  [Rin=1kΩ]  GND
   │
  Vin
```

**增益公式：** `Av = -Rf/Rin`
- 输入阻抗 = Rin（注意负载效应）

### 2.4 差分放大器

```
           Rf=10kΩ
      ┌────[10kΩ]────┐
      │               │
      │  ┌───────┐    │
   R1 │  │       │    │
V1──[10kΩ]─ -     │    │
      │  │  OPA  ├────┴──→ Vout
      │  │  +    │
      └──│───────│──┘
         │       │
V2──[10kΩ]─ +    │
         │       │
        [Rg=10kΩ]│
         │       │
        GND      │
```

**输出公式：** `Vout = (V2 - V1) × (Rf/R1)`
- 抑制共模干扰
- 适合桥式传感器（应变片、压力传感器）

### 2.5 仪表放大器（推荐）

```
         AD620 / INA128
         ┌───────────┐
    RG   │           │
   ┌─┤   │           │
   │ │   │  IN+  OUT ├──→ Vout
   │ │   │  IN-  REF ├──→ Vref(可选)
   └─┤   │           │
         └───────────┘

增益设置：G = 1 + 49.4kΩ/RG
  RG=49.9kΩ → G=2
  RG=5.49kΩ → G=10
  RG=549Ω   → G=91
  RG=49.9Ω  → G=1000
```

**优势：** 高共模抑制比(CMRR>100dB)，适合微弱差分信号。

## 3. 滤波器电路

### 3.1 一阶RC低通滤波器

```
Vin ──[R]──┬──→ Vout
            │
           [C]
            │
           GND

截止频率：fc = 1/(2π×R×C)
  R=10kΩ, C=100nF → fc = 159Hz
  R=1kΩ, C=1μF → fc = 159Hz
```

**适用：** 简单去噪、ADC输入抗混叠

### 3.2 二阶Sallen-Key低通滤波器

```
       R1      R2
Vin──[10kΩ]──[10kΩ]──┬──→ OPA(+)──→ Vout
                      │              │
                     [C1]            └──反馈──┘
                      │
                     GND
                      │
节点 ────────────────[C2=100nF]──→ GND
```

**参数（Butterworth Q=0.707）：**
- R1=R2=R, C1=2C2
- fc = 1/(2π×R×√(C1×C2))
- R=10kΩ, C1=10nF, C2=4.7nF → fc ≈ 1.6kHz

### 3.3 二阶有源带通滤波器

```
           C1
Vin──[C]──┤──[R]──┬──→ OPA(+)──→ Vout
          │       │
         GND     [C2]
                  │
                 GND
```

**中心频率：** f0 = 1/(2π×√(R1×R2×C1×C2))
**带宽：** BW = f0/Q

### 3.4 常用滤波器参数速查

| 信号类型 | 推荐截止频率 | 电路 | 说明 |
|----------|-------------|------|------|
| 直流电压 | 10-100Hz | RC低通 | 去除工频干扰 |
| 电机电流 | 1-10kHz | 二阶低通 | 去PWM纹波 |
| 音频 | 20Hz-20kHz | 带通 | 语音识别 |
| 巡线红外 | 50-500Hz | RC低通 | 去环境光闪烁 |
| 超声波 | 38-40kHz | 带通 | 超声接收滤波 |
| 编码器 | 不滤波 | - | 数字信号直接进MCU |

## 4. 比较器电路

### 4.1 基本电压比较器

```
        Vref (基准电压)
         │
    ┌────┤
    │  ┌─┴───┐
    │  │  +   │
    │  │ COMP ├──→ Vo (开漏/推挽)
    │  │  -   │
    │  └─┬───┘
    │    │
   Vin

当 Vin > Vref → Vo = HIGH
当 Vin < Vref → Vo = LOW
```

### 4.2 迟滞比较器（施密特触发器）

```
            R2=100kΩ
       ┌───[100kΩ]───┐
       │              │
       │  ┌───────┐   │
    R1 │  │  +     │   │
Vref─[10kΩ]─┤ COMP  ├───┴──→ Vo
       │  │  -     │
       │  └───┬───┘
       │      │
      Vin

上阈值：VH = Vref × R2/(R1+R2) + Vsat × R1/(R1+R2)
下阈值：VL = Vref × R2/(R1+R2) - Vsat × R1/(R1+R2)
迟滞量：ΔV = VH - VL
```

**用途：** 消除信号在阈值附近的抖动（按键消抖、电平检测）

### 4.3 LM393 双路比较器典型电路

```
         VCC(3.3V/5V)
          │
        [10kΩ上拉]  ← 开漏输出需上拉
          │
LM393 OUT─┤
          │
    ┌─────┤
    │     └──→ 到MCU GPIO
    │
    └──[100nF]──GND  ← 电源退耦
```

### 4.4 窗口比较器（过压/欠压检测）

```
           ┌─────────┐
  Vcc/HIGH─┤COMP1  + ├──→ 上限比较
           │    -    │     │
           └────┬────┘     │
  Vin──────────┤      ┌───AND──→ 超限报警
           ┌────┴────┐     │
  GND/LOW ─┤COMP2  - ├──→ 下限比较
           │    +    │
           └─────────┘

Vin在范围内→输出HIGH，超出范围→输出LOW
```

## 5. ADC驱动电路

### 5.1 ADC输入保护与驱动

```
传感器信号
    │
   [100Ω]  ← 限流电阻
    │
   [TVS 3.3V]  ← ESD/过压保护 (双向TVS)
    │
    ├────[100nF]──→ GND  ← 抗混叠滤波
    │
    ├────[10kΩ]──→ ADC输入
    │
   GND
```

### 5.2 单端转差分ADC驱动

```
         ┌── OPA_A(+) ──→ ADC_IN+
Vin ──┤                 │
         └── OPA_B(-) ──→ ADC_IN-
              │
            Vcm = VCC/2 (共模电压)
```

**适用：** ADS1115(差分)、STM32内部ADC差分模式

### 5.3 电流-电压转换（跨阻放大器）

```
光电二极管/互感器
    │
    ├──[Rf=10kΩ]──┐
    │              │
    │  ┌───────┐   │
    └──│  -     │   │
       │ OPA   ├───┴──→ Vout = -Iin × Rf
       │  +     │
       └───┬───┘
           │
          Vcm(偏置)
          [Cf=10pF]  ← 反馈电容(稳定)
```

### 5.4 电桥信号调理（压力/应变片）

```
        VCC
         │
      ┌──┴──┐
      R1    R2  ← 应变片(可变电阻)
      │     │
      ├─Vout─┤  ← 差分输出(mV级)
      │     │
      R3    R4
      │     │
      └──┬──┘
         │
        GND

→ AD620仪表放大器(G=100~1000) → 低通滤波 → ADC
```

## 6. 实用电路模板汇总

### 6.1 电池电压检测

```
电池(12V)──[30kΩ]──┬──[10kΩ]──→ GND
                    │
                    ├──[100nF]──→ GND
                    │
                    └──→ ADC(0-3.0V)

分压比: 10/(30+10) = 0.25, 12V×0.25 = 3.0V
```

### 6.2 电流检测（高侧）

```
VCC──→ [Rshunt=0.1Ω] ──→ 负载
              │
         ┌────┴────┐
         │  INA219  │  → I2C输出(直接读电流)
         └─────────┘

或模拟方案:
VCC──→ [Rshunt=0.1Ω] ──→ 负载
              │差分│
         AD620(G=20) → ADC
         1A → 0.1Ω × 20 = 2.0V
```

### 6.3 温度检测（NTC热敏电阻）

```
VCC(3.3V)
   │
  [R_fixed=10kΩ]
   │
   ├────→ ADC输入
   │
  [NTC 10kΩ@25°C]
   │
  GND

B值公式: R_NTC = R0 × exp(B × (1/T - 1/T0))
查表或Steinhart-Hart方程计算温度
```

### 6.4 电机反电动势检测

```
电机端 ──[100kΩ]──┬──[10kΩ]──→ ADC
                   │
                  [100nF] ← 低通滤波
                   │
                  GND

注意：加TVS保护，电机反电势可能超ADC范围
```

## 7. PCB布局建议

| 电路模块 | 布局要点 |
|----------|----------|
| 运放电路 | 退耦电容100nF紧贴电源引脚 |
| 滤波器 | 输入/输出走线避免耦合 |
| 比较器 | 参考电压用独立分压，加退耦 |
| ADC驱动 | 模拟地与数字地单点连接 |
| 电桥 | 等长对称走线，远离干扰源 |

**地线处理：**
```
                    ┌─── 模拟地(AGND)
                    │
传感器 → 调理电路 ──┤──→ 单点连接 ──→ 数字地(DGND)
                    │                        │
                    └─── 电源地(PGND)    MCU/ADC
```
