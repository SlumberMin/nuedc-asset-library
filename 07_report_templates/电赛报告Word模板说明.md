## 一、模板概述

| 文件名 | 用途 | 页数 |
|--------|------|------|
| 电赛报告模板.docx | 主报告模板 | ~20页 |

## 二、快速开始

### 第一步：打开模板
1. 复制 `电赛报告模板.docx` 到你的工作目录
2. **不要直接在原模板上编辑**（保留模板副本）

### 第二步：替换封面信息
在封面页替换以下内容：
```
【题目编号】 → 替换为实际题目编号（如：A题）
【题目名称】 → 替换为完整题目名称
【学校名称】 → 你的学校全称
【队员1姓名】【队员2姓名】【队员3姓名】 → 三位队员姓名
【指导教师】 → 指导老师姓名
【日期】 → 提交日期（YYYY年MM月DD日）
```

### 第三步：编辑正文
按模板中的占位文字逐节填写。

## 四、排版规范

### 4.1 页面设置
```
纸张：A4 (210mm × 297mm)
页边距：上2.5cm，下2.5cm，左3cm，右2.5cm
行距：1.5倍行距
段前段后：0行
```

### 4.2 图表规范

**图片**：
- 宽度不超过页面宽度（约15cm）
- 图片居中对齐
- 图片下方标注图号和图题
- 格式：`图1 系统总体框图`

```word
插入图片 → 右键 → 大小和位置 → 宽度设为14cm → 确定
然后在图片下方添加题注：引用 → 插入题注 → 标签选"图"
```

**表格**：
- 表格居中对齐
- 表格上方标注表号和表题
- 格式：`表1 测试数据记录`
- 表头加粗，内容宋体小四

```word
插入表格 → 选中表格 → 表格属性 → 对齐方式：居中
在表格上方添加题注：引用 → 插入题注 → 标签选"表"
```

**公式**：
- 使用公式编辑器（推荐MathType或Word自带公式）
- 公式居中，编号右对齐
- 格式：`E = mc²  (1)`

### 4.3 参考文献格式

采用GB/T 7714标准：

```
[1] 张三, 李四. 嵌入式系统设计[M]. 北京: 电子工业出版社, 2020.
[2] 王五. 基于STM32的电机控制系统设计[J]. 电子技术应用, 2021, 47(3): 45-48.
[3] Texas Instruments. LM358 Datasheet[EB/OL]. https://www.ti.com, 2020.
```

**文献类型标识**：
| 类型 | 标识 | 说明 |
|------|------|------|
| 专著 | [M] | 书籍、教材 |
| 期刊 | [J] | 学术期刊论文 |
| 学位论文 | [D] | 硕士/博士论文 |
| 会议论文 | [C] | 学术会议论文 |
| 标准 | [S] | 技术标准 |
| 电子文献 | [EB/OL] | 网络资源 |

## 五、图表自动编号

### 5.1 设置题注

```word
操作步骤：
1. 选中图片或表格
2. 引用 → 插入题注
3. 标签选择"图"或"表"
4. 位置选择"所选项目下方"（图）或"所选项目上方"（表）
5. 点击确定
```

### 5.2 交叉引用

```word
在正文中引用图表：
1. 输入"如图"或"见表"
2. 引用 → 交叉引用
3. 引用类型选择"图"或"表"
4. 选择对应的题注
5. 引用内容选择"只有标签和编号"
6. 点击插入
```

**注意**：当增删图表后，全选文档（Ctrl+A）→ 右键 → 更新域，即可自动更新所有编号。

## 八、常见排版问题

### 8.1 图片位置乱跑
**解决**：
```word
右键图片 → 大小和位置 → 文字环绕 → 选择"嵌入型"
```

### 8.2 表格跨页断裂
**解决**：
```word
选中表格 → 表格属性 → 行 → 勾选"允许跨页断行"
选中表头行 → 表格属性 → 行 → 勾选"在各页顶端以标题行形式重复出现"
```

### 8.3 公式编号对不齐
**解决**：
```word
使用制表位：
1. 标尺上设置两个制表位：居中（页面中间）和右对齐（页面右边距）
2. 公式前按Tab，公式和编号之间再按Tab
```

### 8.4 文档太大
**解决**：
- 压缩图片：选中图片 → 格式 → 压缩图片 → 分辨率选150ppi
- 删除嵌入的字体：文件 → 选项 → 保存 → 取消"嵌入字体"

## 十、模板修改指南

如需自定义模板：

### 修改样式
```word
开始 → 样式 → 右键目标样式 → 修改
可修改字体、字号、行距、段前段后等
```

### 添加页眉横线
```word
双击页眉 → 选中文字 → 开始 → 边框 → 下边框
```

```word
页面布局 → 页面设置 → 版式 → 勾选"奇偶页不同"
```
