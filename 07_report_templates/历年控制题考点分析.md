# 2009-2025年全国电赛控制题考点分析

## 一、考点分类体系

控制题核心考点可归纳为以下六大模块：
| 模块 | 考点内容 | 出现频次 |
|------|---------|---------|
| 电机驱动 | 直流电机、步进电机、舵机、BLDC控制 | ★★★★★ |
| 传感器应用 | 编码器、陀螺仪/IMU、超声波、红外、电磁 | ★★★★★ |
| 运动控制 | PID调参、路径规划、姿态解算 | ★★★★★ |
| 机械结构 | 平衡机构、悬挂、传动 | ★★★★ |
| 电源管理 | 电池选型、稳压、功耗优化 | ★★★ |
| 通信协议 | 串口、I2C、SPI、无线 | ★★★ |

## 二、历年题目详细分析

### 2009年
**题目：声音定位系统（E题）**
- 核心考点：信号采集、时延估计、几何定位算法
- 难度：★★★☆
- 关键技术：麦克风阵列、互相关算法、LCD显示
- 点评：偏信号处理，非典型控制题，但涉及传感器融合思想

### 2011年
**题目：帆板控制系统（F题）**
- 核心考点：角度检测、PID闭环控制、舵机驱动
- 难度：★★★☆
- 关键技术：MPU6050/角度传感器、PWM控制、PID参数整定
- 点评：经典PID控制入门题，角度测量精度是区分度

### 2012年
**题目：激光打靶系统（C题）**
- 核心考点：步进电机精确控制、坐标变换、闭环定位
- 难度：★★★★
- 关键技术：步进电机细分驱动、激光定位、摄像头反馈
- 点评：需要精确的坐标映射和校准算法

### 2013年
**题目：旋转倒立摆（F题）**
- 核心考点：倒立摆建模、平衡控制、角度传感
- 难度：★★★★★
- 关键技术：MPU6050姿态解算、PID/极点配置、状态空间建模
- 点评：经典控制理论综合应用，建模能力是关键

### 2014年
**题目：四旋翼自主飞行器（C题）**
- 核心考点：多旋翼姿态控制、传感器融合、自主导航
- 难度：★★★★★
- 关键技术：MPU6050+气压计融合、串级PID、路径规划
- 点评：难度极高的综合题，对PID调参要求极高

### 2015年
**题目：风力摆控制系统（C题）**
- 核心考点：角度/角速度检测、电机驱动、轨迹控制
- 难度：★★★★
- 关键技术：编码器/IMU、直流电机调速、画圆/定角度控制
- 点评：重点考察闭环控制的工程实现能力

### 2016年
**题目：自动泊车系统（C题）**
- 核心考点：路径规划、运动控制、避障
- 难度：★★★★
- 关键技术：超声波/红外测距、差速转向、状态机设计
- 点评：需要良好的状态机设计和路径规划能力

### 2017年
**题目：自适应滤波器（C题）**——模拟信号题
**题目：电磁炮（D题）**
- 核心考点：电磁发射、角度计算、瞄准控制
- 难度：★★★☆
- 关键技术：充电电路、弹道计算、舵机瞄准
- 点评：偏电磁与控制结合，需要电路与算法并重

### 2018年
**题目：滚动小球控制系统（C题）**
- 核心考点：平台倾斜控制、位置检测、双轴PID
- 难度：★★★★
- 关键技术：触摸屏/摄像头位置检测、舵机驱动平板、双轴PID
- 点评：需要精确的位置反馈和稳定的双轴控制

### 2019年
**题目：模拟电磁曲射炮（C题）**
- 核心考点：电磁加速、弹道计算、目标定位
- 难度：★★★☆
- 关键技术：电磁线圈驱动、角度计算、距离测量
- 点评：需要理解电磁发射原理和基本弹道学

### 2020年
**题目：坡道行驶电动小车（D题）**
- 核心考点：坡道行驶、速度控制、角度检测
- 难度：★★★☆
- 关键技术：电机驱动、MPU6050检测坡度、自适应速度控制
- 点评：需要根据不同坡度自适应调整控制策略

### 2021年
**题目：送药小车（F题）**
- 核心考点：循迹导航、避障、目标识别
- 难度：★★★★
- 关键技术：OpenMV/摄像头循迹、超声波避障、坐标导航
- 点评：综合应用题，需要路径规划和多传感器融合

### 2022年
**题目：小车跟随行驶系统（D题）**
- 核心考点：距离保持、速度跟随、无线通信
- 难度：★★★★
- 关键技术：超声波/红外测距、蓝牙/NRF通信、PID速度控制
- 点评：需要稳定可靠的通信和精确的距离控制

### 2023年
**题目：信号失真度测量（C题）**——测量题
**题目：空地协同智能消防系统（F题）**
- 核心考点：多机协同、路径规划、火源识别
- 难度：★★★★★
- 关键技术：视觉识别、无线通信、自主导航
- 点评：题目综合性极强，涉及多个子系统的协同

### 2024年
**题目：单相有源电力滤波器（C题）**——电源题
**题目：自动行驶小车（F题）**
- 核心考点：自主循迹、路径规划、障碍识别
- 难度：★★★★
- 关键技术：摄像头/OpenMV、电机驱动、路径优化
- 点评：视觉循迹是核心，图像处理算法是区分度

### 2025年
**题目：智能仓储物流系统 / 自主巡航小车（预测方向）**
- 核心考点：多传感器融合、路径规划、边缘AI推理
- 难度预测：★★★★★
- 关键技术：RK3588S NPU加速、视觉SLAM、多机协同调度
- 预测考点：
  - 视觉目标检测（YOLOv5n/v8n on NPU）
  - 自适应PID/LADRC控制
  - RTK/视觉定位融合
  - 无线通信协议（WiFi/BLE5.0）
  - 状态机+路径规划

### 2024年控制题技术要求深度分析
**题目：自动行驶小车（F题）**
- 基本功能：
  - 沿指定路线自动行驶（循迹精度要求）
  - 车速控制（PID速度环）
  - 避障功能（超声波/红外）
- 发挥功能：
  - 复杂路径规划
  - 动态避障
  - 目标识别与交互
- **评分重点**：
  - 循迹精度（偏差<2cm为优秀）
  - 响应速度（弯道处理时间）
  - 稳定性（连续运行不脱线）
  - 报告中的理论分析深度

### 2023年控制题技术要求深度分析
**题目：空地协同智能消防系统（F题）**
- 基本功能：
  - 无人机自主起飞/降落
  - 地面车自主导航
  - 火源识别与定位
- 发挥功能：
  - 空地协同控制
  - 多目标处理
  - 实时通信
- **评分重点**：
  - 系统协同能力（空地配合流畅度）
  - 火源识别准确率
  - 控制精度（着陆精度、定位精度）
  - 报告中的系统架构设计

## 四、控制题评分重点与常见失分点

### 4.1 评分重点TOP10
1. **PID调参质量** — 稳定性、响应速度、超调量
2. **硬件可靠性** — 长时间运行不故障
3. **传感器精度** — 数据准确、噪声抑制
4. **控制精度** — 达到题目要求的精度指标
5. **报告理论深度** — 有数学推导，不是"试出来的"
6. **系统架构** — 模块化、可维护、可扩展
7. **抗干扰能力** — 扰动抑制、参数鲁棒性
8. **创新性** — 使用新算法/新平台/新方法
9. **演示稳定性** — 一次成功，不手忙脚乱
10. **答辩表现** — 对作品了如指掌

### 4.2 常见失分点
| 失分原因 | 影响程度 | 避免策略 |
|---------|---------|---------|
| PID参数无理论依据 | ★★★★★ | 提前仿真整定,报告中写出推导过程 |
| 报告抄袭模板痕迹重 | ★★★★★ | 报告用自己的语言,结合实际数据 |
| 硬件不稳定/虚焊 | ★★★★ | 双重检查焊接,加保护电路 |
| 测试数据造假 | ★★★★ | 使用真实数据,不达标的诚实写原因 |
| 忽视发挥部分 | ★★★★ | 优先实现容易的发挥项,即使部分实现 |
| 演示时手忙脚乱 | ★★★ | 提前排练,准备一键启动方案 |
| 答辩一问三不知 | ★★★ | 提前准备常见问题答案 |
| 软件架构混乱 | ★★★ | 使用状态机+模块化架构 |
| 电源设计不合理 | ★★★ | 充分滤波,预留功率裕量 |
| 通信不稳定 | ★★ | 使用成熟协议,加校验和重传 |

### 4.3 失分点避免策略详解

**PID调参策略**：
```
1. 理论计算初始值:
   - 临界比例法: 先找临界增益Ku和临界周期Tu
   - Z-N公式: Kp=0.6Ku, Ti=0.5Tu, Td=0.125Tu
2. 实验微调:
   - 先P后I再D
   - 每次只调一个参数
   - 记录每次调整的效果
3. 高级方法:
   - 串级PID(位置环+速度环)
   - 前馈补偿
   - LADRC带宽整定法
```

**报告写作策略**：
```
1. 方案论证: 列出2-3种方案,用表格对比
2. 理论分析: 公式推导+参数计算过程
3. 测试数据: 真实数据+波形图+分析结论
4. 创新点: 强调使用的新技术/新方法
5. 不足分析: 诚实写不足+改进方向
```

**演示策略**：
```
1. 提前排练: 至少完整演示3次
2. 一键启动: 设计自动运行模式
3. 顺序演示: 先基本功能后发挥功能
4. 说清预期: 演示前说明"接下来演示XX"
5. 失败预案: 准备备用方案
```

## 五、备赛建议

### 基础年份（★★★）：
- 重点掌握PID和电机驱动即可
- 准备好传感器套件和电机驱动模块
- 报告模板提前准备

### 进阶年份（★★★★）：
- 需要加入传感器融合和路径规划
- 掌握串级PID和前馈补偿
- 视觉处理能力（OpenMV/OpenCV）

### 高难度年份（★★★★★）：
- 必须掌握视觉处理和多机协同
- 了解LADRC/滑模/MPC等高级控制
- RK3588S/NPU加速视觉推理
- 边缘AI部署能力

### 所有年份都需要：
- **稳定的硬件平台** + **扎实的PID功底** + **良好的报告写作**
- 模块化代码库 + 成熟的驱动 + 仿真验证
- 团队协作默契 + 时间管理能力
