### 2.2 控制算法设计

#### 2.2.1 PID 控制器设计

**增量式 PID：**

$$\Delta u(k) = K_p[e(k)-e(k-1)] + K_i e(k) + K_d[e(k)-2e(k-1)+e(k-2)]$$

**位置式 PID：**

$$u(k) = K_p e(k) + K_i \sum_{j=0}^{k} e(j) + K_d[e(k)-e(k-1)]$$

#### 2.2.2 选择依据

| 算法 | 适用场景 | 本题是否采用 | 理由 |
|------|---------|------------|------|
| P 控制 | 允许稳态误差 | | |

| PID 控制 | 快速响应，无稳态误差 | | |

### 4.3 关键代码说明

```c
// PID 控制核心代码（关键片段）
typedef struct {
    float Kp, Ki, Kd;
    float error, last_error, prev_error;
    float output;
    float integral;
    float integral_limit;  // 积分限幅
} PID_t;

float PID_Calculate(PID_t *pid, float target, float actual) {
    pid->error = target - actual;

    // 积分分离：误差大于阈值时暂停积分
    if (fabs(pid->error) < INTEGRAL_THRESHOLD) {
        pid->integral += pid->error;
        // 积分限幅
        if (pid->integral > pid->integral_limit) pid->integral = pid->integral_limit;
        if (pid->integral < -pid->integral_limit) pid->integral = -pid->integral_limit;
    }

    // 增量式 PID
    pid->output = pid->Kp * (pid->error - pid->last_error)
                + pid->Ki * pid->error
                + pid->Kd * (pid->error - 2*pid->last_error + pid->prev_error);

    pid->prev_error = pid->last_error;
    pid->last_error = pid->error;

    return pid->output;
}
```
