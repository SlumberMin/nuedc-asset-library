# ⚡ 电赛报告写作速查卡（打印用）

---

## 一、排版规范

| 项目 | 规范 |
|------|------|
| 纸张 | A4 |
| 页边距 | 上下2.5cm，左右2.5cm |
| 正文字体 | 宋体小四（12pt），行距1.5倍 |
| 标题字体 | 一级黑体三号，二级黑体四号，三级黑体小四 |
| 页码 | 底部居中 |
| 图表 | 居中，图在下方标注"图1 xxx"，表在上方标注"表1 xxx" |
| 公式 | 居中，右对齐编号"(1)" |
| 代码 | 等宽字体，灰色背景框 |
| 页数 | 15-20页（不含附录） |

## 二、报告结构模板

```
1. 摘要（1页）★最重要
   - 方案概述（2-3句）
   - 核心技术（1-2句）
   - 测试结果（关键指标数据）
   
2. 方案论证与选择（2页）
   - 方案一/二/三比较（表格）
   - 选择理由
   - 系统总体框图
   
3. 理论分析与计算（2页）
   - 关键公式推导
   - 参数计算过程
   - 电路参数确定依据
   
4. 电路与程序设计（4-6页）
   - 4.1 电源电路
   - 4.2 信号采集电路
   - 4.3 核心处理电路
   - 4.4 驱动/输出电路
   - 4.5 程序总体流程图
   - 4.6 关键算法代码
   
5. 测试方案与结果（3-4页）
   - 测试仪器列表
   - 测试方法描述
   - 数据表格 + 分析
   - 波形截图
   
6. 总结与展望（1页）
```

## 三、常用公式速查

### 基本电路
```
欧姆定律: V = I·R
分压: Vout = Vin · R2/(R1+R2)
RC滤波: fc = 1/(2π·RC)
LC谐振: f = 1/(2π·√(LC))
运放增益: Av = -Rf/R1 (反相), Av = 1+Rf/R1 (同相)
```

### 功率与效率
```
功率: P = V·I = I²·R = V²/R
效率: η = Pout/Pin × 100%
DC-DC效率: η ≈ Vout·Iout / (Vin·Iin)
```

### 信号处理
```
采样定理: fs ≥ 2·fmax
ADC分辨率: ΔV = Vref / 2^n
信噪比: SNR = 20·log10(Vs/Vn) dB
```

### 控制相关
```
PID: u(t) = Kp·e(t) + Ki·∫e·dt + Kd·de/dt
一阶惯性: G(s) = K/(Ts+1)
二阶振荡: G(s) = ωn²/(s²+2ζωn·s+ωn²)
```

### 电机
```
转速: n = 60·f/p (rpm)
反电动势: E = Ke·ω
转矩: T = Kt·I
```

## 五、加分写作技巧

1. **摘要放关键数据**：不要写"较好效果"，写"误差<1%，响应时间<0.5s"
2. **框图要原创**：不要直接截图别人的，自己用Visio/PPT画
3. **测试要完整**：每个指标都要有数据，不能只写"基本满足"
4. **代码要注释**：关键算法加注释，不放全部代码（放附录）
5. **图表要编号**：全文连续编号，正文中必须引用
6. **参考文献**：至少列3-5篇

## 六、常见扣分项

- ❌ 摘要太空，没有数据支撑
- ❌ 没有方案对比，直接写最终方案
- ❌ 电路图没有参数标注
- ❌ 测试数据太少或没有分析
- ❌ 代码没有注释
- ❌ 排版混乱，图表不编号
