# 全国大学生电子设计竞赛备考路线图
## 从零基础到国奖的完整学习路径

---

## 阶段一：基础入门（赛前6-12个月）

### 1.1 电路基础（2-4周）
- [ ] 欧姆定律、基尔霍夫定律
- [ ] RC/RL/RLC电路分析
- [ ] 运放电路（同相/反相放大、比较器、滤波器）
- [ ] 功率器件（MOS管、三极管驱动电路）
- **推荐资源**：《模拟电子技术基础》童诗白

### 1.2 数字电路（2周）
- [ ] 逻辑门、组合逻辑、时序逻辑
- [ ] 555定时器应用
- [ ] 74系列常用芯片
- [ ] FPGA/CPLD基础（可选）

### 1.3 C语言与单片机入门（4-6周）
- [ ] C语言基础（指针、结构体、中断）
- [ ] STM32开发环境搭建（Keil/CubeIDE）
- [ ] GPIO、定时器、中断
- [ ] 串口通信（UART）
- [ ] ADC/DAC
- **推荐资源**：正点原子/野火STM32教程

### 1.4 焊接与调试基础（1-2周）
- [ ] 手工焊接（直插/SOP/TSSOP封装）
- [ ] 万用表、示波器使用
- [ ] 基本故障排查方法

## 阶段二：核心技能（赛前3-6个月）

### 2.1 电机驱动（3-4周）
- [ ] 直流电机：PWM调速、H桥驱动（L298N/TB6612）
- [ ] 步进电机：细分驱动、加减速控制
- [ ] 舵机：PWM控制、角度映射
- [ ] 无刷电机（进阶）：ESC电调、BLDC原理
- **实践项目**：做一个能正反转调速的小车

### 2.2 传感器应用（3-4周）
- [ ] 编码器：正交解码、速度测量
- [ ] MPU6050/IMU：姿态解算（互补滤波/卡尔曼）
- [ ] 超声波：测距原理、多探头防串扰
- [ ] 红外：循迹、避障
- [ ] 电磁传感器：电感/电容检测
- **实践项目**：做一个能测距+测角度的数据采集系统

### 2.3 PID控制（4-6周）⭐最重要
- [ ] PID基本原理（P/I/D各项作用）
- [ ] 位置式PID与增量式PID
- [ ] 手动调参法（先P后I再D）
- [ ] 串级PID（角度环+速度环）
- [ ] 抗积分饱和
- [ ] PID调参实战（至少2个项目）
- **实践项目**：
  - 直流电机速度闭环
  - 平衡小车/倒立摆（终极练习）

### 2.4 通信协议（2周）
- [ ] I2C（传感器通信）
- [ ] SPI（高速通信）
- [ ] 串口协议设计（自定义帧格式）
- [ ] 蓝牙/WiFi/NRF24L01无线通信

## 阶段六：2025年最新技术趋势（备赛重点）

### 6.1 新兴硬件平台
| 平台 | 特点 | 适用场景 |
|------|------|---------|
| **RK3588S** | 八核A76+A55, 6TOPS NPU | 高级视觉处理、边缘AI |
| **MSPM0G3507** | TI ARM M0+, 12bit ADC | 低功耗控制、TI杯指定平台 |
| **OrangePi5** | RK3588S SBC, Linux | 视觉处理、上位机 |
| **STM32H7** | 480MHz M7, 高精度ADC | 高性能控制、信号处理 |
| **ESP32-S3** | 双核240MHz, AI加速 | 无线通信、轻量AI |

### 6.2 视觉AI趋势
- **OpenCV + NPU加速**: 利用RK3588S的NPU进行目标检测(YOLOv5n/v8n)
- **MobileNetV3分类**: 轻量级图像分类用于颜色/形状识别
- **PicoDet**: 嵌入式端高效目标检测
- **颜色空间自适应**: HSV/YCrCb自适应阈值分割
- **深度学习部署**: 模型量化(INT8) + RKNN推理

### 6.3 控制算法演进
- **LADRC(线性自抗扰)**: 替代传统PID, 参数整定更简单(带宽法)
- **超螺旋滑模**: 连续控制信号, 无需边界层, 天然抗抖振
- **自适应卡尔曼**: 实时估计噪声统计量, 适应环境变化
- **MPC(模型预测控制)**: 处理约束优化问题, 适合路径规划
- **模糊PID + 神经网络PID**: 智能参数自整定

### 6.4 2025年考点预测
- 人机协作机器人(视觉+控制+通信)
- 智能仓储/物流系统(多传感器融合+路径规划)
- 自主巡航系统(RTK定位+视觉导航)
- 空地协同(多机通信+协同控制)
- 边缘AI应用(端侧推理+实时决策)

## 阶段七：分阶段训练计划详解

### 7.1 基础阶段（12-6个月前）
| 周次 | 训练内容 | 产出物 |
|------|---------|--------|
| 1-2周 | C语言指针/结构体/中断 | 基础编程练习 |
| 3-4周 | STM32 GPIO/定时器/中断 | LED/按键/定时器实验 |
| 5-6周 | ADC/DAC/PWM | 模拟信号采集实验 |
| 7-8周 | UART/I2C/SPI通信 | 多传感器通信实验 |
| 9-10周 | 焊接练习(直插+SOP) | 焊接评估板 |
| 11-12周 | 万用表/示波器使用 | 基础测量能力 |

### 7.2 进阶阶段（6-3个月前）
| 周次 | 训练内容 | 产出物 |
|------|---------|--------|
| 1-2周 | 直流电机PWM调速+编码器 | 速度闭环模块 |
| 3-4周 | 步进电机细分驱动+加减速 | 精确定位模块 |
| 5-6周 | 舵机PWM控制+角度映射 | 舵机控制模块 |
| 7-8周 | MPU6050/IMU姿态解算 | 姿态检测模块 |
| 9-10周 | PID手动调参(速度环/角度环) | PID调参经验 |
| 11-12周 | 串级PID+抗饱和 | 串级控制模块 |

### 7.3 实战阶段（3-1个月前）
| 周次 | 训练内容 | 产出物 |
