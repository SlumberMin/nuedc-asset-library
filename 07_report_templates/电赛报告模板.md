## 二、理论分析与计算

### 2.1 关键参数计算

**【必须包含的计算内容】**

#### 2.1.1 增益/放大倍数计算

```
题目要求：增益范围 0~60dB

系统增益分配：
  前置固定增益：G1 = 40dB (×100)
  可调增益：G2 = -20dB ~ +30dB
  总增益：G = G1 + G2 = 20dB ~ 70dB（覆盖0~60dB要求）

增益验证：
  最小增益：20dB = ×10 ✓（>0dB）
  最大增益：70dB = ×3162 ✓（满足60dB要求）
```

#### 2.1.2 带宽计算

```
题目要求：频率范围 100kHz ~ 50MHz

运放选型验证：
  OPA847 GBW = 3.9GHz
  闭环增益 G1 = 100 (40dB)
  闭环带宽 BW = GBW/G1 = 3.9GHz/100 = 39MHz
  
  实际带宽需求：>50MHz（留余量）
  → 需要调整增益分配或选择更高GBW运放
```

#### 2.1.3 阻抗匹配计算

```
输入阻抗匹配：Rin = 50Ω（射频标准）
输出阻抗匹配：Rout = 50Ω
终端匹配电阻：RT = 50Ω

功率计算：
  Pout = V²rms/R = (5/√2)²/50 = 250mW
  → 需要输出级驱动能力 > 250mW
```

#### 2.1.4 电源设计计算

```
系统功耗估算：
  MCU: ~100mA @ 3.3V = 0.33W
  模拟电路: ~200mA @ ±5V = 2W
  功率输出: ~500mA @ ±12V = 12W
  总功耗: ~15W

电源选型：12V/2A 开关电源 + LDO稳压
```

#### 3.2.2 关键算法代码（含注释）

```c
/**
 * @brief 增量式PID控制器
 * @param pid: PID结构体指针
 * @param measured: 当前测量值
 * @return: 控制量增量
 */
float PID_Incremental(PID_TypeDef *pid, float measured)
{
    float error = pid->setpoint - measured;
    float delta = pid->Kp * (error - pid->last_error)
                + pid->Ki * error
                + pid->Kd * (error - 2*pid->last_error + pid->prev_error);
    
    pid->prev_error = pid->last_error;
    pid->last_error = error;
    
    return delta;
}
```
