# 电赛高分报告写作指南 V2

> 基于国奖报告反向分析，覆盖结构/排版/数据/图表全要素

---

## 一、报告评分标准

### 1.1 分值分布（20分）

| 评分项 | 分值 | 高分特征 |
|--------|------|----------|
| 方案论证 | 5分 | 2种方案对比，有理有据 |
| 理论分析 | 3分 | 公式推导完整，计算正确 |
| 电路设计 | 4分 | 原理图清晰，参数标注完整 |
| 程序设计 | 3分 | 流程图规范，关键代码注释 |
| 测试数据 | 3分 | 数据完整，误差分析到位 |
| 总结分析 | 2分 | 有深度，有改进方向 |

### 1.2 国奖报告 vs 普通报告

```
普通报告：
  "采用STM32控制，实现了题目要求"
  → 得分：12-14分

国奖报告：
  "综合比较三种方案的精度、成本、开发周期，选择方案二，
   因为其在精度和实现难度之间取得了最佳平衡..."
  → 得分：17-20分
```

## 二、报告结构模板

### 2.1 完整结构

```
1. 摘要（200-300字）
   ├── 系统概述（做什么）
   ├── 核心方案（怎么做）
   ├── 主要指标（做到什么程度）
   └── 关键词（3-5个）

2. 方案论证（1-2页）
   ├── 2.1 方案一：XX方案
   │   ├── 原理说明
   │   ├── 优点
   │   └── 缺点
   ├── 2.2 方案二：YY方案
   │   ├── 原理说明
   │   ├── 优点
   │   └── 缺点
   ├── 2.3 方案比较（表格对比）
   └── 2.4 最终选择及理由

3. 理论分析与计算（2-3页）
   ├── 3.1 系统框图
   ├── 3.2 核心公式推导
   ├── 3.3 关键参数计算
   └── 3.4 误差分析

4. 电路设计（3-5页）
   ├── 4.1 系统总体框图
   ├── 4.2 电源电路
   ├── 4.3 信号调理电路
   ├── 4.4 控制电路
   ├── 4.5 驱动电路
   └── 4.6 显示与人机交互

5. 程序设计（2-3页）
   ├── 5.1 软件架构
   ├── 5.2 主程序流程图
   ├── 5.3 关键子程序流程图
   └── 5.4 核心算法说明

6. 测试与结果（2-3页）
   ├── 6.1 测试条件
   ├── 6.2 测试数据表格
   ├── 6.3 测试波形图
   └── 6.4 误差分析

7. 总结与展望（0.5-1页）
   ├── 完成情况总结
   ├── 创新点说明
   └── 改进方向

参考文献（5-10篇）
```

## 三、各章节写作技巧

### 3.1 摘要写作

#### 高分模板

```
本系统以[主控芯片]为核心，采用[核心方案]实现了[题目要求]。
系统主要由[模块1]、[模块2]、[模块3]组成。
[模块1]采用[方案]实现了[功能]；
[模块2]采用[方案]实现了[功能]；
[模块3]采用[方案]实现了[功能]。
经测试，系统[指标1]达到[数值]，[指标2]达到[数值]，
满足题目基本要求和发挥部分要求。

关键词：[关键词1]；[关键词2]；[关键词3]
```

#### 示例

```
本系统以STM32F407为核心，采用电压电流双闭环控制实现了
高效率数控直流电源。系统主要由Buck变换器、电压采样电路、
电流采样电路和人机交互模块组成。Buck变换器采用同步整流
拓扑实现了92%的转换效率；电压采样采用差分放大电路配合
16位ADC实现了0.1%的采样精度；控制算法采用增量式PID实现
了快速稳定的电压调节。经测试，系统输出电压精度优于0.5%，
负载调整率优于0.3%，纹波电压小于20mV，满足题目全部要求。

关键词：Buck变换器；同步整流；PID控制；数控电源
```

### 3.2 方案论证写作

#### 对比表格模板

```
| 对比项 | 方案一 | 方案二 | 方案三 |
|--------|--------|--------|--------|
| 核心器件 | XX | YY | ZZ |
| 精度 | ±1% | ±0.5% | ±0.1% |
| 成本 | 低 | 中 | 高 |
| 开发难度 | 简单 | 中等 | 复杂 |
| 稳定性 | 一般 | 好 | 好 |
| 综合评价 | ★★☆ | ★★★ | ★★★ |

综合考虑精度、成本和开发周期，选择方案二。
```

#### 常见方案对比（按题目类型）

**电源类**
```
方案一：线性稳压（LM317/LDO）
  优点：纹波小、响应快
  缺点：效率低、发热大

方案二：开关稳压（Buck/Boost）
  优点：效率高、发热小
  缺点：纹波较大、设计复杂

方案三：开关+线性级联
  优点：效率和纹波兼顾
  缺点：成本高、体积大
```

**信号类**
```
方案一：DDS专用芯片（AD9833/AD9850）
  优点：频率精度高、切换快
  缺点：成本高、幅度控制有限

方案二：MCU+DAC软件合成
  优点：灵活、成本低
  缺点：频率上限受MCU主频限制

方案三：FPGA+DAC
  优点：速度高、并行处理
  缺点：开发难度大
```

### 3.3 理论分析写作

#### 公式推导规范

```
错误写法：
  "增益为10倍"

正确写法：
  "根据同相放大器公式：
   Av = 1 + R2/R1 = 1 + 10kΩ/1.11kΩ ≈ 10
   其中R2 = 10kΩ，R1 = 1.11kΩ（取标称值1.1kΩ）
   实际增益 Av = 1 + 10k/1.1k ≈ 10.09"
```

#### 误差分析模板

```
系统误差主要来源：
1. ADC量化误差：ΔV = Vref/2^n = 3.3V/65536 ≈ 0.05mV
2. 电阻精度误差：使用1%精度电阻，最大增益误差±2%
3. 温漂误差：在10℃温度变化下，参考电压漂移约±0.1%
4. 运放失调电压：典型值±0.5mV，折算到输出约±5mV

综合误差 = √(0.05² + 2² + 0.1² + 0.5²) ≈ 2.06%
```

```

建议标注：

```


### 3.5 程序设计写作

#### 流程图绘制规范

```
必须包含：
□ 开始/结束框（圆角矩形）
□ 处理框（矩形）
□ 判断框（菱形）
□ 箭头表示流向
□ 框内文字说明

规范要求：
□ 流程图一页内完成（不跨页）
□ 箭头方向清晰（从上到下、从左到右）
□ 判断框两个出口都标注（是/否 或 Y/N）
□ 关键步骤加注释
```

#### 核心代码展示规范

```c
// 展示核心算法，非完整代码
// 添加详细注释说明算法逻辑

// 增量式PID控制算法
float PID_Incremental(PID_t *pid, float target, float actual)
{
    float error = target - actual;           // 计算误差
    float increment = pid->Kp * (error - pid->last_error)  // 比例项
                    + pid->Ki * error                         // 积分项
                    + pid->Kd * (error - 2*pid->last_error + pid->prev_error); // 微分项
    
    pid->prev_error = pid->last_error;       // 更新历史误差
    pid->last_error = error;
    
    return increment;                        // 返回增量
}
```


## 六、常见报告问题

### 6.1 方面一：方案论证

```
❌ 错误："我们采用STM32作为主控"
✅ 正确："方案一采用51单片机，优点是...缺点是...
         方案二采用STM32，优点是...缺点是...
         综合考虑，选择方案二"

❌ 错误：只写了1种方案
✅ 正确：至少对比2种方案，用表格总结
```

### 6.2 方面二：测试数据

```
❌ 错误："输出电压精度满足要求"
✅ 正确："如表X所示，在1V-5V范围内，输出电压最大
         相对误差为0.3%（设定值3V时），优于题目要求的1%"

❌ 错误：只测了3个点
✅ 正确：全范围内均匀取5-10个测试点
```

### 6.3 方面三：图表质量

```
❌ 错误：模糊的手机拍照
✅ 正确：清晰的示波器截图，带标注

❌ 错误：图片过大占据整页
✅ 正确：图片适当缩放，图题在下方
```
