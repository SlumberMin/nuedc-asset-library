# 电赛答辩PPT模板

> 答辩PPT制作指南，覆盖结构/配色/动画/演讲技巧

---

## 一、答辩概述

### 1.1 答辩形式

```
时间：5-10分钟展示 + 3-5分钟提问
评委：3-5位专家
内容：系统方案、核心设计、测试结果
目标：清晰展示作品，回答专家提问
```

### 1.2 评分权重

```
方案合理性：30%
技术深度：30%
展示效果：20%
回答问题：20%
```

## 二、PPT结构模板

### 2.1 标准结构（10-15页）

```
第1页：封面
  ├── 题目名称
  ├── 学校名称
  ├── 队伍成员
  ├── 指导教师
  └── 日期

第2页：目录
  ├── 方案论证
  ├── 系统设计
  ├── 硬件设计
  ├── 软件设计
  ├── 测试结果
  └── 总结

第3-4页：方案论证
  ├── 题目要求回顾
  ├── 方案对比（表格）
  └── 最终选择及理由

第5页：系统框图
  └── 整体架构图（一页展示清楚）

第6-8页：硬件设计
  ├── 核心电路原理
  ├── 关键参数计算
  └── 实物照片

第9-10页：软件设计
  ├── 软件架构
  ├── 核心算法流程图
  └── 关键代码片段

第11-13页：测试结果
  ├── 测试数据表格
  ├── 测试波形
  └── 指标达成情况

第14页：总结与创新
  ├── 完成情况
  ├── 创新点
  └── 改进方向

第15页：致谢
```

### 2.2 精简结构（5-8页，适合5分钟答辩）

```
第1页：封面
第2页：系统框图 + 方案选择
第3页：核心硬件设计
第4页：核心算法
第5页：测试结果
第6页：总结与创新
```

## 四、配色方案

### 4.1 推荐配色

#### 方案一：科技蓝（推荐）

```
主色：#2B579A（深蓝）
辅色：#5B9BD5（浅蓝）
强调：#FF6600（橙色）
背景：#FFFFFF（白色）
文字：#333333（深灰）
```

#### 方案二：专业灰

```
主色：#333333（深灰）
辅色：#666666（中灰）
强调：#0066CC（蓝色）
背景：#F5F5F5（浅灰）
文字：#333333（深灰）
```

#### 方案三：清新绿

```
主色：#2E7D32（深绿）
辅色：#66BB6A（浅绿）
强调：#FF7043（橙红）
背景：#FFFFFF（白色）
文字：#333333（深灰）
```

### 4.2 配色原则

```
1. 全PPT不超过3种主色
2. 背景色用白色或浅色
3. 文字与背景对比度要高
4. 强调色用于关键数据和结论
5. 避免使用纯红纯绿（色盲友好）
```

## 五、字体规范

### 5.1 字体选择

```
标题：微软雅黑/黑体
正文：微软雅黑/宋体
代码：Consolas/Courier New
数字：Times New Roman/Arial
```

### 5.2 字号规范

```
封面标题：28-36pt
页面标题：24-28pt
正文：18-22pt
注释：14-16pt
图表标注：12-14pt

原则：最后一排观众能看清最小字
```

## 六、动画与切换

### 6.1 推荐动画

```
进入动画：
  - 淡出（最安全）
  - 飞入（适合流程图逐步展示）

强调动画：
  - 放大（突出关键数据）
  - 变色（强调重点）

退出动画：
  - 淡出
  - 飞出

原则：
  - 每页动画不超过3个
  - 动画时间0.3-0.5秒
  - 避免花哨效果（旋转、弹跳）
  - 流程图可用逐步出现
```

### 6.2 页面切换

```
推荐：
  - 淡出（最安全）
  - 推入（适合流程展示）

避免：
  - 翻转
  - 棋盘
  - 随机切换

切换时间：0.5-1秒
```

## 七、演讲技巧

### 7.1 演讲结构

```
开场（30秒）
  "各位老师好，我们是XX大学XX队，题目是XXX"

方案（1分钟）
  "题目要求XXX，我们采用XXX方案，主要考虑是..."

设计（2-3分钟）
  "系统由XX、YY、ZZ三部分组成..."
  "核心电路采用XXX，关键参数是..."

测试（1-2分钟）
  "经测试，各项指标如下..."
  "重点展示XXX波形..."

总结（30秒）
  "本系统完成了XXX，创新点是XXX"
  "谢谢各位老师！"
```

### 7.2 演讲要点

```
1. 语速适中，不要太快
2. 重点内容放慢、加重
3. 眼神交流，不要一直看PPT
4. 手势配合，指向关键图表
5. 准备讲稿，但不要照读
6. 控制时间，不要超时
```

### 7.3 常见问题准备

```
问题1：为什么选择这个方案？
  → 准备2-3个理由，用数据支撑

问题2：这个参数是怎么确定的？
  → 准备计算过程或实验依据

问题3：如果XX条件变化，系统如何应对？
  → 准备扩展性和适应性说明

问题4：创新点在哪里？
  → 准备1-2个具体的创新点

问题5：如果重新做，会改进什么？
  → 准备诚恳的改进方向
```

### 7.4 回答问题技巧

```
1. 认真听完问题，不要打断
2. 不会的诚实说"这个问题我们没有深入研究"
3. 用数据和事实回答，避免主观
4. 回答简洁，不要展开太多
5. 可以用"我们的理解是..."开头
```

## 八、PPT制作检查清单

### 8.1 内容检查

```
□ 封面信息完整（题目、学校、队员、日期）
□ 系统框图清晰完整
□ 方案对比有表格
□ 核心电路有原理图
□ 测试数据有表格和波形
□ 总结有创新点
```

### 8.2 格式检查

```
□ 字体统一（不超过2种）
□ 字号层次分明
□ 配色一致
□ 图表风格统一
□ 动画适度
□ 无错别字
```

### 8.3 演练检查

```
□ 演讲时间控制在规定范围内
□ 每页内容能流畅讲解
□ 关键数据能脱口而出
□ 常见问题有准备答案
□ 至少完整演练3次
```
