# 电赛误差分析模板

> **使用说明**：误差分析是报告的重要评分项，需做到来源明确、计算准确、结论清晰。
> 建议使用方和根法合成总误差。

---

## 一、误差分类体系

### 1.1 误差分类总览

| 误差类型 | 定义 | 特点 | 消除/减小方法 |
|----------|------|------|--------------|
| **系统误差** | 按固定规律变化的误差 | 可重复、可预测、可修正 | 校准、补偿、改进设计 |
| **随机误差** | 不可预测的偶然波动 | 服从统计规律、不可消除 | 多次测量取平均、滤波 |
| **粗大误差** | 明显偏离的异常值 | 超出正常范围 | 剔除（格拉布斯准则） |

### 1.2 误差来源分类框架

```
系统总误差
├── 硬件误差
│   ├── ADC量化误差
│   ├── DAC量化误差
│   ├── 器件非线性误差
│   ├── 温度漂移
│   ├── 电源纹波
│   └── 噪声（热噪声/1/f噪声/散粒噪声）
├── 软件误差
│   ├── 算法截断误差
│   ├── 数值计算舍入误差
│   └── 采样率不足（欠采样/混叠）
├── 测量误差
│   ├── 仪器误差
│   ├── 读数误差
│   └── 环境干扰
└── 系统性偏差
    ├── 零点偏移
    ├── 增益偏差
    └── 非线性失真
```

## 二、ADC量化误差分析

### 2.1 原理

ADC将连续模拟信号量化为离散数字值，量化误差为：

$$e_q = \pm \frac{1}{2} \text{LSB} = \pm \frac{V_{ref}}{2^{N+1}}$$

其中：
- $V_{ref}$ 为ADC参考电压
- $N$ 为ADC位数

- 参考电压：$V_{ref}$ = [XX] V
- 输入满量程：$V_{FS}$ = [XX] V

**计算**：
- LSB = $V_{ref} / 2^N$ = [XX] / [XX] = [XX] V
- 最大量化误差 = ±LSB/2 = ±[XX] V
- 量化信噪比 SNR = 6.02N + 1.76 = [XX] dB
- 量化误差相对值 = $\frac{1}{2^N} \times 100\%$ = [XX]%

### 2.3 改善措施
- 提高ADC分辨率：选用更高位数ADC
- 过采样+抽取：每4倍过采样可增加1位有效分辨率
- 抖动技术：添加适当噪声改善量化特性

$$= \sqrt{[值1]^2 + [值2]^2 + [值3]^2 + [值4]^2 + [值5]^2 + [值6]^2 + [值7]^2}$$

$$\epsilon_{max} = |\epsilon_1| + |\epsilon_2| + |\epsilon_3| + ... = [结果]%$$

**方法三：加权合成法**
