# 07_report_templates

## 模块概述

本模块收录电赛设计报告相关的模板、写作指南、评分标准和答辩技巧。帮助参赛队伍快速撰写高质量设计报告，避免常见扣分项。

## 使用方法

1. **选择模板**：根据题目类型选择对应的报告模板（控制/视觉/电源/信号）
2. **撰写报告**：按照 `电赛高分报告写作指南V2.md` 的建议撰写各章节
3. **自查自纠**：对照 `报告常见错误清单.md` 逐项检查
4. **生成框架**：运行 `报告自动生成脚本.py` 快速生成报告骨架
5. **答辩准备**：参考 `电赛答辩PPT模板.md` 制作PPT

```bash
# 使用报告生成脚本
python 报告自动生成脚本.py --type 控制 --title "你的题目"
```

## 依赖说明

## 常见问题

**Q: 报告用什么格式提交？**
A: 通常为PDF格式。建议用Markdown撰写后导出PDF，或直接用Word排版。

**Q: 报告多少页合适？**
A: 一般15-25页，重点在方案论证和测试数据，不要灌水。

**Q: 摘要怎么写？**
A: 200-300字，包含：题目要求→设计方案→核心指标→测试结果。参考 `高分写作句式库.md`。
