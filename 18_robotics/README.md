# 机器人竞赛优秀方案库

本目录收录了RoboMaster、Robocon、飞思卡尔智能车、ROS2竞赛等机器人竞赛中的优秀方案，并针对电赛平台（MSPM0G3507）进行了适配。

## 目录结构

```
18_robotics/
├── 01_电机控制算法/          # FOC、直接转矩控制等
├── 02_路径规划算法/          # D*、JPS、Hybrid A*等
├── 03_视觉算法/              # 车道线检测、目标检测、二维码识别等
├── 04_系统架构设计/          # 状态机、任务调度、通信架构等
├── 05_调试工具和方法/        # 在线调试、数据可视化、日志系统等
└── README.md                 # 本文件
```

## 使用说明

每个算法模块包含：
- `README.md` - 设计思路和理论分析
- `src/` - 核心代码实现
- `examples/` - 使用示例
- `docs/` - 详细文档
- `adapted/` - 适配电赛平台的版本

## 快速开始

1. 阅读对应模块的README.md了解设计思路
2. 查看examples/目录下的示例代码
3. 使用adapted/目录下的适配版本集成到项目中

## 适配平台

- 主控：MSPM0G3507
- 开发环境：CCS / Keil
- 通信协议：UART、SPI、I2C、CAN
