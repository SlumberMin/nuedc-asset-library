# 电机控制算法库

本模块收录了机器人竞赛中常用的电机控制算法，包括FOC（磁场定向控制）、PID控制、直接转矩控制等。

## 算法列表

### 1. FOC（磁场定向控制）
- **来源**: RoboMaster 优秀开源项目
- **特点**: 高效率、低噪音、精确控制
- **适用**: 无刷直流电机（BLDC）、永磁同步电机（PMSM）
- **文件**: `foc/`

### 2. 增量式PID
- **来源**: 飞思卡尔智能车竞赛
- **特点**: 无积分饱和、响应快
- **适用**: 直流电机速度控制
- **文件**: `pid/incremental_pid.c`

### 3. 位置式PID
- **来源**: Robocon 竞赛
- **特点**: 精确定位、稳态误差小
- **适用**: 舵机、步进电机位置控制
- **文件**: `pid/position_pid.c`

### 4. 串级PID
- **来源**: RoboMaster 步兵机器人
- **特点**: 速度环+位置环双闭环控制
- **适用**: 需要精确位置控制的场合
- **文件**: `pid/cascade_pid.c`

### 5. 直接转矩控制（DTC）
- **来源**: ROS2 竞赛方案
- **特点**: 动态响应快、结构简单
- **适用**: 高性能电机控制
- **文件**: `dtc/`

## 使用方法

1. 选择适合的算法
2. 根据电机参数调整PID参数
3. 集成到项目中
