# FOC算法使用文档

## 1. 算法原理

FOC（Field-Oriented Control，磁场定向控制）是一种高效的电机控制方法，通过将三相交流电机的控制解耦为励磁分量（d轴）和转矩分量（q轴），实现类似直流电机的控制特性。

### 1.1 控制流程

```
三相电流 (ia, ib, ic)
    ↓
Clark变换 (α-β坐标系)
    ↓
Park变换 (d-q坐标系)
    ↓
PI控制器 (id, iq → vd, vq)
    ↓
反Park变换 (α-β坐标系)
    ↓
SVPWM (PWM占空比)
```

### 1.2 关键公式

**Clark变换：**
```
i_α = (2/3) × (i_a - 0.5×i_b - 0.5×i_c)
i_β = (2/3) × (√3/2×i_b - √3/2×i_c)
```

**Park变换：**
```
i_d = i_α×cos(θ) + i_β×sin(θ)
i_q = -i_α×sin(θ) + i_β×cos(θ)
```

**反Park变换：**
```
v_α = v_d×cos(θ) - v_q×sin(θ)
v_β = v_d×sin(θ) + v_q×cos(θ)
```

## 2. 硬件连接

### 2.1 电机驱动电路
```
MSPM0G3507 PWM输出 → 电机驱动器（如DRV8302） → 三相电机
```

### 2.2 电流采样
```
电流传感器（如ACS712） → ADC → MSPM0G3507
```

### 2.3 位置传感器
```
编码器（如AS5048A） → SPI → MSPM0G3507
或
霍尔传感器 → GPIO中断 → MSPM0G3507
```

## 4. 参数整定

### 4.1 电流环PI参数
- **Kp**: 0.1~1.0（根据电机电感调整）
- **Ki**: 0.001~0.01（根据电机电阻调整）
- **调整方法**: 先调整Kp使响应快速，再调整Ki消除稳态误差

### 4.2 速度环PI参数
- **Kp**: 0.01~0.1（根据负载惯量调整）
- **Ki**: 0.0001~0.001（根据系统要求调整）
- **调整方法**: 先调整Kp使速度响应快速，再调整Ki消除稳态误差

## 5. 使用示例

### 5.1 初始化
```c
FOC_HandleTypeDef hfoc;

void main()
{
    // 初始化FOC
    FOC_Init(&hfoc);
    
    // 设置目标速度
    hfoc.speed_ref = 1000.0f; // 1000 rpm
    
    // 启动PWM
    PWM_Start();
}
```

### 5.2 中断服务函数
```c
void PWM_IRQHandler(void)
{
    // 读取电流
    float i_a = ADC_ReadCurrentA();
    float i_b = ADC_ReadCurrentB();
    float i_c = -(i_a + i_b); // 假设三相平衡
    
    // 读取编码器角度
    float angle = Encoder_GetAngle();
    
    // 更新FOC
    FOC_Update(&hfoc, i_a, i_b, i_c, angle);
    
    // 设置PWM占空比
    PWM_SetDuty(PWM_CH_A, hfoc.duty_a);
    PWM_SetDuty(PWM_CH_B, hfoc.duty_b);
    PWM_SetDuty(PWM_CH_C, hfoc.duty_c);
}
```

## 6. 常见问题

### 6.1 电机振动
- 检查电流采样是否正确
- 检查编码器角度是否准确
- 降低PI控制器增益

### 6.2 电机不转
- 检查PWM输出是否正确
- 检查电流采样是否正常
- 检查编码器是否工作正常

### 6.3 电流过大
- 检查电机参数是否匹配
- 降低PI控制器增益
- 检查电流采样精度

## 7. 性能优化

### 7.1 提高控制频率
- 使用更高频率的PWM（如40kHz）
- 优化中断服务函数
- 使用DMA传输

### 7.2 提高电流采样精度
- 使用更高精度的ADC
- 增加采样滤波
- 校准电流传感器

### 7.3 提高位置检测精度
- 使用更高分辨率的编码器
- 实现无传感器FOC（如滑模观测器）
