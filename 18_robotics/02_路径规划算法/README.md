# 路径规划算法库

本模块收录了机器人竞赛中常用的路径规划算法，包括A*、D*、JPS、Hybrid A*等。

## 算法列表

### 1. A*算法
- **来源**: RoboMaster 导航系统
- **特点**: 静态环境下的最优路径搜索
- **适用**: 已知地图的全局路径规划
- **文件**: `astar/`

### 2. D*算法
- **来源**: ROS2 自主导航
- **特点**: 动态环境下的增量式路径规划
- **适用**: 未知或动态变化的环境
- **文件**: `dstar/`

### 3. JPS（Jump Point Search）
- **来源**: RoboMaster 快速导航
- **特点**: 在均匀网格上比A*快数倍
- **适用**: 均匀网格地图
- **文件**: `jps/`

### 4. Hybrid A*
- **来源**: ROS2 自动驾驶
- **特点**: 考虑机器人运动学约束
- **适用**: 非全向移动机器人（如汽车模型）
- **文件**: `hybrid_astar/`

### 5. RRT（快速扩展随机树）
- **来源**: Robocon 机械臂规划
- **特点**: 高维空间快速规划
- **适用**: 复杂环境、高自由度机器人
- **文件**: `rrt/`

## 使用方法

1. 根据环境类型选择合适的算法
2. 配置地图参数
3. 设置起点和终点
4. 调用路径规划函数
5. 执行路径跟踪
