# 调试工具和方法库

本模块收录了机器人竞赛中常用的调试工具和方法，包括数据可视化、日志系统、在线调试等。

## 工具列表

### 1. 数据可视化
- **来源**: RoboMaster 调试系统
- **特点**: 实时数据曲线显示
- **适用**: 参数调试、性能分析
- **文件**: `data_visualization/`

### 2. 日志系统
- **来源**: ROS2 日志系统
- **特点**: 分级日志、文件输出
- **适用**: 错误追踪、调试信息
- **文件**: `logging/`

### 3. 在线调试
- **来源**: 飞思卡尔智能车
- **特点**: 实时变量修改
- **适用**: 参数调整
- **文件**: `online_debug/`

### 4. 性能分析
- **来源**: ROS2 性能工具
- **特点**: CPU占用、执行时间
- **适用**: 性能优化
- **文件**: `profiler/`

### 5. 通信调试
- **来源**: RoboMaster 通信系统
- **特点**: 协议分析、数据包捕获
- **适用**: 通信问题排查
- **文件**: `comm_debug/`

## 调试方法

1. **分层调试**: 从底层硬件到上层应用逐层调试
2. **模块化调试**: 每个模块独立测试后再集成
3. **实时监控**: 使用数据可视化实时监控系统状态
4. **日志记录**: 记录关键信息用于事后分析
