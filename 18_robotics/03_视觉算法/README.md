# 视觉算法库

本模块收录了机器人竞赛中常用的视觉算法，包括车道线检测、目标检测、二维码识别等。

## 算法列表

### 1. 车道线检测
- **来源**: 飞思卡尔智能车竞赛
- **特点**: 边缘检测+霍夫变换
- **适用**: 循迹小车
- **文件**: `lane_detection/`

### 2. 目标检测
- **来源**: RoboMaster 视觉系统
- **特点**: 颜色分割+轮廓检测
- **适用**: 装甲板识别
- **文件**: `target_detection/`

### 3. 二维码识别
- **来源**: Robocon 物料搬运
- **特点**: QR码解码
- **适用**: 二维码识别
- **文件**: `qr_code/`

### 4. 颜色识别
- **来源**: 飞思卡尔智能车
- **特点**: HSV颜色空间转换
- **适用**: 颜色分类
- **文件**: `color_detection/`

### 5. 特征点匹配
- **来源**: ROS2 视觉SLAM
- **特点**: ORB/SIFT特征提取
- **适用**: 视觉定位
- **文件**: `feature_matching/`

## 使用方法

1. 初始化摄像头
2. 采集图像
3. 调用视觉处理函数
4. 解析结果
5. 输出控制指令
