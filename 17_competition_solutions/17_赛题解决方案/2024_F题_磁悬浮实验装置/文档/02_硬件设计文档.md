# 磁悬浮实验装置 — 硬件设计文档

---

## 一、电磁铁设计

### 1.1 电磁铁结构设计

采用**U型铁芯**结构，两侧绕制线圈，形成闭合磁路。悬浮盘为导磁材料，放置在U型铁芯开口上方。

```
        悬浮盘（硅钢片叠片）
    ┌─────────────────────────┐
    │                         │
    │    N ← 磁通 → S        │
    └─────────────────────────┘
         ↑               ↑
    ┌────┴────┐     ┌────┴────┐
    │  线圈A  │     │  线圈B  │    ← 漆包线绕组
    │         │     │         │
    │  铁芯   │     │  铁芯   │    ← 硅钢片叠片
    └────┬────┘     └────┬────┘
         └───────┬───────┘
            铁芯底座（磁轭）
```

### 1.2 电磁铁参数计算

**设计目标**：
- 悬浮高度：2~5cm
- 悬浮盘重量：约50g（含20g配重则为70g）
- 悬浮力需≥70g × 9.8 = 0.686N

**铁芯参数**：
- 材料：硅钢片叠片（DR510-50）
- 铁芯截面积：A = 2cm × 2cm = 4cm² = 4×10⁻⁴ m²
- 铁芯窗口面积：根据线圈匝数确定

**线圈参数**：
- 漆包线直径：0.5mm（AWG24）
- 单个线圈匝数：N = 200匝
- 线圈电阻：R ≈ 2×200×π×(1.5cm)×ρ/A_wire
  - ρ_铜 = 1.72×10⁻⁸ Ω·m
  - A_wire = π×(0.25mm)² = 1.96×10⁻⁷ m²
  - 平均匝长 ≈ 10cm = 0.1m
  - R = ρ×L/A = 1.72×10⁻⁸ × (200×0.1) / 1.96×10⁻⁷ ≈ 1.76Ω
- 双线圈串联总电阻：R_total ≈ 3.5Ω

**悬浮力估算**：
- 在气隙h=2cm时，磁阻主要由气隙决定
- R_air = h/(μ₀×A) = 0.02/(4π×10⁻⁷×4×10⁻⁴) ≈ 3.98×10⁷ A/Wb
- 所需磁通：Φ = √(2μ₀×F×A) = √(2×4π×10⁻⁷×0.686×4×10⁻⁴) ≈ 2.6×10⁻⁴ Wb
- 所需安匝数：NI = Φ×R_air = 2.6×10⁻⁴ × 3.98×10⁷ ≈ 10348 A·turns
- 电流：I = 10348/200 ≈ 51.7A（过大！）

**重新设计**：增加匝数，减小气隙磁阻
- 增加匝数至N=500匝/线圈，双线圈并联
- 重新计算：I = NI/N = 10348/500 ≈ 20.7A（仍偏大）

**优化方案**：采用多电磁铁阵列（4个），每个承担1/4悬浮力
- 单个电磁铁所需力：F_single = 0.686/4 ≈ 0.172N
- 重新计算：Φ_single = √(2×4π×10⁻⁷×0.172×4×10⁻⁴) ≈ 1.3×10⁻⁴ Wb
- NI_single = 1.3×10⁻⁴ × 3.98×10⁷ ≈ 5174 A·turns
- I_single = 5174/500 ≈ 10.3A（可接受，24V供电）

### 1.3 最终电磁铁方案

采用**4个独立电磁铁**组成阵列，对称分布在底座台上：

```
    ┌─────────────────────────┐
    │   [电磁铁1]  [电磁铁2]   │
    │       ○          ○      │  底座台 (18cm×18cm)
    │                         │
    │       ○          ○      │
    │   [电磁铁3]  [电磁铁4]   │
    └─────────────────────────┘
```

每个电磁铁参数：
- 铁芯：硅钢片叠片，截面积2cm²
- 线圈：500匝，0.5mm漆包线
- 电阻：约3.5Ω
- 额定电流：2~8A（可调）
- 额定电压：24V

### 1.4 悬浮盘设计

- 材料：硅钢片叠片（0.35mm厚，叠厚5mm）
- 直径：8cm（满足≤10cm要求）
- 重量：约40~50g
- 表面处理：涂漆防锈

## 二、霍尔传感器设计

### 2.1 传感器选型

选用**SS49E**线性霍尔传感器：
- 灵敏度：1.0~1.75 mV/Gs
- 工作电压：2.7~6.5V
- 输出类型：模拟电压
- 封装：TO-92
- 线性范围：±1000 Gs

### 2.2 传感器布局

采用**4个霍尔传感器**对称布局，用于检测悬浮盘高度和倾斜：

```
    ┌─────────────────────────┐
    │   [H1]          [H2]    │
    │       ○          ○      │  底座台表面
    │                         │
    │       ○          ○      │
    │   [H3]          [H4]    │
    └─────────────────────────┘
```

- H1~H4：检测各点磁场强度
- 高度计算：h = f(B_avg)，其中B_avg为平均磁场强度
- 倾斜检测：通过四点差值判断倾斜方向和角度

### 2.3 信号调理电路

霍尔传感器输出为mV级信号，需放大至ADC可采集范围（0~3.3V）。

**放大电路**（每个通道）：

```
V_Hall ──→ [R1=1kΩ] ──→ (-)OPA ──→ V_out
                              │
                     [Rf=100kΩ] │
                              │
                     [R2=1kΩ] ──→ GND
                              │
                     V_ref ──→ (+)OPA
```

- 增益：G = Rf/R1 = 100
- 偏置：V_ref = 1.65V（中间值）
- 输出范围：1.65V ± 100×V_Hall
- 运放：LM358（双运放，4个传感器需2片）

### 2.4 传感器标定

实际使用前需标定高度-电压关系：
1. 将悬浮盘放置在已知高度
2. 记录ADC值
3. 建立查找表或拟合曲线
4. h = a×V² + b×V + c（二次拟合）

## 三、驱动电路设计

### 3.1 驱动方案

采用**PWM + MOSFET H桥**驱动电磁铁：
- PWM频率：20kHz（人耳不可闻）
- 占空比：0~100%（控制电流大小）
- H桥：实现电流方向控制（可选，本设计仅需单方向）

### 3.2 半桥驱动电路

由于电磁铁只需单方向电流，采用**低边MOSFET开关**即可：

```
         +24V
          │
    ┌─────┴─────┐
    │  电磁铁    │
    │  线圈      │
    │  L + R     │
    └─────┬─────┘
          │
    ┌─────┴─────┐
    │  续流二极管 │ ← 1N5819（肖特基）
    │  (反并联)  │
    └─────┬─────┘
          │
    ┌─────┴─────┐
    │  IRF540N   │ ← N-MOSFET
    │  (N沟道)   │
    └─────┬─────┘
          │
         GND
```

**驱动芯片**：IR2104（半桥驱动器）
- 输入：PWM信号（3.3V逻辑兼容）
- 输出：高/低边栅极驱动
- 自举电容：10μF

### 3.3 MOSFET选型

选用**IRF540N**：
- Vds：100V（>24V，满足要求）
- Id：33A（>10A，满足要求）
- Rds(on)：44mΩ
- 封装：TO-220

### 3.4 续流保护

电磁铁为感性负载，MOSFET关断时会产生反向EMF。续流二极管提供电流泄放路径：
- 选用1N5819肖特基二极管（快恢复，低正向压降）
- 反向耐压：40V（>24V）

### 3.5 电流采样（可选）

串联小电阻（0.1Ω）采样电流，用于过流保护和电流闭环：
- 采样电压：V_sense = I × 0.1Ω
- 最大电流10A时，V_sense = 1V
- 经运放放大后送入ADC

## 四、电源设计

### 4.1 电源架构

```
外部24V DC ──→ [LM2596-5.0] ──→ 5V ──→ [AMS1117-3.3] ──→ 3.3V
    │              │
    │         ┌────┴────┐
    │         │ 5V供电   │
    │         │ 传感器   │
    │         │ 运放     │
    │         └─────────┘
    │
    └──→ 直接供电电磁铁（24V）
```

### 4.2 关键器件

- **LM2596-5.0**：24V→5V降压模块，最大输出3A
- **AMS1117-3.3**：5V→3.3V LDO，为STM32供电
- **滤波电容**：输入端100μF电解+0.1μF陶瓷，输出端47μF电解+0.1μF陶瓷

## 六、PCB设计要点

### 6.1 布局原则

1. **电源分区**：模拟地与数字地单点连接
2. **大电流路径**：电磁铁驱动走线加粗（≥2mm）
3. **信号完整性**：霍尔传感器信号线远离大电流走线
4. **散热**：MOSFET下方大面积铺铜

### 6.2 关键走线

- 电磁铁供电线：2oz铜厚，宽度≥3mm
- ADC信号线：包地处理，避免干扰
- PWM信号线：短且直，减少EMI
