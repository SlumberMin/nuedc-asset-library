# 2023年E题 运动目标控制与自动追踪系统 - 系统方案设计

## 一、题目分析

### 1.1 系统组成
- **红色光斑位置控制系统**（模拟运动目标）：红色激光笔 + 二维云台
- **绿色光斑位置控制系统**（自动追踪）：绿色激光笔 + 二维云台
- 两系统**完全独立**，之间不得有任何通信
- 屏幕：白色，有效面积>0.6×0.6m²，距激光笔1m

### 1.2 基本要求
1. 红色光斑位置复位功能（回到原点），误差≤2cm
2. 红色光斑30s内沿屏幕边线顺时针移动一周，距边线≤2cm
3. 沿A4靶纸四边移动（黑色电工胶带）
4. 靶纸任意旋转角度贴在任意位置

### 1.3 发挥部分
1. 绿色光斑2s内追踪红色光斑，距离≤3cm
2. 追踪红色光斑的边线运动，全程距离≤3cm
3. 暂停键功能

### 1.4 关键约束
- 两系统独立，不得通信
- 光斑直径≤1cm
- 屏幕上无电子元件
- 不得用台式/笔记本电脑

## 二、方案设计

### 红色系统方案
- STM32F103C8T6主控
- OpenMV视觉模块识别屏幕边线/靶纸位置
- 二维SG90云台+红色激光笔
- PID位置控制

### 绿色系统方案（追踪）
- STM32F103C8T6主控
- **光电位置传感器(PSD)** 检测红色光斑位置
- 或 **摄像头** 识别红色光斑
- 二维SG90云台+绿色激光笔
- PID追踪控制

### 关键技术：如何在不通信的前提下追踪？
**方案A**：绿色系统用摄像头直接识别屏幕上的红色光斑
**方案B**：绿色系统用光电传感器检测红色光斑的光强分布
**选择方案A**：摄像头识别精度高，可同时识别红色光斑位置

## 三、系统总体架构

```
红色系统(独立)                绿色系统(独立)
┌─────────────┐              ┌─────────────┐
│ STM32F103   │              │ STM32F103   │
│    ↓        │              │    ↓        │
│ OpenMV      │   屏幕       │ OpenMV      │
│ (识别边线)  │ ←──────────→ │ (识别红点)  │
│    ↓        │   1m距离     │    ↓        │
│ 云台+红激光 │              │ 云台+绿激光 │
└─────────────┘              └─────────────┘
```

## 四、核心模块选型

| 模块 | 红色系统 | 绿色系统 |
|------|----------|----------|
| 主控 | STM32F103C8T6 | STM32F103C8T6 |
| 视觉 | OpenMV4 H7 | OpenMV4 H7 |
| 舵机 | SG90×2 | SG90×2 |
| 激光 | 650nm红光 | 532nm绿光 |
| 电源 | 7.4V锂电池 | 7.4V锂电池 |

## 五、OpenMV程序策略

### 红色系统OpenMV任务
1. 识别屏幕边线（铅笔线）
2. 识别A4靶纸边框（黑色胶带）
3. 输出当前位置坐标给STM32

### 绿色系统OpenMV任务
1. 识别屏幕上的红色光斑（颜色追踪）
2. 输出红色光斑位置坐标给STM32
3. STM32控制绿色激光追踪
