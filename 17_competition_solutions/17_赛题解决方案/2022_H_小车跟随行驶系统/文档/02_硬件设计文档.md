# 2022_H_小车跟随行驶系统 - 硬件设计文档

## 一、系统框图

```
┌────────────────────────────────────────────────────────┐
│                     电源系统                            │
│  ┌──────────┐   ┌──────────┐   ┌──────────────────┐  │
│  │ 12V锂电池│──▶│ 5V DC-DC │──▶│ 3.3V LDO(AMS1117)│  │
│  │ 3S 2200m │   │ LM2596  │   │ 给MCU/OLED供电   │  │
│  └──────────┘   └──────────┘   └──────────────────┘  │
└────────────────────────────────────────────────────────┘
```

## 二、主控最小系统（STM32F407VET6）

### 2.1 核心参数

| 参数 | 值 |
|------|-----|
| 内核 | ARM Cortex-M4F |
| 主频 | 168 MHz |
| Flash | 512 KB |
| SRAM | 192 KB |
| 封装 | LQFP-100 |

### 2.2 电源电路

```
12V ──┬── LM2596(DC-DC) ──▶ 5V (电机驱动/传感器)
      │
      └── AMS1117-3.3(LDO) ──▶ 3.3V (MCU/逻辑)

关键电容:
  - 输入: 220μF/25V 电解电容
  - 输出: 100μF/10V 电解电容 + 100nF 陶瓷电容
  - MCU旁路: 每个VDD引脚 100nF 陶瓷电容
```

### 2.3 时钟电路

```
8MHz 晶振 ──▶ OSC_IN/OSC_OUT
  - 负载电容: 20pF
  - 通过PLL×21 → 168MHz 系统时钟
  
32.768kHz 晶振 ──▶ OSC32_IN/OSC32_OUT (可选，RTC用)
```

### 2.4 复位电路

```
VCC(3.3V) ──┬── 10kΩ上拉 ──▶ NRST
             │
             └── 100nF滤波 ──▶ GND
             
复位按键 ──▶ 并联到10kΩ上拉电阻
```

### 2.5 SWD调试接口

```
SWD接口引脚:
  - SWDIO: PA13
  - SWCLK: PA14
  - VCC: 3.3V
  - GND: GND
  
连接方式: 4线J-Link/ST-Link
```

## 三、电机驱动电路（L298N）

### 3.1 电路原理

```
STM32引脚          L298N           电机
──────            ────           ────
PB12 (IN1)  ──▶  IN1 ──┐
PB13 (IN2)  ──▶  IN2 ──┼──▶ OUT1/OUT2 ──▶ 左电机
PB14 (IN3)  ──▶  IN3 ──┤
PB15 (IN4)  ──▶  IN4 ──┼──▶ OUT3/OUT4 ──▶ 右电机

TIM2_CH1    ──▶  ENA (PWM速度控制-左)
TIM2_CH2    ──▶  ENB (PWM速度控制-右)

电源:
  +VS: 12V (电机供电)
  +5V: 5V (逻辑供电，内部稳压)
  GND: 公共地
```

### 3.2 PWM参数

| 参数 | 值 | 说明 |
|------|-----|------|
| PWM频率 | 20 kHz | 避免电机啸叫 |
| 占空比范围 | 0~100% | 通过CCR寄存器控制 |
| 分辨率 | 12-bit (0~4095) | TIM2 ARR=4095 |
| 死区 | 无 | L298N内部处理 |

### 3.3 续流保护

每个电机并联：
- 100μF/25V 电解电容（低频滤波）
- 100nF 陶瓷电容（高频滤波）
- 1N4007 续流二极管（反向并联）

## 四、传感器电路

### 4.1 超声波模块（HC-SR04×4）

```
STM32引脚         HC-SR04
──────           ───────
PB6 (TRIG1)  ──▶ TRIG (超声波1-前左)
PB7 (ECHO1)  ◀── ECHO (需分压: 5V→3.3V)
PB8 (TRIG2)  ──▶ TRIG (超声波2-前右)
PB9 (ECHO2)  ◀── ECHO (需分压)

分压电路:
  ECHO(5V) ──┬── 1kΩ ──▶ STM32 GPIO(3.3V)
              └── 2kΩ ──▶ GND
```

### 4.2 编码器接口（光电编码器×4）

```
编码器A (左电机):
  - A相: PA0 (TIM2_CH1, 编码器模式)
  - B相: PA1 (TIM2_CH2, 编码器模式)
  - 供电: 5V
  - GND: 公共地

编码器B (右电机):
  - A相: PA6 (TIM3_CH1, 编码器模式)
  - B相: PA7 (TIM3_CH2, 编码器模式)
  - 供电: 5V
  - GND: 公共地

配置: 4倍频计数, 600线 → 2400脉冲/转
```

## 五、通信电路

### 5.1 NRF24L01+PA模块

```
STM32引脚       NRF24L01+PA
──────         ──────────
PA5 (SCK)   ──▶ SCK
PA6 (MISO)  ◀── MISO
PA7 (MOSI)  ──▶ MOSI
PA4 (CS)    ──▶ CSN
PA3 (CE)    ──▶ CE
PB0 (IRQ)   ◀── IRQ (可选)

供电:
  VCC: 3.3V (必须，不能接5V!)
  GND: 公共地

天线: 外置PA天线，有效距离>100m
```

### 5.2 串口通信（调试用）

```
STM32引脚       USB转TTL
──────         ────────
PA9 (TX)    ──▶ RXD
PA10 (RX)   ◀── TXD
GND         ──▶ GND

波特率: 115200
用途: 实时打印调试信息
```

## 六、显示电路

### 6.1 OLED显示模块（0.96寸 I2C）

```
STM32引脚       OLED
──────         ────
PB10 (SCL)  ──▶ SCL
PB11 (SDA)  ◀──▶ SDA
3.3V        ──▶ VCC
GND         ──▶ GND

I2C地址: 0x3C (7-bit)
分辨率: 128×64
用途: 显示系统状态、距离、速度等
```

## 七、PCB设计要点

### 7.1 布局原则

1. **电源区域独立**：电池→DC-DC→LDO，线宽≥2mm
2. **电机驱动区隔离**：大电流区域远离MCU
3. **传感器区域集中**：传感器接口靠近MCU引脚
4. **通信区域**：NRF24L01模块远离电机和电源（减少干扰）

### 7.2 关键走线

| 网络 | 线宽 | 说明 |
|------|------|------|
| 12V电源 | 2mm | 电机供电 |
| 5V电源 | 1.5mm | 传感器供电 |
| 3.3V电源 | 1mm | MCU供电 |
| PWM信号 | 0.3mm | 电机控制 |
| SPI信号 | 0.2mm | NRF24L01通信 |
| I2C信号 | 0.2mm | OLED显示 |

### 7.3 接地策略

```
数字地(DGND) ──┬── 单点接地 ──▶ 电源地(PGND)
               │
模拟地(AGND) ──┘

关键:
  - 电机驱动地单独走线，最后汇合
  - ADC参考地远离数字地
  - 铺地铜皮减少阻抗
```
