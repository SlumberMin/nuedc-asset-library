# 2022_H_小车跟随行驶系统 - 系统方案设计

## 一、题目分析

### 1.1 题目要求概述

本题要求设计一个主从小车跟随行驶系统，包括：
- **前车（领跑车）**：按指定路线行驶，后方设置标识
- **后车（跟随车）**：自动识别前车并保持固定距离跟随行驶
- **超车功能（发挥部分）**：后车具备自动超车能力

### 1.2 核心技术指标

| 指标 | 基本要求 | 发挥部分 |
|------|---------|---------|
| 跟随距离 | 50cm固定 | 30~100cm可调 |
| 跟随精度 | ±10cm | ±5cm |
| 响应速度 | ≤500ms | ≤200ms |
| 行驶速度 | 0.3~0.8m/s | 0.3~1.2m/s |
| 超车成功率 | -- | ≥90% |

## 二、方案论证

### 2.1 目标检测方案对比

| 方案 | 传感器 | 优点 | 缺点 | 选择 |
|------|--------|------|------|------|
| **方案A：视觉方案** | OV5647摄像头+ArUco标记 | 精度高，信息丰富 | 计算量大，延迟高 | **主方案** |
| 方案B：红外方案 | 红外发射/接收对管 | 成本低，响应快 | 精度差，抗干扰弱 | 备用方案 |
| 方案C：超声波方案 | HC-SR04超声波模块 | 测距精度高 | 无法检测方向 | 辅助方案 |
| 方案D：RFID方案 | RFID读写器+标签 | 稳定可靠 | 定位精度差 | 放弃 |

**最终选择：视觉+超声波融合方案**

### 2.2 通信方案对比

| 方案 | 通信方式 | 距离 | 速率 | 选择 |
|------|---------|------|------|------|
| **方案A：2.4G无线** | NRF24L01+PA | 100m | 2Mbps | **主方案** |
| 方案B：蓝牙 | HC-05 | 10m | 3Mbps | 备用 |
| 方案C：WiFi | ESP8266 | 50m | 高 | 备用 |

### 2.3 控制算法方案

| 方案 | 算法 | 优点 | 缺点 | 选择 |
|------|------|------|------|------|
| **方案A：双环PID** | 距离环+方向环 | 稳定可靠 | 参数需仔细整定 | **主方案** |
| 方案B：模糊PID | 自适应参数 | 鲁棒性强 | 调试复杂 | 备用 |
| 方案C：纯跟踪算法 | 几何追踪 | 简单 | 精度不足 | 放弃 |

### 3.2 模块划分

1. **视觉检测模块**（Orange Pi 5）
   - ArUco标记检测
   - 距离估算（基于标记尺寸反投影）
   - 水平偏移量计算

2. **运动控制模块**（STM32F407）
   - 电机PWM控制
   - 编码器测速
   - 速度闭环PID

3. **通信模块**（NRF24L01+PA）
   - 车间状态信息交换
   - 速度指令传输
   - 心跳检测

4. **超声波辅助模块**
   - HC-SR04测距
   - 融合视觉距离估计

## 四、关键算法设计

### 4.1 双环PID控制器

```
距离目标值 ──▶ [距离PID环] ──▶ 前进速度
                                    │
                    ┌───────────────┘
                    ▼
偏移量 ──────▶ [方向PID环] ──▶ 转向速度
                                    │
                    ┌───────────────┘
                    ▼
          [左轮 = 前进 + 转向]
          [右轮 = 前进 - 转向]
```

### 4.2 距离估算算法

基于ArUco标记的针孔相机模型反投影：

```
实际距离 = (标记实际尺寸 × 焦距) / 标记像素宽度
```

标定参数：
- 标记物理尺寸：10cm × 10cm
- 相机焦距：650像素（需标定）
- 标定方法：在已知距离处采集图像，拟合焦距

### 4.3 超车状态机

```
┌────────┐    距离<30cm    ┌──────────┐
│ FOLLOW │ ──────────────▶ │ OVERTAKE │
│ 跟随   │                 │ 超车     │
└────────┘ ◀────────────── └──────────┘
     ▲        超车完成
     │
     │        到达新领跑位
     │       ┌──────────┐
     └────── │   LEAD   │
             │ 领跑     │
             └──────────┘
```
