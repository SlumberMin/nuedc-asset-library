# 2020_C_坡道行驶电动小车 - 硬件设计文档

## 一、电源系统

```
12V锂电池(3S) ──┬── LM2596(DC-DC) ──▶ 5V (电机驱动/传感器)
                 │
                 └── AMS1117-3.3(LDO) ──▶ 3.3V (MCU/逻辑)
```

## 二、主控最小系统

- **MCU**: STM32F407VET6 (168MHz, 512KB Flash, 192KB SRAM)
- **晶振**: 8MHz主晶振
- **调试**: SWD接口 (PA13/PA14)

## 三、传感器电路

### 3.1 MPU6050六轴IMU

```
I2C地址: 0x68
SCL: PB10 (I2C2_SCL)
SDA: PB11 (I2C2_SDA)
VCC: 3.3V
GND: GND

数据输出:
  - 加速度: ±2g量程, 16bit分辨率
  - 陀螺仪: ±250°/s量程, 16bit分辨率
  - 采样率: 100Hz
  
安装要求:
  - 水平安装于小车底盘中心
  - X轴朝前(行驶方向)
  - Z轴朝上
```

### 3.2 编码器

```
左编码器: PA0(TIM2_CH1) + PA1(TIM2_CH2) 编码器模式
右编码器: PA6(TIM3_CH1) + PA7(TIM3_CH2) 编码器模式
参数: 600线, 4倍频 → 2400脉冲/转
速度计算: speed = pulses / (ppr × dt)
```

### 3.3 视觉循线模块

```
OpenMV/OV5647:
  UART1: PA9(TX) + PA10(RX) 115200bps
  安装: 下视朝地, 高度15~20cm
  检测: 黑线/红线, 返回偏移量
```

## 四、执行器电路

### 4.1 电机驱动 (L298N)

```
STM32 → L298N → 电机
PB12(IN1) ──┐
PB13(IN2) ──┼── 左电机 (JGA25-370)
PB14(IN3) ──┤
PB15(IN4) ──┼── 右电机
TIM2_CH1(ENA) ── PWM速度(左)
TIM2_CH2(ENB) ── PWM速度(右)

PWM参数:
  频率: 20kHz
  分辨率: 12-bit (0~4095)
  死区: 无(L298N内部处理)
```

### 4.2 电机选型

```
JGA25-370 12V 直流减速电机:
  - 额定电压: 12V
  - 减速比: 1:34
  - 额定扭矩: 0.3kg·cm
  - 空载转速: 330rpm
  - 适用坡度: ≤25°
```

## 五、显示电路

```
OLED 0.96寸 I2C:
  SCL: PB10 (与MPU6050共用I2C)
  SDA: PB11
  地址: 0x3C
  
显示内容:
  行1: "Slope Car v1.0"
  行2: "Speed: X.X m/s"
  行3: "Slope: XX.X deg"
  行4: "State: 平地/上坡/下坡"
```

## 六、机械结构要点

```
底盘布局:
  ┌─────────────────────┐
  │  电机A    电机B      │
  │  ┌──┐    ┌──┐       │
  │  │  │    │  │       │
  │  └──┘    └──┘       │
  │                     │
  │    MPU6050(中心)    │
  │       ●             │
  │                     │
  │  电池(底部,低重心)   │
  └─────────────────────┘

关键设计:
  - 重心尽量低(电池放底部)
  - 轮距适当(过大难转弯,过小不稳定)
  - 前轮距坡道入口高度需匹配
```
