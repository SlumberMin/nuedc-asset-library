# 2020_C_坡道行驶电动小车 - 系统方案设计

## 一、题目分析

### 1.1 题目要求概述

设计坡道行驶电动小车，要求：
- **平地行驶**：沿指定路线自主行驶
- **坡道检测**：自动识别坡道并调整控制策略
- **坡道行驶**：在坡道上保持速度稳定不失速
- **循线导航**：沿黑线/红线行驶

### 1.2 核心技术指标

| 指标 | 基本要求 | 发挥部分 |
|------|---------|---------|
| 最大坡度 | ≥15° | ≥20° |
| 坡道速度稳定性 | 速度波动<20% | 速度波动<10% |
| 循线精度 | ±3cm | ±1cm |
| 平地速度 | 0.5~1.0m/s | 0.3~1.2m/s |
| 坡道不失速 | 100% | 100% |

## 四、关键算法设计

### 4.1 坡度检测（互补滤波）

```c
// MPU6050数据融合
float ComplementaryFilter(float gyro_angle, float accel_angle) {
    static float filtered = 0;
    // 98%陀螺仪 + 2%加速度计
    filtered = 0.98f * (filtered + gyro_angle * dt) 
             + 0.02f * accel_angle;
    return filtered;
}
```

### 4.2 坡度自适应速度控制

```
目标速度 V_target = V_base + slope_compensation

上坡补偿: slope_compensation = slope_angle × 2.0 (每度增加2个速度单位)
下坡补偿: slope_compensation = slope_angle × 0.8 (适当减速)

最终速度PID:
  V_forward = PID(V_target, V_actual)
  左轮 = V_forward + steering
  右轮 = V_forward - steering
```

### 4.3 防滑控制策略

```c
void AntiSlipControl(void) {
    float target_speed = GetTargetSpeed();
    float actual_speed = Encoder_GetSpeed();
    float slope = MPU6050_GetSlope();
    
    // 检测打滑: 目标速度高但实际速度低
    if (target_speed > actual_speed * 1.5f && slope > 5.0f) {
        // 降低加速度，避免持续打滑
        Motor_SetAcceleration(LOW);
        // 可能需要降低目标速度
        target_speed *= 0.8f;
    }
    
    // PID速度控制
    float pid_out = PID_Calc(&g_speed_pid, target_speed, actual_speed);
    Motor_SetSpeed(pid_out, pid_out);
}
```
