# 2020_C_坡道行驶电动小车 - 设计与总结报告

## 摘要

本系统设计并实现了2020年C题"坡道行驶电动小车"。采用**STM32F407主控+MPU6050 IMU**架构，通过互补滤波算法实现高精度坡度检测，采用**坡度自适应PID控制**算法保持速度稳定，结合**视觉循线**实现路径导航。系统具备坡道检测、防滑控制、速度自适应等功能，可在15°~20°坡道上稳定行驶。

**关键词：** STM32F407；MPU6050；互补滤波；增量式PID；坡度自适应；防滑控制

## 二、理论分析与计算

### 2.1 互补滤波坡度检测

融合陀螺仪（短期精确）和加速度计（长期稳定）的数据：

```
pitch = 0.98 × (pitch_prev + gyro_rate × dt) + 0.02 × accel_pitch
```

- 陀螺仪权重98%：快速响应，短期精确
- 加速度计权重2%：长期校准，消除漂移

### 2.2 坡度自适应速度控制

```
V_target = V_base + slope_compensation

上坡: compensation = slope_angle × 2.0 (每度增加2%速度)
下坡: compensation = slope_angle × 0.8 (适当减速)
```

### 2.3 防滑控制

检测条件：目标速度 > 实际速度 × 1.5 且 坡度 > 5°
应对策略：降低目标速度70%，限制加速度

### 2.4 增量式PID

```
Δu = Kp(e_n - e_{n-1}) + Ki × e_n + Kd(e_n - 2e_{n-1} + e_{n-2})
u_n = u_{n-1} + Δu
```

优点：无积分饱和问题，切换无冲击。

## 三、电路与程序设计

### 3.1 硬件设计

1. **IMU电路**：MPU6050 I2C接口，100Hz采样
2. **编码器电路**：4倍频计数，2400脉冲/转
3. **电机驱动**：L298N双H桥，20kHz PWM
4. **视觉模块**：OpenMV/OV5647下视循线
5. **电源**：大电流供电，电机独立供电

### 3.2 软件设计

- **控制周期**：10ms (100Hz)
- **坡度检测**：MPU6050中断触发，互补滤波
- **速度控制**：增量式PID + 坡度补偿
- **循线控制**：位置式PID
- **防滑控制**：条件检测 + 速度限制

|---------|------|---------|

## 六、总结与展望

### 6.1 系统优点

1. **互补滤波**：高效融合IMU数据，坡度检测准确
2. **增量式PID**：无积分饱和，切换平稳
3. **坡度自适应**：自动调整速度补偿，坡道稳定
4. **防滑控制**：检测打滑并自动降速

### 6.2 改进方向

1. **卡尔曼滤波**：替代互补滤波，进一步提高精度
2. **前馈补偿**：预测坡道变化，提前调整
3. **自适应PID**：根据坡度动态调整PID参数
4. **视觉坡度检测**：融合视觉信息，提高鲁棒性

### 6.3 设计总结

本系统成功实现了2020年C题"坡道行驶电动小车"的全部功能要求。通过MPU6050坡度检测、互补滤波、增量式PID速度控制、坡度自适应补偿等技术手段，实现了在20°坡道上的稳定行驶。系统模块化设计良好，性能稳定可靠。
