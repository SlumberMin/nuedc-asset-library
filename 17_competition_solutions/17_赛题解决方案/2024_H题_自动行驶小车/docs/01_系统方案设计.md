# 2024年全国大学生电子设计竞赛 H题 - 自动行驶小车

## 系统方案设计

### 一、题目分析

#### 1.1 赛题要求概述

本题要求设计一辆自动行驶小车，在规定场地（220cm × 120cm）上沿黑色弧线路径自主行驶。场地由两条直线段和两个半圆弧（R=40cm）组成，弧线黑色宽度1.8cm。

**场地布局说明：**
- 场地尺寸：220cm × 120cm
- 路径由两条直线段（AB段、CD段）和两个半圆弧（BC段、DA段）组成
- A、B、C、D为路径上的四个关键点
- 半圆弧半径R=40cm，弧线宽度1.8cm

#### 1.2 基本要求

| 序号 | 任务要求 | 时间限制 |
|------|----------|----------|
| 1 | A→B 直线行驶 | ≤ 15s |
| 2 | A→B→C→D→A 完整一圈 | ≤ 30s |
| 3 | A→C→B→D→A（反向行驶） | ≤ 40s |
| 4 | 连续行驶4圈 | - |

#### 1.3 约束条件

- 小车尺寸：≤ 25cm × 15cm × 15cm
- 驱动方式：轮式
- **禁止后退**
- **禁止使用摄像头**（不得使用OpenMV、OV2640等视觉模块）
- 经过A、B、C、D四个点时必须有声光提示
- 小车投影必须覆盖弧线顶点（即不能抄近路）

### 二、系统总体方案

#### 2.1 方案选择与论证

**（1）循迹方案选择**

| 方案 | 优点 | 缺点 |
|------|------|------|
| 方案一：红外循迹传感器阵列 | 成本低、响应快、电路简单 | 受环境光影响 |
| 方案二：激光循迹传感器 | 精度高、抗干扰强 | 成本较高 |
| 方案三：摄像头视觉循迹 | 信息丰富 | 赛题禁止使用 |

**选择方案一：红外循迹传感器阵列**，采用7路TCRT5000红外对管，间隔10mm排布，覆盖约6cm宽度，可精确检测1.8cm黑线位置。

**（2）电机驱动方案**

| 方案 | 优点 | 缺点 |
|------|------|------|
| 方案一：L298N驱动 | 电路简单、价格低 | 效率低、压降大 |
| 方案二：TB6612FNG驱动 | 效率高、体积小 | 价格略高 |
| 方案三：MOS管H桥驱动 | 效率最高 | 电路复杂 |

**选择方案二：TB6612FNG**，双通道驱动，可直接驱动两个直流减速电机，PWM频率可达100kHz。

**（3）测速方案**

采用**光电编码器**（增量式），安装在电机轴上，每转200脉冲，通过定时器编码器接口读取，实现精确速度反馈。

**（4）控制器选择**

题目要求使用TI MSPM0系列MCU，但本方案基于**STM32F103C8T6**实现（可移植至MSPM0）：
- 72MHz主频，满足实时控制需求
- 丰富的定时器资源（TIM1-TIM4）
- 多路ADC、UART、I2C、SPI接口
- 开发生态完善，调试方便

#### 2.3 路径规划策略

**正向行驶路径：A → B → C → D → A**
1. A→B：直线段，使用循迹传感器跟踪黑线
2. B→C：右半圆弧（R=40cm），差速转向
3. C→D：直线段，循迹跟踪
4. D→A：左半圆弧（R=40cm），差速转向

**反向行驶路径：A → C → B → D → A**
1. A→C：先走弧线段，再直线
2. C→B：直线段（反向）
3. B→D：弧线段
4. D→A：直线段（反向）

**弧线段行驶策略：**
- 通过编码器测量行驶距离，判断是否进入弧线段
- 在弧线段采用差速控制，内外轮速度差实现转弯
- 内外轮速度比 = (R - d/2) / (R + d/2)，其中d为轮距
- 实时循迹修正，确保投影覆盖弧线顶点

#### 2.4 关键点检测策略

**A、B、C、D四点检测方法：**
- 在路径上预设标记点（可用白色胶带在黑线两侧做标记）
- 通过编码器累积距离 + 循迹传感器组合判断
- 检测到关键点后立即触发声光提示（蜂鸣器响 + LED亮）

**弧线顶点覆盖保证：**
- 循迹传感器实时跟踪黑线中心
- 弧线段采用较低速度，增加循迹精度
- 通过PID控制确保小车中心始终在黑线上方
