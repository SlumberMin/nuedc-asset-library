### 二、核心器件选型

#### 2.1 主控芯片：STM32F103C8T6

| 参数 | 规格 |
|------|------|
| 内核 | ARM Cortex-M3 |
| 主频 | 72MHz |
| Flash | 64KB |
| RAM | 20KB |
| GPIO | 37个 |
| 定时器 | 4个（TIM1-TIM4） |
| ADC | 2个12位ADC，10通道 |
| UART | 3个 |
| I2C | 2个 |
| SPI | 2个 |
| 封装 | LQFP48 |

#### 2.2 电机驱动：TB6612FNG

| 参数 | 规格 |
|------|------|
| 驱动通道 | 双通道H桥 |
| 工作电压 | 2.5V ~ 13.5V |
| 输出电流 | 1.2A（持续）/ 3.2A（峰值） |
| PWM频率 | 最高100kHz |
| 待机模式 | 支持 |

#### 2.3 红外循迹传感器：TCRT5000

| 参数 | 规格 |
|------|------|
| 工作电压 | 3.3V ~ 5V |
| 检测距离 | 1mm ~ 25mm |
| 输出类型 | 数字（比较器输出） |
| 数量 | 7路 |
| 排列间距 | 10mm |

#### 2.4 编码器：霍尔编码器

| 参数 | 规格 |
|------|------|
| 类型 | 增量式霍尔编码器 |
| 分辨率 | 200脉冲/转（减速后约10000脉冲/转） |
| 输出信号 | A/B两相正交信号 |
| 数量 | 2个（左右轮各一个） |

#### 2.5 电机：JGA25-370直流减速电机

| 参数 | 规格 |
|------|------|
| 额定电压 | 6V ~ 12V |
| 减速比 | 1:34 或 1:47 |
| 空载转速 | 约150-300 RPM |
| 堵转扭矩 | 约2-5 kg·cm |
| 带编码器 | 是 |

### 三、关键电路设计

#### 3.1 电源电路设计

```
7.4V锂电池 ──┬── LM7805 ── 5V（传感器、编码器供电）
              │
              ├── AMS1117-3.3 ── 3.3V（MCU供电）
              │
              └── 直接供电 ── TB6612FNG VM（电机驱动）

电容滤波：
- LM7805输入端：100μF电解 + 0.1μF陶瓷
- LM7805输出端：100μF电解 + 0.1μF陶瓷
- AMS1117输入端：10μF电解 + 0.1μF陶瓷
- AMS1117输出端：10μF电解 + 0.1μF陶瓷
```

#### 3.2 TB6612FNG电机驱动电路

```
                    ┌─────────────┐
         AIN1 ────▶│             │──── AOUT1 ── 左电机+
         AIN2 ────▶│  TB6612FNG  │──── AOUT2 ── 左电机-
         PWMA ────▶│  通道A      │
         STBY ────▶│             │
         BIN1 ────▶│  通道B      │──── BOUT1 ── 右电机+
         BIN2 ────▶│             │──── BOUT2 ── 右电机-
         PWMB ────▶│             │
                    └─────────────┘
         VM ── 7.4V（电机电源）
         VCC ── 3.3V（逻辑电源）
         GND ── 公共地

控制逻辑：
┌──────┬──────┬──────────┐
│ AIN1 │ AIN2 │ 电机状态   │
├──────┼──────┼──────────┤
│  0   │  0   │ 制动停止   │
│  1   │  0   │ 正转       │
│  0   │  1   │ 反转       │
│  1   │  1   │ 制动停止   │
└──────┴──────┴──────────┘
PWM占空比控制转速
```

#### 3.3 红外循迹传感器电路

```
VCC(5V) ──┬── TCRT5000发射管 ── GND
           │
           └── TCRT5000接收管 ──┬── 比较器(LM393)输入
                                │
                                └── 10K上拉电阻 ── VCC

比较器输出：
- 检测到黑线：输出低电平(0)
- 未检测到黑线：输出高电平(1)

7路传感器排列（从左到右）：
S1  S2  S3  S4  S5  S6  S7
│   │   │   │   │   │   │
▼   ▼   ▼   ▼   ▼   ▼   ▼
┌───┬───┬───┬───┬───┬───┬───┐
│10mm间距排列，总宽度约60mm  │
└───────────────────────────────┘

传感器安装位置：小车前端，距地面5-10mm
```

#### 3.4 编码器接口电路

```
编码器A相 ──┬── 10K上拉 ── 3.3V
            │
            └── STM32 TIM2_CH1 (PA0)

编码器B相 ──┬── 10K上拉 ── 3.3V
            │
            └── STM32 TIM2_CH2 (PA1)

编码器C相 ──┬── 10K上拉 ── 3.3V
            │
            └── STM32 TIM3_CH1 (PA6)

编码器D相 ──┬── 10K上拉 ── 3.3V
            │
            └── STM32 TIM3_CH2 (PA7)

编码器模式：TIM_EncoderMode_TI12（四倍频）
```

#### 3.5 声光提示电路

```
蜂鸣器驱动：
STM32 PB8 ── 1K电阻 ── NPN三极管(S8050)基极
                        S8050发射极 ── GND
                        S8050集电极 ── 蜂鸣器(有源) ── 5V

LED指示灯：
STM32 PB9 ── 220Ω电阻 ── LED(红色) ── GND
```

#### 3.6 按键电路

```
模式选择按键：
VCC(3.3V) ── 10K上拉 ──┬── STM32 PA4
                        │
                        └── 按键 ── GND

启动按键：
VCC(3.3V) ── 10K上拉 ──┬── STM32 PA5
                        │
                        └── 按键 ── GND
```

### 四、PCB设计要点

1. **电源层**：大面积铺铜，降低阻抗
2. **信号层**：模拟信号与数字信号分区
3. **电机驱动**：大电流走线加宽（≥1mm）
4. **传感器接口**：靠近板边，便于接线
5. **去耦电容**：每个VCC引脚旁放置0.1μF陶瓷电容

  └─────┤                             ├─────┘
