### 三、循迹PID控制算法

#### 3.1 偏差计算

7路传感器从左到右编号为S1~S7，各传感器对应权重：

| 传感器 | S1 | S2 | S3 | S4 | S5 | S6 | S7 |
|--------|----|----|----|----|----|----|----|
| 权重 | -3 | -2 | -1 | 0 | +1 | +2 | +3 |

偏差计算公式：
```
error = Σ(sensor[i] × weight[i]) / Σ(sensor[i])
```

其中 `sensor[i]` 为第i路传感器的输出（0或1）。

#### 3.2 PID控制

采用**增量式PID**控制算法：

```
Δu = Kp × (error - error_last) + Ki × error + Kd × (error - 2×error_last + error_prev)
u = u + Δu
```

**参数整定：**

| 参数 | 直线段 | 弧线段 |
|------|--------|--------|
| Kp | 25 | 35 |
| Ki | 0.1 | 0.2 |
| Kd | 15 | 20 |
| 基础速度 | 200 | 150 |

#### 3.3 差速转向计算

```
左轮速度 = base_speed + correction
右轮速度 = base_speed - correction

correction = u × turn_factor
```

### 四、路径规划算法

#### 4.1 状态机设计

```
┌─────────────────────────────────────────────────────────────┐
│                       路径状态机                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────┐    到达B点    ┌─────────┐    到达C点           │
│  │ A→B直线  │───────────▶│ B→C右弧  │───────────▶          │
│  │ 行驶     │            │ 行驶     │                      │
│  └─────────┘            └─────────┘                      │
│       ▲                                                │    │
│       │                                                ▼    │
│  ┌─────────┐            ┌─────────┐                      │
│  │ D→A左弧  │◀───────────│ C→D直线  │◀─────────────────────┘
│  │ 行驶     │   到达A点   │ 行驶     │    到达D点
│  └─────────┘            └─────────┘
│       │
│       ▼
│  圈数+1，判断是否继续
│
└─────────────────────────────────────────────────────────────┘
```

#### 4.2 距离判断策略

各段距离计算：
- AB段（直线）：约100cm
- BC段（半圆弧）：π × R = π × 40 ≈ 125.7cm
- CD段（直线）：约100cm
- DA段（半圆弧）：π × R = π × 40 ≈ 125.7cm

编码器脉冲到距离的转换：
```
距离(cm) = 脉冲数 × π × 轮径 / (编码器分辨率 × 减速比)
```

以轮径65mm、编码器200线、减速比34:1为例：
```
每脉冲距离 = π × 6.5 / (200 × 4 × 34) ≈ 0.00075cm
每厘米脉冲数 ≈ 1333 脉冲
```

#### 4.3 弧线段控制策略

进入弧线段时：
1. 降低基础速度（从200降至150）
2. 增大PID的Kp、Kd参数
3. 根据传感器偏差实时调整差速
4. 通过编码器监测行驶距离，判断是否离开弧线段

#### 4.4 关键点检测

```
关键点检测逻辑：
IF (累积距离 >= 当前段目标距离 - 容差) THEN
    IF (传感器检测到特定标记 或 偏差突变) THEN
        确认到达关键点
        触发声光提示
        切换到下一段
    END IF
END IF
```

### 五、中断服务程序

#### 5.1 定时器中断（10ms周期）

```
TIM2中断（10ms）：
├── 读取编码器计数值
├── 计算瞬时速度
├── 更新累积距离
├── 循迹PID计算
└── 更新电机PWM
```

#### 5.2 编码器中断

```
TIM2编码器模式：
├── 自动计数
├── 方向判断（正转/反转）
└── 溢出处理
```
