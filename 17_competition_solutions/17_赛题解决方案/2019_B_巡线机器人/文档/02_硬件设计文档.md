# 2019_B_巡线机器人 - 硬件设计文档

## 一、电源系统

```
3S锂电池(12V) ──┬── LM2596(DC-DC) ──▶ 5V (驱动/传感器)
                 │
                 └── AMS1117-3.3(LDO) ──▶ 3.3V (MCU/逻辑)
```

## 二、主控最小系统

- **MCU**: STM32F407VET6 (168MHz) 或 Orange Pi 5 (视觉处理)
- **晶振**: 8MHz
- **调试**: SWD接口

## 三、传感器电路

### 3.1 下视摄像头

```
OV5647 → STM32/OPi5
  MIPI-CSI/USB接口
  安装高度: 15cm
  视场角: 60° (覆盖20cm宽度)
  帧率: 30fps
  分辨率: 320×240 (速度优先)
```

### 3.2 红外阵列（备用）

```
TCRT5000 × 5, 一字排列, 间距1cm
引脚:
  传感器1: PA0 (ADC_CH0)
  传感器2: PA1 (ADC_CH1)
  传感器3: PA2 (ADC_CH2)
  传感器4: PA3 (ADC_CH3)
  传感器5: PA4 (ADC_CH4)
  
检测方式: 模拟量读取, 黑线<阈值, 白地>阈值
偏移量计算: 加权平均法
```

### 3.3 编码器

```
左编码器: PA6(TIM3_CH1) + PA7(TIM3_CH2)
右编码器: PA0(TIM2_CH1) + PA1(TIM2_CH2)
参数: 600线, 4倍频
```

### 3.4 MPU6050

```
I2C: PB10(SCL) + PB11(SDA)
地址: 0x68
用途: 转向角度补偿
```

## 四、执行器电路

### 4.1 TB6612FNG电机驱动

```
STM32 → TB6612 → 电机
PB12(AIN1) ──┐
PB13(AIN2) ──┼── 电机A (左)
PB14(BIN1) ──┤
PB15(BIN2) ──┼── 电机B (右)
TIM2_CH1(PWMA) ── PWM速度(左)
TIM2_CH2(PWMB) ── PWM速度(右)
PA8(STBY)   ── 使能(高有效)

TB6612 vs L298N:
  - TB6612: 效率高(MOSFET), 体积小, 双H桥
  - L298N: 电流大(BJT), 体积大, 发热多
  巡线机器人选TB6612更合适
```

### 4.2 电机选型

```
JGA25-370 12V:
  - 减速比: 1:34
  - 空载转速: 330rpm
  - 适用: 两轮差速驱动
  - 轮子直径: 65mm
  - 最大线速度: ~1.1m/s
```

## 五、底盘机械设计

```
底盘布局 (俯视图):
┌─────────────────────────────┐
│         摄像头(前)           │
│            ●                │
│                             │
│   ┌────┐         ┌────┐   │
│   │电机│         │电机│   │
│   │ A  │         │ B  │   │
│   └──┬─┘         └──┬─┘   │
│      │    MCU+电池   │     │
│      │    ┌──────┐   │     │
│      │    │ STM32│   │     │
│      │    └──────┘   │     │
│      │   ┌────────┐  │     │
│      └───┤  电池  ├──┘     │
│          └────────┘        │
│                             │
│    ○ 万向轮(前)            │
└─────────────────────────────┘

关键尺寸:
  轮距: 16cm (两轮中心距)
  轴距: 12cm (前后轮距)
  摄像头高度: 15cm
  重心: 尽量低且居中
```
