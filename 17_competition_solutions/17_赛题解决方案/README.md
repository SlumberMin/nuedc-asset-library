# 赛题解决方案

本目录收录了全国大学生电子设计竞赛（电赛）历年赛题的解决方案，涵盖 2009 年至 2025 年共 17 年的赛题。每个方案包含 Markdown 方案文档，部分方案附带完整代码工程。

---

## 目录结构说明

- **`YYYY_X_题目名_方案.md`** — 该赛题的解决方案文档（部分年份有带题号和不带题号两个版本）
- **`YYYY_X_题目名/`** — 完整方案子目录，通常包含：
  - `代码/` 或 `code/` — 源代码工程
  - `文档/` 或 `docs/` — 设计文档、原理图等
- **`YYYY_控制题_xxx/`** — 控制类赛题专项方案，含 `README.md` 说明
- **`场景N_xxx_工程框架/`** — 通用工程框架模板（含 `main.c`）
- **`可行性审查报告.md`** — 赛题可行性综合审查

## 如何使用这些方案

### 1. 快速查阅方案思路
直接打开对应年份和题号的 `*_方案.md` 文件，了解该赛题的解题思路、硬件选型、软件架构和关键算法。

### 2. 使用完整工程代码
进入带子目录的赛题文件夹（如 `2023_I_气垫悬浮车/`），其中：
- **`代码/`（或 `code/`）** — 可直接编译的工程源码
- **`文档/`（或 `docs/`）** — 原理图、PCB 设计、测试报告等

### 3. 控制类赛题专项
`YYYY_控制题_xxx/` 目录下的 `README.md` 详细说明了控制类赛题的特定实现方法，适合作为 PID / 电机控制 / 平衡控制等方向的参考。

### 4. 基于工程框架快速开发
`场景N_*_工程框架/` 提供了 `main.c` 模板，可作为新赛题项目的起始框架，根据具体需求修改外设配置和控制逻辑。

### 5. 方案选型参考
阅读 `可行性审查报告.md` 可了解各赛题方案的整体可行性评估，帮助在备赛时选择适合的题目方向。
