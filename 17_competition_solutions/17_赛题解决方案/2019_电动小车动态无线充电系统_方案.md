## 视觉定位方案

### 摄像头安装
- **俯视摄像头**：安装在轨道正上方，OV5647，向下拍摄
- **识别目标**：小车上的LED标记点（不同颜色区分前后）
- **发射线圈定位**：通过轨道中心标记识别

### 定位算法
```python
import cv2
import numpy as np

class DynamicChargerTracker:
    def __init__(self):
        self.camera_matrix = np.load('calib_mtx.npy')
        self.dist_coeffs = np.load('calib_dist.npy')
        self.center = None  # 发射线圈位置（像素）
        self.radius = None  # 轨道半径（像素）

    def detect_car(self, frame):
        """检测小车位置和朝向"""
        hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
        # 前端红色标记
        red_mask = cv2.inRange(hsv, (0,100,100), (10,255,255))
        # 后端蓝色标记
        blue_mask = cv2.inRange(hsv, (100,100,100), (130,255,255))

        red_pos = self._find_marker_center(red_mask)
        blue_pos = self._find_marker_center(blue_mask)

        if red_pos and blue_pos:
            car_center = ((red_pos[0]+blue_pos[0])//2,
                         (red_pos[1]+blue_pos[1])//2)
            angle = np.arctan2(red_pos[1]-blue_pos[1],
                              red_pos[0]-blue_pos[0])
            return car_center, angle
        return None, None

    def calc_alignment(self, car_pos):
        """计算小车与发射线圈的对齐误差"""
        if self.center is None:
            return 0, 0
        dx = car_pos[0] - self.center[0]
        dy = car_pos[1] - self.center[1]
        radial_err = np.sqrt(dx**2+dy**2) - self.radius
        angular_err = np.arctan2(dy, dx)
        return radial_err, angular_err

    def calc_optimal_speed(self, angle, charge_efficiency):
        """根据对齐程度调整速度 - 对齐好时减速充电"""
        # 对齐精度高时降低速度以提高充电效率
        base_speed = 0.3  # m/s
        if charge_efficiency > 0.8:
            return base_speed * 0.6
        elif charge_efficiency > 0.5:
            return base_speed * 0.8
        return base_speed
```

## 无线充电子系统

### 硬件拓扑
```
发射端(固定侧):
  24V DC → 全桥逆变(MOSFET) → 发射线圈 + 串联谐振电容
  频率: 100-200kHz (可调)
  控制: Orange Pi PWM调节逆变频率

接收端(小车侧):
  接收线圈 + 并联谐振电容 → 整流桥 → DC-DC(Buck) → 超级电容/电池
  反馈: 电压/电流采样 → 无线回传至OPi5
```

### 谐振匹配算法
```python
class ResonanceController:
    def __init__(self):
        self.target_freq = 150000  # 150kHz基础频率
        self.pwm_channel = 0

    def adjust_frequency(self, v_rect, i_rect, v_target=12.0):
        """根据接收端反馈调节发射频率以优化耦合"""
        power = v_rect * i_rect
        # 简单P&O最大功率追踪
        if power > self.last_power:
            self.freq_step = abs(self.freq_step)
        else:
            self.freq_step = -abs(self.freq_step)

        self.target_freq += self.freq_step
        self.target_freq = np.clip(self.target_freq, 100000, 250000)
        self._set_inverter_freq(self.target_freq)
        self.last_power = power

    def _set_inverter_freq(self, freq):
        """设置全桥逆变器频率"""
        period_ns = int(1e9 / freq)
        duty_ns = period_ns // 2
        # 通过sysfs或OPi5 PWM接口设置
        with open('/sys/class/pwm/pwmchip0/pwm0/period', 'w') as f:
            f.write(str(period_ns))
        with open('/sys/class/pwm/pwmchip0/pwm0/duty_cycle', 'w') as f:
            f.write(str(duty_ns))
```

## 运动控制

### 循圆控制
```python
class CircularMotionController:
    def __init__(self, target_radius_m=1.0):
        self.R = target_radius_m
        self.Kp_v = 2.0
        self.Kp_w = 3.0
        self.base_speed = 0.3

    def compute(self, radial_err, angle_err, charge_eff):
        """
        radial_err: 径向偏差(像素→米)
        angle_err: 角度偏差
        charge_eff: 充电效率(0~1)
        """
        # 径向修正
        v_correction = self.Kp_v * radial_err
        # 角速度保持切向运动
        w_correction = self.Kp_w * angle_err

        # 充电效率高时降速
        speed_factor = 1.0 - 0.4 * charge_eff
        v = (self.base_speed + v_correction) * speed_factor
        w = self.base_speed / self.R + w_correction

        # 差速模型
        L = 0.15  # 轮距
        v_left = v - w * L / 2
        v_right = v + w * L / 2
        return v_left, v_right
```

## 能量管理

```python
class EnergyManager:
    """超级电容充电管理"""
    def __init__(self):
        self.v_cap_max = 12.6  # 电容组最高电压
        self.v_cap_min = 7.0
        self.state = 'CHARGING'  # CHARGING / RUNNING / IDLE

    def update(self, v_cap, v_rect, i_rect, car_speed):
        charge_power = v_rect * i_rect
        soc = (v_cap - self.v_cap_min) / (self.v_cap_max - self.v_cap_min)

        if charge_power > 0.5:  # 有效充电中
            self.state = 'CHARGING'
            # 限制充电电流保护电容
            if v_cap > self.v_cap_max * 0.95:
                return 'TRICKLE'
            return 'FAST_CHARGE'
        else:
            self.state = 'RUNNING'
            if soc < 0.2:
                return 'LOW_POWER'  # 降低速度
            return 'NORMAL'
```

## Orange Pi 5 接口分配

|------|------|------|
| MIPI CSI-0 | 俯视摄像头 | OV5647定位 |

## 关键算法流程

```
初始化 → 标定轨道中心和半径
  ↓
主循环(30fps):
  ├─ 视觉检测小车位置和朝向
  ├─ 计算径向/角度偏差
  ├─ 采集充电参数(电压/电流/功率)
  ├─ 最大功率追踪(P&O)调节谐振频率
  ├─ 根据充电效率动态调速
  ├─ 差速控制保持圆形轨道
  └─ 能量管理(电容SOC监控)
```
