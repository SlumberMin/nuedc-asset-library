# 2021_F_智能送药小车 - 硬件设计文档

## 一、系统框图

```
┌─────────────────────────────────────────────────────────┐
│                     电源系统                              │
│  ┌──────────┐   ┌──────────┐   ┌──────────────────┐    │
│  │ 12V锂电池│──▶│ 5V DC-DC │──▶│ 3.3V LDO         │    │
│  │ 3S 2200m │   │ LM2596   │   │ 给MCU/逻辑供电   │    │
│  └──────────┘   └──────────┘   └──────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

## 二、主控最小系统

- **MCU**: STM32F407VET6 (168MHz Cortex-M4F)
- **晶振**: 8MHz主晶振 + 32.768kHz RTC晶振
- **调试**: SWD接口 (PA13/PA14)
- **电源**: 3.3V AMS1117 LDO

## 三、传感器电路

### 3.1 OpenMV视觉模块

```
OpenMV引脚     STM32引脚
────────      ──────────
TX (P1)    ──▶ PA10 (UART1_RX)
RX (P0)    ◀── PA9  (UART1_TX)
GND        ──▶ GND
3.3V       ──▶ 3.3V

波特率: 115200
用途: 数字识别结果传输 + 循线偏移量
```

### 3.2 编码器

```
左编码器: PA0(TIM2_CH1) + PA1(TIM2_CH2)
右编码器: PA6(TIM3_CH1) + PA7(TIM3_CH2)
模式: 4倍频, 600线 → 2400脉冲/转
```

### 3.3 超声波避障

```
前左: PB6(TRIG) + PB7(ECHO, 5V→3.3V分压)
前右: PB8(TRIG) + PB9(ECHO, 分压)
后方: PB10(TRIG) + PB11(ECHO, 分压)
```

### 3.4 MPU6050陀螺仪

```
I2C地址: 0x68
SCL: PB10 (I2C2_SCL)
SDA: PB11 (I2C2_SDA)
用途: 转弯角度检测
```

## 四、执行器电路

### 4.1 电机驱动 (L298N)

```
STM32 → L298N → 电机
PB12(IN1) ──┐
PB13(IN2) ──┼── 左电机
PB14(IN3) ──┤
PB15(IN4) ──┼── 右电机
TIM2_CH1(ENA) ── PWM速度(左)
TIM2_CH2(ENB) ── PWM速度(右)
```

### 4.2 舵机控制 (MG996R)

```
送药机构舵机: TIM4_CH1 (PB6) 50Hz PWM
药仓开关舵机: TIM4_CH2 (PB7) 50Hz PWM
脉宽范围: 500μs~2500μs
角度范围: 0°~180°
```

### 4.3 送药机构

```
┌──────────┐
│  药仓    │ ← 舵机控制开合
│  ┌────┐  │
│  │药品│  │ ← 重力下落送药
│  └────┘  │
│    ↓     │
│  传送口   │
└──────────┘
```

## 五、显示电路

```
OLED (0.96寸 I2C):
  SCL: PB10
  SDA: PB11 (与MPU6050共用I2C)
  地址: 0x3C
  
显示内容:
  行1: 系统状态 (药房/运送中/病房)
  行2: 目标病房号
  行3: 当前位置坐标
  行4: 电池电压
```
