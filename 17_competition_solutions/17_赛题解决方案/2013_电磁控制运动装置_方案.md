# 2013 全国大学生电子设计竞赛 - 电磁控制运动装置

## 题目概述
设计一个电磁控制运动装置，通过控制电磁铁的通断和电流方向，使铁质物体（钢珠等）在平板上按预定轨迹运动，实现位置精确控制。

## Orange Pi 5 系统架构

### 核心方案：电磁铁阵列驱动 + 视觉位置反馈 + 闭环运动控制

## 软件架构

### 1. 视觉定位模块（OpenCV）

```python
import cv2
import numpy as np

class BallTracker:
    def __init__(self, camera_id=0):
        self.cap = cv2.VideoCapture(camera_id)
        self.cap.set(cv2.CAP_PROP_FRAME_WIDTH, 640)
        self.cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)
        # 钢珠颜色范围（银色/深色，需根据实际调整）
        self.lower = np.array([0, 0, 80])
        self.upper = np.array([180, 50, 220])

    def get_ball_position(self):
        ret, frame = self.cap.read()
        if not ret:
            return None
        hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
        mask = cv2.inRange(hsv, self.lower, self.upper)
        mask = cv2.morphologyEx(mask, cv2.MORPH_OPEN, np.ones((5,5), np.uint8))
        contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
        if contours:
            c = max(contours, key=cv2.contourArea)
            M = cv2.moments(c)
            if M["m00"] > 0:
                cx = int(M["m10"] / M["m00"])
                cy = int(M["m01"] / M["m00"])
                return (cx, cy)
        return None
```

### 2. 电磁铁控制策略

```python
class ElectromagnetController:
    def __init__(self, uart_port='/dev/ttyS3'):
        import serial
        self.ser = serial.Serial(uart_port, 115200)
        self.num_coils = 8  # 电磁铁数量

    def set_coil(self, coil_id, polarity, power):
        """polarity: 0=关闭, 1=正向吸引, 2=反向排斥; power: 0-255"""
        cmd = bytes([0xAA, coil_id, polarity, power, 0x55])
        self.ser.write(cmd)

    def all_off(self):
        for i in range(self.num_coils):
            self.set_coil(i, 0, 0)

    def attract_ball(self, coil_id, power=200):
        """正向通电吸引钢珠"""
        self.set_coil(coil_id, 1, power)

    def repel_ball(self, coil_id, power=200):
        """反向通电排斥钢珠"""
        self.set_coil(coil_id, 2, power)
```

## 关键技术要点

### 电磁铁布局优化
- **棋盘式布局**：电磁铁按网格排列，间距根据平板尺寸和钢珠直径确定
- **分时复用**：相邻电磁铁交替激活，避免互相干扰
- **梯度控制**：通过调节多个电磁铁电流比例实现平滑运动

### 视觉检测优化
- 使用ROI减少计算量
- 背景差分法消除平板干扰
- 卡尔曼滤波预测钢珠位置，降低延迟

### 控制策略
- 外环：位置PID控制（视觉反馈，~30Hz）
- 内环：电磁铁电流控制（STM32，~1kHz）
- 前馈补偿：根据运动方向预先激活下一组电磁铁
