# 2024年全国大学生电子设计竞赛 - 控制题（H题）自动行驶小车 完整解决方案

## 一、题目分析

### 1.1 题目要求概述

设计一辆自动行驶小车，要求：
- **自动循迹**：沿预设黑色/白色赛道行驶
- **自动避障**：检测并避开赛道上的障碍物
- **定点停车**：在指定区域精确停车
- **坡道行驶（发挥部分）**：在坡道上保持稳定行驶
- **速度控制**：实现变速行驶，弯道减速直道加速

### 1.2 核心技术指标

| 指标 | 基本要求 | 发挥部分 |
|------|---------|---------|
| 循线方式 | 黑线循迹 | 白线/复杂赛道 |
| 循线精度 | ±2cm | ±1cm |
| 避障距离 | 30cm检测 | 50cm提前检测 |
| 定点停车精度 | ±5cm | ±2cm |
| 坡道角度 | 无 | ≤30° |
| 平均速度 | ≥0.5m/s | ≥1.0m/s |

### 1.3 难点分析

1. **高速循线稳定性**：速度越快，循线响应时间越短
2. **弯道与直道速度切换**：需要前瞻预测弯道
3. **坡道行驶**：重力补偿与防滑控制
4. **定点停车精度**：需要精确的编码器计数或视觉定位

## 五、核心算法说明

### 5.1 循线PID算法

> ⚠️ 以下代码基于MSPM0G3507平台，使用实际驱动API，可直接编译。
> 需要链接: `tb6612.h`, `encoder.h`, `advanced_pid.h`, `servo.h`

```c
/* ═══ 公共宏定义 ═══ */
#include <math.h>
#include "tb6612.h"        /* TB6612_SetMotor */
#include "encoder.h"       /* Encoder_GetSpeed */
#include "advanced_pid.h"  /* PID_Init, PID_Calc */
#include "servo.h"         /* Servo_SetAngle */

#define CLAMP(x, lo, hi) ((x) < (lo) ? (lo) : ((x) > (hi) ? (hi) : (x)))
#define BASE_SPEED    2000
#define TURN_LEFT     1
#define TURN_RIGHT    2
#define GO_STRAIGHT   0
```

```c
/* 修复C1: 自定义PID_t → PID_Controller + PID_Calc (实际驱动API) */
/* 修复B4: TURN_LEFT/TURN_RIGHT/GO_STRAIGHT/BASE_SPEED 已定义 */
/* 修复: constrain → CLAMP */

static PID_Controller line_pid;
static int line_pid_inited = 0;

/* 图像处理提取赛道中心线 */
int line_center_detect(uint8_t *image, int width, int height) {
    int sum = 0, count = 0;
    int row_start = (int)(height * 0.7f);
    int row_end   = (int)(height * 0.9f);
    for (int row = row_start; row < row_end; row++) {
        for (int col = 0; col < width; col++) {
            if (image[row * width + col] < 128) {  /* 检测黑线(阈值化) */
                sum += col;
                count++;
            }
        }
    }
    if (count > 0) return sum / count;  /* 返回中心x坐标 */
    return width / 2;  /* 未检测到则返回图像中心 */
}

/* PID计算 — 使用实际驱动API */
float line_pid_calc(float target, float actual) {
    if (!line_pid_inited) {
        PID_Param param = { .kp = 1.5f, .ki = 0.05f, .kd = 0.8f,
                            .output_min = -500.0f, .output_max = 500.0f,
                            .integral_max = 300.0f, .dead_zone = 2.0f };
        PID_Init(&line_pid, &param);
        line_pid_inited = 1;
    }
    return PID_Calc(&line_pid, target, actual);
}
```

### 5.2 避障决策算法

```c
/* 三路超声波避障决策 */
typedef struct {
    float front, left, right;  // 障碍物距离
} Obstacle_t;

int obstacle_avoid(Obstacle_t *obs, float threshold) {
    if (obs->front < threshold) {
        if (obs->left > obs->right) {
            return TURN_LEFT;   /* 左转避障 */
        } else {
            return TURN_RIGHT;  /* 右转避障 */
        }
    }
    return GO_STRAIGHT;  /* 无障碍 */
}
```

### 5.3 坡道识别与控制

```c
/* 修复C1: motor_set_speed → TB6612_SetMotor */
/* 修复B4: BASE_SPEED 已定义 */

/* 通过MPU6050检测坡道 */
void slope_control(float pitch_angle) {
    if (pitch_angle > 5.0f) {  /* 上坡 */
        int spd = CLAMP((int)(BASE_SPEED * 1.5f), 0, 3999);
        TB6612_SetMotor(MOTOR_CH_A, MOTOR_DIR_FORWARD, (uint32_t)spd);
        TB6612_SetMotor(MOTOR_CH_B, MOTOR_DIR_FORWARD, (uint32_t)spd);
    } else if (pitch_angle < -5.0f) {  /* 下坡 */
        int spd = CLAMP((int)(BASE_SPEED * 0.7f), 0, 3999);
        TB6612_SetMotor(MOTOR_CH_A, MOTOR_DIR_FORWARD, (uint32_t)spd);
        TB6612_SetMotor(MOTOR_CH_B, MOTOR_DIR_FORWARD, (uint32_t)spd);
    } else {
        /* 平地正常速度 */
        TB6612_SetMotor(MOTOR_CH_A, MOTOR_DIR_FORWARD, (uint32_t)BASE_SPEED);
        TB6612_SetMotor(MOTOR_CH_B, MOTOR_DIR_FORWARD, (uint32_t)BASE_SPEED);
    }
}
```

## 七、评分要点分析

### 7.1 基本要求评分（50分）

| 评分项 | 分值 | 要点 | 策略 |
|--------|------|------|------|
| 循线行驶 | 20分 | 完整跑完赛道不脱线 | 稳定PID+前瞻预测 |
| 避障功能 | 15分 | 成功避开障碍物 | 三路超声波融合 |
| 定点停车 | 15分 | 停车位置准确 | 编码器精确计数 |

### 7.2 发挥部分评分（50分）

| 评分项 | 分值 | 要点 | 策略 |
|--------|------|------|------|
| 坡道行驶 | 20分 | 上下坡稳定不翻车 | MPU6050角度补偿 |
| 变速行驶 | 15分 | 弯道减速直道加速 | 前瞻预测+速度规划 |
| 高速循线 | 15分 | 平均速度最快 | 优化PID参数+预测控制 |

### 7.3 得分策略

1. **保基础**：先确保循线和避障100%完成，稳定拿40分
2. **攻发挥**：坡道控制是最大加分项（20分），重点攻克
3. **优化速度**：在稳定前提下逐步提高速度
4. **调试技巧**：每轮调试记录参数，避免参数回退

### 7.4 常见扣分点与对策

| 扣分项 | 原因 | 对策 |
|--------|------|------|
| 脱线 | 弯道PID响应慢 | 增大前瞻距离+提前转弯 |
| 撞障碍物 | 检测距离不够 | 增加超声波数量+提前转向 |
| 停车不准 | 编码器误差累积 | 多传感器融合定位 |
| 坡道翻车 | 重心太高 | 降低重心+增加配重 |

## 八、调试经验总结

### 8.1 PID调参口诀

```
先P后D再I，从小到大慢慢提
P大振荡P小慢，D大抖动D小偏
I消稳差I大超，积分限幅要牢记
```

### 8.2 硬件调试要点

1. **重心分布**：电池放低，重心在后轴前方1/3处
2. **传感器安装**：摄像头离地20-30cm，俯角15-20°
3. **电机匹配**：左右电机速度一致性校准
4. **电源管理**：避免电压不稳导致MCU复位

### 8.3 软件调试技巧

1. **串口调试**：实时输出传感器数据和PID参数
2. **分步调试**：先单独测每个模块，再联合调试
3. **参数保存**：将最优参数写入Flash，避免断电丢失
4. **异常处理**：脱线自动停车保护，避免损坏
