# 2017年B题 滚球控制系统 - 系统方案设计

## 一、题目分析

### 1.1 核心要求
- 65cm×65cm光滑平板上分布9个圆形区域(直径3cm)
- 通过控制平板倾斜，使小球按要求运动
- 小球直径≤2.5cm

### 1.2 基本要求
1. 小球在区域2停留≥5s
2. 15s内区域1→区域5，停留≥2s
3. 区域1→区域4→区域5，总时间≤20s
4. 30s内区域1→区域9，停留≥2s

### 1.3 发挥部分
1. 40s内区域1→2→6→9
2. 键盘设定A→B→C→D路径
3. 区域4出发环绕区域5运动≥3周后停在区域9

## 二、系统方案

### 2.1 平板支撑方案
**选择：四舵机支撑平台**
- 4个大扭矩舵机(MG996R)支撑平板四角
- 通过控制各舵机角度实现平板二维倾斜
- 倾斜范围约±15°

### 2.2 小球检测方案
**选择：电阻式触摸屏**
- 4线电阻触摸屏覆盖平板表面
- 实时检测小球XY坐标
- 精度约±1mm，满足要求

### 2.3 控制算法
**选择：双轴级联PID**
- 外环：位置PID（目标位置→目标角度）
- 内环：角度PID（目标角度→舵机输出）

## 三、系统总体架构

```
┌─────────────────────────────────────┐
│           STM32F103C8T6             │
├─────────────────────────────────────┤
│  ┌──────────┐    ┌──────────┐      │
│  │触摸屏    │    │键盘      │      │
│  │(小球位置)│    │(路径设定)│      │
│  └────┬─────┘    └────┬─────┘      │
│       │               │             │
│  ┌────┴───────────────┴────┐       │
│  │    双轴级联PID控制       │       │
│  │  位置PID → 角度PID      │       │
│  └────┬───────────────┬────┘       │
│       │               │             │
│  ┌────┴────┐    ┌─────┴─────┐     │
│  │X轴舵机  │    │Y轴舵机   │     │
│  │×2       │    │×2        │     │
│  └─────────┘    └──────────┘     │
│                                     │
│  ┌──────────┐    ┌──────────┐      │
│  │OLED显示  │    │计时显示  │      │
│  └──────────┘    └──────────┘      │
└─────────────────────────────────────┘
```

## 四、关键选型

| 模块 | 选型 | 理由 |
|------|------|------|
| 主控 | STM32F103C8T6 | 成熟可靠 |
| 舵机 | MG996R×4 | 扭矩大(13kg·cm)，响应快 |
| 检测 | 4线电阻触摸屏 | 精度高，实时性好 |
| 键盘 | 4×4矩阵键盘 | 路径设定 |
| 显示 | OLED + 数码管 | 状态+计时 |
