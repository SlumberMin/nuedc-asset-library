## 二、麦克风阵列设计

### 2.1 INMP441 MEMS麦克风参数

| 参数 | 值 |
|------|-----|
| 接口 | I2S数字输出 |
| 灵敏度 | -26dBFS |
| 信噪比 | 61dB |
| 频率响应 | 60Hz~15kHz |
| 供电 | 3.3V |
| 封装 | QFN-28 |

### 2.2 阵列布局（方形4麦克风）

```
        麦克风3 (0, +d/2)
           ●
           |
           | d=5cm
           |
麦克风1 ───┼─── 麦克风2
(-d/2, 0)  ●|●  (+d/2, 0)
           |
           | d=5cm
           |
           ●
        麦克风4 (0, -d/2)

麦克风间距 d = 5cm (决定最高可定位频率)
最高频率 f_max = c / (2d) = 343 / 0.1 = 3430 Hz
人声基频范围: 85~255Hz，适合本方案
```

### 2.3 接线设计

```
INMP441麦克风接线(每个):
  VDD  → 3.3V
  GND  → GND
  SD   → I2S数据线 (共享，需三态控制或独立I2S)
  WS   → I2S LRCLK
  SCK  → I2S BCLK
  L/R  → GND (左声道) / VDD (右声道)

Orange Pi 5 I2S接口:
  I2S0_SCLK → GPIO1_A0
  I2S0_LRCK → GPIO1_A1
  I2S0_SDO  → GPIO1_A2 (麦克风数据输入)
```

### 2.4 抗干扰设计

1. **电源滤波**：每个麦克风VDD旁 10μF + 100nF 电容
2. **地平面**：模拟地与数字地单点接地
3. **屏蔽**：麦克风阵列用金属屏蔽罩覆盖
4. **走线**：I2S信号线等长走线，差分走线

## 三、步进电机驱动电路

### 3.1 28BYJ-48步进电机参数

| 参数 | 值 |
|------|-----|
| 类型 | 4相8拍 |
| 步距角 | 5.625°/64 |
| 减速比 | 1:64 |
| 驱动电压 | 5V |
| 相电流 | 约240mA |

### 3.2 ULN2003驱动电路

```
Orange Pi 5 GPIO        ULN2003        28BYJ-48
────────────           ───────        ────────
GPIO1_B4 (STEP_X) ──▶ IN1 ──┐
GPIO1_B5 (DIR_X)  ──▶ IN2 ──┼──▶ OUT1~OUT4 ──▶ X轴电机
GPIO1_B6 (PWM_X)  ──▶ IN3 ──┤
GPIO1_B7 (EN_X)   ──▶ IN4 ──┘

GPIO1_C0 (STEP_Y) ──▶ IN5 ──┐
GPIO1_C1 (DIR_Y)  ──▶ IN6 ──┼──▶ OUT5~OUT8 ──▶ Y轴电机
GPIO1_C2 (PWM_Y)  ──▶ IN7 ──┤
GPIO1_C3 (EN_Y)   ──▶ IN8 ──┘

供电: COM引脚接5V
GND: ULN2003 GND接系统GND
```

### 3.3 细分驱动

采用8拍驱动模式，每步1.125°，实现更平滑的运动。

```
拍序:  1   2   3   4   5   6   7   8
IN1:   1   1   0   0   0   0   0   1
IN2:   0   1   1   1   0   0   0   0
IN3:   0   0   0   1   1   1   0   0
IN4:   0   0   0   0   0   1   1   1
```

## 四、激光指示器电路

```
GPIO1_C0 ──┬── 1kΩ ──▶ NPN基极 (S8050)
            │
            └── 10kΩ上拉 ──▶ VCC

S8050集电极 ──▶ 激光模块正极
S8050发射极 ──▶ GND

激光模块负极 ──▶ GND

控制: GPIO高电平 → NPN导通 → 激光开启
```

## 五、摄像头接口

```
OV5647 → Orange Pi 5 MIPI-CSI接口

  - 时钟: 100MHz
  - 分辨率: 640×480@30fps
  - 用途: 视觉确认声源位置
```

## 八、PCB设计要点

1. **I2S信号**：等长走线，阻抗控制，远离电源线
2. **麦克风布局**：精确对称，间距误差<0.5mm
3. **电源分区**：模拟电源与数字电源分离
4. **接地**：大面积铺地，减少地回路
