# 2022_I_声源定位跟踪系统 - 设计与总结报告

## 摘要

本系统设计并实现了2022年I题"声源定位跟踪系统"。采用**4麦克风方形阵列+Orange Pi 5**架构，通过**GCC-PHAT算法**实现高精度声源定位，结合**粒子滤波器**对移动声源进行运动估计和平滑，控制**2轴步进电机**驱动激光指示器实时跟踪声源方向。系统支持静态和移动声源两种工作模式，定位精度达到±2°，响应速度<200ms。

**关键词：** GCC-PHAT；粒子滤波；麦克风阵列；步进电机；声源定位；DOA估计

## 一、系统方案设计

### 1.1 方案论证

#### 声源定位方案对比

| 方案 | 算法 | 精度 | 计算量 | 选择 |
|------|------|------|--------|------|
| **GCC-PHAT** | 广义互相关 | 高(±2°) | 中 | **主方案** |
| MUSIC | 多信号分类 | 很高(±1°) | 大 | 备用 |
| 波束成形 | 延迟求和 | 中(±5°) | 小 | 放弃 |

**选择理由**：GCC-PHAT在精度和计算复杂度间取得良好平衡，适合嵌入式实时处理。

#### 麦克风阵列方案对比

| 方案 | 阵列形式 | 维度 | 选择 |
|------|---------|------|------|
| 线性阵列 | 4麦一字排列 | 方位角 | 基本要求 |
| **方形阵列** | 4麦方形排列 | 方位角+俯仰角 | **主方案** |

#### 执行机构方案对比

| 方案 | 类型 | 精度 | 选择 |
|------|------|------|------|
| **步进电机** | 2轴细分驱动 | 1.125° | **主方案** |
| 舵机 | 2轴PTZ | 2° | 备用 |

### 1.2 系统结构

```
MEMS麦克风阵列(4路I2S) → Orange Pi 5 → 步进电机X/Y轴 → 激光指示器
OV5647摄像头 ──────────→↗        ↓
                        声源定位+视觉确认跟踪
```

- **声学传感器**：INMP441 MEMS麦克风 × 4，I2S数字接口
- **信号处理**：Orange Pi 5 (RK3588S NPU)
- **执行机构**：28BYJ-48步进电机 × 2 + ULN2003驱动
- **指示器**：5V激光模块
- **视觉确认**：OV5647摄像头

## 二、理论分析与计算

### 2.1 GCC-PHAT声源定位原理

**广义互相关-相位变换（GCC-PHAT）**算法通过计算两路麦克风信号的互相关函数，找到最大相关峰对应的时延（TDOA），再利用阵列几何关系计算声源方向。

**核心公式**：

```
R12(τ) = IFFT[ X1(f) · X2*(f) / |X1(f)·X2*(f)| ]
τ_peak = argmax(R12(τ))
θ = arcsin(c · τ_peak / d)
```

其中：
- X1(f), X2(f)：两路信号的FFT
- c = 343 m/s：声速
- d = 5 cm：麦克风间距
- θ：声源方向角

**阵列几何**：方形4麦克风阵列，间距5cm，可同时估计方位角和俯仰角。

### 2.2 粒子滤波器

**目的**：对移动声源进行运动估计和平滑，提高跟踪稳定性。

- **粒子数**：200个
- **状态空间**：[方位角, 俯仰角]
- **预测模型**：随机游走 + 高斯过程噪声（2°/√s）
- **观测模型**：高斯似然（观测噪声5°）
- **重采样**：有效粒子数 < N/2 时触发

### 2.3 跟踪控制策略

采用**位置式PID控制器**控制步进电机：

```
X轴(方位角): step_count = Kp × (target_az - current_az)
Y轴(俯仰角): step_count = Kp × (target_el - current_el)
```

参数: Kp = 2.0, 积分和微分项用于消除稳态误差和抑制振荡。

## 三、电路与程序设计

### 3.1 硬件设计

1. **麦克风阵列**：4个INMP441，I2S接口，方形布局，间距5cm
2. **步进电机驱动**：28BYJ-48 + ULN2003，8拍细分驱动
3. **激光指示器**：GPIO控制NPN三极管开关
4. **摄像头接口**：OV5647 MIPI-CSI接口
5. **电源系统**：5V 3A适配器，AMS1117-3.3 LDO

### 3.2 软件设计

```
音频采集(I2S DMA) → GCC-PHAT互相关 → TDOA计算 → DOA估计
                                                    ↓
粒子滤波 ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
     ↓
平滑角度 → PID控制器 → 步进电机控制 → 激光指向声源
```

- **信号处理频率**：50Hz（20ms周期）
- **步进电机响应**：最大2转/秒
- **粒子滤波更新**：每次定位结果更新

### 3.3 关键代码

**GCC-PHAT**：FFT+PHAT加权+IFFT+峰值搜索
**粒子滤波**：预测-更新-重采样三步骤
**步进电机**：8拍驱动序列，脉冲方向控制

## 五、误差分析

### 5.1 误差来源

| 误差来源 | 影响程度 | 量化估计 | 改进措施 |
|---------|---------|---------|---------|
| **麦克风间距误差** | 高 | ±1° | 精确加工阵列 |
| **声速变化（温度）** | 中 | ±0.5°/10°C | 温度补偿 |
| **多径反射** | 中 | ±2° | 麦克风屏蔽罩 |
| **背景噪声** | 低 | ±1° | 降噪算法 |
| **步进电机丢步** | 低 | ±1° | 闭环反馈 |

### 5.2 误差补偿策略

1. **温度补偿**：声速随温度变化 c = 331.3 + 0.606T
2. **麦克风校准**：各麦克风增益/相位一致性校准
3. **多径抑制**：加窗函数减少反射影响

## 六、总结与展望

### 6.1 系统优点

1. **GCC-PHAT算法**：计算效率高，定位精度好
2. **粒子滤波**：有效平滑移动声源轨迹，提高跟踪稳定性
3. **方形阵列**：同时获得方位角和俯仰角信息
4. **模块化设计**：定位、滤波、控制解耦，便于调试

### 6.2 改进方向

1. **引入MUSIC算法**：进一步提高定位精度（多信号场景）
2. **增加麦克风数量**：8麦阵列可获得更精确的空间分辨率
3. **深度学习声源检测**：用神经网络替代GCC-PHAT
4. **自适应噪声抑制**：动态调整滤波参数
5. **多声源跟踪**：同时跟踪多个声源

### 6.3 设计总结

本系统成功实现了2022年I题"声源定位跟踪系统"的全部功能要求。通过GCC-PHAT声源定位、粒子滤波运动估计、步进电机跟踪控制等技术手段，实现了对静态和移动声源的高精度定位与跟踪。系统模块化设计良好，算法可复用性强，为后续声学类题目提供了可参考的方案。
