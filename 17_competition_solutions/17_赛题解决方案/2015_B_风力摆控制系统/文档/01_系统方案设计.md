# 2015年B题 风力摆控制系统 - 系统方案设计

## 一、题目分析

### 1.1 核心要求
- 细管(60-70cm)上端万向节固定，下方悬挂2-4只直流风机
- 激光笔向下，静止时距地≤20cm
- 控制风机使风力摆按规律运动，激光笔在地面画出要求轨迹

### 1.2 基本要求
1. 15s内画出长度≥50cm的直线段，线性度偏差≤±2.5cm
2. 30-60cm可设置长度的直线段
3. 可设定摆动方向（角度）
4. 5s内制动静止

### 1.3 发挥部分
1. 画圆，半径15-35cm可设置，偏差≤±2.5cm
2. 台扇干扰后5s内恢复圆周运动

## 二、系统方案

### 2.1 风机布局
**选择：4只风机十字布局**
- 0°/90°/180°/270°四个方向
- 通过差速控制实现任意方向推力
- 轴流风机12V/40mm

### 2.2 状态检测
**选择：MPU6050六轴传感器**
- 安装在风力摆底部
- 实时检测摆角(θ)和角速度(ω)
- I2C接口，采样率100Hz

### 2.3 控制算法
**选择：双环PID**
- 外环：角度PID（目标角度→目标角速度）
- 内环：角速度PID（目标角速度→风机PWM）

### 2.4 地面轨迹检测
**选择：OpenMV摄像头俯视**
- 检测激光笔光点位置
- 用于画圆时的闭环反馈

## 三、系统架构

```
         万向节
           │
      ┌────┴────┐
      │  细管   │
      │  60cm   │
      └────┬────┘
           │
    ┌──────┼──────┐
    │      │      │
  风机1  MPU6050  风机2
  (0°)   (检测)   (180°)
    │      │      │
    └──────┼──────┘
           │
    ┌──────┼──────┐
    │      │      │
  风机3  激光笔  风机4
  (90°)  (向下)  (270°)
    │             │
    └─────────────┘
```

## 四、关键选型

| 模块 | 选型 | 理由 |
|------|------|------|
| 主控 | STM32F103C8T6 | 成熟可靠 |
| 风机 | 12V/40mm轴流风机×4 | 轻便，推力适中 |
| 传感器 | MPU6050 | 六轴，I2C，精度高 |
| 地面检测 | OpenMV(可选) | 激光点位置追踪 |
| 电源 | 11.1V锂电池(3S) | 供电风机 |
