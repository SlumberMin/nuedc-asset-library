## 三、核心控制算法设计

### 3.1 循迹PID算法

**选择理由：位置式PID**
- 循迹系统需要连续的位置控制，位置式PID输出直接对应控制量
- 增量式PID更适合执行器本身具有积分特性（如阀门）的场合

**算法参数：**
```
Kp = 2.0   (快速响应横向偏差)
Ki = 0.5   (消除稳态误差)
Kd = 1.0   (抑制超调和振荡)
```

**参数整定方法：**
1. 先将Ki=0, Kd=0，逐渐增大Kp直到系统出现振荡
2. 取Kp为临界值的60%
3. 逐渐增大Ki消除稳态误差
4. 逐渐增大Kd抑制超调
5. 现场微调优化

**位置计算：加权平均法**
```
position = Σ(weight[i] × sensor[i]) / Σ(sensor[i])
```
- 权重从左到右为：-35, -25, -15, -5, +5, +15, +25, +35
- 输出范围：-100 ~ +100

### 3.2 瞄准PID算法

**双轴独立PID控制：**
- 水平轴PID：控制水平舵机角度
- 垂直轴PID：控制垂直舵机角度

**算法参数：**
```
Kp = 1.5   (适中的响应速度)
Ki = 0.3   (避免积分饱和)
Kd = 0.8   (平滑运动)
```

**坐标系定义：**
- OpenMV画面中心为原点(0,0)
- X轴正方向：向右
- Y轴正方向：向下
- 偏差范围：-160 ~ +160 (QVGA分辨率)

**负反馈控制逻辑：**
```
dx > 0 (目标在右边) → 舵机向左调整 → angle_h减小
dy > 0 (目标在下方) → 舵机向上调整 → angle_v减小
```

### 3.3 画圆算法

**原理：** 在循迹行驶的同时，云台做圆周运动

```
θ += (2π) / (正方形周长 / 行驶速度)
angle_h = 90° + R × cos(θ)
angle_v = 90° + R × sin(θ)
```

- R：角度偏移量，根据实际距离标定（约15°对应6cm半径）
- θ：画圆角度，0~2π循环
- 同步保证：行驶1圈对应θ变化2π

## 四、通信协议设计

### 4.1 OpenMV → MSPM0 (UART 115200bps)

**数据格式：** `$X,Y\n` 或 `$X,Y,AREA\n`

| 字段 | 类型 | 范围 | 说明 |
|------|------|------|------|
| $ | char | - | 起始标志 |
| X | float | -160~+160 | 水平像素偏差 |
| Y | float | -120~+120 | 垂直像素偏差 |
| AREA | int | 0~307200 | 目标面积(可选) |
| \n | char | - | 结束标志 |

### 4.2 OpenMV端程序（参考）

```python
import sensor, image, time
from pyb import UART

sensor.reset()
sensor.set_pixformat(sensor.RGB565)
sensor.set_framesize(sensor.QVGA)
sensor.skip_frames(time=2000)

uart = UART(3, 115200)
thresholds = [(30, 100, 15, 127, 15, 127)]  # 红色阈值

while(True):
    img = sensor.snapshot()
    blobs = img.find_blobs(thresholds, pixels_threshold=100)
    if blobs:
        b = max(blobs, key=lambda b: b.pixels())
        cx = b.cx() - 160  # 相对画面中心的偏差
        cy = b.cy() - 120
        uart.write("$%d,%d\n" % (cx, cy))
```

## 五、关键设计要点

### 5.1 实时性保证
- 主循环周期1ms，满足循迹实时性要求
- UART中断接收视觉数据，避免轮询丢失
- PID计算在主循环中执行，避免中断中计算过重

### 5.2 抗干扰设计
- 循迹传感器采用硬件比较器+软件滤波双重处理
- 视觉数据采用滑动窗口均值滤波
- PID输出限幅防止执行器饱和

### 5.3 安全保护
- 脱线检测：所有传感器都未检测到黑线时自动减速停车
- 超时保护：运行超过20s自动停止
- 电机限幅：PWM输出不超过999
- 舵机限幅：角度不超过0~180°范围
