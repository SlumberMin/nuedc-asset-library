# 2025年E题 简易自行瞄准装置 - 系统方案设计

## 一、题目分析

### 1.1 基本要求
1. 小车沿100cm×100cm正方形黑色轨迹自动寻迹逆时针行驶，圈数N可设定(1-5)，行驶时间t≤20s
2. 小车放置在场地任意位置，瞄准模块2s内发射激光击中靶心，D1≤2cm
3. 小车放置在轨迹指定位置，瞄准方向任意，4s内自动瞄准击中靶心，D1≤2cm

### 1.2 发挥部分
1. N=1圈，t≤20s，行驶中激光连续发光瞄准靶心，D1≤2cm
2. N=2圈，t≤40s，同上
3. N=1圈，t≤20s，激光沿半径6cm圆弧画圆，D2≤2cm，同步误差<1/4圈
4. 其他创新

### 1.3 关键技术指标
- 小车尺寸：≤25cm×15cm×25cm
- 靶面：A4紫外感光纸，距AB线50cm，高度≤50cm
- 激光光斑直径≤0.5cm
- 控制器：TI MSPM0系列MCU
- 电源：双独立开关分别控制MSPM0和瞄准模块

## 二、系统方案对比

### 方案一：纯红外循迹 + 开环瞄准
- **循迹**：红外传感器阵列检测黑线，PID控制电机差速转向
- **瞄准**：固定角度激光笔，预设角度瞄准
- **优点**：成本低，实现简单
- **缺点**：瞄准精度差，无法适应不同位置和姿态

### 方案二：红外循迹 + OpenMV视觉瞄准（推荐）
- **循迹**：红外传感器阵列 + 编码器，PID循迹 + 位置记忆
- **瞄准**：OpenMV摄像头识别靶心，二维舵机云台闭环控制
- **优点**：瞄准精度高，可自适应不同位置，画圆功能可实现
- **缺点**：成本略高，视觉处理有一定延迟

### 方案三：纯视觉导航 + 视觉瞄准
- **循迹**：摄像头识别轨迹线导航
- **瞄准**：同方案二
- **优点**：信息量大，可同时处理循迹和瞄准
- **缺点**：计算量大，实时性差，不符合MSPM0限制

### 最终选择：方案二
理由：
1. 红外循迹成熟可靠，响应速度快，满足20s内完成行驶的要求
2. OpenMV视觉瞄准可精确识别靶心位置，闭环控制精度高
3. 系统模块化程度高，调试方便
4. 成本适中，符合竞赛要求

## 四、核心模块选型

| 模块 | 选型 | 理由 |
|------|------|------|
| 主控MCU | TI MSPM0G3507 | 题目要求，ARM Cortex-M0+，80MHz |
| 视觉模块 | OpenMV4 H7 | 高帧率，内置颜色/形状识别，UART输出坐标 |
| 循迹传感器 | TCRT5000×8 | 响应快，检测距离适中，8路覆盖轨迹宽度 |
| 电机驱动 | TB6612FNG | 双路H桥，电流大，体积小，效率高 |
| 直流电机 | N20减速电机+编码器 | 体积小，扭矩足，编码器用于速度闭环 |
| 舵机 | SG90微型舵机×2 | 轻便，响应快，适合二维云台 |
| 激光笔 | 405nm蓝紫激光笔 | 符合题目要求，功率≤10mW |
| 显示 | 0.96寸OLED(I2C) | 低功耗，显示清晰 |
| 电源 | 7.4V锂电池(2S) | 双路独立开关供电 |

## 五、双电源独立控制设计
- **电源1**：7.4V → 5V(稳压) → MSPM0主控板 + 循迹传感器 + 电机驱动
- **电源2**：7.4V → 5V(稳压) → OpenMV + 舵机 + 激光笔
- 两路电源各自独立开关，用LED指示供电状态
