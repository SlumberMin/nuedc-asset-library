## 一、PID调参实战

### 1.1 PID基础原理

```
PID输出 = Kp*e(t) + Ki*∫e(t)dt + Kd*de(t)/dt

各参数作用：
- Kp（比例）：减小当前误差，响应快但可能振荡
- Ki（积分）：消除稳态误差，但会增加超调
- Kd（微分）：抑制变化趋势，减少超调和振荡
```

### 1.2 位置式PID

```c
typedef struct {
    float Kp, Ki, Kd;
    float setpoint;      // 目标值
    float integral;      // 积分累加
    float prev_error;    // 上次误差
    float output_min;    // 输出下限
    float output_max;    // 输出上限
    float integral_max;  // 积分限幅
} pid_t;

float pid_calculate(pid_t *pid, float measured)
{
    float error = pid->setpoint - measured;

    // 积分累加（带限幅）
    pid->integral += error;
    if (pid->integral > pid->integral_max) pid->integral = pid->integral_max;
    if (pid->integral < -pid->integral_max) pid->integral = -pid->integral_max;

    // 微分项
    float derivative = error - pid->prev_error;
    pid->prev_error = error;

    // PID输出
    float output = pid->Kp * error + pid->Ki * pid->integral + pid->Kd * derivative;

    // 输出限幅
    if (output > pid->output_max) output = pid->output_max;
    if (output < pid->output_min) output = pid->output_min;

    return output;
}
```

### 1.3 增量式PID（电赛最常用）

```c
typedef struct {
    float Kp, Ki, Kd;
    float setpoint;
    float prev_error;
    float prev_prev_error;
    float output;
    float output_min;
    float output_max;
} pid_incremental_t;

float pid_incremental_calc(pid_incremental_t *pid, float measured)
{
    float error = pid->setpoint - measured;

    float delta = pid->Kp * (error - pid->prev_error)
                + pid->Ki * error
                + pid->Kd * (error - 2 * pid->prev_error + pid->prev_prev_error);

    pid->prev_prev_error = pid->prev_error;
    pid->prev_error = error;

    pid->output += delta;

    // 输出限幅
    if (pid->output > pid->output_max) pid->output = pid->output_max;
    if (pid->output < pid->output_min) pid->output = pid->output_min;

    return pid->output;
}

// 初始化示例
pid_incremental_t motor_pid = {
    .Kp = 5.0f,
    .Ki = 0.3f,
    .Kd = 0.1f,
    .setpoint = 100.0f,  // 目标转速100 RPM
    .output_min = -1000.0f,
    .output_max = 1000.0f
};
```

### 1.4 PID调参步骤（经验法）

```
步骤1：先调Kp
- Ki=0, Kd=0
- 从小值开始增大Kp（如0.1）
- 观察响应：增大Kp直到系统出现等幅振荡
- 记录此时的Kp为Ku（临界增益）

步骤2：再调Kd
- 在Kp基础上加入Kd
- Kd从Kp/10开始
- 适当增大Kd可以减少超调和振荡
- 过大会导致响应变慢

步骤3：最后调Ki
- Ki从很小值开始（如Kp/100）
- Ki用于消除稳态误差
- 过大会导致积分饱和和超调

参数范围参考（直流电机速度环）：
- Kp: 1~50
- Ki: 0.01~1.0
- Kd: 0~5.0

参数范围参考（角度环/位置环）：
- Kp: 0.5~20
- Ki: 0.001~0.5
- Kd: 0.1~5.0
```

### 1.5 Ziegler-Nichols临界振荡法

```
1. 令Ki=0, Kd=0
2. 逐渐增大Kp，直到系统出现持续等幅振荡
3. 记录临界增益Ku和振荡周期Tu
4. 按下表计算PID参数：

| 类型 | Kp      | Ki        | Kd        |
|------|---------|-----------|-----------|
| P    | 0.5*Ku  | -         | -         |
| PI   | 0.45*Ku | 0.54*Ku/Tu| -         |
| PID  | 0.6*Ku  | 1.2*Ku/Tu | 0.075*Ku*Tu|
```

### 1.6 串级PID（角度+速度双环）

```c
// 外环：角度环（低频率，如100Hz）
// 内环：速度环（高频率，如1000Hz）

typedef struct {
    pid_t angle_pid;  // 外环
    pid_t speed_pid;  // 内环
} cascade_pid_t;

// 外环输出作为内环的设定值
float cascade_calculate(cascade_pid_t *cpid, float angle, float speed)
{
    // 外环：角度误差 → 速度设定值
    cpid->angle_pid.setpoint = 0.0f; // 目标角度
    float speed_ref = pid_calculate(&cpid->angle_pid, angle);

    // 内环：速度误差 → 电机输出
    cpid->speed_pid.setpoint = speed_ref;
    float output = pid_calculate(&cpid->speed_pid, speed);

    return output;
}
```

### 1.7 PID调试技巧

**积分抗饱和**
```c
// 方法1：积分限幅
if (pid->integral > INTEGRAL_MAX) pid->integral = INTEGRAL_MAX;
if (pid->integral < -INTEGRAL_MAX) pid->integral = -INTEGRAL_MAX;

// 方法2：条件积分（输出饱和时停止积分）
if (output < pid->output_max && output > pid->output_min) {
    pid->integral += error;
}

// 方法3：积分分离（误差大时取消积分）
if (fabs(error) > ERROR_THRESHOLD) {
    // 只用PD控制
    output = Kp * error + Kd * derivative;
} else {
    // 使用完整PID
    output = Kp * error + Ki * integral + Kd * derivative;
}
```

**微分滤波**
```c
// 微分项容易受噪声影响，加入低通滤波
float alpha = 0.1f; // 滤波系数
float derivative_raw = error - pid->prev_error;
pid->derivative_filtered = alpha * derivative_raw + (1 - alpha) * pid->derivative_filtered;
```

## 二、编码器调试

### 2.1 增量式编码器

```c
// STM32定时器编码器模式配置
void encoder_init(void)
{
    // TIM3配置为编码器模式
    TIM_Encoder_InitTypeDef encoder_config = {0};
    encoder_config.EncoderMode = TIM_ENCODERMODE_TI12; // 双通道4倍频
    encoder_config.IC1Polarity = TIM_ICPOLARITY_RISING;
    encoder_config.IC1Filter = 0x0F; // 输入滤波
    encoder_config.IC2Polarity = TIM_ICPOLARITY_RISING;
    encoder_config.IC2Filter = 0x0F;
    HAL_TIM_Encoder_Init(&htim3, &encoder_config);
    HAL_TIM_Encoder_Start(&htim3, TIM_CHANNEL_ALL);
}

// 读取编码器计数值
int32_t encoder_read(void)
{
    int16_t count = (int16_t)__HAL_TIM_GET_COUNTER(&htim3);
    __HAL_TIM_SET_COUNTER(&htim3, 0); // 清零
    return count;
}

// 计算转速（RPM）
float get_motor_rpm(void)
{
    static int32_t last_count = 0;
    static uint32_t last_tick = 0;

    int32_t count = encoder_read();
    uint32_t tick = HAL_GetTick();

    float dt = (tick - last_tick) / 1000.0f; // 秒
    if (dt < 0.001f) dt = 0.001f;

    // RPM = (脉冲数 / 每转脉冲数) / 时间(分钟)
    float rpm = (float)count / (ENCODER_PPR * 4) / (dt / 60.0f);

    last_count = count;
    last_tick = tick;

    return rpm;
}
```

### 2.2 编码器常见问题

**问题1：计数方向错误**
```
症状：电机正转时计数值为负
解决：
1. 交换A/B相接线
2. 或修改代码：count = -count;
3. 或修改定时器极性配置
```

**问题2：计数跳变/抖动**
```c
// 原因：信号干扰、滤波不足
// 解决1：硬件滤波（加RC滤波）
// 解决2：增加输入滤波器
encoder_config.IC1Filter = 0x0F; // 增大滤波值

// 解决3：软件中值滤波
int32_t encoder_median_filter(void)
{
    int32_t buf[5];
    for (int i = 0; i < 5; i++) {
        buf[i] = encoder_read();
        delay_us(10);
    }
    // 排序取中值
    sort(buf, 5);
    return buf[2];
}
```

**问题3：速度计算不平滑**
```c
// 使用滑动窗口平均滤波
#define SPEED_FILTER_SIZE 8
float speed_buffer[SPEED_FILTER_SIZE];
int speed_buf_idx = 0;

float get_filtered_rpm(void)
{
    float raw_rpm = get_motor_rpm();
    speed_buffer[speed_buf_idx] = raw_rpm;
    speed_buf_idx = (speed_buf_idx + 1) % SPEED_FILTER_SIZE;

    float sum = 0;
    for (int i = 0; i < SPEED_FILTER_SIZE; i++) {
        sum += speed_buffer[i];
    }
    return sum / SPEED_FILTER_SIZE;
}
```

## 三、电机驱动调试

### 3.1 直流电机驱动（L298N/TB6612）

```c
// TB6612驱动（推荐，效率高）
// AIN1/AIN2 控制方向，PWM控制速度

void motor_set_speed(int16_t speed)
{
    if (speed >= 0) {
        HAL_GPIO_WritePin(GPIOA, AIN1_PIN, GPIO_PIN_SET);
        HAL_GPIO_WritePin(GPIOA, AIN2_PIN, GPIO_PIN_RESET);
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, speed);
    } else {
        HAL_GPIO_WritePin(GPIOA, AIN1_PIN, GPIO_PIN_RESET);
        HAL_GPIO_WritePin(GPIOA, AIN2_PIN, GPIO_PIN_SET);
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, -speed);
    }
}

// PWM频率建议：10kHz~20kHz（避开人耳频率，减少噪声）
```

### 3.2 步进电机驱动（A4988/TMC2209）

```c
// A4988驱动
void stepper_move(int32_t steps, uint16_t speed_delay_us)
{
    // 设置方向
    HAL_GPIO_WritePin(GPIOA, DIR_PIN, steps > 0 ? GPIO_PIN_SET : GPIO_PIN_RESET);
    steps = abs(steps);

    for (int32_t i = 0; i < steps; i++) {
        HAL_GPIO_WritePin(GPIOA, STEP_PIN, GPIO_PIN_SET);
        delay_us(2); // 脉冲宽度
        HAL_GPIO_WritePin(GPIOA, STEP_PIN, GPIO_PIN_RESET);
        delay_us(speed_delay_us); // 控制速度
    }
}

// 微步设置（MS1/MS2/MS3引脚）
// 全步：MS1=L, MS2=L, MS3=L
// 1/2步：MS1=H, MS2=L, MS3=L
// 1/4步：MS1=L, MS2=H, MS3=L
// 1/8步：MS1=H, MS2=H, MS3=L
// 1/16步：MS1=H, MS2=H, MS3=H
```

### 3.3 无刷电机驱动（ESC电调）

```c
// ESC控制信号：50Hz PWM
// 脉宽 1000us = 最小油门（停止）
// 脉宽 1500us = 中位
// 脉宽 2000us = 最大油门

void esc_init(void)
{
    // 初始化时发送最小油门信号（ESC解锁）
    __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, 1000);
    HAL_Delay(2000); // 等待ESC初始化
}

void esc_set_throttle(uint16_t throttle_us)
{
    // 限幅
    if (throttle_us < 1000) throttle_us = 1000;
    if (throttle_us > 2000) throttle_us = 2000;
    __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, throttle_us);
}
```

### 3.4 电机驱动常见故障

| 故障现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 电机不转 | 驱动未使能/接线错误 | 检查EN引脚、接线 |
| 电机只往一个方向 | H桥半边损坏 | 更换驱动模块 |
| 电机抖动不转 | PWM频率太低/电流不足 | 提高PWM频率、增大电源电流 |
| 驱动芯片发烫 | 过流/短路 | 检查电流、加散热片 |
| 有嗡嗡声但不转 | PWM占空比太低 | 增大占空比（启动需要较大电流） |
| 速度不稳定 | 电源内阻大/编码器问题 | 加大电容、检查编码器 |
| 堵转时驱动保护 | 过流保护触发 | 限制最大电流、加电流检测 |

void motor_debug_output(float target, float actual, float output)

### 4.2 PID响应曲线分析

```
理想响应曲线：
设定值 ████████████████████████████████
        _____/‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
       /   ↑超调<5%
实际值/____↓

问题1：响应慢 → 增大Kp
问题2：超调大 → 增大Kd，减小Ki
问题3：振荡   → 减小Kp，增大Kd
问题4：稳态误差 → 适当增大Ki
问题5：积分饱和 → 加入积分限幅
```

### 4.3 电流检测与保护

```c
// 使用ACS712等电流传感器
float read_motor_current(void)
{
    uint16_t adc_val = HAL_ADC_GetValue(&hadc1);
    // ACS712-5A: 185mV/A, 零点2.5V
    float voltage = adc_val * 3.3f / 4096.0f;
    float current = (voltage - 2.5f) / 0.185f;
    return current;
}

// 限流保护
void motor_current_protect(float current_limit)
{
    float current = read_motor_current();
    if (fabs(current) > current_limit) {
        motor_set_speed(0); // 紧急停止
        printf("OVERCURRENT: %.2fA\n", current);
    }
}
```

## 五、实战调试案例

### 案例1：PID参数整定过程
```
目标：直流电机速度控制，目标100 RPM

第1次：Kp=1, Ki=0, Kd=0
结果：响应很慢，稳态误差大（实际80 RPM）

第2次：Kp=10, Ki=0, Kd=0
结果：响应变快，但有稳态误差（实际95 RPM）

第3次：Kp=10, Ki=0.5, Kd=0
结果：稳态误差消除，但有10%超调

第4次：Kp=10, Ki=0.5, Kd=2
结果：超调减小到3%，响应时间<0.5秒 ✓

最终参数：Kp=10, Ki=0.5, Kd=2
```

### 案例2：编码器抖动问题
```
症状：静止时编码器读数±1跳动
原因：信号线靠近电机电源线，电磁干扰
解决：
1. 编码器线改用屏蔽线
2. 增大定时器输入滤波值
3. 软件增加变化阈值（|变化|<3时忽略）
```

### 案例3：电机启动卡顿
```
症状：低速时电机不转，需要手动推一下才转
原因：PWM占空比太低，电机启动扭矩不足
解决：
1. 设置最小启动占空比（如15%）
2. 启动时短时间施加高占空比（启动脉冲）
3. 使用带减速箱的电机，提高扭矩
```
