## 1. 优化目标

| 指标 | 优化前 | 优化后 |

## 2. CPU核心隔离

### 2.1 内核启动参数 (修改 /boot/orangepiEnv.txt)

```bash
# 为RT进程隔离CPU核心
# A76大核: CPU4-7, A55小核: CPU0-3
# 隔离CPU6-7给实时控制和视觉
extraargs=isolcpus=6,7 nohz_full=6,7 rcu_nocbs=6,7 kthread_cpus=0-5
```

修改后更新引导：
```bash
sudo update-initramfs -u
sudo reboot
```

### 2.2 运行时CPU亲和性设置

```bash
# 视觉进程绑定到CPU6
taskset -c 6 ./vision_process

# 控制进程绑定到CPU7
taskset -c 7 ./control_process

# 人机交互进程使用小核
taskset -c 0-3 ./hmi_process
```

### 2.3 Python中设置CPU亲和

```python
import os

def set_cpu_affinity(pid, cores):
    """设置进程CPU亲和性"""
    os.sched_setaffinity(pid, set(cores))

# 视觉进程绑定CPU6
set_cpu_affinity(0, [6])

# 控制进程绑定CPU7
set_cpu_affinity(0, [7])
```

## 3. 进程优先级设置

### 3.1 SCHED_FIFO实时调度

```bash
# 控制进程 - 最高优先级
sudo chrt -f 90 ./control_process

# 视觉进程 - 高优先级
sudo chrt -f 50 ./vision_process

# 人机交互 - 普通优先级
nice -n 10 ./hmi_process
```

### 3.2 Python中设置实时调度

```python
import os
import sched

def set_rt_priority(priority=80):
    """设置SCHED_FIFO实时调度"""
    param = os.sched_param(priority)
    try:
        os.sched_setscheduler(0, os.SCHED_FIFO, param)
        print(f"已设置SCHED_FIFO优先级: {priority}")
    except PermissionError:
        print("需要root权限: sudo python3 script.py")

# 控制进程: 优先级90 (最高)
set_rt_priority(90)
```

### 3.3 优先级规划

| 进程 | 调度策略 | 优先级 | 说明 |
|------|----------|--------|------|
| 控制进程 | SCHED_FIFO | 90 | 最高，1ms周期 |
| 视觉进程 | SCHED_FIFO | 50 | 高，30fps |
| 人机交互 | SCHED_OTHER | 0 | 普通，不抢RT |
| 内核线程 | - | - | 保留在非隔离核 |

## 4. 关闭无用服务

### 4.1 系统服务清理脚本

```bash
#!/bin/bash
# disable_services.sh - 关闭电赛不需要的系统服务

SERVICES=(
    bluetooth.service
    cups.service
    cups-browsed.service
    avahi-daemon.service
    ModemManager.service
    NetworkManager-wait-online.service
    accounts-daemon.service
    polkit.service
    rsyslog.service
    snapd.service
    multipathd.service
    thermald.service
    whoopsie.service
    apport.service
)

for svc in "${SERVICES[@]}"; do
    if systemctl is-active --quiet "$svc" 2>/dev/null; then
        sudo systemctl stop "$svc"
        sudo systemctl disable "$svc"
        echo "已禁用: $svc"
    fi
done

# 关闭不必要的内核模块
sudo modprobe -r bluetooth
sudo modprobe -r btusb
echo "服务清理完成"
```

### 4.2 关闭不必要的内核功能

```bash
# /etc/sysctl.d/99-realtime.conf

# 关闭透明大页 (减少延迟抖动)
echo never | sudo tee /sys/kernel/mm/transparent_hugepage/enabled

# 网络优化 (减少中断)

net.core.wmem_max = 262144

# 关闭NMI watchdog
kernel.nmi_watchdog = 0
```

## 5. 内存优化

### 5.1 锁定内存 (防止换页)

```python
import mmap
import ctypes

def lock_memory():
    """锁定进程内存，防止被换出到swap"""
    libc = ctypes.CDLL('libc.so.6')
    # MCL_CURRENT: 锁定当前所有页面
    # MCL_FUTURE: 锁定未来映射的页面
    libc.mlockall(3)  # MCL_CURRENT | MCL_FUTURE

# 在实时进程启动时调用
lock_memory()
```

### 5.2 预分配内存

```c
// 避免运行时内存分配 (延迟抖动)
// 在初始化阶段预分配所有需要的内存
static vision_result_t g_vision_result __attribute__((aligned(64)));
static control_state_t g_control_state __attribute__((aligned(64)));
```
