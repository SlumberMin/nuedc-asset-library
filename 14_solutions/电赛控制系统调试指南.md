# 电赛控制系统调试指南

## 一、PID基础回顾

### 1.1 PID公式

**位置式PID**：
```
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
```

**增量式PID**（更适合嵌入式）：
```
Δu = Kp·(e[n]-e[n-1]) + Ki·e[n] + Kd·(e[n]-2e[n-1]+e[n-2])
u[n] = u[n-1] + Δu
```

### 1.2 各参数作用

| 参数 | 作用 | 增大的效果 | 减小的效果 |
|------|------|------------|------------|
| Kp | 比例控制 | 响应快，但可能振荡 | 响应慢，稳态误差大 |
| Ki | 积分控制 | 消除稳态误差 | 稳态误差消除慢 |
| Kd | 微分控制 | 抑制超调，增强阻尼 | 超调增大，振荡增多 |

## 二、PID调参流程（工程法）

### 2.1 Ziegler-Nichols临界比例法

**步骤**：

```
1. 令 Ki=0, Kd=0，从小到大增加Kp
2. 当系统出现等幅持续振荡时，记录：
   - 临界增益 Ku（此时的Kp）
   - 振荡周期 Tu
3. 按下表计算PID参数
```

| 控制器类型 | Kp | Ki | Kd |
|-----------|-----|-----|-----|
| P | 0.5Ku | - | - |
| PI | 0.45Ku | 0.54Ku/Tu | - |
| PID | 0.6Ku | 1.2Ku/Tu | 0.075Ku·Tu |

### 2.2 手动调参法（推荐比赛使用）

```
第一步：纯P控制
  - 从小Kp开始，逐步增大
  - 直到系统响应足够快但刚开始出现轻微振荡
  - 记录此时Kp，取70-80%作为工作Kp

第二步：加入D
  - 从小Kd开始增大
  - 观察超调是否减小，振荡是否被抑制
  - Kd过大会导致高频噪声放大

第三步：加入I（如需要）
  - 从小Ki开始
  - 观察稳态误差是否消除
  - Ki过大会导致积分饱和、低频振荡

第四步：微调
  - 反复调整三个参数达到最佳效果
```

### 2.3 常见系统调参经验值

| 控制对象 | Kp范围 | Ki范围 | Kd范围 | 控制周期 |
|----------|--------|--------|--------|----------|
| 直流电机速度 | 0.5-5 | 0.1-1 | 0-0.5 | 10-20ms |
| 舵机角度 | 1-10 | 0-0.5 | 0.5-3 | 20ms |
| 平衡车直立 | 50-200 | 0-1 | 10-50 | 5-10ms |
| 温度控制 | 2-10 | 0.5-2 | 1-5 | 1000ms |
| 巡线差速 | 0.3-2 | 0-0.3 | 0.1-1 | 20-50ms |

## 三、阶跃响应测试

### 3.1 测试方法

```c
// 给系统施加一个阶跃输入，记录输出响应
void step_response_test(void) {
    // 设定目标值突变
    set_target(100);  // 从0跳到100

    // 以固定周期采样记录
    for (int i = 0; i < 500; i++) {
        float actual = read_sensor();
        float error = target - actual;
        float output = pid_calculate(error);

        set_actuator(output);
        record_data(i * dt, target, actual, output);  // 串口输出
        delay_ms(SAMPLE_PERIOD);
    }
}
```

### 3.2 阶跃响应指标

```
           目标值 ─────────────────────
                  ╱‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
                 ╱      超调量
          ╱────╱  ───────────
         ╱   ↑
        ╱    │  上升时间Tr
       ╱     │
  ────╱      │
      ↑      ↑        ↑
    延迟Td  上升时间  调节时间Ts
```

| 指标 | 定义 | 理想范围 |
|------|------|----------|
| 上升时间 Tr | 10%→90%目标值 | 越短越好 |
| 超调量 Mp | 峰值超出目标的百分比 | < 5-15% |
| 调节时间 Ts | 进入±2%误差带的时间 | < 目标响应时间 |
| 稳态误差 | 最终与目标的偏差 | = 0 |

## 四、频率响应测试

### 4.1 正弦扫频测试

```c
// 施加不同频率的正弦输入，记录幅值衰减和相位偏移
void frequency_sweep_test(float f_start, float f_end, float amplitude) {
    for (float freq = f_start; freq <= f_end; freq *= 1.2) {
        // 施加正弦信号10个周期
        for (int cycle = 0; cycle < 10; cycle++) {
            for (int i = 0; i < SAMPLES_PER_CYCLE; i++) {
                float t = i / (float)SAMPLES_PER_CYCLE / freq;
                float input = amplitude * sin(2 * PI * freq * t);
                float output = read_system_output();
                record(freq, t, input, output);
                set_and_wait(input);
            }
        }
    }
}
```

### 4.2 Bode图分析

从扫频数据提取：
- **幅频特性**：每个频率点的输出/输入幅值比（dB）
- **相频特性**：输出相对输入的相位滞后

关键指标：
- **带宽**：-3dB处的频率，决定系统响应速度
- **相位裕度**：增益交叉频率处的相位与-180°的差值
- **增益裕度**：相位交叉频率处增益与0dB的差值

| 稳定性裕度 | 最小值 | 推荐值 |
|-----------|--------|--------|
| 相位裕度 | > 0° | 30°-60° |
| 增益裕度 | > 0dB | 6-12dB |

## 五、稳定性分析

### 5.1 常见不稳定现象及对策

| 现象 | 原因 | 对策 |
|------|------|------|
| 等幅振荡 | Kp过大，阻尼不足 | 减小Kp或增大Kd |
| 发散振荡 | 系统不稳定 | 大幅减小增益，检查采样率 |
| 低频振荡 | Ki过大，积分饱和 | 减小Ki，加积分限幅 |
| 高频抖动 | Kd过大，噪声放大 | 减小Kd，加低通滤波 |
| 稳态误差 | Kp不够大或缺积分 | 适当增大Ki |
| 响应太慢 | 增益过小 | 适当增大Kp |

### 5.2 积分抗饱和（Anti-Windup）

```c
typedef struct {
    float Kp, Ki, Kd;
    float integral;
    float integral_limit;  // 积分限幅
    float output_limit;    // 输出限幅
    float prev_error;
} PID_t;

float pid_calculate(PID_t *pid, float error) {
    // P
    float p_term = pid->Kp * error;

    // I (带抗饱和)
    pid->integral += error * dt;
    if (pid->integral > pid->integral_limit)
        pid->integral = pid->integral_limit;
    if (pid->integral < -pid->integral_limit)
        pid->integral = -pid->integral_limit;
    float i_term = pid->Ki * pid->integral;

    // D (带低通滤波)
    float derivative = (error - pid->prev_error) / dt;
    float d_term = pid->Kd * derivative;
    pid->prev_error = error;

    // 输出限幅
    float output = p_term + i_term + d_term;
    if (output > pid->output_limit) output = pid->output_limit;
    if (output < -pid->output_limit) output = -pid->output_limit;

    return output;
}
```

### 5.3 不完全微分（抑制噪声）

```c
// 一阶低通滤波微分
float filtered_derivative = alpha * (error - prev_error) / dt
                         + (1 - alpha) * prev_filtered_derivative;
// alpha = dt / (dt + Tf), Tf为滤波时间常数
```

## 六、串级PID设计

### 6.1 典型串级结构

```
外环(位置/角度)  ──→  内环(速度/角速度)  ──→  执行器
  慢周期(20ms)         快周期(5-10ms)         电机/舵机
```

### 6.2 调参顺序

```
1. 先调内环（关掉外环，手动给内环设定值）
   - 内环要求：快速响应、无超调或小超调
2. 再调外环（内环已闭合，调节外环参数）
   - 外环参数通常比内环小很多
3. 联调优化
```

## 七、调试检查清单

- [ ] 控制周期固定且稳定（用示波器验证）
- [ ] 传感器数据已滤波（滑动平均/中值滤波/卡尔曼）
- [ ] 执行器输出有限幅保护
- [ ] PID输出方向正确（正误差→正输出）
- [ ] 积分有抗饱和处理
- [ ] 微分有低通滤波
- [ ] 已记录阶跃响应曲线并分析指标
- [ ] 极端条件测试（满载、堵转、传感器异常）
- [ ] 串口调试输出正常，方便赛后分析
