# 电赛评分标准详解

> 逐项解析评分规则，基于国奖作品反推得分策略

---

## 一、评分体系总览

### 1.1 分值构成

```
总分 = 基本要求（50分）+ 发挥部分（30分）+ 设计报告（20分）= 100分

国奖门槛：通常需要 85分以上
省一等奖：通常需要 70分以上
```

### 1.2 评审流程

```
第一轮：作品测试（占60-70%权重）
  └── 专家现场测试，逐项打分

第二轮：报告评审（占20-30%权重）
  └── 专家组盲审设计报告

第三轮：答辩（部分赛区，占10%权重）
  └── 5-10分钟展示+提问
```

---

## 二、基本要求评分详解（50分）

### 2.1 评分原则

```
基本要求是"及格线"，必须全部完成：
- 每项基本要求通常占8-12分
- 完成得满分，未完成按比例扣分
- 基本要求不达标，发挥部分再好也难以获奖
```

### 2.2 典型评分项解析

#### 以2023年电源题为例

| 评分项 | 分值 | 评分标准 | 得分策略 |
|--------|------|----------|----------|
| 输出电压精度 | 10分 | 误差≤1%得满分，每超0.5%扣2分 | 预留校准余量 |
| 负载调整率 | 10分 | ≤1%满分，每超0.5%扣2分 | 加反馈补偿 |
| 纹波电压 | 10分 | ≤50mV满分 | 选好滤波方案 |
| 效率 | 10分 | ≥85%满分 | 选低损耗器件 |
| 保护功能 | 10分 | 过流保护动作正常 | 必须实现 |

#### 以2023年信号题为例

| 评分项 | 分值 | 评分标准 | 得分策略 |
|--------|------|----------|----------|
| 频率精度 | 10分 | 误差≤0.1%满分 | 高精度晶振 |
| 幅度精度 | 10分 | 误差≤1%满分 | DAC校准 |
| 失真度 | 10分 | THD≤1%满分 | 滤波+线性功放 |
| 噪声 | 10分 | 信噪比≥60dB满分 | 布局+屏蔽 |
| 响应速度 | 10分 | 切换时间≤1s满分 | 快速锁定算法 |

### 2.3 得分关键

```
1. 量化指标必须留余量
   要求≤1% → 实际做到≤0.5%
   要求≤50mV → 实际做到≤30mV

2. 测试条件要覆盖
   要求1V~5V输出 → 1V, 2V, 3V, 4V, 5V都要测
   要求不同负载 → 轻载, 满载, 突变都要测

3. 稳定性要好
   连续运行30分钟不漂移
   温度变化时指标不退化
```

## 三、发挥部分评分详解（30分）

### 3.1 评分原则

```
发挥部分是"拉分项"，决定奖项等级：
- 通常3-4个发挥指标
- 每项5-10分
- 完成越多越好，但质量比数量重要
```

### 3.2 常见发挥部分类型

#### 性能提升型

```
例：基本要求输出1V-5V，发挥要求1V-10V
策略：拓扑不变，调整参数范围
难度：★★☆
```

#### 功能扩展型

```
例：基本要求恒压输出，发挥要求恒流模式
策略：增加控制环路切换
难度：★★★
```

#### 创新设计型

```
例：基本要求固定频率，发挥要求频率可调
策略：DDS+PLL方案
难度：★★★★
```

### 3.3 发挥部分得分策略

```
优先级排序：
1. 先确保基本要求满分（50分稳拿）
2. 选择2个最容易的发挥项完成（15-20分）
3. 有余力再做第3个发挥项
4. 最后考虑最难的发挥项

时间分配：
基本要求完成前：不碰发挥部分
基本要求稳定后：集中攻关发挥部分
最后6小时：放弃未完成的发挥项，确保已完成项稳定
```

## 四、设计报告评分详解（20分）

### 4.1 报告评分标准

| 评分项 | 分值 | 评分要点 |
|--------|------|----------|
| 方案论证 | 5分 | 方案比较有理有据，选择合理 |
| 理论分析 | 3分 | 关键公式推导正确，计算无误 |
| 电路设计 | 4分 | 原理图完整，参数标注清楚 |
| 程序设计 | 3分 | 流程图清晰，关键代码有注释 |
| 测试数据 | 3分 | 数据完整，表格规范 |
| 总结分析 | 2分 | 分析到位，改进方向明确 |

### 4.2 报告得分要点

#### 方案论证（5分）- 最容易拿分

```
必须包含：
1. 至少2种方案的对比
2. 每种方案的优缺点分析
3. 最终选择的理由
4. 方案框图

高分写法：
"方案一：采用XX方案，优点是...，缺点是...
 方案二：采用YY方案，优点是...，缺点是...
 综合考虑精度、成本和实现难度，选择方案二"
```

#### 理论分析（3分）

```
必须包含：
1. 核心公式推导
2. 关键参数计算过程
3. 误差分析

高分写法：
给出完整的公式推导过程，而不是直接给出结果
例：增益 = R2/R1 = 10k/1k = 10（带单位和计算过程）
```

#### 测试数据（3分）

```
必须包含：
1. 完整的测试条件说明
2. 表格化数据（非手写）
3. 与指标要求的对比
4. 误差计算

高分写法：
表格+图表双重展示
每组数据重复测量3次取平均值
明确标注是否满足指标要求
```

### 4.3 报告常见扣分点

```
× 没有方案论证，直接写电路设计
× 公式只有结论没有推导
× 测试数据不完整（只测了部分点）
× 图表不清晰（分辨率低/标注不清）
× 没有误差分析
× 总结敷衍（"基本完成了题目要求"）
```

## 五、国奖得分策略

### 5.1 85分得分路线图

```
基本要求：争取48-50分（留1-2分余量）
  ├── 所有量化指标超标完成
  ├── 测试条件全覆盖
  └── 稳定性好

发挥部分：争取20-25分
  ├── 完成2-3个发挥项
  ├── 选择性价比高的发挥项
  └── 确保已完成项稳定

设计报告：争取17-20分
  ├── 方案论证完整
  ├── 理论推导清晰
  ├── 测试数据详实
  └── 排版规范专业
```

### 5.2 不同奖项的得分门槛

```
国一：85-95分（基本满分 + 发挥大部分完成 + 报告优秀）
国二：75-85分（基本基本满分 + 发挥部分完成一半 + 报告良好）
省一：65-75分（基本要求完成 + 发挥部分少量完成 + 报告合格）
省二：55-65分（基本要求大部分完成 + 报告基本合格）
```

### 5.3 评委心理分析

```
评委关注点：
1. 基本要求是否全部完成（第一印象）
2. 指标是否超标（区分度）
3. 稳定性如何（可靠性）
4. 报告是否专业（规范性）
5. 有无创新亮点（加分项）

评委忌讳：
1. 基本要求没完成就做发挥部分
2. 测试时频繁出错需要重启
3. 数据造假或明显不合理
4. 报告格式混乱、错别字多
```
