## 一、电源题分类与历年真题

### 1.1 常见题型

| 类型 | 典型题目 | 核心拓扑 |
|------|---------|---------|
| DC-DC | 可调稳压电源、Boost升压 | Buck/Boost/Buck-Boost |
| 逆变 | 单相/三相逆变器、UPS | 全桥/半桥逆变 |

### 1.2 评分要点

- **效率**（通常要求 >85%，优秀 >90%）
- **输出精度**（电压精度 ±1%，纹波 <50mV）
- **响应速度**（负载突变恢复时间 <100ms）
- **功率因数**（PFC题要求 >0.98）

## 二、DC-DC变换器

### 2.1 Buck降压电路

**拓扑：** Vin → MOSFET → L → C → Vout，续流二极管

**设计公式：**
```
占空比：D = Vout / Vin
电感：L ≥ (Vin - Vout) × D / (f × ΔI_L)
电容：C ≥ ΔI_L / (8 × f × ΔV_out)
纹波：ΔV_out = ΔI_L / (8 × f × C)
```

**参数计算示例（12V→5V/3A，f=200kHz）：**
```
D = 5/12 = 0.417
取ΔI_L = 30% × 3A = 0.9A
L ≥ (12-5) × 0.417 / (200000 × 0.9) = 16.2μH → 选22μH
C ≥ 0.9 / (8 × 200000 × 0.05) = 11.25μF → 选22μF陶瓷×2
```

**常用芯片：**
| 芯片 | 输入范围 | 输出 | 特点 |
|------|---------|------|------|
| TPS5430 | 5.5~36V | 3A | 简单可靠 |
| LM2596 | 4.5~40V | 3A | 便宜易用 |
| MP2307 | 4.75~23V | 3A | 小封装 |
| XL4015 | 8~36V | 5A | 大电流 |

### 2.2 Boost升压电路

**设计公式：**
```
占空比：D = 1 - Vin/Vout
电感：L ≥ Vin × D / (f × ΔI_L)
输出电容：C ≥ Iout × D / (f × ΔV_out)
```

**注意：** Boost电路RHP零点限制带宽，不能太快！

**Type III补偿网络设计：**
```
f_cross = f_RHPZ / 3 = Vin² / (2π × L × Iout_max × 3)
```

### 2.3 Buck-Boost（负压输出）

```
Vout = -Vin × D / (1-D)
```

适合需要负压供电的场景（运放双电源）。

### 2.4 同步整流提高效率

```c
/* 用MOSFET替代二极管，降低导通损耗 */
// 上管：PWM驱动
// 下管：互补PWM（加死区）
// 死区时间：50~100ns

void PWM_Config(void) {
    // TIM1 CH1 和 CH1N 互补输出
    TIM_BDTRInitTypeDef bdtr;
    bdtr.TIM_DeadTime = 80;  // ~100ns @ 72MHz
    bdtr.TIM_Break = TIM_Break_Enable;
    TIM_BDTRConfig(TIM1, &bdtr);
}
```

### 2.5 电感选型

| 参数 | 选择依据 |
|------|---------|
| 电感值 | 由纹波电流决定 |
| 饱和电流 | ≥ 1.3 × I_peak |
| DCR（直流电阻） | 越小效率越高，<50mΩ |
| 磁芯材质 | 铁硅铝（低损耗）或MPP |

## 三、逆变器设计

### 3.1 单相全桥逆变

**拓扑：** H桥 + LC滤波

```
       S1  S3
  Vdc ─┤   ├─ L ─ C ─ Vout
       S2  S4
```

**LC滤波器设计：**
```
截止频率：fc = 1 / (2π√(LC))
取 fc = 10 × f基波，fc < f开关/10

例：50Hz输出，20kHz开关
fc = 500Hz → L = 1mH, C = 100μF
```

### 3.2 SPWM调制

```c
/* 正弦PWM生成 */
#define PWM_PERIOD 3600  // 72MHz / 20kHz = 3600
#define SINE_TABLE_SIZE 200

uint16_t sine_table[SINE_TABLE_SIZE];

void Generate_Sine_Table(void) {
    for (int i = 0; i < SINE_TABLE_SIZE; i++) {
        sine_table[i] = (uint16_t)(PWM_PERIOD * 0.5f * 
                        (1.0f + sinf(2.0f * 3.14159f * i / SINE_TABLE_SIZE)));
    }
}

volatile uint16_t sine_idx = 0;
void TIM1_UP_IRQHandler(void) {
    uint16_t pwm = sine_table[sine_idx];
    TIM1->CCR1 = pwm;
    TIM1->CCR2 = PWM_PERIOD - pwm;  // 互补
    sine_idx = (sine_idx + 1) % SINE_TABLE_SIZE;
}
```

### 3.3 闭环控制（电压外环 + 电流内环）

```c
/* 双环PI控制 */
typedef struct {
    float kp, ki;
    float integral;
    float out_min, out_max;
} PI_Controller;

float PI_Update(PI_Controller *pi, float ref, float fb) {
    float err = ref - fb;
    pi->integral += pi->ki * err;
    if (pi->integral > pi->out_max) pi->integral = pi->out_max;
    if (pi->integral < pi->out_min) pi->integral = pi->out_min;
    float out = pi->kp * err + pi->integral;
    if (out > pi->out_max) out = pi->out_max;
    if (out < pi->out_min) out = pi->out_min;
    return out;
}

/* 逆变器控制 */
PI_Controller v_loop = {0.5, 10, 0, -1, 1};   // 电压环
PI_Controller i_loop = {2.0, 50, 0, -1, 1};   // 电流环

void Inverter_Control(void) {
    float v_ref = 311.0f;  // 220Vrms = 311V峰值
    float v_out = ADC_Read_Voltage();
    float i_out = ADC_Read_Current();
    
    // 电压外环（输出调制波幅值）
    float mod = PI_Update(&v_loop, v_ref, v_out);
    
    // 电流内环（输出PWM占空比）
    float duty = PI_Update(&i_loop, mod * sin_ref, i_out);
    
    Set_PWM_Duty(duty);
}
```

## 四、功率因数校正（PFC）

### 4.1 为什么需要PFC？

整流桥+电容的输入电流是尖脉冲，功率因数只有0.5~0.7，谐波污染严重。

### 4.2 Boost-PFC原理

```
AC → 整流桥 → Boost → DC输出
                ↑
          PFC控制（使输入电流跟踪输入电压波形）
```

### 4.3 PFC控制实现

```c
/* 基于乘法器的PFC控制 */
float PFC_Control(float v_dc, float v_dc_ref, float v_ac_rect, float i_ac) {
    /* 电压环（慢环，10~20Hz带宽） */
    float err_v = v_dc_ref - v_dc;
    static float integ_v = 0;
    integ_v += 0.01f * err_v;  // 很小的Ki
    float vm = 0.5f * err_v + integ_v;
    if (vm < 0) vm = 0;
    
    /* 乘法器：产生电流参考 */
    float v_ac_norm = v_ac_rect / (peak_detector + 0.01f);  // 归一化
    float i_ref = vm * v_ac_norm;
    
    /* 电流环（快环） */
    float err_i = i_ref - i_ac;
    static float integ_i = 0;
    integ_i += 0.5f * err_i;
    float duty = 2.0f * err_i + integ_i;
    
    // 限幅
    if (duty > 0.95f) duty = 0.95f;
    if (duty < 0.05f) duty = 0.05f;
    
    return duty;
}
```

### 4.4 PFC关键参数设计

```
升压电感：L = V²_in_rms × η / (2 × P_out × f_sw × Δi_ratio)
典型值：500μH ~ 2mH

输出电容：C = P_out / (2 × f_line × ΔV_out × V_out)
典型值：220μF ~ 1000μF（450V耐压）

开关频率：65kHz ~ 150kHz
```

### 4.5 常用PFC芯片

| 芯片 | 厂商 | 特点 |
|------|------|------|
| UCC28019 | TI | 连续导通模式，简单 |
| L6562 | ST | 临界导通模式，高效 |
| NCP1654 | ON | 高集成度 |
| ICE3PCS01G | Infineon | 数字控制 |

## 五、PCB布局要点

### 5.1 功率环路最小化

```
关键环路（必须最小化面积）：
  Buck:  Vin → MOSFET → L → C → GND → Vin
  Boost: MOSFET → L → D → C → MOSFET
  
原则：高频电流环路面积最小
```

### 5.2 布局Checklist

- [ ] 输入电容紧贴MOSFET D极和S极
- [ ] 续流二极管/MOS管紧贴开关管
- [ ] 电感远离敏感信号线
- [ ] 采样电阻用Kelvin接法（四线制）
- [ ] 大电流走线 ≥ 1mm/A（铜厚1oz）
- [ ] 功率地和信号地单点连接
- [ ] 散热焊盘打过孔阵列

### 5.3 去耦电容配置

```
MOSFET驱动VCC：100nF陶瓷 + 10μF电解
IC供电：100nF陶瓷（紧贴VCC引脚）
ADC参考：1μF + 100nF
```

## 七、效率优化技巧

### 7.1 开关损耗

```
P_sw = 0.5 × V × I × (t_rise + t_fall) × f_sw

降低方法：
1. 降低开关频率（但电感体积增大）
2. 用更快的驱动（增大驱动电流）
3. ZVS/ZCS软开关
```

### 7.2 导通损耗

```
P_cond = I²_rms × Rds(on)

降低方法：
1. 选低Rds(on)的MOS管
2. 并联MOS管
3. 同步整流
```

### 7.3 磁芯损耗

```
P_core ∝ f^α × B^β  （α≈1.5, β≈2.5）

降低方法：
1. 选低损耗磁芯（铁硅铝 > 铁氧体）
2. 降低纹波电流
3. 适当增大电感量
```
