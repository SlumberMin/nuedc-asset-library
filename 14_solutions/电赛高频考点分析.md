## 二、高频考点TOP10

### 🥇 TOP1：信号放大与调理（出现率85%）

**考查形式：**
- 微弱信号放大（μV~mV级）
- 宽带放大器（DC~数十MHz）
- 低噪声放大器设计
- 自动增益控制(AGC)

**必备知识：**
| 知识点 | 详细内容 |
|--------|----------|
| 运放选型 | 噪声、带宽、压摆率、输入偏置 |
| 增益分配 | 前级高增益利于信噪比、级间匹配 |
| 噪声分析 | 热噪声、散粒噪声、1/f噪声 |
| 阻抗匹配 | 50Ω射频、1MΩ高阻、差分 |
| 滤波器 | 低通/带通/带阻设计 |

**应对策略：**
1. 提前准备2~3种运放组合方案（OPA847/AD8067/THS3091）
2. 熟记噪声计算公式：Vn_total = √(Vn1² + Vn2² + ...)
3. 掌握多级增益分配原则：前级增益尽量大
4. 准备好常用滤波器参数计算模板

---

### 🥈 TOP2：电源设计与功率变换（出现率75%）

**考查形式：**
- Buck/Boost变换器
- 双向DC-DC
- 逆变器(DC-AC)
- 稳压恒流源
- 无线电能传输

**必备知识：**
| 知识点 | 详细内容 |
|--------|----------|
| 拓扑选择 | Buck降压、Boost升压、Buck-Boost |
| 参数计算 | 电感值、电容值、开关频率 |
| 控制方法 | 电压环PID、电流环PID |
| 效率优化 | 同步整流、软开关 |
| 磁性元件 | 电感选型、变压器设计 |

**应对策略：**
1. 准备LM2596/TPS5430等常用电源模块原理图
2. 熟练掌握电感电容计算公式
3. 准备好电流采样电路（霍尔/采样电阻+运放）
4. 掌握MOS管驱动电路设计（IR2104/IR2110）

---

### 🥉 TOP3：电机控制（出现率70%）

**考查形式：**
- 直流电机调速
- 平衡车/倒立摆
- 舵机控制
- 步进电机定位
- BLDC控制

**必备知识：**
| 知识点 | 详细内容 |
|--------|----------|
| 驱动电路 | H桥(L298N/TB6612)、半桥(IR2104) |
| 调速方式 | PWM调速、电压调速 |
| 反馈元件 | 编码器、电位器、MPU6050 |
| 控制算法 | PID串级、LQR |
| 保护措施 | 过流保护、刹车、软启动 |

**应对策略：**
1. 提前调试好编码器读取代码（TIM编码器模式）
2. 准备PID算法库（位置式/增量式/串级）
3. 调试好MPU6050/MPU6500 DMP姿态解算
4. H桥驱动电路提前焊接调试

## 三、各方向核心速查

### 3.1 信号方向必背公式

| 公式 | 含义 | 典型值 |
|------|------|--------|
| Vn = √(4kTRB) | 热噪声 | 1kΩ, 1MHz → 4μV |
| In = √(2qIB) | 散粒噪声 | 1mA, 1MHz → 0.57μA |
| SNR = 20lg(Vs/Vn) | 信噪比 | >40dB为好 |
| NF = SNRin - SNRout | 噪声系数 | <3dB为好 |
| BW = GBW/G | 带宽计算 | 增益越大带宽越小 |
| SR = 2πfVp | 压摆率需求 | 10MHz, 5V → 314V/μs |

### 3.2 电源方向必背公式

| 公式 | Buck | Boost |
|------|------|-------|
| Vout/Vin | D | 1/(1-D) |
| ΔIL | (Vin-Vout)·D/(L·f) | Vin·D/(L·f) |
| Lmin | (Vin-Vout)·D/(ΔIL·f) | Vin·D/(ΔIL·f) |
| Cmin | ΔIL/(8·f·ΔVout) | Iout·D/(f·ΔVout) |
| 效率η | Pout/Pin × 100% | 一般85%~95% |

### 3.3 控制方向必背知识

| 内容 | 要点 |
|------|------|
| PID离散化 | 位置式/增量式公式 |
| 抗积分饱和 | 积分限幅、条件积分 |
| 串级PID | 内环带宽 > 外环带宽×5 |
| 编码器 | 四倍频、方向判断 |
| MPU6050 | DMP姿态解算、互补滤波 |
