# Orange Pi 5 电赛系统总体架构

## 1. 硬件平台

| 项目 | 规格 |
|------|------|
| 主控 | Orange Pi 5 (RK3588S) |
| CPU | 4×A76 + 4×A55 (big.LITTLE) |
| GPU | Mali-G610 MP4 (支持NPU) |
| NPU | 6 TOPS |
| RAM | 8/16 GB LPDDR4x |
| 存储 | eMMC / TF卡 |
| 摄像头 | USB摄像头 / MIPI CSI |

## 2. 三进程架构

```
┌─────────────────────────────────────────────────────┐
│                   Orange Pi 5                        │
│                                                      │
│  ┌──────────┐  共享内存   ┌──────────┐              │
│  │ 视觉进程 │ ◄════════► │ 控制进程 │              │
│  │ (A76×2)  │  < 0.5ms   │ (A76×2)  │              │
│  └────┬─────┘            └────┬─────┘              │
│       │                       │                     │
│       │  共享内存              │  UART/SPI          │
│       │                       │                     │
│  ┌────┴─────┐            ┌────┴─────┐              │
│  │ 摄像头   │            │ 执行机构 │              │
│  │ 驱动     │            │ (电机等) │              │
│  └──────────┘            └──────────┘              │
│       ▲                                             │
│       │ 共享内存                                     │
│  ┌────┴─────┐                                      │
│  │ 人机交互 │                                      │
│  │ (A55×2)  │                                      │
│  └──────────┘                                      │
└─────────────────────────────────────────────────────┘
```

### 2.1 视觉进程 (Vision Process)

- **核心任务**：图像采集、目标检测、坐标解算
- **CPU绑定**：A76大核 2-3
- **优先级**：RT_FIFO 50
- **帧率目标**：≥30fps (检测) / ≥60fps (采集)
- **关键模块**：
  - 摄像头采集线程
  - OpenCV图像预处理
  - YOLOv8/模板匹配推理
  - 目标坐标输出

### 2.2 控制进程 (Control Process)

- **核心任务**：运动规划、PID控制、PWM输出
- **CPU绑定**：A76大核 0-1
- **优先级**：RT_FIFO 80 (最高)
- **控制周期**：1ms (1kHz)
- **关键模块**：
  - 状态机调度
  - PID控制器 (位置环/速度环)
  - 轨迹规划器
  - 电机PWM/SPI驱动

### 2.3 人机交互进程 (HMI Process)

- **核心任务**：显示、按键、调试、参数调优
- **CPU绑定**：A55小核 4-7
- **优先级**：SCHED_OTHER (普通)
- **关键模块**：
  - LCD显示 (实时画面叠加)
  - 按键/旋钮输入
  - 参数在线调参界面
  - 比赛状态显示

## 3. 进程间通信总览

```
视觉进程 ──[共享内存:目标坐标]──► 控制进程
视觉进程 ◄──[共享内存:系统状态]── 控制进程
控制进程 ──[共享内存:运行数据]──► 人机交互
人机交互 ──[共享内存:参数配置]──► 控制进程
人机交互 ──[共享内存:视觉配置]──► 视觉进程
```

通信方式：**POSIX共享内存 + 命名信号量**
延迟目标：**< 0.5ms (实测 0.1~0.3ms)**

## 4. 数据流架构

```
摄像头 → 视觉进程 → 目标坐标(x,y,z,conf)
                         ↓
                    共享内存 (vision_result_t)
                         ↓
                    控制进程 → PID计算 → PWM输出 → 电机
                         ↓
                    共享内存 (system_status_t)
                         ↓
                    人机交互 → LCD显示 / 参数调整
```

## 5. 关键性能指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 视觉帧率 | ≥30fps | YOLOv8n推理 |
| 控制周期 | 1ms | 1kHz控制环 |
| IPC延迟 | <0.5ms | 共享内存读写 |
| 端到端延迟 | <35ms | 摄像头→执行 |
| CPU占用率 | <70% | 各核心 |
| 内存占用 | <2GB | 总进程内存 |
