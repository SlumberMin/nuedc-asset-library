## 二、锂电池选型

### 2.1 电池参数解读

| 参数 | 含义 | 选择要点 |
|------|------|----------|
| 标称电压 | 单节3.7V（满充4.2V） | 2S=7.4V, 3S=11.1V |
| 容量(mAh) | 能量储备 | 按总功耗×比赛时长×1.5选 |
| 放电倍率(C) | 最大放电能力 | 最大电流÷容量=C数 |
| 内阻 | 越小越好 | < 50mΩ为佳 |

### 2.2 选型计算示例

```
假设总功耗30W，比赛持续2小时：
所需容量 = 30W × 2h / 电压 = 60Wh

若用3S(11.1V): 容量 = 60/11.1 ≈ 5.4Ah → 选6000mAh
若用2S(7.4V):  容量 = 60/7.4  ≈ 8.1Ah → 选8000mAh

放电倍率验证：
峰值电流5A，电池6000mAh → 需 ≥ 0.83C → 一般1C即可满足
```

### 2.3 推荐电池配置

| 场景 | 推荐电池 | 续航参考 |
|------|----------|----------|
| 轻载（控制板+小电机） | 2S 2200mAh | 2-3小时 |
| 中载（舵机+电机） | 3S 3300mAh | 1.5-2小时 |
| 重载（大功率电机） | 3S 5000mAh+ | 2-3小时 |
| 双电源方案 | 控制2S2200 + 电机3S3300 | 各自独立 |

## 三、充放电管理

### 3.1 充电方案

**方案A：成品充电板（推荐）**
```
TP4056（单节）→ 1S电池，500mA-1A充电
BQ24610/BQ25895 → 多节电池，支持快充
```

**方案B：成品充电器**
```
IMAX B6平衡充电器 → 1-6S通用，功能全面
单节锂电池充电板 → 便宜简单
```

### 3.2 电量检测

```c
// 3S电池电压检测（分压电阻）
// 12V → 分压(10K+3.3K) → 3.0V → ADC
// ADC参考电压3.3V，12bit

float get_battery_voltage(void) {
    uint16_t adc = ADC_Read(BATTERY_CHANNEL);
    float voltage = adc / 4095.0 * 3.3;  // 分压后电压
    float real_voltage = voltage * (10.0 + 3.3) / 3.3;  // 还原实际电压

    // 电量估算（3S电池）
    if (real_voltage > 12.6) return 100;
    if (real_voltage > 11.1) return (real_voltage - 11.1) / 1.5 * 100;
    if (real_voltage > 9.0)  return 0;  // 低电量警告
    return -1;  // 过放保护
}
```

### 3.3 放电保护

| 保护 | 阈值(3S) | 措施 |
|------|----------|------|
| 低压警告 | < 11.1V | LED闪烁+蜂鸣器 |
| 低压保护 | < 9.9V | 切断电机供电 |
| 过放保护 | < 9.0V | 完全切断输出 |

## 四、多路稳压设计

### 4.1 稳压方案选型

| 器件 | 类型 | 输入 | 输出 | 电流 | 效率 | 适用 |
|------|------|------|------|------|------|------|
| AMS1117-3.3 | LDO | 4.5-12V | 3.3V | 800mA | 低 | 小功率控制板 |
| LM7805 | 线性 | 7-25V | 5V | 1.5A | 低 | 简单5V供电 |
| MP1584 | DC-DC | 4.5-28V | 可调 | 3A | 高(93%) | 大电流5V |
| XL6009 | 升压 | 5-32V | 可调 | 2A | 高 | 升压场景 |
| TPS5430 | DC-DC | 5.5-36V | 可调 | 3A | 高 | 高效降压 |

### 4.2 推荐电源架构

```
        ┌─────────────────────────────────────────────┐
        │               3S 锂电池 (12.6V~9.9V)         │
        └──────┬──────────┬──────────┬────────────────┘
               │          │          │
               ▼          ▼          ▼
         ┌─────────┐ ┌─────────┐ ┌─────────┐
         │MP1584   │ │MP1584   │ │AMS1117  │
         │→ 5V/3A  │ │→ 6V/2A  │ │→ 3.3V   │
         └────┬────┘ └────┬────┘ └────┬────┘
              │           │           │
         ┌────┴────┐  ┌───┴───┐  ┌───┴───┐
         │控制板   │  │舵机   │  │传感器 │
         │显示屏   │  │       │  │通信   │
         │传感器   │  │       │  │模块   │
         └─────────┘  └───────┘  └───────┘

电机直接由电池经H桥驱动，不经过稳压
```

### 4.3 关键设计要点

1. **输入输出压差**：LDO压差大时效率极低、发热严重，大电流场景必须用DC-DC
2. **电感选型**：DC-DC电感值按数据手册推荐，饱和电流≥1.3倍最大电流
3. **输入电容**：DC-DC输入端加100μF电解+100nF陶瓷，靠近芯片
4. **输出电容**：按数据手册推荐值，ESR影响纹波
5. **布局**：DC-DC功率回路面积最小化，远离敏感模拟电路

## 五、电源保护设计

### 5.1 保护电路清单

| 保护类型 | 电路 | 关键参数 |
|----------|------|----------|
| 反接保护 | P-MOSFET或二极管 | 耐压>电池电压，电流>最大负载 |
| 过流保护 | 自恢复保险丝(PPTC) | 额定电流=1.2×正常工作电流 |
| 过压保护 | TVS二极管 | 钳位电压>正常电压，<器件耐压 |
| 防反灌 | 二极管（肖特基） | 正向压降<0.5V |
| 软启动 | RC延时+MOSFET | 防止上电冲击电流 |

### 5.2 反接保护电路（P-MOS方案）

```
电池+ ──── S极 (SI2301 P-MOS)
              │
         G极 ─┤─── 10K电阻 ──→ GND
              │
           D极 ──── 后续电路

接反时：Vgs=0，MOSFET截止，保护电路
正常时：Vgs<0(负)，MOSFET导通，正常供电
```

### 5.3 电机驱动电源隔离

```
关键：电机启停产生大电流尖峰，会干扰控制板电源

方案1：独立电池
  电池1 → 控制系统
  电池2 → 电机驱动

方案2：单电池+LC滤波
  电池 → 电感(100μH) → 控制系统供电
       → 直接 → 电机驱动

方案3：大容量电容缓冲
  电机驱动VCC并联 1000-4700μF电解电容
  每个电机驱动芯片VCC旁路 100nF+10μF
```

## 六、电源调试检查清单

- [ ] 电池电压已测量，确认在安全范围
- [ ] 各路稳压输出已测量，纹波<50mV
- [ ] 上电瞬间无大电流冲击（用万用表电流档串入测量）
- [ ] 电机启停时控制板不复位（加电容/独立供电）
- [ ] 低压保护功能已测试
- [ ] 长时间运行散热可接受（摸一下稳压芯片温度）
- [ ] 电池连接牢固，有防脱落措施
- [ ] 电源开关位置方便操作，紧急断电容易
- [ ] 有备用电池或充电方案
