## 4. 核心技术点速查卡

### 4.1 FFT 频谱分析

```
标准流程：
1. 采样：fs ≥ 2*fmax（Nyquist），通常 fs = 4~10 × fmax
2. 加窗：Hanning（通用）/ Flat-top（幅度精度）/ Blackman（泄漏小）
3. FFT：N点 FFT（N = 2^k），频率分辨率 = fs/N
4. 幅度谱：|X(k)| = sqrt(Re²+Im²) × 2/N（除直流分量）
5. 找峰：定位基波和各次谐波频率
6. 计算THD：THD = sqrt(V2²+V3²+...+Vn²) / V1 × 100%

常见坑：
- 采样点数不够 → 频率分辨率差
- 未加窗 → 频谱泄漏严重
- 未补零 → 栅栏效应
- 非整周期采样 → 幅度误差
```

### 4.2 PID 控制

```
位置式 PID：
  u(k) = Kp*e(k) + Ki*Σe(i)*T + Kd*(e(k)-e(k-1))/T

增量式 PID：
  Δu(k) = Kp*(e(k)-e(k-1)) + Ki*e(k)*T + Kd*(e(k)-2e(k-1)+e(k-2))/T

参数整定经验（Ziegler-Nichols）：
1. 设 Ki=0, Kd=0，逐渐增大 Kp 直到系统等幅振荡
2. 记录临界增益 Ku 和振荡周期 Tu
3. PID: Kp=0.6Ku, Ki=2Kp/Tu, Kd=Kp*Tu/8

抗积分饱和：
  - 积分限幅：|Σe| ≤ Imax
  - 条件积分：仅在误差较小时积分
  - back-calculation：引入反馈消除饱和
```

### 4.3 电机控制

```
直流电机：
  速度控制：PWM占空比 → 电压 → 速度
  位置控制：编码器反馈 + PID

步进电机：
  细分驱动：1/2, 1/4, 1/8, 1/16, 1/32
  加减速：S曲线 / 梯形加减速

无刷电机（BLDC）：
  六步换相：霍尔传感器检测转子位置
  FOC：Clark/Park变换 + SVPWM

舵机：
  PWM控制：50Hz，0.5~2.5ms脉宽 → 0°~180°
```

### 4.4 ADC/DAC 选型

```
ADC 选型要点：
  分辨率：8/10/12/16/24 bit
  采样率：100kSPS~1GSPS
  接口：SPI/I2C（慢）, 并行（快）
  
  电赛常用：
    ADS1115 (16bit, I2C, 慢速高精度)
    ADS8688 (16bit, SPI, 100kSPS)
    STM32内置ADC (12bit, 1~5MSPS)

DAC 选型要点：
  分辨率：8/10/12/16 bit
  建立时间：<1μs（快速）/ <10μs（一般）
  
  电赛常用：
    MCP4725 (12bit, I2C)
    DAC8563 (16bit, SPI)
    STM32内置DAC (12bit)
```

### 4.5 常用通信协议

```
I2C：2线，地址寻址，100/400k/3.4M，适合板内传感器
SPI：4线，全双工，最高50MHz+，适合高速ADC/DAC
UART：2线，异步，115200~1Mbps，适合调试/串口通信
CAN：2线，差分，1Mbps，适合工业控制总线
1-Wire：1线，温度传感器等
```

### 4.6 电源设计速查

```
DC-DC 变换器：
  Buck（降压）：Vo = D*Vi，效率90-95%
  Boost（升压）：Vo = Vi/(1-D)，效率85-92%
  Buck-Boost：Vo = -D*Vi/(1-D)

LDO 选型：
  压差：需 > Dropout Voltage
  功耗：P = (Vi-Vo)*Iout（发热！）
  PSRR：对纹波的抑制

开关电源关键参数：
  开关频率：100kHz~2MHz（频率高→电感小→效率可能低）
  电感选择：L ≥ (Vi-Vo)*D/(fs*ΔI)
  电容选择：C ≥ ΔI/(8*fs*ΔVo)
```

## 5. 赛题解题流程模板

```
第1阶段（开赛前2小时）：审题
  → 明确所有指标要求
  → 区分基本要求 vs 发挥部分
  → 评估哪些能做、哪些难做

第2阶段（2-6小时）：方案确定
  → 选择主控（STM32F103/F407/G431）
  → 选择外围模块
  → 画系统框图
  → 确定关键难点的解决方案

第3阶段（6-24小时）：基础搭建
  → 硬件搭建 + 基本驱动
  → 完成基本要求的功能
  → 调试验证

第4阶段（24-48小时）：优化提升
  → 完成发挥部分
  → 性能优化（精度、速度、效率）
  → 测试所有指标

第5阶段（最后24小时）：收尾
  → 撰写报告
  → 代码整理
  → 最终测试
  → 准备演示
```

## 6. 高频"坑"汇总

| 坑 | 描述 | 避免方法 |
|---|------|---------|
| ADC采样不稳 | 未接地、参考电压不稳 | 独立参考源、滤波电容 |
| 电机干扰 | 电火花干扰单片机 | 光耦隔离、独立电源 |
| PWM频率不当 | 太低有噪声、太高开关损耗 | 电机20kHz+、LED 1kHz+ |
| 运放饱和 | 输出接近电源轨 | 选择轨到轨运放 |
| 振荡器不起振 | 负性阻抗不足 | 选择合适负载电容 |
| 通信丢数据 | 缓冲区溢出 | DMA + 环形缓冲 |
| 滤波器带宽错 | 低通截止频率设错 | 先仿真再焊接 |
| 供电不足 | 电流不够、纹波大 | 预留余量、多级滤波 |
