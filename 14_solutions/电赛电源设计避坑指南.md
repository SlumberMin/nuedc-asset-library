## 电源设计基本原则

### 选型决策树

```
输入电压 > 输出电压？
  ├─ 是 → 需要降压？
  │   ├─ 电流 < 500mA → LDO（简单、低噪声）
  │   └─ 电流 > 500mA → Buck DC-DC（高效）
  │
  └─ 否 → 需要升压 → Boost DC-DC

输出纹波要求高？(ADC/模拟电路)
  └─ LDO 或 LC后级滤波

需要隔离？
  └─ 隔离DC-DC模块
```

### 效率与噪声权衡

| 类型 | 效率 | 噪声 | 复杂度 | 适用 |
|------|------|------|--------|------|
| LDO | 低(η=Vo/Vi) | 极低 | 简单 | 模拟、小功率 |
| Buck | 高(>85%) | 中等 | 中等 | 数字、大功率 |
| Boost | 高(>80%) | 中等 | 中等 | 升压场景 |

## 1. LDO选型与使用

### 1.1 选型要点

**关键参数**:
- **压差 (Dropout Voltage)**: 输入-输出最小压差，LDO的致命限制
- **输出电流**: 预留30%以上余量
- **静态电流**: 电池供电时关注（越低越好）
- **PSRR**: 电源抑制比，模拟电路重要指标
- **热阻**: 大压差时发热严重

**常见型号推荐**:

| 型号 | 输出 | 压差 | 电流 | 特点 |
|------|------|------|------|------|
| AMS1117-3.3 | 3.3V | 1.1V | 1A | 经典，便宜 |
| HT7333 | 3.3V | 0.1V | 500mA | 低压差 |
| ME6211 | 3.3V | 0.1V | 500mA | SOT-23封装 |
| LM1117-5.0 | 5.0V | 1.2V | 800mA | 经典5V |
| XC6206 | 可调 | 0.1V | 200mA | 超小封装 |

### 1.2 常见错误

#### ❌ 错误1: 压差不够
```
问题: 用AMS1117-3.3，输入5V→输出3.3V，压差1.7VOK
     但输入4V→输出3.3V，压差0.7V < 1.1V，输出不稳！

解决: 选择低压差LDO（HT7333压差仅0.1V）
```

#### ❌ 错误2: 散热不足
```
问题: 输入12V→3.3V/1A，功耗=(12-3.3)×1=8.7W
     SOT-223封装热阻约90°C/W，温升=8.7×90=783°C → 必然烧毁！

解决: 
  - 功耗>1W考虑用DC-DC
  - 或用TO-220封装+散热片
  - 功耗计算: P = (Vin - Vout) × Iout
```

#### ❌ 错误3: 输入输出电容不足
```c
// AMS1117要求:
// 输入: 10μF陶瓷电容（靠近Vin引脚）
// 输出: 22μF陶瓷电容（ESR很重要！）

// 典型电路
// Vin ──┤10μF├── AMS1117 ──┤22μF├── Vout
//         │                      │
//        GND                    GND
```

### 1.3 LDO典型电路

```
        ┌─────────┐
Vin ─┬──┤ IN  OUT ├──┬── Vout(3.3V)
     │  │  ADJ/GND │  │
    10μF└─────────┘ 22μF
     │                │
    GND              GND

注意:
1. 电容尽量靠近引脚
2. GND引脚散热焊盘要大面积铺铜
3. 输入电容不能省！（否则可能振荡）
```

## 2. DC-DC Buck电路

### 2.1 常用芯片

| 型号 | 输入范围 | 输出 | 电流 | 特点 |
|------|----------|------|------|------|
| LM2596 | 4.5~40V | 可调 | 3A | 经典，外围简单 |
| MP1584 | 4.5~28V | 可调 | 3A | 小体积，高效 |
| TPS5430 | 5.5~36V | 可调 | 3A | TI品质 |
| XL4015 | 8~32V | 可调 | 5A | 大电流 |
| SY8089 | 2.5~5.5V | 可调 | 3A | 小信号降压 |

### 2.2 关键设计要点

#### 电感选型
```
电感值: L = (Vin - Vout) × Vout / (Vin × f × ΔI)
其中 ΔI 一般取负载电流的30%

饱和电流: Isat > Iout + ΔI/2
DCR(直流电阻): 越小越好，影响效率
```

#### 输出电容选型
```
ESR影响输出纹波: ΔV = ΔI × ESR
陶瓷电容ESR低，纹波小
电解电容ESR高，但容量大
混合使用: 陶瓷(低ESR) + 电解(大容量)
```

#### 反馈电阻
```
Vout = Vref × (1 + R1/R2)
Vref一般为0.6V或1.2V

// 典型值计算 (Vref=0.6V, Vout=5V)
R1/R2 = (5/0.6) - 1 = 7.33
选 R2=10kΩ, R1=73.2kΩ (用75kΩ近似)
```

### 2.3 常见错误

#### ❌ 错误1: 布局不当导致EMI
```
关键回路: 开关管→电感→输出电容→GND→开关管
此回路面积必须最小！

错误: 电感放得太远，走线太长
正确: 电感紧靠IC输出引脚
```

#### ❌ 错误2: 反馈走线受干扰
```
错误: FB走线从电感旁边经过
正确: FB走线远离电感和开关节点，走Kelvin连接
```

#### ❌ 错用二极管方向
```
Buck电路中的续流二极管:
  阳极接SW节点，阴极接Vin
  错接会导致IC烧毁！
  
注意: 同步Buck芯片不需要外部二极管
```

## 3. DC-DC Boost电路

### 3.1 常用场景
- 3.7V锂电池升压到5V
- 5V升压到12V/24V驱动电机
- 太阳能MPPT

### 3.2 典型电路 (以MT3608为例)

```
Vin ─┬──┤L(4.7~22μH)├──┬── D ──┬── Vout
     │                  │       │
    Cin               MOSFET   Cout
    (22μF)           (内部)    (22μF×2)
     │                  │       │
    GND────────────────GND─────GND

FB分压: Vout = Vref × (1 + R1/R2)
```

### 3.3 注意事项

| 问题 | 原因 | 解决 |
|------|------|------|
| 启动失败 | 负载太重 | 先轻载启动再加重 |
| 输出纹波大 | 电容ESR高 | 用低ESR陶瓷电容 |
| 效率低 | 二极管压降 | 用肖特基二极管 |
| 噪声干扰MCU | 开关噪声 | LDO后级滤波 |

## 4. 电源滤波设计

### 4.1 滤波电容选型

```
┌─────────────────────────────────────────────────┐
│ 电容类型   │ 容量范围  │ 频率特性  │ 用途       │
├─────────────────────────────────────────────────┤
│ 电解电容   │ 10~1000μF │ 低频     │ 储能、低频滤波│
│ 陶瓷电容   │ 100nF~10μF│ 高频     │ 去耦、高频滤波│
│ 钽电容     │ 1~100μF   │ 中高频   │ 稳压输出    │
│ C0G/NP0    │ 1pF~100nF │ 极稳定   │ 精密模拟    │
└─────────────────────────────────────────────────┘
```

### 4.2 多级滤波方案

```
电源输入 → 电解100μF → 磁珠 → 陶瓷100nF → MCU VDD
                                    │
                                  10μF
                                    │
                                  GND

要点:
- 100nF去耦电容必须紧靠芯片电源引脚
- 大电容可以稍远（储能）
- 每个电源引脚都要有独立的去耦电容
```

### 4.3 π型LC滤波

```
       L(10μH)
Vin ──┬──┤├──┬── Vout
     │         │
    C1(10μF)  C2(10μF)
     │         │
    GND       GND

适用于: Buck输出二次滤波，降低开关纹波
注意: L的额定电流要够！
```

## 5. PCB电源布线

### 5.1 布线原则

1. **电源线加粗**: 信号线0.25mm够了，电源线至少1mm，大电流>2mm
2. **短而粗**: 走线越短越好，越宽越好
3. **星形拓扑**: 各路电源从电源入口分叉，避免菊花链
4. **地平面**: 至少一层完整的地平面

### 5.2 常见错误

```
❌ 错误: 电源线和信号线并行走线
   → 电源纹波耦合到信号线

❌ 错误: 大电流回路面积过大
   → 天线效应，EMI严重

❌ 错误: 地线太细或断裂
   → 地电位浮动，采样不准

✅ 正确: 
   - 电源线走粗线或铺铜
   - 去耦电容就近放置
   - 模拟地和数字地单点连接
```

## 6. 常见电源Bug

### Bug 6.1: 系统频繁复位
```
现象: MCU随机复位，跑一段时间就死
排查:
  1. 用示波器看VDD波形（AC耦合，100mV/div）
  2. 检查是否有跌落到复位阈值以下
  3. 可能原因:
     - 负载突变(电机启动)导致电压跌落
     - LDO输入电容不够
     - 接触不良(杜邦线)
解决:
  - 加大输出电容(470μF电解+100nF陶瓷)
  - 电机单独供电
  - 使用带欠压保护的LDO
```

### Bug 6.2: ADC采样不准
```
现象: ADC读数随负载变化而变化
原因: 电源内阻太高，负载电流变化导致供电电压变化
解决:
  - ADC参考电压用独立LDO
  - 加LC滤波隔离
  - 使用芯片内部参考电压
```

### Bug 6.3: 电机驱动时MCU死机
```
原因: 电机反电动势/启动浪涌干扰电源
解决:
  - 电机和MCU分路供电
  - 各路电源入口加TVS/ESD保护
  - 电机驱动板加续流二极管
  - 电源入口加100μF+100nF滤波
```

### Bug 6.4: LDO输出振荡
```
现象: LDO输出正弦波形
原因: 输出电容ESR不合适，或电容值太小
解决:
  - 按数据手册要求选择电容(注意ESR范围)
  - 增大电容值
  - 更换电容类型(陶瓷换钽电容)
```

### Bug 6.5: 发热严重
```
排查步骤:
  1. 计算功耗: P_loss = (Vin-Vout)×Iout (LDO)
             P_loss ≈ Pin×(1-η) (DC-DC)
  2. LDO功耗>0.5W考虑换DC-DC
  3. 检查输出是否短路
  4. 检查负载是否超出额定值
```

## 7. 电源保护设计

### 7.1 过流保护

```c
// 软件过流保护（通过电流采样）
#define CURRENT_LIMIT  3.0f  // A

void check_overcurrent(float current) {
    if (current > CURRENT_LIMIT) {
        // 关闭PWM输出
        HAL_TIM_PWM_Stop(&htim1, TIM_CHANNEL_1);
        // 报警
        set_alarm_flag();
    }
}

// 硬件过流保护
// 方案1: 自恢复保险丝(PTC)
// 方案2: 电子保险丝(MOSFET+运放比较)
```

### 7.2 过压保护

```
TVS管选型:
  - 工作电压 > 正常供电电压
  - 钳位电压 < 被保护器件最大耐压
  - 功率 > 浪涌功率

例如: 5V系统用SMBJ5.0A
  - 工作电压: 5.0V
  - 钳位电压: 9.2V
  - 峰值功率: 600W
```

### 7.3 反接保护

```
方案1: 串联二极管（压降0.7V，功耗大）
方案2: P-MOSFET反接保护（推荐！）

       ┌─── P-MOSFET ───┐
Vin+ ──┤S           D├── Vout+
       │    G         │
Vin- ──┴──────────────┴── Vout-

正接: G被拉低，MOS导通
反接: G被拉高，MOS截止 → 保护后级电路
```

## 8. 多路电源设计

### 8.1 典型多路电源方案

```
12V电池
  │
  ├─ Buck → 5V (数字电路、舵机、传感器)
  │
  ├─ Buck → 3.3V (MCU、通信芯片)
  │         └─ LDO → 1.8V (特殊芯片)
  │
  └─ 直接12V (电机驱动)

要点: 各路独立使能控制，按需开关
```

### 8.2 上电时序

```
某些系统要求严格的上电时序:
1. 先核心电源 (1.2V)
2. 再IO电源 (3.3V)
3. 最后外设电源 (5V)

解决方案:
  - 使用带Power Good信号的DC-DC
  - PG信号控制下一级使能
  - 或用RC延时电路
```

## 9. 电池供电设计

### 9.1 锂电池保护

```
必备保护:
1. 过充保护: 单节截止4.2V
2. 过放保护: 单节截止2.7V
3. 过流保护: 短路保护
4. 均衡保护: 多节串联时

推荐芯片: DW01 + 8205A (经典方案)
或使用成品保护板
```

### 9.2 电池电量估算

```c
// 锂电池电压-电量对应关系(单节)
typedef struct {
    float voltage;
    int percent;
} BatteryLUT_t;

BatteryLUT_t battery_table[] = {
    {4.20, 100}, {4.06, 90}, {3.98, 80},
    {3.92, 70},  {3.87, 60}, {3.82, 50},
    {3.78, 40},  {3.73, 30}, {3.68, 20},
    {3.50, 10},  {3.30, 5},  {3.00, 0}
};

int get_battery_percent(float voltage) {
    if (voltage >= 4.20f) return 100;
    if (voltage <= 3.00f) return 0;
    for (int i = 1; i < 12; i++) {
        if (voltage >= battery_table[i].voltage) {
            float ratio = (voltage - battery_table[i].voltage) /
                         (battery_table[i-1].voltage - battery_table[i].voltage);
            return battery_table[i].percent + 
                   (int)(ratio * (battery_table[i-1].percent - battery_table[i].percent));
        }
    }
    return 0;
}
```

### 9.3 充电设计

```
推荐方案: TP4056 (线性充电IC)
- 输入: 5V USB
- 充电电流: 可调(通过Rprog)
  Rprog=10kΩ → 1A
  Rprog=20kΩ → 500mA
  Rprog=100kΩ → 100mA

注意事项:
1. 充电时不能同时大电流放电（LDO温升大）
2. 充电IC热设计要充分
3. 电池温度监测(NTC)
```

## 电源设计检查清单

```
□ 输入电压范围确认
□ 输出电压/电流需求
□ 效率计算（LDO尤其注意散热）
□ 纹波要求（ADC参考电压要特别关注）
□ 保护电路（过流、过压、反接）
□ 去耦电容（每个电源引脚）
□ PCB走线宽度
□ 上电时序要求
□ 电池保护（如有）
□ 测试验证（带载测试、效率测试、纹波测试）
```
