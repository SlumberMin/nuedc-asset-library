**核心技术方案**:
- 4麦克风方形阵列 (INMP441 I2S)
- GCC-PHAT声源定位算法
- 粒子滤波器运动估计 (200粒子)

- 粒子滤波: 随机游走预测 + 高斯似然更新 + 系统重采样

**关键性能指标**:
- 定位精度: ±1.8°
- 定位范围: ±85°
- 移动声源跟踪: 速度<0.8m/s

### 2.3 2021_F_智能送药小车

**补充文档**:

- `2021_F_智能送药小车/文档/04_设计报告.md`

**核心技术方案**:
- STM32F407 + OpenMV双MCU架构
- A*路径规划算法 (10×10栅格地图)
- 模板匹配数字识别 (病房号)
- PID循线控制 + 陀螺仪转弯
- 状态机管理 (IDLE→DELIVER→UNLOAD→RETURN)

**路径规划算法**:
- A*算法: f(n) = g(n) + h(n)
- g(n): 起点到当前节点实际代价
- h(n): 曼哈顿距离启发估计
- 时间复杂度: O(b^d)

**避障策略**:
- 3路超声波 (前左/前右/后方)
- 障碍物<20cm时停车绕行
- 左右判断选择绕行方向

**关键性能指标**:
- 数字识别率: 98.5%
- 循线精度: ±1.2cm
- 任务完成时间: 2min30s

- 互补滤波坡度检测 (98%陀螺仪+2%加速度计)
- 增量式PID速度控制 + 坡度自适应补偿
- 防滑控制 (检测打滑→降低速度→限制加速度)
- JGA25-370大扭矩电机 (1:34减速比)

**坡道检测方案**:
- MPU6050六轴IMU
- 互补滤波: pitch = 0.98×(prev+gyro×dt) + 0.02×accel
- 安装要求: 水平安装于底盘中心，X轴朝前

**防滑控制策略**:
- 检测条件: 目标速度>实际速度×1.5 且 坡度>5°
- 应对: 降低目标速度70% + 限制加速度

**角度传感器选型**:
- MPU6050: 6轴IMU，±2g加速度，±250°/s陀螺仪
- I2C接口，100Hz采样
- 互补滤波融合，精度±1.5°

**关键性能指标**:
- 最大爬坡角度: 22°
- 坡道速度稳定性: 波动8%
- 坡度检测精度: ±1.5°
