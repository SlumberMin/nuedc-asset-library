## 三、新增算法模块清单

### 3.1 第二轮新增的算法测试模块

| 模块 | 核心算法 | 关键特性 | 测试重点 |
|------|---------|---------|---------|
| **LADRC** | 线性自抗扰控制 | 带宽参数化、LESO线性观测器 | 观测器收敛、参数整定简洁性、vs ADRC对比 |
| **超螺旋SMC** | 超螺旋滑模控制 | 连续控制信号、有限时间收敛 | 抖振抑制效果、收敛速度、鲁棒性 |
| **自适应卡尔曼** | 自适应卡尔曼滤波 | 噪声统计量在线估计 | 噪声变化适应、异常值拒绝、vs标准KF对比 |
| **增量式PID** | 增量式PID控制 | 天然抗积分饱和、无扰切换 | 增量计算、饱和避免、位置式PID对比 |
| **陷波滤波器** | 二阶IIR陷波 | 频率选择性衰减 | 陷波深度、Q值影响、自适应频率跟踪 |
| **抗饱和** | 抗积分饱和机制 | 多种方法对比 | 恢复时间、积分累积限制、方法对比 |
| **性能基准** | 全算法性能测试 | MCU资源约束验证 | 计算时间、内存占用、回归检测 |

### 3.2 算法模块技术特点

**LADRC (线性自抗扰控制器)**
- 核心优势: 参数整定简化为2个带宽参数(omega_c, omega_o)
- LESO使用线性增益: beta_i = omega_o^i * C(n,i)
- LSEF使用PD控制: kp = omega_c^2, kd = 2*omega_c
- 适用场景: 需要快速整定、模型不确定的控制系统

**超螺旋滑模控制器**
- 核心优势: 连续控制信号,无需边界层即可抑制抖振
- 算法: u = -k1*sqrt(|s|)*sign(s) + integral(-k2*sign(s))
- 有限时间收敛特性
- 适用场景: 高精度鲁棒控制、扰动抑制

**自适应卡尔曼滤波器**
- 核心优势: 实时估计噪声统计量,适应环境变化
- 方法: 基于新息序列的协方差匹配
- 支持异常值检测与拒绝
- 适用场景: 噪声特性时变的传感器融合
