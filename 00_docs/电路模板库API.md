## 电源电路模板

**文件**: `电源电路模板.md`

### 锂电池充放电方案

#### TP4056 充电方案

```
USB 5V ──┬── TP4056 ──── BAT+ (锂电池正极)
         │   Pin: VIN   BAT- (锂电池负极)
         │
         │   PROG引脚 ── Rprog ── GND
         │   Rprog = 1.2kΩ → 充电电流约1A
         │   Rprog = 2kΩ  → 充电电流约580mA
```

#### 锂电池保护电路 (DW01+8205A)

| 保护功能 | 阈值 | 说明 |
|---------|------|------|
| 过充保护 | 4.25V | 断开充电回路 |
| 过放保护 | 2.5V | 断开放电回路 |
| 过流保护 | 3A | 断开回路 |
| 短路保护 | — | 立即断开 |

#### 集成方案 (IP5306)

充电+放电+升压+电量指示一体方案，推荐新手使用。

### 多路稳压方案

```
锂电池 3.7-4.2V
    ├── [5V] LM7805 / MP1584 → 5V (舵机/电机驱动)
    ├── [3.3V] AMS1117-3.3 → 3.3V (MCU/传感器)
    └── [可调] LM317 → 自定义电压
```

| 稳压器 | 输入 | 输出 | 电流 | 特点 |
|--------|------|------|------|------|
| AMS1117-3.3 | 4.5~12V | 3.3V | 800mA | 低压差，MCU供电首选 |
| LM7805 | 7~25V | 5V | 1.5A | 经典，发热大 |
| MP1584 | 4.5~28V | 可调 | 3A | 开关电源，效率高 |
| LM317 | 3~40V | 可调 | 1.5A | 可调输出 |

### 电源设计要点

| 要点 | 说明 |
|------|------|
| 去耦电容 | 每个IC的VCC旁放100nF + 10μF |
| 电源指示 | 每路输出加LED指示 |
| 保险丝 | 总输入加自恢复保险丝(PPTC) |
| 地线 | 星形接地，避免地环路 |

## 最小系统电路

### STM32F407 最小系统

**文件**: `STM32最小系统电路.md`

#### 核心元件

| 元件 | 型号 | 说明 |
|------|------|------|
| MCU | STM32F407ZGT6 | 168MHz, 1MB Flash, 192KB RAM |
| 晶振(HSE) | 8MHz | 外部高速晶振 |
| 晶振(LSE) | 32.768kHz | RTC用(可选) |
| 电源滤波 | 4×100nF + 4.7μF | VDD/VDDA去耦 |

#### 关键连接

```
供电:
  VDD ── 3.3V (每个VDD脚接100nF去耦电容)
  VDDA ── 3.3V (模拟供电，加100nF + 1μF)
  VBAT ── 3.3V (RTC供电，可接纽扣电池)

时钟:
  OSC_IN (PD0) ── 8MHz晶振 ── OSC_OUT (PD1)
  晶振两端各接20pF负载电容到GND

复位:
  NRST ── 10kΩ上拉到VDD + 100nF到GND + 复位按键

启动配置:
  BOOT0 ── 10kΩ下拉到GND (正常启动)
  BOOT1 ── 10kΩ下拉到GND

调试:
  SWDIO (PA13) ── SWD接口
  SWCLK (PA14) ── SWD接口
```

#### PCB布局建议

- MCU放在板中央
- 晶振紧邻MCU，走线尽量短
- 去耦电容紧贴VDD引脚
- SWD接口放在板边，方便连接

## 电机驱动电路模板

**文件**: `电机驱动电路模板.md`

### TB6612FNG 双路H桥

推荐驱动方案，低损耗，双路输出。

```
VM ── 电机电源 (4.5~13.5V)
VCC ── 逻辑电源 (3.3V/5V)
GND ── 地

AIN1/AIN2 ── STM32 GPIO (方向控制)
PWMA ── STM32 TIM_PWM (速度控制)
STBY ── 3.3V (使能)

AO1/AO2 ── 电机A
BO1/BO2 ── 电机B
```

| 连接 | 说明 |
|------|------|
| VM | 电机电源，接锂电池 |
| VCC | 逻辑电源，接3.3V |
| AIN1/AIN2 | 方向控制，接GPIO |
| PWMA | 速度控制，接TIM PWM输出 |
| STBY | 使能引脚，拉高使能 |

### L298N 双路H桥

经典方案，但压降大(约2V)，发热严重。仅在TB6612电流不够时使用。

### BTS7960 大电流驱动

43A大电流，适合大功率电机。

### 舵机驱动

```
VCC ── 5V (舵机电源)
GND ── GND
信号线 ── STM32 TIM_PWM (50Hz, 0.5~2.5ms脉宽)
```

| 舵机型号 | PWM频率 | 脉宽范围 | 角度范围 |
|---------|---------|---------|---------|
| SG90 | 50Hz | 0.5~2.4ms | 0~180° |
| MG996R | 50Hz | 0.5~2.5ms | 0~180° |
| MG90S | 50Hz | 0.5~2.4ms | 0~180° |

## 传感器电路模板

**文件**: `传感器电路模板.md`

### 超声波 (HC-SR04)

```
VCC ── 5V
Trig ── STM32 GPIO (输出10μs脉冲)
Echo ── 分压后接STM32 GPIO (5V→3.3V: 1kΩ+2kΩ分压)
GND ── GND
```

### 六轴IMU (MPU6050)

```
VCC ── 3.3V
GND ── GND
SCL ── STM32 I2C_SCL (4.7kΩ上拉到VCC)
SDA ── STM32 I2C_SDA (4.7kΩ上拉到VCC)
AD0 ── GND (I2C地址0x68) 或 VCC (地址0x69)
INT ── STM32 GPIO_EXTI (可选，数据就绪中断)
```

### 红外循迹传感器

```
VCC ── 3.3V/5V
GND ── GND
OUTx ── STM32 GPIO (数字输出) 或 ADC (模拟输出)
```

### 激光ToF测距 (VL53L0X)

```
VCC ── 3.3V
GND ── GND
SCL ── I2C_SCL (4.7kΩ上拉)
SDA ── I2C_SDA (4.7kΩ上拉)
XSHUT ── GPIO (使能，可选)
GPIO1 ── GPIO_EXTI (中断，可选)
```

## 信号调理电路模板

**文件**: `信号调理电路模板.md`

### 运算放大器电路

#### 同相放大器

```
        Rf
   ┌───/\/\/───┐
   │            │
Vin──(+)        ├── Vout
     (-) OP-AMP │
      │   ┌─────┘
      └──/\/\/── GND
          Rg

增益: G = 1 + Rf/Rg
```

#### 反相放大器

```
       Rf
Vin──/\/\/──┐
            │
      (-)   ├── Vout
       (+) OP-AMP
        │
       Rg
        │
       GND

增益: G = -Rf/Rg
```

#### 差分放大器

用于抑制共模噪声，适合小信号采集。

### 滤波电路

#### 一阶RC低通滤波

```
Vin──R──┬── Vout
        │
        C
        │
       GND

截止频率: fc = 1/(2πRC)
```

#### 有源低通滤波 (Sallen-Key)

二阶滤波，-40dB/dec滚降，适合ADC前端。

### ADC前端调理

```
信号源 → 放大/衰减 → 低通滤波 → 钳位保护 → ADC输入
```

| 环节 | 电路 | 说明 |
|------|------|------|
| 放大/衰减 | 运放电路 | 调整信号幅度到ADC量程 |
| 低通滤波 | RC/有源滤波 | 抗混叠，截止频率<fs/2 |
| 钳位保护 | 肖特基二极管 | 防止过压损坏ADC |

## 通信电路模板

**文件**: `通信电路模板.md`

### UART

```
STM32_TX ── 1kΩ ── RX_设备
STM32_RX ── 1kΩ ── TX_设备
```

注意：3.3V设备直连，5V设备需电平转换。

### RS232 (MAX3232)

```
STM32_TX ── MAX3232 T1IN ── T1OUT ── DB9 Pin3
STM32_RX ── MAX3232 R1OUT ── R1IN ── DB9 Pin2
VCC ── 3.3V
```

### CAN (TJA1050)

```
STM32_TX(CAN_TX) ── TJA1050 TXD
STM32_RX(CAN_RX) ── TJA1050 RXD
TJA1050 CANH ── 总线CANH
TJA1050 CANL ── 总线CANL
CANH/CANL之间 ── 120Ω终端电阻
```

### RS485 (MAX485)

```
STM32_TX ── MAX485 DI
STM32_RX ── MAX485 RO
GPIO ── MAX485 DE/RE (方向控制)
MAX485 A/B ── 总线
```

## 电路模板使用流程

### 快速使用步骤

```
1. 确定需求
   └── 需要什么功能模块? (电源/驱动/传感器/通信/显示)

2. 查找模板
   └── 在对应 .md 文件中找到匹配的电路方案

3. 复制原理图
   └── 将电路原理图复制到你的工程中

4. 调整参数
   └── 根据实际需求修改电阻/电容值

5. PCB布局
   └── 按照模板中的布局建议放置元件

6. 检查验证
   └── 参考注意事项，检查关键连接
```
