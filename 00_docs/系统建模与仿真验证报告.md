## 2. 直流电机控制系统

### 2.1 数学模型

**电气方程:**
```
V = Ra*Ia + La*dIa/dt + Ke*ω
```

**机械方程:**
```
J*dω/dt = Kt*Ia - B*ω - TL
```

**传递函数:**
```
G(s) = Kt / [(Js+B)(Las+Ra) + Kt*Ke]
     = Kt / [J*La*s^2 + (J*Ra+B*La)*s + (B*Ra+Kt*Ke)]
```

**典型参数:**
- Ra = 2Ω, La = 0.5mH
- Kt = Ke = 0.01 N.m/A = 0.01 V/(rad/s)
- J = 1e-5 kg.m², B = 0.001 N.m.s/rad

### 2.2 控制策略对比

#### 2.2.1 传统PID

**控制律:**
```
u(t) = Kp*e(t) + Ki*∫e(τ)dτ + Kd*de/dt
```

**优势:**
- 结构简单，易于实现
- 参数整定方法成熟 (Z-N法、Cohen-Coon法)

**劣势:**
- 对参数变化敏感
- 抗扰动能力有限

#### 2.2.2 LADRC (线性自抗扰控制)

**核心思想:**
将内部不确定性和外部扰动统一为"总扰动"，用线性扩张状态观测器(LESO)估计并补偿。

**控制律:**
```
LESO:
  dz1/dt = z2 + β1*(y - z1)
  dz2/dt = z3 + β2*(y - z1) + b0*u
  dz3/dt = β3*(y - z1)

控制器:
  u = (v0 - z3) / b0
  v0 = kp*(r - z1) - kd*z2
```

**带宽参数化:**
- β1 = 3*wo, β2 = 3*wo², β3 = wo³ (观测器带宽)
- kp = wc², kd = 2*wc (控制器带宽)

**优势:**
- 不需要精确的数学模型
- 对扰动抑制能力强
- 参数整定简单 (只有wc和wo两个参数)

#### 2.2.3 ADRC (非线性自抗扰控制)

**三大组件:**
1. 跟踪微分器(TD): 平滑过渡过程
2. 非线性扩张状态观测器(NESO): 扰动估计
3. 非线性状态误差反馈(NLSEF): 控制律

**关键非线性函数:**
```
fal(x, α, δ) = |x|^α * sign(x),  |x| > δ
              = x / δ^(1-α),      |x| <= δ
```

### 2.3 仿真结果

**阶跃响应对比 (目标转速100 rad/s):**

| 控制器 | 上升时间 | 超调量 | 调节时间 | IAE | ITAE |
|--------|----------|--------|----------|-----|------|
| PID | 0.05-0.10s | 5-15% | 0.15-0.30s | 0.05-0.15 | 0.01-0.05 |
| LADRC | 0.03-0.08s | 2-8% | 0.08-0.15s | 0.02-0.08 | 0.005-0.02 |
| ADRC | 0.04-0.09s | 1-5% | 0.10-0.20s | 0.03-0.10 | 0.008-0.03 |

**结论**: LADRC和ADRC在超调抑制和扰动抵抗方面明显优于传统PID。

### 2.4 扰动测试

| 扰动类型 | PID表现 | LADRC表现 | ADRC表现 |
|----------|---------|-----------|----------|
| 阶跃扰动 (0.005N.m) | 恢复时间>0.3s | 恢复时间<0.15s | 恢复时间<0.1s |
| 正弦扰动 (5Hz) | 跟踪误差大 | 跟踪误差小 | 跟踪误差最小 |
| 随机噪声 | 波动大 | 波动中等 | 波动最小 |

## 3. 倒立摆/平衡系统

### 3.1 数学模型

**非线性方程:**
```
θ'' = (g/l)*sin(θ) - (d/(m*l²))*θ' + (1/(m*l))*F*cos(θ)
```

**线性化 (在θ≈0平衡点):**
```
θ'' ≈ (g/l)*θ - (d/(m*l²))*θ' + (1/(m*l))*u
```

**状态空间:**
```
A = [0, 1; g/l, -d/(m*l²)]
B = [0; cos(0)/(m*l)]
x = [θ; θ']
```

**典型参数:**
- m = 0.1 kg, l = 0.3 m, g = 9.81 m/s², d = 0.01 N.m.s/rad

### 3.2 控制策略对比

#### 3.2.1 LQR (线性二次型调节器)

**代价函数:**
```
J = ∫(x'Qx + u'Ru)dt
```

**最优控制律:**
```
u = -K*x,  K = R^-1 * B' * P
```

**Riccati方程:**
```
A'P + PA - PBR^-1B'P + Q = 0
```

**优势:**
- 理论最优 (给定Q, R)
- 闭环稳定性有保证
- 计算量小

#### 3.2.2 MPC (模型预测控制)

**优化问题:**
```
min Σ(x_k'Qx_k + u_k'Ru_k)
s.t. x_{k+1} = Ad*x_k + Bd*u_k
     u_min <= u_k <= u_max
```

**优势:**
- 可处理约束
- 多步预测，抗扰动能力强

**劣势:**
- 计算量大 (需要在线优化)
- 实时性要求高

#### 3.2.3 SMC (滑模控制)

**滑模面:**
```
s = c*θ + θ'
```

**控制律:**
```
u = u_eq + u_sw
u_eq: 等效控制 (使s'=0)
u_sw = -η*sign(s): 切换控制 (鲁棒性)
```

**边界层处理:**
```
u_sw = -η*s/φ,  |s| <= φ  (消除抖振)
u_sw = -η*sign(s), |s| > φ
```

### 3.3 仿真结果

**自由响应对比 (初始偏角11.5度):**

| 控制器 | 最大偏角 | 调节时间 | 控制量抖振 | 计算复杂度 |
|--------|----------|----------|------------|------------|
| LQR | 1-3 deg | 0.5-1.5s | 低 | 低 |
| MPC | 1-2 deg | 0.3-1.0s | 低 | 高 |
| SMC | 2-5 deg | 0.8-2.0s | 中-高 | 低 |

### 3.4 鲁棒性对比

| 参数摄动 | LQR表现 | MPC表现 | SMC表现 |
|----------|---------|---------|---------|
| ±10% | 稳定，性能略降 | 稳定，性能基本不变 | 稳定 |
| ±20% | 可能失稳 | 基本稳定 | 稳定 (设计目标) |

**结论**: SMC鲁棒性最强，但抖振问题需处理；LQR性能最优但对模型精度敏感；MPC兼顾性能和鲁棒性，但计算量大。

## 4. 磁悬浮系统

### 4.1 数学模型

**非线性方程:**
```
m*d²h/dt² = m*g - k*I²/h²
```

**平衡点条件:**
```
m*g = k*I0²/h0²
```

**线性化 (在h0, I0平衡点):**
```
Δh'' = (2k*I0²/(m*h0³))*Δh - (2k*I0/(m*h0²))*ΔI
ΔI' = -R/L*ΔI + 1/L*ΔV
```

**状态空间:**
```
A = [0, 1, 0; a, 0, b; 0, 0, -R/L]
B = [0; 0; 1/L]
C = [1, 0, 0]
```

其中: a = 2kI0²/(mh0³), b = -2kI0/(mh0²)

**典型参数:**
- m = 0.02 kg, k = 1e-5 N.m²/A²
- h0 = 0.02 m, I0 = 1 A, g = 9.81 m/s²

### 4.2 控制策略

#### 4.2.1 简单PID

直接用PID控制悬浮高度，输出为线圈电压。

#### 4.2.2 串级PID

**外环 (位置环):**
```
I_ref = Kp_pos*e_pos + Ki_pos*∫e_pos + Kd_pos*de_pos/dt
```

**内环 (电流环):**
```
V = Kp_cur*e_cur + Ki_cur*∫e_cur
e_cur = I_ref - I_actual
```

**优势:**
- 内环快速响应电流变化
- 外环专注于位置精度
- 结构清晰，调参方便

#### 4.2.3 前馈+PID

```
V = V_pid + V_ff
V_ff = Kff * (I_ff - I0)
I_ff = sqrt(m*g*setpoint²/k)
```

### 4.3 仿真结果

**阶跃响应 (从20mm平衡点到25mm):**

| 控制器 | 上升时间 | 超调量 | 调节时间 | 稳态精度 |
|--------|----------|--------|----------|----------|
| PID | 0.05-0.15s | 5-15% | 0.1-0.3s | 高 |
| 串级PID | 0.03-0.10s | 2-8% | 0.05-0.15s | 很高 |
| FF+PID | 0.04-0.12s | 3-10% | 0.08-0.20s | 高 |

### 4.4 轨迹跟踪

正弦轨迹 (振幅3mm, 频率0.5Hz):

| 控制器 | 跟踪误差 | 相位延迟 |
|--------|----------|----------|
| PID | 0.5-1.0 mm | 明显 |
| 串级PID | 0.2-0.5 mm | 小 |
| FF+PID | 0.3-0.8 mm | 中 |

**结论**: 串级PID在磁悬浮系统中表现最优，前馈补偿有助于改善轨迹跟踪。

## 5. 差速小车路径跟踪

### 5.1 运动学模型

**差速驱动:**
```
v = (vL + vR) / 2      (线速度)
ω = (vR - vL) / d       (角速度)
x' = v*cos(θ)
y' = v*sin(θ)
θ' = ω
```

**参数:** 轮距 d = 0.15 m

### 5.2 控制策略

#### 5.2.1 Pure Pursuit (纯追踪)

**算法:**
1. 在参考路径上找到距离机器人Ld处的目标点
2. 计算机器人到目标点的角度α
3. 控制角速度: ω = v * 2*sin(α) / Ld

**关键参数:** 前视距离Ld

#### 5.2.2 Stanley控制器

**控制律:**
```
δ = -θ_e - arctan(k_cross * e_cross / v)
```
- θ_e: 航向误差
- e_cross: 横向误差
- k_cross: 横向误差增益

**优势:** 同时考虑横向误差和航向误差

#### 5.2.3 PID线跟踪

横向误差作为PID输入，输出角速度。

### 5.3 仿真结果

**S形路径跟踪 (v=0.5 m/s):**

| 控制器 | 平均误差 | 最大误差 | RMS误差 |
|--------|----------|----------|---------|
| Pure Pursuit | 2-5 cm | 5-15 cm | 3-7 cm |
| Stanley | 1-3 cm | 3-8 cm | 2-4 cm |
| PID | 3-8 cm | 8-20 cm | 5-10 cm |

**速度影响 (S形路径):**

| 速度[m/s] | PP误差[cm] | Stanley误差[cm] | PID误差[cm] |
|-----------|------------|-----------------|-------------|
| 0.2 | 1-2 | 0.5-1.5 | 2-4 |
| 0.5 | 2-5 | 1-3 | 3-8 |
| 0.8 | 4-8 | 2-5 | 5-12 |
| 1.0 | 6-12 | 3-7 | 8-18 |

**结论**: Stanley综合性能最优，尤其在转弯处；Pure Pursuit平滑性好；PID实现简单但精度较低。

## 6. 摄像头透视变换

### 6.1 针孔相机模型

**投影方程:**
```
[u]   [f  0 cx] [R t] [X]
[v] = [0  f cy] [   ] [Y]
[1]   [0  0  1]       [Z]
                       [1]
```

**内参矩阵:**
```
K = [fx  0  cx]
    [0  fy  cy]
    [0   0   1]
```

### 6.2 畸变模型

**径向畸变:**
```
x_d = x(1 + k1*r^2 + k2*r^4 + k3*r^6)
y_d = y(1 + k1*r^2 + k2*r^4 + k3*r^6)
```

**切向畸变:**
```
x_d += 2*p1*x*y + p2*(r^2 + 2*x^2)
y_d += p1*(r^2 + 2*y^2) + 2*p2*x*y
```

### 6.3 单应性矩阵求解

#### 6.3.1 DLT (Direct Linear Transform)

**原理:** 4对点确定一个Homography矩阵H

**构建A矩阵:**
```
A = [ -x -y -1  0  0  0  ux uy u ]
    [  0  0  0 -x -y -1  vx vy v ]
```

**求解:** SVD分解，取最小奇异值对应的特征向量

#### 6.3.2 RANSAC鲁棒估计

**算法:**
1. 随机选择4个点对
2. DLT求解H
3. 计算所有点的重投影误差
4. 统计内点数量
5. 重复N次，选择内点最多的结果
6. 用所有内点重新拟合

**优势:** 对噪声和误匹配鲁棒

### 6.4 仿真结果

**4点对应法精度:**
- DLT重投影误差: < 0.01 像素 (精确对应)
- RANSAC: 20%外点时仍能正确估计

**鸟瞰图变换应用:**
- 巡线机器人: 消除近大远小，使黑线在图像中均匀宽度
- 定位系统: 图像坐标 -> 物理坐标，精度<5mm
- 测量系统: 图像两点距离 -> 实际距离，精度<1%

## 7. 参数优化方法

### 7.1 优化算法选择

**差分进化 (Differential Evolution):**
- 全局优化算法，不需要梯度信息
- 适合多参数、非凸、有约束的问题
- 参数: 种群大小popsize，最大迭代次数maxiter

### 7.2 代价函数设计

**典型代价函数:**
```
J = w1*overshoot + w2*rise_time + w3*settling_time + w4*IAE + w5*ITAE
```

**权重选择:**
- 稳定性优先: 增大overshoot和settling_time权重
- 响应速度优先: 增大rise_time权重
- 精度优先: 增大IAE和ITAE权重

### 7.3 参数搜索范围

**PID参数搜索:**
```
Kp: [0.1, 100]
Ki: [0.01, 100]
Kd: [0.001, 10]
```

**LADRC参数搜索:**
```
wc: [10, 500]    (控制器带宽)
wo: [100, 2000]  (观测器带宽，通常是wc的2-5倍)
```

## 8. 鲁棒性分析

### 8.1 参数摄动测试

**测试方法:** 对系统参数施加±10%和±20%的随机摄动，重复20次仿真

**评价指标:**
- 平均性能变化
- 性能标准差
- 失稳概率

### 8.2 各系统鲁棒性

| 系统 | 控制器 | ±10%摄动 | ±20%摄动 |
|------|--------|----------|----------|
| 直流电机 | PID | 稳定，性能降10-20% | 可能失稳 |
| 直流电机 | LADRC | 稳定，性能降5-10% | 稳定，性能降10-20% |
| 直流电机 | ADRC | 稳定，性能降3-8% | 稳定，性能降8-15% |
| 倒立摆 | LQR | 稳定 | 可能失稳 |
| 倒立摆 | SMC | 稳定 | 稳定 |
| 磁悬浮 | 串级PID | 稳定 | 稳定 (裕度小) |

### 8.3 鲁棒性提升策略

1. **增大控制器带宽**: 提高响应速度，但可能引入高频噪声
2. **使用自抗扰控制**: 估计并补偿总扰动，对模型不确定性不敏感
3. **滑模控制**: 在滑模面上系统对不确定性不变
4. **增益调度**: 根据工作点调整参数

## 9. 工程应用指南

### 9.1 电赛典型应用对照

| 电赛题目 | 对应系统 | 推荐控制策略 |
|----------|----------|--------------|
| 智能小车 (2011C) | 差速小车 | Stanley控制器 |
| 旋转倒立摆 (2013C) | 倒立摆 | LQR + SMC |
| 四旋翼飞行器 (2014A) | 磁悬浮类似 | 串级PID |
| 风力摆控制 (2015B) | 倒立摆变种 | LQR |
| 小球滚动控制 (2015J) | 倒立摆 | LQR + 路径规划 |

### 9.2 快速调参流程

**PID整定步骤:**
1. 先设置Ki=0, Kd=0，增大Kp直到系统振荡
2. 将Kp减小到振荡值的0.6倍
3. 逐步增大Ki，消除稳态误差
4. 逐步增大Kd，改善动态响应
5. 微调各参数，满足性能指标

**LADRC整定步骤:**
1. 估计控制增益b0 = Kt/(J*La)
2. 设置控制器带宽wc = 期望带宽
3. 设置观测器带宽wo = 3*wc
4. 根据响应微调wc和wo

### 9.3 嵌入式实现注意

1. **采样周期:** 电机控制1-10kHz，运动控制100-1kHz
2. **数值精度:** 使用定点数或适当缩放避免溢出
3. **中断优先级:** 控制中断高于通信中断
4. **看门狗:** 防止控制程序死锁
