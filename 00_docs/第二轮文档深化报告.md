## 三、故障排查手册实战案例补充 (任务B)

### 3.1 控制问题速查手册

**文档位置**：`nuedc-asset-library/14_solutions/控制问题速查手册.md`

**新增内容**：3个实战案例

#### 案例1: PID振荡的快速诊断和修复
- **问题现象**: 磁悬浮系统Kp=0.3时5Hz振荡
- **诊断过程**: 系统化调整P、I、D参数
- **解决方案**: 降低Kp至0.15，增大积分限幅，增加死区
- **效果**: 振荡消除，高度稳定±0.3cm

#### 案例2: 磁悬浮系统失稳的排查流程
- **问题现象**: 小球在特定高度范围持续振荡
- **排查流程**: 传感器线性度→控制频率→电磁铁响应→系统模型分析
- **解决方案**: 增加前馈补偿和状态观测器

#### 案例3: 电机堵转保护的实现方案
- **问题现象**: 直流电机过载堵转，驱动芯片烧毁
- **问题原因**: 无电流检测、无保护机制
- **解决方案**: INA219电流检测 + 软件超时保护 + 硬件过流保护
- **包含代码**: C语言实现的堵转检测函数

**新增内容量**: 约150行

### 3.2 视觉问题速查手册

**文档位置**：`nuedc-asset-library/14_solutions/视觉问题速查手册.md`

**新增内容**：3个实战案例

#### 案例1: 光照变化导致颜色检测失败的处理
- **问题现象**: 不同光照下颜色识别率从95%降至30%
- **问题分析**: HSV颜色在不同光照下偏移
- **解决方案**: 自适应颜色检测器（AdaptiveColorDetector类）
- **包含代码**: 约100行Python代码
- **效果**: 光照变化时识别率保持85%以上

#### 案例2: 摄像头帧率不足的优化方案
- **问题现象**: 双摄像头同时工作时帧率仅25fps（要求30fps）
- **问题分析**: 格式转换、图像处理耗时过长
- **解决方案**: MJPEG格式 + ROI裁剪 + 降低分辨率
- **包含代码**: 优化后的相机类
- **效果**: 帧率从25fps提升至45fps

#### 案例3: OpenCV内存泄漏的排查方法
- **问题现象**: 连续运行30分钟后内存占用从200MB增至2GB
- **排查流程**: tracemalloc定位 → 发现未释放的Mat → 监控内存使用
- **解决方案**: 显式释放 + 预分配缓冲区 + 定期GC
- **包含代码**: 完整的内存监控和优化代码
- **效果**: 内存使用稳定在200-250MB

**新增内容量**: 约200行

### 3.3 系统问题速查手册

**文档位置**：`nuedc-asset-library/14_solutions/系统问题速查手册.md`

**新增内容**：3个实战案例

#### 案例1: 串口通信丢包的解决方案
- **问题现象**: UART通信丢包率5-10%，导致云台控制不连续
- **根本原因**: 中断优先级低、无流控机制
- **解决方案**: 提高中断优先级 + 帧头/长度/校验协议 + 二进制数据包
- **包含代码**: STM32端协议解析状态机 + Python端可靠发送类
- **效果**: 丢包率降至0.1%以下

#### 案例2: 中断优先级配置不当导致的系统崩溃
- **问题现象**: 高负载时进入HardFault
- **根本原因**: 中断优先级冲突导致死锁
- **解决方案**: 规划优先级层次（紧急保护→实时控制→通信→非实时）
- **包含代码**: 正确的中断优先级配置 + 避免在低优先级中断调用阻塞函数
- **效果**: HardFault完全消除

#### 案例3: 电源纹波对ADC采样的影响
- **问题现象**: ADC读数波动±20个LSB，影响测量精度
- **根本原因**: PWM开关噪声 + 电源线和信号线平行走线
- **解决方案**: 分离模拟地/数字地 + 多采样+中值滤波 + 软件低通滤波
- **包含代码**: ADC滤波函数和LowPassFilter结构体
- **效果**: ADC波动从±20降至±3个LSB

**新增内容量**: 约200行
