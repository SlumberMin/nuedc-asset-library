## 电机选型指南

**文件**: `电机选型指南.md`

### 电机类型对比总览

| 特性 | 直流减速电机 | 步进电机 | 舵机 | 无刷电机(BLDC) |
|------|-------------|---------|------|----------------|
| 速度控制 | PWM调速 | 脉冲频率 | 固定角度 | PWM/FOC |
| 位置控制 | 需编码器 | 开环精确 | 内置闭环 | 需编码器 |
| 扭矩 | 大(减速比) | 中等 | 中小 | 大 |
| 精度 | 低(开环) | 高(步距角) | 高(±1°) | 高(闭环) |
| 响应速度 | 快 | 较慢 | 中等 | 快 |
| 噪声 | 中 | 大(低速振动) | 小 | 小 |
| 价格 | 低 | 中 | 低~中 | 高 |
| 驱动复杂度 | 简单(H桥) | 中等(专用IC) | 简单(PWM) | 复杂(ESC/FOC) |
| 适用场景 | 小车、风扇 | 3D打印、转台 | 转向、云台 | 高性能驱动 |

### 常用直流减速电机

| 型号 | 额定电压 | 减速比 | 空载转速 | 额定扭矩 | 重量 | 适用场景 |
|------|---------|--------|---------|---------|------|---------|
| N20 | 3~6V | 1:34~1:150 | 200~600rpm | 0.05~0.2N·m | 10g | 小型智能车 |
| JGA25-370 | 6~12V | 1:10~1:171 | 35~600rpm | 0.1~1.5N·m | 60g | 中型小车 |
| JGB37-520 | 6~24V | 1:6~1:313 | 10~1000rpm | 0.3~5N·m | 120g | 竞赛小车主力 |
| 775电机 | 12~24V | 需配减速箱 | 3000~12000rpm | 低 | 300g | 风扇、泵 |

### 电机驱动方案

| 方案 | 型号 | 电压 | 电流 | 特点 |
|------|------|------|------|------|
| 单路H桥 | L298N | 5~46V | 2A | 经典，压降大(2V) |
| 双路H桥 | TB6612FNG | 4.5~13.5V | 1.2A | 低损耗，推荐 |
| 4路H桥 | L293D | 4.5~36V | 0.6A | 小电机用 |
| MOS-H桥 | BTS7960 | 6~24V | 43A | 大电流驱动 |
| 驱动板 | DRV8833 | 2.7~10.8V | 1.5A | 低压小车首选 |

### 编码器选择

| 类型 | 精度 | 输出信号 | 推荐型号 | 价格 |
|------|------|---------|----------|------|
| 霍尔编码器 | 13PPR(减速后~1000PPR) | AB正交 | JGA25自带 | 低 |
| 光电编码器 | 100~600PPR | AB正交 | LPD3806 | 中 |
| 磁编码器 | 12bit | SPI/PWM | AS5048A | 高 |

### 选型决策流程

```
需要电机?
├── 需要精确角度控制?
│   ├── 是 → 舵机 (SG90/MG996R)
│   └── 否 → 继续
├── 需要连续旋转+高精度定位?
│   ├── 是 → 步进电机 + 驱动器
│   └── 否 → 继续
├── 需要高速/大功率?
│   ├── 是 → 无刷电机 + ESC
│   └── 否 → 直流减速电机
│       ├── 轻载小车 → N20
│       ├── 中载小车 → JGA25-370
│       └── 重载/竞赛主力 → JGB37-520
```

## PCB设计通用规范

**文件**: `PCB设计通用规范.md`

### 布局规范

| 规则 | 说明 |
|------|------|
| 信号流向 | 输入→处理→输出，单向流动 |
| 去耦电容 | 每个VCC引脚旁放置100nF，靠近引脚 |
| 模拟/数字分区 | 模拟地与数字地单点连接 |
| 大电流路径 | 走线加粗，远离信号线 |

### 布线规范

| 规则 | 说明 |
|------|------|
| 最小线宽 | 信号线≥6mil，电源线≥15mil |
| 过孔 | 信号过孔0.3mm，电源过孔0.5mm |
| 差分对 | USB等差分对等长、等间距 |
| 地平面 | 保持完整地平面，避免割裂 |

### 电赛PCB设计建议

- 优先使用**两层板**（嘉立创免费打样24小时出）
- 预留**测试点**，方便示波器测量
- 关键信号线加**地线保护**
- 接插件放在**板边**，方便接线

### 推荐模块（红榜）

| 模块 | 推荐理由 |
|------|---------|
| TB6612FNG | 低损耗双H桥，驱动能力足 |

### 避雷模块（黑榜）

| 模块 | 问题 |
|------|------|
| L298N | 压降大(2V)，发热严重 |
| 某些廉价舵机 | 一致性差，抖动严重 |
| 红外避障模块 | 受环境光干扰大 |
| 便宜的继电器模块 | 响应慢，触点寿命短 |

## 硬件故障排查手册

**文件**: `硬件故障排查手册.md`

### 常见故障速查表

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| MCU不工作 | 供电/晶振/复位 | 测VCC、检查晶振、检查NRST |
| 电机不转 | 驱动/供电/PWM | 测驱动输入、检查PWM信号 |
| 传感器无数据 | 接线/I2C地址/供电 | 示波器看I2C波形、检查地址 |
| ADC读数不准 | 参考电压/噪声 | 测VREF、加滤波电容 |
| 通信异常 | 波特率/电平/接线 | 逻辑分析仪抓包 |
| 发热严重 | 短路/过流/散热 | 检查PCB、测电流 |

### 排查流程

```
故障现象
├── 完全不工作
│   ├── 检查供电 (万用表测各路电压)
│   ├── 检查复位 (NRST是否被拉低)
│   └── 检查晶振 (示波器看波形)
├── 部分功能异常
│   ├── 隔离模块 (逐个断开排查)
│   ├── 检查信号 (示波器/逻辑分析仪)
│   └── 检查通信 (抓包分析)
└── 性能不达标
    ├── 检查电源纹波
    ├── 检查信号完整性
    └── 检查算法参数
```

## 快速选型决策树

### 主控选型

```
需要什么主控?
├── 纯控制任务 (电机/PWM/ADC)
│   ├── 成本敏感 → STM32F103C8T6
│   └── 性能优先 → STM32F407ZGT6
├── 需要视觉处理
│   ├── 本地处理 → Orange Pi 5 (RK3588S)
│   └── 简单视觉 → OpenMV / ESP32-CAM
└── 需要无线通信
    └── ESP32 / nRF24L01
```

### 传感器选型

```
需要什么传感器?
├── 测距
│   ├── <4m → HC-SR04 (超声波)
│   ├── <12m → VL53L0X (激光ToF)
│   └── 长距离 → TFmini (激光雷达)
├── 角度/加速度
│   └── MPU6050 / ICM-20948
├── 颜色/循迹
│   └── TCS34725 / 红外对管阵列
└── 温度
    └── DS18B20 / 热敏电阻
```
