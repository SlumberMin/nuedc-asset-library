## 二、时钟系统差异

### 2.1 时钟架构

| 特性 | STM32F103 | STM32F407 | MSPM0G3507 | TM4C123 |
|------|-----------|-----------|------------|---------|
| **外部晶振** | 8 MHz | 8-25 MHz | 32 MHz (可选) | 16 MHz |
| **内部振荡器** | 8 MHz HSI | 16 MHz HSI | 32 MHz HFXT | 16 MHz PIOSC |
| **PLL倍频** | ×9=72MHz | ×336/8=168MHz | 32MHz→80MHz | 16MHz→400MHz→80MHz |
| **配置方式** | RCC寄存器 | RCC寄存器 | SysConfig图形化 | SysCtlClockSet() |
| **外设时钟分频** | APB1/2分频 | APB1/2分频 | BUSCLK无分频 | 无分频 |

### 2.2 时钟配置代码对比

```c
// STM32F103
RCC->CR |= RCC_CR_HSEON;  // 使能HSE
RCC->CFGR = RCC_CFGR_PLLSRC_HSE | RCC_CFGR_PLLMULL9;  // 8MHz×9=72MHz
RCC->CR |= RCC_CR_PLLON;  // 启动PLL

// STM32F407
RCC->CR |= RCC_CR_HSEON;
RCC->PLLCFGR = RCC_PLLCFGR_PLLSRC_HSE | (168 << 6) | (4 << 24);  // 168MHz
RCC->CR |= RCC_CR_PLLON;
RCC->CFGR = RCC_CFGR_HPRE_DIV1 | RCC_CFGR_PPRE1_DIV2 | RCC_CFGR_PPRE2_DIV1;

// MSPM0G3507 (SysConfig自动生成)
SYSCFG_DL_init();  // 一行搞定，内部配置PLL 32MHz→80MHz

// TM4C123
SysCtlClockSet(
    SYSCTL_SYSDIV_2_5 |    // 400MHz / 2.5 = 80MHz
    SYSCTL_USE_PLL |       // 使用PLL
    SYSCTL_XTAL_16MHZ |   // 16MHz外部晶振
    SYSCTL_OSC_MAIN        // 主振荡器
);
```

| **定时器** | APB1/2 | APB1/2 | BUSCLK (80MHz) | 80MHz |

| **UART** | APB1 (36MHz) | APB1 (42MHz) | BUSCLK (80MHz) | 80MHz |
| **I2C** | APB1 (36MHz) | APB1 (42MHz) | BUSCLK (80MHz) | 80MHz |

## 三、GPIO差异

### 3.1 端口和引脚

| 特性 | STM32 | MSPM0G3507 | TM4C123 |
|------|-------|------------|---------|
| **端口数量** | 7个 (A-G) | 2个 (A-B) | 6个 (A-F) |
| **每端口引脚** | 16个 (0-15) | 32个 (0-31) | 8个 (0-7) |
| **总引脚数** | 37-80个 | 69个 (LQFP64) | 43个 (LQFP64) |
| **输出速度** | 2/10/50MHz可配置 | 固定 (无需配置) | 固定 (无需配置) |
| **内部上拉/下拉** | 可配置 | 可配置 | 可配置 |
| **复用功能** | AFIO (16种) | SysConfig (8种) | GPIOPinConfigure (固定映射) |

### 3.2 GPIO配置代码对比

```c
// STM32F103
GPIO_InitTypeDef GPIO_InitStruct = {0};
GPIO_InitStruct.Pin = GPIO_PIN_5;
GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;
GPIO_InitStruct.Pull = GPIO_PULLUP;
GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_HIGH;
HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);
HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET);

// STM32F407
__HAL_RCC_GPIOA_CLK_ENABLE();
GPIO_InitStruct.Pin = GPIO_PIN_5;
GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;
GPIO_InitStruct.Pull = GPIO_NOPULL;
GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_VERY_HIGH;
GPIO_InitStruct.Alternate = GPIO_AF1_TIM2;  // 复用功能
HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);

// MSPM0G3507 (SysConfig自动生成配置)
#include "ti_msp_dl_config.h"  // 自动生成的配置
DL_GPIO_setPins(GPIOA, DL_GPIO_PIN_5);  // 置高
DL_GPIO_clearPins(GPIOA, DL_GPIO_PIN_5);  // 置低
uint32_t val = DL_GPIO_readPins(GPIOA, DL_GPIO_PIN_5);  // 读取

// TM4C123
SysCtlPeripheralEnable(SYSCTL_PERIPH_GPIOA);
GPIOPinTypeGPIOOutput(GPIO_PORTA_BASE, GPIO_PIN_5);
GPIOPinWrite(GPIO_PORTA_BASE, GPIO_PIN_5, GPIO_PIN_5);  // 置高
GPIOPinWrite(GPIO_PORTA_BASE, GPIO_PIN_5, 0);  // 置低
uint32_t val = GPIOPinRead(GPIO_PORTA_BASE, GPIO_PIN_5);  // 读取
```

### 3.3 引脚复用映射

| 特性 | STM32 | MSPM0G3507 | TM4C123 |
|------|-------|------------|---------|
| **映射方式** | AFIO寄存器 | SysConfig图形化 | 固定引脚映射表 |
| **灵活性** | ⭐⭐⭐⭐⭐ 高 | ⭐⭐⭐ 中 | ⭐⭐ 低 |
| **配置复杂度** | 中等 | 低 (图形化) | 低 (查表) |
| **典型示例** | PA9→USART1_TX (AF7) | PA10→UART0_TX (SysConfig) | PA1→U0RX (固定) |

## 四、定时器差异

### 4.1 定时器资源

| 特性 | STM32F103 | STM32F407 | MSPM0G3507 | TM4C123 |
|------|-----------|-----------|------------|---------|
| **定时器总数** | 8个 | 14个 | 9个 (TIMG8+TIMA2) | 6个 |
| **高级定时器** | TIM1 | TIM1, TIM8 | TIMA0, TIMA1 | 无 |
| **通用定时器** | TIM2-5 | TIM2-5, TIM9-14 | TIMG0-TIMG7 | Timer0-5 |
| **基本定时器** | TIM6-7 | TIM6-7 | 无 | 无 |
| **位宽** | 16-bit | 16-bit | 16-bit/32-bit | 16/32-bit可选 |
| **最大计数** | 65535 | 65535 | 65535/4294967295 | 65535/4294967295 |
| **PWM通道** | 4ch/定时器 | 4ch/定时器 | 4ch/定时器 | 定时器无PWM |
| **编码器模式** | ✅ | ✅ | ✅ QEI模式 | ❌ (用QEI模块) |

### 4.2 PWM配置对比

| 特性 | STM32 | MSPM0G3507 | TM4C123 |
|------|-------|------------|---------|
| **PWM输出来源** | 定时器通道 | 定时器通道 | 专用PWM模块 (PWM0/PWM1) |
| **最大PWM路数** | 8-14路 (定时器数) | 8路 (TIMG数) | 16路 (2模块×8路) |
| **频率配置** | Prescaler + Period | LoadValue | PWMGenPeriodSet |
| **占空比配置** | CCR寄存器 | CaptureCompareValue | PWMPulseWidthSet |
| **对齐方式** | 边沿/中心 | 边沿/中心 | 边沿对齐 |
| **同步** | 支持 | 支持 | 支持 |

### 4.3 PWM配置代码对比

```c
// STM32F103
TIM_OC_InitTypeDef sConfigOC = {0};
sConfigOC.OCMode = TIM_OCMODE_PWM1;
sConfigOC.Pulse = duty;  // 占空比值
sConfigOC.OCPolarity = TIM_OCPOLARITY_HIGH;
HAL_TIM_PWM_ConfigChannel(&htim3, &sConfigOC, TIM_CHANNEL_1);
HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);

// STM32F407 (类似，支持更多模式)
__HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, duty);  // 运行时修改

// MSPM0G3507
DL_TimerA_setCaptureCompareValue(TIMG0, duty, DL_TIMER_CC_0_INDEX);
// 或用封装宏:
PWM_SET_DUTY(TIMG0, DL_TIMER_CC_0_INDEX, duty);

// TM4C123 (使用专用PWM模块)
SysCtlPeripheralEnable(SYSCTL_PERIPH_PWM0);
PWMClockSet(PWM0_BASE, PWM_SYSCLK_DIV_1);  // 80MHz
PWMGenConfigure(PWM0_BASE, PWM_GEN_0, PWM_GEN_MODE_DOWN);
PWMGenPeriodSet(PWM0_BASE, PWM_GEN_0, period);
PWMPulseWidthSet(PWM0_BASE, PWM_OUT_0, pulse);
PWMOutputState(PWM0_BASE, PWM_OUT_0_BIT, true);
PWMGenEnable(PWM0_BASE, PWM_GEN_0);
```

## 五、编码器接口差异 ⭐重要

### 5.1 编码器硬件对比

| 特性 | STM32 | MSPM0G3507 | TM4C123 |
|------|-------|------------|---------|
| **实现方式** | TIM编码器模式 | TIMG QEI模式 | 专用QEI模块 |
| **硬件解码** | ✅ 自动 | ✅ 自动 | ✅ 自动 |
| **4倍频** | ✅ | ✅ | ✅ |
| **速度测量** | ❌ 需软件 | ❌ 需软件 | ✅ 硬件自动 |
| **位置捕获** | ❌ 需软件 | ❌ 需软件 | ✅ 硬件自动 |
| **CPU占用** | 低 | 低 | **零** |
| **资源占用** | 1个定时器 | 1个TIMG | 专用QEI模块 |
| **最大分辨率** | 65535 | 65535 | 32767 |
| **索引信号** | 支持 (TIx) | 不支持 | 支持 (IDX) |

### 5.2 编码器配置对比

```c
// STM32F103 (TIM3编码器模式)
TIM_Encoder_InitTypeDef encoder = {0};
encoder.EncoderMode = TIM_ENCODERMODE_TI12;  // A/B双通道
encoder.IC1Polarity = TIM_ICPOLARITY_RISING;
encoder.IC1Selection = TIM_ICSELECTION_DIRECTTI;
encoder.IC1Prescaler = TIM_ICPSC_DIV1;
encoder.IC1Filter = 0x0F;
encoder.IC2Polarity = TIM_ICPOLARITY_RISING;
encoder.IC2Selection = TIM_ICSELECTION_DIRECTTI;
encoder.IC2Prescaler = TIM_ICPSC_DIV1;
encoder.IC2Filter = 0x0F;
HAL_TIM_Encoder_Start(&htim3, TIM_CHANNEL_ALL);

// MSPM0G3507 (TIMG QEI模式)
DL_Timer_setCounterMode(TIMG12, DL_TIMER_COUNT_MODE_QEI);
DL_Timer_setLoadValue(TIMG12, period);
DL_Timer_enableCounter(TIMG12);

// TM4C123 (专用QEI模块)
QEIConfigure(QEI0_BASE,
    QEI_CONFIG_CAPTURE_A_B |  // A/B双通道捕获
    QEI_CONFIG_NO_RESET_INDEX |  // 无索引信号复位
    QEI_CONFIG_QUADRATURE |  // 正交模式
    QEI_CONFIG_NO_SWAP,  // 不交换A/B
    ENCODER_PPR * 4 - 1);  // 最大位置(4倍频)
QEIVelocityConfigure(QEI0_BASE, QEI_VELDIV_1, vel_period);
QEIVelocityEnable(QEI0_BASE);
QEIEnable(QEI0_BASE);
```

### 5.3 编码器读取对比

```c
// STM32F103
int16_t encoder_value = (int16_t)__HAL_TIM_GET_COUNTER(&htim3);
__HAL_TIM_SET_COUNTER(&htim3, 0);  // 清零

// MSPM0G3507
int16_t cnt = (int16_t)DL_Timer_getTimerCount(TIMG12);
DL_Timer_setTimerCount(TIMG12, 0);  // 清零

// TM4C123 (硬件自动速度测量)
uint32_t position = QEIPositionGet(QEI0_BASE);  // 位置
int32_t velocity = QEIVelocityGet(QEI0_BASE);  // 速度 (硬件自动)
int32_t direction = QEIDirectionGet(QEI0_BASE);  // 方向 (+1/-1)
```

### 5.4 编码器优劣势总结

| 平台 | 优势 | 劣势 |
|------|------|------|
| **STM32** | 引脚灵活，可配置任意定时器 | 需软件计算速度，占用CPU |
| **MSPM0G3507** | 简单易用，API封装好 | 需软件计算速度，占用CPU |
| **TM4C123** | 硬件自动速度/位置/方向测量，零CPU占用 | QEI引脚固定，灵活性低 |

## 八、FPU和DSP差异

### 8.1 浮点运算对比

| 特性 | STM32F103 | STM32F407 | MSPM0G3507 | TM4C123 |
|------|-----------|-----------|------------|---------|
| **FPU** | ❌ 无 | ✅ 单精度 | ❌ 无 | ✅ 单精度 |
| **DSP** | ❌ 无 | ✅ 有 | ❌ 无 | ❌ 无 |
| **乘法耗时** | ~10周期 (软件) | 1周期 (硬件) | ~10周期 (软件) | 1周期 (硬件) |
| **除法耗时** | ~50周期 (软件) | ~5周期 (硬件) | ~50周期 (软件) | ~5周期 (硬件) |
| **PID计算耗时** | ~10μs | ~1μs | ~10-50μs | ~2μs |
| **FPU使能** | 无需 | FPUEnable() | 无需 | FPUEnable() |

### 8.2 浮点运算代码对比

```c
// STM32F103 (无FPU，使用软件浮点库)
// 编译选项: --float-softlib
float pid_calc(float error) {
    return Kp * error + Ki * integral + Kd * diff;  // 软件模拟
}

// STM32F407 (有FPU)
// 编译选项: -mfloat-abi=hard -mfpu=fpv4-sp-d16
SCB->CPACR |= ((3UL << 10*2) | (3UL << 11*2));  // 启用FPU
float pid_calc(float error) {
    return Kp * error + Ki * integral + Kd * diff;  // 硬件加速
}

// MSPM0G3507 (无FPU，使用软件浮点库)
// 编译选项: --float-softlib
float pid_calc(float error) {
    return Kp * error + Ki * integral + Kd * diff;  // 软件模拟
}

// TM4C123 (有FPU)
FPUEnable();
FPULazyStackingEnable();
float pid_calc(float error) {
    return Kp * error + Ki * integral + Kd * diff;  // 硬件加速
}
```

### 8.3 DSP指令对比

| 平台 | DSP指令 | 典型应用 |
|------|---------|---------|
| **STM32F103** | ❌ 无 | 无 |
| **STM32F407** | ✅ Cortex-M4 DSP | FIR/IIR滤波器、FFT、矩阵运算 |
| **MSPM0G3507** | ❌ 无 | 无 |
| **TM4C123** | ❌ 无 (Cortex-M4无DSP) | 无 |

> **注意**: TM4C123使用Cortex-M4F内核，具有FPU但**没有DSP扩展指令集**。STM32F407的Cortex-M4F才有完整的DSP指令。

## 十、DMA差异

### 10.1 DMA硬件对比

| 特性 | STM32F103 | STM32F407 | MSPM0G3507 | TM4C123 |

### 11.1 API命名规范

### 11.2 API封装层次

| 平台 | 底层寄存器 | HAL/DL层 | 业务层 |
|------|-----------|----------|--------|
| **STM32** | CMSIS | HAL + LL | 用户代码 |
| **MSPM0G3507** | CMSIS | DL库 + SysConfig | 用户代码 |
| **TM4C123** | 直接寄存器 | TivaWare | 用户代码 |

### 11.3 API调用对比

```c
// STM32F103 HAL
HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET);

// MSPM0G3507 DL

DL_ADC12_startConversion(ADC0);
DL_UART_main_transmitData(UART0, data);

// TM4C123 TivaWare
GPIOPinWrite(GPIO_PORTA_BASE, GPIO_PIN_5, GPIO_PIN_5);
PWMPulseWidthSet(PWM0_BASE, PWM_OUT_0, pulse);
ADCProcessorTrigger(ADC0_BASE, 3);
UARTCharPut(UART0_BASE, data);
```

## 十三、电赛选型建议

### 13.1 推荐场景

| 场景 | 推荐平台 | 理由 |
|------|---------|------|
| **平衡车** | TM4C123 | QEI硬件速度测量，FPU加速PID |

| **信号发生器** | STM32F407 | DAC+DMA，DSP指令 |
| **无人机** | STM32F407 | 资源丰富，DSP加速姿态解算 |
| **电源控制** | STM32F103 | 成本低，够用 |
| **多路ADC采集** | TM4C123 或 MSPM0G3507 | ADC性能好 |
| **多串口通信** | TM4C123 | 8个UART+FIFO |
| **初学者** | MSPM0G3507 | SysConfig易上手 |

## 十四、代码迁移指南

### 14.1 从STM32迁移到MSPM0G3507

```c
// 1. 头文件替换
// #include "stm32f1xx_hal.h"
#include "ti/driverlib/driverlib.h"

// 2. 初始化替换
// HAL_Init() + SystemClock_Config()
SYSCFG_DL_init();  // 一行搞定

// 3. GPIO替换
// HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET)
DL_GPIO_setPins(GPIOA, DL_GPIO_PIN_5);

// 4. 定时器替换
// __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, duty)
DL_TimerA_setCaptureCompareValue(TIMG0, duty, DL_TIMER_CC_0_INDEX);

// 5. ADC替换
// HAL_ADC_Start() + HAL_ADC_GetValue()
DL_ADC12_startConversion(ADC0);
while (!DL_ADC12_getRawInterruptStatus(ADC0, DL_ADC12_INTERRUPT_MEM0_RESULT_LOADED)) {}
uint16_t val = DL_ADC12_getMemResult(ADC0, DL_ADC12_MEM_IDX_0);
```

### 14.2 从STM32迁移到TM4C123

```c
// 1. 头文件替换
// #include "stm32f1xx_hal.h"
#include "inc/hw_memmap.h"
#include "driverlib/sysctl.h"
#include "driverlib/gpio.h"

// 2. 初始化替换
// HAL_Init() + SystemClock_Config()
FPUEnable();
SysCtlClockSet(SYSCTL_SYSDIV_2_5 | SYSCTL_USE_PLL | SYSCTL_XTAL_16MHZ | SYSCTL_OSC_MAIN);

// 3. GPIO替换
// HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET)
SysCtlPeripheralEnable(SYSCTL_PERIPH_GPIOA);
GPIOPinTypeGPIOOutput(GPIO_PORTA_BASE, GPIO_PIN_5);
GPIOPinWrite(GPIO_PORTA_BASE, GPIO_PIN_5, GPIO_PIN_5);

// 4. 编码器替换 (重大变更)
// TIM编码器模式 → QEI模块
QEIConfigure(QEI0_BASE, QEI_CONFIG_CAPTURE_A_B | QEI_CONFIG_QUADRATURE, max_pos);
QEIEnable(QEI0_BASE);
uint32_t pos = QEIPositionGet(QEI0_BASE);
int32_t vel = QEIVelocityGet(QEI0_BASE);
```

## 十五、常见问题和解决方案

### 15.1 MSPM0G3507 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| SysConfig配置不生效 | 未重新生成代码 | Build → Generate SysConfig Output |
| 无FPU警告 | 编译选项错误 | 添加 --float-softlib |
| 中断优先级错误 | 超过4级 | 仅使用0-3级 |
| 编码器不工作 | QEI模式配置错误 | 使用DL_TIMER_COUNT_MODE_QEI |
| PWM频率错误 | 时钟配置错误 | 确认BUSCLK=80MHz |

### 15.2 TM4C123 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 编译错误MAP_前缀 | API调用错误 | 添加MAP_前缀 |
| QEI无速度值 | 未使能速度测量 | 调用QEIVelocityEnable() |
| FPU异常 | 使能顺序错误 | 先FPUEnable()再FPULazyStackingEnable() |
| PWM无输出 | 未使能输出 | 调用PWMOutputState() |
| GPIO配置失败 | 外设时钟未使能 | 调用SysCtlPeripheralEnable() |
