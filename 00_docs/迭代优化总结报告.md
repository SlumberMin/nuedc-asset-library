## 二、算法模块清单

### 2.1 PID_V2（改进版PID）
- 微分项一阶低通滤波（α=0.3）
- 条件积分抗饱和（Clamping Anti-Windup）
- 独立积分限幅（与输出限幅分离）
- 死区控制（消除稳态抖动）
- 前馈控制接口
- 增量式PID选项

### 2.2 ADRC自抗扰控制器
- TD跟踪微分器（安排过渡过程）
- ESO扩张状态观测器（估计总扰动）
- NLSEF非线性状态误差反馈
- fal非线性函数
- 带宽法参数整定

### 2.3 卡尔曼滤波器
- 一维卡尔曼滤波
- Q/R可调
- 适用于IMU姿态融合

### 2.4 模糊自适应PID
- 7×7模糊规则表
- 在线调整Kp/Ki/Kd
- 适用于宽范围工作系统

## 四、前沿算法应用建议

| 题目类型 | 推荐算法 | 理由 |
|----------|----------|------|
| 磁悬浮 | ADRC | 模型不确定，扰动大 |
| 倒立摆 | LQR+卡尔曼 | 非线性系统，需要状态估计 |
| 小车循迹 | PID_V2 | 成熟可靠，调参方便 |
| 四旋翼 | ADRC+卡尔曼 | 强鲁棒性+姿态融合 |
| 滚球控制 | 模糊PID | 负载变化大，需要自适应 |
| 风力摆 | PID_V2 | 非线性但模型相对明确 |
