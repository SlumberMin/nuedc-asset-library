## 二、新增算法实现

### 2.1 MRAC 模型参考自适应控制

**文件位置**: 
- `common/mrac.h`
- `common/mrac.c`

**设计特点**:
- ✅ 支持一阶和二阶系统（通过枚举选择）
- ✅ MIT规则自适应律：dθ/dt = -γ*e*xm
- ✅ 控制律：u = θ*r（自适应增益）
- ✅ 离散化实现（欧拉前向法）
- ✅ 嵌入式友好：无动态内存，纯浮点运算
- ✅ 参数整定接口：γ（自适应增益）可在线调整
- ✅ 参考模型带宽可配置
- ✅ 输出限幅和状态复位功能

**核心函数**:
```c
void MRAC_InitFirstOrder(MRAC_t *mrac, float am, float gamma, float dt);
void MRAC_InitSecondOrder(MRAC_t *mrac, float wn, float zeta, float gamma, float dt);
float MRAC_Compute(MRAC_t *mrac, float ref, float y_actual);
void MRAC_SetGamma(MRAC_t *mrac, float gamma);
void MRAC_Reset(MRAC_t *mrac);
```

**应用场景**:
- 参数不确定系统
- 模型不确定系统
- 时变系统控制
- 在线自适应调参

## 六、MRAC 算法设计说明

### 6.1 理论基础

**参考模型**:
- 一阶：dxm/dt = -am*xm + am*r
- 二阶：d²xm/dt² + 2*ζ*ωn*dxm/dt + ωn²*xm = ωn²*r

**自适应律（MIT规则）**:
- dθ/dt = -γ*e*xm
- 其中 e = xm - y（参考输出 - 实际输出）

**控制律**:
- u = θ*r

### 6.2 离散化实现

采用欧拉前向法：
```c
// 一阶系统
xm_new = xm + dt * (-am*xm + am*r);
theta = theta - gamma * e * xm * dt;

// 二阶系统
xn1_new = xn1 + dt * xn2;
xn2_new = xn2 + dt * (-wn²*xn1 - 2*ζ*wn*xn2 + wn²*r);
theta = theta - gamma * e * xn1 * dt;
```

### 6.3 收敛条件

- γ > 0（自适应增益为正）
- 参考模型稳定（am > 0）
- 系统满足正实性条件

### 6.4 参数整定建议

- **γ（自适应增益）**:
  - 初始值：0.1-1.0
  - 增大γ → 收敛更快，但可能不稳定
  - 减小γ → 收敛更慢，但更稳定
  - 建议：从0.5开始，根据实际效果调整

- **参考模型带宽（am或wn）**:
  - 应该比实际系统响应快2-5倍
  - am太大 → 控制信号饱和
  - am太小 → 跟踪误差大

## 七、使用指南

### 7.1 MRAC使用示例

```c
#include "mrac.h"

// 定义MRAC结构体
MRAC_t mrac;

// 一阶系统初始化
MRAC_InitFirstOrder(&mrac, 5.0, 0.5, 0.001f);
// am=5（参考模型时间常数），γ=0.5（自适应增益），dt=0.001s

// 二阶系统初始化
MRAC_InitSecondOrder(&mrac, 10.0, 0.7, 0.5, 0.001f);
// wn=10（自然频率），ζ=0.7（阻尼比），γ=0.5，dt=0.001s

// 在控制循环中
float target = 100.0f;
float measurement = get_sensor_reading();
float output = MRAC_Compute(&mrac, target, measurement);
set_motor_output(output);

// 查看自适应参数
float theta = MRAC_GetTheta(&mrac);
```

## 九、与现有算法库的集成

### 9.1 MRAC与其他算法的对比

| 特性 | PID | LADRC | ADRC | MRAC | LQR |
|------|-----|-------|------|------|-----|
| 需要模型 | 否 | 否 | 否 | 否 | 是 |
| 参数数量 | 3 | 2 | 6 | 2 | 多 |
| 自适应能力 | 否 | 否 | 否 | 是 | 否 |
| 性能 | 中 | 高 | 最高 | 高 | 最优 |
| 实时性 | 高 | 高 | 中 | 高 | 中 |
| 实现难度 | 易 | 易 | 中 | 中 | 中 |

### 9.2 推荐应用场景

- **时间紧张、快速出结果**：PID或LADRC
- **有精确模型、追求最优**：LQR或MPC
- **强干扰环境、鲁棒性要求高**：ADRC或超螺旋SMC
- **非线性时变系统**：模糊PID或神经网络PID
- **参数不确定、需要在线自适应**：MRAC或神经网络PID
- **高精度控制、允许复杂性**：串级PID + 前馈
