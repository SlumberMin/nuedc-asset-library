## 2. 倒立摆系统

### 2.1 LQR控制器最优参数
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| Q[0,0] | 200.0 | - | 角度权重 |
| Q[1,1] | 50.0 | - | 角速度权重 |
| R | 1.0 | - | 控制量权重 |
| K | [-44.7, -12.9] | - | 反馈增益 (典型值) |

**LQR调参原则:**
- 增大Q[0,0]: 加快角度收敛，但控制量增大
- 增大R: 减小控制量，但响应变慢
- Q/R比值: 控制响应速度与控制量的权衡

### 2.2 SMC滑模控制器最优参数
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| c | 15.0 | 1/s | 滑模面斜率 |
| eta | 8.0 | - | 切换增益 |
| phi | 0.1 | - | 边界层宽度 (消除抖振) |

**SMC调参原则:**
- c: 滑模面斜率，决定收敛速度
- eta: 必须足够大以克服不确定性，但过大会加剧抖振
- phi: 边界层越小抖振越明显，越大稳态误差越大

### 2.3 MPC控制器参数
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| N | 15 | 步 | 预测时域 |
| Q | diag([200, 50]) | - | 同LQR |
| R | 1.0 | - | 同LQR |
| dt | 0.01 | s | 离散化步长 |

### 2.4 性能指标对比
| 指标 | LQR | MPC | SMC |
|------|-----|-----|-----|
| 最大偏角[deg] | 1-3 | 1-2 | 2-5 |
| 调节时间[s] | 0.5-1.5 | 0.3-1.0 | 0.8-2.0 |
| 控制量抖振 | 低 | 低 | 中-高 |
| 计算复杂度 | 低 | 高 | 低 |
| 鲁棒性 | 中 | 高 | 高 |

### 3.3 前馈+PID参数
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| Kp | 500.0 | V/m | 同简单PID |
| Ki | 200.0 | V/(m.s) | 同简单PID |
| Kd | 50.0 | V.s/m | 同简单PID |
| Kff | 1.0 | - | 前馈增益 |

**磁悬浮调参要点:**
- 系统开环不稳定，PID参数必须在仿真中先验证

- 前馈: 基于平衡电流 I0 = sqrt(m*g*h0^2/k)

## 4. 差速小车

### 4.1 Pure Pursuit控制器
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| 前视距离Ld | 0.5 | m | 0.3-1.0，速度越快越大 |
| k_velocity | 0.5 | - | 速度增益 |

**Pure Pursuit调参:**
- Ld过小: 控制振荡
- Ld过大: 转弯延迟
- 经验公式: Ld = v * (0.5 ~ 1.5)

### 4.2 Stanley控制器
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| k_cross | 1.0 | 1/m | 横向误差增益 |
| k_heading | 2.0 | - | 航向误差增益 |

**Stanley调参:**
- k_cross: 增大会加速纠正横向误差，但可能振荡
- k_heading: 航向对齐权重

## 5. 摄像头透视变换

### 5.1 典型相机内参 (电赛常用)
| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| fx | 800 | 像素 | X方向焦距 |
| fy | 800 | 像素 | Y方向焦距 |
| cx | 320 | 像素 | 主点X (640宽图像) |
| cy | 240 | 像素 | 主点Y (480高图像) |
| k1 | -0.15 | - | 径向畸变系数1 |
| k2 | 0.03 | - | 径向畸变系数2 |
| p1 | 0.01 | - | 切向畸变系数1 |
| p2 | -0.01 | - | 切向畸变系数2 |

### 5.2 标定流程建议
1. 打印7x9棋盘格，方格边长25mm
2. 拍摄15-20张不同角度的标定图像
3. 使用OpenCV `cv2.findChessboardCorners` 检测角点
4. 使用`cv2.calibrateCamera` 求解内参和畸变系数
5. 使用`cv2.undistort` 去畸变
6. 已知地面4点对应，计算Homography矩阵

### 5.3 单应性矩阵求解建议
- **4点对应法 (DLT)**: 快速，适合已知精确对应的场景
- **RANSAC**: 鲁棒，适合有噪声或误匹配的场景
- **归一化DLT**: 先将点归一化到均值0、方差1，提高数值稳定性

## 快速查表：控制算法选型

| 应用场景 | 推荐算法 | 理由 |
|----------|----------|------|
| 直流电机调速 | LADRC > PID | 扰动抑制更强 |
| 倒立摆平衡 | LQR > MPC | 计算简单，效果好 |
| 磁悬浮 | 串级PID | 工程实现简单 |
| 巡线小车 | Stanley | 横向+航向误差综合 |
| 自动泊车 | Pure Pursuit | 平滑轨迹 |
| 视觉定位 | Homography+DLT | 实时性好 |
