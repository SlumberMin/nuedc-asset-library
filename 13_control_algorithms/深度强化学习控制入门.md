# 深度强化学习控制入门

> DQN / PPO / SAC 在电赛控制场景中的应用指南

---

## 1. 强化学习基本框架

```
环境(Environment) ← 动作(Action) ← 智能体(Agent)
     ↓ 状态 s_t+1                          ↑
     └──────────── 奖励 r_t ───────────────┘
```

**核心要素：**
| 要素 | 含义 | 电赛控制中的对应 |
|------|------|-----------------|
| 状态 s | 环境描述 | 传感器读数（编码器、陀螺仪、电压等） |
| 动作 a | 控制决策 | PWM占空比、电机电压、舵机角度 |
| 奖励 r | 评价指标 | 跟踪误差越小奖励越高、超调越大惩罚越重 |
| 策略 π | s→a 映射 | 控制器本身 |
| 折扣因子 γ | 未来奖励权重 | 通常 0.95~0.99 |

---

## 2. DQN（Deep Q-Network）

### 2.1 原理

```
Q(s, a) = r + γ · max_a' Q(s', a')
```

- 用神经网络逼近 Q 函数
- 离散动作空间专用（如：选择前进/后退/左转/右转）

### 2.2 关键技术

| 技术 | 作用 | 实现 |
|------|------|------|
| 经验回放 | 打破数据相关性 | Replay Buffer 循环采样 |
| 目标网络 | 稳定训练 | 每 C 步硬拷贝 / 软更新 |
| ε-greedy | 探索与利用平衡 | ε 从1.0线性衰减到0.01 |
| Double DQN | 减少Q值过估计 | 用主网络选动作，目标网络估Q值 |

### 2.3 电赛应用示例：平衡车直立控制

```python
import torch
import torch.nn as nn
import numpy as np

class DQNController(nn.Module):
    """平衡车DQN控制器 - 离散档位控制"""
    def __init__(self, state_dim=4, n_actions=7):
        super().__init__()
        # 状态: [角度, 角速度, 位置, 速度]
        # 动作: [-100%, -66%, -33%, 0%, 33%, 66%, 100%] PWM
        self.net = nn.Sequential(
            nn.Linear(state_dim, 128),
            nn.ReLU(),
            nn.Linear(128, 128),
            nn.ReLU(),
            nn.Linear(128, n_actions)
        )
    
    def forward(self, state):
        return self.net(state)

class ReplayBuffer:
    def __init__(self, capacity=10000):
        self.buffer = []
        self.capacity = capacity
        self.pos = 0
    
    def push(self, state, action, reward, next_state, done):
        if len(self.buffer) < self.capacity:
            self.buffer.append(None)
        self.buffer[self.pos] = (state, action, reward, next_state, done)
        self.pos = (self.pos + 1) % self.capacity
    
    def sample(self, batch_size):
        indices = np.random.choice(len(self.buffer), batch_size, replace=False)
        batch = [self.buffer[i] for i in indices]
        states, actions, rewards, next_states, dones = zip(*batch)
        return (torch.FloatTensor(states),
                torch.LongTensor(actions),
                torch.FloatTensor(rewards),
                torch.FloatTensor(next_states),
                torch.FloatTensor(dones))

# 奖励函数设计（关键！）
def balance_reward(angle, angle_dot, position, pos_dot):
    """平衡车奖励函数"""
    # 角度偏差惩罚
    r_angle = -abs(angle) * 10
    # 角速度惩罚（阻尼振荡）
    r_damping = -abs(angle_dot) * 1.0
    # 位置保持奖励（在目标位置附近）
    r_position = -abs(position) * 2
    # 倒地惩罚
    if abs(angle) > 0.5:  # ~30度
        return -100
    return r_angle + r_damping + r_position

# 训练循环
def train_dqn(env, controller, target_net, buffer, 
              episodes=1000, gamma=0.99, lr=1e-3, batch_size=64):
    optimizer = torch.optim.Adam(controller.parameters(), lr=lr)
    epsilon = 1.0
    epsilon_decay = 0.995
    
    for ep in range(episodes):
        state = env.reset()
        total_reward = 0
        
        while not done:
            # ε-greedy 选动作
            if np.random.random() < epsilon:
                action = np.random.randint(7)
            else:
                with torch.no_grad():
                    q_values = controller(torch.FloatTensor(state))
                    action = q_values.argmax().item()
            
            next_state, reward, done, _ = env.step(action)
            buffer.push(state, action, reward, next_state, done)
            state = next_state
            total_reward += reward
            
            # 训练
            if len(buffer) >= batch_size:
                states, actions, rewards, next_states, dones = buffer.sample(batch_size)
                
                q_pred = controller(states).gather(1, actions.unsqueeze(1))
                q_next = target_net(next_states).max(1)[0]
                q_target = rewards + gamma * q_next * (1 - dones)
                
                loss = nn.MSELoss()(q_pred.squeeze(), q_target)
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()
        
        epsilon = max(0.01, epsilon * epsilon_decay)
        # 每10步更新目标网络
        if ep % 10 == 0:
            target_net.load_state_dict(controller.state_dict())
```

### 2.4 DQN局限性

- ❌ 只能处理离散动作
- ❌ 连续控制需要离散化，精度受限
- ❌ 高维状态空间训练不稳定

## 3. PPO（Proximal Policy Optimization）

### 3.1 原理

```
L_CLIP = E[min(r_t(θ)A_t, clip(r_t(θ), 1-ε, 1+ε)A_t)]

r_t(θ) = π_θ(a|s) / π_θ_old(a|s)   (概率比)
```

- **策略梯度**方法，直接优化策略
- 支持**连续动作**空间（高斯策略）
- 通过裁剪限制策略更新幅度，训练稳定

### 3.2 Actor-Critic 架构

```
         ┌─────────────────────────────────────┐
         │         共享特征提取层                │
         │   state → [Linear+ReLU] × 2         │
         └──────────┬──────────┬───────────────┘
                    │          │
              ┌─────▼────┐  ┌─▼──────────┐
              │  Actor   │  │   Critic   │
              │ (策略网络)│  │ (价值网络)  │
              │  → μ, σ  │  │  → V(s)    │
              └─────┬────┘  └─┬──────────┘
                    │          │
              动作a_t      状态价值V(s)
```

### 3.3 电赛应用：倒立摆连续力矩控制

```python
import torch
import torch.nn as nn
from torch.distributions import Normal

class ActorCritic(nn.Module):
    """PPO Actor-Critic for continuous control"""
    def __init__(self, state_dim=4, action_dim=1, hidden=128):
        super().__init__()
        # 共享层
        self.shared = nn.Sequential(
            nn.Linear(state_dim, hidden), nn.Tanh(),
            nn.Linear(hidden, hidden), nn.Tanh()
        )
        # Actor: 输出高斯分布参数
        self.mean = nn.Linear(hidden, action_dim)
        self.log_std = nn.Parameter(torch.zeros(action_dim))
        # Critic: 输出状态价值
        self.value = nn.Linear(hidden, 1)
    
    def forward(self, state):
        features = self.shared(state)
        mean = self.mean(features)
        std = self.log_std.exp().expand_as(mean)
        value = self.value(features)
        return mean, std, value
    
    def get_action(self, state):
        mean, std, value = self.forward(state)
        dist = Normal(mean, std)
        action = dist.sample()
        log_prob = dist.log_prob(action).sum(-1)
        return action, log_prob, value
    
    def evaluate(self, states, actions):
        mean, std, values = self.forward(states)
        dist = Normal(mean, std)
        log_probs = dist.log_prob(actions).sum(-1)
        entropy = dist.entropy().sum(-1)
        return log_probs, values.squeeze(-1), entropy

class PPOTrainer:
    def __init__(self, model, lr=3e-4, gamma=0.99, lam=0.95,
                 clip_eps=0.2, epochs=10, batch_size=64):
        self.model = model
        self.optimizer = torch.optim.Adam(model.parameters(), lr=lr)
        self.gamma = gamma
        self.lam = lam
        self.clip_eps = clip_eps
        self.epochs = epochs
        self.batch_size = batch_size
    
    def compute_gae(self, rewards, values, dones):
        """广义优势估计 GAE(λ)"""
        advantages = []
        gae = 0
        for t in reversed(range(len(rewards))):
            if t == len(rewards) - 1:
                next_val = 0
            else:
                next_val = values[t + 1]
            delta = rewards[t] + self.gamma * next_val * (1 - dones[t]) - values[t]
            gae = delta + self.gamma * self.lam * (1 - dones[t]) * gae
            advantages.insert(0, gae)
        return torch.FloatTensor(advantages)
    
    def update(self, trajectory):
        states = torch.FloatTensor(trajectory['states'])
        actions = torch.FloatTensor(trajectory['actions'])
        old_log_probs = torch.FloatTensor(trajectory['log_probs'])
        rewards = trajectory['rewards']
        dones = trajectory['dones']
        
        with torch.no_grad():
            _, old_values, _ = self.model.evaluate(states, actions)
        
        advantages = self.compute_gae(rewards, old_values.numpy(), dones)
        returns = advantages + old_values
        advantages = (advantages - advantages.mean()) / (advantages.std() + 1e-8)
        
        for _ in range(self.epochs):
            log_probs, values, entropy = self.model.evaluate(states, actions)
            ratio = (log_probs - old_log_probs).exp()
            
            surr1 = ratio * advantages
            surr2 = torch.clamp(ratio, 1 - self.clip_eps, 1 + self.clip_eps) * advantages
            policy_loss = -torch.min(surr1, surr2).mean()
            value_loss = nn.MSELoss()(values, returns)
            
            loss = policy_loss + 0.5 * value_loss - 0.01 * entropy.mean()
            
            self.optimizer.zero_grad()
            loss.backward()
            nn.utils.clip_grad_norm_(self.model.parameters(), 0.5)
            self.optimizer.step()

# 倒立摆奖励函数
def pendulum_reward(theta, theta_dot, torque):
    """倒立摆奖励（角度归一化到[-π, π]）"""
    angle_cost = ((theta + np.pi) % (2*np.pi) - np.pi) ** 2
    velocity_cost = 0.1 * theta_dot ** 2
    torque_cost = 0.001 * torque ** 2
    return -(angle_cost + velocity_cost + torque_cost)
```

## 4. SAC（Soft Actor-Critic）

### 4.1 原理

```
J(π) = E[Σ γ^t (r_t + α · H(π(·|s_t)))]
```

- **最大熵**强化学习：在最大化奖励的同时最大化策略熵
- 自动调节温度参数 α（探索与利用自适应平衡）
- 训练最稳定，适合电赛快速调试

### 4.2 双Q网络

```
Q_target = r + γ · (min(Q1_target, Q2_target) - α·log π(a|s))
```

两个Q网络取较小值，减少过估计。

### 4.3 电赛应用：机械臂轨迹跟踪

```python
class SACActor(nn.Module):
    """SAC 策略网络（高斯混合）"""
    def __init__(self, state_dim=6, action_dim=3, hidden=256):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim, hidden), nn.ReLU(),
            nn.Linear(hidden, hidden), nn.ReLU()
        )
        self.mean = nn.Linear(hidden, action_dim)
        self.log_std = nn.Linear(hidden, action_dim)
    
    def forward(self, state):
        features = self.net(state)
        mean = self.mean(features)
        log_std = self.log_std(features).clamp(-20, 2)
        return mean, log_std
    
    def sample(self, state):
        mean, log_std = self.forward(state)
        std = log_std.exp()
        dist = Normal(mean, std)
        # 重参数化技巧
        x = dist.rsample()
        action = torch.tanh(x)
        # 修正log概率（tanh变换的雅可比）
        log_prob = dist.log_prob(x) - torch.log(1 - action.pow(2) + 1e-6)
        log_prob = log_prob.sum(-1, keepdim=True)
        return action, log_prob, mean

class SACQNetwork(nn.Module):
    """双Q网络"""
    def __init__(self, state_dim=6, action_dim=3, hidden=256):
        super().__init__()
        self.q1 = nn.Sequential(
            nn.Linear(state_dim + action_dim, hidden), nn.ReLU(),
            nn.Linear(hidden, hidden), nn.ReLU(),
            nn.Linear(hidden, 1)
        )
        self.q2 = nn.Sequential(
            nn.Linear(state_dim + action_dim, hidden), nn.ReLU(),
            nn.Linear(hidden, hidden), nn.ReLU(),
            nn.Linear(hidden, 1)
        )
    
    def forward(self, state, action):
        sa = torch.cat([state, action], dim=-1)
        return self.q1(sa), self.q2(sa)

class SACAgent:
    def __init__(self, state_dim, action_dim, lr=3e-4,
                 gamma=0.99, tau=0.005, alpha=0.2, auto_alpha=True):
        self.actor = SACActor(state_dim, action_dim)
        self.q_net = SACQNetwork(state_dim, action_dim)
        self.q_target = SACQNetwork(state_dim, action_dim)
        self.q_target.load_state_dict(self.q_net.state_dict())
        
        self.actor_opt = torch.optim.Adam(self.actor.parameters(), lr=lr)
        self.q_opt = torch.optim.Adam(self.q_net.parameters(), lr=lr)
        
        self.gamma = gamma
        self.tau = tau
        
        # 自动温度调节
        if auto_alpha:
            self.target_entropy = -action_dim
            self.log_alpha = torch.zeros(1, requires_grad=True)
            self.alpha_opt = torch.optim.Adam([self.log_alpha], lr=lr)
        self.auto_alpha = auto_alpha
        self.alpha = alpha
    
    def update(self, batch, buffer):
        states, actions, rewards, next_states, dones = buffer.sample(batch)
        
        # 更新Q网络
        with torch.no_grad():
            next_actions, next_log_probs, _ = self.actor.sample(next_states)
            q1_t, q2_t = self.q_target(next_states, next_actions)
            q_target = torch.min(q1_t, q2_t) - self.alpha * next_log_probs
            td_target = rewards + self.gamma * (1 - dones) * q_target
        
        q1, q2 = self.q_net(states, actions)
        q_loss = nn.MSELoss()(q1, td_target) + nn.MSELoss()(q2, td_target)
        
        self.q_opt.zero_grad()
        q_loss.backward()
        self.q_opt.step()
        
        # 更新策略
        new_actions, log_probs, _ = self.actor.sample(states)
        q1_new, q2_new = self.q_net(states, new_actions)
        policy_loss = (self.alpha * log_probs - torch.min(q1_new, q2_new)).mean()
        
        self.actor_opt.zero_grad()
        policy_loss.backward()
        self.actor_opt.step()
        
        # 自动调节alpha
        if self.auto_alpha:
            alpha_loss = -(self.log_alpha * (log_probs + self.target_entropy).detach()).mean()
            self.alpha_opt.zero_grad()
            alpha_loss.backward()
            self.alpha_opt.step()
            self.alpha = self.log_alpha.exp().item()
        
        # 软更新目标网络
        for p, pt in zip(self.q_net.parameters(), self.q_target.parameters()):
            pt.data.copy_(self.tau * p.data + (1 - self.tau) * pt.data)
```

## 6. 电赛实战流程

### 6.1 仿真→实物迁移

```
Step 1: 建立仿真环境
  ├── 物理模型（角度/角速度/力矩）
  ├── 观测空间设计（归一化到[-1,1]）
  └── 奖励函数设计（试错迭代）

Step 2: 仿真训练
  ├── 选择算法（连续动作→PPO/SAC）
  ├── 训练至收敛（奖励曲线稳定）
  └── 导出模型参数（ONNX/TorchScript）

Step 3: 部署到MCU
  ├── 模型量化（float32→int8）
  ├── C推理代码生成
  └── 周期性控制调度（1ms/10ms）

Step 4: 实物微调
  ├── 对比仿真与实物差异
  ├── 调整观测噪声
  └── 必要时做fine-tune
```

### 6.3 常见问题排查

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| 奖励不增长 | 奖励函数设计不合理 | 检查奖励量级，增加中间奖励 |
| 训练震荡 | 学习率过大 / clip太松 | 降低lr，减小clip ε |
| 仿真→实物差距大 | 仿真模型不准确 | 加入噪声扰动训练(domain randomization) |
| 推理太慢 | 网络太大 | 减少层数/宽度，量化部署 |
| 动作饱和 | 输出范围未限制 | 加tanh激活或clip |
| 探索不足 | 熵太低 | 增大α，增加噪声 |
