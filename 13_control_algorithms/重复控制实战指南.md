# 重复控制实战指南

> 电赛控制算法速查 | 基于内模原理的周期性信号精确跟踪

---

## 一、重复控制核心思想

```
内模原理：要无静差跟踪/抑制某频率信号，控制器中必须包含该信号的数学模型
重复控制：在控制器中嵌入周期信号发生器（纯时延正反馈环）
→ 实现对周期为T的任意信号的零稳态误差跟踪
```

**与ILC对比**：
| 特性 | 重复控制 | ILC |
|---|---|---|
| 实时性 | 逐周期在线修正 | 需要离线迭代 |
| 存储需求 | 只存一个周期 | 存多次迭代 |
| 适用场景 | 连续运行 | 重复试验 |
| 实现难度 | 中等 | 较简单 |

## 二、基本原理

### 2.1 内模结构

```
重复控制器 = 周期内模 + 补偿器

周期内模：G_rc(z) = z^(-N) / (1 - z^(-N))

其中N = T/Ts = 一个周期内的采样点数
物理含义：对误差信号进行逐周期积分
```

### 2.2 完整控制结构

```
r(t) ──→ [+] ──→ 重复控制器 ──→ [+] ──→ 被控对象 ──→ y(t)
          ↑-                                     ↓
          └───────────────────────────────────────┘

u_rc(k) = u_rc(k-N) + Q(z) * [Kr * e(k-N) + 补偿项]
```

## 三、Q滤波器设计

### 3.1 为什么需要Q滤波器

```
理想重复控制（Q=1）：z^N = 1处有N个极点在单位圆上 → 临界稳定
Q滤波器作用：将这些极点拉入单位圆内 → 保证稳定
代价：牺牲高频段的跟踪精度
```

### 3.2 常用Q滤波器形式

```c
// 方案1：低通滤波器（推荐）
// Q(z) = (α + 2α*z⁻¹ + α*z⁻²) / (1 + β₁*z⁻¹ + β₂*z⁻²)
// 简化为零相移FIR均值：
// Q(z) = (1/4) * z⁻¹ + (1/2) + (1/4) * z  （三点均值，零相移）

// 方案2：常数Q（最常用，实现最简单）
float Q = 0.95f;  // 0.9~0.98，越接近1精度越高但稳定性越差

// 方案3：截止频率可调的移动平均
float Q_filter(float x[], int idx, int N) {
    // 5点移动平均
    float sum = 0;
    for (int i = -2; i <= 2; i++) {
        sum += x[(idx + i + N) % N];
    }
    return sum / 5.0f;
}
```

### 3.3 Q值选择指南

| 应用场景 | Q值建议 | 说明 |
|---|---|---|
| 高精度、低噪声 | 0.97~0.99 | 跟踪精度高 |
| 一般场合 | 0.93~0.97 | 平衡精度与稳定性 |
| 噪声大/模型不准 | 0.85~0.93 | 优先保证稳定 |
| 初次调试 | 0.90 | 安全起步 |

## 四、补偿器设计

### 4.1 相位超前补偿

```c
// z^m：m步超前，补偿系统相位滞后
// 实际实现：用数组索引偏移

// 补偿器 C(z) = Kr * z^m
// Kr：增益补偿（|G_p(jω)|的倒数近似）
// m：相位补偿步数（系统群延迟近似值）

// 经验取法：
// Kr ≈ 1 / |G_p(e^(jω₀))|  （在基频处）
// m ≈ round(∠G_p(e^(jω₀)) / (2π/N))
```

### 4.2 简化补偿器（电赛实用）

```c
// 大多数电赛场合，用比例补偿+超前1~3步即可
float Kr = 1.0f;   // 先试1，不行再调
int   m  = 2;      // 超前步数，试凑确定

// 调试方法：
// 从m=0开始，逐步增大，观察误差收敛速度
// 误差最小时的m即为最佳超前步数
```

## 五、完整实现模板

```c
/*=============================================================
 * 重复控制器 - 适用于周期性信号跟踪/抑制
 * 周期: N个采样点
 *=============================================================*/
#define N_PERIOD  200      // 一个周期的采样点数

float err_buf[N_PERIOD];   // 误差延迟缓冲区
float u_buf[N_PERIOD];     // 控制量延迟缓冲区
int   buf_idx = 0;         // 缓冲区索引

float Q  = 0.95f;          // Q滤波器系数
float Kr = 0.8f;           // 补偿增益
int   m  = 2;              // 超前步数

// 获取缓冲区中第(offset)步前的值（环形缓冲）
float buf_get(float *buf, int offset) {
    int idx = (buf_idx - offset + N_PERIOD) % N_PERIOD;
    return buf[idx];
}

float RepetitiveControl(float ref, float y_meas) {
    float error = ref - y_meas;
    float u_rc_prev = buf_get(u_buf, N_PERIOD);   // u(k-N)
    float e_prev    = buf_get(err_buf, N_PERIOD - m); // e(k-N+m)

    // 重复控制律
    float u_rc = Q * u_rc_prev + Kr * e_prev;

    // 更新缓冲区
    err_buf[buf_idx] = error;
    u_buf[buf_idx]   = u_rc;
    buf_idx = (buf_idx + 1) % N_PERIOD;

    return u_rc;
}

// 与PID结合使用（推荐）
float RepetitiveControl_withPID(float ref, float y_meas) {
    float error = ref - y_meas;

    // PID反馈（保证暂态性能和鲁棒性）
    float u_pid = Kp * error + Ki * integral + Kd * derivative;

    // 重复控制前馈（消除周期性误差）
    float u_rc = RepetitiveControl(ref, y_meas);

    return u_pid + u_rc;
}
```

## 六、稳定性分析

### 6.1 小增益定理条件

```
重复控制系统稳定的充分条件：

|Q(e^jω) * [1 - Kr * e^(-jωm) * G_p(e^jω)]| < 1,  ∀ω

实际含义：在每个频率点上，开环增益小于1
```

### 6.2 实用稳定性判据

```c
// 简化判断：确保以下三点
// 1. |Q| < 1（通常满足，Q取0.9~0.98）
// 2. |Kr * G_p| ≈ 1（增益匹配）
// 3. 相位误差 < 90°（通过超前补偿m调整）

// 如果不确定，保守取 Q=0.9, Kr=0.5
```

### 6.3 稳定性裕度估算

```
稳定裕度 γ = 1 - max_ω |Q * (1 - C*G_p)|
γ > 0 即稳定，γ越大越稳定

经验值：γ > 0.3 为较安全的稳定裕度
```

## 七、电赛典型应用场景

### 7.1 逆变器输出波形控制

```
任务：输出50Hz标准正弦波
采样率：10kHz → N = 200
重复控制器消除谐波畸变
典型参数：Q=0.95, Kr=0.8, m=2
效果：THD从5%降至<1%
```

### 7.2 有源电力滤波器（APF）

```
任务：补偿电网谐波电流
特点：周期为基波周期（50Hz→N=Fs/50）
重复控制对各次谐波同时有效
典型参数：Q=0.93, Kr=0.7, m=3
```

### 7.3 电机转速脉动抑制

```
任务：抑制电机固有转矩脉动（周期性）
周期 = 电气周期 = 60/(p*n) 秒（p为极对数，n为转速rps）
典型参数：Q=0.95, Kr=0.6, m=1
```

### 7.4 伺服系统周期性跟踪

```
任务：跟踪正弦/三角波参考信号
周期 = 信号周期 / 采样周期
与PID结合：PID负责暂态+抗扰，重复控制消除周期误差
```

## 八、调试流程

```
Step 1: 纯PID调好基本响应（保证系统稳定可运行）
Step 2: 接入重复控制，设Q=0.9, Kr=0.5, m=0
Step 3: 逐步增大Kr（0.5→0.7→0.9），观察误差收敛
Step 4: 调整m（0→1→2→3），找到误差最小的超前步数
Step 5: 微调Q（0.9→0.95→0.98），提高精度
Step 6: 如果出现振荡，回退一步降低Kr或Q
```

## 九、常见问题与对策

| 问题 | 原因 | 对策 |
|---|---|---|
| 误差不收敛 | 补偿器相位不对 | 调整m值 |
| 收敛但有残差 | Q值太低/增益不够 | 增大Q或Kr |
| 高频振荡 | Q值太高/增益过大 | 降低Q或Kr |
| 起始几个周期误差大 | 重复控制需要1~2个周期预热 | 正常现象，加PID平滑 |
| 周期变化后失效 | N值固定不匹配 | 在线估算周期或自适应N |
| 噪声累积 | D型补偿放大噪声 | 加Q滤波或降低Kr |

## 十、与ILC的选择建议

```
选重复控制当：
  ✓ 系统连续运行（如逆变器、APF）
  ✓ 不能中断实验（在线修正）
  ✓ 周期可能微调

选ILC当：
  ✓ 可以重复实验（如机械臂示教）
  ✓ 存储空间充足
  ✓ 需要更快的收敛速度
```
