# 电赛控制题 - 算法选型指南

## 一、算法对比总览

| 算法 | 复杂度 | 计算量 | 模型依赖 | 抗扰性 | 最优性 | 适用场景 |
|------|--------|--------|----------|--------|--------|----------|
| **PID_V2** | 低 | 低 | 无 | 中 | 无 | 通用，首选 |
| **ADRC** | 中 | 中 | 极低 | 强 | 无 | 模型不确定，扰动大 |
| **LQR** | 中 | 低 | 需要 | 中 | 最优 | 多状态系统 |
| **MPC** | 高 | 高 | 需要 | 强 | 最优 | 有约束系统 |
| **FuzzyPID** | 中 | 中 | 无 | 中 | 无 | 负载变化大 |
| **Kalman** | 中 | 中 | 需要 | - | 最优 | 状态估计 |
| **互补滤波** | 低 | 极低 | 无 | - | - | IMU融合 |

## 二、按题目类型选型

### 2.1 小车循迹类
**推荐：PID_V2**
- 理由：成熟可靠，调参方便，计算量小
- 备选：FuzzyPID（路面变化大时）

### 2.2 磁悬浮类
**推荐：ADRC 或 PID_V2**
- 理由：模型不确定，扰动大
- ADRC优势：ESO实时估计扰动
- PID_V2优势：简单可靠

### 2.3 倒立摆类
**推荐：LQR + Kalman**
- 理由：多状态系统，需要状态估计
- LQR优势：最优控制，理论保证稳定
- Kalman优势：融合编码器和IMU

### 2.4 滚球控制类
**推荐：PID_V2 或 FuzzyPID**
- 理由：双轴控制，负载变化
- FuzzyPID优势：自适应不同负载

### 2.5 四旋翼类
**推荐：ADRC + 互补滤波**
- 理由：强鲁棒性，实时性要求高
- ADRC优势：不依赖精确模型
- 互补滤波优势：计算量小

### 2.6 风力摆类
**推荐：PID_V2**
- 理由：非线性但模型相对明确
- 备选：ADRC（需要更强鲁棒性时）

## 三、参数整定方法

### 3.1 PID_V2
1. 先调Kp：增大至系统出现振荡，取60%
2. 再调Ki：消除稳态误差
3. 最后调Kd：抑制超调和振荡
4. 设置d_filter_alpha=0.3（微分滤波）
5. 设置dead_zone（消除稳态抖动）

### 3.2 ADRC（带宽法）
1. 估计b0（系统增益）
2. 设置ESO带宽ωo=5~20
3. 设置控制器带宽ωc=ωo/3~5
4. β01=3ωo, β02=3ωo², β03=ωo³
5. kp=ωc², kd=2ωc

### 3.3 LQR（MATLAB辅助）
1. 建立状态空间模型A,B
2. 定义权重矩阵Q,R
3. 在MATLAB中求解：K=lqr(A,B,Q,R)
4. 将K值复制到STM32代码

### 3.4 Kalman
1. 设置Q（过程噪声）=0.01
2. 设置R（观测噪声）=0.1
3. 根据实际效果调整Q/R比值

## 四、集成建议

### 4.1 最小可行方案
只用PID_V2，覆盖80%的题目需求

### 4.2 标准方案
PID_V2 + Kalman/互补滤波 + 编码器速度闭环

### 4.3 高级方案
ADRC/LQR + Kalman + FuzzyPID + 编码器速度闭环

---

**建议**：根据题目复杂度和团队能力选择算法。电赛现场时间紧张，优先保证基本功能，再考虑高级算法。
