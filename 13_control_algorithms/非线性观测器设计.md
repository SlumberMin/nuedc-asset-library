# 非线性观测器设计

> nuedc-asset-library · 控制算法库 · 非线性观测器

## 1. 概述

### 1.1 为什么需要非线性观测器

- 系统模型是非线性的（电机、倒立摆、无人机等）
- 需要估计无法直接测量的状态（速度、角度、角速度等）
- 需要滤除传感器噪声
- 需要融合多个传感器数据

### 1.2 观测器分类

| 观测器 | 类型 | 适用场景 | 计算量 |
|--------|------|---------|--------|
| Luenberger | 线性 | 线性系统/小偏差线性化 | 最小 |
| 卡尔曼滤波 (KF) | 线性最优 | 线性高斯系统 | 小 |
| **扩展卡尔曼滤波 (EKF)** | 非线性 | 一般非线性系统 | 中 |
| **无迹卡尔曼滤波 (UKF)** | 非线性 | 强非线性系统 | 中偏大 |
| **粒子滤波 (PF)** | 非线性非高斯 | 任意非高斯分布 | 大 |
| **滑模观测器 (SMO)** | 非线性鲁棒 | 鲁棒估计/未知输入 | 小 |

## 2. 扩展卡尔曼滤波 (EKF)

### 2.1 系统模型

```
连续时间:
  ẋ = f(x, u) + w    (过程方程)
  z = h(x, u) + v    (观测方程)

离散时间:
  x[k] = f_d(x[k-1], u[k-1]) + w[k-1]
  z[k] = h_d(x[k], u[k]) + v[k]

其中:
  w ~ N(0, Q)  过程噪声
  v ~ N(0, R)  观测噪声
```

### 2.2 EKF 算法流程

```
初始化:
  x̂[0] = 初始状态估计
  P[0]  = 初始协方差

每个时间步:
  ──── 预测 (Prediction) ────
  1. 计算雅可比矩阵:
     F[k-1] = ∂f/∂x |_{x̂[k-1], u[k-1]}
  
  2. 状态预测:
     x̂⁻[k] = f(x̂[k-1], u[k-1])
  
  3. 协方差预测:
     P⁻[k] = F[k-1] · P[k-1] · F[k-1]ᵀ + Q

  ──── 更新 (Update) ────
  4. 计算观测雅可比:
     H[k] = ∂h/∂x |_{x̂⁻[k]}
  
  5. 卡尔曼增益:
     K[k] = P⁻[k] · H[k]ᵀ · (H[k] · P⁻[k] · H[k]ᵀ + R)⁻¹
  
  6. 状态更新:
     x̂[k] = x̂⁻[k] + K[k] · (z[k] - h(x̂⁻[k]))
  
  7. 协方差更新:
     P[k] = (I - K[k] · H[k]) · P⁻[k]
```

### 2.3 Python 实现

```python
import numpy as np

class EKF:
    def __init__(self, n_states, n_obs):
        self.n = n_states
        self.m = n_obs
        self.x = np.zeros(n_states)      # 状态估计
        self.P = np.eye(n_states) * 10    # 协方差
        self.Q = np.eye(n_states) * 0.01  # 过程噪声
        self.R = np.eye(n_obs) * 0.1      # 观测噪声
    
    def f(self, x, u):
        """状态转移函数 f(x, u) - 需要用户重写"""
        raise NotImplementedError
    
    def h(self, x, u):
        """观测函数 h(x, u) - 需要用户重写"""
        raise NotImplementedError
    
    def F_jacobian(self, x, u):
        """状态雅可比矩阵 ∂f/∂x - 需要用户重写"""
        raise NotImplementedError
    
    def H_jacobian(self, x, u):
        """观测雅可比矩阵 ∂h/∂x - 需要用户重写"""
        raise NotImplementedError
    
    def predict(self, u):
        """预测步骤"""
        F = self.F_jacobian(self.x, u)
        self.x = self.f(self.x, u)
        self.P = F @ self.P @ F.T + self.Q
    
    def update(self, z, u):
        """更新步骤"""
        H = self.H_jacobian(self.x, u)
        
        # 卡尔曼增益
        S = H @ self.P @ H.T + self.R
        K = self.P @ H.T @ np.linalg.inv(S)
        
        # 新息 (innovation)
        y = z - self.h(self.x, u)
        
        # 状态更新
        self.x = self.x + K @ y
        
        # 协方差更新 (Joseph 形式更稳定)
        I_KH = np.eye(self.n) - K @ H
        self.P = I_KH @ self.P @ I_KH.T + K @ self.R @ K.T
    
    def step(self, z, u=None):
        """一步滤波: 预测 + 更新"""
        if u is None:
            u = np.zeros(1)
        self.predict(u)
        self.update(z, u)
        return self.x.copy()
```

### 2.4 电赛实例：角度估计（陀螺仪+加速度计融合）

```python
import numpy as np

class AngleEKF(EKF):
    """双轴角度估计: 状态 = [角度, 角速度偏置]"""
    
    def __init__(self, dt=0.01, process_noise=0.01, gyro_noise=0.001, accel_noise=0.1):
        super().__init__(n_states=2, n_obs=1)
        self.dt = dt
        
        # 噪声协方差
        self.Q = np.array([
            [process_noise, 0],
            [0, 0.0001]
        ])
        self.R = np.array([[accel_noise]])
        
        # 初始状态
        self.x = np.array([0.0, 0.0])
        self.P = np.diag([1.0, 0.1])
    
    def f(self, x, u):
        """x[0] = angle, x[1] = gyro_bias
        u[0] = gyro_rate (rad/s)"""
        angle = x[0] + (u[0] - x[1]) * self.dt
        bias = x[1]  # 偏置随机游走
        return np.array([angle, bias])
    
    def h(self, x, u):
        """观测: 加速度计计算的角度"""
        return np.array([x[0]])
    
    def F_jacobian(self, x, u):
        return np.array([
            [1, -self.dt],
            [0, 1]
        ])
    
    def H_jacobian(self, x, u):
        return np.array([[1, 0]])


# 使用示例
ekf = AngleEKF(dt=0.01)

# 模拟数据
for i in range(1000):
    gyro_rate = 0.1 + np.random.normal(0, 0.01)   # 陀螺仪
    accel_angle = 0.001 * i + np.random.normal(0, 0.05)  # 加速度计
    
    state = ekf.step(np.array([accel_angle]), np.array([gyro_rate]))
    
    if i % 100 == 0:
        print(f"t={i*0.01:.2f}s  角度={state[0]:.4f} rad  偏置={state[1]:.4f} rad/s")
```

### 2.5 电赛实例：直流电机状态估计

```python
class DCMotorEKF(EKF):
    """直流电机: 状态=[电流, 转速], 输入=[电压], 观测=[转速]"""
    
    def __init__(self, dt=0.001, R=1.0, L=0.001, Ke=0.01, Kt=0.01, J=0.001, B=0.0001):
        super().__init__(n_states=2, n_obs=1)
        self.dt = dt
        self.R_ = R      # 电阻
        self.L = L        # 电感
        self.Ke = Ke      # 反电势常数
        self.Kt = Kt      # 转矩常数
        self.J = J        # 转动惯量
        self.B = B        # 摩擦系数
        
        self.Q = np.diag([0.1, 0.01])
        self.R = np.array([[0.1]])
    
    def f(self, x, u):
        """电流 x[0], 转速 x[1], 输入电压 u[0]"""
        i = x[0]
        w = x[1]
        V = u[0]
        
        di = (-self.R_*i - self.Ke*w + V) / self.L
        dw = (self.Kt*i - self.B*w) / self.J
        
        return np.array([i + di*self.dt, w + dw*self.dt])
    
    def h(self, x, u):
        return np.array([x[1]])  # 观测转速
    
    def F_jacobian(self, x, u):
        dt = self.dt
        return np.array([
            [1 - self.R_/self.L*dt, -self.Ke/self.L*dt],
            [self.Kt/self.J*dt,     1 - self.B/self.J*dt]
        ])
    
    def H_jacobian(self, x, u):
        return np.array([[0, 1]])
```

### 2.6 数值雅可比（自动微分）

```python
def numerical_jacobian(f, x, u, eps=1e-6):
    """数值计算雅可比矩阵"""
    n = len(x)
    F = np.zeros((n, n))
    f0 = f(x, u)
    for j in range(n):
        x_plus = x.copy()
        x_plus[j] += eps
        F[:, j] = (f(x_plus, u) - f0) / eps
    return F
```

## 3. 无迹卡尔曼滤波 (UKF)

### 3.1 核心思想

不使用雅可比线性化，而是选取一组 **Sigma 点** 来捕捉均值和协方差，通过非线性函数传播这些点。

### 3.2 Sigma 点生成

对于 n 维状态，生成 2n+1 个 Sigma 点：

```
χ[0] = x̂
χ[i] = x̂ + (√((n+λ)P))_i     for i = 1, ..., n
χ[i] = x̂ - (√((n+λ)P))_{i-n}  for i = n+1, ..., 2n

其中:
  λ = α²(n+κ) - n
  α: 通常取 1e-4 ~ 1
  κ: 通常取 0 或 3-n
  β: 2 (高斯分布最优)
```

### 3.3 UKF 算法

```python
class UKF:
    def __init__(self, n_states, n_obs, alpha=0.1, beta=2, kappa=0):
        self.n = n_states
        self.m = n_obs
        self.x = np.zeros(n_states)
        self.P = np.eye(n_states)
        self.Q = np.eye(n_states) * 0.01
        self.R = np.eye(n_obs) * 0.1
        
        # UKF 参数
        self.alpha = alpha
        self.beta = beta
        self.kappa = kappa
        self.lam = alpha**2 * (n_states + kappa) - n_states
        
        # 权重
        n = n_states
        lam = self.lam
        self.Wm = np.zeros(2*n + 1)
        self.Wc = np.zeros(2*n + 1)
        self.Wm[0] = lam / (n + lam)
        self.Wc[0] = lam / (n + lam) + (1 - alpha**2 + beta)
        for i in range(1, 2*n + 1):
            self.Wm[i] = 1 / (2 * (n + lam))
            self.Wc[i] = 1 / (2 * (n + lam))
    
    def sigma_points(self, x, P):
        """生成 Sigma 点"""
        n = self.n
        lam = self.lam
        
        # 计算矩阵平方根 (Cholesky 分解)
        sqrt_P = np.linalg.cholesky((n + lam) * P)
        
        chi = np.zeros((2*n + 1, n))
        chi[0] = x
        for i in range(n):
            chi[i + 1]     = x + sqrt_P[i]
            chi[n + i + 1] = x - sqrt_P[i]
        return chi
    
    def step(self, z, f_func, h_func, u=None):
        """一步 UKF 滤波"""
        if u is None:
            u = np.zeros(1)
        
        # 生成 Sigma 点
        chi = self.sigma_points(self.x, self.P)
        
        # ---- 预测 ----
        chi_pred = np.array([f_func(chi[i], u) for i in range(2*self.n + 1)])
        
        # 预测均值
        x_pred = np.sum(self.Wm[:, None] * chi_pred, axis=0)
        
        # 预测协方差
        P_pred = self.Q.copy()
        for i in range(2*self.n + 1):
            dx = chi_pred[i] - x_pred
            P_pred += self.Wc[i] * np.outer(dx, dx)
        
        # ---- 更新 ----
        # 观测 Sigma 点
        Z = np.array([h_func(chi_pred[i], u) for i in range(2*self.n + 1)])
        
        # 观测均值
        z_pred = np.sum(self.Wm[:, None] * Z, axis=0)
        
        # 观测协方差 + 互协方差
        S = self.R.copy()
        Pxz = np.zeros((self.n, self.m))
        for i in range(2*self.n + 1):
            dz = Z[i] - z_pred
            dx = chi_pred[i] - x_pred
            S += self.Wc[i] * np.outer(dz, dz)
            Pxz += self.Wc[i] * np.outer(dx, dz)
        
        # 卡尔曼增益
        K = Pxz @ np.linalg.inv(S)
        
        # 更新
        self.x = x_pred + K @ (z - z_pred)
        self.P = P_pred - K @ S @ K.T
        
        return self.x.copy()
```

### 3.4 EKF vs UKF 对比

| 特性 | EKF | UKF |
|------|-----|-----|
| 线性化方式 | 雅可比矩阵 | Sigma 点 |
| 需要计算导数 | 是 | 否 |
| 强非线性性能 | 差（二阶误差） | 好（至少二阶精度） |
| 计算量 | O(n³) | O(n³) |
| 实现难度 | 需要推导雅可比 | 只需 f 和 h |
| 电赛推荐 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

## 4. 粒子滤波 (PF)

### 4.1 核心思想

用一组带权重的随机样本（粒子）表示概率分布，适用于任意非线性、非高斯系统。

### 4.2 算法流程

```
初始化: 从先验分布采样 N 个粒子

对于每个时间步 k:
  1. 预测: 每个粒子通过状态方程传播
     x_i[k] = f(x_i[k-1], u[k-1]) + 噪声
  
  2. 权重更新: 根据观测似然更新权重
     w_i = w_i * p(z[k] | x_i[k])
  
  3. 归一化: w_i = w_i / Σw_i
  
  4. 重采样 (如果有效粒子数过少):
     N_eff = 1 / Σ(w_i²)
     if N_eff < N_threshold: 执行重采样
  
  5. 估计: x̂ = Σ(w_i * x_i)
```

### 4.3 Python 实现

```python
import numpy as np

class ParticleFilter:
    def __init__(self, n_particles, n_states, process_noise, obs_noise):
        self.N = n_particles
        self.n = n_states
        self.process_noise = process_noise
        self.obs_noise = obs_noise
        
        # 初始化粒子 (均匀分布在合理范围内)
        self.particles = np.random.uniform(-1, 1, (n_particles, n_states))
        self.weights = np.ones(n_particles) / n_particles
    
    def predict(self, f_func, u=None):
        """粒子传播"""
        for i in range(self.N):
            noise = np.random.multivariate_normal(
                np.zeros(self.n), self.process_noise
            )
            self.particles[i] = f_func(self.particles[i], u) + noise
    
    def update_weights(self, z, h_func):
        """权重更新: 基于高斯似然"""
        for i in range(self.N):
            z_pred = h_func(self.particles[i])
            innovation = z - z_pred
            # 高斯似然
            det_R = np.linalg.det(self.obs_noise)
            inv_R = np.linalg.inv(self.obs_noise)
            likelihood = np.exp(-0.5 * innovation @ inv_R @ innovation) / \
                        np.sqrt((2*np.pi)**len(z) * det_R)
            self.weights[i] *= likelihood
        
        # 避免退化
        self.weights += 1e-300
        self.weights /= np.sum(self.weights)
    
    def resample(self):
        """系统重采样"""
        N_eff = 1.0 / np.sum(self.weights**2)
        if N_eff < self.N * 0.5:
            indices = self._systematic_resample()
            self.particles = self.particles[indices]
            self.weights = np.ones(self.N) / self.N
    
    def _systematic_resample(self):
        positions = (np.arange(self.N) + np.random.uniform()) / self.N
        cumulative = np.cumsum(self.weights)
        indices = np.zeros(self.N, dtype=int)
        i, j = 0, 0
        while i < self.N:
            if positions[i] < cumulative[j]:
                indices[i] = j
                i += 1
            else:
                j += 1
        return indices
    
    def estimate(self):
        """加权平均估计"""
        return np.average(self.particles, weights=self.weights, axis=0)
    
    def step(self, z, f_func, h_func, u=None):
        self.predict(f_func, u)
        self.update_weights(z, h_func)
        self.resample()
        return self.estimate()
```

### 4.4 粒子滤波适用场景

- 系统高度非线性，EKF/UKF 效果差
- 噪声分布非高斯（多峰、重尾等）
- 状态空间维度 ≤ 5（高维退化严重）
- 电赛中较少用到，但了解原理有助于理解其他滤波器

## 5. 滑模观测器 (SMO)

### 5.1 核心思想

利用不连续的切换项（符号函数）强制观测误差收敛到滑模面，对模型不确定性和外部干扰具有鲁棒性。

### 5.2 基本滑模观测器

考虑系统：
```
ẋ = Ax + Bu + d    (d 为未知扰动)
y = Cx
```

观测器设计：
```
ẋ̂ = Ax̂ + Bu + L(y - Cx̂) + K·sign(y - Cx̂)
```

其中 L 为线性增益，K 为切换增益。

### 5.3 Python 实现

```python
import numpy as np

class SlidingModeObserver:
    def __init__(self, A, B, C, L, K, dt=0.001):
        """
        A, B, C: 系统矩阵
        L: 线性观测增益
        K: 切换增益
        """
        self.A = np.array(A, dtype=float)
        self.B = np.array(B, dtype=float)
        self.C = np.array(C, dtype=float)
        self.L = np.array(L, dtype=float)
        self.K = np.array(K, dtype=float)
        self.dt = dt
        
        self.x_hat = np.zeros(A.shape[0])
    
    def sign(self, x):
        """符号函数 (带死区避免抖振)"""
        epsilon = 0.01
        return np.where(np.abs(x) < epsilon, x / epsilon, np.sign(x))
    
    def step(self, y, u):
        """一步观测"""
        y_hat = self.C @ self.x_hat
        e = y - y_hat  # 观测误差
        
        # 观测器动力学
        dx = (self.A @ self.x_hat + self.B @ u +
              self.L @ e + self.K @ self.sign(e))
        
        self.x_hat += dx * self.dt
        return self.x_hat.copy()
```

### 5.4 电赛实例：无刷直流电机（BLDC）转子位置估计

```python
class BLDC_SMO:
    """无刷直流电机滑模观测器
    估计: α-β 轴电流, 反电势 → 推算转子位置
    """
    
    def __init__(self, Rs, Ls, dt):
        """
        Rs: 相电阻
        Ls: 相电感
        dt: 采样周期
        """
        self.Rs = Rs
        self.Ls = Ls
        self.dt = dt
        
        # 观测器增益
        self.L = np.array([[10, 0], [0, 10]])
        self.K = np.array([[2.0, 0], [0, 2.0]])
        
        # 状态: [i_alpha_hat, i_beta_hat]
        self.x_hat = np.zeros(2)
        
        # 估计的反电势
        self.e_est = np.zeros(2)
    
    def sign(self, x):
        return np.where(np.abs(x) < 0.01, x / 0.01, np.sign(x))
    
    def step(self, v_alpha, v_beta, i_alpha, i_beta):
        """
        v_alpha, v_beta: α-β 轴电压
        i_alpha, i_beta: α-β 轴电流 (测量值)
        返回: (theta_est, omega_est) 转子角度和角速度
        """
        i_meas = np.array([i_alpha, i_beta])
        v = np.array([v_alpha, v_beta])
        
        # 电流观测器
        # di_hat/dt = -(Rs/Ls)*i_hat + (1/Ls)*v - (1/Ls)*e_hat + L*(i - i_hat) + K*sign(i - i_hat)
        error = i_meas - self.x_hat
        di = (-(self.Rs/self.Ls) * self.x_hat + 
              (1/self.Ls) * v -
              (1/self.Ls) * self.e_est +
              self.L @ error + self.K @ self.sign(error))
        
        self.x_hat += di * self.dt
        
        # 反电势估计 (低通滤波)
        # 从滑模等效控制提取: e_est = K * sign(error) (滤波后)
        e_raw = self.K @ self.sign(error)
        alpha_filter = 0.1  # 低通滤波系数
        self.e_est = alpha_filter * e_raw + (1 - alpha_filter) * self.e_est
        
        # 从反电势计算角度
        theta = np.arctan2(-self.e_est[0], self.e_est[1])
        
        # 角速度 (从反电势幅值)
        e_mag = np.linalg.norm(self.e_est)
        Ke = 0.01  # 反电势常数
        omega = e_mag / Ke if Ke > 0 else 0
        
        return theta, omega
```

### 5.5 超螺旋滑模观测器 (Super-Twisting)

解决传统滑模的抖振问题：

```python
class SuperTwistingObserver:
    """超螺旋算法: 二阶滑模，减小抖振"""
    
    def __init__(self, L1, L2, dt=0.001):
        self.L1 = L1  # 切换增益
        self.L2 = L2  # 积分增益
        self.dt = dt
        self.z = 0     # 辅助状态
    
    def update(self, error):
        """
        超螺旋算法:
        u = -L1 * |e|^0.5 * sign(e) + z
        dz/dt = -L2 * sign(e)
        """
        u = -self.L1 * np.sqrt(np.abs(error)) * np.sign(error) + self.z
        self.z -= self.L2 * np.sign(error) * self.dt
        return u
```

## 6. 观测器选型指南

### 6.1 选型决策树

```
系统是否线性？
├── 是 → 卡尔曼滤波 (KF)
└── 否 → 非线性程度如何？
    ├── 轻度非线性 → EKF
    ├── 中度非线性 → UKF（推荐）
    ├── 强非线性 + 非高斯 → 粒子滤波
    └── 需要鲁棒性 → 滑模观测器
```

### 6.2 电赛常见场景推荐

| 场景 | 推荐观测器 | 理由 |
|------|-----------|------|
| 陀螺仪+加速度计融合 | EKF/互补滤波 | 简单有效 |
| 电机转速估计 | EKF/SMO | 成熟方案 |
| BLDC 无感控制 | SMO | 鲁棒性好 |
| 倒立摆角度估计 | UKF | 非线性较强 |
| 四旋翼姿态估计 | EKF/互补滤波 | 计算量适中 |
| 超声波/雷达测距融合 | UKF/EKF | 标准方案 |
| 移动机器人定位 | 粒子滤波/EKF | 取决于环境 |

### 6.3 互补滤波（最简单的"观测器"）

对于电赛初级选手，互补滤波是最实用的角度估计方法：

```c
// 互补滤波: 融合陀螺仪和加速度计
typedef struct {
    float angle;     // 估计角度
    float alpha;     // 滤波系数 (0.96~0.99)
    float dt;        // 采样周期
} complementary_filter_t;

float complementary_update(complementary_filter_t *f, 
                           float gyro_rate, float accel_angle) {
    // 陀螺仪积分 (高频可信) + 加速度计校正 (低频可信)
    f->angle = f->alpha * (f->angle + gyro_rate * f->dt) +
               (1 - f->alpha) * accel_angle;
    return f->angle;
}

// alpha = τ/(τ+dt), τ为时间常数
// τ=0.5s, dt=0.01s: α = 0.5/(0.5+0.01) ≈ 0.98
```

## 7. 调参技巧

### 7.1 EKF 调参

```
Q (过程噪声):
  - 增大 → 估计更信任测量值，跟踪快但噪声大
  - 减小 → 估计更信任模型，平滑但可能滞后
  
R (观测噪声):
  - 增大 → 估计更信任模型
  - 减小 → 估计更信任测量值

经验法则:
  Q/R 比值 ≈ 测量噪声² / 模型预测误差²
```

### 7.2 滑模观测器调参

```
K (切换增益):
  - K 太小: 无法克服不确定性，收敛慢
  - K 太大: 抖振严重
  - 经验: K > |d_max| + ε (d_max 为最大扰动)

L (线性增益):
  - 决定收敛速度
  - L 越大收敛越快，但对噪声敏感

抗抖振:
  1. 用 sat(e) 替代 sign(e)
  2. 边界层厚度 ε: sat(e) = e/ε 当 |e|<ε
```
