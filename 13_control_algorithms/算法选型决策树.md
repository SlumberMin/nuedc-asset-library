# 算法选型决策树

## 一、总体决策流程

```
你的系统是什么类型？
│
├─ 纯角度/位置跟踪 → 是否有精确模型？
│   ├─ 有 → LQR / 状态反馈
│   └─ 无 → PID（角度环）
│
├─ 平衡类（倒立摆/小车） → 速度要求？
│   ├─ 高响应 → 双环PID（角度环+角速度环）
│   └─ 稳态精度优先 → LQR + 前馈
│
├─ 电机调速 → 精度要求？
│   ├─ 一般 → 单环PID
│   ├─ 高精度 → 速度环+电流环 双环PID
│   └─ 高动态 → 增量式PID + 前馈
│
├─ 循迹/寻线 → 路径复杂度？
│   ├─ 直线简单 → PD控制
│   ├─ 弯道多 → PID + 差速
│   └─ 复杂赛道 → 模糊PID
│
├─ 电源/电压调节 →
│   ├─ DC-DC → 电压环PID + 限流保护
│   └─ 逆变器 → PR控制 / 重复控制
│
└─ 信号源/波形 →
    ├─ 正弦 → DDS + AGC
    └─ 任意波形 → DDS + DAC
```

## 三、具体选型决策

### 3.1 电机控制选型

```
电机类型？
├─ 直流有刷
│   ├─ 仅调速 → 增量式PID
│   ├─ 调速+定位 → 速度环PID + 位置环PID
│   └─ 大惯量 → PID + 前馈补偿
│
├─ 直流无刷(BLDC)
│   ├─ 有FOC库 → FOC + 速度环PID
│   └─ 简单驱动 → 六步换相 + PID调速
│
├─ 步进电机
│   ├─ 开环足够 → 脉冲+方向控制
│   └─ 需闭环 → 编码器反馈 + PID
│
└─ 舵机
    ├─ 角度控制 → PWM直接控制
    └─ 高精度 → 电位器反馈 + PID
```

### 3.2 平衡/姿态控制选型

```
平衡系统？
├─ 一维平衡（倒立摆）
│   ├─ 入门 → 单级PID（角度环）
│   └─ 进阶 → 角度环+角速度环 双环PID
│
├─ 二维平衡（平衡小车）
│   ├─ 标准方案 → 直立环PID + 速度环PID + 转向环PID
│   └─ 高性能 → LQR直立 + PID速度 + PD转向
│
└─ 三维姿态（四旋翼）
    ├─ 入门 → 角度环PID + 角速度环PID
    └─ 进阶 → 串级PID + 前馈
```

### 3.3 电源类选型

```
电源类型？
├─ DC-DC变换
│   ├─ Buck/Boost → 电压环PID
│   └─ 高精度 → 电压环+电流环 双环PID
│
├─ 线性稳压
│   └─ 运放反馈 + 功率管（无需数字控制）
│
├─ 逆变器
│   ├─ 工频 → 开环SPWM
│   └─ 高质量 → 电压外环PR + 电流内环PR
│
└─ 功率因数校正(PFC)
    └─ 电流环PID + 电压环PID
```

## 四、算法组合推荐

### 经典组合1：前馈+PID（推荐度★★★★★）
- **适用**：已知系统模型、有可预测扰动
- **原理**：前馈快速响应已知部分，PID消除残余误差
- **公式**：u = Kff·r + PID(e)

### 经典组合2：串级PID（推荐度★★★★★）
- **适用**：需要高动态响应的位置/角度控制
- **原理**：外环给内环设定值，内环快速响应
- **结构**：位置环→速度环→电流环

### 经典组合3：模糊+PID（推荐度★★★★☆）
- **适用**：参数时变、非线性明显的系统
- **原理**：模糊规则在线调整Kp/Ki/Kd
- **优势**：不依赖精确模型

### 经典组合4：LQR+观测器（推荐度★★★★☆）
- **适用**：有状态空间模型的最优控制
- **原理**：最小化二次型性能指标
- **前提**：需要准确的状态空间模型

## 五、速查口诀

> **"不确定用PID，有模型用LQR，要平衡用双环，变参数用模糊，已知扰用前馈"**
