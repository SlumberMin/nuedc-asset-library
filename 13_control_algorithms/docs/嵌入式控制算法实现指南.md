## 1. PID控制

### 1.1 基本原理

PID控制器输出：

```
u(t) = Kp·e(t) + Ki·∫e(τ)dτ + Kd·de(t)/dt
```

### 1.2 离散化实现

**位置式PID：**
```c
err = setpoint - measurement;
integral += err * dt;
derivative = (err - err_prev) / dt;
output = Kp * err + Ki * integral + Kd * derivative;
err_prev = err;
```

**增量式PID：**
```c
err = setpoint - measurement;
output += Kp * (err - err_prev) + Ki * err + Kd * (err - 2*err_prev + err_prev2);
err_prev2 = err_prev;
err_prev = err;
```

### 1.3 改进措施

| 改进项 | 方法 | 作用 |
|--------|------|------|
| 积分限幅 | `integral = CLAMP(integral, -I_MAX, I_MAX)` | 防止积分饱和 |
| 积分分离 | 误差大时关闭积分 | 减少超调 |
| 微分滤波 | 对微分项低通滤波 | 抑制噪声 |
| 死区 | 误差小时输出为0 | 避免抖动 |
| 变积分 | 误差大时减小Ki | 改善动态性能 |

## 2. 力控制算法

### 2.1 阻抗控制

**基本思想：** 将机器人与环境的交互建模为弹簧-阻尼-质量系统：

```
M·ẍ + B·ẋ + K·x = F
```

- **M (惯性):** 控制系统的"重量感"
- **B (阻尼):** 控制能量耗散，B越大越平稳
- **K (刚度):** 控制柔顺性，K越小越"软"

**参数调节经验：**
- 接触作业：K = 100~1000 N/m，B = 10~100 N·s/m
- 装配任务：先设大K，逐步减小至能完成任务

### 2.2 导纳控制

**基本思想：** 测量力，计算运动修正：

```
Md·ẍ_cmd + Bd·ẋ_cmd + Kd·x_cmd = F_des - F_meas
```

**适用场景：**
- 位置控制精度高的场合（如有精确编码器的电机）
- 需要力跟踪的场合（如打磨、抛光）

**阻抗 vs 导纳选择：**

| 特性 | 阻抗控制 | 导纳控制 |
|------|---------|---------|
| 输入 | 运动指令 | 力测量值 |
| 输出 | 力/力矩 | 位置修正 |
| 适用 | 力控精度好 | 位控精度好 |
| 稳定性 | 硬环境好 | 软环境好 |

### 2.3 力/位混合控制

**原理：** 用选择矩阵S将任务空间分为力控方向和位控方向：

```
u = (I - S)·u_pos + S·u_force
```

- S对角元素 = 1：该方向力控
- S对角元素 = 0：该方向位控

**典型应用：**
- 沿墙推磨：切向位控，法向力控
- 插孔装配：轴向力控，径向位控

## 3. 运动控制算法

### 3.1 梯形速度规划

**三段：** 加速 → 匀速 → 减速

```
加速段: v = a_max · t,  0 ≤ t < T_acc
匀速段: v = v_max,       T_acc ≤ t < T_acc + T_const
减速段: v = v_max - a_max·(t-T_acc-T_const)
```

**关键计算：**
```c
T_acc = v_max / a_max;
d_acc = 0.5 * a_max * T_acc²;
if (2*d_acc > distance) {
    // 三角形轮廓，达不到v_max
    T_acc = sqrt(distance / a_max);
    T_const = 0;
} else {
    T_const = (distance - 2*d_acc) / v_max;
}
```

### 3.2 S形速度规划

**七段：** 加加速 → 匀加速 → 减加速 → 匀速 → 加减速 → 匀减速 → 减减速

引入**加加速度 (jerk)** 限制，使加速度连续变化，运动更平滑。

```
j_max = a_max² / v_max    (典型关系)
```

### 3.3 三次多项式轨迹

满足边界条件：`q(0)=q₀, q'(0)=v₀, q(T)=qf, q'(T)=vf`

```
q(t) = a₀ + a₁t + a₂t² + a₃t³
```

4个方程解4个系数，确保位置和速度在起止点连续。

### 3.4 路径插补

**直线插补：**
```c
ratio = traveled / total_length;  // 0~1
x = x_start + ratio * (x_end - x_start);
y = y_start + ratio * (y_end - y_start);
```

**圆弧插补：**
```c
angle = start_angle + ratio * sweep_angle;
x = cx + r * cos(angle);
y = cy + r * sin(angle);
```

## 4. 嵌入式实现要点

### 4.1 定点化

浮点运算在MCU上较慢，可使用定点数：

```c
// Q16.16定点数
typedef int32_t fixed_t;
#define FIXED_SCALE 65536  // 2^16

fixed_t float_to_fixed(float x) { return (fixed_t)(x * FIXED_SCALE); }
float fixed_to_float(fixed_t x) { return (float)x / FIXED_SCALE; }
fixed_t fixed_mul(fixed_t a, fixed_t b) { return (fixed_t)(((int64_t)a * b) >> 16); }
```

### 4.2 定时器中断采样

```c
// STM32示例: TIM2中断, 1kHz
void TIM2_IRQHandler(void) {
    if (TIM2->SR & TIM_SR_UIF) {
        TIM2->SR &= ~TIM_SR_UIF;

        float pos = read_encoder();
        float output = PID_Update(&pid, setpoint, pos);
        set_motor_pwm(output);
    }
}
```

### 4.3 防止积分饱和

```c
integral += err * dt;
if (integral > INTEGRAL_MAX) integral = INTEGRAL_MAX;
if (integral < -INTEGRAL_MAX) integral = -INTEGRAL_MAX;
```

### 4.4 微分项滤波

```c
// 一阶低通滤波: y = α·x + (1-α)·y_prev
float alpha = 0.1;  // 滤波系数, 越小滤波越强
float d_err_raw = (err - err_prev) / dt;
d_err_filtered = alpha * d_err_raw + (1 - alpha) * d_err_filtered_prev;
d_err_filtered_prev = d_err_filtered;
```

### 4.5 查表法替代三角函数

```c
// 预计算sin表 (0~90°, 精度0.1°)
const float sin_table[901]; // 901个点

float fast_sin(float rad) {
    int deg = (int)(rad * 180.0f / M_PI * 10); // 0.1度精度
    if (deg < 0) deg = -deg;
    if (deg > 900) deg = 900;
    return sin_table[deg];
}
```

## 5. 调试技巧

### 5.1 PID调参步骤

1. **先P后I再D：**
   - 先设 Ki=0, Kd=0
   - 增大Kp直到系统开始振荡，取其60%
   - 逐步增大Ki消除稳态误差
   - 适当增大Kd减小超调

2. **Ziegler-Nichols法：**
   - 增大Kp直到等幅振荡，记录临界增益Ku和周期Tu
   - P:  Kp=0.5Ku
   - PI: Kp=0.45Ku, Ki=0.54Ku/Tu
   - PID:Kp=0.6Ku, Ki=1.2Ku/Tu, Kd=0.075Ku·Tu

### 5.2 力控制调试

1. 先在自由空间调试位置控制精度
2. 用低刚度环境（如海绵）测试接触
3. 逐步增大环境刚度
4. 观察力响应的超调和振荡

### 5.3 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 稳态误差 | Kp太小或有干扰 | 增大Kp或加入Ki |
| 振荡 | Kp或Kd太大 | 减小增益，加入滤波 |
| 响应慢 | 增益太小 | 适当增大Kp |
| 力控不稳定 | 环境刚度太大 | 降低阻抗K值 |
| 位置跟踪滞后 | 规划速度太慢 | 提高加速度限制 |

### 5.4 数据记录

建议记录以下数据用于分析：
- 参考值与实际值（位置、速度、力）
- 控制器输出
- 误差及误差积分
- 采样时间戳

```c
// 通过串口发送调试数据
typedef struct {
    uint32_t timestamp;
    float reference;
    float measurement;
    float output;
    float error;
} DebugData_t;
```
