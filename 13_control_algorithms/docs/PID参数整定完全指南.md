## 1. PID基础回顾

### 1.1 PID控制律

```
u(t) = Kp * e(t) + Ki * ∫e(t)dt + Kd * de(t)/dt
```

离散形式:
```
u(k) = Kp*e(k) + Ki*Σe(i)*Δt + Kd*(e(k)-e(k-1))/Δt
```

### 1.2 三个参数的物理含义

| 参数 | 作用 | 增大效果 |
|------|------|----------|
| Kp (比例) | 响应当前误差 | 加快响应, 可能振荡 |
| Ki (积分) | 消除稳态误差 | 消除偏差, 可能超调/振荡 |
| Kd (微分) | 预测误差趋势 | 抑制超调, 抗噪声差 |

### 1.3 性能指标

- **上升时间 (Rise Time)**: 10%→90%响应时间
- **超调量 (Overshoot)**: 最大超调百分比
- **调节时间 (Settling Time)**: 进入±2%或±5%带内的时间
- **稳态误差 (Steady-State Error)**: 最终偏差
- **ITAE / ISE / IAE**: 积分性能指标

## 3. Ziegler-Nichols法详解

### 3.1 闭环临界增益法 (Classic ZN)

**步骤:**
1. 将Ki=0, Kd=0, 从小到大增大Kp
2. 直到系统出现等幅持续振荡
3. 记录此时的**临界增益 Ku** 和**临界周期 Tu**
4. 按下表计算PID参数:

| 控制器 | Kp | Ti (=Kp/Ki) | Td (=Kd/Kp) |
|--------|-----|-------------|-------------|
| P | 0.5*Ku | ∞ | 0 |
| PI | 0.45*Ku | Tu/1.2 | 0 |
| PID (经典) | 0.6*Ku | 0.5*Tu | 0.125*Tu |

**代码实现:**
```c
// 经典ZN规则
kp = 0.6f * Ku;
ki = kp / (0.5f * Tu);   // ki = 1.2 * Ku / Tu
kd = kp * 0.125f * Tu;    // kd = 0.075 * Ku * Tu
```

### 3.2 改进ZN规则

| 规则 | Kp | Ti | Td | 特点 |
|------|-----|-----|-----|------|
| 经典 | 0.6*Ku | 0.5*Tu | 0.125*Tu | 标准, ~25%超调 |
| Pessen | 0.7*Ku | 0.4*Tu | 0.15*Tu | 更快, ~30%超调 |
| 适度超调 | 0.33*Ku | 0.5*Tu | 0.33*Tu | 平衡 |
| 无超调 | 0.2*Ku | 0.5*Tu | 0.33*Tu | 保守, 无超调 |

### 3.3 阶跃响应法 (ZN Open-Loop)

**步骤:**
1. 对被控对象施加阶跃输入
2. 绘制响应曲线
3. 画S形切线, 提取:
   - **L**: 纯滞后时间 (切线与时间轴交点)
   - **T**: 时间常数 (切线与稳态值交点减去L)
4. 计算: `Kp*T*L = K` (过程增益)

| 控制器 | Kp | Ti | Td |
|--------|-----|-----|-----|
| P | T/(K*L) | ∞ | 0 |
| PI | 0.9*T/(K*L) | 3.33*L | 0 |
| PID | 1.2*T/(K*L) | 2*L | 0.5*L |

## 4. 继电反馈法详解

### 4.1 原理

在控制器位置放置一个继电环节(带滞环), 系统会产生自持振荡。

```
         ┌──────────┐     ┌──────────┐
r ──×──►│  继电环节  │────►│  被控对象  │──► y
    │   │ (滞环)    │     │ G(s)     │    │
    │   └──────────┘     └──────────┘    │
    └─────────────────────────────────────┘
```

振荡的**幅值 a** 和**周期 Tu** 与系统临界特性相关:
```
Ku = 4*d / (π*a)
```
其中 `d` 是继电幅值。

### 4.2 实现代码

```c
// 继电滞环逻辑
if (measurement > hysteresis) {
    relay_output = -relay_amplitude;
} else if (measurement < -hysteresis) {
    relay_output = relay_amplitude;
}

// 检测过零点, 计算周期和振幅
// Ku = 4*d / (pi * a)
```

### 4.3 优点

- **闭环运行**: 不需要断开控制回路
- **自动找到振荡点**: 不需要手动调节
- **振幅可控**: 通过调节继电幅值控制振荡幅度
- **实现简单**: 代码量小

### 4.4 注意事项

1. **滞环宽度**: 太小→噪声导致误触发; 太大→振荡不明显
   - 建议: 信号量程的 0.5%~2%
2. **继电幅值**: 影响振荡幅度, 建议控制在执行器量程的 5%~20%
3. **需要足够振荡周期**: 至少3~5个完整周期才能准确估计
4. **超时设置**: 应设为预期周期的 10 倍以上

## 5. 阶跃响应法详解

### 5.1 两点法提取参数

使用响应曲线上的两个特殊点:
- **28.3%点**: y = baseline + 0.283 × (y_ss - baseline)
- **63.2%点**: y = baseline + 0.632 × (y_ss - baseline)

计算:
```
T = 1.5 × (t63.2 - t28.3)
L = t63.2 - T
```

### 5.2 过程增益

```
K = (y_steady_state - y_baseline) / Δu
```

## 6. Cohen-Coon法

适合**大滞后**系统 (L/T > 0.1), 比ZN法更适合一阶加纯滞后模型。

```
G(s) = K * e^(-Ls) / (τs + 1)
```

Cohen-Coon公式:
```
Kp = (1/K) × (1 + 1/(3×τ_l)) × (0.9316/τ_l)   简化形式
其中 τ_l = L/τ (归一化滞后)

Ti = L × (3.33×τ_l + 0.347) / (1 + 2.22×τ_l)
Td = L × 0.473×τ_l / (1 + 2.22×τ_l)
```

## 7. 在线自整定策略

### 7.1 继电反馈自动整定流程

```
1. 记录当前设定值和输出
2. 切换到继电模式
3. 运行足够周期 (4-6个)
4. 提取 Ku, Tu
5. 计算新PID参数
6. 切回PID模式
7. 监控过渡过程
```

### 7.2 条件触发自整定

- 设定值大幅变化时 (>30%)
- 负载突变检测到时
- 性能指标恶化时 (超调增大, 调节时间变长)

## 8. 现场快速整定经验

### 8.1 手动试凑法 (最常用)

**步骤:**
1. **纯P**: Ki=0, Kd=0, 从小Kp开始增大
2. 加入Kp直到系统振荡, 然后回退到 50-60%
3. **加入Ki**: 从小值开始, 直到稳态误差消除且超调可接受
4. **加入Kd**: 从0开始增大, 抑制超调和振荡

### 8.2 经验公式

| 应用场景 | Kp范围 | Ki/Kp | Kd/Kp | 说明 |
|----------|--------|-------|-------|------|
| 电机速度环 | 0.5-5 | 0.1-1.0 | 0.01-0.1 | 响应快 |
| 电机位置环 | 1-20 | 0.01-0.1 | 0.05-0.2 | 需要平滑 |
| 温度控制 | 2-10 | 0.01-0.1 | 0.5-2.0 | 大滞后 |
| 倾角平衡 | 5-50 | 0.5-5 | 0.1-1.0 | 快速响应 |
| 电压/电流 | 0.1-2 | 1-10 | 0-0.01 | 采样率高 |

### 8.3 电赛现场技巧

1. **先用继电法获取大致参数**, 再微调
2. **分环整定**: 先内环(快环), 再外环(慢环)
3. **Ziegler-Nichols参数通常偏激进**, 需要适当降低Kp或增大Ti
4. **加前馈**: 对已知的扰动加前馈补偿, 减轻PID负担
5. **抗饱和必须有**: 实际系统都有执行器饱和

## 9. 常见问题与解决方案

### 9.1 振荡/不稳定

| 原因 | 解决方案 |
|------|----------|
| Kp过大 | 减小Kp到60-70% |
| Ki过大 | 减小Ki, 增大积分时间 |
| 采样率不足 | 提高采样频率 (>10×带宽) |
| 延迟太大 | 使用Smith预估器 |
| 滞后非线性 | 加死区补偿 |

### 9.2 超调过大

- 减小Kp
- 减小Ki (增大Ti)
- 增大Kd (微分抑制)
- 加设定值滤波 (一阶低通)
- 使用输入整形器

### 9.3 响应太慢

- 增大Kp
- 增大Ki
- 加前馈补偿
- 使用级联控制

### 9.4 稳态误差

- 增大Ki (或检查是否积分饱和)
- 加前馈
- 检查传感器标定

## 10. 电赛实战案例

### 10.1 2023年风力摆

```
系统: 电机→摆杆→摆锤位置
整定过程:
1. 继电法: Ku=12.5, Tu=0.8s
2. 经典ZN: Kp=7.5, Ki=18.75, Kd=0.9375
3. 现场微调: Kp=5.0, Ki=8.0, Kd=1.2 (减小超调)
4. 加前馈: 基于cos(θ)的重力补偿
```

### 10.2 2021年智能送药小车

```
系统: 电机→小车→速度/位置
内环(速度): P=3.0, I=5.0, D=0
外环(位置): P=8.0, I=0.5, D=2.0
技巧: 位置环用PD, 到达后切换PI保持
```

### 10.3 2019年模拟电磁曲射炮

```
系统: 电磁线圈→弹丸初速度
整定: 纯P控制 + 查表法
原因: 非线性系统, 传统PID效果差
改进: 分段PID + 前馈查表
```
