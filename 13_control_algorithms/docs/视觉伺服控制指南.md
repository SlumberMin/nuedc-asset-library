# 视觉伺服控制详解

## 1. 概述

视觉伺服(Visual Servoing)是利用**相机反馈实时控制机器人运动**的闭环方法。核心思想是：通过图像信息计算误差，生成速度指令驱动机器人末端或本体运动，直到视觉误差收敛为零。

## 2. 两大范式

### 2.1 IBVS — 基于图像的视觉伺服

**原理**：在**图像空间**中定义误差，不需要三维重建。

```
误差:  e = s - s*    (当前特征 - 期望特征，像素坐标)
控制:  v = -λ · L⁺ · e
```

其中：
- `s` = 当前特征点在图像中的像素坐标 (u, v)
- `s*` = 期望特征点坐标
- `L` = 交互矩阵 (Interaction Matrix / Image Jacobian)
- `L⁺` = L的伪逆 (Moore-Penrose)
- `λ` = 控制增益

**交互矩阵**（对于一个点特征，在深度Z处）：

```
        ⎡ -fx/Z    0     u/Z    uv/fx    -(fx+u²/fx)   v  ⎤
  L  =  ⎢                                                    ⎥
        ⎣   0    -fy/Z   v/Z   (fy+v²/fy)  -uv/fy      -u  ⎦
```

其中 (fx, fy) 是焦距，(u, v) 是相对于主点的坐标。

**优点**：
- 对相机标定误差鲁棒
- 特征直接在图像中跟踪，无需3D重建

**缺点**：
- 可能出现非直线笛卡尔轨迹（相机"绕路"）
- 交互矩阵依赖深度估计Z

### 2.2 PBVS — 基于位置的视觉伺服

**原理**：先从图像中**估计目标的3D位姿**，在任务空间定义误差。

```
误差:  e = [t - t*,  θ·r]    (位移误差 + 旋转误差)
控制:  v = -λ · e
```

其中：
- `t, t*` = 当前/期望平移向量
- `θ·r` = 旋转误差的角轴表示

**优点**：
- 运动轨迹在笛卡尔空间是直线
- 可直接利用先验CAD模型

**缺点**：
- 依赖相机标定和目标模型精度
- 需要3D位姿估计（计算量大）

## 3. IBVS 实现要点

### 3.1 特征点选择
| 特征类型 | 描述 | 维度 |
|---------|------|------|
| 点特征 | SIFT/SURF/ORB检测角点 | 每点2维 |
| 线段特征 | 直线的端点或Plücker坐标 | 每线4维 |
| 面积特征 | 图像矩、椭圆拟合 | 每区域5-6维 |
| 混合特征 | 点+线+面积组合 | 视情况 |

### 3.2 交互矩阵组装

对于 N 个点特征：

```
  ⎡ L₁ ⎤         ⎡ e₁ ⎤
  ⎢ L₂ ⎥   L  =  ⎢ e₂ ⎥    e
  ⎢ ⋮  ⎥         ⎢ ⋮  ⎥
  ⎣ Lₙ ⎦         ⎣ eₙ ⎦

  L ∈ R^{2N × 6},  e ∈ R^{2N}

  v = -λ · L⁺ · e
```

### 3.3 深度估计

交互矩阵依赖每个特征点的深度Z：
- **固定深度近似**：所有特征点使用相同估计深度Z₀
- **自适应深度**：结合IMU/里程计在线估计
- **立体视觉**：双目直接测量
- **深度相机**：RGBD直接获取

### 3.4 增益调节

| λ 值 | 效果 |
|------|------|
| 太小 | 收敛慢，需要更多迭代 |
| 太大 | 震荡、发散 |
| 推荐 | 0.3~1.0（配合dt归一化） |

## 4. PBVS 实现要点

### 4.1 位姿估计方法

1. **PnP求解**：已知3D-2D对应点，求解相机位姿
   - `cv::solvePnP()` / `cv::solvePnPRansac()`
2. **ArUco/AprilTag标记**：直接提供6DoF位姿
3. **模型匹配**：已知CAD模型，ICP/PnP配准

### 4.2 误差设计

旋转误差的常见表达：
- **欧拉角差**：简单但有万向节锁问题
- **角轴表示**：`θ·r = log(R · Rd^T)`，推荐
- **四元数误差**：`e_q = q - q*` 或 `e_q = q*⁻¹ ⊗ q`

### 4.3 任务空间控制律

```c
// 位置增益 + 旋转增益分离调节
vx = -λ_pos * (x - x_des);
vy = -λ_pos * (y - y_des);
vz = -λ_pos * (z - z_des);
wx = -λ_rot * e_roll;
wy = -λ_rot * e_pitch;
wz = -λ_rot * e_yaw;
```

## 5. IBVS vs PBVS 对比

| 特性 | IBVS | PBVS |
|------|------|------|
| 误差空间 | 图像像素 | 笛卡尔3D |
| 3D重建 | 不需要 | 需要 |
| 标定敏感度 | 低 | 高 |
| 轨迹特性 | 可能非直 | 笛卡尔直线 |
| 稳定性 | 局部稳定 | 全局稳定 |
| 计算量 | 较小 | 较大 |
| 适用场景 | 未知目标 | 已知模型 |

## 6. 电赛实战建议

### 6.1 典型应用
- **机械臂抓取**：IBVS跟踪目标色块/标记
- **无人机降落**：IBVS对准地面标记
- **自平衡车跟踪**：PBVS跟踪前方目标

### 6.2 与PID结合
```
视觉伺服输出速度指令 → PID控制器跟踪速度 → 电机PWM
```
视觉伺服作为**外环**产生期望速度，PID作为**内环**控制电机。

### 6.3 实际注意事项
1. **帧率**：相机帧率 ≥ 30fps，延迟 < 50ms
2. **特征稳定性**：使用标记(ArUco)比自然特征更可靠
3. **遮挡处理**：特征丢失时保持最后有效指令或切换到开环
4. **噪声滤波**：对图像特征做低通滤波减小抖动

## 7. 调参流程

```
1. 固定相机，放置标记，获取期望图像
2. 初始化IBVS，λ设为0.3
3. 逐步增大λ直到收敛速度满意（不超过振荡边界）
4. 若使用PBVS，先验证位姿估计精度
5. 加入速度限幅（±2 m/s, ±2 rad/s）
6. 在真实运动平台上测试，调整深度估计Z
```
