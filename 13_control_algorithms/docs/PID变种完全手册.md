## 1. 经典PID回顾

### 传递函数
```
G(s) = Kp + Ki/s + Kd·s
```

### 位置式
```
u(k) = Kp·e(k) + Ki·Σe(j)·Δt + Kd·[e(k)-e(k-1)]/Δt
```

### 增量式
```
Δu(k) = Kp·[e(k)-e(k-1)] + Ki·e(k)·Δt + Kd·[e(k)-2e(k-1)+e(k-2)]/Δt
```

### 参数物理意义
| 参数 | 作用 | 增大效果 | 典型副作用 |
|------|------|---------|-----------|
| Kp   | 快速响应 | 加快响应、减小稳态误差 | 超调增大、振荡 |
| Ki   | 消除稳态误差 | 消除静差 | 积分饱和、超调 |
| Kd   | 抑制超调 | 减小超调、提高稳定性 | 放大噪声、对突变敏感 |

## 2. 增量式PID

### 原理
只计算控制量的增量，输出为 `u(k) = u(k-1) + Δu(k)`

### 优点
- 无积分累积问题
- 手动/自动切换无扰动
- 算法简单，适合嵌入式

### 实现
```c
float delta_u = Kp*(e-e_prev) + Ki*e*dt + Kd*(e-2*e_prev+e_prev2)/dt;
u += delta_u;
```

### 适用场景
- 电机控制
- 简单温度控制
- 资源受限MCU

---

## 3. 位置式PID

### 原理
直接计算绝对控制量，包含完整积分项。

### 积分抗饱和策略
1. **限幅法**: 输出超限时冻结积分
2. **Back-calculation**: `u_sat - u` 反馈修正积分
3. **条件积分**: 仅在输出未饱和时积分
4. **积分限幅**: 直接限制积分项范围

---

## 4. 抗积分饱和PID

### Back-Calculation法
```
积分项 += Ki·e·dt + Kb·(u_sat - u)·dt
```
其中 `Kb` 为回算增益。

### 条件积分法(推荐)
```c
if (u > u_min && u < u_max) {
    integral += e * dt;
}
```

### 积分限幅法
```c
integral += e * dt;
integral = clamp(integral, -I_max, I_max);
```

### 适用场景
- 阀门控制
- 任何存在执行器饱和的场合

---

## 5. 微分先行PID

### 原理
微分项只作用于测量值而非误差，避免设定值突变引起的微分冲击。

### 公式
```
u = Kp·e + Ki·∫e·dt - Kd·(dy/dt)
```

### 优点
- 设定值阶跃变化时无微分尖峰
- 减小超调

### 适用场景
- 设定值频繁跳变的系统
- 伺服控制

---

## 6. 不完全微分PID

### 原理
在微分项上串联一阶低通滤波器，抑制高频噪声。

### 公式
```
D(s) = Kd·s / (Tf·s + 1)
```

### 离散实现
```c
alpha = Tf / (Tf + dt);
D = alpha * D_prev + Kd*(1-alpha)*(e - e_prev)/dt;
```

### 适用场景
- 传感器噪声大的场合
- 所有实际工程应用(推荐默认开启)

---

## 7. 带滤波器PID

### 滤波器类型
1. **一阶低通**: 最常用，抑制高频噪声
2. **二阶低通**: 更强滤波，但有相位滞后
3. **中值滤波**: 去除脉冲干扰
4. **卡尔曼滤波**: 最优估计，适合复杂噪声

### 推荐配置
```c
// 微分滤波系数 0.1~0.5
derivative_filter_coeff = 0.3;
filtered_d = alpha * prev_d + (1-alpha) * raw_d;
```

## 8. 串级PID

### 结构
```
设定值 → [外环PID] → 内环设定值 → [内环PID] → 执行器 → 被控对象
                ←──────── 测量反馈 ←───────────
```

### 参数整定
1. 先调内环(快环): 纯P或PI
2. 再调外环(慢环): PI或PID
3. 内环带宽 > 外环带宽(5~10倍)

### 适用场景
- 电机速度-电流双环
- 温度控制(炉温-加热功率)
- 飞行器姿态控制

## 9. 前馈+PID

### 原理
```
u = u_ff + u_pid
u_ff = Kff·r + Kff_d·(dr/dt)
```

### 设计方法
1. **静态前馈**: `Kff = 1/G(0)` (稳态增益倒数)
2. **动态前馈**: `Kff(s) = 1/G(s)` (需要模型)
3. **实验法**: 测量设定值到输出的传递函数

### 优点
- 提高响应速度
- 减小PID负担
- 不影响稳定性

---

## 10. 增益调度PID

### 原理
根据工作点自动切换PID参数。

### 调度变量
- 输出幅值
- 运行速度
- 温度
- 设定值

### 实现方式
1. **查表插值**: 离散工作点线性插值
2. **函数映射**: `Kp = f(调度变量)`
3. **平滑切换**: 加权平均过渡

### 适用场景
- 工况变化大的系统
- 非线性严重的过程控制
- 化工过程

---

## 11. 模糊PID

### 原理
用模糊规则根据误差(e)和误差变化率(ec)在线调整ΔKp, ΔKi, ΔKd。

### 模糊化
```
语言变量: NB, NM, NS, ZO, PS, PM, PB (7级)
隶属函数: 三角形 / 梯形 / 高斯
```

### 推理规则(7×7规则表)
```
ΔKp: e大 → Kp大; e小 → Kp小
ΔKi: e大且持续 → Ki大; e小 → Ki小
ΔKd: ec大 → Kd大; ec小 → Kd小
```

### 解模糊
- 重心法(最常用)
- 加权平均法(计算量小)
- 最大隶属度法(简单)

### 优点
- 不需要精确数学模型
- 鲁棒性好
- 适合非线性系统

## 12. 模糊PID V2

### V2新增特性
1. **多种隶属函数**: 三角形、梯形、高斯可切换
2. **在线规则自适应**: 根据性能自动微调规则
3. **规则权重系统**: 不同规则可设置不同权重
4. **完整7×7规则表**: NB~PB 七级语言变量
5. **振荡检测**: 自动检测并抑制振荡

### 库中实现
```c
PIDFuzzyV2_t ctrl;
PIDFuzzyV2_Init(&ctrl, Kp, Ki, Kd, dt);
PIDFuzzyV2_SetMFType(&ctrl, MF_GAUSSIAN);
PIDFuzzyV2_SetRanges(&ctrl, 100.0, 100.0, 1.0);
PIDFuzzyV2_EnableOnlineAdapt(&ctrl, 0.01);
output = PIDFuzzyV2_Calculate(&ctrl, setpoint, measurement);
```

## 13. 自适应PID

### 自适应律类型
1. **MIT规则**: `dKp/dt = -γ·e·(∂e/∂Kp)`
2. **梯度下降**: 沿性能指标梯度方向调整参数
3. **Lyapunov**: 保证稳定性条件下的参数调整

### 常见自适应策略
- **误差驱动**: 误差大→增大Kp; 误差持续→增大Ki
- **性能指标驱动**: ITAE, ISE, IAE等
- **模型辨识**: 在线辨识模型参数→重新整定PID

---

## 14. 自适应PID V2

### V2支持的6种自适应律

| 自适应律 | 原理 | 特点 | 适用场景 |
|---------|------|------|---------|
| 梯度下降 | `Δθ = -γ·∂J/∂θ` | 简单、收敛慢 | 一般自适应 |
| Lyapunov | `V=e²/2>0, dV/dt<0` | 稳定性保证 | 安全关键系统 |
| MRAC | 参考模型跟踪 | 动态性能好 | 需要指定响应 |
| σ修正 | `Δθ = γeφ - σγ(θ-θ₀)` | 防参数漂移 | 长时间运行 |
| e修正 | `Δθ = γeφ - σγ|e|θ` | 误差消失时归零 | 短暂扰动 |
| 切换律 | 大误差激进/小误差保守 | 快速收敛+精确 | 宽工况范围 |

### 振荡检测与抑制
V2内置32点误差滑动窗口，计算符号变化频率作为振荡指标：
```
oscillation_metric = sign_changes / window_size
```
当振荡指标高时，自动衰减自适应增益(0.2~1.0倍)。

### 库中实现
```c
PIDAdaptiveV2_t ctrl;
PIDAdaptiveV2_Init(&ctrl, Kp, Ki, Kd, dt);
PIDAdaptiveV2_SetAdaptLaw(&ctrl, ADAPT_LAW_SIGMA_MOD);
PIDAdaptiveV2_SetAdaptGain(&ctrl, 0.05, 0.01, 0.01);
PIDAdaptiveV2_SetSigma(&ctrl, 0.02);
PIDAdaptiveV2_SetParamRange(&ctrl, 0.1, 50, 0, 20, 0, 10);
output = PIDAdaptiveV2_Calculate(&ctrl, setpoint, measurement);
```

## 15. 鲁棒PID

### 设计方法
1. **H∞优化**: 最小化最坏情况下的性能指标
2. **μ综合**: 处理结构化不确定性
3. **定量反馈理论(QFT)**: 基于频域约束

### 简化鲁棒PID
```
Kp = 0.6·Ku
Ki = 2·Kp / (Tu·π)
Kd = Kp·Tu / (8·π)
```
其中 Ku 为临界增益，Tu 为临界振荡周期。

## 16. 非线性PID

### 变增益PID
```
Kp = Kp0 + α·|e|
Ki = Ki0 / (1 + β·|e|)
Kd = Kd0·(1 + γ·|e|)
```

### 死区PID
```c
if (fabs(e) < dead_band) {
    u = 0;
} else {
    u = pid_calc(e);
}
```

### 死区+限幅
适用于阀门等有死区的执行器。

## 17. 分数阶PID

### 传递函数
```
G(s) = Kp + Ki/s^λ + Kd·s^μ
```
其中 0 < λ, μ < 2 (通常 λ=0.5~1.5, μ=0.5~1.5)

### 近似方法
- Oustaloup滤波器(5阶近似)
- Charef近似
- 连分数展开

### 优点
- 更灵活的调节自由度
- 对模型不确定性更鲁棒

### 适用场景
- 电化学系统
- 热传导控制
- 分数阶特性明显的物理系统

## 18. Smith预估PID

### 原理
针对纯滞后系统 `G(s)·e^(-Ls)`，用模型补偿滞后。

### 结构
```
设定值 → [PID] → 执行器 → [G(s)·e^(-Ls)] → 输出
              ↑                |
              └── [G(s)] ──┬───┘
                           └── [e^(-Ls)] ──→ 反馈
```

### 设计
```
控制器 = Gc(s) = 1/G(s)  (模型精确时)
实际: 调节PID使 Gc·G/(1+Gc·G) 稳定
```

### 适用场景
- 化工过程(大滞后)
- 网络控制(通信延迟)
- 滞后时间 L > 2T 的系统

---

## 19. 内模控制PID

### 原理
基于被控对象模型设计控制器，使闭环传递函数等于期望模型。

### 设计步骤
1. 模型: `Gm(s) = Km / (Tm·s+1)^n`
2. 期望响应: `Gd(s) = 1 / (Tc·s+1)^n`
3. 控制器: `C(s) = Gd(s) / [Gm(s)·(1-Gd(s))]`

### 一阶IMC-PID
```
Kp = Tm / (Km·Tc)
Ki = Kp / Tm
Kd = 0
```

### 优点
- 只有一个调参参数(Tc)
- 系统性设计方法

## 20. 基于扰动观测器的PID

### DOB原理
```
扰动估计 d̂ = (u - ŷ_model)  经低通滤波
补偿: u_total = u_pid - d̂
```

### 结构
```
设定值 → [PID] → Σ → 执行器 → 被控对象 → 输出
               -d̂ ↑
           [DOB估计] ←──────┘
```

### 适用场景
- 扰动频繁的系统
- 需要快速抗扰的场合
- 机械臂、无人机

## 21. 神经网络PID

### 结构
```
输入层: e(k), e(k-1), e(k-2), y(k), y(k-1)
隐藏层: 10~20个神经元
输出层: Kp, Ki, Kd
```

### 训练方法
- BP神经网络在线调整
- 强化学习优化
- 深度学习离线训练

### 优点
- 自动学习最优参数
- 适应复杂非线性

### 缺点
- 计算量大
- 训练不稳定
- 实时性差(嵌入式)

## 22. 遗传算法优化PID

### 流程
```
1. 初始化种群(随机PID参数)
2. 适应度评估(IAE/ITAE/ISE)
3. 选择、交叉、变异
4. 重复2-3直到收敛
```

### 适应度函数
```
J = w1·IAE + w2·Mp + w3·ts + w4·|u_max|
IAE = Σ|e|
Mp = 超调量
ts = 调节时间
```

### 适用场景
- 离线整定
- 多目标优化
- 复杂系统建模

## 23. 选型指南

### 按场景选型

| 场景 | 推荐PID变种 | 理由 |
|------|-----------|------|
| 简单电机控制 | 增量式PID | 简单、无积分问题 |
| 温度控制 | 带滤波器PID + 抗饱和 | 慢过程、有饱和 |
| 位置伺服 | 微分先行PID + 前馈 | 设定值跳变频繁 |
| 化工过程 | Smith预估PID | 大滞后 |
| 变工况系统 | 增益调度PID | 工况变化大 |
| 不确定系统 | 自适应PID V2 | 参数未知或时变 |
| 非线性系统 | 模糊PID V2 | 不需要精确模型 |
| 扰动频繁 | DOB + PID | 快速抗扰 |
| 高精度要求 | 分数阶PID | 更多调节自由度 |
| 多目标优化 | 遗传算法优化PID | 离线全局优化 |

### 整定流程
```
1. 纯P调节 → 找临界振荡点(Ku, Tu)
2. Z-N公式计算初始参数
3. 逐步微调: 先P → 再I → 最后D
4. 如有振荡: 减小Kp或增大Kd
5. 如有稳态误差: 增大Ki
6. 如有噪声: 加滤波、减小Kd
```

### 嵌入式实现注意事项
1. **定点化**: 使用Q16.16格式避免浮点开销
2. **积分限幅**: 防止溢出
3. **微分滤波**: 必须加，否则噪声放大
4. **采样时间**: 确保定时器精度
5. **保护逻辑**: 加入输出限幅和故障检测
