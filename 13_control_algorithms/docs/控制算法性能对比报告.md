# 控制算法性能对比报告

## 1. 概述

本报告对比分析电赛常用控制算法的性能特征，为不同应用场景选型提供参考依据。

| **前馈控制** | ★★★★★ | ★★★ | ★★ | ★★★ | ★★ | 已知模型的轨迹跟踪 |

## 3. 性能指标对比（典型阶跃响应测试）

### 3.1 测试条件
- 被控对象: 直流电机 (J=0.01 kg·m², b=0.1, Kt=0.1)
- 输入: 阶跃信号 (0→1 rad)
- 采样周期: 1ms

## 4. 各算法详细分析

### 4.1 位置式PID
- **优势**: 实现简单，调参直观，适用面广
- **劣势**: 积分饱和、响应速度受限
- **电赛应用**: 大多数基础控制任务

### 4.2 增量式PID
- **优势**: 无冲击切换、输出为增量不易饱和、适合PWM控制
- **劣势**: 响应略慢于位置式
- **电赛应用**: 电机调速、舵机控制
- **优化**: 加入微分滤波、增量限幅后性能显著提升

### 4.3 串级PID
- **优势**: 内环快速抑制扰动、外环精确跟踪、带宽分离设计
- **劣势**: 参数整定更复杂(需先整定内环再整定外环)
- **电赛应用**: 电机位置控制(位置环+速度环)、云台控制(角度环+角速度环)
- **推荐**: 电机类题目首选方案

### 4.4 前馈控制
- **优势**: 基于模型补偿，理论上可实现零延迟跟踪
- **劣势**: 依赖准确模型，单独使用无纠偏能力
- **电赛应用**: 机械臂轨迹跟踪、已知运动规律的跟随控制
- **最佳实践**: 前馈+PID组合，前馈提速、PID纠偏

### 4.5 Bang-Bang控制
- **优势**: 时间最优，大误差时响应最快
- **劣势**: 精度差，在目标附近振荡
- **电赛应用**: 快速粗定位 + PID精定位的两阶段策略
- **改进**: 带滞回或PD切换的混合模式

### 4.6 视觉伺服(IBVS)
- **优势**: 直接在图像空间闭环，无需精确标定
- **劣势**: 计算量大、延迟高、需相机标定
- **电赛应用**: 视觉追踪、二维码定位、色块跟随

## 5. 选型建议

### 按应用场景

| 场景 | 推荐方案 | 备选方案 |
|------|---------|---------|
| 电机调速 | 增量式PID | 位置式PID |
| 电机位置控制 | 串级PID(位置+速度) | 前馈+PID |
| 角度控制(陀螺仪) | 串级PID(角度+角速度) | 模糊PID |
| 快速到位 | Bang-Bang + PID | 前馈+PID |
| 视觉追踪 | IBVS + PID | PID + 坐标变换 |
| 循迹小车 | 增量式PID | 差速模糊控制 |
| 悬挂/平衡 | 串级PID | 状态空间(LQR) |

### 按调参难度

1. **最简单**: Bang-Bang (2个参数)
2. **简单**: 增量式PID (3个参数)
3. **中等**: 位置式PID (3+参数)
4. **较难**: 串级PID (6+参数，需分步整定)
5. **困难**: 视觉伺服 (需标定+多参数)

## 6. 调参建议

### 串级PID整定步骤
1. 先断开外环，整定内环(速度环): 先P后I，使速度响应快且无超调
2. 接入外环，整定外环(位置环): 先P后D，I通常很小或为0
3. 内环带宽应为外环的5~10倍

### 前馈增益标定
1. 速度前馈Kv: 让系统匀速运动，调节Kv使PID输出为零
2. 加速度前馈Ka: 让系统做加减速，调节Ka减小跟踪误差

## 7. 总结

- **通用首选**: 位置式/增量式PID
- **电机控制首选**: 串级PID
- **高精度跟踪**: 前馈 + PID
- **时间最优**: Bang-Bang + PID混合
- **视觉控制**: IBVS视觉伺服

在电赛中，建议根据题目要求灵活组合以上算法，以达到最优控制效果。
