## 1. 概述

### 1.1 为什么需要非线性控制?

PID等线性控制器在工作点附近效果良好,但当系统存在以下情况时需要非线性控制:

| 场景 | 示例 | 推荐方法 |
|------|------|----------|
| 大范围运动 | 机械臂全关节运动 | 反馈线性化、反步法 |
| 模型不确定 | 负载变化、摩擦未知 | 自适应控制、滑模控制 |
| 混沌/不稳定 | 倒立摆、混沌振子 | 反馈线性化、Lyapunov法 |
| 状态约束 | 角度/速度限制 | 最优控制、反步法 |
| 快速响应 | 电机伺服、平衡车 | 滑模控制 |

### 1.2 非线性系统基本形式

**仿射非线性系统** (最常见):
```
ẋ = f(x) + g(x)u
y = h(x)
```

**严格反馈形式** (适合反步法):
```
ẋ₁ = f₁(x₁) + g₁(x₁)x₂
ẋ₂ = f₂(x₁,x₂) + g₂(x₁,x₂)x₃
...
ẋₙ = fₙ(x₁,...,xₙ) + gₙ(x₁,...,xₙ)u
```

## 2. 反馈线性化

### 2.1 核心思想

通过非线性状态反馈将非线性系统变换为线性系统,再用成熟的线性控制方法设计。

### 2.2 数学原理

对于 SISO 仿射非线性系统:
```
ẋ = f(x) + g(x)u
y = h(x)
```

**相对阶**: 使 L_g L_f^{r-1} h(x) ≠ 0 的最小整数 r。

当 r = n (系统维数) 时,可通过坐标变换:
```
z = [h(x), L_f h(x), L_f² h(x), ..., L_f^{n-1} h(x)]
```

和反馈:
```
u = (v - L_f^n h(x)) / (L_g L_f^{n-1} h(x))
```

将系统化为 n 阶积分器链:
```
z₁' = z₂
z₂' = z₃
...
zₙ' = v
```

### 2.3 倒立摆示例

```
系统: ml²θ'' = mgl·sin(θ) - b·θ' + u
输出: y = θ
相对阶: r = 2 = n ✓

变换后: y'' = (g/l)·sin(θ) - b/(ml²)·θ' + u/(ml²)
令 v = -k₁(θ-θ_ref) - k₂θ'  (极点配置)
u = ml²·(v - (g/l)·sin(θ) + b/(ml²)·θ')
```

### 2.4 优点与局限

| 优点 | 局限 |
|------|------|
| 精确线性化,无近似误差 | 需要精确的系统模型 |
| 可直接使用线性控制理论 | 要求相对阶 = 系统维数 |
| 理论保证性强 | 可能产生奇异点(分母为零) |
| 实现相对简单 | 不直接处理不确定性 |

### 2.5 代码使用

```c
#include "feedback_linearization.h"

// 定义倒立摆参数
fbl_pendulum_params_t pend_params = {
    .m = 1.0f, .l = 1.0f, .g = 9.81f, .b = 0.1f
};

// 配置控制器
fbl_config_t config = {
    .n = 2,
    .f_func = fbl_pendulum_f,
    .g_func = fbl_pendulum_g,
    .h_func = fbl_pendulum_h,
    .user_data = &pend_params
};

fbl_controller_t ctrl;
fbl_init(&ctrl, &config, 0.001f);

// 设置闭环极点 (期望: -5, -5)
float poles[] = {-5.0f, -5.0f};
fbl_set_poles(&ctrl, poles);

// 控制循环
while (running) {
    float x[2] = {theta, theta_dot};
    float u = fbl_update(&ctrl, x, theta_ref, current_time);
    apply_control(u);
}
```

## 3. 反步法

### 3.1 核心思想

对严格反馈系统,从最内层状态开始,逐层设计虚拟控制律,
每步都构造 Lyapunov 函数保证该步的稳定性。

### 3.2 设计步骤 (二阶系统)

```
系统:
ẋ₁ = f₁(x₁) + g₁(x₁)x₂
ẋ₂ = f₂(x₁,x₂) + g₂(x₁,x₂)u
```

**Step 1**: 定义 z₁ = x₁ - x₁ᵈ (跟踪误差)
```
选择虚拟控制 α₁(x₁) 使 z₁ → 0
α₁ = (-c₁·z₁ - f₁(x₁)) / g₁(x₁)

Lyapunov函数: V₁ = 0.5·z₁²
```

**Step 2**: 定义 z₂ = x₂ - α₁
```
V₂ = V₁ + 0.5·z₂² = 0.5·z₁² + 0.5·z₂²

dV₂/dt = z₁·z₁' + z₂·z₂'
       = z₁·(g₁·z₂ + g₁·α₁ + f₁ - x₁ᵈ')
         + z₂·(f₂ + g₂·u - ∂α₁/∂x₁·ẋ₁ - ∂α₁/∂x₁ᵈ·ẋ₁ᵈ)

选择 u 使 dV₂/dt < 0:
u = (-c₂·z₂ - z₁·g₁ - f₂ + ∂α₁/∂x₁·ẋ₁) / g₂
```

### 3.3 优点与局限

| 优点 | 局限 |
|------|------|
| 系统化设计,逐层保证稳定 | 仅适用于严格反馈形式 |
| 不需要精确线性化 | "计算膨胀"——高阶系统计算量大 |
| 可自然处理参数不确定 | 需要知道系统的下三角结构 |
| 有严格的Lyapunov稳定性证明 | 虚拟控制可能过于保守 |

## 4. 滑模控制

### 4.1 核心思想

设计一个滑模面 s(x) = 0,使系统状态一旦到达滑模面就沿其滑动到目标。
控制律分两部分: 等效控制(保持在滑模面) + 切换控制(到达滑模面)。

### 4.2 设计方法

**滑模面设计**:
```
s = c·e + ė        (一阶滑模面, e = x - x_ref)
s = (d/dt + c)^n · ∫e dt   (高阶滑模面)
```

**控制律**:
```
u = u_eq + u_sw

等效控制 (令 ṡ = 0):
u_eq = (-f(x) - c·ė + ẍ_ref) / g(x)

切换控制 (趋近律):
u_sw = -η·sign(s) / g(x)    (符号函数,有抖振)
u_sw = -η·sat(s/φ) / g(x)  (饱和函数,减抖振)
u_sw = (-η·s) / (|s| + δ) / g(x)  (连续近似)
```

### 4.3 抖振抑制策略

| 方法 | 公式 | 特点 |
|------|------|------|
| 边界层法 | sat(s/φ) | 简单有效,牺牲有限精度 |
| 连续近似 | s/(|s|+δ) | 更平滑 |
| 高阶滑模 | Super-Twisting | 本质消除抖振 |
| 模糊滑模 | 模糊规则切换 | 自适应边界层 |

### 4.4 Super-Twisting算法 (二阶滑模)

```
u = -λ|s|^0.5 · sign(s) + v
v̇ = -W · sign(s)

优点: 连续控制信号,无抖振
```

### 4.5 优点与局限

| 优点 | 局限 |
|------|------|
| 对匹配不确定性完全鲁棒 | 符号函数导致抖振 |
| 设计简单,易于实现 | 不确定性上界需已知 |
| 响应快速 | 滑模面设计需要经验 |
| 不依赖精确模型 | 仅处理匹配不确定性 |

## 5. Lyapunov直接法

### 5.1 核心思想

不经过线性化,直接选择 Lyapunov 函数 V(x),设计控制律使 dV/dt < 0。

### 5.2 设计步骤

1. **选择候选 Lyapunov 函数**:
   ```
   V(x) = 0.5·x'Px  (二次型)
   V(x) = 0.5·ml²θ'² + mgl(1-cos θ)  (物理能量)
   ```

2. **计算 dV/dt**:
   ```
   dV/dt = (∂V/∂x)·(f(x) + g(x)u)
   ```

3. **选择 u 使 dV/dt < 0**:
   ```
   dV/dt = L_f V + L_g V · u < 0
   若 L_g V ≠ 0:
   u = -(L_f V + γ·L_f V / |L_g V|²)  其中 γ > 0
   ```

### 5.3 倒立摆示例

```
V = 0.5·ml²θ'² + mgl(1-cos θ) + 0.5·k·θ²   (能量 + 位置惩罚)

dV/dt = θ'·(ml²θ'' + mgl·sin θ + k·θ)
      = θ'·(u - b·θ' + k·θ)

选择: u = -k·θ - (b + k_d)·θ'  (k_d > 0)
则: dV/dt = -k_d·θ'² ≤ 0  ✓ (渐近稳定)
```

## 6. 自适应控制

### 6.1 模型参考自适应控制 (MRAC)

```
参考模型: ẋ_m = A_m·x_m + B_m·r
控制律:   u = θ'·φ  (θ为可调参数, φ为回归向量)
自适应律: θ̇ = -Γ·e·φ  (MIT规则)
         θ̇ = -Γ·e·φ - σ·Γ·θ  (σ修正防漂移)
```

### 6.2 自校正调节器 (STR)

```
1. 在线辨识 (RLS): θ̂_{k+1} = θ̂_k + K_k·(y_k - φ_k'θ̂_k)
2. 基于θ̂设计控制器
3. "确定性等价原理": 将θ̂视为真值
```

### 6.3 增益调度

```
根据已知的调度变量(如工作点),查表或插值选择不同控制器参数。
要求: 调度变量变化慢于系统动态
```

## 7. 最优控制

### 7.1 动态规划 (DP)

**Bellman方程**:
```
V*(x,k) = min_u { l(x,u) + V*(f(x,u), k+1) }
```

**求解步骤** (逆向递推):
```
1. V(N, x) = l_f(x)  (终端代价)
2. k = N-1 → 0:
   对每个状态网格点 x_i:
     V(k, x_i) = min_u { l(x_i, u) + V(k+1, f(x_i, u)) }
     u*(k, x_i) = argmin { ... }
3. 查询: 在线查表获取 u*(x_current, k)
```

### 7.2 线性二次调节器 (LQR)

```
系统: ẋ = Ax + Bu
代价: J = ∫(x'Qx + u'Ru)dt + x_f'Px_f

解Riccati方程: A'P + PA - PBR⁻¹B'P + Q = 0
最优控制: u = -R⁻¹B'Px = -Kx
```

### 7.3 应用场景

| 方法 | 适用 | 计算量 | 实时性 |
|------|------|--------|--------|
| DP | 离散小规模 | 离线大 | 在线极快 |
| LQR | 线性/线性化系统 | 离线中 | 在线极快 |
| MPC | 有约束系统 | 在线大 | 需要快速求解器 |

## 8. 算法选型指南

```
┌─────────────────────────────────────────────────────┐
│               系统是否有精确模型?                      │
│         ┌───是───┐              ┌───否───┐           │
│         ↓        │              ↓        │           │
│   是否仿射形式?   │         ┌───在线辨识───┐          │
│    ┌─是─┐ ┌─否─┐│         ↓              │          │
│    ↓    │ ↓    ││     MRAC / STR         │          │
│  相对阶=n?│     ││                        │          │
│  ┌是┐┌否┐│     ││                        │          │
│  ↓  │↓  ││     ││                        │          │
│ 反馈│滑模│反步法││                        │          │
│ 线性│控制│     ││                        │          │
│ 化  │   │     ││                        │          │
└─────────────────────────────────────────────────────┘

选择标准:
1. 模型精确 + 相对阶=n → 反馈线性化 (最简单)
2. 模型精确 + 严格反馈 → 反步法 (系统化)
3. 模型不确定 + 匹配 → 滑模控制 (鲁棒)
4. 模型不确定 + 未知参数 → 自适应控制
5. 有约束 + 可离线计算 → 最优控制/DP
6. 快速实现 + 小角度 → Lyapunov直接法
```

## 9. 嵌入式实现要点

### 9.1 数值计算注意

```c
// Lie导数数值微分 — 选择合适步长
float delta = 1e-4f;  // 太小: 数值噪声; 太大: 近似误差

// 避免除零 (反馈线性化)
if (fabsf(denominator) < 1e-6f) {
    denominator = (denominator >= 0) ? 1e-6f : -1e-6f;
}

// 参数限幅 (自适应控制)
theta = fminf(theta_max, fmaxf(theta_min, theta));

// 饱和函数替代符号函数 (滑模控制)
float sat_s = (fabsf(s) <= phi) ? s/phi : (s > 0 ? 1.0f : -1.0f);
```

## 10. 电赛实战案例

### 10.1 倒立摆/平衡车

```
推荐: 反步法 或 反馈线性化
原因:
- 系统模型明确 (单摆/双摆)
- 大角度运动需要非线性方法
- 可叠加自适应处理摩擦不确定

参数整定:
- 反步法: c1=3~5, c2=5~10
- 反馈线性化: 极点 -3~-10
```

### 10.2 机械臂轨迹跟踪

```
推荐: 反馈线性化 + 前馈补偿
原因:
- 各关节耦合非线性,精确模型可用
- 反馈线性化可同时解耦和线性化
- 计算量在MCU可接受范围内

实现:
u = M(q)·(v + Kd·ė + Kp·e) + C(q,q̇)·q̇ + G(q)
```

### 10.3 电机伺服系统

```
推荐: 滑模控制 (参数不确定时) 或 反馈线性化 (模型精确时)
原因:
- 电机参数随温度、负载变化
- 滑模对参数变化鲁棒
- 响应速度快

参数整定:
- 滑模面: c = 10~50
- 切换增益: η = 系统不确定性上界的1.5倍
- 边界层: φ = 0.01~0.1
```

### 10.4 四旋翼姿态控制

```
推荐: 反步法 + 自适应
原因:
- 姿态动力学为严格反馈形式
- 质量/惯量不确定 → 自适应补偿
- 逐层设计便于模块化

结构:
外环 (位置): PID
内环 (姿态): 反步法 + 自适应
```
