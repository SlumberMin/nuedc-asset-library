# PID调参速查卡

## 快速调参流程（Ziegler-Nichols法）

### 步骤1：确定系统类型
- 一阶系统：响应慢，无超调
- 二阶系统：响应快，可能有超调
- 积分系统：位置控制等

### 步骤2：基础调参（试凑法）
```
1. 先调P：从小到大，直到系统振荡
2. 再调I：消除稳态误差，注意超调
3. 最后调D：抑制超调，加快响应
```

### 步骤3：Ziegler-Nichols临界比例度法
1. 令Ki=0, Kd=0
2. 逐渐增大Kp，直到系统等幅振荡
3. 记录临界增益Ku和振荡周期Tu
4. 按下表计算PID参数：

| 控制器类型 | Kp      | Ki       | Kd       |
|-----------|---------|----------|----------|
| P         | 0.5Ku   | -        | -        |
| PI        | 0.45Ku  | 0.54Ku/Tu| -        |
| PID       | 0.6Ku   | 1.2Ku/Tu | 0.075KuTu|

## 常见问题与解决方案

### 问题1：响应太慢
**症状**：系统达到目标值需要很长时间
**解决**：
- 增大Kp（加快响应）
- 适当增大Ki（消除稳态误差）
- 检查执行机构是否有死区

### 问题2：超调过大
**症状**：超过目标值很多才回来
**解决**：
- 减小Kp
- 减小Ki
- 增大Kd（抑制超调）
- 使用积分分离或抗饱和

### 问题3：稳态误差大
**症状**：无法达到目标值
**解决**：
- 增大Ki（主要手段）
- 检查是否有外部扰动
- 使用前馈补偿

### 问题4：振荡/不稳定
**症状**：系统持续振荡
**解决**：
- 减小Kp（首要措施）
- 减小Kd（可能过大）
- 增加滤波（微分项滤波）
- 检查采样时间是否合适

### 问题5：积分饱和
**症状**：响应慢，退出饱和时有大超调
**解决**：
- 使用抗饱和算法
- 限制积分项范围
- 条件积分（误差小时才积分）

## 参数经验范围（参考值）

### 电机速度控制
- Kp: 0.1~10
- Ki: 0.01~1
- Kd: 0.001~0.1

### 电机位置控制
- Kp: 1~100
- Ki: 0.01~1
- Kd: 0.1~10

### 温度控制
- Kp: 1~50
- Ki: 0.001~0.1
- Kd: 0~1

### 平衡车/倒立摆
- Kp: 10~100
- Ki: 0~10
- Kd: 0.1~10

### 飞行器姿态控制
- Kp: 1~10
- Ki: 0.1~1
- Kd: 0.1~1

## 采样时间选择原则

```
采样时间 Ts 应满足：
1. Ts << 系统时间常数（一般 Ts < T/10）
2. Ts << 期望上升时间（一般 Ts < Tr/10）
3. Ts > 控制器计算时间
4. 采样频率 > 2×系统带宽频率

经验值：
- 电机控制：1~10 ms
- 温度控制：100~1000 ms
- 平衡控制：1~5 ms
- 飞控姿态：1~4 ms
```

## 微分项滤波

### 问题：微分项对噪声敏感
### 解决：一阶低通滤波

```
滤波系数 α = Ts / (Tf + Ts)
Tf为滤波时间常数，一般取 Tf = (0.1~0.2) × Kd/Kp

滤波后微分 = α × 上次滤波值 + (1-α) × 当前微分值
```

## 串级PID调参顺序

```
1. 先调内环（响应快的环）
   - 通常只需要P或PD控制
   - 确保内环稳定且响应快

2. 再调外环（响应慢的环）
   - 内环作为外环的执行机构
   - 外环输出作为内环的设定值

典型应用：
- 电机位置-速度串级
- 平衡车角度-速度串级
- 飞控角度-角速度串级
```

## 快速诊断流程图

```
系统不稳定？
├─ 是 → 减小Kp，检查采样时间
└─ 否 → 响应太慢？
         ├─ 是 → 增大Kp
         └─ 否 → 有稳态误差？
                  ├─ 是 → 增大Ki
                  └─ 否 → 超调过大？
                           ├─ 是 → 增大Kd，减小Ki
                           └─ 否 → 调参完成✓
```

## 常用PID变体

| 变体类型 | 适用场景 | 优点 |
|---------|---------|------|
| 增量式PID | 执行机构接受增量 | 无扰切换，抗积分饱和 |
| 位置式PID | 执行机构接受绝对值 | 直观，易于理解 |
| 速度式PID | 需要速度输出 | 计算量小 |
| 带滤波PID | 噪声环境 | 抑制微分噪声 |
| 前馈+PID | 已知扰动模型 | 提高响应速度 |
| 串级PID | 多变量耦合系统 | 提高抗扰性 |

## 实用公式汇总

### 位置式PID
```
u(t) = Kp × e(t) + Ki × ∫e(t)dt + Kd × de(t)/dt

离散化：
u(k) = Kp × e(k) + Ki × Σe(i)×Ts + Kd × (e(k)-e(k-1))/Ts
```

### 增量式PID
```
Δu(k) = u(k) - u(k-1)
       = Kp × (e(k)-e(k-1)) + Ki × e(k)×Ts + Kd × (e(k)-2e(k-1)+e(k-2))/Ts

u(k) = u(k-1) + Δu(k)
```

### 积分抗饱和（反计算）
```
u_sat = clamp(u_raw, min, max)
积分修正 = e + Kb × (u_sat - u_raw) / Tt
```

### 前馈补偿
```
u_ff = Kff × setpoint + Kff_d × d(setpoint)/dt

总输出 = PID输出 + 前馈输出
```
