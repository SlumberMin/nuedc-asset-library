# 控制算法选型指南

> 电赛常用控制算法对比与选型建议

## 一、算法总览

| 算法 | 复杂度 | 鲁棒性 | 精度 | 实时性 | 典型应用 |
|------|--------|--------|------|--------|----------|
| PID | ★★☆☆☆ | ★★☆☆☆ | ★★★☆☆ | ★★★★★ | 电机/温度/液位 |
| 模糊PID | ★★★☆☆ | ★★★★☆ | ★★★★☆ | ★★★★☆ | 非线性/时变系统 |
| 滑模控制 | ★★★☆☆ | ★★★★★ | ★★★★☆ | ★★★★☆ | 电机/倒立摆/机器人 |
| ADRC | ★★★★☆ | ★★★★★ | ★★★★★ | ★★★☆☆ | 高精度位置/姿态 |
| LQR | ★★★☆☆ | ★★★☆☆ | ★★★★★ | ★★★★★ | 倒立摆/平衡车 |
| MPC | ★★★★★ | ★★★★☆ | ★★★★★ | ★★☆☆☆ | 轨迹跟踪/多变量 |
| 卡尔曼 | ★★★☆☆ | ★★★★☆ | ★★★★★ | ★★★★☆ | 传感器融合/滤波 |

## 二、选型决策树

```
需要控制什么？
│
├─ 简单系统(电机转速、温度)
│  ├─ 无特殊要求 → PID
│  ├─ 参数变化大 → 模糊PID
│  └─ 传感器噪声大 → PID + 卡尔曼滤波
│
├─ 快速响应系统(电机位置、平衡)
│  ├─ 需要强鲁棒性 → 滑模控制
│  ├─ 有精确模型 → LQR
│  └─ 未知模型+有扰动 → ADRC
│
├─ 复杂系统(多变量、有约束)
│  ├─ 有模型+有约束 → MPC
│  └─ 无模型+有约束 → ADRC + 约束投影
│
└─ 传感器融合
   ├─ 两路传感器 → 互补滤波
   ├─ 需要速度估计 → 卡尔曼滤波
   └─ 姿态估计(加速度计+陀螺仪) → 互补/卡尔曼/四元数
```

## 三、各算法详解

### 3.1 PID控制

**优点**: 简单、成熟、几乎所有MCU都能实现  
**缺点**: 对非线性、时变系统效果差，需要调参  
**电赛场景**: 90%的题目都可以用PID解决

**推荐配置**:
- 电机速度环: 位置式PID + 积分分离
- 温度控制: 位置式PID + 积分限幅
- 平衡控制: 位置式PID + 微分先行

**参数范围参考**:

| 控制对象 | Kp | Ki | Kd |
|----------|-----|-----|-----|
| 直流电机速度 | 5~50 | 1~20 | 0~5 |
| 直流电机位置 | 1~10 | 0~2 | 0.5~5 |
| 温度控制 | 2~20 | 0.1~5 | 0~10 |
| 舵机角度 | 1~5 | 0~1 | 0.1~2 |

### 3.2 模糊PID

**优点**: 无需精确模型，自适应调整参数  
**缺点**: 规则表设计依赖经验，计算量比PID大  
**电赛场景**: 系统特性不确定、参数变化大的场合

**适用场景**:
- 水温控制(加热/散热特性不同)
- 机械臂(不同姿态惯量不同)
- 任何"调了一组PID参数但不同工况效果差异大"的情况

### 3.3 滑模控制

**优点**: 对参数摄动和外部干扰有极强鲁棒性  
**缺点**: 存在抖振问题(输出高频振荡)  
**电赛场景**: 需要快速响应且鲁棒性要求高的场合

**抖振抑制技巧**:
1. 用饱和函数(sat)替代符号函数(sgn)
2. 边界层法: 在滑模面附近用线性控制
3. 输出低通滤波
4. 趋近律选择指数趋近律

### 3.4 ADRC自抗扰

**优点**: 不依赖模型，自动估计补偿扰动  
**缺点**: 参数多(12个)，调试难度大  
**电赛场景**: 精度要求高、扰动大的场合

**参数整定顺序**:
1. 先调TD的r(速度因子)
2. 再调ESO的β1, β2, β3(观测器增益)
3. 最后调NLSEF的β0, β1

### 3.5 LQR最优控制

**优点**: 理论最优，可计算反馈增益  
**缺点**: 需要精确的线性数学模型  
**电赛场景**: 倒立摆、平衡车等有明确数学模型的场合

**权重矩阵设计**:
- Q大 → 重视状态误差 → 响应快但控制量大
- R大 → 重视控制能量 → 响应慢但平滑
- 先设Q=I, R=1, 再根据效果调整

### 3.6 MPC模型预测

**优点**: 可处理约束，多变量优化  
**缺点**: 计算量大，需要数学模型  
**电赛场景**: 有明确约束条件的轨迹跟踪

**嵌入式优化**:
- 预测时域Np取3~5
- 控制时域Nc取1~3
- 用梯度法替代QP求解

### 3.7 卡尔曼滤波

**优点**: 最优估计，可估计不可测量的状态  
**缺点**: 需要噪声统计特性  
**电赛场景**: 传感器数据处理、状态估计

**典型应用**:
- 编码器测速去噪
- 加速度计+陀螺仪姿态融合
- ADC采样值滤波
- 速度估计(从位置测量)

## 四、组合使用推荐

| 场景 | 推荐方案 |
|------|----------|
| 电机速度控制 | PID + 卡尔曼滤波 |
| 电机位置控制 | 串级PID(位置外环+速度内环) |
| 平衡车 | LQR或PD + 卡尔曼(互补) |
| 倒立摆 | 滑模控制 或 LQR |
| 温度控制 | 模糊PID 或 PID + 积分分离 |
| 视觉跟踪 | PID + 卡尔曼滤波 |
| 轨迹跟踪 | MPC 或 ADRC |
| 机械臂 | ADRC 或 计算力矩法+PID |

## 五、电赛实战建议

1. **首选PID**: 除非有明确理由，否则PID是最快实现、最可靠的方案
2. **加滤波**: 任何传感器数据都建议先过卡尔曼或低通滤波
3. **串级控制**: 位置控制用串级PID(位置环+速度环)效果远好于单环
4. **前馈+反馈**: 对已知扰动(如重力)加前馈补偿，比纯反馈好得多
5. **积分分离**: 任何PID都建议加积分分离，避免积分饱和
6. **先调P再调I最后D**: 手动调参的基本顺序
