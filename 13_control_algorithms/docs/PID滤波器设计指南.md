# PID滤波器设计指南

## 1. 概述

在实际嵌入式系统中，传感器信号不可避免地包含噪声。PID控制器的**微分项(D项)**对高频噪声极为敏感，因为微分运算本质上是一个高通滤波器——噪声中的高频分量会被放大，导致控制输出剧烈抖动，甚至引起执行器损坏。

**滤波PID**通过在微分通道和/或输出通道加入低通滤波器，有效解决这一问题。

---

## 2. 问题分析: 为什么需要滤波

### 2.1 微分项的噪声放大效应

假设传感器噪声为白噪声 `n(t)`，其功率谱密度为 `S_n(f) = N₀`。

- 微分运算的频率响应: `H(f) = j·2πf`
- 微分后噪声功率谱: `S_out(f) = |H(f)|² · S_n(f) = (2πf)² · N₀`

**结论**: 频率越高，微分后的噪声越大。1kHz处的噪声被放大约4000倍！

### 2.2 输出抖动问题

即使比例项和积分项正常，微分项的噪声也会导致:
- 控制输出存在高频毛刺
- PWM信号不稳定 → 电机发出异响
- 执行器频繁换向 → 寿命缩短
- 电流纹波增大 → EMI问题

### 2.3 设定值突变冲击

当设定值发生阶跃变化时:
- 误差 `e = SP - PV` 突变
- 微分项 `D = kd · de/dt` 产生脉冲
- 输出出现巨大尖峰

---

## 3. 滤波器设计

### 3.1 一阶IIR低通滤波器

最简单且嵌入式友好的滤波器:

```
y[n] = α · x[n] + (1-α) · y[n-1]
```

其中:
- `x[n]`: 当前输入
- `y[n]`: 当前输出(滤波后)
- `y[n-1]`: 上一次输出
- `α`: 滤波系数 (0 < α ≤ 1)

**等价形式**(计算更高效):
```
y[n] = y[n-1] + α · (x[n] - y[n-1])
```

### 3.2 截止频率计算

一阶低通滤波器的截止频率 `fc` 与 `α` 的关系:

```
fc = -ln(1-α) / (2π·dt)
```

反算:
```
α = 1 - exp(-2π·fc·dt)
```

### 3.3 常用参数速查表

| 采样周期 dt | α = 0.1 | α = 0.2 | α = 0.3 | α = 0.5 |
|:-----------:|:-------:|:-------:|:-------:|:-------:|
| 0.5 ms      | 32 Hz   | 70 Hz   | 113 Hz  | 221 Hz  |
| 1 ms        | 16 Hz   | 35 Hz   | 56 Hz   | 110 Hz  |
| 2 ms        | 8 Hz    | 18 Hz   | 28 Hz   | 55 Hz   |
| 5 ms        | 3.2 Hz  | 7 Hz    | 11 Hz   | 22 Hz   |
| 10 ms       | 1.6 Hz  | 3.5 Hz  | 5.6 Hz  | 11 Hz   |

### 3.4 选择滤波系数的原则

1. **截止频率 > 信号带宽**: 滤波器不应影响有用信号
2. **截止频率 < 噪声主频**: 尽可能多地滤除噪声
3. **折中**: 在噪声抑制和响应速度之间取平衡

**经验法则**: 截止频率取系统带宽的5~10倍

## 4. 滤波位置选择

### 4.1 方案A: 仅微分滤波

```
D项: d_raw = kd · (e[n] - e[n-1]) / dt
     d_filtered = α_d · d_raw + (1-α_d) · d_filtered_prev
输出: u = P + I + D_filtered
```

**优点**: 只影响D通道，PI部分不受影响
**适用**: 噪声主要影响微分项时

### 4.2 方案B: 仅输出滤波

```
D项: d = kd · (e[n] - e[n-1]) / dt (不滤波)
输出: u_raw = P + I + D
      u_filtered = α_out · u_raw + (1-α_out) · u_filtered_prev
```

**优点**: 整个输出平滑
**缺点**: 引入额外相位滞后，影响稳定性
**适用**: 输出信号整体需要平滑时

### 4.3 方案C: 微分+输出双滤波 (推荐)

```
D项: d_raw = kd · (e[n] - e[n-1]) / dt
     d_filtered = α_d · d_raw + (1-α_d) · d_filtered_prev
输出: u_raw = P + I + D_filtered
      u_filtered = α_out · u_raw + (1-α_out) · u_filtered_prev
```

**推荐**: 微分滤波用较小α(如0.1~0.3)，输出滤波用较大α(如0.3~0.5)

## 5. 微分信号来源: 误差微分 vs PV微分

### 5.1 对误差微分 (传统方法)

```
D = kd · (e[n] - e[n-1]) / dt
  = kd · ((SP[n]-PV[n]) - (SP[n-1]-PV[n-1])) / dt
```

**问题**: 当SP发生阶跃时，D项产生巨大脉冲

### 5.2 对PV微分 (推荐)

```
D = kd · (-(PV[n] - PV[n-1])) / dt
```

**优点**:
- SP变化时D项不会突变(因为D只看PV的变化)
- 跟踪控制中特别有用
- 大多数工业PID采用此方式

**注意**: 对PV微分时，D项的符号是**负的PV变化率**，即阻尼作用

### 5.3 对比

| 特性 | 误差微分 | PV微分 |
|:----:|:-------:|:-----:|
| SP阶跃响应 | D项脉冲大 | D项无冲击 |
| 跟踪控制 | 可能不稳定 | 稳定 |
| 调节控制 | 两者相近 | 两者相近 |
| 实现复杂度 | 简单 | 简单 |
| **推荐度** | ★★★ | ★★★★★ |

## 6. 嵌入式实现要点

### 6.1 定点化注意事项

对于定点MCU:
```
// Q16.16格式
alpha = FLOAT_TO_Q16(0.2f);   // α = 0.2
y = Q16_MUL(alpha, x) + Q16_MUL(Q16_ONE - alpha, y_prev);
```

**注意**: α不能太小，否则定点精度不足。建议α ≥ 0.05。

### 6.2 首次运行处理

```c
if (first_run) {
    d_filtered = 0.0f;     // 或 d_raw
    out_filtered = raw_output;  // 避免从0跳变
    first_run = 0;
}
```

### 6.3 运算量分析

每次调用PID+滤波器的额外运算:
- 一阶IIR: 2次乘法 + 1次加法
- 如果D和输出都滤波: 4次乘法 + 2次加法
- 对于100MHz MCU，约0.1~0.5μs

### 6.4 资源占用

- 额外RAM: 每个滤波器1个float(4字节)
- 额外Flash: 约50~100字节代码

## 7. 调参指南

### 7.1 步骤

1. **先不加滤波器**: 用标准PID调出基本可用的kp、ki、kd
2. **观察D项噪声**: 记录D项原始值，评估噪声水平
3. **设定微分滤波α**: 从0.3开始，逐步减小到噪声可接受
4. **观察输出抖动**: 如果输出仍抖动，加入输出滤波
5. **设定输出滤波α**: 从0.5开始，逐步减小
6. **验证动态性能**: 确保滤波没有使系统变慢太多
7. **微调PID增益**: 滤波引入的滞后可能需要略微调整增益

### 7.2 故障排除

| 现象 | 可能原因 | 解决方案 |
|:----:|:-------:|:-------:|
| 输出仍有高频毛刺 | D滤波α太大 | 减小α |
| 响应变慢/超调增大 | 滤波引入滞后太大 | 增大α或只用D滤波 |
| 稳态有小振荡 | 微分噪声残余 | 加入死区或减小kd |
| SP突变时输出尖峰 | 使用误差微分 | 改用PV微分 |
| 定点运算溢出 | α太小精度不足 | 增大α或增大Q格式位宽 |

## 8. 进阶: 高阶滤波器

### 8.1 二阶IIR (Biquad)

当一阶滤波器衰减不够时，可使用二阶:

```
y[n] = b0·x[n] + b1·x[n-1] + b2·x[n-2] - a1·y[n-1] - a2·y[n-2]
```

Butterworth二阶LPF系数(截止频率fc, 采样率fs):
```
ωc = 2π·fc/fs
K = tan(ωc/2)
norm = 1 / (1 + √2·K + K²)
b0 = K² · norm
b1 = 2·K² · norm
b2 = K² · norm
a1 = 2·(K² - 1) · norm
a2 = (1 - √2·K + K²) · norm
```

### 8.2 移动平均滤波器

```
y[n] = (1/N) · Σ x[n-k], k=0..N-1
```

- 优点: 无相位失真(线性相位)
- 缺点: 需要N个存储单元，延迟N/2个采样周期
- 适用: 对相位要求严格的场合

### 8.3 中值滤波器

```
y[n] = median(x[n], x[n-1], x[n-2])
```

- 优点: 对脉冲噪声(尖峰)效果极好
- 缺点: 对连续噪声效果一般
- 适用: 编码器偶发跳变、ADC毛刺

## 9. 实际工程经验

### 9.1 电赛常见场景

| 场景 | 推荐滤波方案 | 典型参数 |
|:----:|:----------:|:-------:|
| 直流电机速度环 | D滤波 + PV微分 | α_d=0.2, dt=1ms |
| 舵机位置环 | 输出滤波 | α_out=0.5 |
| 平衡车 | D滤波(PV微分) | α_d=0.15 |
| 温度控制 | 输出滤波 | α_out=0.3 |
| 电流环 | D滤波 | α_d=0.3, dt=100μs |

### 9.2 常见误区

1. **滤波不能代替好的传感器**: 硬件滤波(RC滤波)永远优于软件滤波
2. **不要过度滤波**: 会使系统迟钝，甚至不稳定
3. **D滤波和输出滤波不要都用很小的α**: 累积滞后会很大
4. **注意滤波器的初始化**: 首次运行时应设初始值，避免从0跳变
5. **采样频率要足够高**: 至少是信号带宽的10倍
