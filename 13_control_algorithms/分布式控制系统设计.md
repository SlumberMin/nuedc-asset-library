## 一、分布式控制概述

### 1.1 核心概念

分布式控制：多个独立智能体通过**局部通信**协调行为，实现**全局目标**。

| 对比项 | 集中式 | 分布式 |
|--------|--------|--------|
| 通信拓扑 | 星型（中心节点） | 网状/图 |
| 容错性 | 单点故障 | 部分节点失效可容忍 |
| 扩展性 | 差 | 好 |
| 计算负担 | 集中于主机 | 分散到各节点 |

### 1.2 电赛典型场景

- 多车协同（避障/编队/接力）
- 分布式传感器网络（温度/湿度场测量）
- 多机械臂协同搬运
- 无人机编队飞行（视觉/通信组网）

## 二、图论基础

### 2.1 通信拓扑建模

```
图 G = (V, E, A)
  V = {1, 2, ..., n}    节点集合（智能体）
  E ⊆ V×V              边集合（通信链路）
  A = [aij]             邻接矩阵
```

### 2.2 关键矩阵

```
度矩阵：D = diag(d1, d2, ..., dn), di = Σj aij
拉普拉斯矩阵：L = D - A
```

**拉普拉斯矩阵性质**：
- 行和为0 → 总有特征值0
- 连通图 → 0是单重特征值
- 第二小特征值λ₂ > 0（代数连通度）→ 决定一致性收敛速度

### 2.3 常见拓扑

```
全连接：   所有节点互连，λ₂最大
环形：     每节点连左右邻居
星形：     中心节点连所有
链式：     1-2-3-...-n
随机图：   概率连边
```

## 三、一致性算法

### 3.1 离散一致性

```
xi(k+1) = xi(k) + ε·Σj aij·(xj(k) - xi(k))

矩阵形式：
x(k+1) = (I - εL)·x(k)
```

收敛条件：0 < ε < 1/λₘₐₓ(L)

### 3.2 连续一致性

```
ẋi = Σj aij·(xj - xi)

矩阵形式：
ẋ = -L·x
```

收敛到平均值：x∞ = (1ᵀx₀/n)·1

### 3.3 带参考状态的一致性

```
ẋi = Σj aij·(xj - xi) + bi(x₀ - xi)
```

bi > 0 表示节点i能接收参考状态x₀（leader信息）。

### 3.4 领航-跟随一致性

```
Leader:  ẋ₀ = f(x₀)
Follower: ẋi = Σj aij·(xj - xi) + bi(x₀ - xi) + ui

其中 ui 为额外控制输入（如编队偏移量）
```

## 四、编队控制

### 4.1 基于一致性的编队

```
ẋi = Σj aij·[(xj - xi) - Δij]

Δij = 期望相对位置差
```

### 4.2 虚拟结构法

```
1. 定义虚拟编队中心 xc
2. 每个agent追踪 xc + δi（δi为编队偏移）
3. 编队中心由领导者驱动

ẋi = -K(xi - (xc + δi))
```

### 4.3 行为法

```
各行为加权求和：
ui = w₁·u_formation + w₂·u_obstacle_avoid + w₃·u_goal_seeking + w₄·u_velocity_match
```

### 4.4 队形变换

```
平移：δi' = δi + Δc（整体平移）
旋转：δi' = R(θ)·δi（整体旋转）
缩放：δi' = α·δi（整体缩放）
```

## 五、分布式控制器设计

### 5.1 分布式PID

```
ui = Kp·ei + Ki·∫ei + Kd·ėi

ei = Σj aij·(xj - xi) + bi·(xref - xi)  // 一致性误差
```

### 5.2 分布式MPC

```
每个agent求解局部优化：
min Σk ‖xi(k)‖²Q + ‖ui(k)‖²R

s.t. 动力学约束 + 邻居耦合约束
```

### 5.3 分布式事件触发控制

```
触发条件：‖ei(t)‖ ≥ σ·‖xi(t)‖ + δ

ei = xi(t) - xi(tₖ)  // 上次触发时的状态
σ ∈ (0,1), δ > 0

仅在触发时刻通信 → 节省带宽
```

## 七、STM32实现模板

```c
// 一致性控制核心
#define MAX_NODES 8
#define ALPHA     0.5f  // 一致性增益

typedef struct {
    float x, y, theta;   // 本节点状态
    float x_ref, y_ref;  // 参考位置（编队偏移）
    uint8_t id;
    uint8_t adj[MAX_NODES]; // 邻接关系
    float neighbor_x[MAX_NODES];
    float neighbor_y[MAX_NODES];
} Agent_t;

void Consensus_Update(Agent_t *agent, float dt) {
    float ux = 0, uy = 0;
    
    for (int j = 0; j < MAX_NODES; j++) {
        if (agent->adj[j]) {
            // 一致性项 + 编队偏移
            ux += ALPHA * ((agent->neighbor_x[j] - agent->x) 
                           - (agent->x_ref - agent->neighbor_x_ref[j]));
            uy += ALPHA * ((agent->neighbor_y[j] - agent->y)
                           - (agent->y_ref - agent->neighbor_y_ref[j]));
        }
    }
    
    // Leader跟踪项（如果有）
    if (has_leader_info) {
        ux += ALPHA * (leader_x - agent->x);
        uy += ALPHA * (leader_y - agent->y);
    }
    
    agent->x += ux * dt;
    agent->y += uy * dt;
}
```

## 八、调试要点

| 问题 | 原因 | 解决 |
|------|------|------|
| 不收敛 | 拓扑不连通 | 确保图连通，检查通信 |
| 收敛慢 | λ₂太小 | 增加通信链路 |
| 通信丢包 | 距离/干扰 | 降速率、加重传、增大功率 |
| 时钟不同步 | 时间戳漂移 | 用相对时间或NTP同步 |
| 队形发散 | 增益过大 | 降低α，加饱和限幅 |
