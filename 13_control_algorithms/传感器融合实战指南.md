# 传感器融合实战指南

> 互补滤波 + 卡尔曼滤波 + Mahony + Madgwick 全解析

---

## 一、为什么需要传感器融合？

### 1.1 单一传感器的局限

| 传感器 | 优点 | 缺点 |
|--------|------|------|
| 加速度计 | 测量绝对角度（无漂移） | 高频噪声大、受振动影响 |
| 陀螺仪 | 高频响应好、精度高 | 存在零漂、积分累积误差 |
| 磁力计 | 提供绝对航向 | 易受磁场干扰 |
| 编码器 | 位置精确 | 只能测量相对位移 |

### 1.2 融合的核心思想

> **取长补短**：用陀螺仪的快速响应弥补加速度计的噪声，用加速度计的绝对参考弥补陀螺仪的漂移。

## 二、互补滤波（入门级）

### 2.1 原理

最简单的传感器融合方法：

```
角度 = α × (角度 + 陀螺仪 × dt) + (1-α) × 加速度计角度
```

- **高频部分**（快速变化）信任陀螺仪
- **低频部分**（缓慢变化/绝对参考）信任加速度计
- α 通常取 0.95 ~ 0.99

### 2.2 代码实现

```c
typedef struct {
    float angle;      // 融合后的角度
    float alpha;      // 滤波系数（0.95~0.99）
    float dt;         // 采样周期
} ComplementaryFilter_t;

void Complementary_Init(ComplementaryFilter_t *cf, float alpha, float dt) {
    cf->angle = 0;
    cf->alpha = alpha;
    cf->dt = dt;
}

float Complementary_Update(ComplementaryFilter_t *cf, float acc_angle, float gyro_rate) {
    // 陀螺仪积分（高频通道）
    cf->angle = cf->alpha * (cf->angle + gyro_rate * cf->dt)
              + (1 - cf->alpha) * acc_angle;  // 加速度计修正（低频通道）
    return cf->angle;
}
```

### 2.3 α值选择

| α值 | 效果 | 适用场景 |
|-----|------|----------|
| 0.90 | 更信任加速度计 | 振动小、需要快速收敛 |
| 0.95 | 平衡 | 一般应用 |
| 0.98 | 更信任陀螺仪 | 振动大、短期精度要求高 |
| 0.99 | 高度信任陀螺仪 | 强振动环境 |

### 2.4 优缺点

**优点**：实现简单、计算量小、调参直观
**缺点**：参数固定、无法处理非线性、精度有限

## 三、卡尔曼滤波（经典级）

### 3.1 原理简介

卡尔曼滤波是一种**最优估计算法**，基于状态空间模型：

```
预测步骤：
  x̂⁻ = A × x̂ + B × u     （状态预测）
  P⁻  = A × P × Aᵀ + Q     （协方差预测）

更新步骤：
  K  = P⁻ × Hᵀ × (H × P⁻ × Hᵀ + R)⁻¹  （卡尔曼增益）
  x̂  = x̂⁻ + K × (z - H × x̂⁻)            （状态更新）
  P  = (I - K × H) × P⁻                   （协方差更新）
```

### 3.2 一维卡尔曼滤波实现

```c
typedef struct {
    float x;      // 状态估计值（角度）
    float P;      // 估计协方差
    float Q;      // 过程噪声协方差
    float R;      // 测量噪声协方差
    float K;      // 卡尔曼增益
} KalmanFilter_t;

void Kalman_Init(KalmanFilter_t *kf, float Q, float R, float P0) {
    kf->x = 0;
    kf->P = P0;
    kf->Q = Q;   // 过程噪声（陀螺仪噪声）
    kf->R = R;   // 测量噪声（加速度计噪声）
}

float Kalman_Update(KalmanFilter_t *kf, float gyro_rate, float acc_angle, float dt) {
    // === 预测步骤 ===
    kf->x += gyro_rate * dt;        // 状态预测（陀螺仪积分）
    kf->P += kf->Q;                 // 协方差预测

    // === 更新步骤 ===
    kf->K = kf->P / (kf->P + kf->R);   // 卡尔曼增益
    kf->x += kf->K * (acc_angle - kf->x); // 状态更新
    kf->P *= (1 - kf->K);               // 协方差更新

    return kf->x;
}
```

### 3.3 参数调节指南

| 参数 | 含义 | 调大效果 | 调小效果 |
|------|------|----------|----------|
| Q（过程噪声） | 陀螺仪不确定度 | 更信任加速度计、响应快但噪声大 | 更信任陀螺仪、平滑但响应慢 |
| R（测量噪声） | 加速度计不确定度 | 更信任陀螺仪、平滑 | 更信任加速度计、响应快 |
| P0（初始协方差） | 初始不确定度 | 收敛快 | 收敛慢 |

**经验取值**：
```
Q = 0.001 ~ 0.01  （陀螺仪噪声方差）
R = 0.01 ~ 1.0    （加速度计噪声方差）
P0 = 1.0          （初始值，较快收敛）
```

### 3.4 二维卡尔曼滤波（角度+角速度）

```c
typedef struct {
    float x[2];     // 状态：[角度, 角速度偏置]
    float P[2][2];  // 协方差矩阵
    float Q[2][2];  // 过程噪声
    float R;        // 测量噪声
} KalmanFilter2D_t;

float Kalman2D_Update(KalmanFilter2D_t *kf, float gyro, float acc_angle, float dt) {
    // 预测
    kf->x[0] += (gyro - kf->x[1]) * dt;  // 角度 = (陀螺仪-偏置)*dt
    // kf->x[1] 不变（偏置假设不变）
    
    float P00 = kf->P[0][0] + dt * (dt*kf->P[1][1] - kf->P[0][1] - kf->P[1][0]) + kf->Q[0][0];
    float P01 = kf->P[0][1] - dt * kf->P[1][1];
    float P10 = kf->P[1][0] - dt * kf->P[1][1];
    float P11 = kf->P[1][1] + kf->Q[1][1];
    kf->P[0][0] = P00; kf->P[0][1] = P01;
    kf->P[1][0] = P10; kf->P[1][1] = P11;
    
    // 更新
    float S = kf->P[0][0] + kf->R;
    float K0 = kf->P[0][0] / S;
    float K1 = kf->P[1][0] / S;
    
    float y = acc_angle - kf->x[0];  // 残差
    kf->x[0] += K0 * y;
    kf->x[1] += K1 * y;
    
    float P00_new = (1 - K0) * kf->P[0][0];
    float P01_new = (1 - K0) * kf->P[0][1];
    float P10_new = kf->P[1][0] - K1 * kf->P[0][0];
    float P11_new = kf->P[1][1] - K1 * kf->P[0][1];
    kf->P[0][0] = P00_new; kf->P[0][1] = P01_new;
    kf->P[1][0] = P10_new; kf->P[1][1] = P11_new;
    
    return kf->x[0];
}
```

## 四、Mahony互补滤波（AHRS级）

### 4.1 原理

Mahony滤波器是一种**基于四元数的互补滤波**，专为AHRS（姿态航向参考系统）设计：
- 用重力向量作为参考，修正陀螺仪漂移
- 用磁力计修正航向
- 输出四元数或欧拉角

### 4.2 完整代码实现

```c
#include <math.h>

typedef struct {
    float q0, q1, q2, q3;  // 四元数
    float Kp, Ki;           // 增益参数
    float integralFBx, integralFBy, integralFBz;  // 积分反馈
    float dt;
} MahonyAHRS_t;

MahonyAHRS_t mahony = {1, 0, 0, 0, 10.0, 0.0, 0, 0, 0, 0.001};

void Mahony_UpdateIMU(MahonyAHRS_t *ahrs, 
                       float gx, float gy, float gz,   // 陀螺仪 (rad/s)
                       float ax, float ay, float az)    // 加速度计 (归一化)
{
    float recipNorm;
    float halfvx, halfvy, halfvz;
    float halfex, halfey, halfez;
    float qa, qb, qc;

    // 加速度计归一化
    recipNorm = sqrt(ax*ax + ay*ay + az*az);
    if (recipNorm < 1e-6f) return;  // 避免除零
    ax *= recipNorm;  ay *= recipNorm;  az *= recipNorm;

    // 估计重力方向（从四元数计算）
    halfvx = ahrs->q1 * ahrs->q3 - ahrs->q0 * ahrs->q2;
    halfvy = ahrs->q0 * ahrs->q1 + ahrs->q2 * ahrs->q3;
    halfvz = ahrs->q0 * ahrs->q0 - 0.5f + ahrs->q3 * ahrs->q3;

    // 计算误差（叉积）
    halfex = (ay * halfvz - az * halfvy);
    halfey = (az * halfvx - ax * halfvz);
    halfez = (ax * halfvy - ay * halfvx);

    // 积分反馈（修正陀螺仪偏置）
    if (ahrs->Ki > 0.0f) {
        ahrs->integralFBx += ahrs->Ki * halfex * ahrs->dt;
        ahrs->integralFBy += ahrs->Ki * halfey * ahrs->dt;
        ahrs->integralFBz += ahrs->Ki * halfez * ahrs->dt;
        gx += ahrs->integralFBx;
        gy += ahrs->integralFBy;
        gz += ahrs->integralFBz;
    }

    // 比例反馈
    gx += ahrs->Kp * halfex;
    gy += ahrs->Kp * halfey;
    gz += ahrs->Kp * halfez;

    // 四元数更新
    gx *= (0.5f * ahrs->dt);
    gy *= (0.5f * ahrs->dt);
    gz *= (0.5f * ahrs->dt);
    
    qa = ahrs->q0;  qb = ahrs->q1;  qc = ahrs->q2;
    ahrs->q0 += (-qb * gx - qc * gy - ahrs->q3 * gz);
    ahrs->q1 += (qa * gx + qc * gz - ahrs->q3 * gy);
    ahrs->q2 += (qa * gy - qb * gz + ahrs->q3 * gx);
    ahrs->q3 += (qa * gz + qb * gy - qc * gx);

    // 四元数归一化
    recipNorm = sqrt(ahrs->q0*ahrs->q0 + ahrs->q1*ahrs->q1 + 
                     ahrs->q2*ahrs->q2 + ahrs->q3*ahrs->q3);
    ahrs->q0 /= recipNorm;
    ahrs->q1 /= recipNorm;
    ahrs->q2 /= recipNorm;
    ahrs->q3 /= recipNorm;
}

// 四元数转欧拉角
void QuaternionToEuler(float q0, float q1, float q2, float q3,
                       float *roll, float *pitch, float *yaw) {
    *roll  = atan2(2*(q0*q1 + q2*q3), 1 - 2*(q1*q1 + q2*q2));
    *pitch = asin(2*(q0*q2 - q3*q1));
    *yaw   = atan2(2*(q0*q3 + q1*q2), 1 - 2*(q2*q2 + q3*q3));
}
```

### 4.3 参数调节

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| Kp | 2.0 ~ 50.0 | 比例增益，大则收敛快但可能振荡 |
| Ki | 0.0 ~ 0.1 | 积分增益，用于消除陀螺仪偏置 |

**调试步骤**：
1. Ki=0，调Kp：从10开始，直到姿态稳定
2. 加入小的Ki（如0.005），修正陀螺仪偏置
3. 观察是否出现漂移或振荡

## 五、Madgwick滤波器（高效级）

### 5.1 原理

Madgwick滤波器使用**梯度下降法**来估计姿态：
- 计算误差函数的梯度
- 沿梯度反方向更新四元数
- 与陀螺仪积分结果融合

### 5.2 核心代码

```c
typedef struct {
    float q0, q1, q2, q3;
    float beta;  // 梯度下降步长
    float dt;
} MadgwickAHRS_t;

MadgwickAHRS_t madgwick = {1, 0, 0, 0, 0.1, 0.001};

void Madgwick_UpdateIMU(MadgwickAHRS_t *ahrs,
                         float gx, float gy, float gz,
                         float ax, float ay, float az)
{
    float recipNorm;
    float s0, s1, s2, s3;
    float qDot1, qDot2, qDot3, qDot4;
    float _2q0, _2q1, _2q2, _2q3;
    float _4q0, _4q1, _4q2;
    float _8q1, _8q2;
    float q0q0, q1q1, q2q2, q3q3;

    // 加速度计归一化
    recipNorm = sqrt(ax*ax + ay*ay + az*az);
    if (recipNorm < 1e-6f) return;
    ax /= recipNorm;  ay /= recipNorm;  az /= recipNorm;

    // 辅助变量
    _2q0 = 2.0f * ahrs->q0;  _2q1 = 2.0f * ahrs->q1;
    _2q2 = 2.0f * ahrs->q2;  _2q3 = 2.0f * ahrs->q3;
    _4q0 = 4.0f * ahrs->q0;  _4q1 = 4.0f * ahrs->q1;
    _4q2 = 4.0f * ahrs->q2;
    _8q1 = 8.0f * ahrs->q1;  _8q2 = 8.0f * ahrs->q2;
    q0q0 = ahrs->q0 * ahrs->q0;  q1q1 = ahrs->q1 * ahrs->q1;
    q2q2 = ahrs->q2 * ahrs->q2;  q3q3 = ahrs->q3 * ahrs->q3;

    // 梯度下降（重力方向误差函数的梯度）
    s0 = _4q0 * q2q2 + _2q2 * ax + _4q0 * q1q1 - _2q1 * ay;
    s1 = _4q1 * q3q3 - _2q3 * ax + 4.0f * q0q0 * ahrs->q1 - _2q0 * ay - _4q1 + _8q1 * q1q1 + _8q1 * q2q2 + _4q1 * az;
    s2 = 4.0f * q0q0 * ahrs->q2 + _2q0 * ax + _4q2 * q3q3 - _2q3 * ay - _4q2 + _8q2 * q1q1 + _8q2 * q2q2 + _4q2 * az;
    s3 = 4.0f * q1q1 * ahrs->q3 - _2q1 * ax + 4.0f * q2q2 * ahrs->q3 - _2q2 * ay;

    // 归一化梯度
    recipNorm = sqrt(s0*s0 + s1*s1 + s2*s2 + s3*s3);
    s0 /= recipNorm;  s1 /= recipNorm;  s2 /= recipNorm;  s3 /= recipNorm;

    // 四元数导数（陀螺仪测量 + 梯度修正）
    qDot1 = 0.5f * (-ahrs->q1 * gx - ahrs->q2 * gy - ahrs->q3 * gz) - ahrs->beta * s0;
    qDot2 = 0.5f * ( ahrs->q0 * gx + ahrs->q2 * gz - ahrs->q3 * gy) - ahrs->beta * s1;
    qDot3 = 0.5f * ( ahrs->q0 * gy - ahrs->q1 * gz + ahrs->q3 * gx) - ahrs->beta * s2;
    qDot4 = 0.5f * ( ahrs->q0 * gz + ahrs->q1 * gy - ahrs->q2 * gx) - ahrs->beta * s3;

    // 积分
    ahrs->q0 += qDot1 * ahrs->dt;
    ahrs->q1 += qDot2 * ahrs->dt;
    ahrs->q2 += qDot3 * ahrs->dt;
    ahrs->q3 += qDot4 * ahrs->dt;

    // 归一化
    recipNorm = sqrt(ahrs->q0*ahrs->q0 + ahrs->q1*ahrs->q1 + 
                     ahrs->q2*ahrs->q2 + ahrs->q3*ahrs->q3);
    ahrs->q0 /= recipNorm;  ahrs->q1 /= recipNorm;
    ahrs->q2 /= recipNorm;  ahrs->q3 /= recipNorm;
}
```

### 5.3 beta参数调节

| beta值 | 效果 |
|--------|------|
| 0.01 ~ 0.05 | 高精度、慢收敛 |
| 0.1（推荐） | 平衡 |
| 0.3 ~ 1.0 | 快收敛、可能有噪声 |

## 七、实战案例：MPU6050姿态融合

### 7.1 硬件连接

```
MPU6050    STM32
  VCC  ──→ 3.3V
  GND  ──→ GND
  SCL  ──→ PB6 (I2C1_SCL)
  SDA  ──→ PB7 (I2C1_SDA)
  INT  ──→ PA0 (外部中断)
```

### 7.2 初始化代码

```c
#include "mpu6050.h"

void MPU6050_Init(void) {
    I2C_Write(MPU6050_ADDR, 0x6B, 0x01);  // 退出睡眠，使用X轴陀螺仪时钟
    I2C_Write(MPU6050_ADDR, 0x1B, 0x18);  // 陀螺仪量程 ±2000°/s
    I2C_Write(MPU6050_ADDR, 0x1C, 0x00);  // 加速度计量程 ±2g
    I2C_Write(MPU6050_ADDR, 0x1A, 0x03);  // DLPF 42Hz
    I2C_Write(MPU6050_ADDR, 0x19, 0x04);  // 采样率 200Hz
}

void MPU6050_ReadData(int16_t *ax, int16_t *ay, int16_t *az,
                       int16_t *gx, int16_t *gy, int16_t *gz) {
    uint8_t buf[14];
    I2C_ReadBytes(MPU6050_ADDR, 0x3B, buf, 14);
    *ax = (buf[0] << 8) | buf[1];
    *ay = (buf[2] << 8) | buf[3];
    *az = (buf[4] << 8) | buf[5];
    *gx = (buf[8] << 8) | buf[9];
    *gy = (buf[10] << 8) | buf[11];
    *gz = (buf[12] << 8) | buf[13];
}
```

### 7.3 主循环融合

```c
int main(void) {
    MPU6050_Init();
    
    // 选择融合算法（三选一）
    ComplementaryFilter_t cf;
    Complementary_Init(&cf, 0.98, 0.005);
    
    // 或者使用Mahony
    // mahony.Kp = 10;  mahony.Ki = 0.005;
    
    while (1) {
        int16_t ax, ay, az, gx, gy, gz;
        MPU6050_ReadData(&ax, &ay, &az, &gx, &gy, &gz);
        
        // 转换为物理单位
        float acc_angle = atan2(ax, az) * 57.2958f;  // 度
        float gyro_rate = gy / 16.4f;  // °/s（±2000°/s量程）
        
        // 互补滤波
        float angle = Complementary_Update(&cf, acc_angle, gyro_rate);
        
        // 或Mahony
        // Mahony_UpdateIMU(&mahony, gx/131, gy/131, gz/131, ax, ay, az);
        // QuaternionToEuler(mahony.q0, mahony.q1, mahony.q2, mahony.q3, &roll, &pitch, &yaw);
        
        printf("Angle: %.2f\n", angle);
        HAL_Delay(5);
    }
}
```

## 八、调试技巧

1. **先验证传感器数据**：确保原始数据正确
2. **分步调试**：先单独测试陀螺仪积分，再加融合
3. **对比分析**：同时输出原始值和融合值对比
4. **静止测试**：静止时角度应稳定不漂移
5. **动态测试**：快速运动时应跟踪准确
