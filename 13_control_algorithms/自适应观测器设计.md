# 自适应观测器设计

> 电赛控制算法库 | 参数自适应 + 状态估计 + 鲁棒性设计

---

## 一、自适应观测器概述

### 1.1 核心思想

自适应观测器 = **状态观测器** + **参数在线估计**

当系统模型参数未知或时变时，传统观测器（Luenberger/Kalman）性能下降。自适应观测器同时估计**系统状态**和**未知参数**。

### 1.2 适用场景

| 场景 | 说明 |
|------|------|
| 电机参数变化 | 电阻随温度变化、电感饱和 |
| 负载扰动未知 | 负载转矩、摩擦力变化 |
| 传感器受限 | 部分状态不可直接测量 |
| 模型不确定性 | 系统建模误差较大 |

## 二、标准自适应观测器

### 2.1 系统模型

```
ẋ = Ax + Bu
y = Cx
```

其中 A = A₀ + ΔA(θ)，θ 为未知参数向量。

**参数化表示**：
```
ẋ = A₀x + Bu + Φ(x,u)θ
y = Cx
```
Φ(x,u) 为已知的回归矩阵。

### 2.2 自适应观测器结构

```
状态估计器：
  x̂̇ = A₀x̂ + Bu + Φ(x,u)θ̂ + L(y - Ĉx̂)

参数自适应律：
  θ̂̇ = ΓΦᵀ(x,u)CᵀP(y - Ĉx̂)
```

其中：
- L：观测器增益矩阵
- θ̂：参数估计值
- Γ > 0：自适应增益矩阵
- P > 0：Lyapunov方程解

### 2.3 Lyapunov稳定性分析

```
V = eᵀPe + (θ-θ̂)ᵀΓ⁻¹(θ-θ̂)

其中 e = x - x̂（状态估计误差）

要求：
  A₀ - LC 的特征值均在左半平面
  P(A₀-LC) + (A₀-LC)ᵀP = -Q, Q > 0
```

## 三、基于滑模的鲁棒自适应观测器

### 3.1 设计动机

标准自适应观测器对有界扰动鲁棒性不足。引入滑模项增强鲁棒性。

### 3.2 观测器结构

```
x̂̇ = A₀x̂ + Bu + Φ(x,u)θ̂ + L(y - Ĉx̂) + ρ·sgn(y - Ĉx̂)
```

其中 ρ 为滑模增益，需大于扰动上界。

### 3.3 参数自适应律（σ修正）

```
θ̂̇ = ΓΦᵀ(x,u)CᵀP(y - Ĉx̂) - σΓθ̂
```

σ修正项防止参数漂移。

### 3.4 边界层实现（消除抖振）

```
用 sat(s/ε) 替代 sgn(s)

sat(s/ε) = {
  s/ε,        |s| ≤ ε
  sgn(s),     |s| > ε
}
```

## 四、扩展自适应观测器（AEKF）

### 4.1 扩展状态向量

将参数θ增广为状态：

```
z = [x; θ]

ż = [A₀z₁ + Bu + Φ(z₁,u)z₂; 0] = f(z,u)
y  = Cz₁ = h(z)
```

### 4.2 EKF迭代

```
预测：
  ẑ⁻ₖ = f(ẑₖ₋₁, uₖ₋₁)
  P⁻ₖ  = FₖPₖ₋₁Fₖᵀ + Q

更新：
  Kₖ   = P⁻ₖHₖᵀ(HₖP⁻ₖHₖᵀ + R)⁻¹
  ẑₖ   = ẑ⁻ₖ + Kₖ(yₖ - h(ẑ⁻ₖ))
  Pₖ   = (I - KₖHₖ)P⁻ₖ
```

其中 F = ∂f/∂z，H = ∂h/∂z 为雅可比矩阵。

## 五、电赛典型应用

### 5.1 电机转速与负载观测

```
系统模型：
  J·ω̇ = Kt·i - B·ω - Tl
  Tl̇  = 0（负载缓变假设）

增广状态：x = [ω, Tl]ᵀ
观测器：
  ω̂̇  = (Kt·i - B̂·ω̂ - T̂l)/Ĵ + L₁(ω_sensor - ω̂)
  T̂l̇ = L₂(ω_sensor - ω̂)
```

### 5.2 电感/电阻在线辨识

```
V = Ri + L·di/dt + e

参数向量：θ = [R, L]ᵀ
回归向量：Φ = [i, di/dt]

自适应律实时更新 R̂, L̂
→ 用于FOC控制器前馈补偿
```

## 六、STM32实现模板

```c
// 自适应观测器结构体
typedef struct {
    float x_hat[4];      // 状态估计
    float theta[2];      // 参数估计
    float L[4];          // 观测器增益
    float gamma;         // 自适应增益
    float sigma;         // σ修正系数
    float rho;           // 滑模增益
    float Ts;            // 采样周期
} AdaptiveObserver;

void AdaptiveObs_Init(AdaptiveObserver *obs) {
    memset(obs->x_hat, 0, sizeof(obs->x_hat));
    obs->theta[0] = 1.0f;  // R初始估计
    obs->theta[1] = 0.001f; // L初始估计
    obs->L[0] = 100.0f; obs->L[1] = 50.0f;
    obs->gamma = 10.0f;
    obs->sigma = 0.01f;
    obs->rho = 0.5f;
}

void AdaptiveObs_Update(AdaptiveObserver *obs, float y, float u) {
    float y_hat = obs->x_hat[0];
    float e_y = y - y_hat;
    
    // 状态更新（Euler法）
    float dx0 = obs->theta[0] * obs->x_hat[0] + obs->theta[1] * u 
                + obs->L[0] * e_y + obs->rho * sgn_f(e_y);
    float dx1 = obs->L[1] * e_y;
    
    obs->x_hat[0] += dx0 * obs->Ts;
    obs->x_hat[1] += dx1 * obs->Ts;
    
    // 参数自适应（σ修正）
    obs->theta[0] += obs->Ts * (obs->gamma * e_y * obs->x_hat[0] 
                                - obs->sigma * obs->theta[0]);
    obs->theta[1] += obs->Ts * (obs->gamma * e_y * u 
                                - obs->sigma * obs->theta[1]);
}

// 符号函数（带边界层）
static inline float sgn_f(float x) {
    if (x > 0.01f) return 1.0f;
    if (x < -0.01f) return -1.0f;
    return x / 0.01f;
}
```

## 七、调试要点

| 参数 | 调试建议 |
|------|---------|
| 观测器增益L | 先调大加快收敛，再减小降低噪声敏感度 |
| 自适应增益Γ | 过大→参数振荡，过小→收敛慢 |
| σ修正系数 | 通常0.001~0.1，防止参数漂移 |
| 滑模增益ρ | 需大于扰动上界，但不宜过大 |
| 采样频率 | 通常为系统带宽的10倍以上 |

### 常见问题

1. **参数收敛慢**：增大Γ或增加持续激励信号
2. **状态估计振荡**：减小L或增大Q权重
3. **参数漂移**：加入σ修正或死区修正
4. **抖振**：用边界层替代符号函数
