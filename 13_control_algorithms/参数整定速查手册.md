# 参数整定速查手册

## 一、PID 参数整定

### 1.1 Ziegler-Nichols 临界比例法（推荐首选）

**步骤：**
1. 令 Ki=0, Kd=0，仅用比例控制
2. 逐渐增大 Kp，直到系统出现**等幅振荡**
3. 记录此时 Kp=Ku（临界增益），振荡周期 Tu（临界周期）
4. 按下表计算参数：

| 控制器类型 | Kp | Ki | Kd |
|-----------|-----|-----|-----|
| 纯比例 P | 0.5·Ku | 0 | 0 |
| PI | 0.45·Ku | 0.45·Ku/(0.83·Tu) | 0 |
| PID经典 | 0.6·Ku | 0.6·Ku/(0.5·Tu) | 0.6·Ku·0.125·Tu |
| PID改进 | 0.33·Ku | 0.66·Ku/Tu | 0.11·Ku·Tu |

### 1.2 Ziegler-Nichols 阶跃响应法

**步骤：**
1. 系统开环，施加阶跃输入
2. 记录响应曲线，提取参数：
   - **K**：稳态增益 = Δ输出/Δ输入
   - **L**：延迟时间（响应开始前的等待）
   - **T**：时间常数（从开始响应到63.2%稳态值）
3. 计算：

| 控制器类型 | Kp | Ti | Td |
|-----------|-----|-----|-----|
| P | T/(K·L) | ∞ | 0 |
| PI | 0.9·T/(K·L) | L/0.3 | 0 |
| PID | 1.2·T/(K·L) | 2L | 0.5L |

### 1.3 手动整定经验法（实战最常用）

**口诀：先P后I再D，先粗后细**

| 步骤 | 操作 | 观察指标 |
|------|------|----------|
| 1. 设Ki=0, Kd=0 | 从小到大增大Kp | 响应加快但未振荡 |
| 2. Kp到临界 | 减小到振荡消失值的60-80% | 有超调但收敛 |
| 3. 加入Ki | 从小到大增大Ki | 消除稳态误差 |
| 4. Ki过大会振荡 | 减小Ki到合适值 | 无静差且稳定 |
| 5. 加入Kd | 从小到大增大Kd | 减小超调、加快响应 |
| 6. Kd过大 | 减小Kd | 噪声不被放大 |

### 1.4 各场景参数范围参考

| 控制对象 | Kp范围 | Ki范围 | Kd范围 | 采样周期 |
|----------|--------|--------|--------|----------|
| 直流电机调速 | 0.5~5 | 0.01~0.5 | 0~0.1 | 1~10ms |
| 平衡车直立 | 10~50 | 0~1 | 0.5~5 | 5~10ms |
| 平衡车速度 | 0.5~3 | 0.01~0.1 | 0 | 10~50ms |
| 舵机位置 | 1~10 | 0~0.5 | 0.1~1 | 10~20ms |
| 温度控制 | 2~20 | 0.01~0.5 | 1~10 | 100~1000ms |
| 电压调节 | 0.1~2 | 0.01~0.5 | 0~0.1 | 0.1~1ms |
| 电流环 | 0.5~5 | 0.1~2 | 0~0.05 | 0.1~1ms |
| 循迹(差速) | 1~10 | 0~0.5 | 0.5~5 | 10~50ms |

### 1.5 常见问题诊断

| 现象 | 原因 | 解决方法 |
|------|------|----------|
| 响应太慢 | Kp太小 | 增大Kp |
| 振荡不收敛 | Kp太大或Ki太大 | 减小Kp和Ki |
| 稳态有静差 | Ki太小或为0 | 增大Ki |
| 超调过大 | Kp过大、Kd太小 | 减小Kp、增大Kd |
| 高频抖动 | Kd过大或采样噪声 | 减小Kd、加滤波 |
| 积分饱和 | 积分项过大 | 加积分限幅或抗饱和 |
| 启动冲击 | 积分累积过大 | 启动时清零积分项 |

## 二、串级PID整定

### 整定顺序：从内环到外环

```
外环(位置/角度) → 内环(速度/角速度)
```

| 环路 | 整定要点 |
|------|----------|
| 内环（速度/角速度） | 先整定，带宽要高，响应要快 |
| 外环（位置/角度） | 后整定，带宽应为内环的1/5~1/10 |

**经验公式：** 外环Kp ≈ 内环带宽 / (2π × 外环期望带宽)

## 三、模糊PID整定

### 模糊规则表（ΔKp / ΔKi / ΔKd）

输入：误差e 和 误差变化率ec，语言变量：NB NM NS ZO PS PM PB

**ΔKp 规则表（7×7）：**

| e╲ec | NB | NM | NS | ZO | PS | PM | PB |
|------|-----|-----|-----|-----|-----|-----|-----|
| NB | PB | PB | PM | PM | PS | ZO | ZO |
| NM | PB | PB | PM | PS | PS | ZO | NS |
| NS | PM | PM | PM | PS | ZO | NS | NS |
| ZO | PM | PM | PS | ZO | NS | NM | NM |
| PS | PS | PS | ZO | NS | NS | NM | NM |
| PM | PS | ZO | NS | NM | NM | NM | NB |
| PB | ZO | ZO | NM | NM | NM | NB | NB |

### 模糊隶属度函数参数

| 语言变量 | 三角形中心(e) | 三角形中心(ec) |
|----------|---------------|----------------|
| NB | -3 | -3 |
| NM | -2 | -2 |
| NS | -1 | -1 |
| ZO | 0 | 0 |
| PS | 1 | 1 |
| PM | 2 | 2 |
| PB | 3 | 3 |

> **量化因子**：Ke = 3/emax, Kec = 3/ecmax
> **比例因子**：Ku_Kp = Kp_max/3, Ku_Ki = Ki_max/3, Ku_Kd = Kd_max/3

## 四、LQR 参数整定

### 4.1 Q、R矩阵选取原则

```
性能指标 J = ∫(x'Qx + u'Ru)dt
```

| 参数 | 增大效果 | 减小效果 |
|------|----------|----------|
| Q（状态权重） | 状态更快收敛、控制量增大 | 允许更大状态偏差、控制量减小 |
| R（控制权重） | 控制量减小、响应变慢 | 控制量增大、响应变快 |

### 4.2 LQR整定步骤

1. 建立状态空间模型：ẋ = Ax + Bu
2. 设 Q = diag(q1, q2, ..., qn), R = diag(r1, r2, ..., rm)
3. 初始设定：qi = 1/(xi_max²), rj = 1/(uj_max²)
4. 求解 Riccati 方程得 K
5. 仿真验证，按需调整 Q、R

### 4.3 典型Q/R参考值（平衡车）

| 状态量 | 典型q值 | 说明 |
|--------|---------|------|
| 角度 θ | 100~1000 | 角度偏差权重最大 |
| 角速度 θ̇ | 1~10 | 角速度权重适中 |
| 位置 x | 1~10 | 位置偏差权重较小 |
| 速度 ẋ | 1~10 | 速度权重适中 |
| R（控制量） | 0.01~1 | 越小控制越激进 |

## 五、前馈补偿参数

### 5.1 速度前馈

```
u_ff = Kff × d(setpoint)/dt
```

- Kff 一般从 0 开始，逐步增大
- 当前馈量恰好补偿系统惯性时，超调最小
- 典型值：Kff = 0.5~2.0

### 5.2 加速度前馈

```
u_ff = Kff1 × d(setpoint)/dt + Kff2 × d²(setpoint)/dt²
```

- Kff1 补偿摩擦/阻力
- Kff2 补偿惯性
- 适用于高速运动控制

## 六、采样周期选择

| 控制对象 | 推荐采样周期 | 原则 |
|----------|-------------|------|
| 电流环 | 50~200μs | 带宽最高 |
| 速度环 | 1~5ms | 带宽中等 |
| 位置环/角度环 | 5~20ms | 带宽较低 |
| 温度控制 | 0.5~5s | 惯性大、慢系统 |
| 电压环(DC-DC) | 10~100μs | 开关频率决定 |

**经验法则：** 采样周期 ≤ 系统时间常数 / 10

## 七、滤波器参数速查

### 一阶低通滤波

```
y[n] = α·x[n] + (1-α)·y[n-1]
α = Ts / (Ts + τ)
```

| 滤波对象 | 截止频率 | τ (时间常数) | α (Ts=10ms) |
|----------|----------|-------------|-------------|
| 陀螺仪 | 20~50Hz | 3~8ms | 0.55~0.77 |
| 加速度计 | 5~20Hz | 8~32ms | 0.24~0.56 |
| 电流采样 | 100~500Hz | 0.3~1.6ms | 0.86~0.97 |
| 编码器速度 | 10~50Hz | 3~16ms | 0.38~0.77 |
| 电压采样 | 50~200Hz | 0.8~3ms | 0.77~0.93 |

### 滑动平均滤波

```
y[n] = (x[n] + x[n-1] + ... + x[n-N+1]) / N
```

| 滤波对象 | 窗口大小N | 延迟 |
|----------|-----------|------|
| 陀螺仪 | 4~8 | 较小 |
| 超声波 | 5~10 | 中等 |
| 温度 | 10~20 | 较大 |
