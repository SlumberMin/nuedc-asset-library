# 最优控制实战指南

> 适用场景：电赛中需要系统性能最优（能量最小、时间最短、误差最小）的控制问题

## 一、最优控制核心思想

```
给定系统 x̂ = Ax + Bu，找控制律 u(t) 使性能指标 J 最小化
J = ∫[xᵀQx + uᵀRu]dt + xᵀ(t_f)Sx(t_f)
     状态代价     控制代价      终端代价
```

**核心权衡**：Q大→状态收敛快但控制量大；R大→控制量小但收敛慢

---

## 二、LQR（线性二次调节器）

### 2.1 连续LQR

**问题**：x̂ = Ax + Bu，最小化 J = ∫(xᵀQx + uᵀRu)dt

**解法**：求解代数Riccati方程（ARE）：
```
AᵀP + PA - PBR⁻¹BᵀP + Q = 0
最优增益：K = R⁻¹BᵀP
控制律：u = -Kx
```

### 2.2 离散LQR

**离散系统**：x[k+1] = Ax[k] + Bu[k]

**离散Riccati方程**：
```
P = AᵀPA - AᵀPB(R + BᵀPB)⁻¹BᵀPA + Q
K = (R + BᵀPB)⁻¹BᵀPA
```

### 2.3 LQR设计流程（电赛实战）

```
步骤1：建立状态空间模型 x̂ = Ax + Bu
步骤2：选择Q、R矩阵
        ┌ 常用：Q = diag(q1, q2, ..., qn), R = diag(r1, r2, ..., rm)
        ├ Q对角元素越大→对应状态越快收敛
        └ R对角元素越大→对应控制量越小
步骤3：求解Riccati方程得P → 计算K = R⁻¹BᵀP
步骤4：仿真验证闭环响应
步骤5：加入状态估计器（若无法全状态测量）
```

### 2.4 Q/R调参口诀

| 目标 | 调整策略 |
|------|---------|
| 响应更快 | 增大Q对角元素 / 减小R |
| 控制量更平滑 | 减小Q / 增大R |
| 某个状态优先 | 增大该状态对应的qᵢ |
| 执行器饱和 | 增大对应rⱼ |

### 2.5 STM32 LQR代码模板

```c
// LQR增益（离线计算好）
float K[2] = {-10.0f, -3.16f};  // K = [k1, k2]

// 控制循环
float LQR_Update(float x1, float x2) {
    // x1=位置误差, x2=速度误差
    float u = -(K[0]*x1 + K[1]*x2);
    // 限幅
    if(u > U_MAX) u = U_MAX;
    if(u < -U_MAX) u = -U_MAX;
    return u;
}
```

## 三、LQG（线性二次高斯）

### 3.1 LQG = LQR + Kalman滤波器

```
当状态不可直接测量时：
  1. 用Kalman滤波器估计状态 x̂
  2. 用LQR对估计状态施加控制 u = -Kx̂

分离定理：控制器设计和观测器设计可独立进行
```

### 3.2 Kalman滤波器回顾

```
预测：x̂⁻[k] = Ax̂[k-1] + Bu[k-1]
       P⁻[k] = AP[k-1]Aᵀ + Q_n

更新：K_kal = P⁻Hᵀ(HP⁻Hᵀ + R_n)⁻¹
       x̂[k] = x̂⁻[k] + K_kal(z[k] - Hx̂⁻[k])
       P[k] = (I - K_kalH)P⁻[k]
```

### 3.3 LQG完整架构

```
         ┌──────────┐    u = -Kx̂    ┌──────────┐
参考 r ──→│ LQR控制器 │──────────────→│  被控对象  │──→ 输出 y
         └──────────┘                └──────────┘
              ↑                           │
              │ x̂ (状态估计)               │ y (测量)
         ┌──────────┐                     │
         │ Kalman滤波 │←───────────────────┘
         └──────────┘
```

### 3.4 电赛典型应用：倒立摆LQG

```c
// 状态：x = [θ, θ̇, x, ẋ]ᵀ
// 观测：z = [θ_encoder, x_encoder]ᵀ (只测角度和位移)

// Kalman滤波器估计4个状态
void Kalman_Update(float z_angle, float z_pos) {
    // 预测
    mat_mul(A, x_hat, x_pred, 4, 4, 1);
    mat_add(x_pred, Bu, x_pred, 4, 1);
    // 更新（简化）
    float innovation[2] = {z_angle - x_pred[0], z_pos - x_pred[2]};
    for(int i=0; i<4; i++)
        x_hat[i] = x_pred[i] + K_kal[i][0]*innovation[0] + K_kal[i][1]*innovation[1];
}

// LQR控制
float LQG_Control(void) {
    Kalman_Update(read_angle(), read_position());
    float u = 0;
    for(int i=0; i<4; i++) u -= K_lqr[i] * x_hat[i];
    return saturate(u, -PWM_MAX, PWM_MAX);
}
```

## 四、H₂最优控制

### 4.1 H₂性能指标

```
H₂范数：||T(s)||₂ = √(1/2π ∫|T(jω)|²dω)

物理意义：白噪声输入下输出的均方根值（RMS能量）
目标：最小化噪声到输出的传递函数H₂范数
```

### 4.2 H₂ vs LQR

| 对比 | LQR | H₂ |
|------|-----|-----|
| 噪声模型 | 确定性 | 随机（白噪声） |
| 性能指标 | 状态+控制加权积分 | 输出能量 |
| 设计自由度 | Q, R选择 | 加权函数选择 |
| 鲁棒性 | 无显式保证 | 频域整形 |

### 4.3 H₂加权函数设计

```
灵敏度函数 S(s) = 1/(1+P(s)C(s))
补灵敏度 T(s) = P(s)C(s)/(1+P(s)C(s))

设计目标：
  min ||W₁S||₂² + ||W₂T||₂²

W₁(s)：性能加权（低频大→跟踪好）
W₂(s)：鲁棒加权（高频大→抗噪声）
```

## 五、H∞最优控制

### 5.1 H∞核心思想

```
H∞范数：||T(s)||∞ = max_ω |T(jω)|

含义：最坏情况下的增益峰值
目标：保证在所有频率上性能不超过某个上界γ

min γ, 使得 ||T(s)||∞ < γ
```

### 5.2 H∞ vs H₂

| 对比 | H₂ | H∞ |
|------|-----|-----|
| 优化目标 | 平均性能 | 最坏情况性能 |
| 鲁棒性 | 较弱 | 强 |
| 对不确定性 | 敏感 | 鲁棒 |
| 计算复杂度 | 较低 | 较高 |
| 适用场景 | 噪声抑制 | 模型不确定性大 |

### 5.3 混合灵敏度H∞设计

```
广义被控对象 P(s)：
        ┌         ┐
        │ W₁      -W₁G │
P(s) = │ 0        W₂G │
        │ I       -G   │
        └         ┘

求控制器K(s)使闭环||T_zw||∞ < γ
```

### 5.4 MATLAB H∞设计示例

```matlab
% 被控对象
s = tf('s');
G = 10 / (s^2 + 2*s + 10);

% 加权函数
W1 = (s/1.5 + 1) / (s/0.01 + 1);   % 低频高增益→好的跟踪
W2 = (s/10) / (s/100 + 1);          % 高频高增益→鲁棒性

% H∞综合
[K, CL, gamma] = hinfsyn(sysic(W1, W2, G), 1, 1);
% gamma即为最优H∞范数上界
```

### 5.5 电赛中的简化H∞

```
电赛中完整H∞设计计算量大，简化方案：
1. 先用PID设计基本控制器
2. 加入频率整形：低频高增益PI + 高频低通滤波
3. 增益裕度 > 6dB，相位裕度 > 45°
4. 验证在参数变化±20%时性能满足要求
```

## 六、参数整定方法汇总

### 6.1 LQR参数整定

```
方法1：Bryson法则
  Q = diag(1/x₁_max², 1/x₂_max², ...)
  R = diag(1/u₁_max², 1/u₂_max², ...)
  → 状态和控制量归一化后再调

方法2：主导极点法
  先用PID确定期望极点 → 反推Q, R使LQR极点接近

方法3：迭代试凑
  从Q=I, R=ρI开始，逐步调整ρ（控制代价权重）
  ρ大→保守（慢）  ρ小→激进（快但可能饱和）
```

### 6.2 参数整定速查表

| 参数 | 增大效果 | 减小效果 |
|------|---------|---------|
| Q[i][i] | 状态i收敛更快 | 状态i收敛更慢 |
| R[j][j] | 控制量j更小更平滑 | 控制量j更大更激进 |
| Q_n (过程噪声) | Kalman增益大→更信任测量 | 更信任模型预测 |
| R_n (测量噪声) | Kalman增益小→更信任模型 | 更信任测量 |

### 6.3 常见问题诊断

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 控制量剧烈震荡 | Q太大/R太小 | 减小Q或增大R |
| 响应太慢 | Q太小/R太大 | 增大Q或减小R |
| 稳态误差 | 模型有偏差 | 加积分项或修正模型 |
| 状态估计不准 | Q_n/R_n不匹配 | 重新标定噪声参数 |
| 仿真好实际差 | 模型精度不够 | 用系统辨识提高模型精度 |

## 七、电赛最优控制选型决策树

```
需要最优控制？
├── 状态全可测？
│   ├── 是 → LQR（最简单高效）
│   └── 否 → LQG（LQR + Kalman）
├── 模型不确定性强？
│   ├── 是 → H∞（鲁棒性最强）
│   └── 否 → LQR/LQG
├── 噪声为主要问题？
│   ├── 是 → H₂（最优噪声抑制）
│   └── 否 → LQR
└── 计算资源有限？
    ├── 是 → LQR（离线算好K，在线只做矩阵乘法）
    └── 否 → 可考虑H∞
```

## 八、实战技巧

### 8.1 从PID平滑过渡到LQR

```
已调好PID：Kp=50, Ki=10, Kd=5
等价LQR参考：K ≈ [50, 5]（PD项）
再微调Q/R在此基础上优化
```

### 8.2 参考输入跟踪

```
LQR调节器（regulator）默认将状态镇定到零
跟踪非零参考：
  1. 前馈：u_ff = (C(A-BK)⁻¹B)⁻¹ · r
  2. 积分器扩展：加入误差积分状态
  3. 伺服补偿器：内模原理
```

### 8.3 抗积分饱和

```c

float u_lqr = LQR_Update(x1, x2);
float u_total = u_lqr + u_integral;

if(fabs(u_total) > U_MAX) {

} else {
    u_integral += Ki * error * dt;
}
```
