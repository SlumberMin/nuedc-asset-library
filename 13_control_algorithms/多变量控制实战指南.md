# 多变量控制实战指南

## 一、多变量系统概述

### 1.1 什么是多变量系统

多输入多输出（MIMO）系统：多个输入变量与多个输出变量之间存在**耦合**关系。

```
        ┌──────────┐
u₁ ──→│          │──→ y₁
        │  被控对象  │
u₂ ──→│  G(s)    │──→ y₂
        └──────────┘

G(s) = ┌ g₁₁(s)  g₁₂(s) ┐
       └ g₂₁(s)  g₂₂(s) ┘
```

**耦合**意味着改变u₁不仅影响y₁，还会影响y₂。

### 1.2 电赛中的多变量场景

| 场景 | 输入 | 输出 | 耦合表现 |
|------|------|------|---------|
| 双电机同步 | 电压1, 电压2 | 转速1, 转速2 | 机械耦合/负载耦合 |
| 倒立摆 | 小车力, 摆杆力 | 位置, 角度 | 动力学耦合 |
| 温控系统 | 加热功率, 风扇转速 | 温度, 湿度 | 热湿耦合 |
| 两轮平衡车 | 左轮PWM, 右轮PWM | 前进速度, 转向角 | 运动学耦合 |
| 电源并联 | 电流1, 电流2 | 总电压, 均流 | 电气耦合 |

## 二、耦合分析与解耦

### 2.1 相对增益矩阵（RGA）

RGA用于判断变量间的耦合程度，指导输入输出配对。

#### RGA计算

```
Λ = G ⊗ (G⁻¹)ᵀ
```

其中 ⊗ 表示逐元素乘积（Hadamard积）。

#### 2×2系统RGA计算

```
G = ┌ g₁₁  g₁₂ ┐
    └ g₂₁  g₂₂ ┘

det(G) = g₁₁·g₂₂ - g₁₂·g₂₁

RGA:
λ₁₁ = g₁₁·g₂₂ / det(G)
λ₁₂ = 1 - λ₁₁
λ₂₁ = 1 - λ₁₁
λ₂₂ = λ₁₁
```

#### RGA解读规则

```
λᵢⱼ ≈ 1  → uⱼ 与 yᵢ 强关联，适合配对
λᵢⱼ ≈ 0  → uⱼ 与 yᵢ 弱关联，避免配对
0.3 < λᵢⱼ < 0.7  → 耦合严重，需要解耦
λᵢⱼ < 0  → 配对后系统不稳定，必须避免
```

### 2.2 静态解耦

在稳态工作点用解耦矩阵消除稳态耦合。

#### 解耦原理

```
欲使 u → y 的传递为对角阵：

G·D = diag

其中D为解耦矩阵，u = D·v（v为解耦后的控制量）

对2×2系统：
D = G⁻¹ · diag = ┌ d₁₁  d₁₂ ┐
                  └ d₂₁  d₂₂ ┘
```

#### 静态解耦实现

```c
// 静态解耦 - 2×2系统
// 输入: v1, v2 (解耦后控制量)
// 输出: u1, u2 (实际输入)

// 解耦矩阵 (稳态增益的逆)
// D = inv(G_static)
float d11 =  1.2, d12 = -0.3;
float d21 = -0.4, d22 =  1.1;

float u1 = d11 * v1 + d12 * v2;
float u2 = d21 * v1 + d22 * v2;
```

### 2.3 动态解耦

在全频率范围内消除耦合，比静态解耦效果更好。

#### 对角矩阵解耦（理想解耦）

对于2×2系统：

```
D(s) = G(s)⁻¹ · diag{g₁₁(s), g₂₂(s)}

简化后：
D₁₁ = g₁₁ / (g₁₁·g₂₂ - g₁₂·g₂₁) · g₁₁
D₁₂ = -g₁₂ / (g₁₁·g₂₂ - g₁₂·g₂₁) · g₂₂
D₂₁ = -g₂₁ / (g₁₁·g₂₂ - g₁₂·g₂₁) · g₁₁
D₂₂ = g₂₂ / (g₁₁·g₂₂ - g₁₂·g₂₁) · g₂₂
```

#### 简化解耦（Bristol方法）

忽略动态耦合项，只做静态解耦+动态补偿：

```
D = ┌ 1           -g₁₂/g₁₁ · (g₂₂/g₁₂的静态值) ┐
    └ -g₂₁/g₂₂    1                               ┘
```

### 2.4 前馈解耦

利用前馈补偿消除通道间干扰：

```
         ┌────┐
u₁ ──→│ G₁₁│──→ y₁
   │    └────┘      ↑
   │                │
   │  ┌────┐        │ 耦合干扰
   └─→│FF₁₂│────────┘
      └────┘

FF₁₂ = -G₂₁/G₂₂  (前馈补偿器)
```

#### 嵌入式前馈解耦实现

```c
// 前馈解耦控制器
typedef struct {
    float Kff12;   // 前馈增益 u1→y2 补偿
    float Kff21;   // 前馈增益 u2→y1 补偿
    float alpha;   // 滤波系数
    float ff12_filt; // 前馈滤波状态
    float ff21_filt;
} FF_Decoupler;

void FF_Decoupler_Init(FF_Decoupler* d, float Kff12, float Kff21, float alpha) {
    d->Kff12 = Kff12;
    d->Kff21 = Kff21;
    d->alpha = alpha;
    d->ff12_filt = 0;
    d->ff21_filt = 0;
}

void FF_Decoupler_Calc(FF_Decoupler* d, float u1, float u2, 
                        float* u1_out, float* u2_out) {
    // 前馈补偿 + 低通滤波
    float ff12 = d->Kff12 * u1;
    float ff21 = d->Kff21 * u2;
    
    d->ff12_filt = d->alpha * d->ff12_filt + (1 - d->alpha) * ff12;
    d->ff21_filt = d->alpha * d->ff21_filt + (1 - d->alpha) * ff21;
    
    // 解耦输出
    *u1_out = u1 + d->ff21_filt;  // 补偿u2对y1的影响
    *u2_out = u2 + d->ff12_filt;  // 补偿u1对y2的影响
}
```

## 三、状态空间控制

### 3.1 状态空间模型

```
ẋ = Ax + Bu
y = Cx + Du
```

离散化（零阶保持）：

```
x(k+1) = Gx(k) + Hu(k)
y(k)   = Cx(k) + Du(k)
```

其中：
```
G = e^(A·Ts)        (状态转移矩阵)
H = (∫₀^Ts e^(Aτ)dτ) · B  (输入矩阵)
```

### 3.2 能控性与能观性

#### 能控性矩阵

```
Qc = [B  AB  A²B  ...  Aⁿ⁻¹B]

rank(Qc) = n → 完全能控
```

#### 能观性矩阵

```
Qo = [C; CA; CA²; ...; CAⁿ⁻¹]

rank(Qo) = n → 完全能观
```

### 3.3 极点配置（状态反馈）

给定期望极点位置 p₁, p₂, ..., pₙ，求状态反馈增益K：

```
u = -Kx + r    (状态反馈+参考输入)

闭环系统: ẋ = (A-BK)x + Br
```

#### MATLAB求解

```matlab
% 系统矩阵
A = [0 1; -2 -3];
B = [0; 1];
C = [1 0];

% 期望极点
poles = [-5, -8];

% 极点配置
K = place(A, B, poles);

% 验证
eig(A - B*K)  % 应等于期望极点
```

#### Ackermann公式（手动计算）

```
K = [0 0 ... 0 1] · Qc⁻¹ · αd(A)
```

其中 αd(A) = 期望特征多项式代入A。

### 3.4 状态观测器

当状态不可直接测量时，用观测器估计：

#### 全阶Luenberger观测器

```
x̂(k+1) = Gx̂(k) + Hu(k) + L[y(k) - Cx̂(k)]
```

- **L** = 观测器增益
- **x̂** = 状态估计值

```
观测器极点 = eigenvalues(G - LC)

L = acker(G', C', desired_poles)'  % MATLAB
```

#### 降阶观测器

当部分状态可直接测量时，只估计不可测状态，减小计算量。

```
对于n维系统，p个输出可测：
降阶观测器维数 = n - p
```

### 3.5 LQR最优控制

线性二次型调节器，自动计算最优状态反馈增益：

```
min J = ∫(xᵀQx + uᵀRu)dt

最优控制: u = -Kx
K = R⁻¹BᵀP
```

其中P是代数Riccati方程的解：

```
AᵀP + PA - PBR⁻¹BᵀP + Q = 0
```

#### MATLAB实现

```matlab
A = [0 1; -2 -3];
B = [0; 1];
Q = diag([100, 1]);  % 状态权重
R = 0.1;             % 控制权重

[K, P, e] = lqr(A, B, Q, R);
% K = 最优增益
% P = Riccati解
% e = 闭环极点
```

#### Python实现

```python
import numpy as np
from scipy.linalg import solve_continuous_are

def lqr(A, B, Q, R):
    """求解LQR最优增益"""
    P = solve_continuous_are(A, B, Q, R)
    K = np.linalg.inv(R) @ B.T @ P
    return K

# 示例
A = np.array([[0, 1], [-2, -3]])
B = np.array([[0], [1]])
Q = np.diag([100, 1])
R = np.array([[0.1]])

K = lqr(A, B, Q, R)
print(f"LQR增益: K = {K}")
print(f"闭环极点: {np.linalg.eig(A - B @ K)[0]}")
```

## 四、内模控制（IMC）

### 4.1 基本原理

内模控制的核心思想：**如果控制器内部包含被控对象的模型，则可以精确消除稳态误差**。

```
           ┌─────┐
r ──→(+)──→│ Q(s)│──→ u ──→┌─────┐──→ y
      ↑    └─────┘         │ G(s)│
      │                    └─────┘
      │         ┌─────┐      │
      └─────────│ Gm(s)│←────┘
                └─────┘
         内模（对象模型）

等效反馈控制器：
C(s) = Q(s) / (1 - Q(s)·Gm(s))
```

### 4.2 IMC设计步骤

```
Step 1: 获取对象模型 Gm(s)
Step 2: 分解 Gm(s) = Gm+(s) · Gm-(s)
        Gm+(s) = 最小相位部分（含所有右半平面零点和时滞）
        Gm-(s) = 最小相位部分（稳定、可逆）
Step 3: 设计 Q(s) = Gm-⁻¹(s) · F(s)
        F(s) = 低通滤波器（保证Q(s)物理可实现）
Step 4: 计算等效控制器 C(s)
```

### 4.3 IMC滤波器设计

```
F(s) = 1 / (λs + 1)ⁿ

λ = 滤波器时间常数（可调参数）
n = 滤波器阶数（保证Q(s)可实现）
```

- λ小→响应快，鲁棒性差
- λ大→响应慢，鲁棒性好

### 4.4 IMC-PID等价

IMC可以转化为PID参数，这是最实用的PID整定方法之一：

#### 一阶+时滞对象

```
Gm(s) = K·e^(-θs) / (τs+1)

IMC-PID参数：
Kp = (2τ + θ) / (2K(λ + θ))
Ti = τ + θ/2
Td = τθ / (2τ + θ)
```

#### 二阶+时滞对象

```
Gm(s) = K·e^(-θs) / (τ₁s+1)(τ₂s+1)

IMC-PID参数：
Kp = (τ₁ + τ₂) / (K(λ + θ))
Ti = τ₁ + τ₂
Td = τ₁τ₂ / (τ₁ + τ₂)
```

### 4.5 IMC实现示例

```c
// IMC控制器实现（离散化）
typedef struct {
    // 内模参数
    float K;      // 模型增益
    float tau;    // 时间常数
    float lambda; // 滤波器参数
    float Ts;     // 采样周期
    
    // 状态
    float ym;     // 模型输出
    float u_prev; // 上一步控制量
    float e_prev; // 上一步误差
} IMC_Controller;

void IMC_Init(IMC_Controller* imc, float K, float tau, float lambda, float Ts) {
    imc->K = K;
    imc->tau = tau;
    imc->lambda = lambda;
    imc->Ts = Ts;
    imc->ym = 0;
    imc->u_prev = 0;
    imc->e_prev = 0;
}

float IMC_Update(IMC_Controller* imc, float y_meas, float r) {
    // 内模输出（模型预测）
    float a_m = exp(-imc->Ts / imc->tau);
    imc->ym = a_m * imc->ym + (1 - a_m) * imc->K * imc->u_prev;
    
    // 模型误差（反馈滤波）
    float model_err = y_meas - imc->ym;
    
    // 滤波器（一阶低通）
    float a_f = exp(-imc->Ts / imc->lambda);
    float filtered_err = a_f * imc->e_prev + (1 - a_f) * model_err;
    imc->e_prev = filtered_err;
    
    // 控制量 = 模型逆 + 滤波后的模型误差补偿
    float u_model_inv = r / imc->K;  // 简化的模型逆
    float u = u_model_inv + filtered_err / imc->K;
    
    imc->u_prev = u;
    return u;
}

// IMC-PID等价实现
typedef struct {
    float Kp, Ki, Kd;
    float integral;
    float prev_error;
    float Ts;
    float out_min, out_max;
} IMC_PID;

void IMC_PID_Init(IMC_PID* pid, float K, float tau, float theta, 
                   float lambda, float Ts) {
    // IMC-PID参数整定
    pid->Kp = (2*tau + theta) / (2*K*(lambda + theta));
    pid->Ki = pid->Kp / (tau + theta/2);
    pid->Kd = pid->Kp * (tau*theta) / (2*tau + theta);
    
    pid->Ts = Ts;
    pid->integral = 0;
    pid->prev_error = 0;
    pid->out_min = -100;
    pid->out_max = 100;
}

float IMC_PID_Update(IMC_PID* pid, float error) {
    pid->integral += error * pid->Ts;
    float derivative = (error - pid->prev_error) / pid->Ts;
    pid->prev_error = error;
    
    float output = pid->Kp * (error + pid->Ki * pid->integral + pid->Kd * derivative);
    
    // 限幅
    if (output > pid->out_max) output = pid->out_max;
    if (output < pid->out_min) output = pid->out_min;
    
    return output;
}
```

### 5.1 多变量控制设计流程

```
┌─────────────────────────────────────────────────┐
│              多变量控制设计流程                    │
├─────────────────────────────────────────────────┤
│                                                 │
│  Step 1: 系统建模                                │
│  ├─ 辨识传递函数矩阵 G(s)                        │
│  └─ 或建立状态空间模型 (A,B,C,D)                 │
│                                                 │
│  Step 2: 耦合分析                                │
│  ├─ 计算RGA矩阵                                 │
│  └─ 确定输入输出配对                              │
│                                                 │
│  Step 3: 解耦设计                                │
│  ├─ 若耦合弱：单独设计各回路PID + 前馈            │
│  ├─ 若耦合中：静态解耦 + 各回路PID                │
│  └─ 若耦合强：动态解耦 或 状态空间方法             │
│                                                 │
│  Step 4: 各通道控制器设计                         │
│  ├─ PID/IMC-PID（简单系统）                      │
│  ├─ 状态反馈+观测器（状态空间）                    │
│  └─ 预测控制DMC/GPC（复杂系统）                   │
│                                                 │
│  Step 5: 仿真验证                                │
│  ├─ 单通道响应                                   │
│  ├─ 交叉耦合测试                                 │
│  └─ 鲁棒性测试                                   │
│                                                 │
│  Step 6: 实物调试                                │
│  └─ 参数微调                                     │
└─────────────────────────────────────────────────┘
```

### 5.2 双电机同步控制案例

#### 问题描述

两台直流电机通过机械轴耦合，要求：
- 两电机转速同步
- 负载突变时保持同步

#### 系统模型

```
G(s) = ┌ G₁₁(s)  G₁₂(s) ┐    G₁₁ = G₂₂ = K/(τs+1)  (主通道)
       └ G₂₁(s)  G₂₂(s) ┘    G₁₂ = G₂₁ = Kc/(τc·s+1) (耦合通道)
```

#### 解耦+PID方案

```c
// 双电机同步控制器
typedef struct {
    PID_Controller pid1;  // 电机1 PID
    PID_Controller pid2;  // 电机2 PID
    float Kff12;          // 前馈增益 1→2
    float Kff21;          // 前馈增益 2→1
    float sync_Kp;        // 同步误差增益
} DualMotor_Controller;

void DualMotor_Init(DualMotor_Controller* ctrl, 
                     float K1, float tau1, float K2, float tau2,
                     float Kc, float sync_Kp) {
    // 初始化各通道PID（IMC整定）
    float lambda1 = tau1 * 0.3;
    float lambda2 = tau2 * 0.3;
    IMC_PID_Init(&ctrl->pid1, K1, tau1, 0, lambda1, 0.01);
    IMC_PID_Init(&ctrl->pid2, K2, tau2, 0, lambda2, 0.01);
    
    // 前馈解耦增益
    ctrl->Kff12 = -Kc / K2;
    ctrl->Kff21 = -Kc / K1;
    
    // 同步控制增益
    ctrl->sync_Kp = sync_Kp;
}

void DualMotor_Update(DualMotor_Controller* ctrl,
                       float speed_ref1, float speed_ref2,
                       float speed_meas1, float speed_meas2,
                       float* u1, float* u2) {
    // 速度误差
    float err1 = speed_ref1 - speed_meas1;
    float err2 = speed_ref2 - speed_meas2;
    
    // 同步误差
    float sync_err = speed_meas1 - speed_meas2;
    
    // PID计算（含同步补偿）
    float u_pid1 = IMC_PID_Update(&ctrl->pid1, err1 - ctrl->sync_Kp * sync_err);
    float u_pid2 = IMC_PID_Update(&ctrl->pid2, err2 + ctrl->sync_Kp * sync_err);
    
    // 前馈解耦
    *u1 = u_pid1 + ctrl->Kff21 * u_pid2;
    *u2 = u_pid2 + ctrl->Kff12 * u_pid1;
}
```

## 六、多变量控制快速选型

```
耦合程度评估（RGA分析）
│
├─ RGA接近单位阵（弱耦合）
│  └─ 各通道独立PID + 前馈补偿
│
├─ RGA偏离但<0.7（中等耦合）
│  └─ 静态解耦 + 各通道PID
│
├─ RGA>0.7 或含时滞（强耦合）
│  ├─ 2×2系统 → 动态解耦 + PID
│  └─ 高阶系统 → 状态空间/预测控制
│
└─ 需最优性能
   └─ LQR/LQG（状态空间）或 多变量MPC
```

### 计算资源评估

| 方法 | 计算量 | 适用平台 |
|------|--------|---------|
| 解耦+PID | 低 | STM32/Arduino |
| 状态反馈+观测器 | 中 | STM32/DSP |
| LQR | 中 | STM32/DSP |
| 多变量MPC | 高 | DSP/ARM Cortex-M4+ |
| IMC-PID | 低 | 任意平台 |
