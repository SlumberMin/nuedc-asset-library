## 5. 选型建议

### 5.1 按题目类型推荐

| 题目类型 | 推荐算法 | 理由 |
|----------|----------|------|
| 电源类（DC-DC） | 增量PID / Fuzzy-PID | 简单可靠，资源占用少 |
| 信号源/放大器 | LQR | 精度高，线性系统最优 |
| 电机控制 | ADRC / 滑模控制 | 抗扰强，鲁棒性好 |
| 温度控制 | Smith预估 + PID | 温度系统有纯滞后 |
| 小车/平衡 | LQR / MPC | 多变量协调控制 |
| 飞控类 | ADRC | 不依赖精确模型 |
| 视觉跟踪 | MPC | 可处理约束和预测 |
| 简单功能验证 | PID | 快速实现，够用就行 |

### 5.2 选型决策流程

```
是否需要处理约束？
├── 是 → MPC
└── 否 → 是否多变量系统？
         ├── 是 → LQR
         └── 否 → 系统模型是否精确？
                  ├── 否 → ADRC（不依赖模型）
                  └── 是 → 是否有纯滞后？
                           ├── 是 → Smith预估+PID
                           └── 否 → 是否需要高鲁棒性？
                                    ├── 是 → 滑模控制
                                    └── 否 → 响应速度优先？
                                             ├── 是 → Fuzzy-PID
                                             └── 否 → 增量PID
```

### 5.3 电赛实战建议

1. **首选 PID 或增量PID**：90% 的题目用 PID 就能解决，调参简单
2. **备选 ADRC**：当 PID 效果不好时，ADRC 不需要精确建模，调参也有规律
3. **慎用 MPC**：计算量大，在 MSPM0 上可能来不及
4. **滑模控制注意抖振**：实际使用时需要做边界层近似
5. **不要为了用算法而用算法**：评委看的是效果，不是算法复杂度

## 6. 附录：快速调参指南

### PID 手动调参（Ziegler-Nichols 改进法）
1. 先只加 P，从小到大，直到系统出现等幅振荡 → 记录 Ku, Tu
2. PID: Kp=0.6Ku, Ki=2Kp/Tu, Kd=Kp*Tu/8
3. 实际取 0.7 倍 Kp，减少超调
4. 微调：超调大→减Kp增Kd；稳态误差大→增Ki

### ADRC 参数简化整定
1. 跟踪微分器：r（跟踪速度），h0（滤波因子）
2. ESO：β01, β02, β03（观测器增益，与采样频率相关）
3. 非线性反馈：β1, β2（PD增益），α1, α2（非线性因子）
4. 简化：先只调 b0（系统增益估计），效果不明显再调其他
