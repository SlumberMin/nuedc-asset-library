# 滑模控制进阶指南

> nuedc-asset-library · 控制算法库  
> 适用场景：电机高精度伺服、逆变器控制、飞行器姿态、机器人关节

---

## 1. 经典滑模回顾

### 1.1 基本原理

```
系统: ẋ = f(x) + g(x)u
切换面: s(x) = c·e + ė    (e = x_d - x)
控制律: u = u_eq + u_sw
  u_eq = -[c·ė + f(x)] / g(x)        (等效控制)
  u_sw = -η·sign(s) / g(x)            (切换控制)
```

### 1.2 存在问题
| 问题 | 表现 | 原因 |
|------|------|------|
| 抖振 | 控制信号高频振荡 | sign(s)不连续 |
| 低阶滑动 | 有限时间到达但精度受限 | 切换面阶次低 |
| 参数敏感 | 性能受模型不确定性影响 | 控制律依赖模型 |

## 2. 积分滑模控制（ISMC）

### 2.1 设计思想
在切换面中引入积分项，消除稳态误差，增强鲁棒性。

### 2.2 切换面设计

```
s = ė + λ·e + γ·∫e·dt

其中：
  λ > 0: 比例增益
  γ > 0: 积分增益
  积分项从t=0开始，系统初始状态在滑模面上
```

### 2.3 完整控制律

```c
// 积分滑模控制器
typedef struct {
    float lambda;      // 比例增益
    float gamma;       // 积分增益
    float eta;         // 切换增益
    float phi;         // 边界层厚度
    float integral_e;  // 误差积分
    float dt;          // 采样周期
} ISMC_t;

float ISMC_Calculate(ISMC_t *ctrl, float e, float de) {
    // 更新积分
    ctrl->integral_e += e * ctrl->dt;
    
    // 切换面
    float s = de + ctrl->lambda * e + ctrl->gamma * ctrl->integral_e;
    
    // 等效控制 (假设g(x)=1)
    float u_eq = -ctrl->lambda * de - ctrl->gamma * e;
    
    // 切换控制（带边界层）
    float u_sw;
    if (fabsf(s) > ctrl->phi) {
        u_sw = -ctrl->eta * (s > 0 ? 1.0f : -1.0f);
    } else {
        u_sw = -ctrl->eta * s / ctrl->phi;
    }
    
    return u_eq + u_sw;
}
```

### 2.4 参数整定
```
λ: 决定滑动模态动态，取 5~20 (取决于系统带宽)
γ: 消除稳态误差，取 λ/10 ~ λ/5
η: 克服不确定性，取不确定性上界 + 余量
φ: 边界层，取 0.01~0.1 (平衡抖振与精度)
```

## 3. 高阶滑模控制（HOSM）

### 3.1 原理
将不连续控制作用于s的高阶导数，实际控制信号连续，消除抖振。

### 3.2 二阶滑模 — Twisting算法

```
系统: ẍ = f(x,t) + u, s = 0为滑模面
控制器:
  u = -r₁·sign(s) - r₂·sign(ṡ)

收敛条件:
  (r₁ + r₂) > K_m
  (r₁ - r₂) > K_M
  其中 K_m ≤ |s̈| ≤ K_M
```

```c
// Twisting二阶滑模
typedef struct {
    float r1;          // 增益1
    float r2;          // 增益2
    float s_prev;      // 上次滑模面值
    float dt;
} TwistingSMC_t;

float TwistingSMC_Calculate(TwistingSMC_t *ctrl, float s) {
    // 估计ṡ (差分近似)
    float ds = (s - ctrl->s_prev) / ctrl->dt;
    ctrl->s_prev = s;
    
    // Twisting控制律
    float u = -ctrl->r1 * sign(s) - ctrl->r2 * sign(ds);
    
    return u;
}
```

### 3.3 二阶滑模 — 超螺旋（Super-Twisting）算法 ⭐

**最常用的二阶滑模，控制信号本身连续，无需ṡ信息。**

```
控制律:
  u = u₁ + u₂
  u̇₁ = -λ·|s|^α · sign(s)
  u₂ = -k·|s|^ρ · sign(s)

典型参数: α = 0.5, ρ = 0.5
简化形式:
  u̇₁ = -λ·sign(s)
  u₂ = -k·|s|^0.5 · sign(s)

收敛条件:
  k > K_m / √(2(λ - K_M))
  λ > K_M
```

```c
// 超螺旋控制器
typedef struct {
    float lambda;      // 积分增益
    float k;           // 比例增益
    float u1;          // 积分项输出
    float dt;
} SuperTwisting_t;

float SuperTwisting_Calculate(SuperTwisting_t *ctrl, float s) {
    float abs_s = fabsf(s);
    float sign_s = (s > 0) ? 1.0f : ((s < 0) ? -1.0f : 0.0f);
    float sqrt_abs_s = sqrtf(abs_s);
    
    // 积分项更新
    ctrl->u1 += (-ctrl->lambda * sign_s) * ctrl->dt;
    
    // 比例项
    float u2 = -ctrl->k * sqrt_abs_s * sign_s;
    
    return ctrl->u1 + u2;
}
```

### 3.4 超螺旋参数整定
```
步骤:
1. 估计系统不确定性的上界 K_m, K_M
2. 选 λ > K_M (建议 λ = 2*K_M)
3. 选 k > K_m / √(2*(λ - K_M)) (建议 k = 1.5*理论值)
4. 仿真验证，逐步增大至满足性能要求
```

## 4. 自适应滑模控制（ASMC）

### 4.1 问题背景
当不确定性上界未知时，固定增益η过大导致抖振，过小导致不收敛。

### 4.2 增益自适应律

```
自适应切换增益:
  η̇ = Γ·(|s| - ε)

其中:
  Γ > 0: 自适应速率
  ε > 0: 目标边界层

当 |s| > ε 时 η 增大，|s| < ε 时 η 减小
```

```c
// 自适应滑模控制器
typedef struct {
    float lambda;       // 切换面斜率
    float eta;          // 自适应切换增益
    float gamma;        // 自适应速率
    float epsilon;      // 目标边界
    float eta_min;      // 增益下限
    float eta_max;      // 增益上限
    float phi;          // 边界层
    float dt;
} AdaptiveSMC_t;

float AdaptiveSMC_Calculate(AdaptiveSMC_t *ctrl, float e, float de) {
    // 切换面
    float s = de + ctrl->lambda * e;
    
    // 等效控制
    float u_eq = -ctrl->lambda * de;
    
    // 切换控制（饱和函数）
    float u_sw;
    if (fabsf(s) > ctrl->phi) {
        u_sw = -ctrl->eta * (s > 0 ? 1.0f : -1.0f);
    } else {
        u_sw = -ctrl->eta * s / ctrl->phi;
    }
    
    // 自适应律
    float abs_s = fabsf(s);
    ctrl->eta += ctrl->gamma * (abs_s - ctrl->epsilon) * ctrl->dt;
    
    // 限幅
    if (ctrl->eta < ctrl->eta_min) ctrl->eta = ctrl->eta_min;
    if (ctrl->eta > ctrl->eta_max) ctrl->eta = ctrl->eta_max;
    
    return u_eq + u_sw;
}
```

### 4.3 自适应+超螺旋组合

```
将超螺旋中的k和λ也做自适应:
  k̇ = α₁·(|s| - δ₁)
  λ̇ = α₂·(|s| - δ₂)

效果：自动调整增益，对时变不确定性有良好适应性
```

## 5. 模糊滑模控制（FSMC）

### 5.1 思路
用模糊推理替代切换控制律中的sign/sat函数，平滑控制信号。

### 5.2 设计框架

```
输入: s (滑模面), ṡ (滑模面导数)
输出: Δu (切换控制的平滑替代)

模糊规则示例 (7条):
IF s is NB AND ṡ is NB THEN Δu is PB
IF s is NM AND ṡ is NB THEN Δu is PM
IF s is ZO AND ṡ is ZO THEN Δu is ZO
IF s is PB AND ṡ is PB THEN Δu is NB
...

控制律: u = u_eq + Δu_fuzzy
```

## 6. 滑模控制在电赛中的应用

### 6.1 电机速度环

```
被控对象: J·ω̇ = K_t·i - B·ω - T_L
切换面: s = ė_ω + λ·e_ω
控制输出: 电流参考 i_ref

超螺旋实现 (推荐):
  无需测加速度，抗负载扰动强
```

### 6.2 DC-DC变换器

```
Buck变换器滑模:
  s = α·(V_ref - V_o) + β·d(V_ref - V_o)/dt

切换面设计要点:
  - 包含输出电压误差及其导数
  - 控制量为占空比 (限幅0~1)
  - 需要低通滤波器平滑输出
```

### 6.3 平衡车/倒立摆

```
多变量滑模:
  s₁ = c₁·e_x + ė_x        (位置)
  s₂ = c₂·e_θ + ė_θ        (角度)
  s = k₁·s₁ + s₂           (组合切换面)

推荐: 超螺旋 + 自适应增益
```

## 7. 工程实践要点

### 7.1 抖振抑制方法对比

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 边界层法 | sat(s)替代sign(s) | 简单 | 牺牲精度 |
| 超螺旋 | 高阶滑模 | 连续控制 | 参数整定复杂 |
| 模糊替代 | 模糊推理 | 平滑 | 规则设计复杂 |
| 滤波法 | 低通滤波输出 | 易实现 | 引入相位滞后 |
| 自适应 | 动态调整增益 | 自动平衡 | 可能收敛慢 |

### 7.2 离散化注意事项

```
采样频率要求:
  - 至少 > 10× 滑模带宽频率
  - 电机控制: 建议 > 10kHz
  - DC-DC: 建议 > 100kHz (或固定频率PWM)

离散超螺旋:
  u₁[k] = u₁[k-1] - λ·Ts·sign(s[k])
  u₂[k] = -k·|s[k]|^0.5 · sign(s[k])
  
  注意: Ts要足够小，否则离散化破坏稳定性
```

### 7.3 调参流程

```
1. 先用经典PI/PD控制，获取系统带宽和响应特性
2. 设计切换面 s = ė + λe，λ取系统带宽的2~5倍
3. 先用饱和函数sat(s)，边界层φ取误差允许范围
4. 逐步增大切换增益η，直到抗扰性能满足
5. 替换为超螺旋，按收敛条件设定λ和k
6. 加入自适应律，放宽参数敏感性
```

## 8. 参考代码模板

### 8.1 通用滑模控制器框架

```c
typedef enum {
    SMC_CLASSIC,
    SMC_INTEGRAL,
    SMC_SUPERTWISTING,
    SMC_ADAPTIVE
} SMC_Type_t;

typedef struct {
    SMC_Type_t type;
    float lambda;       // 切换面斜率
    float eta;          // 切换增益
    float phi;          // 边界层
    // 积分滑模
    float gamma;
    float integral_e;
    // 超螺旋
    float k_st;
    float u1_integral;
    // 自适应
    float gamma_adapt;
    float epsilon_adapt;
    // 通用
    float dt;
    float output_min;
    float output_max;
} SMC_t;

float SMC_Calculate(SMC_t *ctrl, float e, float de, float dde) {
    float s, u = 0;
    
    switch (ctrl->type) {
    case SMC_CLASSIC:
        s = de + ctrl->lambda * e;
        u = -ctrl->lambda * de - ctrl->eta * sat(s, ctrl->phi);
        break;
        
    case SMC_INTEGRAL:
        ctrl->integral_e += e * ctrl->dt;
        s = de + ctrl->lambda * e + ctrl->gamma * ctrl->integral_e;
        u = -ctrl->lambda * de - ctrl->gamma * e 
            - ctrl->eta * sat(s, ctrl->phi);
        break;
        
    case SMC_SUPERTWISTING:
        s = de + ctrl->lambda * e;
        float sign_s = sign(s);
        float sqrt_s = sqrtf(fabsf(s));
        ctrl->u1_integral -= ctrl->k_st * ctrl->dt * sign_s;
        u = ctrl->u1_integral - ctrl->eta * sqrt_s * sign_s 
            - ctrl->lambda * de;
        break;
        
    case SMC_ADAPTIVE:
        s = de + ctrl->lambda * e;
        u = -ctrl->lambda * de - ctrl->eta * sat(s, ctrl->phi);
        ctrl->eta += ctrl->gamma_adapt * (fabsf(s) - ctrl->epsilon_adapt) * ctrl->dt;
        if (ctrl->eta < 0.1f) ctrl->eta = 0.1f;
        break;
    }
    
    // 输出限幅
    return constrain(u, ctrl->output_min, ctrl->output_max);
}

static inline float sat(float s, float phi) {
    if (fabsf(s) > phi) return (s > 0) ? 1.0f : -1.0f;
    return s / phi;
}

static inline float sign(float s) {
    if (s > 0) return 1.0f;
    if (s < 0) return -1.0f;
    return 0.0f;
}
```
