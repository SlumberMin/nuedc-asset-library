# 非线性控制实战指南

> 电赛非线性控制方法速查与实战

---

## 一、为什么需要非线性控制

| 场景 | 线性控制局限 | 非线性控制优势 |
|------|-------------|---------------|
| 大范围角度摆动 | 小角度线性化误差大 | 全局稳定 |
| 变负载系统 | PID参数需频繁整定 | 自适应补偿 |
| 存在摩擦/死区 | 稳态振荡 | 可显式补偿 |
| 快速响应需求 | 响应慢/超调大 | 有限时间收敛 |

## 二、反馈线性化（Feedback Linearization）

### 2.1 核心思想

通过非线性状态变换将原系统转化为线性系统，再用线性控制方法设计。

### 2.2 适用模型：SISO仿射非线性系统

```
ẋ = f(x) + g(x)u
y = h(x)
```

### 2.3 相对度（Relative Degree）

输出y对时间求导直到显式出现u：

```
y' = Lfh(x)          # 若 Lfh ≠ 0 且不依赖u → r=1
y'' = Lf²h(x) + LgLfh·u  # 若 Lfh=0 且 LgLfh≠0 → r=2
```

- `Lfh = ∇h·f` 为李导数
- 相对度r = 输出需微分几次才出现u

### 2.4 设计步骤（以倒立摆为例，r=2）

```
# 系统: θ̈ = f(θ,θ̇) + g(θ)u

# 第1步：计算相对度
# y = θ, y' = θ̇, y'' = f(θ,θ̇) + g(θ)u  → r=2

# 第2步：选择控制律
v = -k1*(θ - θ_ref) - k2*(θ̇ - θ̇_ref) + θ̈_ref
u = (v - f(θ,θ̇)) / g(θ)

# 第3步：闭环 → (θ-θ̈_ref)'' + k2*(θ̇-θ̇_ref)' + k1*(θ-θ_ref) = 0
# 特征方程 s² + k2*s + k1 = 0，选k1,k2使极点在左半平面
```

### 2.5 C实现框架

```c
// 反馈线性化控制器
typedef struct {
    float k1, k2;       // 线性化后PD增益
    float x1_ref, x2_ref; // 参考状态
} FBL_Controller;

float fbl_compute(FBL_Controller *c, float x1, float x2,
                  float f_val, float g_val, float x2_ref_dot, float x1_ref_dot2) {
    float e1 = x1 - c->x1_ref;
    float e2 = x2 - c->x2_ref;
    float v = -c->k1 * e1 - c->k2 * e2 + x1_ref_dot2;
    float u = (v - f_val) / g_val;
    return u;
}
```

### 2.6 注意事项

- 要求精确的系统模型 f(x), g(x)
- g(x)不能为零（奇异点）
- 模型失配时性能急剧下降
- 实际中常与自适应结合使用

## 三、反步法（Backstepping）

### 3.1 核心思想

对严格反馈系统，逐步构造Lyapunov函数，从最后一个子系统往回"反推"控制律。

### 3.2 适用模型：严格反馈系统

```
ẋ₁ = f₁(x₁) + g₁(x₁)x₂
ẋ₂ = f₂(x₁,x₂) + g₂(x₁,x₂)x₃
...
ẋₙ = fₙ(x) + gₙ(x)u
```

### 3.3 两步反步法设计

```
# Step 1: 视x₂为虚拟控制
# 令 α₁(x₁) = 稳定x₁子系统的虚拟控制律
# e₁ = x₁ - x₁_ref
# V₁ = ½e₁²
# α₁ = (-c₁·e₁ - f₁(x₁)) / g₁(x₁)   使 V̇₁ = -c₁·e₁²

# Step 2: 设计真实控制u使x₂跟踪α₁
# e₂ = x₂ - α₁
# V₂ = V₁ + ½e₂²
# u = (1/g₂)(-c₂·e₂ - ∂α₁/∂x₁·(f₁+g₁x₂) - g₁·e₁ - f₂)
```

### 3.4 C实现（二级严格反馈）

```c
typedef struct {
    float c1, c2;  // 设计参数
} Backstep_Ctrl;

float backstep_compute(Backstep_Ctrl *c,
                       float x1, float x2, float x1_ref,
                       float f1, float g1, float f2, float g2,
                       float alpha1_dot) {
    float e1 = x1 - x1_ref;
    // 虚拟控制
    float alpha1 = (-c->c1 * e1 - f1) / g1;
    float e2 = x2 - alpha1;
    // 真实控制
    float u = (-c->c2 * e2 - alpha1_dot * (f1 + g1*x2)
               - g1*e1 - f2) / g2;
    return u;
}
```

### 3.5 反步法优势

- 系统化的Lyapunov设计
- 可自然处理参数不确定性（加自适应律）
- 适合电赛中多级串联结构（如电机+丝杆+负载）

## 四、滑模控制（Sliding Mode Control）

### 4.1 核心思想

设计滑模面 s=0，迫使系统状态在有限时间到达滑模面并沿其滑动。

### 4.2 设计步骤

```
# 第1步：设计滑模面
s = c·e + ė     (二阶系统)
# e为跟踪误差，c>0

# 第2步：设计趋近律
# 等速趋近: u = u_eq - k·sign(s)
# 指数趋近: u = u_eq - k·sign(s) - λ·s

# 第3步：计算等效控制 u_eq
# 令 ṡ = 0 求解 u_eq

# 第4步：完整控制律
u = u_eq - k·sign(s)    # k > |扰动上界|
```

### 4.3 C实现

```c
typedef struct {
    float c;      // 滑模面斜率
    float k;      // 切换增益
    float lambda; // 指数趋近系数
    float dt;
} SMC_Controller;

float smc_compute(SMC_Controller *c, float e, float e_dot,
                  float u_eq) {
    float s = c->c * e + e_dot;
    // 趋近律：指数趋近 + 等速趋近
    float u = u_eq - c->k * sgn(s) - c->lambda * s;
    return u;
}

// 符号函数
static inline float sgn(float x) {
    if (x > 0.01f) return 1.0f;
    if (x < -0.01f) return -1.0f;
    return 0.0f;
}
```

### 4.4 抖振抑制

| 方法 | 实现 | 效果 |
|------|------|------|
| 边界层法 | sign(s) → sat(s/φ) | 简单有效，最常用 |
| 连续趋近 | -k·s/(\|s\|+δ) | 平滑但收敛慢 |
| 高阶滑模 | 超螺旋算法 | 消除抖振，复杂度高 |

```c
// 饱和函数替代符号函数
float sat(float s, float phi) {
    if (s > phi) return 1.0f;
    if (s < -phi) return -1.0f;
    return s / phi;
}
```

### 4.5 参数选取经验

```
滑模面斜率 c：决定收敛速度
  - c大 → 收敛快但对噪声敏感
  - c小 → 平滑但慢
  - 经验: c ≈ 3~10 × 期望带宽

切换增益 k：必须大于扰动上界
  - k > |d_max| + η (η>0为裕量)
  - 通常取 k = 1.5~3 × 估计扰动最大值

边界层 φ：
  - φ = 0.01~0.1 × 满量程
  - 太小→抖振，太大→稳态误差
```

## 五、自适应控制

### 5.1 核心思想

在线估计未知参数，实时更新控制律。

### 5.2 MRAC（模型参考自适应）

```
# 参考模型: ẋ_m = a_m·x_m + b_m·r
# 实际系统: ẋ = a·x + b·u  (a,b未知)
# 控制律: u = θ₁·r + θ₂·x
# 自适应律 (MIT规则改良):
# θ̇₁ = -γ₁·e·r / (1 + r² + x²)
# θ̇₂ = -γ₂·e·x / (1 + r² + x²)
```

### 5.3 自适应反步法

```c
// 在反步法基础上加入参数估计
typedef struct {
    float theta_hat;  // 参数估计值
    float gamma;      // 自适应增益
    float c1, c2;
} Adaptive_Backstep;

float adaptive_backstep(Adaptive_Backstep *c,
                        float x1, float x2, float x1_ref,
                        float phi1, float phi2,
                        float dt) {
    float e1 = x1 - x1_ref;
    float alpha1 = (-c->c1 * e1 - c->theta_hat * phi1);
    float e2 = x2 - alpha1;

    // 自适应律（梯度法）
    c->theta_hat += c->gamma * e1 * phi1 * dt;
    // 参数投影（防止发散）
    if (c->theta_hat > 100.0f) c->theta_hat = 100.0f;
    if (c->theta_hat < -100.0f) c->theta_hat = -100.0f;

    float u = (-c->c2 * e2 - c->theta_hat * phi2
               - e1) / 1.0f;
    return u;
}
```

### 5.4 自适应增益γ选取

- γ太大 → 参数振荡
- γ太小 → 自适应太慢
- 经验: γ = 0.1~10，从大到小试
- 实际电赛中: γ = 1 通常是合理起点

## 六、方法对比与选型指南

| 方法 | 模型依赖 | 实现难度 | 抗扰性 | 电赛适用场景 |
|------|---------|---------|--------|-------------|
| 反馈线性化 | 高（需精确模型） | 中 | 低 | 模型已知的机械系统 |
| 反步法 | 中（需结构） | 中 | 中 | 多级串联系统 |
| 滑模 | 低 | 低 | **强** | **电赛首选** |
| 自适应 | 低 | 中 | 中 | 参数变化大的系统 |

### 电赛推荐方案

```
首选: 滑模控制（鲁棒性强、实现简单）
进阶: 自适应滑模（参数变化场景）
高阶: 反步法+自适应（多级系统、有时间调参）
```

## 七、电赛实战技巧

1. **先PID后非线性**：先用PID跑通，再替换为非线性控制器
2. **模型辨识先行**：阶跃响应法获取系统参数
3. **滑模最实用**：4天比赛中，滑模控制性价比最高
4. **仿真验证**：用MATLAB/Simulink先验证再移植
5. **抗积分饱和**：所有非线性控制器都需加输出限幅
