# 迭代学习控制（ILC）实战指南

> 电赛控制算法速查 | 适用周期性任务的高精度跟踪控制

---

## 一、ILC核心思想

```
重复执行同一任务 → 利用前次误差修正本次输入 → 逐次逼近理想轨迹
```

**数学描述**：
$$u_{k+1}(t) = Q(q)[u_k(t) + L \cdot e_k(t)]$$

- $u_k(t)$：第k次迭代的控制输入
- $e_k(t) = y_d(t) - y_k(t)$：第k次跟踪误差
- $L$：学习律（核心参数）
- $Q(q)$：Q滤波器（抑制高频噪声）

---

## 二、三种基本学习律

### 2.1 P型ILC

```c
// 最简单：只用误差本身学习
u[k+1][t] = u[k][t] + Kp * e[k][t];

// 优点：实现简单，对初值偏差不敏感
// 缺点：收敛速度慢
// 适用：低阶系统、精度要求不高
```

**收敛条件**：$\|1 - K_p B\| < 1$（B为系统增益）

### 2.2 D型ILC

```c
// 用误差的微分学习，跟踪效果更好
u[k+1][t] = u[k][t] + Kd * (e[k][t+1] - e[k][t]);  // 一阶差分近似de/dt

// 优点：收敛快，对系统阶次不敏感
// 缺点：放大噪声
// 适用：需要快速收敛的场合
```

**收敛条件**：$\|1 - K_d C B\| < 1$（C为系统输出矩阵）

### 2.3 PD型ILC（推荐）

```c
// 综合P型和D型优点
u[k+1][t] = u[k][t] + Kp * e[k][t] + Kd * (e[k][t+1] - e[k][t]);

// 推荐参数选择（经验值）：
// Kp = 0.3~0.8 / B（B为系统直流增益）
// Kd = 0.1~0.5 / (CB)
// 通常Kd < Kp
```

### 2.4 高阶ILC

```c
// 利用多次迭代历史信息
u[k+1][t] = a1*(u[k][t]+L*e[k][t]) + a2*(u[k-1][t]+L*e[k-1][t]) + a3*(u[k-2][t]+L*e[k-2][t]);
// a1+a2+a3=1，通常a1>0.7
```

## 三、Q滤波器设计

```
目的：抑制学习过程中的高频噪声积累
形式：Q(z) = (1+2z⁻¹+z⁻²)/4  （二阶均值滤波器，常用）
```

```c
// Q滤波器实现（滑动平均）
float Q_filter(float u_raw, float u_prev1, float u_prev2) {
    return (u_raw + 2.0f * u_prev1 + u_prev2) / 4.0f;
}

// 或低通滤波器形式
// Q(s) = ωc / (s + ωc)，截止频率选在系统带宽附近
```

**Q滤波器取值建议**：
| 鲁棒性需求 | Q值 | 效果 |
|---|---|---|
| 噪声大 | Q=0.8~0.9 | 稳定但收敛慢 |
| 噪声小 | Q=0.95~1.0 | 收敛快 |
| 纯理论 | Q=1.0 | 最优收敛 |

## 四、完整实现模板

```c
/*=============================================================
 * ILC控制器 - 适用于周期性跟踪任务
 * 周期: N个采样点 = 一个完整周期
 * 最大迭代次数: K_max
 *=============================================================*/
#define N_PERIOD   500      // 一个周期的采样点数
#define K_MAX      20       // 最大迭代次数

float u_mem[K_MAX+1][N_PERIOD];  // 控制量存储
float e_mem[K_MAX+1][N_PERIOD];  // 误差存储
float y_d[N_PERIOD];             // 期望轨迹（预存）

float Kp_ilc = 0.5f;
float Kd_ilc = 0.2f;
float Q_val  = 0.9f;

int iter = 0;  // 当前迭代次数
int step = 0;  // 当前周期内步序

float ILC_update(int k, int t) {
    if (k == 0) return 0.0f;  // 首次迭代用零输入

    float u_raw = u_mem[k-1][t] + Kp_ilc * e_mem[k-1][t];
    if (t > 0) {
        u_raw += Kd_ilc * (e_mem[k-1][t] - e_mem[k-1][t-1]);
    }

    // Q滤波
    float u_filtered = Q_val * u_raw + (1.0f - Q_val) * u_mem[k-1][t];
    return u_filtered;
}

// 在控制定时器中断中调用
void ILC_ControlISR(void) {
    float y_meas = read_sensor();
    float error = y_d[step] - y_meas;

    e_mem[iter][step] = error;
    float u_out = ILC_update(iter, step);
    u_mem[iter][step] = u_out;

    set_actuator(u_out);

    step++;
    if (step >= N_PERIOD) {
        step = 0;
        iter++;
        if (iter > K_MAX) iter = K_MAX;  // 保持最新迭代数据
        // 计算本轮最大误差用于收敛判断
        float max_err = 0;
        for (int i = 0; i < N_PERIOD; i++) {
            float ae = fabsf(e_mem[iter-1][i]);
            if (ae > max_err) max_err = ae;
        }
        if (max_err < 0.01f) {
            // 收敛达标，可切换到正常运行
        }
    }
}
```

## 五、收敛性分析

### 5.1 收敛条件总结

| 学习律 | 收敛条件 | 收敛速度 |
|---|---|---|
| P型 | $\|1-K_p B\|<1$ | 慢，$\rho \approx |1-K_p B|$ |
| D型 | $\|1-K_d CB\|<1$ | 中等 |
| PD型 | $\|1-K_p B - K_d CB\|<1$ | 快 |
| Q-PD型 | $Q\|1-K_p B-K_d CB\|<1$ | 取决于Q |

### 5.2 收敛速度估计

```
误差衰减比 ρ = |1 - Kp*B - Kd*C*B| * Q
第k次迭代误差 ≈ ρ^k * e_0
达到目标精度所需迭代次数 ≈ ln(e_target/e_0) / ln(ρ)

例：ρ=0.6, e_0=1.0, e_target=0.01
    k ≈ ln(0.01)/ln(0.6) ≈ 9次迭代
```

### 5.3 不收敛怎么办？

1. **减小学习增益** Kp、Kd（保守但安全）
2. **增大Q滤波**（牺牲收敛速度换稳定性）
3. **检查系统模型**是否时变（ILC假设系统参数不变）
4. **检查初态偏差**（每次迭代起点不一致导致发散）

## 六、电赛典型应用场景

### 6.1 倒立摆周期性摆起

```
任务：重复执行摆起-平衡过程
ILC优势：每次摆起的控制力矩逐次优化
典型参数：N=200, Kp=0.4, Kd=0.15, Q=0.9
```

### 6.2 小车循迹（固定赛道）

```
任务：沿固定赛道反复跑圈
ILC优势：第1圈学习赛道特征，后续圈逐渐提速
典型参数：N=赛道长度/采样步长, Kp=0.5, Kd=0.2
关键：位置编码器精确记录每个点的误差
```

### 6.3 电机转速周期性跟踪

```
任务：跟踪周期性速度曲线（如正弦波）
ILC优势：消除稳态跟踪误差
典型参数：N=速度周期/采样周期, Kp=0.6, Kd=0.3
```

### 6.4 机械臂轨迹跟踪

```
任务：重复抓取-放置动作
ILC优势：逐步消除动态误差
典型参数：N=动作周期/采样周期, Kp=0.3, Kd=0.2
```

## 七、与PID结合的混合控制

```
实际应用中，ILC通常与反馈PID配合使用：
  u_total = u_ILC + u_PID

PID负责：抑制非重复性扰动、保证每次迭代稳定性
ILC负责：消除重复性误差、逐次提高跟踪精度
```

```c
float pid_output = PID_calculate(error);
float ilc_output = u_mem[iter][step];  // ILC前馈
float total = ilc_output + pid_output;
set_actuator(total);
```

## 八、调试经验总结

| 问题 | 原因 | 解决 |
|---|---|---|
| 前几次迭代振荡 | 增益过大 | 降低Kp、Kd |
| 收敛但有残差 | Q滤波太强/模型误差 | 增大Q或加PID补偿 |
| 每次结果不同 | 初态不一致 | 加初始位置复位机构 |
| 高频噪声积累 | D型学习律 | 加Q滤波或换P型 |
| 周期不准确 | N取值不对 | 精确标定周期采样数 |
