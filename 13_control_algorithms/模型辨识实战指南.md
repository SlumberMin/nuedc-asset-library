## 1. 为什么要系统辨识？

```
物理建模（白箱）← 结合 → 系统辨识（黑/灰箱）→ 精确控制模型
     ↑                        ↑
  知道结构但参数未知         从输入输出数据推断
```

**电赛中的典型场景：**
- 电机转速 ↔ PWM 的传递函数
- 倒立摆角度 ↔ 力矩的动态特性
- 温控系统的加热/散热模型
- 电压环/电流环的参数整定

## 2. 阶跃响应法（Step Response Method）

### 2.1 原理

对系统施加阶跃输入，记录输出响应，从响应曲线提取模型参数。

### 2.2 一阶系统辨识

```
G(s) = K / (τs + 1)

阶跃响应: y(t) = K·(1 - e^(-t/τ))
```

**辨识步骤：**
```
         y(t)
    K ────────────────────────  ─┐
    │                    ·       │ y(τ) = 0.632K
    │               ·            │
    │           ·                │
    │        ·                   │
    │     ·                      │
    │   ·                        │
    │  ·                         │
    ├────────────── t ──────────┘
    0     τ
```

1. 施加幅值为 A 的阶跃输入
2. 记录输出达到 63.2% 稳态值的时间 → **τ（时间常数）**
3. 稳态值 / A → **K（增益）**

### 2.3 二阶系统辨识（S形曲线）

```
G(s) = K·ω²_n / (s² + 2ζω_n·s + ω²_n)
```

对于过阻尼（ζ>1）的S形响应：

```
         y(t)
    K ────────────────────────
    │                  ····
    │              ··
    │           ··      拐点处切线
    │         ·       ╱
    │       ·       ╱  → 与稳态线交点 = t1+t2
    │      ·      ╱
    │     ·     ╱
    │    ·    ╱
    │   ·   ╱
    │  ·  ╱
    │ · ╱
    ├──────────────────────── t
    0   t1  t2
```

**辨识步骤：**
1. 找到响应曲线拐点
2. 过拐点做切线，与 0 和 K 的交点分别为 t1、t2
3. **K** = 稳态增益
4. **τ₁** = t1，**τ₂** = t2 - t1
5. 模型：G(s) = K / ((τ₁s+1)(τ₂s+1))

### 2.4 代码实现

```python
import numpy as np
from scipy.optimize import curve_fit

def first_order_step(t, K, tau, delay):
    """一阶加滞后模型"""
    y = np.zeros_like(t)
    mask = t >= delay
    y[mask] = K * (1 - np.exp(-(t[mask] - delay) / tau))
    return y

def identify_first_order(t, y_data, u_step=1.0):
    """
    阶跃响应法辨识一阶加滞后系统
    输入: 时间序列t, 输出数据y_data, 阶跃幅值u_step
    输出: 增益K, 时间常数tau, 滞后delay
    """
    # 初始猜测
    y_ss = y_data[-1]  # 稳态值
    K_guess = y_ss / u_step
    tau_guess = t[np.argmin(np.abs(y_data - 0.632 * y_ss))]
    delay_guess = 0
    
    # 曲线拟合
    popt, pcov = curve_fit(first_order_step, t, y_data,
                           p0=[K_guess, tau_guess, delay_guess],
                           bounds=([0, 0, 0], [np.inf, np.inf, t[-1]/2]))
    K, tau, delay = popt
    return K, tau, delay

# 实际使用
t = np.linspace(0, 5, 500)
# 模拟采集数据（真实参数: K=2, tau=0.5, delay=0.1）
y_true = first_order_step(t, K=2.0, tau=0.5, delay=0.1)
y_noisy = y_true + np.random.normal(0, 0.02, len(t))

K, tau, delay = identify_first_order(t, y_noisy)
print(f"辨识结果: K={K:.3f}, τ={tau:.3f}, delay={delay:.3f}")
```

### 2.5 阶跃法优缺点

| 优点 | 缺点 |
|------|------|
| ✅ 直观简单 | ❌ 需要系统达到稳态 |
| ✅ 不需要复杂设备 | ❌ 对噪声敏感 |
| ✅ 适合现场调试 | ❌ 只能辨识低阶模型 |
| ✅ 一人即可操作 | ❌ 大信号激励可能损坏设备 |

## 3. 频率响应法（Frequency Response Method）

### 3.1 原理

```
对系统施加正弦输入: u(t) = A·sin(ωt)
记录稳态输出:        y(t) = A·|G(jω)|·sin(ωt + φ(ω))

扫频 → 得到幅频特性 |G(jω)| 和相频特性 φ(ω)
```

### 3.2 辨识流程

```
┌─────────────────────────────────────────────┐
│ 1. 准备信号发生器 + 数据采集                  │
│ 2. 选择频率点 (对数间隔, 如 0.1~100Hz)        │
│ 3. 每个频率施加正弦信号，等待稳态             │
│ 4. 测量输入输出幅值比 → |G(jω)|              │
│ 5. 测量输出滞后输入的相位 → φ(ω)             │
│ 6. 绘制 Bode 图                              │
│ 7. 从 Bode 图拟合传递函数                     │
└─────────────────────────────────────────────┘
```

### 3.3 代码实现

```python
import numpy as np
from scipy import signal

def frequency_identification(u_input, y_output, fs, freqs_to_test):
    """
    频率响应法系统辨识
    
    参数:
        u_input: 输入信号（各频率正弦拼接或单独实验）
        y_output: 输出信号
        fs: 采样频率
        freqs_to_test: 需要辨识的频率列表(Hz)
    
    返回:
        freqs: 频率点
        magnitude_db: 幅值(dB)
        phase_deg: 相位(度)
    """
    # 使用互功率谱法（抗噪性好）
    f, Pxy = signal.csd(u_input, y_output, fs=fs, nperseg=1024)
    f, Pxx = signal.welch(u_input, fs=fs, nperseg=1024)
    
    # 频率响应 = 互功率谱 / 自功率谱
    H = Pxy / Pxx
    magnitude_db = 20 * np.log10(np.abs(H))
    phase_deg = np.degrees(np.angle(H))
    
    return f, magnitude_db, phase_deg

def bode_to_transfer_function(freqs, mag_db, phase_deg, order=2):
    """
    从Bode图拟合传递函数
    
    简化版：手动辨识
    """
    # 从Bode图提取关键参数
    # 低频增益
    K = 10 ** (mag_db[0] / 20)
    
    # 找转折频率（幅值下降3dB的点）
    ref_mag = mag_db[0]
    for i, m in enumerate(mag_db):
        if ref_mag - m >= 3:
            omega_c = freqs[i] * 2 * np.pi
            break
    
    # 从相位裕度估算阻尼比
    # 一阶系统: G(s) = K / (s/omega_c + 1)
    num = [K]
    den = [1/omega_c, 1]
    sys = signal.TransferFunction(num, den)
    
    return sys

def generate_sweep_signal(freqs, amplitudes, fs, cycles_per_freq=5):
    """生成扫频信号"""
    t_total = 0
    for f, n in zip(freqs, [cycles_per_freq]*len(freqs)):
        t_total += n / f
    
    t = np.arange(0, t_total, 1/fs)
    u = np.zeros_like(t)
    offset = 0
    
    for f, amp in zip(freqs, amplitudes):
        duration = cycles_per_freq / f
        n_samples = int(duration * fs)
        t_local = np.arange(n_samples) / fs
        u[offset:offset+n_samples] = amp * np.sin(2*np.pi*f*t_local)
        offset += n_samples
    
    return t[:offset], u[:offset]

# 实际使用
fs = 1000  # 采样率 1kHz
freqs = np.logspace(-1, 2, 20)  # 0.1Hz ~ 100Hz，20个点
amps = np.ones_like(freqs) * 2.0  # 输入幅值2V

t, u = generate_sweep_signal(freqs, amps, fs)
# 假设y是从实际系统采集的输出
# f, mag, phase = frequency_identification(u, y, fs, freqs)
```

## 4. 最小二乘法（Least Squares）

### 4.1 离散差分方程模型

```
y(k) + a₁y(k-1) + ... + aₙy(k-n) = b₁u(k-1) + ... + bₘu(k-m) + e(k)

向量形式: y = Φ·θ + e

θ = [a₁, ..., aₙ, b₁, ..., bₘ]ᵀ  (待辨识参数)
```

### 4.2 递推最小二乘（RLS）

```
θ̂(k) = θ̂(k-1) + K(k)·[y(k) - φᵀ(k)·θ̂(k-1)]

K(k) = P(k-1)·φ(k) / [λ + φᵀ(k)·P(k-1)·φ(k)]
P(k) = [I - K(k)·φᵀ(k)]·P(k-1) / λ
```

**λ = 遗忘因子**（0.95~0.999），使算法跟踪时变参数。

### 4.3 代码实现

```python
import numpy as np

class RecursiveLeastSquares:
    """递推最小二乘辨识器"""
    def __init__(self, n_params, forgetting_factor=0.99, P0=1000):
        self.n = n_params
        self.lambda_ = forgetting_factor
        self.theta = np.zeros(n_params)          # 参数估计
        self.P = np.eye(n_params) * P0           # 协方差矩阵
        self.history = []                         # 参数历史
    
    def update(self, phi, y):
        """
        单步更新
        phi: 回归向量 [y(k-1), ..., y(k-na), u(k-1), ..., u(k-nb)]
        y: 当前输出 y(k)
        """
        phi = np.array(phi).reshape(-1, 1)
        y_hat = (phi.T @ self.theta.reshape(-1, 1))[0, 0]
        error = y - y_hat
        
        # 增益向量
        K = self.P @ phi / (self.lambda_ + phi.T @ self.P @ phi)
        
        # 参数更新
        self.theta = self.theta + (K * error).flatten()
        
        # 协方差更新
        self.P = (self.P - K @ phi.T @ self.P) / self.lambda_
        
        self.history.append(self.theta.copy())
        return y_hat, error

def identify_transfer_function(u, y, na=2, nb=2, delay=1):
    """
    辨识离散传递函数 G(z) = (b1*z^-1 + ... + b_nb*z^-nb) / (1 + a1*z^-1 + ... + a_na*z^-na)
    
    参数:
        u: 输入序列
        y: 输出序列
        na, nb: 分母/分子阶次
        delay: 纯滞后拍数
    """
    n_params = na + nb
    rls = RecursiveLeastSquares(n_params, forgetting_factor=0.998)
    
    N = len(y)
    for k in range(max(na, nb + delay), N):
        # 构建回归向量
        phi = []
        for i in range(1, na + 1):
            phi.append(-y[k - i])          # -y(k-1), -y(k-2), ...
        for i in range(1, nb + 1):
            phi.append(u[k - delay - i + 1])  # u(k-d), u(k-d-1), ...
        
        rls.update(phi, y[k])
    
    # 提取最终参数
    a = np.concatenate(([1], rls.theta[:na]))     # 分母系数
    b = np.concatenate(([0], rls.theta[na:]))      # 分子系数
    
    return a, b, rls.history

# 实例：电机转速模型辨识
def motor_identification_demo():
    """电机PWM→转速 辨识示例"""
    # 生成测试数据
    fs = 100  # 100Hz采样
    N = 2000
    t = np.arange(N) / fs
    
    # PRBS输入信号（伪随机二进制序列）
    np.random.seed(42)
    u = np.random.choice([-1, 1], size=N) * 50  # ±50% PWM变化
    
    # 真实系统: G(z) = 0.15*z^-1 / (1 - 0.85*z^-1)
    y = np.zeros(N)
    for k in range(1, N):
        y[k] = 0.85 * y[k-1] + 0.15 * u[k-1]
    y += np.random.normal(0, 0.5, N)  # 加噪声
    
    # 辨识
    a, b, history = identify_transfer_function(u, y, na=1, nb=1, delay=1)
    print(f"辨识结果:")
    print(f"  a = {a}")  # 应接近 [1, -0.85]
    print(f"  b = {b}")  # 应接近 [0, 0.15]
    
    # 转换为连续传递函数
    # G(s) = K / (τs + 1), 其中 τ = -Ts / ln(a1), K = b1 * τ / Ts
    Ts = 1/fs
    tau = -Ts / np.log(a[1])
    K = b[1] * tau / Ts
    print(f"  连续模型: G(s) = {K:.3f} / ({tau:.3f}s + 1)")

motor_identification_demo()
```

## 5. 子空间辨识法（Subspace Identification）

### 5.1 原理

```
状态空间模型:
  x(k+1) = A·x(k) + B·u(k) + w(k)
  y(k)   = C·x(k) + D·u(k) + v(k)

子空间法: 直接从输入输出数据估计 A, B, C, D
无需迭代优化，基于线性代数分解
```

### 5.2 核心思想

```
构建Hankel矩阵:

        ┌ y(0)   y(1)  ...  y(j-1) ┐
        │ y(1)   y(2)  ...  y(j)   │
  Y_p = │  ⋮      ⋮    ⋱    ⋮     │  (past outputs)
        │ y(i-1) y(i) ... y(i+j-2)│
        ├──────────────────────────│
  Y_f = │ y(i)   y(i+1)... y(i+j-1)│  (future outputs)
        │ y(i+1) y(i+2)... y(i+j)  │
        │  ⋮      ⋮    ⋱    ⋮     │
        │ y(2i-1) y(2i)...y(2i+j-2)│
        └──────────────────────────┘

通过 SVD 分解提取可观子空间 → 估计系统矩阵
```

### 5.3 代码实现（N4SID算法）

```python
import numpy as np
from scipy import signal as sig

def n4sid(y, u, order, i=None):
    """
    N4SID 子空间辨识算法
    
    参数:
        y: 输出数据 (N × ny)
        u: 输入数据 (N × nu)
        order: 系统阶次
        i: Hankel行数 (默认=2*order)
    
    返回:
        A, B, C, D: 状态空间矩阵
    """
    y = np.atleast_2d(y).T if y.ndim == 1 else y
    u = np.atleast_2d(u).T if u.ndim == 1 else u
    
    N, ny = y.shape
    nu = u.shape[1]
    
    if i is None:
        i = 2 * order
    j = N - 2 * i + 1
    
    # 构建 Hankel 矩阵
    def build_block(data, rows, cols, offset=0):
        block = np.zeros((rows * data.shape[1], cols))
        for r in range(rows):
            for c in range(cols):
                block[r*data.shape[1]:(r+1)*data.shape[1], c] = data[offset + r + c]
        return block
    
    # Past and future blocks
    Up = build_block(u, i, j, 0)
    Uf = build_block(u, i, j, i)
    Yp = build_block(y, i, j, 0)
    Yf = build_block(y, i, j, i)
    
    # 构建 instrumental variable
    Wp = np.vstack([Up, Yp])
    
    # 投影
    # O = Yf @ [Wp; Uf]^+ @ [Wp; Uf]
    WU = np.vstack([Wp, Uf])
    # 使用最小二乘投影
    proj = Yf @ np.linalg.pinv(WU) @ WU
    
    # SVD分解
    U_svd, S_svd, Vt_svd = np.linalg.svd(proj, full_matrices=False)
    
    # 截取前n阶
    U1 = U_svd[:, :order]
    S1 = np.diag(S_svd[:order])
    
    # 可观性格拉姆矩阵
    Gamma = U1 @ np.sqrt(S1)
    
    # 提取 C 和 A
    C = Gamma[:ny, :]
    A = np.linalg.pinv(Gamma[:ny*(i-1), :]) @ Gamma[ny:, :]
    
    # 提取 B 和 D（通过最小二乘）
    # 构建回归矩阵
    X = np.zeros((order, j))
    for k in range(i):
        X += np.linalg.pinv(Gamma[:ny*(i-k), :]) @ Yf[ny*k:ny*(k+1), :]
    X = X / i
    
    # 求解 B, D
    regressors = np.vstack([X[:, :-1], u[:j-1].T])
    target = np.vstack([X[:, 1:], y[:j-1].T])
    BD = target @ np.linalg.pinv(regressors)
    
    A_est = BD[:order, :order]
    B_est = BD[:order, order:]
    D_est = BD[order:, order:]
    
    return A_est, B_est, C, D_est

# 验证示例
def subspace_demo():
    """子空间辨识验证"""
    # 创建测试系统（二阶振荡）
    A_true = np.array([[0.9, 0.1], [-0.1, 0.8]])
    B_true = np.array([[1.0], [0.5]])
    C_true = np.array([[1.0, 0.0]])
    D_true = np.array([[0.0]])
    
    # 仿真
    N = 1000
    u = np.random.randn(N, 1) * 0.5
    x = np.zeros((2, 1))

        y[k] = C_true @ x + D_true * u[k]

```
需要快速整定PID参数？

### 7.2 阶跃法快速PID整定流程（Ziegler-Nichols）

```python
def zn_tuning(K, tau, delay):
    """
    Ziegler-Nichols 阶跃响应法整定PID
    K: 过程增益, tau: 时间常数, delay: 滞后
    """
    a = K * delay / tau  # 无量纲参数
    
    # 经典ZN公式
    params = {
        'P':   {'Kp': 1/a},
        'PI':  {'Kp': 0.9/a, 'Ti': 3.33*delay},
        'PID': {'Kp': 1.2/a, 'Ti': 2*delay, 'Td': 0.5*delay}
    }
    
    pid = params['PID']
    Ki = pid['Kp'] / pid['Ti']
    Kd = pid['Kp'] * pid['Td']
    
    return pid['Kp'], Ki, Kd

# 示例：电机模型 K=2, τ=0.3s, delay=0.05s
Kp, Ki, Kd = zn_tuning(K=2.0, tau=0.3, delay=0.05)
print(f"PID参数: Kp={Kp:.2f}, Ki={Ki:.2f}, Kd={Kd:.4f}")
```

### 7.3 PRBS信号生成（用于最小二乘辨识）

```python
def generate_prbs(n_bits=10, n_samples=1000, amplitude=1.0):
    """
    生成伪随机二进制序列(PRBS)
    最大长度序列: N = 2^n_bits - 1
    """
    # LFSR生成最大长度序列
    register = [1] * n_bits
    prbs = []
    
    for _ in range(2**n_bits - 1):
        prbs.append(register[-1])
        new_bit = register[-1] ^ register[-2]  # XOR反馈
        register = [new_bit] + register[:-1]
    
    # 转换为 ±1 信号
    prbs = np.array(prbs) * 2 - 1
    
    # 重复到所需长度
    repeats = n_samples // len(prbs) + 1
    prbs = np.tile(prbs, repeats)[:n_samples]
    
    return prbs * amplitude

# 使用
prbs = generate_prbs(n_bits=8, n_samples=2000, amplitude=50)
# prbs 作为电机PWM输入，采集转速响应，然后用RLS辨识
```
