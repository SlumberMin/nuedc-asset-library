## 1. 采样控制系统概述

### 1.1 基本结构

```
参考输入 r(t) → [比较器] → e(t) → [ADC/采样] → e*(k) → [数字控制器] → u*(k) → [DAC/零阶保持] → u(t) → [被控对象] → y(t)
                  ↑                                                                              |
                  └──────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 关键术语

| 术语 | 符号 | 说明 |
|------|------|------|
| 采样周期 | T | 两次采样之间的时间间隔 |
| 采样频率 | fs = 1/T | 每秒采样次数 |
| 零阶保持器 | ZOH | 将离散信号在 T 内保持不变 |
| 量化 | — | 将连续幅值映射为有限离散值 |

## 2. Z 变换基础

### 2.1 定义

$$
X(z) = \sum_{k=0}^{\infty} x(kT) z^{-k}
$$

### 2.2 常用 Z 变换对

| 时域 x(t) | 拉氏变换 X(s) | Z 变换 X(z) |
|-----------|---------------|-------------|
| δ(t) | 1 | 1 |
| 1(t) | 1/s | z/(z-1) |
| t | 1/s² | Tz/(z-1)² |
| e^(-at) | 1/(s+a) | z/(z-e^(-aT)) |
| te^(-at) | 1/(s+a)² | Tze^(-aT)/(z-e^(-aT))² |
| sin(ωt) | ω/(s²+ω²) | z·sin(ωT) / (z²-2z·cos(ωT)+1) |
| cos(ωt) | s/(s²+ω²) | z(z-cos(ωT)) / (z²-2z·cos(ωT)+1) |

### 2.3 Z 变换性质

| 性质 | 表达式 |
|------|--------|
| 线性 | Z[ax₁+x₂] = aX₁(z) + X₂(z) |
| 时移（右移） | Z[x(k-n)] = z^(-n) X(z) |
| 时移（左移） | Z[x(k+n)] = z^n X(z) - Σ x(k)z^(n-1-k) |
| 初值定理 | x(0) = lim(z→∞) X(z) |
| 终值定理 | x(∞) = lim(z→1) (z-1)X(z) |
| 卷积定理 | Z[x₁*x₂] = X₁(z)·X₂(z) |

## 3. 连续系统离散化方法

### 3.1 方法对比

| 方法 | 变换公式 | 精度 | 稳定性 | 适用场景 |
|------|---------|------|--------|---------|
| 前向差分 | s = (z-1)/T | 低 | 可能不稳定 | 简单系统快速原型 |
| 后向差分 | s = (z-1)/(Tz) | 中 | 稳定 | 一阶滤波器设计 |
| 双线性变换(Tustin) | s = (2/T)(z-1)/(z+1) | 高 | 稳定 | **推荐默认方法** |
| 脉冲不变法 | 保持脉冲响应 | 中 | 可能混叠 | 数字滤波器设计 |
| 阶跃不变法 | 保持阶跃响应 | 高 | 可能混叠 | 控制系统设计 |
| 零极点匹配法 | s+α → z-e^(-αT) | 高 | 好 | 低阶系统精确匹配 |

### 3.2 双线性变换详解（推荐）

```
s = (2/T) * (z-1)/(z+1)

即: z = (1 + sT/2) / (1 - sT/2)
```

**频率预畸变**：双线性变换会将整个 s 左半平面映射到单位圆内，但频率发生畸变：

$$
\omega_a = \frac{2}{T} \tan\left(\frac{\omega_d T}{2}\right)
$$

其中 ω_a 为模拟频率（预畸变后），ω_d 为期望数字频率。

**设计流程**：
1. 给定期望数字截止频率 ω_d
2. 计算预畸变模拟频率 ω_a = (2/T)·tan(ω_d·T/2)
3. 用 ω_a 设计模拟滤波器 H(s)
4. 用双线性变换替换 s → (2/T)(z-1)/(z+1) 得到 H(z)

### 3.3 Python 离散化示例

```python
import numpy as np
from scipy import signal

# 连续系统: G(s) = 10/(s+10)
num_c = [10]
den_c = [1, 10]
sys_c = signal.TransferFunction(num_c, den_c)

T = 0.001  # 1ms 采样周期

# 方法1: 双线性变换
sys_d_tustin = sys_c.to_discrete(T, method='tustin')

# 方法2: 后向差分
sys_d_backward = sys_c.to_discrete(T, method='backward_diff')

# 方法3: 脉冲不变法
sys_d_impulse = sys_c.to_discrete(T, method='impulse')

# 方法4: 前向差分
sys_d_zoh = sys_c.to_discrete(T, method='zoh')

print("双线性变换:", sys_d_tustin)
print("差分方程系数 b:", sys_d_tustin.num)
print("差分方程系数 a:", sys_d_tustin.den)
```

### 3.4 手动双线性变换示例

```python
# 手动实现双线性变换: G(s) = ωn²/(s² + 2ζωn·s + ωn²)
# 令 K = 2/T

def bilinear_biquad(wn, zeta, T):
    """二阶系统双线性变换"""
    K = 2.0 / T
    K2 = K * K
    
    # 连续: wn^2 / (s^2 + 2*zeta*wn*s + wn^2)
    # 替换 s = K*(z-1)/(z+1):
    # 分母 = K2*(z-1)^2 + 2*zeta*wn*K*(z-1)*(z+1) + wn^2*(z+1)^2
    
    a0 = K2 + 2*zeta*wn*K + wn**2
    a1 = 2*(wn**2 - K2)
    a2 = K2 - 2*zeta*wn*K + wn**2
    
    b0 = wn**2
    b1 = 2*wn**2
    b2 = wn**2
    
    # 归一化
    b = [b0/a0, b1/a0, b2/a0]
    a = [1, a1/a0, a2/a0]
    return b, a

# 示例: wn=2*pi*100, zeta=0.707, T=1/10000
b, a = bilinear_biquad(2*np.pi*100, 0.707, 1/10000)
print(f"b = {b}")
print(f"a = {a}")
```

## 4. 采样率选择

### 4.1 奈奎斯特-香农定理

$$
f_s > 2 f_{max}
$$

采样频率必须大于信号最高频率的 2 倍，否则发生**混叠(Aliasing)**。

### 4.2 工程采样率选择准则

| 应用场景 | 最低采样率 | 推荐采样率 | 说明 |
|---------|-----------|-----------|------|
| 控制系统 | 10× 闭环带宽 | 20~50× 闭环带宽 | 保证相位裕度 |
| 音频信号 | 2× 20kHz | 44.1kHz/48kHz | 标准音频采样率 |
| 电力测量 | 2× 基波频率 | 64~256点/周波 | 保证谐波分析精度 |
| 电机控制 | 10× 电频率 | PWM频率同频 | 常见10~20kHz |
| 温度/压力 | 2× 变化频率 | 1~100Hz | 慢过程 |
| ADC前端 | — | 根据信号带宽 | 配合抗混叠滤波器 |

### 4.3 控制系统采样率经验公式

```
fs ≥ (5~10) × f_bw        # f_bw: 闭环带宽
fs ≥ (10~20) × f_natural   # f_natural: 系统自然频率
fs ≥ (20~50) × f_damping   # 针对高精度伺服
```

### 4.4 过采样与欠采样

**过采样(Oversampling)**：
- 采样率远高于奈奎斯特率
- 优点：降低对抗混叠滤波器的要求、提高信噪比、可通过抽取提高有效分辨率
- 典型：Σ-Δ ADC 过采样率 64~256 倍

**欠采样(Undersampling/带通采样)**：
- 对带通信号，采样率可低于 2× 最高频率
- 条件：$$ f_s > 2B $$ 且 $$ \frac{2f_H}{n} \leq f_s \leq \frac{2f_L}{n-1} $$
- 其中 B 为带宽，f_H/f_L 为上下边频

## 5. 混叠(Aliasing)

### 5.1 混叠原理

当信号频率 f > fs/2 时，高频分量会"折叠"到低频：

$$
f_{alias} = |f - n \cdot f_s| \quad (n \text{ 为使 } f_{alias} < f_s/2 \text{ 的整数})
$$

### 5.2 混叠示例

```
信号频率: 1.2 kHz
采样频率: 1.0 kHz
混叠频率: |1200 - 1000| = 200 Hz  ← 在采样数据中表现为 200Hz
```

### 5.3 抗混叠滤波器设计

**要求**：在 fs/2 处衰减 ≥ ADC 有效位数 × 6dB

| ADC 分辨率 | fs/2 处最低衰减 | 滤波器阶数(巴特沃斯, fs/10过渡带) |
|-----------|---------------|----------------------------------|
| 8 bit | 48 dB | 2~3 阶 |
| 10 bit | 60 dB | 3~4 阶 |
| 12 bit | 72 dB | 4~5 阶 |
| 16 bit | 96 dB | 5~6 阶 |

**典型抗混叠方案**：

```
信号 → [RC低通 fc=信号带宽] → [有源滤波器 fc=信号带宽, 4~6阶] → ADC
```

### 5.4 抗混叠滤波器设计示例

```python
# 4阶巴特沃斯抗混叠滤波器
# 信号带宽: 1kHz, 采样率: 10kHz
from scipy import signal

fs = 10000
fc = 1000  # 截止频率 = 信号带宽
order = 4

# 设计模拟滤波器
b, a = signal.butter(order, fc, btype='low', analog=True)

# 验证衰减
f_test = fs / 2  # 奈奎斯特频率
w, h = signal.freqs(b, a, [2*np.pi*f_test])
attenuation_dB = -20*np.log10(np.abs(h[0]))
print(f"在 fs/2 = {f_test}Hz 处衰减: {attenuation_dB:.1f} dB")
# 4阶巴特沃斯在 5kHz (5×fc) 处约衰减 48dB → 适合 8~10bit ADC
```

## 6. 量化(Quantization)

### 6.1 量化模型

```
模拟值 x → [量化器 Q(·)] → 数字值 x_q
量化误差 e = x_q - x,  |e| ≤ Δ/2
其中 Δ = Vref / 2^N (LSB)
```

### 6.2 量化噪声

假设量化误差均匀分布，量化噪声功率：

$$
\sigma_e^2 = \frac{\Delta^2}{12}
$$

**信噪比(SNR)**：

$$
SNR = 6.02N + 1.76 \text{ dB} \quad (\text{满量程正弦波})
$$

| 位数 N | SNR (dB) | 应用 |
|--------|----------|------|
| 8 | 49.9 | 简单控制 |
| 10 | 62.0 | 一般测量 |
| 12 | 74.0 | 精密控制 |
| 16 | 98.1 | 高精度测量 |
| 24 | 146.2 | 音频/仪器 |

### 6.3 量化对控制的影响

**极限环(Limit Cycle)**：由于量化，系统可能在两个相邻值之间振荡，无法收敛到稳态。

**解决方法**：
1. 增加 ADC 分辨率
2. 在控制器中加入**死区(Dead Zone)**：当 |e| < 1 LSB 时输出保持
3. 抖动(Dithering)：添加小幅随机噪声，打破极限环
4. 使用更高精度内部计算（32位定点/浮点）

### 6.4 定点数实现

```c
// Q15 定点数: 1位符号 + 15位小数, 范围 [-1, 1)
typedef int16_t q15_t;

#define Q15_SCALE 32768
#define FLOAT_TO_Q15(x) ((q15_t)((x) * Q15_SCALE))
#define Q15_TO_FLOAT(x) ((float)(x) / Q15_SCALE)

// Q15 乘法
q15_t q15_mul(q15_t a, q15_t b) {
    return (q15_t)(((int32_t)a * b) >> 15);
}

// PID 控制器 (Q15)
typedef struct {
    q15_t Kp, Ki, Kd;
    q15_t integral;
    q15_t prev_error;
    q15_t out_min, out_max;
} pid_q15_t;

q15_t pid_update(pid_q15_t *pid, q15_t error) {
    // 积分
    pid->integral += q15_mul(pid->Ki, error);
    
    // 微分
    q15_t derivative = error - pid->prev_error;
    q15_t d_term = q15_mul(pid->Kd, derivative);
    
    // 比例
    q15_t p_term = q15_mul(pid->Kp, error);
    
    // 输出
    int32_t output = (int32_t)p_term + pid->integral + (int32_t)d_term;
    
    // 限幅
    if (output > pid->out_max) output = pid->out_max;
    if (output < pid->out_min) output = pid->out_min;
    
    pid->prev_error = error;
    return (q15_t)output;
}
```

## 7. 数字控制器实现

### 7.1 差分方程实现

数字控制器 H(z) 的一般形式：

$$
H(z) = \frac{b_0 + b_1 z^{-1} + ... + b_m z^{-m}}{1 + a_1 z^{-1} + ... + a_n z^{-n}}
$$

对应差分方程：

$$
y(k) = \sum_{i=0}^{m} b_i \cdot x(k-i) - \sum_{j=1}^{n} a_j \cdot y(k-j)
$$

### 7.2 Direct Form I 实现

```c
// 二阶 IIR 滤波器 (Direct Form I)
typedef struct {
    float b[3];  // 分子系数
    float a[3];  // 分母系数 (a[0]=1)
    float x[3];  // 输入延迟
    float y[3];  // 输出延迟
} iir_df1_t;

float iir_update(iir_df1_t *f, float input) {
    // 移位
    f->x[2] = f->x[1]; f->x[1] = f->x[0]; f->x[0] = input;
    f->y[2] = f->y[1]; f->y[1] = f->y[0];
    
    // 计算
    f->y[0] = f->b[0]*f->x[0] + f->b[1]*f->x[1] + f->b[2]*f->x[2]
              - f->a[1]*f->y[1] - f->a[2]*f->y[2];
    
    return f->y[0];
}
```

### 7.3 Direct Form II (转置) 实现（推荐）

```c
// 二阶 IIR 滤波器 (Transposed Direct Form II) - 内存效率更高
typedef struct {
    float b[3];
    float a[3];
    float w[2];  // 内部状态
} iir_tdf2_t;

float iir_update_tdf2(iir_tdf2_t *f, float input) {
    float w0 = input - f->a[1]*f->w[0] - f->a[2]*f->w[1];
    float output = f->b[0]*w0 + f->b[1]*f->w[0] + f->b[2]*f->w[1];
    f->w[1] = f->w[0];
    f->w[0] = w0;
    return output;
}
```

### 7.4 级联二阶节 (SOS) 实现高阶滤波器

```python
# Python: 将高阶滤波器分解为级联二阶节
from scipy import signal
import numpy as np

# 设计 8 阶巴特沃斯低通滤波器
sos = signal.butter(8, 1000, btype='low', fs=10000, output='sos')

# 每一行 sos = [b0, b1, b2, a0, a1, a2]
print(f"共 {sos.shape[0]} 个二阶节")

# 级联实现
def sos_filter(sos, x):
    """级联二阶节滤波"""
    y = x.copy()
    for section in sos:
        b = section[:3]
        a = section[3:]
        y = signal.lfilter(b, a, y)
    return y
```

## 8. 数字滤波器设计速查

### 8.1 IIR 滤波器（无限脉冲响应）

| 类型 | 特点 | 阶数要求 | 相位 |
|------|------|---------|------|
| Butterworth | 最大平坦幅频 | 中 | 非线性 |
| Chebyshev I | 通带等纹波 | 低 | 非线性 |
| Chebyshev II | 阻带等纹波 | 低 | 非线性 |
| Elliptic | 通带+阻带等纹波 | 最低 | 非线性 |
| Bessel | 最平坦群延迟 | 高 | 近似线性 |

### 8.2 FIR 滤波器（有限脉冲响应）

| 设计方法 | 特点 | 阶数要求 |
|---------|------|---------|
| 窗函数法 | 简单，可线性相位 | 高 |
| 频率采样法 | 灵活 | 高 |
| Parks-McClellan (等纹波) | 最优 | 中 |
| 最小二乘法 | 最小化误差平方和 | 中 |

```python
# FIR 滤波器设计示例
from scipy import signal

# 方法1: 窗函数法

fs = 10000
b_fir = signal.firwin(N, fc, fs=fs)
```

## 9. 电赛实战：STM32 采样系统设计

### 9.1 ADC 配置模板

```c
// STM32F4 ADC DMA 采样配置 - 10kHz 采样率
#define SAMPLE_RATE  10000
#define SAMPLE_BUF   1024

volatile uint16_t adc_buf[SAMPLE_BUF];

void ADC_DMA_Init(void) {
    // ADC 配置: 12位, 连续扫描
    ADC1->CR1 = ADC_CR1_SCAN;
    ADC1->CR2 = ADC_CR2_CONT | ADC_CR2_DMA | ADC_CR2_EXTTRIG |
                ADC_CR2_EXTSEL_2 | ADC_CR2_EXTSEL_1 | ADC_CR2_EXTSEL_0; // TIM3 TRGO
    ADC1->SMPR2 = ADC_SMPR2_SMP0_2; // 84 cycles
    ADC1->SQR3 = 0; // CH0
    
    // DMA 配置
    DMA2_Stream0->PAR = (uint32_t)&ADC1->DR;
    DMA2_Stream0->M0AR = (uint32_t)adc_buf;
    DMA2_Stream0->NDTR = SAMPLE_BUF;
    DMA2_Stream0->CR = DMA_SxCR_CIRC | DMA_SxCR_MINC |
                       DMA_SxCR_PSIZE_0 | DMA_SxCR_MSIZE_0 |
                       DMA_SxCR_TCIE;
    
    // TIM3 触发: 10kHz
    RCC->APB1ENR |= RCC_APB1ENR_TIM3EN;
    TIM3->PSC = 83;        // 84MHz / 84 = 1MHz
    TIM3->ARR = 99;        // 1MHz / 100 = 10kHz
    TIM3->CR2 = TIM_CR2_MMS_1; // Update event as TRGO
    
    ADC1->CR2 |= ADC_CR2_ADON;
    TIM3->CR1 |= TIM_CR1_CEN;
}
```

### 9.2 软件抗混叠 + 数字滤波

```c
// 结合过采样 + 数字滤波
#define OVERSAMPLE  16  // 16倍过采样 → 提高 2位有效分辨率

uint16_t oversample_avg(uint16_t *buf, int n) {
    uint32_t sum = 0;
    for (int i = 0; i < n; i++) sum += buf[i];
    return (uint16_t)(sum / n);  // 或右移 log2(n) 位
}

// 简单一阶 IIR 低通: y[n] = α*x[n] + (1-α)*y[n-1]
typedef struct {
    float alpha;
    float y;
} iir_lpf_t;

float iir_lpf_update(iir_lpf_t *f, float x) {
    f->y = f->alpha * x + (1.0f - f->alpha) * f->y;
    return f->y;
}

// alpha 计算: alpha = 2π*fc*T / (2π*fc*T + 1)
// fc=100Hz, T=100us: alpha = 2*3.14*100*0.0001 / (2*3.14*100*0.0001 + 1) ≈ 0.059
```

## 10. 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 控制输出振荡 | 采样率过低/量化噪声 | 提高采样率/增加分辨率 |
| 相位滞后过大 | 采样延迟+ZOH延迟 | 提高采样率/前馈补偿 |
| 极限环振荡 | 量化死区 | 加抖动/提高分辨率 |
| 高频噪声放大 | 微分项对噪声敏感 | 低通滤波微分/用观测器替代 |
| 频谱混叠 | 缺少抗混叠滤波器 | 加模拟低通滤波器 |
| 控制器不稳定 | 离散化方法不合适 | 换用双线性变换/提高采样率 |
