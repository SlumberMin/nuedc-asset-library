# 模糊控制进阶指南

> nuedc-asset-library · 控制算法库  
> 适用场景：非线性系统、模型未知系统、多变量耦合、智能调节

---

## 1. 经典模糊控制回顾

### 1.1 基本结构

```
输入 → 模糊化 → 规则库 → 推理机 → 解模糊 → 输出
        ↑                              ↓
       量化因子                     比例因子

典型: 双输入(e, Δe) → 单输出(Δu)
```

### 1.2 局限性
- 规则依赖专家经验，难以自优化
- 固定参数不能适应工况变化
- 高维问题规则爆炸
- Type-1模糊集无法处理高不确定性

## 2. 自适应模糊控制

### 2.1 自调节量化/比例因子

```
基本思想: 根据系统状态在线调整模糊控制器的缩放参数

自适应律:
  K_e(t)  = K_e0  + ΔK_e(t)     (误差量化因子)
  K_de(t) = K_de0 + ΔK_de(t)    (误差变化率因子)
  K_u(t)  = K_u0  + ΔK_u(t)     (输出比例因子)
```

```c
// 自适应模糊控制器
typedef struct {
    float K_e;          // 误差量化因子
    float K_de;         // 误差变化率因子
    float K_u;          // 输出比例因子
    float alpha;        // 调整速率
    float e_prev;
    float de_prev;
} AdaptiveFuzzy_t;

void AdaptiveFuzzy_UpdateGains(AdaptiveFuzzy_t *ctrl, float e, float de) {
    // 规则: |e|大 → 增大K_e(快速响应), 减小K_u(防超调)
    //       |e|小 → 减小K_e, 增大K_u(提高精度)
    float abs_e = fabsf(e);
    float abs_de = fabsf(de);
    
    // 基于e的大小自适应
    ctrl->K_e = 1.0f + ctrl->alpha * abs_e;
    
    // 基于de的大小自适应
    ctrl->K_de = 0.5f + ctrl->alpha * abs_de;
    
    // 输出因子反向调节
    if (abs_e > 0.5f) {
        ctrl->K_u *= 0.98f;  // 误差大时减小输出
    } else {
        ctrl->K_u *= 1.02f;  // 误差小时增大输出
    }
    ctrl->K_u = constrain(ctrl->K_u, 0.1f, 10.0f);
}
```

### 2.2 基于Lyapunov的自适应模糊

```
目标: 用模糊系统逼近未知非线性函数，自适应律保证稳定性

系统: ẋ = f(x) + g(x)·u
模糊逼近: f̂(x) = θ^T · ξ(x)  (θ为参数向量, ξ为模糊基函数)

自适应律 (保证Lyapunov稳定):
  θ̇ = -γ·e·ξ(x)

控制律:
  u = [-f̂(x) + ẋ_d - k·e] / g(x)
```

```c
// Lyapunov自适应模糊
typedef struct {
    float *theta;       // 参数向量 [n_rules]
    float gamma;        // 自适应增益
    int n_rules;
    float k;            // 反馈增益
    float dt;
} LyapunovFuzzy_t;

float LyapunovFuzzy_Control(LyapunovFuzzy_t *ctrl, 
                             float e, float x[], float x_d_dot) {
    // 计算模糊基函数 ξ(x)
    float xi[MAX_RULES];
    ComputeFuzzyBasis(ctrl->n_rules, x, xi);
    
    // 模糊逼近 f̂(x)
    float f_hat = 0;
    for (int i = 0; i < ctrl->n_rules; i++) {
        f_hat += ctrl->theta[i] * xi[i];
    }
    
    // 控制律
    float u = (-f_hat + x_d_dot - ctrl->k * e);
    
    // 自适应律
    for (int i = 0; i < ctrl->n_rules; i++) {
        ctrl->theta[i] -= ctrl->gamma * e * xi[i] * ctrl->dt;
    }
    
    return u;
}
```

## 3. 神经模糊系统（ANFIS）

### 3.1 结构

```
Layer 1: 模糊化层 — 隶属函数参数 (前提参数)
Layer 2: 规则层 — 计算规则触发强度 (T-norm)
Layer 3: 归一化层 — 归一化触发强度
Layer 4: 结论层 — 线性组合 (结论参数)
Layer 5: 输出层 — 求和得到输出

本质: Takagi-Sugeno模糊系统 + 神经网络学习
```

### 3.2 T-S模糊模型

```
规则 i: IF x₁ is A₁ⁱ AND x₂ is A₂ⁱ THEN yᵢ = a₀ⁱ + a₁ⁱ·x₁ + a₂ⁱ·x₂

输出: y = Σ(wᵢ·yᵢ) / Σ(wᵢ)
  wᵢ = μ_A₁ⁱ(x₁) · μ_A₂ⁱ(x₂)  (规则权重)

优势: 局部线性，可用线性控制理论分析
```

```c
// 简化ANFIS推理 (2输入, n_rules条规则)
typedef struct {
    float center[2][7];   // 隶属函数中心
    float sigma[2][7];    // 隶属函数宽度
    float consequent[7][3]; // 结论参数 [a0, a1, a2] per rule
    int n_rules;
} ANFIS_t;

float ANFIS_Predict(ANFIS_t *model, float x1, float x2) {
    float w_sum = 0, wy_sum = 0;
    
    for (int i = 0; i < model->n_rules; i++) {
        // 高斯隶属函数
        float mu1 = expf(-powf(x1 - model->center[0][i], 2) 
                         / (2 * model->sigma[0][i] * model->sigma[0][i]));
        float mu2 = expf(-powf(x2 - model->center[1][i], 2) 
                         / (2 * model->sigma[1][i] * model->sigma[1][i]));
        
        // 触发强度
        float w = mu1 * mu2;
        
        // 结论 (线性)
        float y_i = model->consequent[i][0] 
                   + model->consequent[i][1] * x1 
                   + model->consequent[i][2] * x2;
        
        w_sum += w;
        wy_sum += w * y_i;
    }
    
    return (w_sum > 0) ? wy_sum / w_sum : 0;
}
```

### 3.3 在线学习 (混合学习)

```
阶段1 (前向): 固定前提参数，用最小二乘更新结论参数
阶段2 (反向): 固定结论参数，用梯度下降更新前提参数

前提参数更新:
  center_new = center_old - η·∂E/∂center
  sigma_new  = sigma_old  - η·∂E/∂sigma

适合离线训练后部署到嵌入式
```

## 4. 分层模糊控制

### 4.1 问题：规则爆炸

```
经典模糊: n个输入, 每个m个模糊集 → m^n 条规则
  2输入×7模糊集 = 49规则 (可接受)
  4输入×7模糊集 = 2401规则 (不可接受!)
```

### 4.2 分层结构

```
方案: 将高维问题分解为多个低维子问题

例: 4输入(e, de, ∫e, dde) → 2级分层

Level 1:
  模糊控制器1: (e, de) → z₁
  模糊控制器2: (∫e, dde) → z₂

Level 2:
  模糊控制器3: (z₁, z₂) → u

规则数: 49 + 49 + 49 = 147 (vs 2401)
```

```c
// 分层模糊控制器
typedef struct {
    FuzzyController_t level1_ctrl[2];  // 两个子控制器
    FuzzyController_t level2_ctrl;     // 上层融合控制器
} HierarchicalFuzzy_t;

float HierarchicalFuzzy_Calculate(HierarchicalFuzzy_t *ctrl,
                                   float e, float de, float ie, float dde) {
    // Level 1
    float z1 = Fuzzy_Calculate(&ctrl->level1_ctrl[0], e, de);
    float z2 = Fuzzy_Calculate(&ctrl->level1_ctrl[1], ie, dde);
    
    // Level 2
    float u = Fuzzy_Calculate(&ctrl->level2_ctrl, z1, z2);
    
    return u;
}
```

### 4.3 设计原则
- 按物理意义分组（快变量/慢变量）
- 同级子控制器独立设计
- 上层控制器做协调融合
- 可递归分层（3级以上输入）

## 5. Type-2 模糊控制

### 5.1 Type-1 vs Type-2

```
Type-1: 隶属函数 μ(x) ∈ [0,1] — 一个确定值
Type-2: 隶属函数 μ(x) ∈ [0,1] — 本身是一个模糊集

Type-2隶属函数: μ̃(x) = ∫∫ f_x(u)/u  (u ∈ J_x ⊆ [0,1])
  J_x: x的主隶属度 (footprint of uncertainty, FOU)
```

### 5.2 区间Type-2 (IT2) — 实用简化

```
只需存储上隶属函数和下隶属函数:
  μ̄(x): 上隶属函数 (upper MF)
  μ_(x): 下隶属函数 (lower MF)
  FOU = {(x,u) | μ_(x) ≤ u ≤ μ̄(x)}

优点: 计算量与Type-1相当，但能处理更大不确定性
```

```c
// 区间Type-2模糊推理
typedef struct {
    float center;       // 隶属函数中心
    float sigma_upper;  // 上隶属函数宽度 (较大)
    float sigma_lower;  // 下隶属函数宽度 (较小)
} IT2_MF_t;

// 计算IT2隶属度区间
typedef struct {
    float upper;
    float lower;
} IT2_Membership_t;

IT2_Membership_t IT2_Gaussian(IT2_MF_t *mf, float x) {
    IT2_Membership_t mu;
    float diff = x - mf->center;
    mu.upper = expf(-0.5f * (diff * diff) / (mf->sigma_upper * mf->sigma_upper));
    mu.lower = expf(-0.5f * (diff * diff) / (mf->sigma_lower * mf->sigma_lower));
    return mu;
}

// 简化型降阶 (Karnik-Mendel算法近似)
float IT2_Defuzzify(IT2_MF_t mfs[], float w_upper[], float w_lower[], 
                     int n_rules, float y_rules[]) {
    // 简化的中心集降阶
    float num = 0, den = 0;
    for (int i = 0; i < n_rules; i++) {
        float w_avg = (w_upper[i] + w_lower[i]) / 2.0f;
        num += w_avg * y_rules[i];
        den += w_avg;
    }
    return (den > 0) ? num / den : 0;
}
```

### 5.3 Type-2在电赛中的适用场景

| 场景 | 不确定性来源 | Type-2优势 |
|------|-------------|-----------|
| 传感器噪声大 | 测量不确定性 | FOU覆盖噪声范围 |
| 负载变化剧烈 | 模型参数不确定 | 上下MF适应不同工况 |
| 环境温度变化 | 材料参数漂移 | 鲁棒性更强 |
| 多传感器融合 | 信息冲突 | 处理矛盾信息 |

## 6. 模糊PID参数自整定

### 6.1 结构

```
输入: e, Δe
输出: ΔKp, ΔKi, ΔKd

在线调整:
  Kp = Kp0 + ΔKp
  Ki = Ki0 + ΔKi
  Kd = Kd0 + ΔKd
```

### 6.2 典型规则表 (49条)

```
ΔKp规则表 (e × Δe → ΔKp):

         Δe: NB   NM   NS   ZO   PS   PM   PB
e: NB  | PB   PB   PM   PM   PS   ZO   ZO  |
   NM  | PB   PB   PM   PS   PS   ZO   NS  |
   NS  | PM   PM   PM   PS   ZO   NS   NS  |
   ZO  | PM   PM   PS   ZO   NS   NM   NM  |
   PS  | PS   PS   ZO   NS   NS   NM   NM  |
   PM  | PS   ZO   NS   NM   NM   NM   NB  |
   PB  | ZO   ZO   NM   NM   NM   NB   NB  |

ΔKi规则表:
         Δe: NB   NM   NS   ZO   PS   PM   PB
e: NB  | NB   NB   NM   NM   NS   ZO   ZO  |
   NM  | NB   NB   NM   NS   NS   ZO   ZO  |
   NS  | NB   NM   NS   NS   ZO   PS   PS  |
   ZO  | NM   NM   NS   ZO   PS   PM   PM  |
   PS  | NM   NS   ZO   PS   PS   PM   PB  |
   PM  | ZO   ZO   PS   PS   PM   PB   PB  |
   PB  | ZO   ZO   PS   PM   PM   PB   PB  |

ΔKd规则表:
         Δe: NB   NM   NS   ZO   PS   PM   PB
e: NB  | PS   NS   NB   NB   NB   NM   PS  |
   NM  | PS   NS   NB   NM   NM   NS   ZO  |
   NS  | ZO   NS   NM   NM   NS   NS   ZO  |
   ZO  | ZO   NS   NS   NS   NS   NS   ZO  |
   PS  | ZO   ZO   ZO   ZO   ZO   ZO   ZO  |
   PM  | PB   NS   PS   PS   PS   PS   PB  |
   PB  | PB   PM   PM   PM   PS   PS   PB  |
```

### 6.3 完整实现

```c
#define N_MF 7  // NB NM NS ZO PS PM PB

typedef struct {
    float Kp0, Ki0, Kd0;    // PID初始参数
    float Ke, Kde, Ku;      // 量化/比例因子
    float rule_Kp[N_MF][N_MF];
    float rule_Ki[N_MF][N_MF];
    float rule_Kd[N_MF][N_MF];
    // 隶属函数参数
    float mf_centers[N_MF]; // {-6,-4,-2,0,2,4,6}
    float mf_sigma;
    // 状态
    float e_prev;
    float integral;
    float dt;
    float out_min, out_max;
} FuzzyPID_t;

void FuzzyPID_Init(FuzzyPID_t *pid) {
    pid->mf_centers[0] = -6; pid->mf_centers[1] = -4;
    pid->mf_centers[2] = -2; pid->mf_centers[3] = 0;
    pid->mf_centers[4] = 2;  pid->mf_centers[5] = 4;
    pid->mf_centers[6] = 6;
    pid->mf_sigma = 1.5f;
    // 初始化规则表 (使用上表)
    // ... (省略，可从数组初始化)
}

float FuzzyPID_Calculate(FuzzyPID_t *pid, float setpoint, float pv) {
    float e = setpoint - pv;
    float de = (e - pid->e_prev) / pid->dt;
    pid->e_prev = e;
    pid->integral += e * pid->dt;
    
    // 量化
    float e_n = e * pid->Ke;
    float de_n = de * pid->Kde;
    
    // 模糊推理 (简化的加权平均)
    float dKp = 0, dKi = 0, dKd = 0;
    float w_sum = 0;
    
    for (int i = 0; i < N_MF; i++) {
        for (int j = 0; j < N_MF; j++) {
            float mu_e  = gaussian_mf(e_n,  pid->mf_centers[i], pid->mf_sigma);
            float mu_de = gaussian_mf(de_n, pid->mf_centers[j], pid->mf_sigma);
            float w = mu_e * mu_de;
            
            dKp += w * pid->rule_Kp[i][j];
            dKi += w * pid->rule_Ki[i][j];
            dKd += w * pid->rule_Kd[i][j];
            w_sum += w;
        }
    }
    
    if (w_sum > 0) {
        dKp /= w_sum; dKi /= w_sum; dKd /= w_sum;
    }
    
    // 更新PID参数
    float Kp = pid->Kp0 + dKp * pid->Ku;
    float Ki = pid->Ki0 + dKi * pid->Ku;
    float Kd = pid->Kd0 + dKd * pid->Ku;
    if (Kp < 0) Kp = 0;
    if (Ki < 0) Ki = 0;
    if (Kd < 0) Kd = 0;
    
    // PID输出
    float u = Kp * e + Ki * pid->integral + Kd * de;
    return constrain(u, pid->out_min, pid->out_max);
}
```

## 7. 模糊控制在电赛中的应用

### 7.1 温度控制

```
输入: 温度误差e, 温度变化率Δe
输出: PWM占空比增量Δu

隶属函数: 三角形 (计算快)
  e: [-20, -10, -5, 0, 5, 10, 20]°C
  Δe: [-5, -2.5, -1, 0, 1, 2.5, 5]°C/s

推荐: 模糊PID自整定
```

### 7.2 平衡车

```
输入: 角度误差e_θ, 角速度ω, 位置误差e_x, 速度v
方案: 分层模糊
  Level 1-1: (e_θ, ω) → 倾斜力矩
  Level 1-2: (e_x, v) → 位置修正
  Level 2: 融合 → 电机PWM
```

### 7.3 电源输出调节

```
输入: 电压误差e_V, 负载电流I_load
输出: 占空比调整量

Type-2模糊适用: 负载突变时不确定性大
```

## 8. 调参技巧与对比

### 8.1 不同模糊控制方法对比

| 方法 | 复杂度 | 自适应性 | 鲁棒性 | 适用场景 |
|------|--------|---------|--------|---------|
| 经典Mamdani | 低 | 无 | 中 | 简单系统 |
| T-S模糊 | 中 | 弱 | 中 | 可线性化系统 |
| 自适应模糊 | 中 | 强 | 强 | 参数变化系统 |
| 神经模糊 | 高 | 最强 | 强 | 需要数据训练 |
| 分层模糊 | 中 | 中 | 中 | 高维系统 |
| Type-2模糊 | 高 | 中 | 最强 | 高不确定性 |

### 8.2 调参流程

```
1. 确定输入输出变量及论域范围
2. 设计隶属函数 (通常3~7个，三角形或高斯)
3. 建立规则表 (从专家经验或PID经验出发)
4. 调整量化因子/比例因子 (最重要!)
5. 微调规则和隶属函数
6. 测试边界条件和抗扰性能
```
