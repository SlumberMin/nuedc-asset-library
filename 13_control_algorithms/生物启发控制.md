# 生物启发控制算法

## 概述

生物启发控制算法模拟自然界生物的群体行为、进化机制和神经结构，用于解决复杂优化和控制问题。在电赛中，这类算法常用于参数优化、路径规划、自适应控制等场景。

---

## 1 遗传算法（Genetic Algorithm, GA）

### 1.1 基本原理

模拟达尔文进化论的"自然选择、适者生存"机制。

```
种群初始化 → 适应度评估 → 选择 → 交叉 → 变异 → 新一代
    ↑                                              |
    └──────────── 终止条件未满足 ──────────────────┘
```

### 1.2 核心操作

#### 编码方式

| 编码 | 适用场景 | 示例 |
|------|----------|------|
| 二进制编码 | 离散优化 | 10110101 |
| 实数编码 | 连续参数优化 | [1.5, 3.2, 0.8] |
| 排列编码 | 组合优化 | [3,1,4,1,5] |
| 树编码 | 程序进化 | 表达式树 |

#### 选择算子

| 方法 | 原理 | 特点 |
|------|------|------|
| 轮盘赌选择 | 概率与适应度成正比 | 简单，易早熟 |
| 锦标赛选择 | 随机选k个取最优 | 可调压力 |
| 排序选择 | 按排名分配概率 | 避免超级个体 |
| 精英保留 | 最优直接复制 | 保证收敛 |

#### 交叉算子

**单点交叉（二进制）：**
```
父代1:  1011|0101
父代2:  0110|1100
子代1:  1011|1100
子代2:  0110|0101
```

**模拟二进制交叉SBX（实数编码）：**
```
子代 = 0.5×[(1+β)×父代1 + (1-β)×父代2]
β = (2u)^(1/(η+1))  若 u ≤ 0.5
β = (1/(2(1-u)))^(1/(η+1))  若 u > 0.5
```

#### 变异算子

**均匀变异：** 随机替换某基因值

**高斯变异（实数编码）：**
```
x' = x + N(0, σ)
σ = σ₀ × (1 - t/T)  // 自适应变异强度
```

### 1.3 在控制中的应用

#### PID参数优化

**编码：** [Kp, Ki, Kd] → 实数编码

**适应度函数：**
```python
def fitness(individual):
    Kp, Ki, Kd = individual
    # 仿真系统响应
    t, y = simulate_pid(Kp, Ki, Kd)
    # ITAE指标（越小越好）
    itae = np.sum(t * np.abs(reference - y)) * dt
    # 超调量惩罚
    overshoot = max(0, (max(y) - reference) / reference - 0.05)
    return 1.0 / (itae + 10 * overshoot + 1e-6)
```

**典型参数设置：**
| 参数 | 推荐值 |
|------|--------|
| 种群大小 | 50-200 |
| 交叉概率 | 0.7-0.9 |
| 变异概率 | 0.01-0.1 |
| 迭代代数 | 100-500 |

### 1.4 完整实现

```python
import numpy as np

class GeneticAlgorithm:
    def __init__(self, fitness_func, bounds, pop_size=100, 
                 pc=0.8, pm=0.1, max_gen=200):
        self.fitness_func = fitness_func
        self.bounds = np.array(bounds)  # [(lb,ub), ...]
        self.dim = len(bounds)
        self.pop_size = pop_size
        self.pc = pc  # 交叉概率
        self.pm = pm  # 变异概率
        self.max_gen = max_gen
    
    def initialize(self):
        lb, ub = self.bounds[:, 0], self.bounds[:, 1]
        return lb + (ub - lb) * np.random.rand(self.pop_size, self.dim)
    
    def select(self, pop, fitness):
        # 轮盘赌选择
        prob = fitness / fitness.sum()
        idx = np.random.choice(len(pop), size=len(pop), p=prob)
        return pop[idx].copy()
    
    def crossover(self, parent1, parent2):
        # SBX交叉
        eta = 20
        u = np.random.rand(self.dim)
        beta = np.where(u <= 0.5, 
                       (2*u)**(1/(eta+1)),
                       (1/(2*(1-u)))**(1/(eta+1)))
        child1 = 0.5 * ((1+beta)*parent1 + (1-beta)*parent2)
        child2 = 0.5 * ((1-beta)*parent1 + (1+beta)*parent2)
        return child1, child2
    
    def mutate(self, individual):
        # 高斯变异
        for i in range(self.dim):
            if np.random.rand() < self.pm:
                sigma = (self.bounds[i, 1] - self.bounds[i, 0]) * 0.1
                individual[i] += np.random.normal(0, sigma)
                individual[i] = np.clip(individual[i], 
                                       self.bounds[i, 0], 
                                       self.bounds[i, 1])
        return individual
    
    def optimize(self):
        pop = self.initialize()
        best_solution = None
        best_fitness = -np.inf
        
        for gen in range(self.max_gen):
            fitness = np.array([self.fitness_func(ind) for ind in pop])
            
            # 更新最优
            max_idx = fitness.argmax()
            if fitness[max_idx] > best_fitness:
                best_fitness = fitness[max_idx]
                best_solution = pop[max_idx].copy()
            
            # 选择
            pop = self.select(pop, fitness)
            
            # 交叉变异
            new_pop = []
            for i in range(0, self.pop_size, 2):
                p1, p2 = pop[i], pop[i+1]
                if np.random.rand() < self.pc:
                    c1, c2 = self.crossover(p1, p2)
                else:
                    c1, c2 = p1.copy(), p2.copy()
                new_pop.extend([self.mutate(c1), self.mutate(c2)])
            pop = np.array(new_pop)
            
            if gen % 20 == 0:
                print(f"Gen {gen}: best fitness = {best_fitness:.6f}")
        
        return best_solution, best_fitness
```

## 2 粒子群优化（PSO）

### 2.1 基本原理

模拟鸟群觅食行为：每个粒子代表一个候选解，通过跟踪个体最优和全局最优来更新位置。

### 2.2 算法流程

```
初始化粒子群（位置+速度）
循环：
    对每个粒子计算适应度
    更新个体最优 pBest
    更新全局最优 gBest
    更新速度和位置
直到满足终止条件
```

### 2.3 核心公式

**速度更新：**
$$v_i^{t+1} = w \cdot v_i^t + c_1 r_1 (pBest_i - x_i^t) + c_2 r_2 (gBest - x_i^t)$$

**位置更新：**
$$x_i^{t+1} = x_i^t + v_i^{t+1}$$

**参数说明：**
| 参数 | 含义 | 推荐值 |
|------|------|--------|
| w | 惯性权重 | 0.4~0.9（线性递减） |
| c₁ | 认知系数 | 1.5~2.0 |
| c₂ | 社会系数 | 1.5~2.0 |
| r₁, r₂ | 随机数 | [0,1]均匀分布 |

### 2.4 改进变体

| 变体 | 改进点 | 效果 |
|------|--------|------|
| 自适应PSO | w随迭代线性递减 | 平衡探索与开发 |
| 混沌PSO | 混沌映射初始化 | 增加多样性 |
| 离散PSO | 离散化位置更新 | 解决组合优化 |
| 多目标PSO | Pareto前沿 | 多目标优化 |

### 2.5 完整实现

```python
import numpy as np

class PSO:
    def __init__(self, fitness_func, bounds, n_particles=50,
                 w=0.7, c1=1.5, c2=1.5, max_iter=200):
        self.fitness_func = fitness_func
        self.bounds = np.array(bounds)
        self.dim = len(bounds)
        self.n_particles = n_particles
        self.w = w
        self.c1 = c1
        self.c2 = c2
        self.max_iter = max_iter
    
    def optimize(self):
        lb, ub = self.bounds[:, 0], self.bounds[:, 1]
        
        # 初始化位置和速度
        pos = lb + (ub - lb) * np.random.rand(self.n_particles, self.dim)
        vel = (ub - lb) * (np.random.rand(self.n_particles, self.dim) - 0.5)
        
        pBest_pos = pos.copy()
        pBest_fit = np.array([self.fitness_func(p) for p in pos])
        gBest_idx = pBest_fit.argmax()
        gBest_pos = pBest_pos[gBest_idx].copy()
        gBest_fit = pBest_fit[gBest_idx]
        
        for t in range(self.max_iter):
            # 线性递减惯性权重
            w = self.w - (self.w - 0.4) * t / self.max_iter
            
            for i in range(self.n_particles):
                r1, r2 = np.random.rand(self.dim), np.random.rand(self.dim)
                
                # 更新速度
                vel[i] = (w * vel[i] + 
                         self.c1 * r1 * (pBest_pos[i] - pos[i]) +
                         self.c2 * r2 * (gBest_pos - pos[i]))
                
                # 更新位置
                pos[i] += vel[i]
                pos[i] = np.clip(pos[i], lb, ub)
                
                # 评估适应度
                fit = self.fitness_func(pos[i])
                
                # 更新pBest
                if fit > pBest_fit[i]:
                    pBest_fit[i] = fit
                    pBest_pos[i] = pos[i].copy()
                    
                    # 更新gBest
                    if fit > gBest_fit:
                        gBest_fit = fit
                        gBest_pos = pos[i].copy()
            
            if t % 20 == 0:
                print(f"Iter {t}: gBest fitness = {gBest_fit:.6f}")
        
        return gBest_pos, gBest_fit
```

## 3 蚁群算法（Ant Colony Optimization, ACO）

### 3.1 基本原理

模拟蚂蚁通过信息素寻找最短路径的集体行为。

### 3.2 核心机制

**信息素更新：**
$$\tau_{ij}(t+1) = (1-\rho)\tau_{ij}(t) + \sum_{k=1}^{m}\Delta\tau_{ij}^k$$

**转移概率：**
$$p_{ij}^k = \frac{[\tau_{ij}]^\alpha \cdot [\eta_{ij}]^\beta}{\sum_{l \in allowed} [\tau_{il}]^\alpha \cdot [\eta_{il}]^\beta}$$

其中：
- τ_ij：信息素浓度
- η_ij = 1/d_ij：启发式信息（距离倒数）
- α：信息素重要程度
- β：启发式信息重要程度
- ρ：信息素蒸发率

### 3.3 TSP问题求解

```python
import numpy as np

class ACO_TSP:
    def __init__(self, distances, n_ants=20, alpha=1.0, beta=2.0,
                 rho=0.5, Q=100, max_iter=100):
        self.distances = distances
        self.n = len(distances)
        self.n_ants = n_ants
        self.alpha = alpha
        self.beta = beta
        self.rho = rho
        self.Q = Q
        self.max_iter = max_iter
        self.tau = np.ones((self.n, self.n)) * 0.1  # 信息素
        self.eta = 1.0 / (distances + np.eye(self.n) * 1e-10)  # 启发信息
    
    def solve(self):
        best_path = None
        best_length = np.inf
        
        for t in range(self.max_iter):
            paths = []
            lengths = []
            
            for ant in range(self.n_ants):
                path = self._construct_path()
                length = self._path_length(path)
                paths.append(path)
                lengths.append(length)
                
                if length < best_length:
                    best_length = length
                    best_path = path.copy()
            
            self._update_pheromone(paths, lengths)
            
            if t % 10 == 0:
                print(f"Iter {t}: best length = {best_length:.2f}")
        
        return best_path, best_length
    
    def _construct_path(self):
        start = np.random.randint(self.n)
        path = [start]
        visited = {start}
        
        for _ in range(self.n - 1):
            current = path[-1]
            prob = np.zeros(self.n)
            for j in range(self.n):
                if j not in visited:
                    prob[j] = (self.tau[current,j]**self.alpha * 
                              self.eta[current,j]**self.beta)
            prob /= prob.sum()
            next_city = np.random.choice(self.n, p=prob)
            path.append(next_city)
            visited.add(next_city)
        
        return path
    
    def _path_length(self, path):
        length = 0
        for i in range(len(path) - 1):
            length += self.distances[path[i], path[i+1]]
        length += self.distances[path[-1], path[0]]  # 回到起点
        return length
    
    def _update_pheromone(self, paths, lengths):
        self.tau *= (1 - self.rho)  # 蒸发
        for path, length in zip(paths, lengths):
            for i in range(len(path) - 1):
                self.tau[path[i], path[i+1]] += self.Q / length
            self.tau[path[-1], path[0]] += self.Q / length
```

### 3.4 参数影响

| 参数 | 过大 | 过小 | 推荐值 |
|------|------|------|--------|
| α | 搜索停滞 | 忽略经验 | 1~2 |
| β | 贪婪搜索 | 随机搜索 | 2~5 |
| ρ | 遗忘太快 | 记忆冗余 | 0.1~0.5 |

## 4 神经网络在控制中的应用

### 4.1 神经网络控制结构

#### 直接逆控制

```
参考输入 → [神经网络控制器] → 控制信号 → [被控对象] → 输出
```

**训练：** 最小化 ||u - N⁻¹(y_d)||²

#### 神经网络自适应控制

```
参考输入 → [+] → [神经网络] → [+] → 被控对象 → 输出
           ↑-          ↑                  |
           |      [自适应律]              |
           └─────────────────────────────┘
```

#### 模型参考自适应控制（MRAC）

```
参考模型 → y_m
                ↘ (比较)
参考输入 → 神经网络控制器 → 被控对象 → y
                ↗ (误差驱动更新)
```

### 4.2 RBF神经网络控制

**径向基函数：**
$$\phi_j(x) = \exp\left(-\frac{||x-c_j||^2}{2\sigma_j^2}\right)$$

**网络输出：**
$$f(x) = \sum_{j=1}^{N} w_j \phi_j(x)$$

**控制律设计：**
```python
import numpy as np

class RBFNetwork:
    def __init__(self, n_input, n_hidden, lr=0.01):
        self.n_input = n_input
        self.n_hidden = n_hidden
        self.lr = lr
        
        # 初始化中心、宽度、权重
        self.centers = np.random.randn(n_hidden, n_input) * 0.5
        self.sigma = np.ones(n_hidden) * 1.0
        self.weights = np.random.randn(n_hidden) * 0.1
    
    def forward(self, x):
        # 计算RBF输出
        self.h = np.exp(-np.sum((x - self.centers)**2, axis=1) / 
                       (2 * self.sigma**2))
        return np.dot(self.weights, self.h)
    
    def train(self, x, y_target):
        y_pred = self.forward(x)
        error = y_target - y_pred
        
        # 更新权重
        self.weights += self.lr * error * self.h
        
        # 更新中心
        for j in range(self.n_hidden):
            self.centers[j] += (self.lr * error * self.weights[j] * 
                               self.h[j] * (x - self.centers[j]) / 
                               self.sigma[j]**2)
        
        return error
```

### 4.3 强化学习控制

#### Q-Learning 基础

$$Q(s,a) \leftarrow Q(s,a) + \alpha[r + \gamma \max_{a'} Q(s',a') - Q(s,a)]$$

**参数：**
- α：学习率
- γ：折扣因子
- ε-greedy：探索率

#### 用于控制系统的强化学习

```python
import numpy as np

class QLearningController:
    def __init__(self, n_states, n_actions, lr=0.1, gamma=0.95, epsilon=0.1):
        self.q_table = np.zeros((n_states, n_actions))
        self.lr = lr
        self.gamma = gamma
        self.epsilon = epsilon
    
    def choose_action(self, state):
        if np.random.rand() < self.epsilon:
            return np.random.randint(self.q_table.shape[1])
        return self.q_table[state].argmax()
    
    def learn(self, state, action, reward, next_state):
        current = self.q_table[state, action]
        target = reward + self.gamma * self.q_table[next_state].max()
        self.q_table[state, action] += self.lr * (target - current)
```

## 5 算法对比与选型

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| GA | 全局搜索强，易并行 | 收敛慢，参数敏感 | 参数优化、调度 |
| PSO | 简单高效，易实现 | 易早熟，连续空间 | 连续优化、PID调参 |
| ACO | 正反馈机制，并行性 | 收敛慢，存储大 | 路径规划、组合优化 |
| 神经网络 | 非线性逼近，自适应 | 训练数据需求大 | 复杂系统建模与控制 |
| 强化学习 | 无需模型，自学习 | 样本效率低 | 序列决策控制 |

## 6 电赛实战应用

### 6.1 倒立摆控制（PSO优化PID）

```python
def inverted_pendulum_pid(params):
    Kp, Ki, Kd = params
    # 倒立摆参数
    M, m, l, g = 1.0, 0.1, 0.5, 9.8
    
    # 仿真
    x, dx, theta, dtheta = 0, 0, 0.1, 0  # 初始偏角0.1rad
    dt = 0.01
    total_error = 0
    
    for t in range(500):
        # PID控制力
        error = -theta
        integral = max(-10, min(10, integral + error * dt))
        derivative = (error - prev_error) / dt
        F = Kp * error + Ki * integral + Kd * derivative
        prev_error = error
        
        # 状态更新（简化模型）
        ddtheta = ((M+m)*g*np.sin(theta) - np.cos(theta)*F) / (l*(M+m*np.sin(theta)**2))
        dtheta += ddtheta * dt
        theta += dtheta * dt
        
        total_error += abs(theta) * dt
    
    return -total_error  # 最小化误差
```
