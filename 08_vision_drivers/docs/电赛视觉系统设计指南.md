## 1. 相机选型

### 1.1 传感器类型对比

| 特性 | 全局快门 (Global Shutter) | 卷帘快门 (Rolling Shutter) |
|------|--------------------------|---------------------------|
| 运动畸变 | 无 | 有（果冻效应） |
| 高速拍摄 | 优秀 | 一般 |
| 价格 | 较高 | 较低 |
| 推荐场景 | 电赛移动目标检测 | 静态场景 |
| 典型型号 | OV9281, OV2311 | OV5640, IMX219 |

**推荐：全局快门相机**（电赛场景目标多为运动物体）

### 1.2 推荐型号

#### 入门级（50元以内）
- **OV9281 USB模块** - 640×480@120fps，全局快门，USB2.0/3.0
- 适用：基础巡线、色块检测

#### 进阶级（100-200元）
- **OV2311 USB模块** - 640×480@120fps, 1600×1200@60fps，全局快门
- 适用：高精度识别、多目标跟踪

#### 高性能级
- **海康MV-CS系列工业相机** - 支持硬件触发、更高帧率
- 适用：精密测量、高速检测

### 1.3 接口选择

| 接口 | 带宽 | 延迟 | 推荐度 |
|------|------|------|--------|
| USB3.0 | 5Gbps | 低 | ⭐⭐⭐⭐⭐ |
| USB2.0 | 480Mbps | 中 | ⭐⭐⭐ |
| MIPI CSI | 高 | 极低 | ⭐⭐⭐⭐ |
| GbE网口 | 1Gbps | 中 | ⭐⭐⭐ |

**Orange Pi 5 推荐 USB3.0** - 带宽充足，驱动成熟

### 2.1 USB3.0连接要点

```
Orange Pi 5 USB3.0 端口
    │

    └── 供电不足时使用带电源的USB HUB
```

1. **固定稳固** - 使用螺丝/扎带固定，避免振动

## 5. 图像处理算法

### 5.1 自动曝光

```
场景              目标亮度  策略
─────────────────────────────────────
室内竞赛场        128      中心加权
室外晴天          100      点测光+降曝光
弱光环境          150      提增益+提曝光
逆光场景          128      ROI测光
```

### 5.2 HDR场景

- **适用场景**：窗口旁、明暗交界处
- **方法**：3帧包围曝光（EV-2/0/+2），线性加权融合
- **注意**：全局快门无果冻效应，帧间对齐精度高

### 5.3 多帧降噪

- **适用场景**：弱光环境、需要高画质
- **方法**：4-8帧时域中值/均值融合
- **效果**：信噪比提升√N倍（4帧=2倍）

### 10.2 双目摄像头配置

```python
import cv2
import numpy as np
from threading import Thread, Lock

class StereoCameraSystem:
    def __init__(self, cam_left_id=0, cam_right_id=1):
        self.cam_left = cv2.VideoCapture(cam_left_id)
        self.cam_right = cv2.VideoCapture(cam_right_id)

        # 设置相同的分辨率和帧率
        for cam in [self.cam_left, self.cam_right]:
            cam.set(cv2.CAP_PROP_FRAME_WIDTH, 640)
            cam.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)
            cam.set(cv2.CAP_PROP_FPS, 60)
            cam.set(cv2.CAP_PROP_FOURCC, cv2.VideoWriter_fourcc('M', 'J', 'P', 'G'))

        # 帧同步锁
        self.lock = Lock()
        self.frame_left = None
        self.frame_right = None

        # 相机参数（标定得到）
        self.K_left = None   # 左相机内参
        self.K_right = None  # 右相机内参
        self.D_left = None   # 左相机畸变
        self.D_right = None  # 右相机畸变
        self.R = None        # 旋转矩阵
        self.T = None        # 平移向量
        self.Q = None        # 视差-深度映射矩阵

    def capture_sync(self):
        """同步采集双相机图像"""
        with self.lock:
            ret_l, self.frame_left = self.cam_left.read()
            ret_r, self.frame_right = self.cam_right.read()
            return ret_l and ret_r, self.frame_left, self.frame_right

    def calibrate_stereo(self, left_images, right_images, board_size=(9, 6)):
        """双目标定"""
        # 准备标定板3D坐标
        objp = np.zeros((board_size[0]*board_size[1], 3), np.float32)
        objp[:, :2] = np.mgrid[0:board_size[0], 0:board_size[1]].T.reshape(-1, 2)

        obj_points = []
        img_points_left = []
        img_points_right = []

        for img_left, img_right in zip(left_images, right_images):
            gray_left = cv2.cvtColor(img_left, cv2.COLOR_BGR2GRAY)
            gray_right = cv2.cvtColor(img_right, cv2.COLOR_BGR2GRAY)

            ret_l, corners_l = cv2.findChessboardCorners(gray_left, board_size)
            ret_r, corners_r = cv2.findChessboardCorners(gray_right, board_size)

            if ret_l and ret_r:
                obj_points.append(objp)
                img_points_left.append(corners_l)
                img_points_right.append(corners_r)

        # 单目标定
        ret_l, self.K_left, self.D_left, _, _ = cv2.calibrateCamera(
            obj_points, img_points_left, gray_left.shape[::-1], None, None)
        ret_r, self.K_right, self.D_right, _, _ = cv2.calibrateCamera(
            obj_points, img_points_right, gray_right.shape[::-1], None, None)

        # 双目标定
        flags = cv2.CALIB_FIX_INTRINSIC
        ret, self.K_left, self.D_left, self.K_right, self.D_right, self.R, self.T, E, F = \
            cv2.stereoCalibrate(obj_points, img_points_left, img_points_right,
                               self.K_left, self.D_left,
                               self.K_right, self.D_right,
                               gray_left.shape[::-1],
                               criteria=(cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 100, 1e-6),
                               flags=flags)

        # 计算校正映射
        self.R1, self.R2, self.P1, self.P2, self.Q, roi1, roi2 = \
            cv2.stereoRectify(self.K_left, self.D_left,
                            self.K_right, self.D_right,
                            gray_left.shape[::-1],
                            self.R, self.T,
                            alpha=0)

        self.map_left = cv2.initUndistortRectifyMap(
            self.K_left, self.D_left, self.R1, self.P1,
            gray_left.shape[::-1], cv2.CV_16SC2)
        self.map_right = cv2.initUndistortRectifyMap(
            self.K_right, self.D_right, self.R2, self.P2,
            gray_right.shape[::-1], cv2.CV_16SC2)

        return ret

    def compute_disparity(self, frame_left, frame_right):
        """计算视差图"""
        # 校正图像
        rect_left = cv2.remap(frame_left, *self.map_left, cv2.INTER_LINEAR)
        rect_right = cv2.remap(frame_right, *self.map_right, cv2.INTER_LINEAR)

        # 转灰度
        gray_left = cv2.cvtColor(rect_left, cv2.COLOR_BGR2GRAY)
        gray_right = cv2.cvtColor(rect_right, cv2.COLOR_BGR2GRAY)

        # SGBM立体匹配
        window_size = 5
        min_disp = 0
        num_disp = 64  # 必须是16的倍数

        stereo = cv2.StereoSGBM_create(
            minDisparity=min_disp,
            numDisparities=num_disp,
            blockSize=window_size,
            P1=8 * 3 * window_size ** 2,
            P2=32 * 3 * window_size ** 2,
            disp12MaxDiff=1,
            uniquenessRatio=10,
            speckleWindowSize=100,
            speckleRange=32,
            preFilterCap=63,
            mode=cv2.STEREO_SGBM_MODE_SGBM_3WAY
        )

        disparity = stereo.compute(gray_left, gray_right).astype(np.float32) / 16.0
        return disparity, rect_left

    def disparity_to_3d(self, disparity, point_2d):
        """视差转3D坐标"""
        x, y = point_2d
        # 使用Q矩阵进行重投影
        point = np.array([x, y, disparity[y, x], 1.0])
        point_3d = self.Q @ point
        point_3d /= point_3d[3]  # 齐次坐标转换
        return point_3d[:3]  # 返回x, y, z (mm)
```

### 11.2 数据帧协议设计

```python
# 数据帧格式
"""
帧头(1B) | 命令(1B) | 长度(1B) | 数据(NB) | 校验(1B) | 帧尾(1B)
  0xAA   |  CMD    |  LEN    |  DATA   |  XOR   |  0x55
"""

class ProtocolDefine:
    # 帧头帧尾
    HEADER = 0xAA
    TAIL = 0x55

    # 命令定义
    CMD_DETECTION_RESULT = 0x01   # 视觉检测结果
    CMD_GIMBAL_CONTROL = 0x02     # 云台控制命令
    CMD_MOTOR_CONTROL = 0x03      # 电机控制命令
    CMD_SENSOR_DATA = 0x04        # 传感器数据
    CMD_SYSTEM_STATUS = 0x05      # 系统状态
    CMD_ACK = 0x06                # 应答确认

    # 数据格式
    @staticmethod
    def build_frame(cmd, data):
        """构建数据帧"""
        frame = bytearray()
        frame.append(ProtocolDefine.HEADER)
        frame.append(cmd)
        frame.append(len(data))
        frame.extend(data)

        # 计算校验和 (XOR)
        checksum = 0
        for b in data:
            checksum ^= b
        frame.append(checksum)
        frame.append(ProtocolDefine.TAIL)
        return bytes(frame)

    @staticmethod
    def parse_frame(buffer):
        """解析数据帧"""
        if len(buffer) < 5:
            return None, None, None

        if buffer[0] != ProtocolDefine.HEADER or buffer[-1] != ProtocolDefine.TAIL:
            return None, None, None

        cmd = buffer[1]
        length = buffer[2]
        data = buffer[3:3+length]
        checksum = buffer[3+length]

        # 验证校验和
        calc_checksum = 0
        for b in data:
            calc_checksum ^= b
        if checksum != calc_checksum:
            return None, None, None

        return cmd, data, checksum
```

```python
    def send_detection_result(self, x, y, width, height, confidence):
        """发送检测结果"""
        # 数据打包: x(2B), y(2B), w(2B), h(2B), conf(1B)
        data = struct.pack('<hhhhB', int(x), int(y), int(width), int(height), int(confidence*100))
        frame = ProtocolDefine.build_frame(ProtocolDefine.CMD_DETECTION_RESULT, data)
        self.ser.write(frame)

    def send_gimbal_command(self, angle_pan, angle_tilt):
        """发送云台控制命令"""
        data = struct.pack('<hh', int(angle_pan*10), int(angle_tilt*10))  # 0.1度精度
        frame = ProtocolDefine.build_frame(ProtocolDefine.CMD_GIMBAL_CONTROL, data)
        self.ser.write(frame)
```

```c
// UART接收中断
void UART2_IRQHandler(void) {
    uint8_t data = UART2->DR;

    // 帧解析状态机
    static enum {
        STATE_IDLE,
        STATE_HEADER,
        STATE_CMD,
        STATE_LENGTH,
        STATE_DATA,
        STATE_CHECKSUM,
        STATE_TAIL
    } state = STATE_IDLE;

    static uint8_t cmd, length, index, checksum;
    static uint8_t buffer[64];

    switch (state) {
        case STATE_IDLE:
            if (data == 0xAA) state = STATE_HEADER;
            break;
        case STATE_HEADER:
            cmd = data;
            state = STATE_CMD;
            break;
        case STATE_CMD:
            length = data;
            index = 0;
            state = STATE_LENGTH;
            break;
        case STATE_LENGTH:
            buffer[index++] = data;
            if (index >= length) state = STATE_DATA;
            break;
        case STATE_DATA:
            checksum = data;
            state = STATE_CHECKSUM;
            break;
        case STATE_CHECKSUM:
            if (data == 0x55) {
                // 帧完整，处理数据
                Process_Command(cmd, buffer, length);
            }
            state = STATE_IDLE;
            break;
    }
}

// 处理检测结果
void Process_Command(uint8_t cmd, uint8_t *data, uint8_t length) {
    switch (cmd) {
        case 0x01: {  // 检测结果
            int16_t x, y, w, h;
            uint8_t conf;
            memcpy(&x, data, 2);
            memcpy(&y, data+2, 2);
            memcpy(&w, data+4, 2);
            memcpy(&h, data+6, 2);
            conf = data[8];

            // 计算云台角度
            float angle_pan = Pixel_to_Angle_X(x);
            float angle_tilt = Pixel_to_Angle_Y(y);

            // 控制云台
            Gimbal_SetAngle(angle_pan, angle_tilt);

            // 可选：开启激光
            if (conf > 80 && abs(x-320) < 20 && abs(y-240) < 20) {
                Laser_On();
            } else {
                Laser_Off();
            }
            break;
        }
        case 0x02: {  // 云台控制命令
            int16_t angle_pan, angle_tilt;
            memcpy(&angle_pan, data, 2);
            memcpy(&angle_tilt, data+2, 2);
            Gimbal_SetAngle((float)angle_pan/10.0, (float)angle_tilt/10.0);
            break;
        }
    }
}
```

```python
"""
二进制协议:
  优点: 数据量小, 解析快, 无精度损失
  缺点: 调试不直观

文本协议 (JSON/CSV):
  优点: 可读性强, 易调试
  缺点: 数据量大, 解析慢

电赛建议: 使用二进制协议
"""

"""
不是每帧都发送:
  - 只在检测到目标时发送
  - 设置发送间隔 (如50ms一次)
  - 使用变化检测 (坐标变化>阈值才发送)
"""

"""
坐标压缩:
  - 使用相对坐标 (相对于上一帧)

  - 减小数据包大小
```
