# RK3588S 硬件加速使用手册

## 概述

RK3588S集成了三大硬件加速引擎：

| 加速器 | 算力 | 适用场景 | 模块路径 |
|--------|------|----------|----------|
| **RGA** | 2D图形加速 | 缩放/裁剪/旋转/格式转换 | `rga/` |
| **Mali-G610** | 6核GPU | 滤波/边缘检测/颜色处理 | `gpu/` |
| **NPU** | 6 TOPS | AI推理(分类/检测/分割) | `npu/` |

---

## 1. RGA 硬件2D加速

### 1.1 快速开始

```python
from rga.rga_utils import RgaUtils, RgaRotate

rga = RgaUtils()

# 缩放 (1080p→320x240, ~0.8ms, 比OpenCV快15倍)
small = rga.resize(frame, (320, 240))

# 裁剪
roi = rga.crop(frame, (100, 100, 400, 400))

# 旋转
rotated = rga.rotate(frame, RgaRotate.ROT_90)

# 格式转换 (YUYV→BGR)
from rga.rga_convert import RgaConverter, PixelFormat
conv = RgaConverter()
bgr = conv.yuyv_to_bgr(yuyv_frame, 640, 480)
```

### 1.2 RGA缩放专用优化

```python
from rga.rga_resize import RgaResizer

resizer = RgaResizer()

# 保持宽高比缩放
result = resizer.resize(img, (640, 480), keep_aspect=True, pad_value=0)

# 图像金字塔
pyramid = resizer.build_pyramid(img, levels=5, scale_factor=0.5)

# ROI裁剪+缩放 (单次RGA调用)
result = resizer.roi_resize(img, (100, 100, 200, 200), (64, 64))
```

### 1.3 RGA格式转换

```python
from rga.rga_convert import RgaConverter, PixelFormat

conv = RgaConverter()

# USB摄像头 YUYV → BGR
bgr = conv.convert(yuyv_data, PixelFormat.YUYV, PixelFormat.BGR, 640, 480)

# VPU解码 NV12 → BGR
bgr = conv.convert(nv12_data, PixelFormat.NV12, PixelFormat.BGR, 1920, 1080)

# BGR → NV12 (给硬件编码器)
nv12 = conv.bgr_to_nv12(bgr_frame)
```

## 2. Mali GPU 加速

### 2.1 GPU滤波

```python
from gpu.gpu_filter import GpuFilter

gpu = GpuFilter()

# 高斯滤波
blurred = gpu.gaussian_blur(img, (5, 5))

# 中值滤波 (去椒盐噪声)
denoised = gpu.median_blur(img, 5)

# 双边滤波 (保边去噪)
smooth = gpu.bilateral_filter(img, 9, 75, 75)

# 自定义卷积核
import numpy as np
kernel = np.array([[-1,-1,-1],[-1,8,-1],[-1,-1,-1]], dtype=np.float32)
edges = gpu.custom_filter(img, kernel)

# 锐化
sharpened = gpu.sharpen(img, strength=1.5)
```

### 2.2 GPU边缘检测

```python
from gpu.gpu_edge import GpuEdgeDetector

edge = GpuEdgeDetector()

# Canny边缘检测
edges = edge.canny(img, threshold1=50, threshold2=150)

# 自适应Canny (自动阈值)
edges = edge.auto_canny(img, sigma=0.33)

# Sobel梯度
magnitude = edge.sobel(img, dx=1, dy=1, ksize=3)

# 完整Sobel梯度 (含方向)
gx, gy, mag, angle = edge.sobel_gradient(img)

# Scharr高精度梯度
grad = edge.scharr(img)
```

### 2.3 GPU颜色处理

```python
from gpu.gpu_color import GpuColorProcessor

color = GpuColorProcessor()

# 颜色空间转换
hsv = color.convert(img, 'bgr', 'hsv')
lab = color.convert(img, 'bgr', 'lab')

# 颜色检测
mask = color.detect_color(img, 'red')
mask = color.detect_color(img, 'blue')

# 多颜色同时检测
masks = color.detect_multi(img, ['red', 'blue', 'green'])

# 颜色追踪
bbox = color.track_color(img, 'green')  # 返回 (x, y, w, h)

# 主色调提取
colors, percentages = color.dominant_color(img, k=3)

# 自定义颜色范围
color.add_color('my_red', lower=[0, 150, 150], upper=[10, 255, 255])
```

## 3. NPU 推理加速

### 3.1 基本推理

```python
from npu.rknn_inference import RKNNInference

# 加载模型
engine = RKNNInference('yolov5s.rknn')

# 分类推理
results = engine.classify(image, top_k=5, labels=['cat', 'dog', ...])
# 返回 [(class_id, confidence, label), ...]

# 检测推理
detections = engine.detect(image, conf_threshold=0.5)
# 返回 [DetectionResult(class_id, class_name, confidence, bbox), ...]

# 通用推理
blob = engine.preprocess_image(image, input_size=(640, 640))
result = engine.infer(blob)
# result.outputs: List[np.ndarray]
# result.inference_ms: float
```

### 3.2 多模型管理

```python
from npu.rknn_inference import RKNNModelPool, ModelConfig

pool = RKNNModelPool(max_models=4)

# 添加检测模型
pool.add_model('detector', 'yolov5s.rknn', ModelConfig(
    model_path='yolov5s.rknn',
    input_size=(640, 640),
    mean_values=[0, 0, 0],
    std_values=[255, 255, 255],
))

# 添加分类模型
pool.add_model('classifier', 'mobilenetv2.rknn')

# 流水线推理
detections = pool.infer('detector', image)
for det in detections:
    crop = image[det.bbox[1]:det.bbox[3], det.bbox[0]:det.bbox[2]]
    cls_result = pool.infer('classifier', crop)
```

### 3.3 模型量化

```python
from npu.rknn_quantize import RKNNQuantizer, QuantConfig, CalibrationDataset

# 准备校准数据
dataset = CalibrationDataset('./calibration_images/', max_samples=200)

# 配置量化
config = QuantConfig(
    model_path='yolov5s.onnx',
    output_path='yolov5s_int8.rknn',
    quantized_dtype='asymmetric_quantized-8',
    mean_values=[[0, 0, 0]],
    std_values=[[255, 255, 255]],
    reorder_channel='0 1 2',
)

# 执行量化 (在x86主机上执行)
quantizer = RKNNQuantizer()
quantizer.quantize(config, dataset)
```

### 5.2 注意事项

1. **RGA对齐要求**: 输入输出图像宽度需16字节对齐
2. **NPU量化**: INT8模型精度通常损失<1%，但速度提升2x
3. **内存管理**: 大批量处理时及时释放RKNN资源
4. **散热**: 长时间满负载需确保散热，否则会降频
5. **DMA-BUF**: 跨模块(VPU↔RGA↔NPU)传递数据时使用零拷贝
